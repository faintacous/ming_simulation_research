# 明代海上航线（嘉靖至崇祯，1521-1644）

## 概述

明代中后期，尽管名义上实行海禁，但海上贸易从未中断。隆庆元年（1567）开海后，漳州月港成为唯一合法的商人出海港口，繁盛时期拥有 **18条航线**，通往 **47个国家和地区**。海上贸易形成三条主干线：东南亚航线、马尼拉大帆船航线（跨太平洋）、果阿-欧洲航线。同时，走私贸易网络（以双屿港为代表）也在海禁期间繁荣一时。这一时期，大量白银通过三条渠道流入中国，深刻改变了明代经济结构。

---

## 一、月港：明代唯一合法出海港

### 基本信息

- **位置**：福建漳州府龙溪县（今漳州龙海区海澄镇）
- **开放时间**：隆庆元年（1567年），明穆宗批准"准贩东西二洋"
- **功能**：唯一合法的民间商人出海贸易港口
- **别称**："天子南库"（因贡献大量关税收入）

### 18条航线通47国

月港在繁盛时期拥有18条航线，覆盖东南亚、东亚、南亚乃至更远地区的47个国家和地区。

**东洋航线**（主要目的地）：
1. 吕宋（菲律宾马尼拉）——最重要航线
2. 日本长崎
3. 琉球
4. 台湾（鸡笼/淡水）

**西洋航线**（主要目的地）：
5. 占城（越南中部）
6. 暹罗（泰国）
7. 柬埔寨
8. 北大年（马来半岛东岸）
9. 大泥（马来半岛）
10. 麻六甲（马六甲）
11. 爪哇
12. 苏门答腊
13. 渤泥（文莱）
14. 苏禄（菲律宾南部）
15. 美洛居（摩鹿加群岛）
16. 旧港（巨港，苏门答腊南部）
17. 万丹（爪哇西部）
18. 其他（含东帝汶、印度沿岸等）

### 月港的税收贡献

| 年份 | 月港税银 | 福建全省税银 | 占比 |
|------|---------|------------|------|
| 万历四十一年（1613） | **3.5万两以上** | 6万两 | **58%以上** |

## 二、三条主干航线

### 1. 东南亚航线

**主要路线**：
```
漳州月港 → 台湾海峡 → 南海 → 分叉：
  ├─ 吕宋（马尼拉）：约7-10天
  ├─ 占城/暹罗/柬埔寨：约15-20天
  ├─ 北大年/麻六甲：约20-30天
  └─ 爪哇/苏门答腊：约30-40天
```

**贸易特点**：
- **中国输出**：生丝、丝绸、瓷器、铁器、铜钱、纸张、药材
- **中国输入**：胡椒、苏木、象牙、犀角、香料、锡
- 航程利用季风：冬季（东北风）南下，夏季（西南风）北返
- 每年一个贸易周期

**贸易规模**：
- 万历年间，每年从月港出发的商船约 **100-200艘**
- 马尼拉是最大目的地，每年约有 **30-50艘** 中国商船到达

### 2. 马尼拉大帆船航线（跨太平洋线）

**航线构成**（两段接力）：

**第一段：月港-马尼拉（中国商船）**
```
漳州月港 → 台湾南端 → 吕宋海峡 → 马尼拉
航程：约7-10天（顺风）
```

**第二段：马尼拉-阿卡普尔科（西班牙大帆船）**
```
马尼拉 → 太平洋（北纬40度线附近）→ 阿卡普尔科（墨西哥）
东行航程：约4-6个月
西行航程：约2-3个月（利用赤道洋流和信风）
```

**贸易特点**：
- 时间跨度：1565-1815年，延续250年
- 中国商品从月港集中运往马尼拉，再由西班牙大帆船转运美洲
- 主要商品：中国生丝和丝绸 → 美洲白银
- 大帆船排水量高达 **1000-2000吨**，远超同期中国商船
- 每年通常 **1-2艘** 大帆船往返

**航线时间表**：
```
年初（1-2月）：中国商船离开月港，南下马尼拉
3-6月：马尼拉集市交易
6-7月：大帆船从马尼拉东行
次年1-3月：抵达阿卡普尔科
3-4月：阿卡普尔科集市交易
4-5月：大帆船西行返回
年末（11-12月）：抵达马尼拉
```

> 来源：[马尼拉大帆船 - 百度百科](https://baike.baidu.com/item/%E9%A9%AC%E5%B0%BC%E6%8B%89%E5%A4%A7%E5%B8%86%E8%88%B9/3792808)；[世界白银史漫谈7：马尼拉大帆船贸易 - 知乎](https://zhuanlan.zhihu.com/p/678592483)

### 3. 果阿-欧洲航线（葡萄牙航线）

**主要路线**：
```
澳门 → 马六甲海峡 → 印度洋 → 果阿（印度）→ 好望角 → 里斯本（葡萄牙）
```

**航程时间**：
- 澳门至果阿：约 **2-3个月**（顺风）
- 果阿至里斯本：约 **4-6个月**
- 全程单趟：约 **6-9个月**
- 往返一趟需约 **1.5-2年**

**支线——澳门-长崎航线**（葡萄牙转口）：
```
澳门 → 日本长崎
航程：约10-15天
```

- 1557年葡萄牙人获准居留澳门后开辟
- 输出中国丝绸，换回日本白银
- 每年白银输入量高达 **450万两**（峰值估计）

**贸易特点**：
- 葡萄牙人充当中日贸易中间人
- 中国生丝在日本可获利 **100-200%**
- 果阿是葡属印度的首府，也是亚洲贸易网络的中心节点

> 来源：[明朝中后期海外白银输入的三条主要渠道](http://economy.guoxue.com/?p=7514)；[樊树志：全球化贸易与大明帝国 - 知乎](https://zhuanlan.zhihu.com/p/495555259)

## 三、双屿港走私网络

### 基本信息

- **位置**：浙江宁波府外海舟山群岛中的六横岛
- **活跃时期**：嘉靖三年至二十七年（1524-1548），约 **24年**
- **性质**：海禁时期最大的国际自由贸易市场（走私港）

### 港口规模

- 葡萄牙人在此建设完善的港区设施：
  - 市政厅 1座
  - 教堂 1座
  - 医院 1座
  - 民居约 **1000幢**
- 常住人口约 **3000人**，其中葡萄牙人约 **1200名**
- 鼎盛时期"舶客拥万众"，可聚集上万人

### 贸易网络

参与者来自十余个国家和地区：
- 葡萄牙商人
- 日本商人
- 福建/浙江中国商人
- 暹罗商人
- 马来商人
- 以及其他欧亚非洲商人

**贸易额**：
- 葡萄牙人在双屿港与日本的贸易额超过 **300万葡元**
- 商品带往日本可赚取 **3-4倍利润**

### 主要走私商人

- **汪直**（王直）：嘉靖年间最大的海商/海盗首领，曾以双屿港为基地
  - 嘉靖二十四年（1545）到日本博多津，招诱日本商人来双屿港贸易
  - "泊客拥有徒众万余人"

### 覆灭

嘉靖二十七年（1548），浙江巡抚 **朱纨** 派都指挥 **卢镗** 率兵进攻双屿港，将港口建筑全部摧毁，并用石头填塞港口水道，彻底终结了双屿港的贸易活动。

此后走私重心转移至福建沿海和月港一带。

> 来源：[双屿港 - 维基百科](https://zh.wikipedia.org/zh-hans/%E5%8F%8C%E5%B1%BF%E6%B8%AF)；[双屿港，存在于讲述中的海岛 - 三联](https://www.lifeweek.com.cn/h5/article/detail.do?artId=193403)

## 四、白银输入的三条渠道

### 渠道概览

明代中后期，白银通过三条主要渠道大量流入中国，为"一条鞭法"等财税改革提供了货币基础。

| 渠道 | 路线 | 白银来源 | 流入方式 |
|------|------|---------|---------|
| 日本渠道 | 中日直接贸易 + 澳门-长崎转口 | 日本石见银山等 | 贸易顺差 |
| 马尼拉渠道 | 月港-马尼拉-阿卡普尔科 | 美洲（墨西哥/秘鲁银矿） | 丝银贸易 |
| 欧洲渠道 | 里斯本-好望角-果阿-澳门 | 美洲（经欧洲转运） | 贸易顺差 |

### 各渠道白银流入数据

#### 日本渠道

| 项目 | 数据 |
|------|------|
| 明清时期总流入量 | 约 **2亿两** |
| 中日直接贸易占比 | 约1/3（约6700万两） |
| 澳门-长崎转口占比 | 约2/3（约1.3亿两） |
| 澳门-长崎年均输入（峰值） | 约 **450万两/年** |

#### 马尼拉渠道（美洲白银经太平洋）

| 时期 | 美洲白银流入量（吨） | 备注 |
|------|---------------------|------|
| 1590-1602年 | 约 **2,010吨** | 初期 |
| 1602-1636年 | 约 **2,400吨** | 高峰期 |
| 1636-1644年 | 约 **210吨** | 衰退期 |
| **合计** | **约4,620吨** | 约1.2亿两 |

- 万历二十八年（1600）后，每年从马尼拉经澳门流入的白银最低 **200万两**

#### 欧洲渠道（经好望角）

- 主要通过葡萄牙商人将美洲白银经欧洲、印度洋运至澳门
- 与马尼拉渠道合计，美洲白银流入中国约 **4亿两**（明清合计）

### 白银流入总量估算

| 来源 | 明清时期总量（约） |
|------|-------------------|
| 日本白银 | 2亿两 |
| 美洲白银（经马尼拉） | 2亿两 |
| 美洲白银（经欧洲/印度洋） | 2亿两 |

## 五、主要出口商品

### 丝绸

- 明代出口商品中丝绸居首位
- 主要产地：苏州、杭州、湖州、南京
- 以生丝出口量最大，其次为织成品
- 在马尼拉市场，中国生丝可换取等重白银

### 瓷器

- 景德镇为最大生产中心
- 明代中后期每年产瓷约 **18万担**
- 外销瓷以青花瓷为主
- 通过月港、澳门等港口出口

### 其他出口商品

- 棉布、茶叶、铁器、铜钱
- 纸张、漆器、药材
- 金锭（少量）

## 六、主要进口商品

### 白银

- 最重要的"进口商品"
- 以西班牙银元（"番银"）和日本银锭为主
- 流入白银成为明代货币白银化的物质基础

### 其他进口商品

- **胡椒**：东南亚主要出口品，明初曾作为官员俸禄发放
- **香料**：丁香、肉豆蔻、檀香
- **苏木**：东南亚产，用于染料和药材
- **象牙、犀角**：奢侈品
- **锡**：马来半岛出产
- **玳瑁、珊瑚**：装饰品原料

## 七、海上贸易的管理制度

### 引票制度

隆庆开海后，月港实行引票管理：
- 商船出海需向海防馆领取"船引"（执照）
- 船引注明船只大小、目的地、船员人数
- 限定每年出海船引数量（初期约50张，后增至约100张以上）
- 需缴纳"引税"

### 市舶司与海防馆

- 福建海防馆（月港）：负责出海贸易管理
- 广东市舶司：管理朝贡贸易（后逐渐转为商业管理）
- 浙江市舶司：嘉靖年间废止后恢复

### 贸易禁区

即使开海后，仍有限制：
- 禁止私运军器、硝磺等违禁品
- 禁止前往日本（因倭寇问题，实际上私下贸易不断）
- 限定船只大小和武装
