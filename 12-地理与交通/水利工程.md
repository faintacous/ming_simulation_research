# 明代水利工程（嘉靖至崇祯，1521-1644）

## 概述

明代水利工程的核心任务是"治黄保运"——治理黄河以保障漕运畅通。黄河自南宋以来夺淮入海，其泛滥直接威胁京杭运河的通航。明代先后出现万恭、潘季驯等杰出水利专家，潘季驯四度总理河道，创造性地提出"束水攻沙"理论和三堤体系。南旺分水工程（戴村坝）解决了运河最高段的水源问题，是古代水利工程的杰作。运河沿线的分段管理和疏浚维护制度，保障了漕运体系的长期运转。

## 一、潘季驯四度治河

### 人物简介

- **姓名**：潘季驯（1521-1595）
- **字**：时良、惟良
- **号**：印川
- **籍贯**：浙江湖州府归安县（今湖州市）
- **科举**：嘉靖二十九年（1550）进士
- **治河生涯**：嘉靖四十四年至万历二十年（1565-1592），27年间四次出任总理河道，实际治河近 **12年**
- **著作**：《河防一览》（14卷），系统总结治河经验

### 四次治河时间线

#### 第一次：嘉靖四十四年至隆庆四年（1565-1570）

| 项目 | 内容 |
|------|------|
| 背景 | 嘉靖四十四年黄河大决沛县，淹没昭阳湖，运道淤塞百余里 |
| 职务 | 佥都御史，总理河道 |
| 主要任务 | 配合工部尚书朱衡治河 |
| 工程 | 参与开凿南阳新河（自南阳河口至留城，全长约70公里） |
| 方案分歧 | 潘主张恢复黄河故道，朱衡主张继续开凿盛应期旧渠，最终采纳朱衡方案 |

#### 第二次：万历四年至六年（1576-1578）

| 项目 | 内容 |
|------|------|
| 背景 | 黄河再次决口，运河受阻 |
| 职务 | 右都御史，总理河道 |
| 主要措施 | 开始实践"束水攻沙"理论，初步筑堤约束河道 |
| 成效 | 取得一定成效，但因朝中矛盾被调离 |

#### 第三次：万历六年至八年（1578-1580）——最重要

| 项目 | 内容 |
|------|------|
| 背景 | 张居正主政时期，获得充分支持 |
| 职务 | 右都御史兼工部左侍郎，总理河道 |
| 核心工程 | 全面推行"束水攻沙"和"蓄清刷黄"方案 |
| 具体措施 | ①修筑高家堰蓄淮清水；②大规模修筑缕堤约束黄河；③建遥堤防洪；④设格堤淤滩 |
| 成效 | **整治后黄河十余年未发生大的决溢**，河道基本稳定 |

#### 第四次：万历十六年至二十年（1588-1592）

| 项目 | 内容 |
|------|------|
| 背景 | 万历十五年黄河再次决口 |
| 职务 | 右都御史兼工部右侍郎，总理河道 |
| 核心工程 | 大筑黄河下游三省（河南、山东、南直隶）长堤 |
| 具体措施 | 将黄河两岸大堤全部连接起来，形成完整的堤防体系 |
| 成效 | 河道基本趋于稳定，河患显著减少 |
| 结局 | 万历二十年被免职回乡，三年后去世 |

## 二、"束水攻沙"三堤体系

### 理论基础

潘季驯在万恭《治水筌蹄》的基础上，系统发展出"束水攻沙"和"蓄清刷黄"两大治河理论：

**束水攻沙**：
> 将黄河约束在一条河道之内，不分流，以水流汇聚之力冲刷河底泥沙，确保河道深度。这一理论反对传统的"分流减势"做法，主张"合流攻沙"。

**蓄清刷黄**：
> 修筑高家堰，将东流的淮河拦腰截断，蓄积巨大水库（洪泽湖），在与黄河交汇处留出一个小豁口，使清水喷薄宣泄而出，抵御黄河浊流泥沙。

### 四种堤防体系

潘季驯创造性地将堤防工程分为四种类型，因地制宜配合使用：

#### 1. 缕堤（近堤）

| 项目 | 内容 |
|------|------|
| 位置 | 紧逼水滨的河道两岸 |
| 功能 | **核心堤防**。约束河水在主河道中流动，提高流速，冲刷泥沙 |
| 特点 | 堤身坚固、高大，直接面对水流冲击 |
| 与"束水攻沙"关系 | 缕堤是束水攻沙的主要工程手段 |

#### 2. 遥堤（远堤）

| 项目 | 内容 |
|------|------|
| 位置 | 缕堤之外数里至十余里处 |
| 功能 | **防洪堤防**。一旦缕堤溃决，遥堤阻止洪水向两岸扩散 |
| 特点 | 高度与缕堤相当或略低，但更为厚实 |
| 作用 | 作为最后防线，保护两岸农田和村庄 |

#### 3. 格堤

| 项目 | 内容 |
|------|------|
| 位置 | 遥堤与缕堤之间，**横向修筑** |
| 功能 | 将遥堤与缕堤之间的空间分隔成若干格子 |
| 作用 | 万一缕堤冲决，横流遇格即止，水退沙留，可以**淤滩**，起到淤滩刷槽的效果 |
| 特点 | 间隔修筑，不必连续 |

#### 4. 月堤

| 项目 | 内容 |
|------|------|
| 位置 | 遥堤、缕堤之外的特别险要河段 |
| 功能 | 额外加固的堤防，形如半月 |
| 特点 | 仅在特别危险的地段修筑 |

### 三堤体系示意

```
河道中心（水流）
  │
  ├── 缕堤（近堤）── 紧贴河道，束水攻沙
  │     │
  │     ├── 格堤 ── 横向分隔（淤滩刷槽）
  │     │
  ├── 遥堤（远堤）── 防洪最后防线
  │
  ├── 月堤（特殊段）── 额外加固
```

## 三、万恭治河

### 人物简介

- **姓名**：万恭
- **字**：肃卿
- **籍贯**：江西南昌
- **科举**：嘉靖二十三年（1544）进士
- **著作**：《治水筌蹄》——中国水利史上的重要著作

### 治河经历

万恭任河道总理的时间虽短，但贡献显著：

#### 主要治河措施

1. **治理淮南运河（里运河）**：
   - 扬州至淮安段是京杭运河最重要的河段之一
   - 疏浚河道，加固堤防

2. **改筑南运口**：
   - 南运口位于清江（今淮安市西），是淮南运河与黄河交汇的枢纽
   - 在原天妃口重建石闸
   - 闸门管理原则："运尽而黄水盛发，则闭闸绝黄；水落则启天妃闸以利商船"
   - 有效防止黄河泥沙倒灌运河

3. **整治会通河减水闸**：
   - 改建引水工程
   - 完善闸坝管理

### 《治水筌蹄》的理论贡献

万恭的核心贡献在于理论创新：
- 首次系统提出利用水流力量冲刷泥沙的思想（"以水治沙"）
- 反对传统的"分流减势"理论
- 其思想被潘季驯继承和发展，成为"束水攻沙"理论的先驱
- 清代大型水利资料汇编《行水金鉴》大量引用《治水筌蹄》

## 四、运河分段管理与疏浚

### 分段管理体制

#### 管理机构演变

| 时期 | 管理方式 | 备注 |
|------|---------|------|
| 永乐年间 | 漕运总兵官（陈瑄）统管河漕 | 军政合一 |
| 成化七年（1471） | 设河道总督（王恕），驻济宁 | 河漕分管 |
| 成化年间 | 漕河分三段管理 | 工部派郎中提督 |
| 嘉靖以后 | 河道总督+分段专官+各地方官 | 体制完善 |

#### 分段管理制度

成化年间，将运河分为三段管理：

| 段别 | 范围 | 管理机构 |
|------|------|---------|
| 北段 | 通州至临清 | 工部都水司派郎中提督 |
| 中段 | 临清至济宁至淮安 | 工部都水司派郎中提督 |
| 南段 | 淮安至杭州 | 工部都水司派郎中提督 |

**关键节点的专官**：
- 徐州洪：工部主事1人驻守
- 吕梁洪：工部主事1人驻守
- 南旺闸、沽头闸、庆丰闸：各设管闸主事
- 临清、济宁等闸和泉源：工部外派司官管理

### 疏浚工程

运河疏浚是持续性的大工程：

#### 疏浚频率

- 常规疏浚：每年春季（漕运前）进行例行疏浚
- 大规模疏浚：视淤塞程度不定期进行
- 应急疏浚：黄河决口后的紧急清淤

#### 主要疏浚工程案例

| 时间 | 地点 | 内容 |
|------|------|------|
| 永乐九年（1411） | 会通河全线 | 工部尚书宋礼主持，修复元代旧河道 |
| 嘉靖四十五年（1566） | 南阳新河 | 朱衡主持，开凿70公里新河道 |
| 万历三十二年（1604）前后 | 泇运河 | 避开黄河影响的新运河段 |
| 崇祯年间 | 多段 | 因财政困难，疏浚不力 |

### 闸坝管理

运河沿线设有大量水闸（闸坝），用于调节水位：

- **会通河段**：闸门最密集，因地势高需要精密调水
  - 自济宁至临清设闸约 **38座**
  - 每座闸设闸官和闸夫管理
  - 采用"顺开倒闭"的操作方式

- **里运河段**：设有洪泽湖调水闸门系统

## 五、会通河与南旺分水工程

### 会通河概况

会通河是京杭运河中最关键也最困难的河段：
- **范围**：山东临清至台儿庄（济宁段为核心）
- **长度**：约500公里
- **难点**：地处运河最高点（水脊），南旺比北边临清高 **30米**，比南边徐州高 **40米**
- **水源不足**：地势高而水源有限，常处于浅水或无水状态

### 南旺分水工程

#### 工程背景

永乐九年（1411），工部尚书 **宋礼** 主持治理运河，遇到南旺段水源不足的难题。后遇汶上县民间水利专家 **白英**，在其建议下建设了南旺分水枢纽。

#### 白英其人

- 汶上县运河民夫领班
- 数十年行船和治水经验
- 对汶水流域水文、地形了如指掌
- 提出了天才的分水方案

#### 工程核心——戴村坝

| 项目 | 数据 |
|------|------|
| 位置 | 山东东平县南城子村 |
| 功能 | 截断大汶河，引水入南旺分水口 |
| 全长 | **1,599.5米** |
| 结构 | 主石坝 + 窦公堤 + 泄洪坝 三部分 |
| 主石坝长 | 443米，略成弧形 |

**主石坝的精妙设计**（三级溢流）：

| 坝段 | 位置 | 高度关系 |
|------|------|---------|
| 乱石坝 | 中间 | **最低**（先溢流） |
| 滚水坝 | 左侧 | 比乱石坝高 **10厘米** |
| 玲珑坝 | 右侧 | 比滚水坝高 **20厘米** |

- 三坝随大汶河水位升降**分级漫水**：枯水时只有乱石坝溢流，丰水时三坝依次漫水
- 这种分级溢流设计精妙控制了入运河的水量

#### 分水原理——"三七分水"

```
大汶河水 → 戴村坝拦截
  ↓
人工开挖小汶河（约30公里）→ 引水至南旺分水口
  ↓
南旺分水口：
  ├── 七分北流 → 供应北段运河（至临清/天津/北京）
  └── 三分南流 → 供应南段运河（至徐州/淮安）
```

- 利用地势落差（汶水高于南旺约 **13米**）实现自然引水
- "七分朝天子（北），三分下江南（南）"

#### 配套工程——水柜与陡门

- **水柜**：在汶上、东平等地滨湖区域设蓄水库（"水柜"），利用安山湖、南旺湖等天然湖泊蓄水
- **陡门**（水闸）：控制水量进出
- 管理原则："柜以蓄泉，门以泄涨"——丰水期蓄水，枯水期放水

#### 工程效果

| 项目 | 数据 |
|------|------|
| 完工后首年漕运量 | **646万石** |
| 此后常年运量 | 约 **400万石** |
| 比元代提升 | 约 **10倍** |

## 六、黄河改道对漕运的影响

### 黄河夺淮入海的格局

南宋建炎二年（1128）以来，黄河夺淮入海，下游河道不断南移，至明代已严重影响运河通航。黄河在苏北与运河交叉，泥沙淤塞是运河最大的威胁。

### 嘉靖至万历间的重大事件

#### 嘉靖五年至七年（1526-1528）

- 黄河水骤涨，"清河以北，兖州以南，水势弥茫，田庐淹没"
- 嘉靖六年：黄河水溢入漕渠，沛北庙道口淤塞数十里，漕船受阻
- 嘉靖七年：总河盛应期奏请于昭阳湖东开凿新河，但工程半途而废

#### 嘉靖四十四年（1565）——关键转折

- **事件**：黄河大决沛县，漫昭阳湖，由沙河入二洪，"浩渺无际"
- **后果**：运道淤塞百余里，漕运完全中断
- **应对**：朱衡、潘季驯联合治河
- **工程**：开凿南阳新河（自南阳河口至留城，约 **70公里**）
- **意义**：这是明代运河避开黄河影响的第一次重大改线

#### 隆庆三年至四年（1569-1570）

- 黄河在沛县再次决堤
- 2000余艘漕船阻塞在邳州
- 隆庆四年九月，黄河在邳州决堤，大量漕船梗阻、人员伤亡

#### 万历年间——泇运河的开凿

- **万历三年（1575）**：河道总督傅希挚首提开凿泇运河，被户部驳回
- **总河舒应龙**：开凿韩庄运河，连接微山湖与泇河
- **万历三十二年（1604）**：再次提议
- **万历三十四年（1606）**：总河曹时聘完成泇运河工程
  - "大加展拓，建坝遏沙，修堤度纤，置邮驿，设兵巡，增河官，立公署"
  - 泇运河绕开了黄河最危险的徐州段
  - 大大降低了黄河对漕运的影响

### 治河理念的转变

| 时期 | 主流理念 | 代表人物 |
|------|---------|---------|
| 嘉靖前期 | 分流减势（分散黄河水量） | 传统做法 |
| 嘉靖末期 | 合流攻沙（束水集中冲沙） | 万恭（先驱） |
| 万历年间 | 束水攻沙+蓄清刷黄 | 潘季驯（集大成） |

## 七、其他重要水利工程

### 高家堰（洪泽湖大堤）

- 位于淮安以西，拦截淮河形成洪泽湖
- 潘季驯重点加固，用于"蓄清刷黄"
- 堤长约60华里
- 蓄水后形成巨大水库，以清水冲刷黄河入海泥沙

### 归仁堤

- 保护运河免受黄河洪水侵袭的重要堤防
- 位于今江苏宿迁一带

### 里运河堤防

- 淮安至扬州段运河东西两岸修筑堤防
- 防止运河水外溢和外水侵入
- 万恭曾重点治理此段
