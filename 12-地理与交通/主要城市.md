# 明代主要城市（嘉靖至崇祯，1521-1644）

## 概述

明代中后期是中国城市发展的重要时期。北京和南京作为两京，是全国的政治中心；苏州、杭州、扬州等江南城市是经济文化中心；广州、佛山是对外贸易和手工业重镇；景德镇、临清、淮安则因特殊产业或交通地位而繁荣。万历年间中国总人口约 **1.5亿**，城市化水平在世界范围内处于领先地位。

## 二、南京——陪都

### 城垣规模

南京城是明初都城，永乐迁都后仍保留完整的六部等衙门建制，为"陪都"。南京城墙是世界上现存最长的古代城墙。

#### 京城城墙

| 项目 | 数据 |
|------|------|
| 修建时间 | 洪武二年至二十六年（1369-1393） |
| 周长 | **35.267公里**（世界最长） |
| 围合面积 | **55平方公里** |
| 城墙高度 | 14-21米 |
| 城墙厚度 | 顶部约4-9米 |
| 城门 | **13座**：正阳门、通济门、聚宝门（今中华门）、三山门、石城门、清凉门、定淮门、仪凤门、钟阜门、金川门、神策门、太平门、朝阳门 |

#### 外郭城

| 项目 | 数据 |
|------|------|
| 周长 | 超过 **60公里** |
| 城门 | **18座** |
| 功能 | 城防外围，多为土墙 |

### 人口

| 时期 | 人口数据 | 备注 |
|------|---------|------|
| 洪武年间 | 约47万户 | 含应天府各县 |
| 嘉靖至万历 | 估计 **50-70万** | 城区人口 |

### 经济功能

- 织造业：设南京织造局，生产宫廷用丝绸
- 商业：秦淮河两岸商业繁荣
- 科举：南京国子监为全国最高学府之一
- 出版：金陵刻书为全国三大刻书中心之一

> 来源：[明南京城 - 百度百科](https://baike.baidu.com/item/%E6%98%8E%E5%8D%97%E4%BA%AC%E5%9F%8E/4682853)

## 三、苏州——江南经济中心

### 城市概况

| 项目 | 数据 |
|------|------|
| 行政级别 | 苏州府（应天巡抚驻地） |
| 府辖范围 | 吴县、长洲县、吴江县、昆山县、常熟县、嘉定县、太仓州等 |
| 人口（全府） | 约 **235万人**（占南直隶约1/5） |
| 城市化水平 | "城郭者十之四五，市镇者十之三四，乡村者十之一二" |
| 城区估计人口 | **50万以上** |

### 经济功能

- **丝织业**："东北半城皆居机户"
  - 官营织造局：匠丁约 **1,700人**
  - 民间织工：数千至上万人
  - 与南京、杭州合称三大织造基地，供应全国 **80%** 的生丝与丝织品
  - 万历年间出现织工暴动（资本主义萌芽的标志性事件）
- **棉纺织业**：嘉定县为棉纺中心，产品销往全国
- **商业**："苏州样"成为全国消费时尚的标杆
- **出版印刷**：与金陵并称刻书中心

> 来源：[苏州、杭州丝织业中的资本主义萌芽](http://economy.guoxue.com/?p=4552)；[明朝苏州府人口有多少？](http://www.qulishi.com/article/201903/323907.html)

## 五、扬州——盐业之都

### 城市概况

| 项目 | 数据 |
|------|------|
| 行政级别 | 扬州府 |
| 城区人口 | 估计 **数十万** |
| 核心地位 | 两淮盐运使司驻地、运河与长江交汇处 |

### 经济功能

- **盐业**：两淮盐业是明代最大的盐产区
  - "煮海之利，重于东南，两淮为最"
  - 万历年间盐商达 **100余家**
  - 盐商资本超过 **3,000万两白银**
  - 盐商发展为全国三大商业资本集团之一
- **运河枢纽**：南北商品在此中转
- **文化**：因盐商富裕，文化艺术繁荣

> 来源：[扬州盐商与苏商文化](https://jssdfz.jiangsu.gov.cn/n100/20210511/i6425.html)；[《说扬州》：论明清商业之都的沉浮 - 新华网](http://www.news.cn/book/20230103/c979fd01b65540bdab5437fa932bdf33/c.html)

| 项目 | 数据 |

- **造船业**：广东沿海造船业发达

## 七、佛山——手工业重镇

### 城市概况

| 项目 | 数据 |
|------|------|
| 行政级别 | 南海县属镇（非府城，但经济规模超过许多府城） |
| 地位 | 与汉口、景德镇、朱仙镇并称"天下四大镇" |

### 经济功能

- **冶铁业**：
  - "两广铁货所都，七省需焉"
  - 从事铁器制造达 **万余家**
  - 主要产品：铁锅、铁钉、铁线、针
  - 明朝政府规定广东所有铁矿石必须运至佛山统一冶炼
  - 铁器不仅内销全国，还通过广州出口海外

- **陶瓷业**：
  - 佛山石湾窑享誉海内外
  - "凡广州陶器皆出石湾"（屈大均《广东新语》）
  - 天启年间发展到八大行业，后扩至二十余行
  - 石湾陶瓷大量出口东南亚

> 来源：[自古广佛本是一家 - 广州市政府](https://www.gz.gov.cn/zlgz/whgz/content/post_8814877.html)；[近代历史时期佛山经济的衰落与调适发展](https://m.fx361.com/news/2019/0328/4961010.html)

## 八、景德镇——瓷都

### 城市概况

| 项目 | 数据 |
|------|------|

| 性质 | 全国最大的制瓷中心 |
| 人口 | "民以陶为业" |

### 瓷器生产规模

| 项目 | 数据 |
|------|------|
| 官窑（御窑厂） | **58座** |
| 民窑 | **900余座** |
| 从业人口 | **10余万人** |
| 年产量 | 约 **18万担**（每担约200个碗） |

**重要生产数据**：

### 御窑厂

- 洪武二年（1369）设立，专为皇室烧造瓷器
- 嘉靖至万历年间产量最大
- 主要产品：青花瓷、五彩瓷、斗彩瓷
- 御窑产品不仅供宫廷使用，也用于赏赐和外交

## 九、临清——运河第一商埠

### 城市概况

| 项目 | 数据 |
|------|------|
| 行政级别 | 临清州（东昌府） |
| 位置 | 南运河与会通河交汇处 |
| 人口（明末极盛期） | 约 **百万**（含流动人口），土著约10万 |

### 经济功能

- **运河枢纽**：南北商品在此中转
- **钞关税收**：万历六年（1578）临清钞关征税 **83,000余两**
  - 居全国八大钞关之首
  - 占全国课税额的 **四分之一**
  - 是同年山东省税收（8,860两）的近 **10倍**
- **商业规模**：
  - 店铺至少 **600家以上**，极盛期大小店铺超千家
  - 徽商占从事工商业人数的 **90%**
- **特色产业**：临清砖（贡砖），用于紫禁城等皇家建筑
- **地位**："北方最大的商业都会"（隆庆-万历时期）

> 来源：[临清运河钞关 - 维基百科](https://zh.wikipedia.org/zh-hans/%E4%B8%B4%E6%B8%85%E8%BF%90%E6%B2%B3%E9%92%9E%E5%85%B3)；[临清运河钞关：见证一城芳华600年 - 齐鲁网](https://liaocheng.iqilu.com/lcyaowen/2017/0401/3481427.shtml)

## 十、淮安——漕运枢纽

### 城市概况

| 项目 | 数据 |
|------|------|
| 行政级别 | 淮安府 |
| 特殊地位 | 漕运总督驻地 |
| 位置 | 运河与黄淮交汇处 |

### 经济功能

- **漕运管理中心**：漕运总督、漕运总兵均驻扎于此

- **钞关**：淮安板闸设有淮关
- **盐务**：两淮盐运的重要节点
- **商业**：因漕运和盐务而成为"运河名都"
- 明代与扬州并列为"长江以北少数几个最繁荣的城市"

## 十一、其他重要城市简述

### 武昌/汉口

- 湖广布政使司驻地（武昌）
- 汉口因长江与汉水交汇而成为"九省通衢"
- 商业枢纽，连接上下游

### 成都

- 四川布政使司驻地
- 蜀锦、蜀绣产地
- 川茶集散中心

### 开封

- 河南布政使司驻地
- 明代仍为中原重镇
- 崇祯十五年（1642）黄河决堤被淹

### 太原

- 山西布政使司驻地
- 晋商大本营
- 连接蒙古贸易的后方基地

### 西安

- 陕西布政使司驻地
- 西北军事重镇
- 茶马古道东端起点之一
