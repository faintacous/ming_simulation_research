# 明代关隘与要塞（嘉靖至崇祯，1521-1644）

## 概述

明代长城防御体系是中国古代军事工程的巅峰之作。从东部山海关到西部嘉峪关，万里长城上设有众多关隘要塞，构成了抵御北方游牧民族的核心防线。其中山海关、居庸关、紫荆关、雁门关、嘉峪关五大关隘各具特色，承担着不同的战略功能。明代在长城沿线设九大军镇（"九边"），常年驻军约 **80万**，关隘是这一庞大防御体系的关键节点。

## 一、山海关——"天下第一关"

### 基本信息

| 项目 | 数据 |
|------|------|
| 位置 | 河北省秦皇岛市东北15公里处 |
| 始建 | 明洪武十四年（1381），大将军 **徐达** 主持修筑 |
| 命名 | 依山（燕山）襟海（渤海），故名山海关 |
| 战略地位 | "边郡之咽喉，京师之保障"，明长城东端起点 |
| 别称 | "天下第一关"、中国长城"三大奇观之一" |

### 城墙规模

| 项目 | 数据 |
|------|------|
| 关城周长 | 约 **4公里**（4,796米） |
| 城墙高度 | **11.6-14米** |
| 城墙厚度 | 顶部 **12-15米**，基部更宽 |
| 城墙材质 | 夯土筑成，内外包城砖 |
| 城门 | **4座**（东、西、南、北各一） |
| 东门楼 | "天下第一关"匾额所在 |

### 防御设施

- 瓮城、箭楼、角楼、敌台
- 城外深壕
- 东面连接长城直达老龙头入海处
- 西面连接长城延伸至角山
- 形成"城、关、山、海"一体的防御体系

### 驻军

| 时期 | 驻军规模 | 备注 |
|------|---------|------|
| 洪武十四年初建 | **15,100人** | 燕山等卫所屯兵 |
| 万历至天启年间 | 数万人 | 关宁锦防线核心 |
| 崇祯末年 | 吴三桂部约 **5万** | 最后的关宁军 |

### 修建沿革时间表

| 时间 | 事件 |
|------|------|
| 洪武十四年（1381） | 徐达筑城建关设卫，修筑三十二关 |
| 洪武十五年（1382） | 改名山海卫 |
| 成化年间 | 增修敌台和箭楼 |
| 嘉靖年间 | 加固城墙，增设防御工事 |
| 万历年间 | 配合辽东防线加强防御 |
| 天启年间 | 孙承宗构筑关宁锦防线，山海关为后方基地 |
| 崇祯十七年（1644） | 吴三桂引清军入关，中国历史转折点 |

### 战略位置

- 距北京（明都城）：**380公里**
- 距盛京（后金/清都城，今沈阳）：**461公里**
- 正好位于北京与盛京之间，是连接关内关外的唯一安全通道
- 辽西走廊最窄处仅 **8公里**（山海关段），山海之间无法迂回

### 历史战役

#### 宁远大捷（天启六年/1626）

- 后金努尔哈赤率约 **6万** 大军进攻宁远城
- 袁崇焕以红夷大炮击退后金军
- 努尔哈赤负伤，数月后病亡
- 山海关作为后方保障了宁远的坚守

#### 宁锦大捷（天启七年/1627）

- 皇太极率军进攻锦州和宁远
- 明军以城防和大炮击退清军
- 关宁锦防线的完整性得到验证

#### 山海关之战（崇祯十七年/1644）

- 李自成攻占北京后，率军进攻山海关
- 吴三桂开关迎入清军（多尔衮部约 **8万** 骑兵）
- 吴三桂与清军联合击败李自成起义军
- 此战决定了明清易代的走向

## 二、居庸关——京师北大门

### 基本信息

| 项目 | 数据 |
|------|------|
| 位置 | 北京市昌平区 |
| 始建 | 秦代已有关口，现存关城为明洪武元年（1368）重建 |
| 修建者 | 大将军 **徐达**、副将军 **常遇春** |
| 战略地位 | 京师西北门户，"内三关"之首 |
| 别称 | "天下九塞"之一 |

### 城墙规模

| 项目 | 数据 |
|------|------|
| 城垣高度 | **四丈二尺**（约13.4米） |
| 城垣厚度 | **两丈五尺**（约8米） |
| 城门 | **南北两座**券城 |
| 重门 | 2座 |
| 城楼 | 各5间 |
| 敌楼 | 四面共 **15座** |

### 防御体系

居庸关不仅是一座关城，而是一个完整的防御区域：

| 项目 | 数据 |
|------|------|
| 管辖范围 | 东西长 **105公里**，南北宽 **90公里** |
| 长城长度 | **75公里** |
| 城池 | **6座** |
| 屯堡 | **61座** |
| 驿站 | **4个** |

### 修建沿革时间表

| 时间 | 事件 |
|------|------|
| 洪武元年（1368） | 徐达、常遇春督建关城 |
| 永乐年间 | 增修加固 |
| 景泰年间 | 因土木之变后加强防御 |
| 嘉靖年间 | 增修敌台、烽燧 |
| 万历年间 | 配合蓟镇防线全面加固 |

### 军事功能

- 与八达岭构成双重防线
- 南口为关沟的南端入口
- 控制北京通往张家口、大同的交通要道
- 土木之变（正统十四年/1449）后防御地位大幅提升
- 隶属蓟镇（蓟州镇），是京师防卫的第一道屏障

## 三、紫荆关——内三关的核心

### 基本信息

| 项目 | 数据 |
|------|------|
| 位置 | 河北省易县紫荆岭上 |
| 始建 | 战国时期，历代沿用 |
| 历代名称 | 秦汉称上谷关，东汉名五阮关，宋金称金坡关，元以后称紫荆关 |
| 战略地位 | "内三关"之一（与居庸关、倒马关合称） |
| 别称 | "紫塞金城" |

### 城墙规模

| 项目 | 数据 |
|------|------|
| 长城总长（易县段） | **21公里** |
| 关城面积 | 约 **3平方公里** |
| 城墙保存长度 | 约 **12,000米**（1.2万米） |
| 城门 | **3座** |
| 水门 | **3座** |
| 敌楼 | **2座** |
| 特色 | **九门九关**石头城（全国独一无二） |

### 修建沿革时间表

| 时间 | 事件 |
|------|------|
| 洪武初年 | 改用石条作基础，以砖砌面封顶，用石灰碎石灌注 |
| 正统年间 | 因土木之变后加固 |
| 景泰年间 | 扩建 |
| 弘治年间 | 改筑加固 |
| 嘉靖年间 | 修缮 |

### 关城石匾

- 北门："紫荆关""河山带砺"（明万历年间书刻）
- 南门："紫塞金城"（明万历年间书刻）

### 军事功能

- 控制太行山东麓通往北京的要道
- 与居庸关、倒马关形成"内三关"防御体系
- 内长城从居庸关向西南延伸至紫荆关、倒马关
- 土木之变时也先曾由紫荆关方向进逼北京

## 四、雁门关——中原北方屏障

### 基本信息

| 项目 | 数据 |
|------|------|
| 位置 | 山西省代县县城西北，雁门山腰 |
| 始建 | 唐朝（现存关城） |
| 现存关城修建 | 明洪武七年（1374） |
| 战略地位 | 内长城重要关口，"中华第一关" |
| 别称 | "九塞尊崇第一关" |

### 城墙规模

| 项目 | 数据 |
|------|------|
| 关城围城周长 | **5公里多** |
| 城墙高度 | **二丈**（约6.4米） |
| 城门 | **3座**：东门、西门、小北门 |
| 城墙材质 | 石座砖身，雉堞为齿 |
| 外围石墙 | 大石墙 **3道**，小石墙 **25道** |

### 防御设施

- 围城随山势而建，南端与关城东西两翼相连
- 城墙向北沿山脊延伸到谷底
- 关城正北山岗上有明清驻军 **营房旧址**
- 东南有 **校场**（练兵场）
- 万历年间复筑门楼

### 修建沿革时间表

| 时间 | 事件 |
|------|------|
| 唐代 | 始建 |
| 洪武七年（1374） | 重建现存关城 |
| 正统年间 | 加固 |
| 嘉靖年间 | 修缮扩建 |
| 万历年间 | 复筑门楼 |

### 军事功能

- 控制山西北部通往中原的关键通道
- 防御蒙古从大同方向南下的重要关口
- 隶属太原镇（山西镇）防区
- 明代九边重镇中大同镇与山西镇的分界线附近
- 历史上杨家将抗辽的传说即与此关有关

### 历史战役

- 正统至成化年间，蒙古多次从雁门关方向侵入山西
- 嘉靖年间，俺答汗多次威胁雁门关防线
- 隆庆和议后，雁门关军事压力减轻

> 来源：[雁门关 - 维基百科](https://zh.wikipedia.org/zh-hans/%E9%9B%81%E9%97%A8%E5%85%B3)；[长城十三关——关关有故事](http://www.silkroads.org.cn/portal.php?mod=view&aid=18289)

### 驻军

- 嘉峪关隶属甘肃镇防区
- 关城及附近堡寨常年驻军数千人
- 甘肃镇总兵力约 **数万人**

### 军事功能

- 控制河西走廊通往西域的唯一通道
- 明代对西域的控制以嘉峪关为界
- 关外即为蒙古和西域各部势力范围
- "出了嘉峪关，两眼泪不干"——关外为荒凉之地
- 嘉靖年间因吐鲁番威胁而大规模加固

### 历史战役

- 正德年间，吐鲁番曾一度攻破嘉峪关防线，占领哈密
- 嘉靖初年，吐鲁番再次威胁嘉峪关，明朝实行"闭关绝贡"政策
- 嘉靖年间多次与蒙古各部在关外交战

## 七、明代长城防御体系总览

### 九边军镇与关隘分布

```
【西端】嘉峪关 ←─── 甘肃镇 ←── 宁夏镇 ←── 延绥镇 ←── 太原镇（雁门关）
                                                              ↓
                                       大同镇 ←── 宣府镇 ←── 蓟镇（居庸关/紫荆关）
                                                              ↓
                                                     辽东镇 → 山海关 【东端】
```

### 长城总体数据

| 项目 | 数据 |
|------|------|
| 明长城总长 | 约 **8,851.8公里**（2012年国家文物局测量） |
| 九边军镇 | 9个 |
| 总兵力 | 约 **80万** |
| 主要关隘 | 数十座 |
| 敌台 | 数千座 |
| 烽火台 | 数千座 |
| 堡寨 | 数百座 |

### 内外长城

明代长城分为内外两道：
- **外长城**：从北京东部怀柔向北经宣府到张家口，再到大同，沿北部边境延伸
- **内长城**：从八达岭向西南至紫荆关、倒马关
- **内三关**：居庸关、紫荆关、倒马关
- **外三关**：雁门关、宁武关、偏头关

### 关宁锦防线（天启至崇祯）

明末最重要的防线，以山海关为后方基地：

```
山海关 → 宁远（今兴城）→ 锦州
```

- 孙承宗、袁崇焕先后构筑
- 配备红夷大炮等火器
- 是明朝在辽东对抗后金/清的最后防线
- 清军26年间无法正面突破山海关
