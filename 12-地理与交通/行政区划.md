# 明代行政区划（嘉靖至崇祯，1521-1644）

## 概述

明代实行"两京十三省"的行政区划体系。全国分为北直隶、南直隶和十三布政使司（省），下辖约 **162府**（含军民府）、**255州**（含直隶州和属州）、**1173县**。在省级管理上实行"三司并立"体制，布政使司（民政）、按察使司（司法监察）、都指挥使司（军事）三权分立、互不统属，各对中央负责。

## 二、三司并立管理体制

### 三司的设置

明初朱元璋废除行省制度（因行省权力过大），改设三司分治：

#### 1. 承宣布政使司（简称"布政司"，俗称"省"）

| 项目 | 内容 |
|------|------|
| 主官 | 左、右布政使（从二品） |
| 职权 | 一省民政、财政、户籍 |
| 属官 | 左右参政（从三品）、左右参议（从四品） |
| 分司 | 按地域分若干"道"，由参政/参议分管 |

#### 2. 提刑按察使司（简称"按察司"）

| 项目 | 内容 |
|------|------|
| 主官 | 按察使（正三品） |
| 职权 | 一省司法、监察、驿传 |
| 属官 | 副使（正四品）、佥事（正五品） |
| 分司 | 设若干分巡道、兵备道 |

#### 3. 都指挥使司（简称"都司"）

| 项目 | 内容 |
|------|------|
| 主官 | 都指挥使（正二品） |
| 职权 | 一省军事（卫所管理、操练、屯田） |
| 属官 | 都指挥同知、都指挥佥事 |
| 下辖 | 各卫指挥使司、各所千户所 |

### 三司的运作原则

- **互不统属**：三司平级，分掌军、政、法，相互制衡
- **各对中央**：布政司对户部和朝廷负责，按察司对都察院负责，都司对五军都督府和兵部负责
- **遇大事会商**：重要事务由三司会同商议处理
- **巡抚凌驾三司**：明中后期（弘治以后），巡抚逐渐成为实际上的一省最高长官，凌驾三司之上

### 巡抚与总督

三司并立的弊端在于事权分散、协调困难，因此明中后期逐渐增设：

- **巡抚**：正德以后逐渐常设化，统管一省军政，成为实际省级长官
- **总督**：管辖两省以上的军政大员（如两广总督、蓟辽总督等）
- 另有漕运总督、河道总督等专职总督

## 三、行政层级

### 四级行政体系

明代行政层级为省（布政使司）—府—州/县，但实际运作中存在四级：

```
第一级：布政使司（省）/ 两京直隶
  ↓
第二级：府 / 直隶州
  ↓
第三级：属州 / 县
  ↓
第四级：（属州下辖的县）
```

### 府

- 府为省以下最重要的行政单位
- 长官为知府（正四品）
- 辖若干州、县
- 重要府（如应天府、顺天府）知府品级更高

### 州

分两类：
- **直隶州**：直属于布政使司，地位与府平级，可辖县
- **属州**：隶属于府，介于府与县之间

### 县

- 最基层行政单位
- 长官为知县（正七品）
- 全国约1173个县

## 四、特殊行政区域

### 1. 两京直隶

北直隶和南直隶不设布政使司，其府州直接隶属于中央：
- **北直隶**：顺天府（北京）、保定府、河间府、真定府、顺德府、广平府、大名府、永平府，及保安州、延庆州等
- **南直隶**：应天府（南京）、苏州府、松江府、常州府、镇江府、凤阳府、淮安府、扬州府、徽州府、宁国府、太平府、池州府、安庆府、庐州府，及徐州、滁州、和州、广德州等

### 2. 军事卫所

明代在全国设立卫所系统，与行政区划并行：
- **卫**：约 **493卫**，每卫5600人（额定）
- **所**：约 **359所**（千户所），每所1120人（额定）
- 卫所遍布全国，边境地区尤为密集
- 部分卫所（如天津卫、威海卫）后来演变为城市

### 3. 土司地区

西南地区（云南、贵州、广西、四川、湖广西部）设有大量土司：
- **宣慰司**、**宣抚司**、**安抚司**、**长官司**等
- 土司世袭，实行间接统治
- 明代推行"改土归流"政策，逐步以流官取代土司

### 4. 九边军镇

明代在北部边境设九个军事重镇（"九边"），与行政区划并行：

| 军镇 | 驻地 | 管辖范围 |
|------|------|---------|
| 辽东镇 | 辽阳 | 辽东地区 |
| 蓟镇 | 三屯营 | 北京东北 |
| 宣府镇 | 宣化 | 北京西北 |
| 大同镇 | 大同 | 山西北部 |
| 太原镇（山西镇） | 偏关 | 山西西北 |
| 延绥镇（榆林镇） | 榆林 | 陕北 |
| 宁夏镇 | 银川 | 宁夏 |
| 固原镇 | 固原 | 甘肃东部 |
| 甘肃镇 | 张掖 | 河西走廊 |

- 九边总兵力常年维持约 **80万** 官军
- 每镇约 **10万** 士兵（额定）

## 五、全国人口数据

### 各时期人口估算

| 时期 | 人口估算 | 数据来源/依据 |
|------|---------|-------------|
| 洪武十四年（1381） | 约 **5,987万** | 户部统计（户籍人口） |
| 十四世纪后期 | 约 **6,500万** | 何炳棣研究 |
| 永乐年间 | 约 **6,600万** | 户部统计 |
| 弘治年间（1500前后） | 约 **5,300万**（官方） | 户口流失严重，实际远高于此 |
| 嘉靖年间 | 约 **6,000万**（官方） | 实际约 **1-1.2亿** |
| 万历六年（1578） | 约 **6,069万**（官方） | 户部统计数据 |
| 万历二十八年（1600） | 约 **1.5亿** | 学者估算（两京十三省） |
| 崇祯末年（1644） | 约 **1.5-2亿** | 学者估算 |

**说明**：
- 明代官方户口统计因赋役关系，百姓有隐匿户口的动机，实际人口远高于官方数字
- 学者估算的1.5-2亿是较为广泛接受的晚明人口数字
- 何炳棣等学者的研究对修正官方数据贡献很大

### 各省人口分布（嘉靖至万历间估算）

人口密度最高的地区：
1. **南直隶**（江苏/安徽）：最大人口密集区
2. **浙江**：紧随其后
3. **江西**：人口稠密
4. **湖广**：人口较多
5. **山东/河南**：北方人口密集区

人口密度较低的地区：
- 云南、贵州、广西（山地多、少数民族地区）
- 陕西西部（甘肃、宁夏部分）

## 六、财政体制

### 税赋区域差异

明代各省税赋负担差异极大：
- **南直隶**（尤其苏州、松江等府）：税赋最重，"天下赋税出于江南，而江南尤重于苏松"
- **苏州府**：洪武年间田赋约占全国总额的 **1/10**
- **西南诸省**：税赋相对较轻

### 一条鞭法

万历九年（1581），张居正在全国推行一条鞭法：
- 将各种赋税和徭役合并为一条
- 统一折算为白银征收
- 简化税制，减少中间盘剥
- 这是明代最重要的财税改革
