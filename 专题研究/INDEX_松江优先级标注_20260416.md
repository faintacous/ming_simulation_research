# 人物名录 INDEX 松江优先级标注

生成时间：2026-04-16
标注范围：`人物名录/INDEX.md` 全部人物（约 430 人）
时间窗口：嘉靖元年（1521）— 崇祯末年（1644）
优先级定义：
- **P1** 和松江府有关系的重要官员（松江籍二品以上/入阁大臣；在松江任过职且有显著政绩的高级官员；影响松江重大事件的外地高官）
- **P2** 和松江府有关系的商人（棉布巨商、漕粮包揽商、在松江设庄的外埠商人、海商）
- **P3** 松江府历任官员（嘉靖-崇祯期间担任过松江府/华亭/上海/青浦知府、县令、同知、通判、推官、儒学教授等）
- **P4** 松江府的读书人（松江籍进士/举人有著作者、文社成员、画派宗师、学派传人）
- **P5** 其他在松江府生活过的名人（流寓、闲居、著述、行医、修道等任一类）
- **无关** 与松江府无任何可查关系

行政区划注意：明代松江府辖华亭县、上海县、青浦县（万历元年增设）。**嘉定县、太仓州、崇明县属苏州府**，不在本标注的"松江府"范围内。归有光在嘉定讲学、孙元化（嘉定人）、王忬/王世贞（太仓人）因此均标"无关"（近邻但非本府）。

## 工作总结（<200 字）

INDEX 内 430 人中仅 9 人（2.1%）与松江府有可查关系，其中 7 人属 P1（4 人松江籍大臣 + 3 人外籍高官任职松江辖区）、2 人属 P3。P2、P4、P5 在 INDEX 范围内基本为空。

最大缺口：(1) **P3 松江历任官员**——明代嘉靖-崇祯 123 年间松江府知府约 40-50 人、三县县令合计约 150 人，INDEX 仅命中 2 人，建档率约 1%；(2) **P2 松江棉布/漕粮商人**全部空白；(3) **P4 松江读书人**层面，莫是龙、陈继儒、陈子龙、夏允彝四大核心人物未被 INDEX 收录（筛选年代窗漏收或超出）。

特别发现：(a) 董其昌、徐光启虽归入"文人学者"类，但按明代官秩他们是 P1 重要官员级别（南京礼部尚书、礼部尚书兼大学士），应在人物优先级上获得 P1 权重而非纯文化人物权重；(b) INDEX 现有 3 篇松江相关传记（徐阶、董其昌、海瑞），其余 P1（陆树声、徐光启、董传策、林润）亟需建档。
