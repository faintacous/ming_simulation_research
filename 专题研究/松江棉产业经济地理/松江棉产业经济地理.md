# 松江棉产业经济地理

## 概述

本文档聚焦松江府棉纺织业的**空间经济结构**：产业链各环节的地理分布、城乡分工、市镇集散体系、贸易路线网络、以及赋税征解的空间逻辑。研究起点为黄敬斌《郡邑之盛：明清松江城的空间形态与经济职能》（《史林》2016年第6期），并延伸至相关文献。

---

## 一、产业链空间分工

### 1.1 棉花种植的区域分布

| 地区 | 种植比例 | 来源 |
|------|---------|------|
| 松江府东乡 | "大半植棉"，**棉七稻三** | 多种方志 |
| 嘉定、太仓 | **三分宜稻，七分宜木棉** | 吴伟业《木棉吟序》 |
| 嘉定极端情况 | **棉九稻一**，乃至全种棉花 | 康熙《嘉定县志》卷一《风俗》 |
| 松江府西部 | 稻作比重较高 | — |

**关键特征**：松江府内部存在东西分异——东部（靠近海岸的沙质土壤区）以棉为主，西部（低洼水田区）以稻为主。棉作区农户需向外购粮，形成对湖广米粮的结构性依赖。

### 1.2 纺纱与织布的城乡分工

黄敬斌论文引用的关键史料揭示了城乡之间的纺织分工：

| 生产环节 | 空间分布 | 来源 |
|---------|---------|------|
| **纺纱** | 城市居民（"红女"）以纺纱为主 | 康熙十一年碑刻："城市红女，悉力纺纱，售之乡民" |
| **织布** | 乡村农户以织布为主 | 同上："村野当农隙之候，妇女弄梭" |
| **在城机户** | 府城内存在专业织户，织造高端棉布 | 崇祯《松江府志》郑友玄《布解议》 |

**城市机户的关键数据**（《布解议》）：
- 华亭县年解额：梭布 **16,185匹** + 棉布 **48,935匹** = 合计 **65,120匹**
- "在城机户，惯织官布者，原自有人"——城内存在专业织造群体
- 官布分二线、三线，定价每匹 **三钱八分** 或 **四钱**
- 按每机户两天成布一匹估算，65,120匹仅需 **约360户** 一年产量

**西嶋定生的观点**（《中国经济史研究》）：
- 松江府城织布业的特征是**织造高级棉布**
- 城市机户"本质上是以政府收买为前提的定货生产者"
- 黄敬斌反驳：城市机户并非全部为官府生产，相当部分产品进入市场

### 1.3 棉布品种的产地地理

| 品种 | 产地 | 特征 | 等级 |
|------|------|------|------|
| **三梭布** | 车墩一带 | 幅阔三尺余，光洁细密 | 高端 |
| **飞花布**（丁娘子布） | 双庙桥（府城东门外） | "花皆飞起，尤为精软" | 高端 |
| **尤墩布** | 松江府城 | 厚且重，用于制暑袜 | 高端 |
| **眉织** | 松江府城 | 织花绒布 | 高端 |
| **番布** | 乌泥泾 | 黄道婆传授技术之地 | 高端 |
| **标布**（大布） | 浦东三林塘（最精） | 标准规格商品布 | 中等 |
| **扣布**（小布/短头布） | 叶榭 | 幅窄 | 中等 |
| **稀布** | 龙华、七宝 | — | 中等 |
| **高丽布**（洋袍） | 高桥、洋泾 | — | 中等 |
| **紫花布** | 松江广域 | 蓝白相间，即Nankeen | 出口品种 |

**空间特征**：高端布种集中于**府城及近郊**（飞花、尤墩、眉织），中等布种分布于**周边市镇**（标布在三林塘，扣布在叶榭），形成以府城为中心的品质梯度。

### 1.4 染踹加工业的空间转移

| 时期 | 加工中心 | 规模 | 来源 |
|------|---------|------|------|
| 明代 | 松江枫泾、朱泾 | "数百家布号，皆在松江枫泾、洙泾乐业，而染坊、踹坊、商贾悉从之" | 顾公燮 |
| 清康熙初年 | 苏州开始集中 | 字号**四五十至六七十家** | 邱澎生研究 |
| 康熙五十九年 | 苏州 | 染坊 **64家** | 官方记录 |
| 雍正年间 | 苏州 | 踹坊 **450余家**，踹匠 **万余人**，日加工 **约12万匹** | 官方调查 |
| 康熙至雍正 | 苏州 | 踹匠、染匠合计 **至少2万人以上** | 雍正年间调查 |

**空间转移逻辑**：
- 明代布号集中于松江枫泾、朱泾等产地市镇
- 清初开始向苏州集中——苏州阊门一带成为全国棉布加工贸易中心
- 原因：苏州的市场辐射力、水路交通优势、商业基础设施
- 松江本地仅保留牙行（代苏州布商收货），布号字号多迁苏州

## 二、市镇集散体系

### 2.1 松江府城的商业地位

黄敬斌论文的核心发现：松江府城在棉布贸易中**并不具有突出的中心地位**，但仍是分散集散地之一。

**府城布业数据**（康熙十一年碑刻）：
| 类别 | 数量 |
|------|------|
| 华亭县布牙 | **25家** |
| 华亭县布铺 | **57家** |
| 在城布铺 | **8家** |
| 在城染坊 | **6家** |

乾隆元年碑刻提到徽商吴舆璠"投治西外开张富有字号"——府城西门外有较大布商字号。

### 2.2 核心集散市镇

| 市镇 | 功能 | 来源 |
|------|------|------|
| **枫泾** | 明代主要布号集中地 | 顾公燮；黄敬斌论文 |
| **朱泾（洙泾）** | 明代主要布号集中地，"万家灯火似都城" | 顾公燮 |
| **车墩** | 三梭布产地，"细布出东门，盖车墩去东门不远" | 正德《松江府志》 |
| **泗泾** | 米市兼有布市 | 近代调查 |
| **朱家角** | "商贾凑集，贸易花布" | — |
| **华阳桥** | 府城东门外，东西长街终点 | 黄敬斌论文 |
| **三林塘** | 标布最精产地 | — |

### 2.3 商路网络

黄汴《一统路程图记》记载苏松间水路商路之多，"路须多迂，布商不可少也"：

| 路线 | 特点 |
|------|------|
| 苏州→嘉兴→松江→上海 | 主干线 |
| 松江→官塘→苏州 | 直达线 |
| 松江→太仓→苏州 | 北线 |
| 松江→南翔→上海 | 东线 |
| 松江→金山卫 | 南线 |
| 松江→吴淞所 | 东北线 |
| 松江→乌泥泾 | 短途 |

**特征**：布商收购棉布必须深入各地市镇乡村，不能仅依赖一两个中心集散地。至少明代，江南棉布的贸易集散地相当分散。

### 3.2 折征比价

| 布种 | 折米比价 | 时期 | 来源 |
|------|---------|------|------|
| 阔白三梭布 | 1匹 = **米2石5斗** | 宣德八年（1433） | 周忱奏定 |
| 阔白棉布 | 1匹 = **米1石** 或 **9斗8升** | 宣德八年 | 同上 |

### 3.3 江南官布在全国的占比

苏、松、常三府所贡各色棉布占全国征解量的近 **80%**。松江府赋税额居全国最高水平，"苏松赋税半天下"。

## 四、棉布价格的空间与品级差异

| 品种/品级 | 价格 | 时期 | 来源 |
|----------|------|------|------|
| 普通白棉布 | **一钱五分至两钱/匹** | 明代 | 陈江讲座 |
| 松江云布精品 | **三两银子/匹** | 明代 | 李绍文《云间杂识》 |
| 贡品三绫布 | 可达**一百两银子/匹** | 明代 | 同上 |
| 三线细布（官布） | **七钱/匹** | 明代 | 聂绍昌《布解议》 |
| 二线细布（官布） | **六钱/匹** | 明代 | 同上 |
| 阔白线布（官布） | **三钱四分/匹** | 明代 | 同上 |
