# 南京布（Nankeen）全球贸易史

## 概述

南京布（Nankeen/Nanking cloth）是明清时期中国最重要的棉布出口商品，主要产自松江府及江南地区，因经南京集散而得名。16世纪起通过月港、广州出口至东南亚、欧洲、美洲，18世纪末至19世纪初达到贸易高峰——1819年单年出口超过335.9万匹。1786-1833年的48年间，累计出口超过4400万匹。南京布的外销被郭卫东称为"丝绸之路的续篇"，标志着中国对世界贸易的贡献从高端丝绸转向大众棉布。19世纪30年代后，随英国工业革命机织棉布的冲击，南京布贸易急剧衰落。

## 一、名称由来——为什么叫"Nankeen"而非"Songjiang cloth"

### 1.1 名称的形成过程

| 时间 | 称呼 | 使用者 | 来源 |
|------|------|--------|------|
| 1580年代 | "Songjiang cloth" | 葡萄牙、西班牙商人 | 郭卫东论文 |
| 16-17世纪 | "mantas de Lanquin" | 西班牙人（经马尼拉贸易） | 张礼骏研究 |
| 18世纪初 | "Nanking cloth" / "Nankeen" | 英国东印度公司 | 英国东印度公司贸易清单 |
| 1755年 | 最早英文文献记录 | 英语世界 | OED词典 |

### 1.2 命名逻辑——三个中心

南京布实际并非产于南京，而是产于以松江为中心的江南地区。名称由来与明清江南棉纺织的产业分工有关：

| 功能中心 | 城市 | 角色 |
|---------|------|------|
| **织造中心** | 松江府 | 棉布生产的核心产区 |
| **浆染中心** | 芜湖 | 棉布染色加工 |
| **贸易中心** | 南京 | 棉布集散与批发贸易 |

[来源：《天工开物》："凡棉布寸土皆有，而织造尚松江，浆染尚芜湖"](https://baike.baidu.com/item/松江棉布/1105408)

### 1.3 "南京"取代"松江"的原因

1. **行政地理因素**：南京（南直隶首府）是江南地区最大、最知名的城市，外国商人以地区中心城市指代该地区产品
2. **贸易中转角色**：南京位于长江要冲，是棉布从产区运往广州的重要中转站
3. **福建商人的转述**：16-17世纪西班牙人经马尼拉与福建商人贸易，福建商人告知此布来自"南京"（南直隶辖区），西班牙人遂以此命名
4. **品牌效应**：南京在欧洲的知名度远高于松江，"Nanking"成为一个便于传播的商业标签

[来源：Francis Academic Press论文 "Study on the Name of 'Nanking Cloth'"](https://francis-press.com/papers/12592)

## 二、紫花布的制作工艺

### 2.1 天然棕/黄色南京布（Yellow Nankeen）

这是出口欧洲的主要品种，也是"Nankeen"最经典的含义。

| 特征 | 说明 |
|------|------|
| **原料** | 天然黄色（棕色）棉花品种，属 Gossypium arboreum 或 Gossypium herbaceum |
| **颜色** | 天然浅黄/淡棕色（pale yellow / buff），无需染色 |
| **紫花棉特征** | 花开为紫色，成熟后棉果为黄色，纤维细长柔软 |
| **织造** | 平纹或斜纹组织（plain weave / twill），手工木织机 |
| **后整理** | "finished without size or bleach"——不上浆、不漂白 |
| **耐久性** | 抗褪色，随时间形成柔和光泽（patina） |
| **质地** | 紧密、坚固（firm-textured, durable） |

[来源：Grokipedia "Nankeen"](https://grokipedia.com/page/Nankeen)；[Wikipedia "Nankeen"](https://en.wikipedia.org/wiki/Nankeen)

### 2.2 蓝印花南京布（Blue Nankeen）

另一种重要品种，是蓝白相间的印花棉布。

制作工序：
1. 手工雕刻镂空花版（木版或厚纸版）
2. 用黄豆粉和石灰调制防染浆
3. 将花版铺在白色棉布上，刮浆刀将防染浆刮入花纹空隙
4. 待干后放入靛蓝染缸，每次浸染20分钟后取出氧化透风30分钟
5. 反复浸染6-8次达到所需蓝色深度
6. 刮去防染浆，露出白色花纹
7. 水洗、晾干、熨烫

[来源：Smithsonian Folklife Magazine "蓝印花布"](https://folklife.si.edu/magazine/blue-calico-cloth-nantong-chinese)

## 三、贸易路线

### 3.1 国内流通路线

```
松江府（织造）→ 芜湖（浆染）→ 南京（集散）
                                    ↓
                              经长江水路南下
                                    ↓
                              广州（对外出口港）
```

中间环节：松江布商 → 牙行经纪 → 长途贩运商 → 广州十三行（行商）→ 外国商人

### 3.2 出口路线

**路线一：月港-马尼拉-美洲（16-17世纪为主）**
```
松江 → 福建月港 → 马尼拉（吕宋）→ 阿卡普尔科（墨西哥）
                                       ↓
                                    西班牙本土/拉美各地
```
- 1567年隆庆开海后，月港成为合法贸易港
- 福建商人从月港出发，将棉布、丝绸、瓷器运至马尼拉
- 再由马尼拉大帆船运至墨西哥

**路线二：广州-欧洲（18-19世纪为主）**
```
松江/江南 → 广州（Canton）→ 好望角 → 英国/法国/荷兰/北欧
                            → 经大西洋 → 美国东海岸
```
- 1715年英国东印度公司在广州设立永久"商馆"（factory）
- 广州十三行垄断对外贸易
- 各国东印度公司（英、荷、瑞典、丹麦、法国）从广州采购

**路线三：广州-美国（1784年后）**
```
广州 → 好望角/合恩角 → 纽约/费城/波士顿/塞勒姆
```
- 1784年"中国皇后号"（Empress of China）开启中美直接贸易

## 五、欧洲各国的市场接受

### 5.1 英国

- **东印度公司角色**：1715年在广州设永久商馆，系统采购南京布
- **早期需求**：1734年首次尝试订购100匹，此后逐年增加
- **用途**：男性夏季裤装（trousers）、马裤（breeches）、马甲（waistcoats）、儿童服装
- **摄政时期（Regency era, 1811-1820）**：南京布达到时尚巅峰
- **军事用途**：热带殖民地军队制服
- 1807年购买288,000匹，1817年购买643,000匹

### 5.2 法国

- 1786年购买72,000匹
- 用于时尚市场——福楼拜《包法利夫人》中提及"nankeen dress"
- 拿破仑战争期间贸易中断

### 5.3 荷兰

- 1786年购买98,200匹——当年各国中最多
- 很可能充当转口贸易角色，将南京布分销至欧洲其他地区
- 荷兰东印度公司（VOC）参与棉布贸易

### 5.4 北欧国家

- **丹麦**：1786年购买78,000匹，数量令人瞩目
- **瑞典**：瑞典东印度公司（1731-1813年运营）将南京布列为重要进口商品，称其为"东印度贸易中最精良的织品"

### 5.5 美国

- **1784年**："中国皇后号"从纽约首航广州，返程载回**20,000双南京布裤子**及大量棉布
- **贸易规模**：1801年美国从广州输出140万匹；1809-1810年达376.9万匹
- **1819年美国分销**：不仅供应国内，还转口至欧洲、南美洲和太平洋岛屿
- 美国成为南京布最大的买家和分销商
- 波士顿、纽约、费城、塞勒姆（Salem）均为重要贸易港

### 7.3 语义衍生

"Nankeen"一词从面料名称衍生出多重含义：
- 指代一种浅黄/暗黄色（颜色名称）
- 泛指用这种面料制作的裤子
- 用于命名澳大利亚的鸟类（Nankeen Kestrel、Nankeen Night Heron）和植物
- 指代中国风格的蓝白瓷器

## 八、与英国工业革命棉纺织品的竞争与衰落

### 8.1 竞争时间线

| 年份 | 事件 |
|------|------|
| 1733 | 约翰·凯伊发明飞梭，引发"纱荒" |
| 1764 | 哈格里夫斯创制珍妮纺纱机 |
| 1785 | 卡特莱特发明水力织布机 |
| 1819 | 南京布出口达到最高峰335.9万匹 |
| 1819-1820 | 出口量"断崖式下跌"——从335.9万匹骤降至91万匹 |
| 1825 | 英国已有蒸汽机1.5万台、马力37.5万匹 |
| 1830年代 | 曼彻斯特地区85%的全球棉布在兰开夏加工 |
| 1832 | 南京布出口降至17.05万匹 |
| 1833 | 英国东印度公司对华贸易垄断权终结 |
| 1840年代 | 英国机织棉布反向出口中国 |
| 甲午战前(1894) | 洋布仅占中国棉布消费额的13.5%——土布仍顽强生存 |

### 8.2 衰落原因分析

1. **价格优势逆转**：英国机织布以蒸汽动力大规模生产，成本远低于中国手工布
2. **质量竞争**：机织布更均匀、更洁白、品种更丰富
3. **中国技术停滞**：郭卫东指出"墨守成法，不知改良"——中国纺织技术长期停滞
4. **市场替代**：英国棉布先占领欧美本土市场，再向中国和亚洲倾销
5. **原料竞争**：美国南方种植园生产的白色棉花取代了天然彩色棉花的地位

### 8.3 南京布在美国的特殊命运

美国人曾尝试在南方种植园种植天然黄色的"Nankeen cotton"，试图本土化生产。但19世纪中期以后：
- 白色棉花主导国际市场
- 黄色南京棉沦为"有色人种棉花"（colored people's cotton）
- 被奴隶穿着使用，市场价格大幅下跌
- 从原来的"异国高级面料"变为低等级织物

## 九、明代通过月港出口东南亚的早期贸易

### 9.1 月港贸易背景

| 方面 | 说明 |
|------|------|
| 位置 | 福建漳州龙海九龙江入海口 |
| 开放时间 | 1567年隆庆开海 |
| 性质 | 明代唯一合法的私人海外贸易港口 |
| 顾炎武记载 | "闽人通番皆自漳州月港出洋" |

### 9.2 出口商品构成

月港出口商品以丝绸、棉布、瓷器为大宗。漳州本地手工业发达，绒、纱、缎、毡均在国内外市场畅销。但最大宗的棉布来自松江，经内贸网络运至福建再出口。

### 9.3 对菲律宾棉布贸易的影响

- 中国棉布质优价廉，迅速占领菲律宾市场
- 1591年：菲律宾总督发现当地土著已完全依赖中国衣料，不再种棉织布，下令禁止
- 1592年：中国商人甚至收购菲律宾原棉，运回中国加工后再以成品棉布返销
- 棉布成为中国在菲律宾**销路最大**的商品

### 9.4 白银回流数据

| 时期 | 年均白银流入量 |
|------|-------------|
| 1570年代 | 约28.5万两 |
| 1580年代 | 约88.9万两 |
| 1600年后 | 约100万两 |

## 存疑 & 待查

1. **1786年之前的系统出口数据缺失**：郭卫东论文以1786年为起点，此前的葡萄牙、西班牙时期出口数据需要进一步搜集
2. **月港棉布出口的具体品种和数量**：明代通过月港出口的棉布中，松江布占多大比例？是否有区分紫花布和其他品种？
3. **"mantas de Lanquin"的具体含义**：西班牙文献中这个词是否特指松江紫花布，还是泛指中国棉布？
4. **1809-1810年美国单年进口376.9万匹与1819年峰值335.9万匹的矛盾**：郭卫东论文中这两个数据是否口径不同？需核实
5. **芜湖浆染环节的具体操作**：从松江到广州的完整物流链中，棉布是否全部经过芜湖？
6. **法国市场的具体数据**：1786年之后法国的逐年采购量数据缺失
7. **紫花棉的品种鉴定**：Gossypium arboreum 还是 Gossypium herbaceum？与现代天然彩棉的关系？
8. **福楼拜和狄更斯之外的文学引用**：是否有简·奥斯汀、萨克雷、梅尔维尔等人的作品直接提及Nankeen？
9. **南京布在日本市场的贸易数据**：日本是否也进口南京布？规模如何？
