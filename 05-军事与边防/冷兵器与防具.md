# 冷兵器与防具

## 概述

明代中后期（嘉靖至崇祯，1521-1644）虽然火器迅速发展，冷兵器仍是明军的基本装备。明军制式冷兵器种类丰富，包括刀、枪、弓弩、盾牌等。防具方面，随着火器普及，传统铁札甲逐渐被布面甲和棉甲取代，以兼顾防冷兵器与早期火器。本篇聚焦嘉靖至崇祯年间各类冷兵器和防具的规格、使用场景和制度规定。

## 一、刀

### 1. 腰刀

| 项目 | 数据 |
|------|------|
| 地位 | 明军最基本的制式近战兵器，几乎人手一把 |
| 刀身长度 | 约二尺至二尺五寸（64-80厘米） |
| 全长（含柄） | 约三尺至三尺二寸（96-102厘米） |
| 重量 | 约一斤十两至二斤（约1.1-1.2公斤） |
| 刃型 | 单刃微弯，前宽后窄 |
| 材质 | 铁制或钢制 |
| 使用方式 | 单手持握，配刀鞘悬于腰间 |
| 装备范围 | 步兵、骑兵、水兵均配备 |

戚继光《纪效新书》记载戚家军腰刀："长三尺二寸，重一斤十两"（约102厘米，1.92市斤）。
[来源：《纪效新书》]

### 2. 长刀（戚氏长刀/倭刀）

| 项目 | 数据 |
|------|------|
| 背景 | 嘉靖抗倭战争中，明军发现日本刀法厉害，戚继光遂引入并改良 |
| 刀刃长度 | 五尺（约160厘米） |
| 铜护刃 | 一尺 |
| 刀柄长度 | 一尺五寸（约48厘米） |
| 全长 | 六尺五寸（约208厘米） |
| 重量 | 二斤八两（约1.475公斤） |
| 使用方式 | 双手持握 |
| 用途 | 戚继光称"此自倭犯中国始有之"，专门对抗倭寇 |

[来源：戚继光《纪效新书·长刀解》]

### 3. 程氏单刀（苗刀前身）

| 项目 | 数据 |
|------|------|
| 设计者 | 程宗猷（晚明武术家），著《单刀法选》 |
| 刀身长度 | 三尺八寸（约122厘米） |
| 刀柄长度 | 一尺二寸（约38厘米） |
| 全长 | 约五尺（约160厘米） |
| 特点 | 参考日本大太刀改良，双手持握 |
| 后续影响 | 民国时期武术家刘玉春统称此类刀为"苗刀" |

[来源：程宗猷《单刀法选》；苗刀 - 百度百科](https://baike.baidu.com/item/%E8%8B%97%E5%88%80/3479104)

### 4. 倭腰刀

| 项目 | 数据 |
|------|------|
| 特点 | 采用中式刀柄和装具，搭配日式刀条 |
| 全长 | 85-95厘米 |
| 刀刃长度 | 65-80厘米 |
| 使用 | 明军仿制或进口日本刀条，装配中式刀装 |

[来源：[明代倭刀及其使用与传布研究](https://m.fx361.com/news/2020/0921/11125196.html)]

### 5. 其他制式刀

| 刀名 | 特点 | 用途 |
|------|------|------|

## 二、枪

### 1. 长枪

| 项目 | 数据 |
|------|------|
| 地位 | "百兵之王"，明军步兵最重要的长兵器 |
| 枪杆材质 | 竹制或木制 |
| 全长 | 一般九尺至一丈二尺（约288-384厘米） |
| 枪头 | 铁制，长约六寸至一尺（19-32厘米） |
| 枪缨 | 红缨，防止血液流到枪杆导致手滑 |
| 用途 | 步兵列阵突刺，鸳鸯阵的核心杀伤兵器 |

《武备志》记明代常用枪有六种：长枪、枪式一至三、铁钩枪、龙刀枪。
[来源：《武备志》]

**鸳鸯阵中长枪兵的配置**：

每队12人中配长枪兵4人，居阵型第四排，是主要杀伤力量。长枪兵在藤牌手和狼筅手掩护下，以长枪突刺敌人。
[来源：戚继光《纪效新书》]

### 2. 标枪（飞枪/投枪）

| 项目 | 数据 |
|------|------|
| 全长 | 约四尺至五尺（128-160厘米） |
| 重量 | 轻于长枪 |
| 射程 | 投掷距离约20-30步（30-50米） |
| 使用方式 | 盾牌手通常携带两支标枪，左手持盾和一支标枪，右手持另一支 |
| 战术 | 先投掷标枪扰乱敌阵，再拔腰刀近战 |

[来源：新华网《中国古代标枪》](http://www.news.cn/milpro/20241128/d23b4e79818c4f78a8c73ff42c5730c5/c.html)

### 3. 狼筅

| 项目 | 数据 |
|------|------|
| 材质 | 大毛竹，保留枝桠 |
| 全长 | 约一丈至一丈二尺（320-384厘米） |
| 设计者 | 戚继光推广使用 |
| 用途 | 以密集枝桠挡住敌人视线和行动，掩护后排长枪手突刺 |
| 配置 | 鸳鸯阵中每队2人持狼筅 |
| 特点 | 造价极低，不需铁制枪头 |

[来源：《纪效新书》]

## 三、弓弩

### 1. 弓

#### 弓力标准

《天工开物》记载明代弓力分四等：

| 等级 | 弓力（斤） | 说明 |
|------|-----------|------|
| 上力 | 120斤 | 军中精锐标准 |
| 中力 | 80-100斤 | 普通军士标准 |
| 下力 | 60斤 | 最低军用标准 |
| 虎力 | 超过120斤 | 罕见的极强弓力 |

弓力测量方法：以足踏弦就地，秤钩搭挂弓腰，弦满之时推移秤锤所压，则知弓力多少。
[来源：宋应星《天工开物》]

#### 弓的特点

| 项目 | 数据 |
|------|------|
| 类型 | 复合弓（角、木、筋三材制作） |
| 弓身长度 | 约三尺至四尺（96-128厘米） |
| 特点 | 明代弓被称为"软弓"（相对蒙古硬弓），但最软的战弓拉力也有约60磅（29公斤） |
| 军中口号 | "软弓、长箭、快马、轻刀"——明军骑兵四利器 |

[来源：[揭秘明朝骑兵四大利器 - 凤凰网](https://finance.ifeng.com/a/20170822/15593429_0.shtml)]

#### 箭的规格

| 项目 | 数据 |
|------|------|
| 箭杆长度 | 约二尺（约64厘米） |
| 簇长 | 约一寸（约3.2厘米） |
| 箭杆材质 | 竹制（空心竹箭杆最常见） |
| 全重 | 约五六钱（约18-22克，竹箭较轻） |
| 空心竹箭杆 | 长二尺六寸，连簇全重五六钱 |

[来源：《兵录》]

#### 射程

| 条件 | 距离 |
|------|------|
| 弓力60斤（下力弓） | 有效射程约70-80步（约105-120米） |
| 弓力100斤（中上力弓） | 有效射程约100-120步（约150-180米） |
| 弓力120斤（上力弓） | 有效射程约130-150步（约195-225米） |

### 2. 弩

| 项目 | 数据 |
|------|------|
| 明代地位 | 明中叶后弩在陆军中的地位下降，逐渐被火器取代 |
| 水军使用 | 水军战船上仍配备药弩（可发射带毒的弩箭） |
| 苍山船配置 | 药弩4张，弩箭100支 |
| 海沧船配置 | 药弩6张，弩箭100支 |
| 射程 | 约100-200步（150-300米） |
| 特点 | 精度高、不受风雨影响（相比火器），但射速慢 |

[来源：《筹海图编》；明朝水师 - 百度百科]

## 五、甲胄

### 1. 布面甲（暗甲）

| 项目 | 数据 |
|------|------|
| 别名 | 暗甲、青布铁甲、布面铁甲、钉甲 |
| 起源 | 元代开始使用，明代中后期成为主流 |
| 结构 | 外层为布料（棉布），内层夹缀铁甲片，甲片以铆钉固定 |
| 重量 | 约24-50斤（嘉靖时青布铁甲每副24-25斤；重型暗甲约50斤） |
| 甲片特点 | 甲片硕大，不需凿孔穿条，省工省料 |
| 防护能力 | 可抵挡冷兵器劈砍突刺，对早期火器霰弹有较强防护力 |
| 优势 | 造价低廉、易于大批量生产、可配发下层兵士 |
| 适用环境 | 北方冬季保暖，兼顾防御功能 |

[来源：布面甲 - 维基百科](https://zh.wikipedia.org/zh-hans/%E5%B8%83%E9%9D%A2%E7%94%B2)；[明朝盔甲 - 腾讯新闻](https://news.qq.com/rain/a/20200704A0ACSH00)

### 2. 棉甲

| 项目 | 数据 |
|------|------|
| 材质 | 棉花七斤，用布缝如夹袄 |
| 制作方法 | "棉花七斤，用布缝如夹袄，两臂过用脚踹实，以不胖胀为度，晒干收用"（朱国祯《涌幢小品》） |
| 重量 | 较轻（纯棉甲约10-15斤，内衬铁片者更重） |
| 明代改良 | 为对抗火器，开始内衬铁片，外用铜钉固定 |
| 防护原理 | 棉花纤维可吸收火器弹丸的冲击力，减弱穿透效果 |
| 优势 | 制作容易、价格低廉、重量轻、方便大批量生产 |
| 使用场景 | 普通步兵、辅兵 |

[来源：棉甲 - 百度百科](https://baike.baidu.com/item/%E6%A3%89%E7%94%B2/9619119)

### 3. 锁子甲

| 项目 | 数据 |
|------|------|
| 结构 | 以金属环相互扣连成网状 |
| 重量 | 约15-25斤 |
| 防护能力 | 防刺能力强，防砍劈能力中等 |
| 明代使用 | 御林军护身盔甲，高级军官和精锐部队使用 |
| 造价 | 高昂，制作耗时 |
| 缺点 | 制作复杂、造价高，难以大批量装备 |

[来源：[锁子甲 - 北京市公安局](https://gaj.beijing.gov.cn/jcbwg/zgdc/gjdjc/202007/t20200713_1947062.html)]

### 4. 甲胄演变趋势（嘉靖至崇祯）

| 时期 | 主流甲胄 | 原因 |
|------|----------|------|
| 嘉靖以前 | 铁札甲（明甲）为主 | 传统制式 |
| 嘉靖年间 | 布面甲逐步普及 | 火器增多，需兼顾防弹；北方作战需保暖 |
| 万历年间 | 布面甲成为主流 | 性价比高，适合大规模装备 |
| 天启-崇祯 | 布面甲/棉甲为主 | 财政紧张，廉价甲胄更受欢迎 |

### 5. 头盔

| 类型 | 特点 |
|------|------|
| 明制铁盔 | 铁制，带护颈帘和前额遮阳 |
| 笠形盔 | 形似斗笠，铁制或皮制 |
| 八瓣盔 | 八片铁片拼接而成 |
| 棉盔 | 棉制衬铁，与棉甲配套 |

## 七、武器质量与供给问题

### 嘉靖至崇祯时期的问题

| 问题 | 表现 |
|------|------|
| 偷工减料 | 军器局和地方作坊以次充好，刀枪铁质低劣 |
| 武器朽坏 | 长期不更新维护，甲胄残缺、刀枪锈蚀 |
| 供给不足 | 财政紧张导致武器更新周期延长 |
| 地域差异 | 边镇部队装备较好，内地卫所极差 |
