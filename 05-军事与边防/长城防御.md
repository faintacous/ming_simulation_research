# 明代长城防御体系

## 概述

明长城是中国历史上修筑规模最大、技术最先进、保存最完整的长城。明代自洪武年间开始修筑，至万历年间基本完工，前后持续约200年。明长城东起鸭绿江畔辽东虎山，西至甘肃嘉峪关，总长度约**8,851.8公里**（国家文物局2009年测量数据：人工墙体6,259.6公里、壕堑359.7公里、天然险阻2,232.5公里）。

明长城不是单一的线性城墙，而是由**城墙、关隘、敌台、烽燧、墩台、城堡**组成的立体纵深防御体系。在京畿地区还有内外两道长城。

> 敌台和烽燧数据来源：2006-2010年国家文物局会同国家测绘局组织的长城资源调查，涉及16省445县，由361个专业机构、1,295名专业技术人员历时4年完成。

### 内外长城

明代在京畿地区修有内外两道长城：

- **外长城**（边墙）：从山海关向西经蓟镇、宣府、大同一线
- **内长城**（次边）：从居庸关向西南经紫荆关、倒马关、雁门关、宁武关、偏头关一线，全长约1,600余里
- **内三关**：居庸关、紫荆关、倒马关
- **外三关**：雁门关、宁武关、偏头关

内外长城构成京师的两道防线，使得即便外长城被突破，内长城仍可阻止敌军深入。

## 二、主要关隘逐一详述

### 1. 山海关 ——"天下第一关"

| 项目 | 内容 |
|------|------|
| **位置** | 今河北秦皇岛市山海关区，辽东镇与蓟州镇交界处 |
| **地理特征** | 北依燕山，南临渤海，长城在此与海相接，扼守辽西走廊咽喉 |
| **建置沿革** | 洪武十四年（1381）大将军徐达筑关设卫，初名"山海卫"；成化以后称"山海关" |
| **关城规模** | 周长约4公里（约8里），城高约14米（约4丈2尺），厚约7米（约2丈1尺）；东、西、南、北四门，东门额题"天下第一关"（明代书法家萧显题） |
| **建制** | 设山海关卫、山海守御千户所；后期设山海关总兵（独立于蓟镇和辽东） |
| **驻军** | 嘉靖年间约5,000-8,000人；万历至天启年间增至数万人（因辽东战事） |
| **附属设施** | 老龙头（长城入海处，设有宁海城）、角山长城（山海关北翼）、南翼城、北翼城、东罗城、西罗城、威远城，共7座卫城、10大关隘 |

**战略地位**："两京锁钥无双地，万里长城第一关"。华北平原通往东北的唯一陆路通道。崇祯末年吴三桂引清兵入关即发生于此（1644年）。

### 2. 居庸关 ——"天下九塞之一"

| 项目 | 内容 |
|------|------|
| **位置** | 今北京市昌平区西北，太行山八达岭段 |
| **地理特征** | 位于太行山与燕山交汇处的峡谷中（关沟），两侧山势陡峭，谷道狭长约40里 |

| **关城规模** | 关城周长约4,000米（约13里），城墙高约12米，城门二（南门"居庸关"、北门"北门锁钥"）；关城内有云台（元代过街塔基座） |
| **建制** | 设居庸关千户所，隶属昌平镇（后归蓟镇管辖） |
| **驻军** | 嘉靖年间约3,000-5,000人；含八达岭守军 |
| **附属设施** | 八达岭关城（居庸关北口外关，"居庸之险不在关城而在八达岭"）、南口城、上关城、岔道城 |

## 三、空心敌台

### 起源与推广

空心敌台是明代长城防御技术的重大创新，由蓟镇总兵**戚继光**于隆庆三年至万历初年（1569-1573）在蓟镇率先大规模建造。此前长城上的敌台多为实心台（骑墙墩台），士兵只能站在台顶，无法避风雨、储存武器弹药。

### 空心敌台规格

| 项目 | 数据 |
|------|------|
| 台体高度 | 三四丈不等（约12-15米），高于城墙墙顶约3-5米 |
| 台基尺寸 | 周围阔十二丈至十七八丈不等（底部约12m x 10m，大台可达15m x 12m） |
| 台体结构 | 外部砖石砌筑，内部砖拱券结构（拱顶），分上下二至三层 |
| 底层 | 下筑基与城墙平，外出一丈四五尺有余，内出五尺有余 |
| 中层 | 空豁，四面箭窗，可驻兵30-50人，存放武器弹药粮食 |
| 顶层（台面） | 建楼橹，环以垛口，可架设佛郎机等火器 |
| 箭窗/铳眼 | 每面3-4个，上下交错排列，可向外射击 |
| 门道 | 入口设在城墙墙顶一侧，通过梯道上下，门可封闭 |
| 间距 | 约100-200米（一箭至两箭之地），确保相邻敌台火力交叉覆盖 |

### 驻兵配置

| 配置项 | 数量/职责 |
|--------|----------|
| 百总 | 1名——专管调度攻打 |
| 台头、副 | 2名——专管台内军器辎重 |
| 主客军士 | 30-50名不等（战时可容百人） |
| 标准火器 | 佛郎机8架 |
| 存粮 | 可支持短期坚守 |

### 戚继光论空心敌台

戚继光在《练兵实纪》中详述空心敌台的设计理念：

> "台高五丈，虚中为三层，台宿百人。铠仗糗粮具备。……令守台军五十名，于垛口内住宿，盘诘行人，举放火炮信号。……每台火器以佛郎机八架为率。"

## 四、烽燧系统

### 总体数据

| 项目 | 数据 |
|------|------|
| 全国烽燧总数 | 约**5,723座** |
| 分布范围 | 沿长城墙体内外两侧及通往内地的延伸线 |
| 间距 | "五里一墩，十里一台"（约2.5-5公里），以目视可见为原则 |
| 台体高度 | 约5-8米（约1丈5尺至2丈5尺） |
| 台基尺寸 | 底部约6-8米见方 |
| 台体结构 | 北方多为夯土或砖石结构；西北多为纯夯土 |
| 顶部设施 | 垛墙、柴草堆（昼烟夜火）、旗杆、火炮 |
| 驻守人数 | 每台约5-10人（墩军/烽军） |
| 配备器材 | 旗杆、灯笼、火铳/火炮、干粮、饮水、火池、柴草 |

### 烽火信号规则

#### 按敌军数量（成化二年/1466法令）

> "令边候举放烽炮，若见敌一、二人至百余人举放一烽一炮，五百人二烽二炮，千人以上三烽三炮，五千以上四烽四炮，万人以上五烽五炮。"

| 敌情等级 | 白昼信号（烟） | 夜间信号（火） | 炮声信号 |
|---------|--------------|--------------|---------|
| 1-100余人 | 举**一烽烟** | 举**一火** | 放**一炮** |
| 约500人 | 举**二烽烟** | 举**二火** | 放**二炮** |
| 千人以上 | 举**三烽烟** | 举**三火** | 放**三炮** |
| 五千以上 | 举**四烽烟** | 举**四火** | 放**四炮** |
| 万人以上 | 举**五烽烟** | 举**五火** | 放**五炮** |

#### 按敌军方向（以旗色/灯数辨别）

**白天放炮举旗**：
> "一炮青旗贼在东，南方连炮旗色红。白旗三炮贼西至，四炮玄旗北路逢。"

**夜晚改用悬灯**：
> "一灯一炮贼从东，双灯双炮看南风，三灯三炮防西面，四灯四炮北方攻。"

#### 信号传递规则

1. **逐站传递**：看到前一站信号后立即重复，逐站接力
2. **传递速度**：一昼夜可达7,000余里。蓟镇前线到北京理论上可在2-3小时内完成
3. **验证机制**：各站传递后需记录时间，事后查验延误。延误者军法处置
4. **解除信号**：敌军退去后举放"平安火"（单火一炬），表示警报解除
5. **虚报惩罚**：无敌情而举烽者，军法从事

#### 烽火燃料

| 燃料 | 用途 | 备注 |
|------|------|------|
| 狼粪（狼烟） | 昼间发烟信号 | 浓烟不易被风吹散（实际上各种柴草也常用） |
| 干柴/稻草 | 夜间发火信号 | 火光明亮，远处可见 |
| 硫磺/硝石 | 辅助燃料 | 加快点燃速度 |
| 火药/火铳/小炮 | 声响信号 | 配合烟火使用 |

## 五、墩台体系（非烽燧类）

### 墩台分类

| 类型 | 功能 | 规格 | 位置 |
|------|------|------|------|
| **骑墙墩台**（实心敌台） | 加强墙体防御，供士兵登高瞭望射击 | 高约5-8米，底部约6-8米见方，实心夯土/砖石 | 城墙之上，间距约200-300米 |
| **空心敌台** | 驻兵防御、火力点（详见上节） | 高约12-15米，底部约10-12米 | 城墙之上，间距约100-200米 |
| **烽火台/烽燧** | 传递军事信号 | 高约5-8米（详见上节） | 城墙内外两侧 |
| **镇北台** | 最大单体墩台，瞭望指挥 | 高约30余米，底部约80米见方，四层台体 | 延绥镇榆林北 |

### 镇北台（长城最大墩台）详述

| 项目 | 数据 |
|------|------|
| 位置 | 陕西榆林市北约5公里，延绥镇长城线上 |
| 建造时间 | 万历三十五年（1607），延绥巡抚涂宗浚督建 |
| 总高度 | 约30.51米（四层台体叠加） |
| 第一层（基台） | 底边约82米x80米（南北x东西），高约10米 |
| 第二层 | 底边约54米x43米，高约11米 |
| 第三层 | 底边约28米x22米，高约5米 |
| 第四层（顶台） | 底边约12米x10米，高约4.5米 |
| 建筑材料 | 夯土包砖 |
| 功能 | 瞭望指挥、烽火信号传递、控制互市（台北即款贡城/互市场地） |

## 七、长城修筑的主要时期

| 时期 | 主要工程 | 主持者 | 备注 |
|------|---------|--------|------|
| 洪武年间（1368-1398） | 山海关、嘉峪关等关城修筑；辽东边墙；老龙头、小河口长城 | 徐达、冯胜 | 奠基阶段 |
| 永乐年间（1402-1424） | 蓟镇、宣府、大同沿线修缮 | — | 北边防御全面展开 |
| 正统-景泰年间（1436-1457） | 土木堡之变后紧急加固内长城 | 于谦等 | 紧急防御 |
| 成化年间（1465-1487） | 辽阳段（成化三年/1467）；宁夏河东长城387里（成化十年/1474） | 韩斌、徐廷璋、范瑾 | 大规模修建 |
| 弘治年间（1488-1505） | 固原镇长城修筑 | — | 九边体系定型 |
| 嘉靖年间（1522-1566） | 大同、宣府大修；蓟镇加固 | 翁万达等 | 北虏压力最大（正统至嘉靖末为大规模兴筑主要时期，历时约130年） |
| 隆庆年间（1567-1572） | 蓟镇包砖空心敌台（1568起）；北路龙门所外边补筑 | 戚继光、谭纶 | 长城防御技术巅峰 |
| 万历年间（1573-1620） | 大同镇屯堡257座、敌台1,028所砖包；延绥大边长城数百里；辽东边墙加固（宽甸六堡等） | 张守中、李成梁等 | 最后大规模修建 |
| 天启-崇祯年间（1621-1644） | 山海关-宁远-锦州防线 | 孙承宗、袁崇焕 | 关宁锦防线（非传统长城） |
