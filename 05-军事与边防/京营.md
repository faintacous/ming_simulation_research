# 京营

## 一、京营概述

京营是明代驻守京师的中央禁卫军，为全国最精锐的常备兵力。京营的组织形式历经多次变革，其兵力和战斗力从永乐年间的鼎盛到崇祯年间的衰落，折射出整个明代军事制度的盛衰。

---

## 二、三大营（永乐朝创设）

### 1. 创设背景

永乐年间，明成祖朱棣以靖难之役起家，深知军事力量的重要性。他将京师迁至北京后，将原南京留守兵力和北方新编兵力整合为三大营。

### 2. 三大营编制

#### 五军营

| 项目 | 内容 |
|------|------|
| 创设 | 永乐年间（约1403-1407年） |
| 职能 | 步骑兵主力，负责列阵冲杀 |
| 编制 | 分中军、左哨、左掖、右哨、右掖五部 |
| 兵力 | 约20万-25万人（永乐年间全盛时） |
| 兵种 | 以步兵和骑兵为主 |
| 训练 | 列阵操演、骑射、步战 |
| 武器 | 刀枪弓弩、骑兵配马 |
| 指挥 | 提督总兵官1人（勋臣充任） |

#### 三千营

| 项目 | 内容 |
|------|------|
| 创设 | 永乐年间 |
| 名称由来 | 初以归附蒙古骑兵3,000人为骨干编成，故名 |
| 职能 | 精锐骑兵部队，负责巡哨、突击 |
| 编制 | 分五司：左哨、右哨、左掖、右掖、中军 |
| 兵力 | 约7万-8万人（永乐年间，含后续扩充） |
| 兵种 | 以骑兵为主 |
| 特点 | 包含大量蒙古降兵和边地善骑者 |
| 武器 | 弓箭、马刀、长枪 |
| 指挥 | 提督总兵官1人 |

#### 神机营

| 项目 | 内容 |
|------|------|
| 创设 | 永乐年间（一般认为创于永乐初年，不晚于永乐八年第一次北征） |
| 职能 | 火器部队，世界上最早的独立建制火器部队之一 |
| 编制 | 分中军、左哨、左掖、右哨、右掖五部 |
| 兵力 | 约5万-7万人（永乐年间） |
| 兵种 | 火铳手、火炮手 |
| 武器配置 | 火铳（手铳）、碗口铳、火炮、火箭 |
| 作战方式 | 列阵远程射击，为步骑兵提供火力掩护 |
| 指挥 | 提督总兵官1人 |

### 3. 三大营合计兵力

| 时期 | 总兵力 | 备注 |
|------|--------|------|
| 永乐年间 | 约40万（一说约50万） | 含从各地抽调的精锐 |
| 宣德年间 | 约35万-40万 | 基本维持 |
| 正统年间 | 名义约50万，实际约20-25万 | 空额严重 |

### 4. 永乐年间京营的作战记录

- 永乐八年至二十二年（1410-1424年）：明成祖五次亲征漠北，京营三大营为核心兵力
- 每次北征动员京营兵力约30万-50万
- 神机营在作战中通常部署在阵前，以火器杀伤敌军后，五军营步骑冲击，三千营骑兵迂回包抄

## 三、十团营（景泰朝改编）

### 1. 改编背景

正统十四年（1449年）土木堡之变，英宗被俘，京营三大营精锐随征几乎全军覆没。于谦主持北京保卫战后，对京营进行重大改编。

### 2. 十团营编制

景泰元年（1450年），于谦奏请将三大营残余兵力改编为**十团营**：

| 项目 | 内容 |
|------|------|
| 改编时间 | 景泰元年（1450年） |
| 编制 | 将三大营兵力打散重编为10个团营 |
| 每团营 | 设坐营都督1人（统帅）、都指挥等若干 |
| 兵力分配 | 每团营约数千人至万余人 |
| 改编原则 | 将步、骑、火器兵混合编组，使每个团营都具备独立作战能力 |
| 选锋 | 从十团营中选拔精锐组成"选锋"，作为机动打击力量 |
| 总兵力 | 约15万-20万（土木堡之变后京营大损） |
| 指挥 | 总督京营戎政1人（文臣，地位高于坐营都督） |

### 3. 十团营的意义

- **混合编组**：改变了三大营按兵种分营的旧制，每团营步骑火器齐备
- **文臣督军**：设"总督京营戎政"一职，以文臣节制武将，加强文官对京营的控制
- **选锋制度**：精锐集中，提高了可用战力

## 四、十二团营（成化朝改编）

### 1. 改编过程

| 时间 | 事件 |
|------|------|
| 天顺元年（1457年） | 英宗复辟，废十团营，恢复三大营旧制（但已名存实亡） |
| 成化二年（1466年） | 再次改制，将三大营改编为十二团营 |
| 成化三年（1467年） | 十二团营正式运行 |

### 2. 十二团营编制

| 项目 | 内容 |
|------|------|
| 编制 | 12个团营 |
| 名称 | 奋武、耀武、练武、显武、敢勇、果勇、效勇、鼓勇、立威、伸威、扬威、振威 |
| 每营 | 设坐营官1人（勋臣）、监枪内臣1人（宦官监军） |
| 总指挥 | 总督京营戎政（文臣） + 协理京营戎政（勋臣） |
| 兵力 | 每营约数千至万人 |
| 总兵力 | 约12万-15万 |

### 3. 宦官监军的引入

成化年间十二团营的一大变化是**宦官正式进入京营指挥体系**：
- 每营设"监枪内臣"1人，由司礼监派出
- 名义上监督操练和火器使用，实际上干预军务
- 这为此后宦官控制京营埋下伏笔

## 五、三大营恢复（嘉靖朝）

### 1. 恢复过程

| 时间 | 事件 |
|------|------|
| 正德年间 | 十二团营弊端严重，兵力空虚 |
| 嘉靖二十九年（1550年） | 庚戌之变：俺答汗骑兵兵临北京城下，京营不堪一战 |
| 嘉靖二十九-三十年 | 废十二团营，恢复三大营旧名（五军营、三千营、神机营） |

### 2. 恢复后的编制

| 项目 | 内容 |
|------|------|
| 名义 | 恢复三大营 |
| 实际 | 仍按营制编组，每营下设若干小营（哨） |
| 总兵力 | 名义约38万（嘉靖奏疏），实际不足5-6万可用之兵 |
| 指挥 | 总督京营戎政（文臣）+ 协理京营戎政（勋臣） |

### 3. 嘉靖京营整顿措施

| 措施 | 内容 |
|------|------|
| 清查兵额 | 清查空额、虚冒 |
| 加强训练 | 定期操练制度 |
| 补充兵力 | 从各地选调精壮补充京营 |
| 效果 | 短期有所改善，长期仍难以扭转颓势 |

| 节点 | 兵力损失 | 影响 |
|------|----------|------|

## 七、京营的日常管理与弊端

### 1. 管理体制

| 时期 | 管理者 | 备注 |
|------|--------|------|
| 永乐-正统 | 勋臣提督 | 武将主导 |
| 景泰以后 | 总督京营戎政（文臣）+ 协理京营戎政（勋臣） | 文臣主导 |
| 成化以后 | 加监枪内臣（宦官） | 三方共管 |

### 2. 主要弊端

| 弊端 | 表现 |
|------|------|
| 占役 | 勋贵、宦官、文武官员大量占用京营军士从事私人劳役（修宅、搬运、种地等） |
| 空饷 | 将领虚报兵额，冒领饷银；嘉靖时空饷比例达60%以上 |
| 老弱充数 | 老弱残疾不予淘汰，点卯充数 |
| 训练废弛 | "三日一操"形同虚设，军士不识弓马 |
| 武器朽坏 | 火器、弓弩长期不修缮，堪用者极少 |
| 将不知兵 | 勋臣世袭统领，多不通军事 |

### 3. 占役的严重程度

据嘉靖朝奏疏记载：
- 勋戚占役京营军士多者数千人，少者数百人
- 各官府衙门役使京营军士修造、搬运
- 工部等衙门常年借用京营军士充当工役
- 京营实际在营操练者，有时不足在册兵力的十分之一

## 八、崇祯元年李邦华整顿京营

### 1. 李邦华到任时的京营状况

崇祯元年（1628年），李邦华以协理京营戎政身份到任视察：

| 项目 | 数据 |
|------|------|
| 名义兵力 | 约10万以上 |
| 实际可用 | 仅存约17,000人 |
| 占役问题 | 勋贵、宦官大量占用京营军士做私役 |
| 虚冒问题 | 将领虚报兵额，冒领饷银 |
| 编制膨胀 | 通过增设选锋、旗手、壮勇等名目，编制扩至12万以上 |

### 2. 整顿措施与结果

| 措施 | 内容 |
|------|------|
| 清查占役 | 清退被勋贵占用的京营军士 |
| 核实兵额 | 清查虚冒，核实实际在营人数 |
| 恢复操练 | 重建训练制度 |
| 受阻 | 襄城伯李守锜担任提督后"恨邦华挠之"，处处掣肘 |
| 结局 | 己巳之变后李邦华被免职，改革成果付之东流 |

[来源：[曹循：明代京营制度新探 - 腾讯新闻](https://news.qq.com/rain/a/20231101A08X9400)]

> 三大备操营"不习战阵"而食粮与正军等，三大备操营沦为权贵安插家仆和占役军士的场所，"占役"和"虚冒"问题越发严重。

## 九、崇祯末年京营的最终状态

### 1. 李自成进攻北京时的京营

崇祯十七年（1644年）三月，李自成大军逼近北京：

| 项目 | 情况 |
|------|------|
| 名义兵力 | 数万 |
| 实际可战之兵 | 约数千人 |
| 士气 | 极度低落，大量逃散 |
| 武器 | 多数朽坏不堪使用 |
| 饷银 | 长期欠饷，军心涣散 |
| 城防 | 北京内城城墙周长约40里，兵力严重不足 |

### 2. 崇祯最后的努力

- 崇祯帝试图从内帑拨银犒赏守城军士，但国库空虚
- 命太监曹化淳等督守各门
- 三月十八日，李自成兵至城下，京营守军大量投降或溃散
- 三月十九日，北京城破
