# 战船与水师

## 概述

明代水师是当时世界上规模最大的海军力量之一。永乐年间郑和船队远航西洋，鼎盛时期拥有3,800艘各类船只。嘉靖至崇祯时期（1521-1644），水师虽已从永乐盛世大幅萎缩，但在抗倭战争和海防中仍发挥重要作用。明代战船以福船、广船、沙船、苍山船等为主要船型，各有特色。

## 一、主要战船类型

### 1. 福船

| 项目 | 数据 |
|------|------|
| 产地 | 福建沿海 |
| 结构特点 | 底尖上阔，首尖尾宽两头翘 |
| 船型分级 | 一号福船（九丈）、二号福船（八丈） |
| 长度 | 一号约九丈（约28.8米），二号约八丈（约25.6米） |
| 水密隔舱 | 舱壁由水密隔舱构成，具备良好的抗沉性能 |
| 载员 | 可容百人以上 |
| 武器配置 | 大发贡（重型火炮）、佛郎机、碗口铳、火箭等 |
| 适用海域 | 深水远洋，适合外海作战 |
| 吃水 | 深，浅水区受限 |
| 战术优势 | "高大如城，可以俯瞰射击；遇小船可直接冲撞击沉" |

[来源：《筹海图编》；[明代水师考——大福船 - 知乎](https://zhuanlan.zhihu.com/p/1895292835114242284)]

**福船的作战评价**（《筹海图编》）：

> "福船高大如城，非人力可驱，全仗风势，上可下击，下难仰攻。"

### 2. 广船

| 项目 | 数据 |
|------|------|
| 产地 | 广东 |
| 来源 | 由东莞"乌糟"船和新会"横江"船演变而来 |
| 结构特点 | 船身坚固，不追求巨大体积，注重实用作战功能 |
| 材质 | 铁力木等硬木建造，坚固耐用 |
| 特点 | 船底平坦，吃水较福船浅 |
| 武器 | 配备佛郎机、碗口铳等火器 |
| 优势 | 坚固耐撞，适合近海作战 |
| 劣势 | 速度不及福船，远洋能力较弱 |

[来源：[古船扬帆·明代三大海船 - 知乎](https://zhuanlan.zhihu.com/p/553935671)]

### 3. 沙船

| 项目 | 数据 |
|------|------|
| 产地 | 长江流域及北方沿海 |
| 结构特点 | 平头、方艄、平底、船身较宽 |
| 帆桅 | 多桅多帆 |
| 吃水 | 浅，适合浅水航行 |
| 适用海域 | 长江口、渤海、黄海等浅水区域 |
| 特点 | 可在沙洲、浅滩通行 |
| 用途 | 内河、近海巡逻和运输 |

## 二、水师编制

### 1. 水师卫所

明初水师隶属卫所体系，沿海设有专门的水军卫所：

| 项目 | 数据 |
|------|------|
| 水军卫总数 | 约24卫（分布于沿海各省） |
| 分布 | 浙江、福建、广东、山东、直隶沿海 |
| 每卫兵额 | 名义5,600人（实际常不足） |
| 名义总兵力 | 约13万人（24卫 x 5,600） |

### 2. 水寨制度

嘉靖年间，沿海设立水寨作为海防据点：

| 区域 | 主要水寨 | 备注 |
|------|----------|------|
| 浙江 | 金山卫水寨、定海水寨等 | 抗倭前线 |
| 福建 | 烽火门水寨、铜山水寨、浯屿水寨等 | 重要海防据点 |
| 广东 | 南澳水寨等 | 南方海防 |

### 3. 水寨兵力衰减数据

## 三、船队规模变化

### 1. 鼎盛时期（永乐年间，1420年前后）

| 类型 | 数量 | 说明 |
|------|------|------|
| 巡船 | 1,350艘 | 沿海巡逻 |
| 战船 | 1,350艘 | 作战用途 |
| 大船（南京新江口基地） | 400艘 | 含远洋宝船 |

| 时期 | 状况 |
|------|------|
| 嘉靖初 | 远洋船队已不存在，沿海战船老化严重 |
| 嘉靖中后期 | 因抗倭需要，大量建造新型战船 |
| 万历年间 | 援朝战争动员水师，短暂恢复一定规模 |
| 天启-崇祯 | 水师继续衰落，战船数量和质量均下降 |

## 四、海战战术

### 1. 戚继光的水战原则

戚继光提出"水战，火为第一"的原则：

| 战术阶段 | 内容 |
|----------|------|
| 远距离 | 以火器（佛郎机、火箭等）压制敌方，令敌不敢露头 |
| 中距离 | 使用火箭、喷筒等纵火武器攻击敌船 |
| 近距离 | 组织士兵攀援敌船，以冷兵器搏杀 |
| 水陆配合 | 陆军阻击登陆敌军，水军打击海上敌船 |

[来源：[明代抗倭的海防战略 - 中国军网](http://www.81.cn/gfbmap/content/2021-08/12/content_296276.htm)]

### 2. 抗倭海战的主要战术

| 战术 | 说明 |
|------|------|
| 以大击小 | 利用福船体量优势，冲撞倭寇小船 |
| 火攻 | 用火箭、火球攻击倭船木质船体 |
| 编队作战 | 大船居中，小船两翼包抄 |
| 封锁航道 | 控制港口和水道，截断倭寇退路 |

### 3. 台州大捷水战（嘉靖四十年，1561年）

戚继光在台州之战中：
- 陆军组成一头两翼的冲锋部队，击溃登陆倭寇
- 水军同时打击海上倭寇船队
- 水陆协同，全歼倭寇

[来源：戚继光 - 维基百科](https://zh.wikipedia.org/zh-hans/%E6%88%9A%E7%BB%A7%E5%85%89)

## 七、水师衰落原因

| 原因 | 表现 |
|------|------|
| 海禁政策 | 仁宣以后收缩海防线，停止远洋航行 |
| 卫所兵员锐减 | 水军卫所兵额严重不足 |
| 战船老化 | 大型舰船停止建造，旧船逐渐朽坏 |
| 经费不足 | 造船和维护费用被挪用或削减 |
| 岛屿基地放弃 | 外海岛屿的军事据点撤回 |
| 海盗与走私 | 沿海治安恶化，水师疲于应付 |
