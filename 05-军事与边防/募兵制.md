# 募兵制

## 概述

明代募兵制是在卫所制度崩溃背景下逐步发展起来的军事制度。从正统年间零星募兵应急，到嘉靖年间大规模制度化，募兵逐渐取代卫所兵成为明军主力。募兵制以银饷养兵、兵将相习为特征，提高了军队战斗力，但也造成了沉重的财政负担，最终成为明末财政危机的重要推手。新式募兵不隶卫所、以民籍身份成为经制兵，军饷从嘉靖初年的年饷约9两递增到崇祯年间名义30余两，有"浙兵半天下"之说。

---

## 一、从卫所到募兵的转变过程

### 时间线

| 时期 | 事件 | 说明 |
|------|------|------|
| 正统年间（1436-1449） | 零星募兵 | 卫所兵逃亡严重，部分边镇开始临时募兵应急 |
| 景泰年间（1450-1456） | 募兵增加 | 土木堡之变后京营严重缺额，大量募兵补充 |
| 成化年间（1465-1487） | 常态化趋势 | 各边镇普遍出现募兵，但仍以卫所为主体 |
| 弘治年间（1488-1505） | 规模扩大 | 部分卫所逃亡率达70%-80%，军官奉命出兵时临时雇用矿工、盐工充数 |
| 嘉靖年间（1522-1566） | 制度化转折 | 倭寇侵扰与庚戌之变，募兵全面制度化，成为明军主力 |
| 隆庆-万历年间（1567-1620） | 募兵为主 | 营兵制确立，卫所军事功能基本丧失，仅保留管理军籍和屯田职能 |
| 天启-崇祯年间（1621-1644） | 完全依赖募兵 | 辽东战事激增募兵需求，军饷负担空前沉重 |

### 转变的关键节点

1. **嘉靖三十四年（1555年）**：戚继光受命在浙江义乌募兵，创建戚家军，标志着募兵制的成功典范 [来源：《明史·戚继光传》]
2. **嘉靖二十九年（1550年）**：庚戌之变，京营在册38万仅可调5-6万，暴露卫所制度的彻底失败 [来源：《明史·兵志》]
3. **隆庆二年（1568年）**：戚继光调任蓟镇总兵，以募兵编练北方边军，营兵制推广至九边

## 二、募兵的来源

| 来源类别 | 具体人群 | 说明 |
|----------|----------|------|
| 农民 | 义乌、处州等地农民 | 戚继光的主要募兵对象，义乌矿农最为悍勇 |
| 矿工 | 浙江、福建、广东矿徒 | 习惯械斗，体力强健，戚继光尤重用之 |
| 盐工/盐徒 | 沿海盐场工人 | 生活艰苦，体质过硬 |
| 流民 | 各地流移避差人户 | 兵源最广泛的来源之一 |
| 商贩 | 远引客商 | 辽东等地招募对象之一 |
| 罪犯 | 犯该徒杖以下罪名者 | 可以原免罪行入伍 |
| 土著/边民 | 辽东、陕西等地边民 | 熟悉边情，善骑射 |
| 归附蒙古人 | 降附的蒙古骑兵 | 主要编入三千营等骑兵部队 |

### 募兵选拔标准

戚继光在《纪效新书》中明确提出选兵标准：
- **优先**：乡野老实之人、矿工农夫
- **排除**：城市油滑之徒、衙门奸巧之人
- **体格**：面黑、手粗、脚大、有力气者
- **年龄**：通常为15-40岁壮年

## 四、募兵的编制和管理

### 营兵制编制体系

募兵采用营兵制编制，与卫所制并行但独立运作：

| 层级 | 主官 | 兵力规模 | 说明 |
|------|------|----------|------|
| 镇 | 总兵官 | 数万至十余万 | 统辖一镇军务 |
| 协 | 副总兵 | 数千至万余 | 协助总兵 |
| 营 | 参将/游击 | 约3,000-6,000人 | 独立作战单位 |
| 部（司） | 守备/千总 | 约500-1,000人 | |
| 哨 | 把总 | 约100-200人 | |
| 队 | 队长 | 约10-12人 | 基层战术单位 |

### 戚继光的编制创新

戚继光在《纪效新书》《练兵实纪》中确立了严密的编制体系：

**步营编制**（以《练兵实纪》为准）：
- 全营2,699人
- 鸟铳手1,080人（占40%）
- 配置鸳鸯阵、火器队等战术编组

**车营编制**：
- 全营约3,109人
- 鸟铳手512名
- 佛郎机手768名
- 火器人员占全营41%

**鸳鸯阵基本单位**（12人/队）：
| 位置 | 兵种 | 人数 | 武器 |
|------|------|------|------|
| 前排 | 队长 | 1 | 腰刀 |
| 第二排 | 牌兵 | 2 | 长牌、藤牌各一，配腰刀和标枪 |
| 第三排 | 狼筅兵 | 2 | 狼筅 |
| 第四排 | 长枪兵 | 4 | 长枪 |
| 第五排 | 镋钯手 | 2 | 镋钯 |
| 后方 | 火头兵 | 1 | 不参战，负责伙食 |

### 管理特点

1. **兵将相习**：募兵由将领亲自招募训练，兵与将之间有私属关系（区别于卫所"兵将分离"）
2. **赏罚严明**：戚继光制定了严格的军法——"凡临阵，贼一人入阵，两旁不救者，两旁皆斩"
3. **连坐法**：一小旗有逃者，全旗受罚；一队有退者，队长与之同罪
4. **训练制度**：每日操练，月有月考，季有季考

## 五、著名的募兵案例

### 1. 戚继光义乌募兵（1559年）

- **背景**：嘉靖三十八年（1559年），戚继光在浙江抗倭，原有卫所兵不堪用
- **地点**：金华府义乌县及周边永康、东阳、处州等地
- **规模**：初募约3,000-4,000人
- **来源**：义乌矿农为主，兼及周边农民
- **选兵标准**：选"乡野老实之人"，排斥"城市油滑之徒"
- **成效**：戚家军抗倭战绩辉煌——台州大捷九战九胜，歼倭5,500余人，己方阵亡仅约20人
- **编制创新**：创鸳鸯阵，12人为一队，长短兵器搭配
- **意义**：证明了募兵制的军事优越性，成为全国推广的范本

### 2. 俞大猷的募兵实践

- 在福建、广东等地募集兵员抗倭
- 善用水兵，主张"以大船击小船，以众击寡"
- 创鸳鸯船战法

### 3. 辽东募兵（万历末-崇祯）

- **背景**：萨尔浒之战后辽东军事形势紧迫
- **规模**：辽东驻军由万历初约8万增至天启年间约14万
- **来源**：关内各省招募，浙兵、川兵入辽
- **问题**：饷银浩大，欠饷严重，兵变频发

### 4. 李成梁家丁制

- 辽东总兵李成梁以私人家丁为核心战力
- 家丁待遇优厚，装备精良
- 实为募兵制的极端化——精锐集中于少数将领手中，加剧将领割据倾向

## 六、募兵制的利弊分析

### 优势

| 方面 | 说明 |
|------|------|
| 战斗力 | 募兵经过选拔和训练，战斗力远超卫所兵 |
| 灵活性 | 可根据需要随时扩募或裁减 |
| 兵将相习 | 将领与士兵长期磨合，配合默契 |
| 激励机制 | 以饷银为激励，赏罚分明 |

### 弊端

| 方面 | 说明 |
|------|------|
| 财政负担 | 养兵费用巨大，嘉靖后军费成为财政最大支出项 |
| 兵为将有 | 兵将间形成私属关系，助长军阀倾向 |
| 质量参差 | 战事紧急时降低标准，兵员素质难保 |
| 欠饷兵变 | 饷银不足导致频繁兵变，如崇祯年间多次 |

## 七、募兵的地域特征

### "浙兵半天下"

嘉靖抗倭战争后，浙江处州、绍兴、金华、台州等地的募兵在战场上声名鹊起，各地争相招募浙兵，形成了"浙兵半天下"的局面。

| 来源地 | 特点 | 备注 |
|--------|------|------|
| 义乌 | 矿农为主，悍勇善斗 | 戚继光首选兵源 |
| 处州（丽水） | 山民体格健壮 | 处州兵名声远扬 |
| 绍兴 | — | 浙兵主要来源之一 |
| 金华 | — | 浙兵主要来源之一 |
| 台州 | — | 浙兵主要来源之一 |
| 广东 | "惯战民兵" | 嘉靖三十四年南京招募广东兵3,000人 |

[来源：曹循《明代军制演进与盛衰之变》；[明代中后期东南地区兵制变迁](http://210.14.65.66/upload/2023/0707/758f203d-3559-4136-9263-266fc6ab2e1f.pdf)]

### 募兵的民兵性质

万历时总兵侯继高撰《全浙兵制》详列各营"军兵"与"民兵"员额。其中"民兵"与"军兵"一起编为营伍，隶于将领，是朝廷经制官军中的民籍兵士。新式募兵不隶卫所，以民籍身份而成为经制兵，这是与卫所军户兵的根本区别。
