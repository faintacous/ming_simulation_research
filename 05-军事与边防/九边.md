# 明代九边重镇

> 聚焦时段：嘉靖至崇祯（1521-1644），兼及永乐时期对比数据
> 本文涵盖九边名称与驻地、兵力变迁、行政管理体系、军费支出与互市贸易

---

## 一、九边概述

明代九边是沿北部边境长城一线设置的九个军事重镇，东起鸭绿江，西至嘉峪关，全线绵延万余里，是明帝国抵御蒙古势力南下的核心防御体系。

九边建制并非一次成形，而是从永乐到弘治年间逐步确立：
- **初设四镇**：辽东、宣府、大同、延绥
- **继设三镇**：宁夏、甘肃、蓟州
- **后增二镇**：太原（山西镇）、固原（陕西镇）

弘治年间九边体制基本定型，此后沿用至明亡。

> **来源**：《九边建制》百度百科；蒋栋良《明代的"九边"是怎样一种存在？》，观察者网，2018-01-06

### 3.2 兵力变迁分析

- **永乐时期**：明成祖五次北征，边防兵力充实，辽东镇独占23万，为最大军镇
- **万历时期**：经嘉靖末年整顿，册额达到峰值88.5万（万历四十八年），宣府镇以15.1万居首
- **崇祯时期**：辽东战事消耗、财政困难、兵员逃亡，总兵力缩减至59万，辽东镇从23万锐减至8万
- **关键转折**：万历四十六年（1618）辽东战事爆发，两年零八个月辽饷支出超过2000万两，九边体系逐渐瓦解

## 四、三组行政管理体系

九边实行军事、民政、监察三套平行管理体系，相互制约，共同维护边防秩序。

### 4.1 军事体系

```
总督（跨镇统辖）
 └── 总兵官（一镇最高军事长官）
      ├── 副总兵
      ├── 参将（分守各路）
      ├── 游击将军（机动部队）
      └── 守备、千总、把总（基层军官）
```

**四大定设总督：**

| 总督名称 | 管辖范围 | 设立/定设时间 |
|----------|----------|--------------|
| 蓟辽总督 | 蓟州、辽东、保定三镇 | 嘉靖年间设立 |
| 宣大总督 | 宣府、大同、山西三镇 | 嘉靖二十九年（1550）定设 |
| 三边总制（总督） | 延绥、宁夏、甘肃三镇 | 驻固原，弘治年间设 |
| 固原镇 | 三边总制兼管 | 作为三边总制驻地 |

**总兵官的演变：**
- 明初：临战派遣，战后卸任
- 永乐后：北方总兵官逐渐常驻化
- 嘉靖后：总兵官权力被巡抚、总督分割，地位下降

### 4.2 民政体系

```
巡抚（以都御史出任，兼理军务）
 ├── 布政使司（管理财政、民政）
 ├── 兵备道（兵备副使/佥事，文官管军）
 └── 管粮郎中（户部派驻，管理军饷转运）
```

**巡抚的崛起：**
- 正统元年（1436），辽东镇首设都御史巡抚
- 嘉靖初年，各地镇守太监陆续撤回
- 巡抚成为各地驻军实际上的首脑，统揽军政大权

### 4.3 监察体系

```
都察院
 ├── 巡按御史（一年一任，巡察各地）
 │    ├── 审录囚犯
 │    ├── 考察官吏
 │    ├── 弹劾违法
 │    └── 体察民情
 └── 巡关御史（专门巡视关隘）
```

**监察制度要点：**
- 巡按御史任期一年，有权直接向皇帝上奏弹劾
- 万历二十八年起，改由巡按御史负责边防阅视
- 形成督、抚、按、道兼用的边防监察体制
- 巡按虽品级低（正七品），但代天子巡察，权力极大

### 4.4 三套体系的相互制约

| 制约关系 | 具体表现 |
|----------|----------|
| 总督制约总兵 | 总督节制数镇总兵，掌握战略决策权 |
| 巡抚制约总兵 | 巡抚以文制武，主持一方军政 |
| 巡按制约巡抚 | 巡按御史可监督弹劾巡抚 |
| 三司制约巡抚 | 布政使司、按察使司、都指挥使司独立运作 |
| 兵备道制约将领 | 文官兵备道监管地方军务 |

> **来源**：《明代九边军事统率制度的变迁》中国社会科学院历史研究所；《明朝巡按御史都"巡"什么？》澎湃新闻；《总督》维基百科

### 5.3 军费来源构成

| 屯田收入 | 明初主要来源，正统后逐渐萎缩 |
| 民运粮 | 各省按定额运粮至边镇 |
| 京运年例银 | 户部太仓库拨银，嘉靖后为主要来源 |
| 盐引开中 | 商人运粮至边换取盐引（明初重要渠道） |
| 屯粮折银 | 屯田实物折银交纳 |

### 5.4 嘉靖四十五年经制改革

嘉靖四十四年（1565），户部尚书高耀奏准清查各边镇实际兵马数额，核定"应支岁用"。次年根据各镇回报，折中裁定军费定额，称为"嘉靖四十五年议定经制"。这是明代首次系统核定九边军费的尝试。

### 5.5 崇祯三饷

| 加派名称 | 开始时间 | 年征额 | 用途 |
|----------|----------|--------|------|
| 辽饷 | 万历四十六年（1618）始征 | 约520万两 | 辽东军费 |
| 剿饷 | 崇祯十年（1637） | 约280万两 | 镇压农民起义 |
| 练饷 | 崇祯十二年（1639） | 约730万两 | 练兵 |

三饷合计年加派超过1500万两，远超正常田赋收入，成为压垮明朝财政的最后稻草。

## 六、互市贸易

### 6.1 隆庆和议（俺答封贡）

#### 背景
- 嘉靖十三年（1534）起，蒙古土默特部俺答汗数百次向明朝提出"通贡互市"
- 明世宗一律拒绝，蒙古以武力掠夺弥补贸易需求
- 嘉靖二十九年（1550）"庚戌之变"：俺答汗兵临北京城下

#### 和议达成
- 隆庆四年（1570），俺答汗之孙把汉那吉叩关投明
- 隆庆五年（1571），明穆宗封俺答汗为"顺义王"
- 同时开放十余处边境互市市场

### 6.2 互市贸易类型

| 类型 | 说明 | 特点 |
|------|------|------|
| 贡市 | 伴随朝贡进行的贸易 | 官方主导，规模有限 |
| 官市 | 官方组织的定期互市 | 双方政府定价 |
| 民市 | 允许民间自由交易 | 规模最大，活力最强 |
| 月市 | 按月定期开放的市场 | 常态化交易 |

### 6.3 贸易规模实例

**隆庆五年（1571）张家口堡交易记录：**
- 官市交易期：6月13日至26日（不到半月）
- 官市成交马匹：1993匹
- 官市成交金额：白银15,277两
- 平均马价：每匹约7.67两
- 私市交易量：马骡牛羊约9000头（远超官市）

### 6.4 互市的影响

| 方面 | 积极影响 | 消极影响 |
|------|----------|----------|
| 军事 | 边境和平近半世纪，军事压力大减 | 部分边军因和平而松懈 |
| 经济 | 促进蒙汉经济交流，减少军费支出 | 白银外流 |
| 社会 | 边境居民安居乐业 | 走私问题时有发生 |
| 政治 | 明蒙关系改善 | 对东部女真防范不足 |

### 6.5 互市市场分布

隆庆和议后开放的主要互市点：
- **宣府镇**：张家口堡（最大互市点）
- **大同镇**：得胜堡、新平堡
- **山西镇**：水泉营
- **延绥镇**：红山市口
- **宁夏镇**：清水营

互市维持了近半个世纪的边境和平，直至明朝灭亡。

> **来源**：《俺答封贡》维基百科、百度百科；《隆庆和议之马市开设》人民政协报，2025-01-14；《在万里长城的心脏做生意》中国日报网

## 七、九边防御体系的结构层级

### 7.1 防御纵深

```
第一道防线：长城边墙 + 敌台 + 烽燧
    ↓
第二道防线：关城 + 堡寨 + 军台
    ↓
第三道防线：卫所城 + 镇城
    ↓
纵深支援：内地卫所 + 京营
```

### 7.2 各镇下辖防区划分

每镇按地理分为若干"路"（段），每路设参将或游击驻守：
- **蓟州镇**：分东、中、西三协，下辖十二路
- **宣府镇**：分东、中、西三路及万全右卫
- **大同镇**：分东、中、西三路
- **辽东镇**：分东、西二路

### 7.3 兵种构成

| 兵种 | 职能 | 占比 |
|------|------|------|
| 主兵（守兵） | 常驻本镇防守 | 约70% |
| 客兵 | 从他镇或内地调防 | 约20% |
| 机动兵（游兵） | 巡逻、增援 | 约10% |

> **来源**：《明代驻守长城的兵力分配》网易订阅；《九边建制》百度百科

## 八、九边的衰落

### 8.1 衰落的主要原因

1. **屯田破坏**：军屯土地被将领侵占，军士沦为佃户
2. **兵员空额**：将领吃空饷，实际在营兵力远低于册额
3. **军饷拖欠**：财政困难导致边军长期欠饷，士气低落
4. **指挥混乱**：文官与武将争权，指挥体系效率低下
5. **两线作战**：辽东后金崛起与西北民变同时爆发

### 8.2 "明军不满饷，满饷不可敌"

这句话反映了明末边军的真实状态：
- 崇祯年间九边实际在营兵力可能不到册额的一半
- 即便有兵，也因欠饷数月乃至数年而无心战守
- 但凡能足额发饷、认真训练，明军战力并不逊色

> **来源**：《"明军不满饷，满饷不可敌"》腾讯新闻；曹循《明代军制演进与盛衰之变》知乎
