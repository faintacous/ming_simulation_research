# 军事后勤

## 概述

明代军事后勤体系经历了从"屯田自给"到"全面依赖财政拨饷"的根本转变。洪武年间朱元璋以屯田制度实现"养兵百万不费百姓一粒米"的构想，但中叶以后屯田崩溃、募兵制兴起，军饷成为国家财政最大的支出项。嘉靖至崇祯年间（1521-1644），军饷来源从开中法、盐引、太仓银库到最终的"三饷"加派，折射出明代财政从宽裕到崩溃的全过程。

## 一、军饷来源体系

### 1. 开中法

| 项目 | 内容 |
|------|------|
| 创设时间 | 洪武三年（1370年） |
| 基本原理 | 商人将粮食运至边镇军仓，换取盐引（食盐运销许可证），再凭盐引到指定盐场支取食盐贩卖获利 |
| 三步流程 | 报中（商人应招纳粮）-> 守支（等待领取盐引）-> 市易（凭盐引支盐贩卖） |
| 核心作用 | 利用盐利吸引商人主动运粮至边镇，解决边防军粮运输难题 |
| 附带效果 | 商人在边地就地开垦"商屯"种粮纳入军仓，促进边地开发 |

[来源：开中法 - 百度百科](https://baike.baidu.com/item/%E5%BC%80%E4%B8%AD%E6%B3%95/1705625)；[开中法 - 维基百科](https://zh.wikipedia.org/zh-hans/%E9%96%8B%E4%B8%AD%E6%B3%95)

### 2. 叶淇变法（弘治年间）

| 项目 | 内容 |
|------|------|
| 时间 | 弘治年间（1488-1505年） |
| 改革者 | 户部尚书叶淇 |
| 内容 | 改"纳粮开中"为"纳银开中"——商人不再运粮至边地，改为直接向运司缴纳银两换取盐引 |
| 盐引价格 | 每引盐输银三四钱不等 |
| 短期效果 | 太仓银增至百余万两，国家财政收入骤增 |
| 长期恶果 | 边地盐商大都举家内迁，商屯迅速破坏，边军粮食储备大减 |

**叶淇变法后的制度**：

- 运司收银解至太仓，再由太仓分拨各边
- "开中法"并未完全中断，此后"开中法"与"折色法"并存（双轨盐法体制）
- 但纳粮开中占比日益缩小，边镇军粮日益依赖银两采购

[来源：[盐引·公债·资本市场 - 中国经济史论坛](http://economy.guoxue.com/?p=2832)]

### 3. 盐引制度

| 项目 | 内容 |
|------|------|
| 起源 | 宋代创设盐引制度 |
| 明代盐引 | 官府向盐商发放的食盐运销许可凭证 |
| 交换物 | 除粮米外，有时也可用布绢、银钱、马匹等换取 |
| 两淮盐引 | 最重要的盐引来源，利润最大 |
| 问题 | 盐引超发导致贬值，商人积引难兑 |

[来源：盐引 - 百度百科](https://baike.baidu.com/item/%E7%9B%90%E5%BC%95/3422978)

### 4. 太仓银库

| 项目 | 内容 |
|------|------|
| 性质 | 户部太仓银库，国家中央财政最重要的银库 |
| 收入来源 | 各省解送的赋税银、盐课银、关税等 |
| 主要支出 | 边饷为最大支出项 |
| 鼎盛时期 | 弘治年间太仓银多至百余万两（叶淇变法后短期效果） |
| 万历四十六年 | 岁入仅389万两，岁出边饷381万两，已入不敷出 |

[来源：[赫赫武功的万历三大征 - 国家人文历史](https://www.gjrwls.com/military/20240415/964547718443696128.html)]

## 三、三饷加派

### 1. 辽饷

| 项目 | 内容 |
|------|------|
| 开征时间 | 万历四十六年（1618年） |
| 起因 | 辽东防卫后金的军事需要 |
| 提议者 | 户部尚书李汝华 |
| 初始税率 | 亩加三厘五毫 |
| 递增 | 次年复加三厘五毫，万历四十八年（1620年）又加二厘 |
| 合计税率 | 前后平均每亩加征银九厘 |
| 初始总额 | 520万零62两 |
| 崇祯四年（1631年）调整 | 提高到每亩一分二厘，共银660万两 |
| 加关税盐课后 | 共得银740万8,298两 |

### 2. 剿饷

| 项目 | 内容 |
|------|------|
| 开征时间 | 崇祯十年（1637年） |
| 起因 | 镇压农民起义 |
| 提议者 | 兵部尚书杨嗣昌 |
| 内容 | 奏请增兵12万，加征"剿饷" |
| 总额 | 280万两 |
| 停征 | 崇祯十三年（1640年）被迫停止 |

### 3. 练饷

| 项目 | 内容 |
|------|------|
| 开征时间 | 崇祯十二年（1639年） |
| 起因 | 崇祯十一年九月至十二年三月，清兵多次长驱内地 |
| 提议者 | 杨嗣昌 |
| 总额 | 730余万两 |

### 4. 三饷总览

| 饷名 | 年征额 | 开征时间 | 停征时间 |
|------|--------|----------|----------|
| 辽饷 | 740万两（崇祯四年后） | 万历四十六年（1618年） | 未停 |
| 剿饷 | 280万两 | 崇祯十年（1637年） | 崇祯十三年（1640年） |
| 练饷 | 730余万两 | 崇祯十二年（1639年） | 未停 |

**对比**：御史郝晋言："万历末年，合九边饷止二百八十万。今加派辽饷至九百万。剿饷三百三十万，业已停罢，旋加练饷七百三十馀万。"

[来源：三饷 - 维基百科](https://zh.wikipedia.org/zh-hans/%E4%B8%89%E9%A4%89)；[三饷 - 百度百科](https://baike.baidu.com/item/%E4%B8%89%E9%A5%B7/1716153)

## 四、粮草运输方式

### 1. 漕运

| 项目 | 内容 |
|------|------|
| 定义 | 利用内陆河流（主要是大运河）运送粮食 |
| 明代规定 | 自1415年起，漕运全部经由内河，停止海运 |
| 漕船数量 | 约12,000只 |
| 漕军数量 | 约120,000人 |
| 中转仓库 | 淮安常盈仓为大型中转仓，常蓄150万石粮食 |
| 年运量 | 约400万石（高峰期） |
| 目的地 | 主要运往京师和北方边镇 |

[来源：漕运 - 维基百科](https://zh.wikipedia.org/zh-hans/%E6%BC%95%E8%BF%90)；[文史漫步·明清漕运的经济账](https://www.gzszx.gov.cn/wstd/wsmb/32884.shtml)

### 2. 驿站系统

| 项目 | 内容 |
|------|------|
| 设置标准 | 主要交通线上每60里设驿站，每10里设递铺 |
| 军用驿传 | 边地设卫所管理的军用驿传系统（塘铺） |
| 驿站功能 | 传递军情、转运军需物资（粮食、武器等） |
| 驿站规模 | 以盂城驿为例：鼎盛时厅房100多间、驿马65匹、驿船18条、马夫水夫200多人 |
| 递运所 | 专门负责运输官方物资的机构 |

[来源：[明代大运河沿线驿站选址初探](http://dyyjy.zjgsu.edu.cn/show.asp?cid=404)]

### 3. 开中法下的粮草运输

| 方式 | 说明 |
|------|------|
| 商人运粮 | 商人自行组织运力将粮食运至边镇军仓 |
| 商屯 | 商人在边地就近开垦种粮，减少运输成本 |
| 叶淇变法后 | 改为银两拨付，边镇就地采购粮食 |

## 五、军屯的实际产出

### 1. 屯田制度设计

| 项目 | 内容 |
|------|------|
| 授田标准 | 每军丁受田约50亩 |
| 屯种比例 | 边地七分屯种三分守城，内地八分屯种二分守城 |
| 征粮标准 | 每军丁纳"屯田籽粒" |
| 官府供给 | 耕牛、农具、种子 |

### 2. 屯田面积变化

| 时期 | 面积 | 备注 |
|------|------|------|
| 洪武年间 | 893,194顷（约8,930万亩） | 鼎盛期 |
| 万历年间 | 减少249,000余顷 | 比洪武减少约28% |
| 崇祯年间 | 大幅萎缩 | 具体数据难考 |

### 3. 屯田产出的衰减原因

| 原因 | 影响 |
|------|------|
| 军官侵占 | 将屯田变为私产，不再缴纳籽粒 |
| 权贵夺田 | 勋贵、宦官大量侵占屯田 |
| 军士逃亡 | 无人耕种，田地荒废 |
| 水利失修 | 灌溉设施年久失修，产量下降 |
| 赋额僵化 | 田地减少而征收定额不变 |

## 六、崇祯朝军饷危机

### 1. 财政收支失衡

| 项目 | 数据 |
|------|------|
| 太仓年收入 | 约400万两（崇祯初年） |
| 军费年支出 | 远超1,000万两（含三饷加派后） |
| 年缺口 | 至少100万两，多时超过400万两 |

### 2. 欠饷问题

| 时间 | 事件 | 后果 |
|------|------|------|
| 崇祯年间 | 各地边兵长期欠饷 | 兵变频发 |
| 崇祯十七年（1644年） | 唐通部勤王军每人仅获赏银五钱 | 唐通投降李自成 |
| 崇祯十七年 | 北京守军"三大营精锐"长期欠工资 | 守军无心作战 |

### 3. 军饷危机的连锁反应

| 环节 | 表现 |
|------|------|
| 朝廷加税 | 三饷加派，百姓负担沉重 |
| 民变加剧 | 加税导致更多农民起义 |
| 镇压需更多军费 | 又加征剿饷 |
| 军队欠饷 | 兵变、投敌频发 |
| 财政崩溃 | 恶性循环，最终无力维持 |

### 4. 崇祯最后的财政努力

| 措施 | 效果 |
|------|------|
| 号召大臣捐饷 | 大臣多推脱，收效甚微 |
| 动用内帑 | 国库已空虚 |
| 加征三饷 | 激起更大民变 |
| 裁撤驿站 | 节省开支，但裁出的驿卒李自成成为农民军领袖 |

## 七、军饷发放标准的演变

### 1. 卫所兵月粮

| 时期 | 标准 | 备注 |
|------|------|------|
| 洪武制 | 有家小者月支1石，无妻小者月支6斗 | 以实物（粮食）为主 |
| 明中叶后 | 折色比例大增，实际收入缩水 | 折银后常被克扣 |

### 2. 募兵年饷变化

| 时期 | 年饷标准 | 备注 |
|------|----------|------|
| 英宗时期 | 银一两，布二匹 | 最初募兵极少 |
| 弘治年间 | 银五两/人 | 募兵规模扩大 |
| 嘉靖初年 | 约9-10两/年 | 日给工食银四分 |
| 嘉靖末年 | 约16-18两/年 | 戚家军年饷约10两，加赏功银更多 |
| 隆庆-万历初 | 约14-20两/年 | 月饷1.2-1.5两，另发安家银、行粮 |
| 万历中后期 | 约20-25两/年 | 辽东前线饷银较高 |
| 天启年间 | 约24-36两/年 | 辽军月饷约2两，入辽客兵近3两/月 |
| 崇祯年间 | 名义30余两/年 | 实际常欠饷，发放率仅三成 |

[来源：[募兵制军饷 - 腾讯新闻](https://news.qq.com/rain/a/20230810A04KDL00)；曹循《明代军制演进与盛衰之变》]

### 3. 特殊激励

| 项目 | 金额 | 说明 |
|------|------|------|
| 安家银 | 3两/人 | 入伍时一次性发放 |
| 行粮银 | 2两/人 | 行军途中补贴 |
| 首功赏银 | 约30两/首级 | 戚继光制度 |
| 衣装银 | 约1-2两 | 置办军装 |
