# 火器

## 一、概述

明代是中国古代火器发展的高峰期。从洪武朝大规模使用火铳，到永乐朝建立世界最早的独立建制火器部队（神机营），再到嘉靖朝引进西方火器技术，明代火器经历了从自主发展到中西融合的历程。

## 二、单兵火器

### 1. 鸟铳（鸟嘴铳）

| 项目 | 数据 |
|------|------|
| 引入时间 | 嘉靖二十七年（1548年），由葡萄牙人传入，又说嘉靖年间从缴获倭寇武器中仿制 |
| 原型 | 西方火绳枪（arquebus/matchlock musket） |
| 击发方式 | 火绳击发——龙头夹持燃烧的火绳，扣扳机后火绳落入火门点燃引药 |
| 铳管长度 | 约3-4尺（约96-128厘米），《武备志》载"铳筒长三尺有余" |
| 全长 | 约5-6尺（约160-192厘米，含木托） |
| 口径 | 约1-1.5厘米（装铅子直径约1厘米） |
| 重量 | 约5-8斤（约3-5公斤） |
| 射程 | 有效射程约80-100步（约120-150米），最大射程约200步（300米） |
| 装药量 | 约3钱（约11克）火药 |
| 弹丸 | 铅子1枚，重约3钱（约11克） |
| 射速 | 熟练射手每分钟约1-2发 |
| 制造数量 | 嘉靖至万历年间制造总量估计达数万至十余万支 |
| 装备部队 | 戚家军、蓟镇军、辽东军、神机营等 |
| 制造成本 | 约1-3两银（不同时期和质量差异大） |
| 名将使用 | 戚继光在蓟镇练兵时大量装备鸟铳，其步兵营约40%配鸟铳 |

**戚继光对鸟铳的评价**（《练兵实纪》）：
- "鸟铳……利能洞甲，射能命中，弓矢弗如也"
- 认为鸟铳在精度和穿甲能力上优于弓箭

**戚继光步兵营鸟铳配置**：
- 一营3,000人中配鸟铳约1,080-1,200支
- 每哨设鸟铳手若干，与冷兵器兵混合编组

### 2. 三眼铳

| 项目 | 数据 |
|------|------|
| 出现时间 | 明中期（约成化至嘉靖年间） |
| 结构 | 三根铳管并列固定于一个木柄上 |
| 全长 | 约4-5尺（约130-160厘米） |
| 铳管长 | 约1-1.5尺（约32-48厘米） |
| 口径 | 约2-3厘米 |
| 重量 | 约5-8斤（含木柄） |
| 射程 | 有效射程约30-50步（约45-75米），最大射程约80步 |
| 装药量 | 每管约2-3钱火药 |
| 弹丸 | 每管装铅子或铁子1-3枚 |
| 射速 | 三管可依次射击，射完后可当铁锤使用 |
| 特点 | 精度低、射程近，但火力密度高、可作钝器使用 |
| 装备部队 | 北方骑兵广泛使用，明末京营、边军普遍装备 |
| 数量 | 制造量极大，是明代装备最广泛的火器之一 |

> 三眼铳虽然性能不如鸟铳，但因结构简单、造价低廉、可兼作冷兵器，在北方骑兵中极为流行。

### 1. 佛郎机炮

#### 引入

| 项目 | 内容 |
|------|------|
| 原型 | 葡萄牙/西班牙的后装式短管火炮（Breech-loading swivel gun） |
| 引入时间 | 正德年间末至嘉靖初年（约1521-1522年） |
| 引入途径 | 正德十六年（1521年）屯门海战中缴获葡萄牙船上的佛郎机炮 |
| 仿制 | 嘉靖元年至二年（1522-1523年）开始仿制 |

#### 各型号规格

| 型号 | 重量 | 炮管长 | 口径（约） | 弹重 | 射程 | 备注 |
|------|------|--------|-----------|------|------|------|
| 大佛郎机（一号） | 约300-500斤（180-300公斤） | 约5-6尺 | 约8-10厘米 | 约5-10斤 | 约500步（750米）以上 | 城防/舰载 |
| 二号佛郎机 | 约150-200斤 | 约4-5尺 | 约6-8厘米 | 约3-5斤 | 约300-400步 | 野战/城防 |
| 三号佛郎机 | 约70-100斤 | 约3-4尺 | 约5-6厘米 | 约1-3斤 | 约200-300步 | 野战 |
| 四号佛郎机 | 约30-50斤 | 约2-3尺 | 约3-4厘米 | 散弹 | 约150-200步 | 骑兵/船用 |
| 五号佛郎机（小佛郎机） | 约10-20斤 | 约1-2尺 | 约2-3厘米 | 散弹 | 约100步 | 轻便型 |

**佛郎机的关键特征**：
- **后装式**：使用预装火药和弹丸的"子铳"（弹药筒），从炮尾装入
- 每门佛郎机配4-9个子铳，可快速更换
- 射速远高于前装炮：每分钟可发射4-6发
- 缺点：后膛密封性差，气密不好，射程和威力不如前装炮

**制造数量**：
- 嘉靖年间大量制造
- 从嘉靖至万历年间，明军总计装备了约**四至五万门**佛郎机（含各种型号）
- 戚继光蓟镇车营：每辆偏厢车配佛郎机2门，一营128辆车共256门
- 明末各边镇普遍装备

**无敌大将军炮（重型佛郎机）**：
- 全重约1,050斤（母铳1,000斤，子铳约50斤）
- 配3个子炮，实操测算前三炮总共花费不到20秒
[来源：[佛郎机炮 - 维基百科](https://zh.wikipedia.org/zh-hans/%E4%BD%9B%E9%83%8E%E6%A9%9F%E7%A0%B2)]

## 七、火器在各时期的战场应用

### 1. 明初（洪武-永乐）

- 以手铳、碗口铳、大将军炮为主
- 神机营是核心火器部队
- 永乐北征中，神机营以火铳齐射阻遏蒙古骑兵冲击

### 2. 明中期（嘉靖-万历）

| 战争 | 火器使用 |
|------|----------|
| 抗倭战争 | 鸟铳成为主力，戚继光大量装备；虎蹲炮用于近战 |
| 庚戌之变后北边防御 | 佛郎机广泛部署于长城沿线 |
| 万历援朝（1592-1598年） | 佛郎机、鸟铳、大将军炮在平壤之战等战役中发挥关键作用 |
| 万历播州之役（1600年） | 火器攻坚 |

### 3. 明末（天启-崇祯）

| 战争 | 火器使用 |
|------|----------|
| 宁远之战（1626年） | 红夷大炮击退后金军，据传努尔哈赤受伤 |
| 宁锦之战（1627年） | 红夷大炮再次发挥城防优势 |
| 松锦之战（1641-1642年） | 明军依赖火炮防御 |
| 孔有德叛变（1633年） | 大量红夷大炮及操炮手被带至后金 |

## 九、明代火器的局限

| 局限 | 具体表现 |
|------|----------|
| 制造质量不稳定 | 偷工减料、贪墨严重，火器炸膛事故频发 |
| 训练不足 | 许多火器兵不熟悉操作，实际射击命中率低 |
| 后勤困难 | 火药供应不稳定，前线常缺弹药 |
| 技术停滞 | 嘉靖后自主创新减少，依赖引进西方技术 |
| 制度障碍 | 军器局和兵仗局的官僚化管理效率低下 |
| 天气限制 | 火绳枪和火炮在雨天无法使用 |
