# 明代海防体系

## 概述

明代海防体系是中国历史上第一个系统性的沿海军事防御网络。明太祖朱元璋建国之初即面临倭寇（日本海盗及流亡海上的张士诚、方国珍残部）威胁，洪武年间开始大规模建设沿海卫所和海防设施。海防重心区域为东南沿海——**山东、南直隶（江苏）、浙江、福建、广东**五省。嘉靖年间（1521-1566）倭寇大规模入侵达到顶峰；隆庆元年（1567）部分开放海禁后，倭患逐渐平息。

明代海防体系包括：**沿海卫所城堡、水寨（水军基地）、水师舰队、沿海烽堠、巡洋会哨制度、海禁政策**六大要素。

## 一、沿海卫所分布

### 总体数据

明代沿海卫所总计约**54卫、99所**（统计口径有差异）。每卫额定5,600人，每所额定1,120人。洪武年间理论总兵力约40-50万人。

汤和奉命在沿海六十里设一军卫，三十里设一守御千户所。洪武年间在江南、北，浙东、西等处筑59座城，得兵58,750余人。

## 三、巡海制度

### 巡洋会哨制度

明代巡洋会哨制度是沿海防御的核心运作机制，真正形成于**正统时代**（1436-1449）。

| 制度要素 | 具体内容 |
|---------|---------|
| 巡洋 | 水寨战船分班轮流出海巡逻，每班巡逻期约1-3个月 |
| 会哨 | 相邻水寨的巡逻船在指定海域交界处"会哨"（会合点验），防止巡逻空白区 |
| 连界会哨 | 给沿海各都司卫所划分明确巡逻海域，在交汇处建立连界会哨制度以监察履职 |
| 巡逻季节 | 春夏（3-9月）为旺季（倭寇多利用季风来犯），秋冬减少 |
| 陆上巡逻 | 沿海卫所兵丁沿海岸线巡逻，与水上巡逻配合 |

### 制度演变

| 时期 | 变化 |
|------|------|
| 洪武年间 | "诏滨海卫所每百户置船二艘，巡运海上盗贼，巡检司亦如之" |
| 英宗时期 | 巡海制度局限性凸显，开始调整：明确各卫所巡逻海域、建立连界会哨 |
| 嘉靖年间 | 沿海防务打破卫所防御区划，形成新的防御区域。福建分三路，各置参将，恢复五水寨，设总兵统领 |

## 五、海禁政策的建立与演变

### 海禁体制（洪武-隆庆，1368-1567）

| 时间 | 政策内容 |
|------|---------|
| 洪武四年（1371） | 朱元璋下令"寸板不许下海"，禁止民间私人出海贸易 |
| 洪武十四年（1381） | 设沿海卫所，建立系统海防 |
| 洪武二十三年（1390） | 严申海禁，"敢有私下诸番互市者，必置之重法" |
| 洪武二十七年（1394） | 禁止民间使用"双桅以上船只"，限制船只尺寸 |
| 永乐年间 | 一方面厉行海禁，一方面官方组织郑和下西洋（1405-1433），官方垄断海外贸易 |
| 宣德以后 | 郑和下西洋停止，海禁继续执行但力度波动 |
| 嘉靖年间 | 海禁最严厉时期，"片板不许入海"。但正是海禁导致沿海民众生计断绝，大量"通番"走私，最终酿成嘉靖大倭寇 |

### 海禁的矛盾

- **正面逻辑**：切断倭寇/海盗的陆上补给和贸易网络
- **实际后果**：沿海渔民、商人丧失生计，反而投靠海盗集团。嘉靖大倭寇的头目王直（汪直）、徐海、林凤等大多为中国人，真正的日本倭寇只占少数
- **核心矛盾**：朝廷将海上贸易视为安全威胁而非经济利益，海禁政策与东南沿海的商业经济严重冲突

> "市通则寇转而为商，市禁则商转而为寇。"

## 六、嘉靖抗倭时期的海防调整

### 嘉靖大倭寇（1547-1567）

| 阶段 | 时间 | 主要事件 |
|------|------|---------|
| 酝酿期 | 1520s-1547 | 走私贸易增加，双屿（舟山群岛）成为国际走私港 |
| 爆发期 | 1548 | 朱纨捣毁双屿港，走私商人转为海盗 |
| 高潮期 | 1552-1556 | 王直集团大规模入侵，倭寇深入江浙腹地 |
| 转折期 | 1555-1562 | 胡宗宪/戚继光/俞大猷先后主持平倭 |
| 尾声期 | 1563-1567 | 福建/广东残余倭寇肃清 |

### 海防调整措施

| 调整方向 | 具体措施 |
|---------|---------|
| 设立统一指挥 | 嘉靖三十二年（1553）设浙直总督（总督浙直福建军务），代表人物：张经、胡宗宪 |
| 恢复水寨 | 福建恢复五水寨旧制，以把总领之 |
| 增设兵备道 | 各省增设海道副使/兵备道，分段管理海防 |
| 分路防御 | 福建沿海分成三路，各置参将；另设总兵统领全省三路五寨 |
| 改建城防 | 嘉靖四十五年（1566）在月港设立海澄县，加强海防管理 |
| 军事创新 | 戚继光创建鸳鸯阵、训练戚家军，大规模使用火器 |

### 倭寇成因的认识转变

嘉靖倭乱发生后，朝野曾发生禁海问题争论。有识之士看到"海禁"与海寇之间的关系：

> "市通则寇转而为商，市禁则商转而为寇。"

这一认识最终推动了隆庆开海。

## 七、隆庆开关后海防的变化

### 隆庆开海（1567）

| 项目 | 内容 |
|------|------|
| 时间 | 隆庆元年（1567） |
| 推动者 | 福建巡抚涂泽民奏请 |
| 开放范围 | 仅允许"东洋"和"西洋"两条航线，通过福建漳州月港出海 |
| 禁止方向 | 仍禁止赴日本贸易（因倭寇记忆），但走私赴日从未中断 |
| 管理方式 | 每年申请制，需领取"船引"并交纳引税 |
| 年征税银 | 月港年征税银约2-3万两（万历年间） |
| 效果 | 倭患迅速平息，沿海经济繁荣。流入中国白银估计约3-4亿两 |

### 海防政策转变

| 方面 | 变化 |
|------|------|
| 从军事防御到经济疏导 | 通过合法化民间贸易解决倭寇问题 |
| 卫所体制松动 | 沿海卫所不再承担严密封锁任务，功能转向治安维护 |
| 新威胁出现 | 荷兰人（红毛番）开始在东南海域活动 |

### 万历至崇祯海防

| 时期 | 主要态势 |
|------|---------|
| 万历初 | 倭患平息，海防以防备海盗为主 |
| 万历二十年代 | 丰臣秀吉侵朝（1592-1598），明朝水师参加抗倭援朝（露梁海战等） |
| 万历三十年代 | 荷兰人占据澎湖（1604、1622两次） |
| 天启年间 | 荷兰占台湾南部（1624），西班牙占台湾北部（1626） |
| 崇祯年间 | 郑芝龙崛起控制东南海域；1633年料罗湾海战击败荷兰舰队 |

## 八、沿海烽堠体系

### 建设背景

明代沿海烽堠（烽火台）体系与北方长城烽燧并行，构成海上预警网络。元末以来倭寇屡犯沿海，明初即在海岸高地建立烽火台，嘉靖倭患高峰期烽堠设置尤为密集。

### 布局原则

| 项目 | 标准 |
|------|------|
| 选址 | 高山险绝处优先，无山则置平地高处 |
| 间距 | 每座相距约30-50里（15-25公里），确保视线可及 |
| 构造 | 方形石砌或圆形土墩，高数丈 |
| 驻守 | 每台数人至十余人 |

### 沿海信号规则

沿海烽堠的信号体系侧重反映海上敌情的阶段性变化：

| 敌情阶段 | 信号 |
|----------|------|
| 望见海面出现盗船 | 放1炮，举烽1把 |
| 海盗船逼近海岸 | 放2炮，举烽2把 |
| 海盗登陆 | 放3炮，举烽3把 |

信号方式：白天燃烟，夜间举火，全天候辅以放炮声响传递。

### 烽堠网络运作流程

```
前沿烽堠（海岛/岬角）
  │ 发现敌船 → 举烽放炮
  ↓
沿岸烽堠（海岸高地）
  │ 逐站传递 → 信号沿海岸线扩散
  ↓
水寨/巡检司
  │ 判断敌情规模 → 派船出海拦截
  ↓
卫所城
  │ 调集陆上兵力 → 准备岸防
  ↓
府城/省城
  │ 上报巡抚 → 协调各路增援
```

### 实例：浙江台州沿海烽堠

嘉靖时期台州沿海烽堠布局尤密，烽堠耸立于群峰之巅，一路烟楼相望、烽墩相连，曲折跌宕数百里。形成以卫所为点、巡检司和寨汛烽堠为线的连续海防预警网络。敌情可在短时间内传至卫所指挥部，各处迅速进入临战状态。

> **来源**：《明代台州烽堠与抗倭考述》台州地方志；《烽火台》百度百科

## 十、海防战略关键人物

### 朱纨（?-1549）

| 项目 | 内容 |
|------|------|
| 官职 | 都御史，巡抚浙江兼管福建海道 |
| 任期 | 嘉靖二十六年至二十八年（1547-1549） |
| 主张 | 海陆一体防御：备战船+建保甲+打击通倭势族 |
| 军功 | 双屿之役、诏安之役，捣毁倭寇巢穴 |
| 结局 | 触动通倭权贵利益，遭弹劾，被撤职后自杀 |
| 影响 | 朱纨之死导致海防整饬半途而废，倭患随后急剧恶化 |
