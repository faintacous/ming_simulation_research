# 武官体系

## 一、中央军事机构：五军都督府

### 1. 设置沿革

| 时间 | 变化 |
|------|------|
| 洪武十三年（1380年） | 废大都督府，分设中、左、右、前、后五军都督府 |
| 此后至明末 | 五军都督府体制未变，但权力逐渐被兵部侵夺 |

### 2. 品级与职官

| 官职 | 品级 | 员数（每府） | 说明 |
|------|------|-------------|------|
| 左都督 | 正一品 | 1员 | 各府最高长官 |
| 右都督 | 正一品 | 1员 | |
| 都督同知 | 从一品 | 无定员 | |
| 都督佥事 | 正二品 | 无定员 | |
| 经历 | 从五品 | 1员 | 属官 |
| 都事 | 从七品 | 1员 | 属官 |

### 3. 职能

| 职能 | 内容 |
|------|------|
| 统兵权 | 掌管各卫所的军籍、军政、屯田等事务 |
| 分管辖区 | 五府分管全国各都司卫所（详见卫所制度篇） |
| 战时统帅 | 朝廷命将出征时，通常由都督府勋臣担任总兵官 |
| 制衡关系 | 有统兵权无调兵权——调兵权归兵部 |

### 4. 五军都督府与兵部的权力制衡

| 机构 | 权力 | 限制 |
|------|------|------|
| 五军都督府 | 统兵权（掌管军籍、军政、屯田、操练） | 无调兵权 |
| 兵部 | 调兵权（调遣、铨选武官、军政考核） | 无统兵权 |

> 五府"有统兵之权而无调兵之权"，兵部"有调兵之权而无统兵之权"，相互牵制。

[来源：[五军都督府 - 维基百科](https://zh.wikipedia.org/zh-hans/%E4%BA%94%E8%BB%8D%E9%83%BD%E7%9D%A3%E5%BA%9C)；[什么是五军都督府 - 知乎](https://zhuanlan.zhihu.com/p/438203321)]

### 5. 权力变迁

| 时期 | 都督府地位 |
|------|-----------|
| 洪武-永乐 | 权重，勋臣掌实权 |
| 宣德-正统 | 逐渐削弱，兵部权力上升 |
| 成化以后 | 都督府沦为虚衔，实权归兵部 |
| 嘉靖以后 | 都督衔成为武将的荣衔/加衔，不再掌管实际军务 |
| 万历-崇祯 | 完全成为荣衔机构 |

> 明代中后期的实际格局：**兵部掌军令（调兵、军政、铨选武官），五军都督府仅存名义**。

## 二、地方军事机构：都指挥使司

### 1. 全国设置

| 类别 | 数量 | 分布 |
|------|------|------|
| 都指挥使司 | 13个 | 对应十三布政使司 |
| 行都指挥使司 | 5个 | 边防要地 |
| 留守司 | 2个 | 凤阳（中都）、钟祥（兴都） |

### 2. 都指挥使司职官

| 官职 | 品级 | 员数 | 职责 |
|------|------|------|------|
| 都指挥使 | 正二品 | 1员 | 一省军事最高长官 |
| 都指挥同知 | 从二品 | 2员 | 佐官 |
| 都指挥佥事 | 正三品 | 4员 | 佐官 |
| 经历 | 正六品 | 1员 | 文案 |
| 都事 | 正七品 | 1员 | 文案 |
| 断事 | 正六品 | 1员 | 司法 |
| 司狱 | 从九品 | 1员 | 刑狱 |

### 3. 都司的实际权力变化

- 明初都指挥使与布政使、按察使并列为一省三司长官
- 明中叶后，巡抚制度确立，都指挥使受巡抚节制
- 镇戍营兵制兴起后，都指挥使只管辖卫所事务，边防实权归总兵等营兵将领

## 三、镇戍营兵制武官体系

### 1. 总兵官（总兵）

| 项目 | 内容 |
|------|------|
| 品级 | 无固定品级（明初为临时差遣，以公侯伯充任；后渐固定化，通常挂都督或都督同知衔，正一品至正二品） |
| 正式定制 | 万历以后逐渐以正二品为常 |
| 职责 | 一镇最高军事长官，统辖全镇兵马 |
| 管辖兵力 | 一镇数万至十余万不等 |
| 设置 | 全国主要总兵官约20余人（嘉靖以后），九边各设1人 |
| 任命 | 由兵部推举，皇帝任命，给"将印"（后改为"关防"） |
| 节制关系 | 受巡抚、总督节制 |

### 2. 副总兵（副将）

| 项目 | 内容 |
|------|------|
| 品级 | 从二品（通常挂都督佥事或都指挥使衔） |
| 职责 | 协助总兵，分守一路或一协 |
| 管辖兵力 | 数千至万余人 |
| 设置 | 每镇1-3人不等 |
| 任命 | 兵部题准，皇帝任命 |

### 3. 参将

| 项目 | 内容 |
|------|------|
| 品级 | 正三品（通常挂都指挥同知或都指挥佥事衔） |
| 职责 | 分守一路，驻守重要关口或城堡 |
| 管辖兵力 | 2,000-5,000人 |
| 设置 | 每镇3-10余人不等 |
| 别称 | 分守参将、守备参将 |

### 4. 游击（游击将军）

| 项目 | 内容 |
|------|------|
| 品级 | 从三品（通常挂都指挥佥事或指挥使衔） |
| 职责 | 率游兵机动作战，不固定驻防地点 |
| 管辖兵力 | 1,000-3,000人 |
| 设置 | 每镇数人至十余人 |
| 特点 | 机动灵活，负责巡逻、追剿、增援 |

### 5. 守备

| 项目 | 内容 |
|------|------|
| 品级 | 正四品至正五品（挂指挥使或指挥同知衔） |
| 职责 | 驻守一城一堡 |
| 管辖兵力 | 500-2,000人 |
| 设置 | 数量最多，每镇数十人 |

### 6. 把总

| 项目 | 内容 |
|------|------|
| 品级 | 正六品至正七品（挂指挥佥事或千户衔） |
| 职责 | 统率一营或一部兵力 |
| 管辖兵力 | 100-500人 |
| 设置 | 数量众多 |

### 7. 千总（坐营中军官）

| 项目 | 内容 |
|------|------|
| 品级 | 正五品至正六品 |
| 职责 | 统率千人左右，或为参将、游击的佐官 |
| 管辖兵力 | 500-1,000人 |

| 层级 | 官职 | 品级 | 管辖兵力 | 相当于 |

| 3 | 参将 | 正三品 | 一路（2,000-5,000） | 旅长 |
| 4 | 游击将军 | 从三品 | 游兵（1,000-3,000） | 团长（机动） |
| 5 | 守备 | 正四至五品 | 一城堡（500-2,000） | 营长 |
| 6 | 千总 | 正五至六品 | 500-1,000 | 营/连级 |
| 7 | 把总 | 正六至七品 | 100-500 | 连/排级 |
| 8 | 百总 | 正七品 | 约100人 | 排级 |

## 四、文臣统军制度

### 1. 总督

| 项目 | 内容 |
|------|------|
| 品级 | 正二品（加兵部尚书衔则为正一品） |
| 职责 | 总督军务，节制辖区内各镇总兵 |
| 管辖范围 | 通常跨二至三省/镇 |
| 设置 | 嘉靖以后逐步固定化 |

**主要总督**：

| 总督名称 | 驻地 | 管辖 | 设置时间 |
|----------|------|------|----------|
| 蓟辽总督 | 密云/蓟州 | 蓟镇、辽东、保定 | 嘉靖年间 |
| 宣大总督 | 阳和/大同 | 宣府、大同、山西 | 嘉靖年间 |
| 三边总督 | 固原 | 延绥、宁夏、甘肃 | 弘治年间 |
| 两广总督 | 肇庆 | 广东、广西 | 成化年间 |
| 川贵总督 | — | 四川、贵州 | 不常设 |
| 七省总督 | — | 万历援朝设 | 临时差遣 |

### 2. 巡抚

| 项目 | 内容 |
|------|------|
| 品级 | 正三品至从二品（加都御史衔） |
| 职责 | 一省/一镇军政最高长官，兼管民政 |
| 军事权 | 节制辖区内参将以下武官；对总兵有监督权 |
| 设置 | 宣德年间开始派出，嘉靖以后各省固定 |

### 3. 文武关系

| 时期 | 关系 |
|------|------|
| 明初-正统 | 武将地位较高，勋臣掌军 |
| 景泰以后 | 文臣督军成为定制（于谦首创总督京营戎政） |
| 嘉靖以后 | "以文制武"完全确立，总督、巡抚节制总兵 |
| 万历以后 | 武将地位进一步下降，总兵见巡抚需行跪拜礼 |

## 五、武官选任方式

### 1. 世袭

| 项目 | 内容 |
|------|------|
| 适用范围 | 卫所武官（指挥使至百户） |
| 规则 | 嫡长子袭职，无嫡长则庶长，无子则弟或侄 |
| 袭职程序 | 报兵部核准，比试武艺后袭职 |
| 降袭 | 若比试不合格，降一级袭职（如指挥使之子降为指挥同知） |
| 减袭 | 明中叶后为控制冗官，部分职位递减袭职 |
| 存在时间 | 贯穿整个明代，但实际意义在中后期大为降低 |

### 2. 武举

| 项目 | 内容 |
|------|------|
| 创设 | 天顺八年（1464年）颁布第一部《武举法》，弘治六年（1493年）定制 |
| 层级 | 武生→武乡试（武举人）→武会试（武进士）→武殿试 |
| 考试频率 | 每三年一科，与文科同年 |
| 考试内容 | "先之以谋略，次之以武艺"——先试策论，不合格者取消武试资格 |
| 录取名额 | 武乡试各省不等，武会试约100-200名/科 |
| 授官 | 中式者授官、月俸禄三石米，前往军营报道 |
| 参加范围 | 具有军事才能的所有官民均可参加 |
| 与世袭的关系 | 嘉靖中叶前，会试武举中只有副千户以上的武官才能被推用为将领，因此武举实为世袭制的补充 |

**武举考试具体内容**：

| 科目 | 内容 | 标准 |
|------|------|------|
| 马射 | 骑马奔射三箭 | 射中靶心为上 |
| 步射 | 步行射箭若干支 | 射中数和精度评判 |
| 开弓 | 拉硬弓 | 嘉靖时标准为三力（约120斤力）以上 |
| 舞刀 | 舞大刀 | 标准刀重80斤、100斤、120斤三等 |
| 掇石 | 举石锁 | 标准石重200斤、250斤、300斤三等 |
| 策论 | 笔试军事策论 | 嘉靖以后增加，此前纯考武艺 |

### 3. 战功

| 项目 | 内容 |
|------|------|
| 方式 | 以实际战功升迁 |
| 标准 | 斩首级数（首功制）为主要考核标准 |
| 具体标准 | 斩敌首一级，升官一级（但各时期、各战场标准不同） |
| 问题 | 首功制导致杀良冒功——杀普通百姓冒充敌兵首级邀功 |
| 典型案例 | 戚继光、李成梁、俞大猷等皆以战功晋升 |
| 弊端 | 买功、冒功现象严重 |

### 4. 荐举

| 项目 | 内容 |
|------|------|
| 方式 | 由总督、巡抚或兵部推荐 |
| 适用 | 中高级武官的选拔 |
| 程序 | 督抚保举→兵部核准→皇帝任命 |
| 问题 | 门户之见、行贿买缺 |

### 5. 行伍出身

| 项目 | 内容 |
|------|------|
| 方式 | 从士兵、基层军官逐级升迁 |
| 途径 | 军功 + 年资 + 上官提拔 |
| 实例 | 明末许多将领出身行伍 |

### 营兵武官俸禄

营兵制武官的俸禄相对灵活，除基本俸禄外还有：
- **廉俸/养廉银**：部分地区另给
- **柴薪银、马价银**：额外补贴
- **家丁饷银**：将领可用部分饷银豢养家丁

## 七、明末武官体系的问题

### 1. 文武失衡

| 现象 | 表现 |
|------|------|
| 武将地位极低 | 总兵（正二品）见巡抚（正三品加衔）需行跪拜礼 |
| 文官掣肘 | 武将作战需报巡抚、总督批准，贻误战机 |
| 监军太监 | 宦官监军干预军务，武将受多重束缚 |
| 功过不公 | 战胜归功督抚，战败武将受罚 |

### 2. 将领私兵化

| 现象 | 表现 |
|------|------|
| 家丁制度膨胀 | 将领核心战力依赖私养家丁 |
| 兵为将有 | 士兵认将不认朝廷 |
| 拥兵自重 | 崇祯年间左良玉、吴三桂等将领号令不行 |

### 3. 武官来源的变化

| 时期 | 主要来源 | 特点 |
|------|----------|------|
| 洪武-永乐 | 开国功臣子弟、世袭武官 | 勋贵掌军 |
| 正统-弘治 | 世袭 + 战功 | 世袭武官能力退化 |
| 嘉靖-万历 | 战功 + 武举 + 荐举 | 实战型将领增多 |
| 天启-崇祯 | 行伍 + 荐举 + 战功 | 底层出身将领增多（如毛文龙等） |
