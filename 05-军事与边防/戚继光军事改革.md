# 戚继光军事改革

## 概述

戚继光（1528-1588），字元敬，号南塘，山东蓬莱人（登州卫世袭军户出身）。他是明代最杰出的军事家、军事改革家，一生主要分为两个阶段：

1. **浙闽抗倭**（嘉靖三十四年至嘉靖四十五年，1555-1566）：在浙江、福建抗击倭寇，创建"戚家军"，发明鸳鸯阵，九战九捷，著《纪效新书》
2. **镇守蓟州**（隆庆二年至万历十一年，1568-1583）：任蓟镇总兵16年，训练车营步营骑营，修建空心敌台，著《练兵实纪》，蓟镇16年间无大规模敌寇入犯

戚继光的军事改革涵盖**阵法、编制、兵器、练兵、筑城**五大方面，其核心思想是：以严格训练的职业化军队取代腐朽的卫所世袭兵，以系统化的战术体系取代个人武勇。

## 一、鸳鸯阵

### 概述

鸳鸯阵是戚继光在浙江抗倭时期（1559-1561年间）创制的步兵战术阵型。针对倭寇善用日本刀（太刀/打刀）近身搏斗、个人武艺精湛的特点，戚继光设计了一种以**长短兵器配合、远近火力搭配**为核心的小队战术阵型。

### 12人编组详解

鸳鸯阵的基本单位为**一队12人**，由队长1人统率，配置如下：

```
阵型示意（面向敌方）：

                    ← 敌方 →

        [盾牌手A]          [盾牌手B]
    [狼筅手C]                  [狼筅手D]
  [长枪手E]  [镋钯手F]    [长枪手G]  [镋钯手H]
    [短兵手I]                  [短兵手J]
              [火兵K]
              [队长L]
```

#### 各位置详细配置

| 位次 | 名称 | 人数 | 武器装备 | 职责 |
|------|------|------|---------|------|
| A/B | **藤牌手**（盾牌手） | 2人 | 藤牌（圆形藤编盾牌，直径约2尺/65cm）+ 腰刀（戚家刀）+ 标枪（每人2支，用于投掷） | 位于最前排，用藤牌掩护全队推进。遇敌先投标枪，再以藤牌挡住敌人刀锋，掩护后方狼筅手。藤牌质轻（约5-6斤），可挡刀箭 |
| C/D | **狼筅手** | 2人 | 狼筅（用大毛竹制成，留枝杈并绑上铁尖，全长约一丈三尺/约4.3米，铁制者杆长5米，附枝最长60cm最短25cm，重约7-8斤）+ 腰刀 | 位于藤牌手之后。狼筅枝杈繁多，横扫可挡住敌人前进路线，铁尖可刺伤敌人。主要功能是用密集的竹枝"罩住"敌人（尤其克制倭寇的日本刀劈砍，刀砍在竹枝上会被缠住） |
| E/G | **长枪手** | 2人（一说4人） | 长枪（枪杆长约一丈二尺/约4米，一说长达6米，铁枪头）+ 腰刀 | 位于狼筅手之后（左右各一或各二）。趁狼筅手制住敌人时，从两侧以长枪刺杀。长枪是鸳鸯阵中主要的杀伤武器 |
| F/H | **镋钯手**（又称"叉手"） | 2人 | 镋钯（山字形铁兵器，原长七八尺，戚继光加长至一丈二尺/约4米；顶端凹下处可放置火箭）+ 腰刀 | 与长枪手搭配，位于枪手内侧。镋钯可格挡敌人兵器、叉住敌人身体，也可杀伤敌人。负责保护枪手侧翼及全队后方 |
| I/J | **短兵手**（短刀手） | 2人 | 戚家刀（仿日本刀型制的中国改良刀，刀身微弯，长约3尺5寸/约1.1米）+ 腰刀（备用） | 位于阵型后部，当敌人突破前排防线或混战时，短兵手以刀近身格杀。也负责保护全队后方 |
| K | **火兵** | 1人 | 携带火具（火石火绒）、干粮、炊具、腰刀 | 负责全队的后勤——做饭、烧水、搬运辎重。战斗时在队尾协助警戒 |
| L | **队长** | 1人 | 旗帜（认旗，用于指挥）+ 哨子/锣鼓 + 腰刀 | 在队中或队尾指挥全队进退。以旗帜和口令指挥阵型变化 |

> **编组说明**：关于12人的具体分配，《纪效新书》不同版本（14卷本与18卷本）记载略有差异。上表为综合主流文献的常见配置。部分版本中长枪手为4人、镋钯手为2人，也有版本中长枪手2人、镋钯手2人、短兵手2人。核心原则不变：盾牌掩护→狼筅罩压→长兵刺杀→短兵补杀。

### 鸳鸯阵的变化阵型

鸳鸯阵的精妙之处在于可根据地形和战况灵活变阵：

#### 1. 标准鸳鸯阵（两路纵队）

- 队形：如上图所示，左右各6人，两两对称
- 适用：开阔地带正面推进
- 特点：攻守兼备，正面宽约3-4米

#### 2. 两仪阵（一分为二）

鸳鸯阵一分为二，成两个6人横队，左右分开包抄。适用于窄路、巷战、追击，或需要夹击时。

#### 3. 三才阵（一分为三）

将12人分为前、中、后三组，每组4人：
- **前组（天）**：2盾牌手 + 2狼筅手——掩护
- **中组（地）**：长枪手/镋钯手——主攻
- **后组（人）**：短兵手 + 火兵 + 队长——殿后/支援

适用于狭窄地形（小巷、田埂、山路），阵型收窄为纵队推进。

#### 4. 一头两翼阵

```
              [盾A]
        [狼筅C] [狼筅D]
   [枪E][镋F]     [枪G][镋H]
     [短兵I]        [短兵J]
              [盾B]
           [火兵K][队长L]
```

- 特点：一面藤牌在前为"头"，两组枪钯手展开为"翼"
- 适用：需要宽正面推进或包抄时

#### 5. 大鸳鸯阵

多个鸳鸯阵联合组成大阵，适用于较大规模战斗。

#### 6. 小三才阵（最小战斗单元）

紧急情况下每3人为一组：盾牌手+狼筅手+长枪手，形成最小战斗单元。

### 战术配合原则

1. **远距离**：火铳/弓弩先行压制（在阵外配属火器手）
2. **中距离**：藤牌手投掷标枪（每人2支）逼敌惊慌躲闪
3. **近距离**：狼筅手以巨大枝杈扫压、缠绕敌方兵器并扫倒敌人
4. **刺杀**：长枪手趁敌倒地或被狼筅制住时刺杀
5. **补杀/护翼**：镋钯手护卫侧后，对漏网之敌补杀；短兵手近身格杀突入之敌

> **核心理念**：以集体协同的阵法对抗敌方个人武艺，"虽一人之勇无所施"。好的战术是让士兵在最大限度保护自己的情况下杀伤敌人。

## 二、车营

### 概述

车营是戚继光在镇守蓟州时期（1568-1583）创建的新型军事编制，是为应对蒙古骑兵的大规模冲击而设计。与南方抗倭的步兵阵型不同，北方防御需要对抗骑兵集群，车营以偏厢车为核心，结合步兵、骑兵、火器，形成"移动城墙"式的防御-进攻体系。

### 偏厢车

| 项目 | 数据 |
|------|------|
| 车型 | 偏厢车（单侧有木板防护的战车） |
| 车体规格 | 长约1丈3尺（约4.2米），宽约5尺（约1.6米），高约7尺（约2.3米） |
| 防护 | 一侧（面向敌方）设厚木板屏障，可挡箭矢；木板上开铳眼/炮眼 |
| 车轮 | 两轮，直径约3尺（约1米） |
| 牵引 | 每车配骡马2匹牵拉行进，战斗时卸下骡马 |
| 火器配置 | 每车配佛郎机炮2门（装在车上），另有火铳/鸟铳若干 |
| 随车步兵 | 每车配步兵约20人（车正1人，车副1人，舵手1人，其余为战斗兵） |

### 车营编制——五级体制

戚继光车营编制为**128辆偏厢车**组成的完整战斗单位（一说129辆含主将车），采用五级编制：

#### 编制层级

| 层级 | 名称 | 数量 | 内容 | 指挥官 |
|------|------|------|------|--------|
| **第一级** | 车 | 1辆 | 偏厢车1辆 + 随车官兵20人 | 车正（百总级） |
| **第二级** | 衡（横排） | 4辆 | 4辆偏厢车为一衡 | 管衡官（把总级） |
| **第三级** | 冲（纵列） | 8辆 | 2衡 = 8辆车为一冲 | 千总 |
| **第四级** | 营（分营） | 32辆 | 4冲 = 32辆车为一分营 | 把总/中军 |
| **第五级** | 全营 | 128辆 | 4分营 = 128辆车 | 车营主将（参将/副总兵） |

#### 车营全编制详表

| 兵种 | 人数 | 备注 |
|------|------|------|
| 偏厢车正兵（车上战斗人员） | 约2,560人 | 128车 x 20人/车 |
| 骑兵（营内骑兵队） | 约512人 | 配属骑兵用于追击 |
| 辎重兵/火兵 | 约400人 | 后勤、弹药运输 |
| 将官/旗鼓/传令 | 约100余人 | 指挥系统 |
| **车营总兵力** | **约3,000-4,000人** | 一完整车营 |

### 车营武器配置

#### 每辆偏厢车的武器

| 武器 | 数量 | 用途 |
|------|------|------|
| 佛郎机炮（子母铳） | 2门 | 车载火炮，射程约300-500步（约450-750米） |
| 鸟铳（火绳枪） | 4支 | 精确射击，射程约100步 |
| 火箭架 | 1架 | 一窝蜂/火箭筒，齐射 |
| 长枪 | 4支 | 近战防御 |
| 大刀 | 2把 | 近战 |
| 藤牌 | 2面 | 防护 |

#### 全营火器统计

| 火器类型 | 数量（128辆车合计） | 备注 |
|---------|-------------------|------|
| 佛郎机炮 | 256门 | 每车2门 |
| 鸟铳 | 512支 | 每车4支 |
| 火箭架/一窝蜂 | 128架 | 每车1架 |
| 其他火器（火铳、三眼铳等） | 约200余件 | 配属步兵/骑兵 |
| **火器合计** | **约1,100余件** | — |

#### 火器占比

| 指标 | 数据 |
|------|------|
| 车营中火器兵占比 | 约**50-60%**（火器操作人员占总兵力一半以上） |
| 每车火器与冷兵器比 | 火器7件（2炮+4铳+1火箭架）vs 冷兵器8件（4枪+2刀+2盾），约**47%:53%** |
| 与同时代欧洲比较 | 明代车营的火器占比与16世纪欧洲军队（约40-50%火器兵）大致相当 |

### 车营战术

#### 防御阵型——车城/车阵

```
         ← 敌骑兵冲击方向 →

    ┌─车─车─车─车─车─车─车─车─┐
    车                          车
    车     步兵方阵（中央）       车
    车     骑兵（预备队）         车
    车                          车
    └─车─车─车─车─车─车─车─车─┘
```

1. **结车为城**：128辆偏厢车首尾相连，围成方阵（或圆阵），木板防护面朝外
2. **火力输出**：敌骑兵冲近时，车上佛郎机、鸟铳、火箭齐射
3. **步兵守车**：步兵在车后以长枪拒敌，防止骑兵突入
4. **骑兵反击**：敌骑被火力杀伤后退时，打开车阵缺口，骑兵出击追杀

#### 进攻阵型——行军战斗转换

1. 行军时：128辆车排成纵列行军，每4辆一排
2. 遇敌时：迅速展开为方阵，展开时间要求在"一柱香"（约15-30分钟）内完成
3. 进攻时：车阵缓慢向前推进，步兵跟随，火器持续射击

步营是戚继光在蓟州时期编练的步兵主力单位，与车营配合作战。步营以鸳鸯阵为基础单元，但扩展为更大规模的编制。

## 四、骑营编制

### 概述

骑营是戚继光在蓟州编练的骑兵单位，主要用于追击和机动作战。蓟镇骑兵以辽东/宣大马匹为主，兵源包括北方募兵和"跳荡兵"（精锐骑手）。

### 骑营编制表

| 层级 | 名称 | 人数 | 编成 | 指挥官 |
|------|------|------|------|--------|
| 第一级 | 伍 | 5骑 | 基本单位 | 伍长 |
| 第二级 | 队 | 10骑 | 2伍为1队 | 队长 |
| 第三级 | 旗 | 50骑 | 5队为1旗 | 旗总 |
| 第四级 | 局 | 200骑 | 4旗为1局 | 百总 |
| 第五级 | 司 | 约600骑 | 3局为1司 | 千总 |
| 第六级 | 营 | 约2,700骑 | 3-4司 + 直属队 | 营将（参将/副总兵） |

### 骑营武器配置

| 武器 | 配备情况 | 备注 |
|------|---------|------|
| 弓箭 | 每骑必备 | 骑射为基本技能 |
| 马刀/腰刀 | 每骑必备 | 近战兵器 |
| 长枪/马枪 | 约30%骑兵配备 | 冲锋用 |
| 三眼铳 | 约30-40%骑兵配备 | 骑兵专用火器：三管火铳，轮流射击后可当锤使用 |
| 链甲/棉甲 | 视供应情况 | 骑兵优先配发甲胄 |

### 三眼铳（骑兵特色火器）

| 项目 | 数据 |
|------|------|
| 结构 | 三根铳管并列固定在一根木柄上 |
| 全长 | 约4尺（约1.3米） |
| 重量 | 约5-8斤（约3-5公斤） |
| 射程 | 有效射程约30-50步（约45-75米） |
| 使用方式 | 骑马冲锋时先射三发（一管一发，轮流点火），射完后翻转以铁铳管当锤击打敌人 |
| 优点 | 制造简单、操作方便、兼具火器和冷兵器功能 |
| 缺点 | 射程近、精度差、装填慢（实际上一次冲锋只射三发） |

## 五、三大营协同——车步骑配合

### 戚继光的"三大营"体制

在蓟镇，戚继光建立了车营、步营、骑营三种编制的协同作战体系：

| 营种 | 数量（蓟镇全镇约） | 每营兵力 | 主要任务 |
|------|-------------------|---------|---------|
| 车营 | 约3-4营 | 约3,000-4,000人 | 防御、火力输出 |
| 步营 | 约6-8营 | 约2,700-3,000人 | 近战、攻坚 |
| 骑营 | 约3-4营 | 约2,700人 | 机动、追击、侦察 |
| 辎重营 | 约2-3营 | 约1,000-2,000人 | 后勤补给 |

### 协同战术

1. **遇敌先结车城**：车营前出，128辆偏厢车结成方阵，以火力阻滞敌骑兵冲击
2. **步兵守车接战**：步营在车阵内外列阵，以鸳鸯阵/方阵抵御冲入车阵的敌兵
3. **骑兵出击追杀**：待敌骑受挫后退时，骑营从车阵缺口冲出，追击歼灭
4. **三营依次展开**：车营为"体"（防御核心），步营为"翼"（侧翼保护和近战），骑营为"角"（机动出击）

## 六、练兵法

### 选兵标准

戚继光在《纪效新书》中详述了严格的选兵标准，明确反对招收"油滑"之人：

#### 选兵"四要"

| 标准 | 内容 |
|------|------|
| **要朴实** | 选农村壮丁，不选城市游民。"堪为上等兵者，惟有乡野老实之人" |
| **要壮健** | 身材中等以上，臂力过人。"凡选兵，以力为先"——能挽弓1石以上者优先；能负重50斤行军一日者合格 |
| **要胆大** | 面目黧黑、眼神坚毅者优先。"白净面皮"者不选（认为怯懦） |
| **要手足灵便** | 反应敏捷、四肢协调。测试方法：快速跑跳、翻越障碍 |

#### 选兵"六不要"

| 不选类型 | 原因 |
|---------|------|
| 城市游民/市井无赖 | "油滑奸巧之徒"，纪律差、易逃跑 |
| 官府衙役/胥吏之家 | 习惯钻营，不服管束 |
| 面白细嫩者 | 认为胆小怯懦（与时代偏见有关） |
| 性格伶俐/巧言令色者 | "巧者太巧，不可用" |
| 年过四十者 | 体力下降 |
| 曾犯军法/逃兵 | 纪律隐患 |

#### 兵源地偏好

戚继光在浙江抗倭时，特别从**义乌县**招募矿工为兵。原因：
- 义乌矿工常因矿产争端发生械斗，天性剽悍
- 矿工体力好、吃苦耐劳
- 矿工有集体作业习惯，容易训练纪律
- 著名的"戚家军"核心即为义乌兵，约3,000-4,000人

### 训练科目

#### 1. 基础训练

| 科目 | 内容 | 训练周期 |
|------|------|---------|
| 体能 | 每日跑步/负重行军（全副武装行军30里为合格），举石锁、拉硬弓 | 持续训练 |
| 队列 | 行进、转向、变阵（以旗号/鼓声为令）、集合/解散 | 入伍前3个月重点 |
| 号令 | 识别旗语、鼓声（进鼓/退金/集合旗/分散旗等12种以上号令）、夜间火号 | 入伍前3个月重点 |

#### 2. 武艺训练

| 科目 | 内容 | 考核标准 |
|------|------|---------|
| 长枪 | 杨家枪法（以马家枪法为基础改良），基本招式：拦、拿、扎 | 熟练掌握6路枪法 |
| 刀法 | 戚家刀法（参考日本刀法改良），基本招式：劈、砍、撩、挡 | 熟练掌握4路刀法 |
| 藤牌 | 盾牌防御+标枪投掷+配合近战 | 准确投掷标枪命中5丈外靶 |
| 狼筅 | 横扫、下压、上挑（要求覆盖正面3-4米宽度） | 能在行进中保持狼筅平稳横扫 |
| 弓箭 | 步射/骑射 | 步射：50步内中靶；骑射：30步内中靶 |
| 鸟铳 | 装填、瞄准、射击（含三段击战术） | 80步内命中人形靶 |
| 火炮 | 佛郎机操作、虎蹲炮操作 | 每门炮组3人，射速达到规定标准 |

#### 3. 阵法训练

| 科目 | 内容 | 考核标准 |
|------|------|---------|
| 鸳鸯阵 | 12人小队配合演练——标准阵→三才阵→两仪阵变阵 | 在一炷香时间内完成3次变阵 |
| 方阵 | 多队联合列阵——方阵/圆阵/长蛇阵 | 百人以上方阵在半炷香内列成 |
| 车阵 | 偏厢车展开/收拢/行进战斗转换 | 128辆车一柱香（30分钟）内结成车城 |
| 夜战 | 夜间行军、夜间变阵（以火号/金鼓为令） | 能在夜间无月光条件下保持队形 |

#### 4. 实战演练

| 科目 | 内容 |
|------|------|
| 对抗演习 | 分红蓝两军，以钝兵器（木枪/竹刀）实战对抗，按中招次数评分 |
| 攻城/守城 | 模拟攻城、守城演练 |
| 山地战 | 山地行军、伏击、反伏击 |
| 水战 | 舟师操练（限浙闽时期） |

### 考核制度

#### 考核周期

| 类型 | 周期 | 内容 |
|------|------|------|
| 日常考核 | 每日 | 队长检查本队出勤、装备、操练情况 |
| 月考 | 每月 | 各兵种武艺考核（弓、铳、枪、刀） |
| 季考 | 每季 | 阵法合练考核 |
| 年度大阅 | 每年1-2次 | 全军大检阅，总兵亲自主持 |

#### 奖惩规则——戚继光的赏罚体系

**赏格（奖励）**：

| 功绩 | 赏赐 |
|------|------|
| 斩首一级（斩获敌人首级1颗） | 赏银30两 + 升一级 |
| 斩首累计3级 | 赏银加倍 + 升二级 |
| 先登城者（攻城第一个登上城墙） | 赏银50两 + 升三级 |
| 射击/武艺考核第一名 | 赏银1-3两 |
| 月考优等 | 赏肉/酒/银若干 |
| 全队/全营无败绩 | 全队赏银，队长升级 |

**罚则（惩罚）**：

| 过失 | 惩处 |
|------|------|
| 临阵脱逃 | **斩首**（军法从事，绝不宽恕） |
| 全队退却 | 队长斩首；如全旗退却，旗总斩首（逐级追责） |
| 友军有难不救 | 主将降级/罚银；严重者斩 |
| 军令不从 | 杖责40-100（按情节轻重） |
| 装备不整 | 杖责20-40 |
| 武艺考核不合格 | 扣粮饷、补训；连续不合格者逐出 |
| 扰民/抢掠 | 杖责80-100，严重者斩 |
| 吃空饷/虚报战功 | 军官斩首，株连家属 |

#### 连坐制度

戚继光实行严格的**连坐法**（源自《纪效新书》"束伍篇"）：

| 层级 | 连坐规则 |
|------|---------|
| 伍（5人） | 一人逃跑，其余4人连坐（杖责或罚银）；全伍逃跑，伍长斩 |
| 队（12人） | 队长战死而全队溃散，全队皆斩；全队有1人逃跑而其余不追回者，全队连坐 |
| 旗（48人） | 一队溃散而他队不救，他队队长连坐 |

**核心原则**："有功同赏，有罪同罚"——鼓励集体荣誉感和互相监督。

### 军纪条例

戚继光制定了极为详细的军纪条例（《纪效新书》"禁令篇"），主要条目包括：

| 条目 | 内容 |
|------|------|
| 禁赌博 | 军中赌博者杖100，聚赌者斩 |
| 禁酗酒 | 操练/行军时饮酒者杖40；醉酒误事者杖80-100 |
| 禁盗窃 | 偷盗同袍财物者杖100；偷盗军需物资者斩 |
| 禁争斗 | 营内斗殴者双方各杖40；持刀伤人者斩 |
| 禁扰民 | 驻扎时损害百姓庄稼/财物者，赔偿并杖责 |
| 禁私出 | 未经许可离营者杖40；逾时不归者按逃兵论 |
| 禁喧哗 | 夜间喧哗者杖20；传播谣言/惑众者斩 |

### 核心思想

1. **"练兵以练心为上"**：先训练纪律和服从意识，再训练武艺
2. **"兵不贵多而贵精"**：3,000精兵胜过30,000乌合之众
3. **"练为战，非练为看"**：所有训练科目必须面向实战，反对花架子
4. **"号令明、赏罚信"**：号令系统清晰、赏罚严格执行是一切的基础
5. **"器械利而后战"**：工欲善其事必先利其器，详定每种兵器的制作标准

## 九、戚继光军事改革的历史评价

### 成就

| 方面 | 具体成就 |
|------|---------|
| 战绩 | 浙闽抗倭九战九捷（台州大捷等），歼倭寇数万；蓟镇16年无大规模入犯 |
| 制度 | 创建了中国古代最系统的军队训练体系，影响延续至清代 |
| 战术 | 鸳鸯阵是中国古代步兵战术的巅峰之一；车营是火器时代的创新 |
| 筑城 | 蓟镇空心敌台1017座，大幅提升长城防御能力 |
| 著作 | 《纪效新书》和《练兵实纪》是中国军事史上最重要的实战型兵书之一 |

### 局限

| 方面 | 具体局限 |
|------|---------|
| 个人依附 | 戚继光的改革依赖内阁首辅张居正的政治支持，张居正去世（1582）后戚继光即被弹劾罢免 |
| 推广有限 | 戚继光的练兵法仅在蓟镇全面实施，其他八镇未能复制（缺乏经费和政治支持） |
| 火器发展停滞 | 虽然戚继光大量使用火器，但明代火器技术在万历以后停滞不前，未能像欧洲那样持续革新 |
| 制度回潮 | 戚继光离任后，蓟镇营兵制迅速退化，重回旧军体制 |
