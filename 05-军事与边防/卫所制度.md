# 卫所制度

## 概述

卫所制度是明朝军事制度的根基，由朱元璋在元末割据时期萌芽、洪武年间定型，采用"寓兵于农"的思路，以军户世袭、屯田自养为核心。制度设计的初衷是"养兵百万不费百姓一粒米"，但明中叶后因逃亡、侵占、管理腐败等原因迅速崩溃，至嘉靖朝已名存实亡，被募兵制实质取代。

## 二、军户世袭制度

### 1. 军户来源

| 类别 | 说明 |
|------|------|
| 从征军 | 随朱元璋起义的军队，最初骨干 |
| 归附军 | 元朝和割据势力投降的军队 |
| 谪发军（恩军） | 因犯罪被发配充军者 |
| 垛集军 | 按《垛集令》抑配民户入伍者，后期主要来源 |

### 2. 世袭规则

- **父死子继，世代为兵**：军户身份固定，不得脱籍
- 军户一户出一名正丁服役，其余为"余丁"，正丁死亡或残疾由余丁递补
- 嫡长子优先袭职，无嫡长则庶长，无子则弟或侄
- 军户不得分户、不得从事他业（实际执行松弛）
- 军户地位低于民户，社会歧视严重

### 3. 垛集法

垛集法是卫所兵员的主要补充方式：

| 项目 | 内容 |
|------|------|
| 基本单位 | 民3户为1"垛"，其中1户为正户出军丁，其余2户为贴户 |
| 递补规则 | 正军死亡，由贴户壮丁递补 |
| 永乐以后变化 | 正户与贴户壮丁轮流更代为军 |
| 正式法令 | 朝廷颁行《垛集令》，使垛集成为定制 |
| 问题 | 民户被抑配为军户后极力逃避，加剧逃亡 |

[来源：《明朝兵制》 - 百度百科](https://baike.baidu.com/item/%E6%98%8E%E6%9C%9D%E5%85%B5%E5%88%B6/10711969)

## 三、屯田制度

### 1. 屯田比例规定

| 区域 | 屯种比例 | 守城比例 |
|------|----------|----------|
| 边地 | 七分屯种 | 三分守城 |
| 内地 | 八分屯种 | 二分守城 |

洪武二十五年（1392年）下诏："命天下卫所军卒，自今以十之七屯种，十之三守城，务尽力开垦，以足军食。"
[来源：《明史·食货志》]

### 2. 屯田规模数据

| 时期 | 屯田面积 | 备注 |
|------|----------|------|
| 洪武年间（鼎盛） | 893,194顷（约8,930万亩） | 明代军屯最高峰 |
| 永乐年间 | 维持约89万顷 | 基本持平 |
| 万历年间 | 比洪武时减少249,000余顷 | 减少约28% |
| 崇祯年间 | 大幅萎缩 | 具体数据难考 |

[来源：卫所制度 - 百度百科；军屯 - 维基百科](https://zh.wikipedia.org/zh-hans/%E8%BB%8D%E5%B1%AF)

### 3. 屯田管理与产出

| 项目 | 内容 |
|------|------|
| 授田标准 | 每军丁受田一份（约50亩） |
| 官府供给 | 耕牛、农具、种子 |
| 征粮标准 | 按份征收"屯田籽粒"，上交官仓 |
| 管理 | 卫所指挥、千户提督屯种，年终造册赴京比较 |
| 考核 | 洪武三十五年（1402年）令各卫所年终以上仓并给军子粒数目造册 |

### 4. 屯田破坏原因

| 原因 | 表现 |
|------|------|
| 军官侵占 | 卫所军官将屯田占为私产，役使军士为其耕种 |
| 权贵夺田 | 勋贵、宦官大量侵夺屯田 |
| 水旱灾害 | 屯田缺乏水利维护，产量下降 |
| 军士逃亡 | 屯田无人耕种，荒废增多 |
| 赋额未减 | 田地减少但征收定额不变，剩余军士负担加重 |

## 四、卫所崩溃：逃亡与兵力数据

### 1. 逃亡数据

| 时间 | 逃亡人数/比例 | 来源 |
|------|-------------|------|
| 洪武三年（1370年） | 47,986人 | 《明史·兵志》 |
| 正统三年（1438年） | 1,633,664人（逃亡官军总数） | 《明史·兵志》 |
| 嘉靖年间 | 部分卫所逃亡率达70% | 学术研究 |
| 16世纪初 | 一些卫所逃亡率达80% | 学术研究 |

[来源：卫所制 - 维基百科](https://zh.wikipedia.org/zh-hans/%E8%A1%9B%E6%89%80%E5%88%B6)；[澎湃新闻·军事体制与王朝运行](https://m.thepaper.cn/kuaibao_detail.jsp?contid=7302846)

### 2. 实际兵力变化

| 时期 | 名义兵额 | 实际兵力（估计） | 备注 |
|------|----------|----------------|------|
| 洪武年间 | 约180万 | 约150-180万 | 制度初创，逃亡已存在但比例低 |
| 永乐年间 | 约270万 | 约200-270万 | 鼎盛期 |
| 正统年间 | 约270万 | 约150-200万 | 空额严重化 |
| 嘉靖八年（1529年） | — | 约97万 | 桂萼公布天下兵籍数据 |
| 万历年间 | — | 约80-90万 | 继续萎缩 |
| 崇祯年间 | — | 卫所兵已近乎瓦解 | 实际战力依赖募兵 |

[来源：《明史·兵志》；桂萼兵籍数据见卫所制 - 维基百科]

### 3. 逃亡原因分析

| 原因 | 具体表现 |
|------|----------|
| 军官欺压 | 克扣军粮、役使军士为私人劳役 |
| 屯田侵占 | 军官和权贵侵夺屯田，军士生计无着 |
| 戍地遥远 | 军户需远赴异地戍守，路途艰辛 |
| 社会歧视 | 军户地位低下，子女婚配困难 |
| 待遇微薄 | 月粮常被克扣，实际到手不足定额一半 |
| 世袭僵化 | 后代不愿服役，千方百计逃籍 |

## 五、嘉靖至崇祯时期的卫所状态

### 1. 军事功能丧失

嘉靖朝以后，卫所的军事功能已基本丧失：

| 方面 | 状况 |
|------|------|
| 战斗力 | 卫所兵"不堪一战"，嘉靖二十九年庚戌之变暴露无遗 |
| 军事训练 | 操练形同虚设，军士不识弓马 |
| 武器装备 | 武器朽坏，甲胄残缺 |
| 实际角色 | 卫所仅保留管理军籍和屯田的行政功能 |

### 2. 残存功能

| 功能 | 说明 |
|------|------|
| 军籍管理 | 卫所仍管理军户户籍 |
| 屯田赋税 | 屯田籽粒仍是地方财政来源之一 |
| 地方治安 | 部分内地卫所承担基层治安 |
| 驿传保障 | 一些卫所负责驿站运营 |

### 3. 嘉靖以后的制度调整

| 调整 | 内容 |
|------|------|
| 营兵制兴起 | 以募兵编成的营兵逐步取代卫所兵执行军事任务 |
| 卫所清军 | 朝廷多次清查军籍，试图追回逃亡军户，效果有限 |
| 屯田清丈 | 多次清丈屯田，追缴被侵占田地，收效甚微 |
| 卫所改编 | 部分边地卫所兵被编入营兵体系 |

## 六、卫所制度与募兵制的并行关系

### 制度对比

| 维度 | 卫所制 | 募兵制 |
|------|--------|--------|
| 兵源 | 军户世袭 | 自愿应募 |
| 待遇 | 屯田自养+月粮 | 银饷（年饷9-30余两） |
| 训练 | 松弛 | 将领亲自训练 |
| 兵将关系 | 兵将分离 | 兵将相习 |
| 编制 | 卫所体系 | 营哨体系 |
| 财政负担 | 理论上自给 | 需国库拨饷 |
| 战斗力 | 明中叶后极低 | 较高 |
| 主导时期 | 洪武至正德 | 嘉靖至崇祯 |

### 并行阶段

| 时期 | 关系 |
|------|------|
| 正统-弘治 | 卫所为主，零星募兵补充 |
| 嘉靖年间 | 并行共存，募兵渐成主力 |
| 隆庆-万历 | 募兵主导，卫所名存实亡 |
| 天启-崇祯 | 完全依赖募兵，卫所仅存行政外壳 |
