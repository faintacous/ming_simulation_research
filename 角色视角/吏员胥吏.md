# 明朝吏员胥吏职业生命周期研究报告（1521-1644）

**研究时段**：嘉靖元年（1521）至崇祯十七年（1644）

---

## 一、概览与背景

### 1.1 官与吏的分野

明代官制中，"官"与"吏"有着严格的区分：
- **官**：通过科举正途出身，在朝廷吏部挂号，有品级，为朝廷命官
- **吏**：不入流，无品级，负责具体事务执行，是基层行政的实际操作者

黄宗羲对这一结构有精辟概括："今天下无封建之国，有封建之吏。"——吏员凭借专业垄断、信息控制和在地优势，成为基层实际掌权者，而官员的频繁流动反而强化了吏员的隐性权力。

### 1.2 吏员规模数据

| 时期 | 吏员数量 | 备注 |
|------|----------|------|
| 洪武十三年（1380） | 中央六部443人 | 官105人，吏443人，官吏比约1:4 |
| 正德年间（1506-1521） | 约55,000人 | 据郑晓《今言》载 |
| 明末（崇祯时期） | "每县殆不止千人矣" | 包括经制吏与额外吏、差役 |

> **来源**：郑晓《今言》；《万历会典》卷7；《明史》卷71《选举志》

### 1.3 制度性背景：《大明会典》的编制逻辑

《大明会典》明确指出"在外各衙门事体繁简不同……俱不开载"，即中央并未给全国州县吏典制定统一名额表。这意味着地方州县的吏典定额须从地方志个案中抽样研究，中央官修文献仅给出框架性原则。《大明会典》卷7"吏员"、卷12"考核二吏员（承差知印附）"、卷13"诸司职掌（吏员升转阶梯）"是研究吏员制度的核心文本。

## 二、入行途径

### 2.1 三大主要来源

明代吏员来源主要有三种方式：

#### （1）佥充（明初主要方式）
- **条件**：农民身家无过，年龄三十岁以下，能书写、识字会算
- **程序**：自愿申请→里老乡绅审核→布政司备案→成为正式吏员
- **来源**：《大明会典》规定，"选佥农吏是明代胥吏的主要来源"

#### （2）罚充（惩戒性补充方式）
- **对象**：
  - 生员考试不谙文理者（廪膳十年以上发附近充吏，六年以上发本处充吏）
  - 监生有过者
  - 官员犯过被谪充吏者
- **说明**：成化以后基本停止此方式

> **来源**：《明史·选举志》；《盟水斋存犊》

#### （3）求充/纳银充吏（明中后期主流）
- **背景**：景泰以后逐渐取代佥充成为主要参充方式
- **程序**：纳银→考试→授职
- **纳银额度**：最低约15两银子
- **考试内容**：刑名、行移、写字三项
- **特点**：
  - "民家子弟初在本处纳银充吏，谓之农民"
  - "挂名卯簿，则家里不坐重差，免一丁，免力役"
  - 援纳者甚众，往往不得实缺，遂有"候缺吏"

> **来源**：刘涛《明代吏员的参充及指参》；《明史·选举志》

另有"许各保贴书二名"的地方保结与贴书制度（《大明会典》），说明入行途径不止于纳银正途，地方保结也是一条路径。

### 2.2 其他来源

| 来源类型 | 说明 |
|----------|------|
| 元代旧吏 | 明初留用，后逐渐淘汰 |
| 还俗僧道 | 正统时"营充吏役" |
| 皂隶买通 | 原皂隶身分，买通官员得以充吏 |
| 退役复充 | 已退役胥吏贿赂官员再次充吏 |

### 2.3 顶首银制度

吏职实际上成为可交易的"产权"：
- **顶首银（顶头银）**：新旧吏员交接时，接任者支付给卸任者的费用
- **价格区间**：
  - 地方衙门：数十至数百两
  - 京师权要之吏：数千金
  - 大学士许讚称："在京各衙门吏典交代，顶头银两渐至数百两"
- **性质**：职位买卖的灰色收入

> **来源**：焦竑记载；黄宗羲《明夷待访录·胥吏》；《明代吏员和吏治》

## 三、六房书吏与三班衙役

### 3.1 组织架构

明代县衙实行"三班六房"制：

```
县衙结构金字塔：
┌─────────────────┐
│     知县（主官）     │  有品级朝廷命官
│   县丞、主簿（佐贰）  │  有品级朝廷命官
├─────────────────┤
│    典史（首领官）     │  无品级，不入流
├─────────────────┤
│     六房书吏        │  经制吏员
│     三班衙役        │  差役
├─────────────────┤
│  书手、算手、贴书等   │  编外人员
│  白役、防夫、快手等   │  杂役
└─────────────────┘
```

### 3.2 六房书吏

六房对应中央六部，分房管理各类事务：

| 房科 | 主要职责 | 重要职位 |
|------|----------|----------|
| **吏房** | 人事管理、官员考核升迁、公文档案 | 掌案书吏 |
| **户房** | 钱粮税赋、黄册编修、徭役征派 | 算手、粮书 |
| **礼房** | 科举考试、学务、典礼、祭祀 | 礼书 |
| **兵房** | 民兵、驿递、军械 | 兵书 |
| **刑房** | 案件审理、刑名律例、监狱管理 | 刑书、狱典 |
| **工房** | 工程营造、水利、工匠管理 | 工书 |

> **制度规定**："正统元年奏准，天下裁减衙门，每房止存司吏一名，典吏二名"

从地方志材料可见六房有时左右分列：左吏户礼、右兵刑工（这是制度推断，实际排布因地而异）。

#### 六房书吏具体职责（据《明会典》"新官到任各房供报须知式样"）

**吏房**：
- 管理本县官员履历、考核、升迁
- 负责各类公文档案的书写、抄录
- 掌握官员任免的关键信息

**户房**：
- 钱粮征收、核算
- 黄册（户籍册）编修
- 徭役分派
- 案例：崇祯时广州府粮道吏职，书手可修改申报材料决定官职选拔

**刑房**：
- 案件登记、审理记录
- 律例查阅、判决文书起草
- 监狱管理

> **来源**：《明会典》；马伯庸《胥吏的盛宴》；内乡县衙博物馆资料

### 3.3 三班衙役

| 班次 | 职责 | 人员构成 |
|------|------|----------|
| **皂班** | 站堂、喝威、行刑、仪仗 | 皂隶 |
| **壮班** | 维持治安、看监狱、守城门 | 民壮 |
| **快班** | 缉捕罪犯、传唤、押解 | 捕快、快手 |

**补充说明**：
- 有时增设"捕班"，与快班合称"捕快"
- 三班衙役并非专职，多为轮流服役
- 黄宗羲《明夷待访录·胥吏》将"皂隶、快手、承差"归于官府雇役链条，说明其与六房书吏不同，偏执行与差遣

**皂隶工食银**（有明文数据）：
- 弘治后期形成"直堂把门看监皂隶每名每年工食银10两"等规则，并有闰月加给细则
- 嘉靖元年（1522）"动支1000两，雇募在京公差官员长跟皂隶"，反映财政化雇役
- 嘉靖六年（1527）"京办皂隶每100名减6名"，反映控编与减费

> **来源**：《图书集成》所引《明会典》嘉靖条，含皂隶工食与裁减数字

### 3.4 其他吏员类型

| 类型 | 职位示例 | 说明 |
|------|----------|------|
| 首领官 | 典史 | 掌管一县吏员、差役 |
| 承发房 | 承发 | 公文收发传递 |
| 架阁库 | 管库书吏 | 档案保管 |
| 专管吏 | 闸吏、坝吏、驿吏、攒典 | 专门机构吏员 |
| 书算人员 | 书手、算手、主文、贴书 | 协助文书、会计 |

> **来源**：《明代吏胥制度及其历史地位》；马伯庸《胥吏的盛宴》

## 四、日常工作与职能

### 4.1 核心工作内容

书吏的核心职能可归纳为四类：

| 类别 | 主要职责 |
|------|----------|
| **主管吏员** | 起草文案、拟定呈稿 |
| **案牍吏员** | 抄写收发、保管文案、档案管理 |
| **司财吏员** | 保管、收支钱粮及其它实物 |
| **司狱吏员** | 分管狱吏、管理人犯 |

### 4.2 文书工作流程

书吏在衙门文书工作中占据核心地位：

```
案件处理流程：
1. 收案登记 → 书吏接收诉状，登记造册
2. 查阅例案 → 书吏检索类似案例、相关律例
3. 拟定文稿 → 书吏起草处理意见
4. 呈送长官 → 官员"略加润色"
5. 誊写发出 → 书吏缮写正式文书
6. 归档保管 → 书吏存档备查
```

> **典型案例**："书吏检阅成案，比照律，呈之司官，司官略加润色，呈之堂官，堂官若不驳斥，则此案定矣"

役班的核心则是"值堂、传唤、拘捕、押解、巡缉、差遣"。书吏与衙役构成"文案+执行"的二元结构，是州县治理真实运行的骨架。

### 4.3 专业依赖

官员对书吏的深度依赖体现在：

| 依赖方面 | 具体表现 |
|----------|----------|
| **刑名** | 律例繁杂，官员不熟悉，需书吏检索 |
| **钱谷** | 赋税计算复杂，需书吏核算 |
| **例案** | 成例汗牛充栋，书吏掌握 |
| **文书** | 公文格式、行移规矩，书吏精通 |

晚明州县中，知县流动快、任期短，吏役"在地化"导致其事实影响力不断上升。这种"熟手治理"结构是明代基层行政的根本特征。

> **来源**：黄宗羲《明夷待访录·胥吏》

## 五、收入与灰色地带

### 5.1 法定收入

#### （1）吏员工食银
- 正德以后：书吏每年约6两银子
- 皂隶工食银另有明文规定（见3.3节）

#### （2）收入对比表

| 人员类型 | 年收入（约） | 备注 |
|----------|-------------|------|
| 正七品知县 | 俸银45两+补贴 | 另有大量陋规 |
| 中央部门书吏 | 工食银约6两 | 法定收入极低 |
| 地方书吏 | 工食银约6两 | 实际多靠陋规 |

| | 起送农民银 | 0.5-0.8两 |

| | 起送吏农拨缺银 | 兵刑工5两，户3两，礼1-0.5两 |

> **来源**：海瑞《淳安县政事序》；冯桂芬估计

#### （3）书吏灰色收入总额估计

据晚清学者冯桂芬估计（适用于明代时需注意时代差异）：
- 吏部四司书吏：每年约300万两
- 兵部、户部、工部四部书吏：每年不少于1000万两
- 全国书吏灰色收入总额：超过2000万两
- 当时全国财政收入：约4000多万两
- **比例**：书吏灰色收入高达国家财政的一半

[注：冯桂芬为清代学者，此估算用于明代情况需谨慎，但其揭示的结构性规律对明代仍有参考价值]

> **来源**：冯桂芬《校邠庐抗议》

**《大明律》的法律规制**：《大明律》卷23"受赃"对灰色收入设定了极严的法律门槛：
- 有禄官"枉法赃80贯绞"
- 不枉法赃到120贯为流三千里上限段
- 无禄人（含典吏类）枉法赃到120贯亦可至绞

说明国家在法条上高压，但实践上仍长期存在"例规化"灰色收入，两者长期并存。

### 5.3 顶首银收入

| 职位类型 | 顶首银价格 |
|----------|------------|
| 一般地方吏职 | 数十至数百两 |
| 京师权要之吏 | 数千金 |
| 户部银库书吏 | 数百至数千两 |

### 5.4 收入特征分析

| 特征 | 说明 |
|------|------|
| **合法收入极低** | 工食银仅约6两/年，不足以养家 |
| **灰色收入丰厚** | 陋规、部费、顶首银等 |
| **与职位相关** | 户房、刑房、吏房等要害部门收入最高 |
| **地域差异大** | 京师、富裕县份收入远高于偏远地区 |

## 六、任期与出路

### 6.1 任期制度

#### （1）考满制度

| 考核阶段 | 时间 | 内容 |
|----------|------|------|
| **初考** | 三年 | 第一次考核，称职者升转 |
| **再考** | 六年 | 第二次考核 |
| **通考（京考）** | 九年 | 三考满，获得出身资格 |

《大明会典》卷13给出了吏员的品级内转阶梯（未入流→九品→八品……），说明吏并非完全静止身份，理论上有升转空间。同时又有"在外大小衙门典吏不许升转；36个月考满给由赴京听用"的规则，体现"地方锁定+期满输送"的双轨机制。

#### （2）出身等级

根据《大明会典》，三考满后按衙门品级授予不同出身：

| 衙门品级 | 吏员职位 | 授予出身 |
|----------|----------|----------|
| 一品衙门 | 提控 | 从七品 |
| 二品衙门 | 都吏 | 从七品 |
| 一、二品衙门 | 掾史、典吏 | 正八品 |
| 二品衙门 | 令史、典吏 | 正八品 |
| 三品衙门 | 令史 | 从八品 |
| 三品衙门 | 典吏 | 正九品 |
| 四品衙门 | 司吏 | 正九品 |
| 四品衙门 | 典吏 | 从九品 |
| 五品衙门 | 司吏、书吏 | 从九品 |
| 六至九品衙门 | 吏典 | 杂职出身 |

> **来源**：《大明会典》；《明史·选举志》

### 6.2 出路变化

#### 明初（洪武至永乐）
- "三途并用"：科目、举贡、吏员并进
- 吏员考满后可升任郡县之职
- 有吏员出身做大官者：滕懋德、张度、胡祯、徐辉、李友直、徐晞、尤祺等做到尚书；王钟、刘本道、陈宁、汪河、叶春、刘敏、王敏等做到侍郎

#### 明中期（宣德至成化）
- 宣德七年：吏员三考满后需经吏部考试
  - 考试内容：文义、行移、书写
  - 三项合格为一等，两项合格为二等
  - "三者俱无可取者，罢为民"

#### 明中后期（成化以后）
- 重科举轻吏途，"吏员与科举，高下天渊"
- 吏与官之比约3:1，候缺困难
- 成化时大学士商辂疏奏："吏员考满，冠带听选有经十二三年未得除授者"
- 有的终因老、疾、贫被罢为民

> **来源**：《明史·选举志》；顾炎武《日知录》

### 6.3 万历年间宛平县官员出身统计

据宛平知县沈榜统计（洪武元年至万历十九年）：

| 职位 | 总人数 | 吏员出身 | 比例 |
|------|--------|----------|------|
| 县令 | 25人 | 0人 | 0% |
| 县丞 | 47人 | 1人 | 2.1% |
| 主簿 | 19人 | 3人 | 15.8% |
| 典史 | 17人 | 16人 | 94.1% |

> **来源**：沈榜《宛署杂记》

这一数据直观说明了明中后期吏途的实际天花板：吏员的晋升上限基本止于典史，进入县级正职序列的比例极低。

### 6.4 后期出路困境

| 问题 | 表现 |
|------|------|
| 候缺时间过长 | 十余年不得除授 |
| 任职地域偏远 | 即使得官，也到边远地区任小官 |
| 升迁天花板 | 最高只能升至杂职官 |
| 盘踞衙门 | 役满不到吏部候职，私相顶替 |

## 七、吏员的实际权力

### 7.1 权力来源

| 权力来源 | 说明 |
|----------|------|
| **专业垄断** | 掌握刑名、钱谷、例案专业知识 |
| **信息垄断** | 控制档案、文书、信息流通 |
| **程序控制** | 文书必经书吏之手，可上下其手 |
| **在地优势** | 土著世守，熟悉地方情况 |

### 7.2 弄权方式

#### （1）把持政务

> "州县曰可，吏曰不可，斯不可矣。卿贰、督抚曰可，部吏曰不可，斯不可矣……天子曰可，部吏曰不可，其不可者亦半焉。其权直出宰辅大臣之上。"
> ——郑观应《盛世危言》

具体体现为掌握流程权（立案先后、文书格式、回驳节奏）、掌握信息权（钱粮册、词讼卷、里甲册）、掌握执行接口（与皂快承差/捕役衔接）。

#### （2）舞文弄法
- 窜改公文日期（如崇祯广州府案例："五月二十八日"改为"九月二十八日"）
- 隐匿、改易、焚毁档案
- 夹私诬罔，贿赂行文

#### （3）挟例弄权
- "法可知而例不可知，吏胥得操其两可之权"
- 利用例案繁杂、矛盾之处上下其手

#### （4）包揽词讼
- 与本地劣绅、痞棍串通
- 恐吓乡愚，勒索无辜

### 7.3 "缺主"现象

| 特征 | 说明 |
|------|------|
| **定义** | 独掌一司事务的书吏，成为实际"主管" |
| **盘踞** | "交通贿赂，上下朋奸，书办尚有更换，缺主总无改移" |
| **世袭** | "子孙世业，遂成积蠹" |

### 7.4 地缘垄断

明代后期中央各部门胥吏职位被绍兴人高度垄断：

> "一入衙门，前后左右皆绍兴人"
> ——沈德符《万历野获编》

绍兴师爷群体形成了跨代际、跨地域的职业网络，成为明清之际行政专业化的典型案例。

## 八、风险与惩处

### 8.1 法律地位

| 方面 | 规定 |
|------|------|
| **社会地位** | 被视为"贱役"，不得参加科举 |
| **任职限制** | 五年役满必须退役 |
| **重役禁止** | 禁止连续充役 |

### 8.2 常见犯罪类型

#### （1）监守自盗
- 侵盗仓库钱粮
- 案例：宋佑"侵匿一百八十金，以充私裳"

#### （2）受贿枉法
- 收受贿赂，曲法裁断
- 户部堂司书吏百余人，十余年接受贿银70万两（注：此案例为康熙末年清代案例，非明代数据）

#### （3）勒索讹诈
- 向百姓、商人勒索
- 案例：漕运坐粮厅书吏，十年贪污10万两

#### （4）舞文弄法
- 窜改公文、销毁档案
- 夹私诬罔，包揽词讼

### 8.3 刑罚制度

#### （1）充军刑

| 等级 | 距离 | 说明 |
|------|------|------|
| 附近 | 1000里 | 最轻 |
| 沿海 | 1500里 | |
| 边卫 | 2000里 | |
| 边远 | 2500里 | |
| 极边 | 4000里 | |
| 烟瘴 | 4000里 | 最重（南方瘴疠之地）|

#### （2）期限类型

| 类型 | 说明 |
|------|------|
| **终身充军** | 本人充军至死 |
| **永远充军** | 子孙世代充军，直至"丁尽户绝" |

#### （3）适用罪名

- 《大明律》中适用充军的罪名：46条
- 《问刑条例》中充军罪：252种
- 嘉靖二十九年（1550）颁《充军条例》：213种犯罪适用充军

**《大明律》受赃条款具体门槛**：
- 有禄官：枉法赃80贯绞；不枉法赃120贯为流三千里上限段
- 无禄人（含典吏类）：枉法赃120贯亦可至绞

国家在法条上对受赃设置极高压力，但实践上"例规化"灰色收入长期存在，"吏卒贪赃，岂能尽革"（朱元璋语）。

> **来源**：《大明律》；《问刑条例》；《明会典》

### 8.4 实际惩处情况

| 问题 | 表现 |
|------|------|
| **法不责众** | "吏卒贪赃，岂能尽革"（朱元璋语） |
| **官官相护** | 官员依赖书吏，往往姑息 |
| **贿赂横行** | 上下朋奸，难以追究 |
| **刺字羞辱** | 吏员犯罪要刺字（军职官员免刺） |

**政治风险**（嘉靖至崇祯）：考察、巡按刷卷、清查册籍时集中爆雷；崇祯后期财政崩坏、军需加派，基层吏役被迫在"催科—维稳—自保"之间高压运转，战乱冲击下的流离失所也是重要风险。

### 8.5 典型案例

#### 案例一：甘肃十一仓侵盗案
- 犯人武忠等将侵盗官仓看作"预存养老资金"
- 企图事发后回原籍过富裕生活

#### 案例二：金时英案
- 因"侵粮问革"后无力缴纳赎罚
- 要挟养父出钱，"要胁诬控"

#### 案例三：康熙末年户部书吏案（清代案例，仅作结构性参考）
- 户部堂司书吏百余人
- 接受办草豆商人"馈送"
- 几年间得银70万两
- "一尘不染者，仅一、二人而已"

### 9.2 "吏有封建"现象

| 对比 | 官员 | 吏员 |

- 形成"吏役团体"利益共同体
- 胥吏"日以伙，每县殆不止千人矣"

## 十、结论与评价

### 10.1 制度困境

| 困境 | 表现 |
|------|------|
| **低薪制** | 法定收入不足以养家，默许灰色收入 |
| **升迁难** | 明中后期吏途堵塞，九年考满难获官职 |
| **地位低** | 被视为"贱役"，不得科举 |
| **管制松** | 五年役满规定流于形式，重役普遍 |

### 10.2 历史影响

| 方面 | 影响 |
|------|------|
| **吏治腐败** | 顾炎武抨击明朝胥吏"皆虎狼也" |
| **基层治理** | 吏员成为基层实际掌权者 |
| **社会矛盾** | 盘剥百姓，加剧社会矛盾 |
| **制度衰亡** | 成为明朝政治腐败的重要表现 |

### 10.3 核心悖论

明代胥吏制度存在根本性的悖论：

1. **必要性**：庞大帝国的治理离不开胥吏
2. **低贱性**：法律地位低下，不得科举
3. **专业性**：掌握刑名钱谷专业知识
4. **腐败性**：低薪导致依赖灰色收入
5. **顽固性**：世袭化、地缘化，难以改革

这一悖论在明代无解，并延伸至清代，形成了中国传统行政长期依赖"隐性专业阶层"运转的制度特征。

## 存疑 & 待查

1. "全国书吏灰色收入超过2000万两"的估计出自冯桂芬（清代），直接用于明代需谨慎，但其揭示的结构性规律可参考。
2. 户部书吏"十余年接受贿银70万两"的案例为康熙末年（清代），非明代数据，不宜作为明代直接证据。
3. 《大明会典》明确说"在外各衙门事体繁简不同……俱不开载"，即中央并未给全国州县吏典统一名额表，做量化研究需转向地方志个案抽样。
4. 吏员皂隶工食银的嘉靖年间变化（嘉靖元年"动支1000两雇募皂隶"、嘉靖六年"每100名减6名"）值得深入追踪，可从《图书集成》所引《明会典》嘉靖条查找原文。
5. 绍兴师爷群体在明代（相对于清代）的规模与影响有待进一步考证，《万历野获编》为重要线索，但仍需更多史料支撑。
6. 地方三班六房的定额数据，须转向地方志个案研究，中央会典明确"不统载"。
