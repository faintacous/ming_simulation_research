# 明朝佃农的职业生命周期研究（1521-1644）

## 引言

明代中后期，随着土地兼并加剧与商品经济发展，佃农群体逐渐成为农业生产的主力军。据《明史》及各类地方志记载，嘉靖至崇祯年间，全国耕地中约70%由佃农耕作，江南地区更出现"有田者什一，为人佃作者什九"的局面（顾炎武《日知录》卷十）。

本文以个体佃农（以户为单位）的完整职业经历为主线，从入行、日常生活、经济状况、发展路径、风险困境到最终结局，系统考察这一时期佃农群体的生存状态。研究材料主要来源于《明实录》《明会典》《天下郡国利病书》《沈氏农书》《实政录》以及大量明清契约文书和地方志。

### 生命周期分期模型

从宏观视角看，佃农户的职业生命周期可分为四个阶段：

1. **进入期（约15-25岁）**：通过继承佃权、婚入佃户家庭、或灾后失地转佃进入。关键约束包括押租、种子/农具借贷、与地主或里甲网络的绑定。
2. **扩张期（约25-40岁）**：劳动力最强，租种面积扩大；常见"主业稻麦+副业纺织/季节性雇工"。在江南，商品化更强（棉、桑、米市）；在北方，粮食与灾荒风险更突出。
3. **高压期（约40-55岁）**：家庭赡养与赋役压力叠加。万历"一条鞭法"后（1580s），税役货币化加强，佃农对银与市场价格波动更敏感。可能出现债务累积、典当器具、转为短租或兼打工。
4. **衰退/退出期（约55岁后）**：由子代接手佃种；转为雇农、佃仆或非农小业；灾荒/战乱下流亡、入伍或参加流民武装（晚崇祯期显著）。

这一周期叠加三个阶段性外部冲击：
- **嘉靖（1521-1567）**：土地兼并与地方差异扩大，佃农依附关系增强。
- **隆庆-万历（1567-1620）**：白银流通与市场扩张，佃农收入机会增加，但货币税负风险同步上升。
- **天启-崇祯（1620-1644）**：气候异常、灾荒、军费摊派和治安恶化，职业生命周期明显"缩短且中断"。

## 第一章 入行：如何成为佃农

### 一、入行的制度性背景

明代佃农来源复杂，按《明会典》及各地赋役黄册记载，成为佃农主要有以下几种途径：

**（一）自耕农破产沦佃**

明初朱元璋推行黄册制度，"许民垦辟为己业"，造就了大量自耕农。但随着土地自由买卖的发展，"田宅无分界，人人得以自买自卖"（海瑞《海瑞集》），土地兼并日趋严重。

据安徽休宁县二十七都五图的编审册数据统计，万历十年（1582年）至万历四十年（1612年），当地土地分配基尼系数从0.34下降至0.18，表明土地逐渐分散（栾成显《明代黄册研究》）。但这并不意味着农民处境改善——大量自耕农因无法承担赋役而被迫卖地沦为佃农。

**典型案例**：万历年间，徽州府休宁县胡玄应家族七十年间买进土地110笔，合计44.875亩，平均每次交易仅0.4亩。这些土地绝大部分来自破产自耕农的零星出卖（《明清徽州社会经济资料丛编》第一集）。

**（二）流民客佃**

嘉靖以后，随着"一条鞭法"推行，户籍管理松弛，"客佃"（外地来的佃农）大量增加。《沈氏农书》记载，明末江南"客佃"数量激增，田主之家"要宽恤佃户，不敢退佃"。

成化年间，河南开封、怀庆等府"连岁灾伤，人民离散，外来军民，畏惧粮差，不肯尽数承佃"（《明宪宗实录》卷五十六）。这说明当时已出现跨省流动的佃农群体。

**（三）押租承佃**

明代万历年间（1573-1620），福建个别地区开始实行押租制。佃农承佃时需向地主缴纳押金（又称"顶首""佃头银""粪土银"），作为履行地租缴纳义务的担保。

《天下郡国利病书》引《漳州府志》记载："佃户出力耕田，如雇佣取值，岂得称其田主？缘得田之家，见目前小利，得受粪土银若干，名曰佃头银。"这是明代押租制的典型记录。

### 二、承佃的程序与条件

**（一）契约订立**

明代租佃普遍采用契约形式。根据《万书萃宝》《三台万用正宗》等明代日用类书所载契式，承佃流程如下：

1. **招佃**：地主发布招佃信息，或通过保人介绍
2. **看田**：佃农实地查看土地肥瘠、水利条件
3. **议价**：双方商定租额、租期、押租等条款
4. **立契**：书写佃约，明确双方权利义务
5. **中保**：邀请中见人、代笔人见证
6. **交押**：佃农交付押租银（如有约定）
7. **领田**：佃农正式接管土地耕种

**典型契式**（据万历《万书萃宝》）：

> "某里某都立佃人某，今佃到某里某都某名下山田若干亩，每年议纳乾员租谷若干石，早六冬四理还，凭本宅量秤，不得少欠。如有少欠，听凭田主即时召佃，佃人不得执占。今恐无凭，立此佃约为照。"

**（二）承佃条件**

1. **生产能力**：南方佃户需自有住屋、自备牛犁和种子；北方佃户往往"居住业主之庄屋，其牛、犁、谷种间亦资养于业主"（清代总督奏章，可推及明代）。
2. **身份要求**：一般佃农需为"良民"身份。明律规定，佃户见田主"不论齿序，并行以少事长之礼"（洪武五年诏），但佃户并非贱民，与地主不存在主仆关系。
3. **担保机制**：部分地区要求保人担保，如佃户欠租逃亡，保人需承担连带责任。

**（三）入行的经济门槛**

明代中后期，承佃土地需要一定的经济基础：

- **押租银**：福建漳州地区，"粪土银"（押租）数额因地而异，一般相当于半年至一年地租
- **农具置办**：江南地区，一套完整农具（犁、耙、水车等）约需银3-5两
- **种子肥料**：首年种子、肥料投入约需银1-2两
- **生活用度**：从春耕到秋收，约需预留半年口粮

据《沈氏农书》记载，明末江南一个完整佃农的生产资料投入约需银10-15两，这对于贫困农民而言是不小的门槛。

## 第二章 日常生活与工作

### 一、居住与饮食

**（一）居住条件**

明代佃农的居住状况因地区而异：

**南方佃户**："自居己屋，自备牛、种"（清代奏章述明代遗制）。江南佃农多居住于村庄中的茅屋或瓦房，屋舍简陋但独立。

**北方佃户**："居住业主之庄屋"，依附性较强。如河南、山东等地，佃农往往聚居于地主庄田附近，形成"庄佃"聚落。

**佃仆世仆**：在徽州、宁国等地，存在"佃仆"制度。佃仆"住庄""住屋"于地主庄屋，"耕种田垣，看管山场"（《窦山公家议》）。崇祯三年（1630年），吴五得同妻汪爱弟投到地主程继英的杨坑辛塘庄屋居住，充当佃仆。

**（二）饮食结构**

明代佃农饮食以粗粝为主，因季节和地区有所差异：

**主食**：南方以稻米为主，北方以小麦、杂粮为主。但佃农往往只能食用劣质米面，"好米换银自用，劣质米交租"（明末史料）。

**副食**：蔬菜、腌菜为主，肉类仅在年节时食用。江南佃农"每日两餐，早食粥，晚食饭，佐以腌菜"（《沈氏农书》）。

**饮食量**：成年男劳动力每日约需米1.5-2升，妇女、儿童减量。据《沈氏农书》计算，两口之家全年口食约需银10两。

### 二、作息与劳作

**（一）全年农事安排**

据《沈氏农书》记载，明代江南佃农的全年劳作安排如下：

| 月份 | 主要农事 |
|------|----------|
| 正月 | 修农具、积肥料、垦田 |
| 二月 | 种桑秧、浇菜苗 |
| 三月 | 布秧、捉害虫 |
| 四月 | 插秧、耘田 |
| 五月 | 耘田、灌溉 |
| 六月 | 收割早稻、犁田 |
| 七月 | 插晚秧、耘田 |
| 八月 | 耘田、灌溉 |
| 九月 | 收获晚稻 |
| 十月 | 打谷、晒谷、交租 |
| 十一月 | 种麦、蚕豆 |
| 十二月 | 编织养蚕工具、购置年货 |

**（二）每日作息**

农忙季节（插秧、收割期）：
- 寅时（3-5点）：起身，简单进食
- 卯时（5-7点）：下田劳作
- 午时（11-13点）：田间休息，进食干粮
- 未时（13-15点）：继续劳作
- 酉时（17-19点）：收工回家
- 戌时（19-21点）：晚餐，修补农具
- 亥时（21-23点）：就寝

农闲季节：
- 从事副业（纺织、养蚕、砍柴等）
- 为地主服杂役（挑水、劈柴、看门等）
- 外出打短工

**（三）劳动强度**

据《沈氏农书》记载，一个壮劳动力可种水田8亩、桑地4亩，全年无休。顾炎武描述苏州佃农："佃人竭一岁之力，粪壅工作，一亩之费可一缗，而收成之日，所得不过数斗。"（《日知录》卷十）

### 三、家庭副业与手工业

明代中后期，随着商品经济发展，佃农家庭副业日益重要。隆庆开关、白银流通扩大之后，佃农对市场的依赖显著加深——副业收入不再仅是补贴，而成为维持生存的必要条件。

**（一）蚕桑丝织**

江南地区，"田功半荒而衣帛食鲜"（《双林镇志》）。农家妇女"自十岁以外，皆早暮拮据，以糊其口"，从事缫丝织绢。

据张履祥记载："妇女2名，每年可以织绢120疋。每匹1两，值平价银1钱，计得价银120两。"扣除成本，实际收入约28两（《补农书》）。

**（二）棉纺织业**

松江府一带，"视他郡虽劳苦倍之，而男女皆能自立"（《松江府志》）。农家妇女纺纱织布，"以织作为业"。

据方行研究，农户年产布80匹，约共为钱21600文，加上粮食收入，年收入共约为钱34725文，远高于单纯种田。

**（三）其他副业**

- **养殖业**：养猪、养鸡、养鱼
- **采集业**：砍柴、烧炭、采药
- **手工业**：编织、陶匠、石工

## 第三章 收入与经济状况

### 一、正式收入：地租形态与租率

明代地租主要有三种形态：分成租、定额租、货币租。

**（一）分成租**

北方地区多实行分成租，一般"见税什五"（对半分），即地主与佃户各取收成的50%。

福建、湖广等地也有实行分成租的，"收一石则五斗，收十石则人分五石"（《天下郡国利病书》）。

**（二）定额租**

南方地区多实行定额租。据《休西胡玄应家买田契抄件》（万历十七年至崇祯十年）统计，56笔土地交易全部采用定额租，租额如下：

| 土地等级 | 每亩租额 |
|----------|----------|
| 上田 | 干糯5-30斤，或银3分-2钱 |
| 中田 | 干糯3-15斤 |
| 下田 | 干糯1-8斤 |
| 塘 | 银2分-5分 |

苏州地区，"私租之重者至一石二三斗，少亦八九斗"（《日知录》）。按亩产2-3石计算，地租率约为40%-60%。

定额租制对佃农的影响具有两面性：丰年佃农可保留超出定额的部分，但歉年则无论实收多少都须足额缴租，形成刚性压力。万历一条鞭法推行后，部分定额租折银征收，佃农进一步暴露于银价波动风险之中。

**（三）货币地租**

明代货币地租尚不普遍，主要在江南经济作物区出现。

嘉靖九年（1530年），祁门县胡三乞等租种契约规定："其田每年议还硬租早谷五秤，若交银，每年交文（纹）银贰钱伍分"（南开大学图书馆藏契约）。

万历四十年（1612年），歙县吴承魁租约规定："递年议租谷交纳文（纹）银陆钱伍分正"（《明清徽州社会经济资料丛编》）。

太仓州学田租规定："早租米一石，折银五钱"（崇祯《太仓州志》卷四）。

### 二、额外负担：附加租与劳役

佃农除正租外，还需承担各种额外剥削：

**（一）大秤大斛**

地主使用特制大秤收租。安徽贵池县兴孝乡，地主每亩征收田租二石，"而收之者用租秤。租秤者，每石一百二十斤也。出粜则用发秤。发秤者，每石九十斤也"。一进一出，佃农多损失30%（《楼山堂集》卷十三）。

**（二）附加租**

| 名目 | 内容 | 地区 |
|------|------|------|
| 桶面 | 每租一石收耗折一斗 | 江西石城县 |
| 淋尖 | 催租仆役于栳外横征 | 福建泉州 |
| 冬牲 | 缴纳鸡鸭等，或折银 | 福建北部、安徽徽州 |
| 租膳 | 田丁供田主一饭 | 福建泉州 |
| 脚米 | 送租上门时的损耗 | 江南各地 |
| 信鸡 | 年节缴纳鸡只 | 安徽徽州 |

**（三）劳役地租**

佃农需为地主提供各种无偿劳役：

- **生产劳役**：修水利、整田埂、看山林
- **家务劳役**：抬轿、赶车、挑担、做饭
- **年节劳役**：按节送礼、办婚丧差事

《实政录》记载，河南一等佃户"向主家称贷，轻则加三，重则加五，谷花始收，当场扣取。勤动一年，依然冻馁"（吕坤《实政录》卷二）。

### 三、实际收入水平

**（一）江南地区**

据《沈氏农书》及方行研究，一个经营12亩（8亩水田+4亩桑地）的佃农家庭：

- 产米：24.2石，值银24.2两
- 产桑：360个（每个20斤），值银36两
- **总产值**：约60.2两

扣除地租（约50%）、生产成本（约30%），净收入约12-15两。

**（二）苏州地区**

顾炎武记载："吴中之民，有田者什一，为人佃作者什九。其田甚窄，而凡沟渠道路，皆并其税于田之中。岁仅秋禾一熟，一亩之收不能至三石，少者不过一石有余。而私租之重者至一石二三斗，少亦八九斗。"

据此计算：
- 亩产：1-3石
- 地租：0.8-1.3石
- 佃户所得：0.2-1.2石

"至有今日完租，而明日乞贷者"，佃农实际收入仅能维持温饱。

**（三）北方河南**

吕坤《实政录》记载："近见佃户缺食，便向主家称贷，轻则加三，重则加五，谷花始收，当场扣取。勤动一年，依然冻馁。"

北方佃农因生产资料多仰赖地主，实际收入更低，"勤动一年，依然冻馁"是普遍现象。

### 四、区域比较

佃农的收入与风险结构存在显著区域差异：

| 维度 | 江南（商品化型） | 华北（灾荒型） | 湖广（移民垦殖型） |
|------|------------------|----------------|---------------------|
| 租额形式 | 定额租为主，渐折银 | 分成租为主 | 定额/分成混合 |
| 副业比例 | 高（纺织、蚕桑） | 低（季节性雇工） | 中（垦殖扩张） |
| 主要风险 | 银价波动、市场依赖 | 旱蝗灾害、兵燹 | 水土疫病、客佃冲突 |
| 上升机会 | 较多（永佃、佃富农） | 较少 | 中等（新垦地） |

## 第四章 晋升与发展路径

### 一、永佃权的获得

明代中叶以后，在福建、江西、安徽等地，逐渐出现"永佃制"的萌芽。永佃权指佃户长期甚至永远享有土地耕种权，地主不能随意撤佃。

**（一）永佃权的来源**

1. **垦荒投入**：佃户开垦荒地、投入工本，获得长期耕种权
2. **押租转化**：交纳高额押租，获得"不限年月"的承佃权
3. **长期耕作**："久佃成业"，形成事实上的永佃
4. **买耕**：自耕农出卖土地时保留耕作权

**（二）典型案例**

万历年间福建漳州出现"一田三主"现象：

> "漳民受田者，往惮输赋税；而潜割本户米配租若干石以贱售之，其买者亦利以贱得之。当大造年，辄收米入户，一切粮差皆其出办。于是得田者坐食租税，于粮差概无所与，曰小税主。其得租者，但有租无田，曰大税主。民间仿效成习，久之租与税遂分为二。而佃户又以粪土银私授受其间，而一田三主之名起焉。"（顾炎武《天下郡国利病书》）

这里的"粪土银"即押租，佃户通过交纳押租获得长期耕种权（田面权），地主保留收租权（田底权）。

**（三）永佃契式**

明代日用类书记载的"不限年月"佃约格式：

> "某都某图立佃人某，今佃到某都某图某名下晚田若干亩，凭保人某作保，当得纹银若干两正。其田佃人小心耕作，不许荒废。每年秋收，照约交纳定额谷物，送仓无误。若佃人不愿耕作，将田退还业主，取回原银，两相交付。今恐无凭，立此佃约为照。"

### 二、押租与佃权流转

**（一）押租制的功能**

1. **保障地主利益**：防止欠租，"若有欠租，便可扣抵"
2. **保障佃农权利**：押租不退，地主不得撤佃
3. **佃权流转**：押租可以转让，形成佃权市场

**（二）佃权买卖**

明代中后期，部分地区出现佃权买卖现象。佃户可将承佃权（田面）出卖或转佃给他人，收取"皮租"或"小租"。

崇祯十五年（1642年），祁门县程氏契约记载，佃户将"力垒"（佃权）出卖，"先尽山主，毋得变卖他人"。这说明佃权已成为可交易的财产权利（《明清徽州社会经济资料丛编》第一集）。

### 三、佃富农的兴起

明代后期，在江南经济发达地区，部分佃农通过经营副业、精耕细作，积累了一定财富，形成"佃富农"阶层。

**特征**：
- 自有完善的生产资料（牛、农具、种子）
- 雇佣短工协助耕作
- 经营多种副业（蚕桑、纺织、养殖）
- 有储蓄，可交纳押租获得永佃权

《沈氏农书》作者沈氏即为佃富农典型，其经营12亩土地，年产总值达60两银，生活水平超过普通自耕农。

佃富农的兴起与隆庆-万历间白银货币化、市场扩张密切相关。商品化程度越高的地区（如松江棉布、湖州蚕丝），佃农上升为佃富农的概率越大。但这一上升路径在崇祯后期因灾荒与兵燹而基本中断。

## 第五章 风险与困境

### 一、自然风险

**（一）水旱灾害**

明代中后期，气候转寒（处于"小冰期"），灾害频发。据《明实录》记载：

- 嘉靖年间：大旱、大水、蝗灾频繁
- 万历年间："万历十五年至十七年，连岁灾伤"
- 崇祯年间："连年旱蝗，赤地千里"

天启六年（1627年），陕西大旱，"饥民相食"，直接引发明末农民起义。

**（二）病虫害**

蝗灾、螟虫等病虫害对农业生产威胁极大。佃农缺乏抗灾能力，"一遭虫荒，便致破产"。

### 二、经济风险

**（一）地租剥削**

定额租制下，无论丰歉，佃农都需按约交租。遇灾年，"田主不少蠲免，佃户只得借贷完租"。

顾炎武记载："吴中之民……至有今日完租，而明日乞贷者。"

**（二）高利贷盘剥**

明代中后期，高利贷盛行，佃农深受其害：

1. **地主放贷**："佃户缺食，便向主家称贷，轻则加三，重则加五，谷花始收，当场扣取"
2. **专业放贷者**：明末周之夔《弃草文集》记载："佃户在春末已无耕本、无日食，不得不向放生谷之人借生作活。冬熟时先须将田中所收新谷加息完债，谷债未了，租债又起……极贫者生谷债本竟莫能偿，只随冬收加息，子什其母。"
3. **银钱借贷**：苏州地区农民借债，谚称为"出门加一"（月息10%）。广东钦州"三数月间，一本而偿一利，穷民无偿，反复加算，仅及一年，一两及七八两，或准子女米谷，或准人畜，甚至以死偿之"。

**（三）赋役转嫁**

明代后期，贵族、勋戚、官僚"千方百计逃避赋役"，"富家占地万亩不纳一粒米"，赋役转嫁到无地少地的佃农身上。

嘉靖时，江西巡抚唐龙揭露："遇有造册时，行贿里胥，有飞洒现在人户者，名为活洒；有暗藏逃绝户内者，名为死寄……这些人逃避的赋役，大多转嫁到无地或少地的小农身上。"

**（四）一条鞭法后的货币化风险**

万历九年（1581年）一条鞭法全面推行后，税役折银征收。佃农被迫将实物产出换成白银缴税，暴露于两重波动：一是粮价波动（丰年粮贱银贵，佃农实际税负加重）；二是银铜比价波动（佃农日常以铜钱交易，折银时受兑换率挤压）。这一货币化进程在提高行政效率的同时，也将市场风险系统性地传导至底层佃农。

### 三、人身压迫

**（一）超经济强制**

尽管明代法律上佃户与地主"止行以少事长之礼"，但实际生活中，地主仍可利用经济手段约束佃户。

北方佃户"居住业主之庄屋，其牛、犁、谷种间亦资养于业主，故一经退佃，不特无田可耕，亦并无屋可住。故佃户畏惧业主，而业主得奴视而役使之"（清代奏章，可推及明代）。

**（二）佃仆制度**

在徽州、宁国等地，存在"佃仆""世仆"制度。佃仆"住庄""住屋"于地主，"耕种田垣，看管山场"，人身依附关系极强。

万历十九年（1591年），程究保给地主程仙立还的文书中说："先年造牛栏厕所在主山上，其地亦不起租，以准看守坟茔之资"（《窦山公家议》）。

佃仆被视为贱民，"累世相承"，不能摆脱。他们除种地外，还需承担各种杂役，"婚配、葬山"皆受地主控制。

### 四、社会动荡

**（一）兵燹战乱**

明代中后期，边患不断，内乱频仍。嘉靖倭寇侵扰东南，万历援朝战争，天启、崇祯年间农民起义，都给佃农带来深重灾难。

**（二）流民问题**

"流民"是明代严重的社会问题。自耕农破产、佃农失地后，沦为流民。嘉靖以后，"流民"日众，"客佃"大量增加。

## 第六章 退出与结局

### 一、正常退出：退佃与迁移

**（一）退佃程序**

明代佃农原则上可自由退佃，但实际操作中受经济条件限制。退佃流程：

1. 提前告知地主（一般需提前一季）
2. 清理账目，交清欠租
3. 退还押租（如有）
4. 移交土地、房屋
5. 换取"退佃文书"

**（二）迁移自由**

明后期，随着黄册制度弛废，佃农迁徙相对自由。《沈氏农书》说，田主"不敢退佃"，说明退佃已成地主头痛的问题。

但佃农能否自由迁移，还取决于经济条件。如北方佃户"居住业主之庄屋"，退佃后"并无屋可住"，实际难以迁移。

### 二、被迫退出：逃亡与流亡

**（一）欠租逃亡**

当佃农无力缴纳地租时，往往选择逃亡。正统以后，"逃佃"现象日益严重。

成化年间，河南开封、怀庆等府，"人民离散，外来军民，畏惧粮差，不肯尽数承佃，以致田地抛荒"。

**（二）灾荒流亡**

遇重大灾害，佃农被迫流亡他乡。崇祯年间，陕西、河南连年大旱，"饥民遍地，流亡载道"。

《明史·流贼列传》记载："崇祯元年，陕西大饥，延绥缺饷，固原兵劫州库。白水贼王二，府谷贼王嘉胤，宜川贼王左挂、飞山虎、大红狼等，一时并起。"

### 三、激烈反抗：抗租与起义

**（一）抗租斗争**

明代中后期，佃农抗租斗争此起彼伏，主要形式包括：

1. **较斗抗租**：反对地主大斗收租
   - 正统十三年（1448年），福建沙县邓茂七起义，起因是反对"馈田主"（送租上门）及额外勒索。
   - 顺治三年（1646年），福建宁化黄通"较桶"抗租，"倡谕诸乡，凡纳租皆以十六升为率，移耕、冬牲、豆、送仓诸例皆罢"。

2. **拖欠地租**："丰收之年也声称歉收，拖欠地租'渐以成风'"

3. **以次充好**："将收获的好米换取银钱自用，劣质米交租"

**（二）参加起义**

明末农民起义中，佃农是主要参与者。

**李自成起义**（1629-1645）：
- 李自成本人出身贫苦，曾为银川驿卒
- 提出"贵贱均田""迎闯王，不纳粮"口号
- 佃农、流民纷纷响应，"从者如流"

**张献忠起义**：
- 崇祯十四年（1641年）攻下襄阳
- 十七年（1644年）建立大西政权

**城市贫民和世仆的参加**：《中国史纲要》指出，"城市贫民和世仆的参加起义，是明末农民起义的一个显著的特点。"

### 四、悲惨结局：破产与死亡

**（一）债务破产**

佃农在高利贷和地租双重压迫下，往往陷入债务陷阱，"子什其母"，最终破产。

"收成甫毕，贫佃家已无寸储"，"勤动一年，依然冻馁"。

**（二）卖儿鬻女**

破产佃农被迫出卖子女、妻子偿债。《明实录》记载，"饥民卖儿鬻女，易子而食"的情况在灾年十分普遍。

**（三）死亡**

- **饿死**：灾年"人相食"，"饿殍遍野"
- **病死**：缺医少药，瘟疫流行
- **被打死**：遭地主私刑拷打致死
- **战死**：参加起义军后战死

## 第七章 个案分析：一个佃农的典型生涯

### 人物设定

**姓名**：王阿大（虚构，综合多例史料）
**生卒**：嘉靖二十年（1541年）- 崇祯三年（1630年），享年89岁
**籍贯**：苏州府吴江县
**身份**：佃农（后成为佃富农）

### 生平经历

**第一阶段：入佃（嘉靖四十年，1561年）**

王阿大20岁时，父亲王老汉因欠税卖地，全家沦为佃农。通过保人介绍，承佃地主张氏水田5亩，立佃约如下：

> "吴江县立佃人王阿大，今佃到张名下田五亩，每年额租米五石，秋收送仓，凭本宅量秤，不得短少。今恐无凭，立此佃约为照。"

年成好时，亩产米2石，5亩收10石，交租5石，自余5石。但"一亩之费可一缗"（约银1两），扣除成本后所剩无几，"今日完租，而明日乞贷"。

**第二阶段：艰难维持（隆庆、万历初年）**

王阿大娶妻李氏，育有二子一女。全家主要靠王阿大一人耕作，妻女从事纺织补贴家用。

据《沈氏农书》类似案例计算：
- 5亩水田年产米10-12石
- 交租5石
- 自余5-7石
- 四口之家年需口粮约12石
- 缺口5-7石，靠借贷和纺织收入弥补

**第三阶段：转型副业（万历十年以后）**

王阿大逐渐将部分水田改种桑树，兼营养蚕缫丝。妻女"夙夜赶趁"，纺织绢布。

据张履祥记载类似案例：
- 2名妇女年织绢120匹，值银120两
- 扣除成本（丝、工具、口食）92两
- 净收入28两

加上农业收入，家庭年净收入可达30-40两，逐渐摆脱贫困。

**第四阶段：获得永佃权（万历三十年，1602年）**

王阿大积累一定财富后，向地主张氏交纳"押租银"20两，获得"不限年月"的永佃权。佃约规定：

> "佃人王阿大，今交押租银二十两，永远耕作张名下田五亩，每年额租米五石，永不加租。若退佃，押租银如数退还。"

**第五阶段：成为佃富农（天启年间）**

王阿大雇佣短工协助耕作，自己主要经管蚕桑。家中购置织机2台，成为"佃富农"。

据《沈氏农书》类似案例：
- 种水田8亩、桑地4亩
- 产米24.2石，值银24.2两
- 产桑值银36两
- 总产值60.2两
- 扣除地租、成本，净收入约20-25两

**第六阶段：晚年与结局（崇祯年间）**

崇祯元年（1628年），陕西大旱，流民四起。崇祯三年（1630年），王阿大年已89岁，在家寿终。

其子继承佃权，继续耕作。但不久后，李自成起义军进入江南，王家命运随时代洪流浮沉。

### 个案评析

王阿大的生涯是明代中后期佃农的典型案例：

1. **入佃原因**：自耕农破产，被迫沦佃
2. **生存困境**：单纯种田难以维持，"今日完租，而明日乞贷"
3. **发展路径**：通过家庭副业（纺织）积累财富
4. **地位提升**：交纳押租获得永佃权，成为佃富农
5. **最终结局**：善终，但后代命运受时代影响

需要注意的是，王阿大的上升轨迹代表了江南商品化地区的最优情形。在华北灾荒频发区或湖广移民垦殖区，佃农的生命周期往往更短、更脆弱，多数止步于"艰难维持"阶段即被灾荒或战乱中断。

## 结论

明代嘉靖至崇祯年间（1521-1644），佃农群体经历了复杂的历史变迁。这一时期，租佃关系在法制上趋于平等，佃农摆脱贱民身份；在经济上，定额租、押租制、永佃权等新制度出现，佃农经营独立性增强；在实际生活中，商品经济发展为佃农提供了更多生存机会。

然而，佃农的基本处境并未根本改变。高额地租、高利贷盘剥、自然灾害、战乱动荡，使大多数佃农仍生活在温饱线上。"今日完租，而明日乞贷"是他们生活的真实写照。

从生命周期视角看，佃农何时最可能"上升"（扩佃、获永佃权）或"下坠"（失佃、流亡），取决于三个关键变量的交互：**家庭劳动力结构**（副业能力）、**租佃稳定性**（定额租vs分成租、有无永佃权）和**外部冲击暴露**（灾荒、银价波动、兵燹）。一条鞭法后，市场化既为佃农提供了通过副业上升的渠道，也放大了系统性风险——崇祯危机中佃农职业生命周期的大规模中断，正是这一双刃剑效应的集中体现。

明末农民起义的爆发，正是这一矛盾的总爆发。天灾、暴政、剥削，最终将广大佃农推向了反抗的道路。从邓茂七到李自成，佃农和流民成为推翻明王朝的主要力量。

明代佃农的职业生命周期，既是个人命运的起伏，也是整个封建社会走向没落的缩影。
