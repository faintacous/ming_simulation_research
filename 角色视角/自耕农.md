# 自耕农：职业生命周期深度研究

> 聚焦时段：嘉靖至崇祯（1521-1644）
> 核心视角：一个拥有土地、自主经营的小农家庭，从立足、维持到最终破产或存续的完整生命历程。

---

## 一、土地与家庭

### 1.1 自耕农的土地面积

明代自耕农的土地持有量因地区、时段差异极大，但总体趋势是从明初的相对均匀走向明后期的严重分化。

**明初的土地分配**

洪武年间通过大规模屯田、移民和均田政策，自耕农比例较高。据《明史》洪武二十六年（1393）数据，全国人口约6054万，耕地约8.5亿亩（明亩，约合今亩5亿余亩），人均耕地约14明亩。以每户5口计，户均约70明亩（折今亩约40余亩）。但这一数字包含了大量官田和军屯田在内，实际民户的均田面积要小得多。

**嘉靖至万历时期**

到明中后期，土地兼并加剧，自耕农户均耕地大幅缩减。根据《明代农业生产力的发展和生产关系的演变》的研究：

- **上户**："田粮十石以上"，按亩税三升五合计算，有地约300亩以上，属地主阶层
- **中户**："田粮五石左右"，有地约140亩，介于地主与自耕农之间
- **下户**：自耕农和半自耕农，通常拥有10-50亩不等

实际上，一个典型的自耕农家庭在嘉靖至万历年间，北方拥有15-30亩旱地，南方拥有10-20亩水田，基本处于温饱线上下。万历时人沈榜在《宛署杂记》中的调查显示，下户占总人口的绝大多数（约97%），这些人中包含了自耕农和佃农。

**土地的碎片化特征**

自耕农的土地通常并非连片，而是因历代买卖、典当、继承等原因分散在村庄各处。一个拥有20亩地的农户，其田地可能分为七八块，分布在村子不同方位，每块仅二三亩甚至更少，与他人的田地相互夹杂。这种碎片化增加了耕作的劳动成本和管理难度。

**万历至崇祯的急剧恶化**

到万历后期至崇祯年间，全国自耕农比例急剧下降。据研究估计，明后期佃农耕作的土地占全部耕地的70%左右。顾炎武《日知录》记载苏州"有田者什一，为人佃作者什九"——自己有田的人只占十分之一。当然这是江南极端情况，北方自耕农比例略高，但整体趋势不可逆转。

### 1.2 家庭结构

明代自耕农家庭以核心家庭为主体，通常呈现以下结构：

**典型家庭规模**：根据黄册（户口登记簿）记录，标准农户5-7口人——户主夫妇、老人（若在世）、2-4个子女。里甲制度以110户为一里，丁口数直接关系徭役摊派。

**家庭分工**：男性主管田间耕作和运粮纳税；女性纺织、养蚕、饲养家禽家畜、做饭洗衣并协助田间劳动；老人看护幼儿、做轻活、编织竹器；儿童放牛拾柴割草，少数可入私塾。

**宗族关系**：同姓聚居普遍，宗族提供互助——农忙互帮、婚丧出力、纠纷调解。但宗族内部也有贫富分化，富户常通过放贷蚕食同族贫户土地。

### 1.3 房屋

明代农民的住房严格受到等级制度限制。洪武二十六年定制：

- **庶民房屋**：不得超过三间五架，不许用斗拱、彩色装饰
- **实际民居**：北方多为土坯房或窑洞，用泥土混合秸秆筑墙，屋顶覆以茅草或灰瓦
- **南方民居**：多为竹木结构或砖木结构，富裕地区用砖瓦，贫困地区仍为茅屋

一个典型自耕农家庭的住宅：

- **北方**：正房2-3间，泥坯墙、灰瓦或茅草顶，有火炕取暖；院内有猪圈、鸡舍、简易仓房
- **南方**：2-3间木构或砖木房，有的为二层（楼上储粮），厨房与正屋相连或分开

### 1.4 家产

一个中等自耕农家庭的典型家产包括：

- **生产资料**：耕牛1头（或与邻户合养）、犁耙各1套、锄头镰刀若干、石磨1盘、水车（南方）
- **家具**：木桌1-2张、长凳数条、木床1-2张、衣柜或木箱1-2个、灶台、水缸
- **储备**：存粮若干石（收获后留种、口粮、备荒）
- **家禽家畜**：猪1-2头、鸡鸭数只、或养蚕若干

耕牛是最重要的生产资料。一头耕牛在万历年间价值约3-5两银子，对自耕农家庭而言是一笔大投资。许多贫困自耕农无力独自购买耕牛，需与邻户合养或租用，这也是他们区别于富裕农户的重要标志。

## 二、农业生产周期

### 2.1 一年四季的农事安排

明代农业遵循二十四节气指导的传统农历节奏，南北方因作物和气候差异而有所不同。

#### 南方水稻区（以长江中下游为例）

| 季节 | 月份 | 主要农事 |
|------|------|----------|
| 春 | 正月 | 修整农具、清理沟渠、积肥备种、浸种催芽 |
| 春 | 二月 | 育秧——撒种入秧田，覆稻草保暖 |
| 春 | 三月 | 犁田耙田、施肥（人畜粪、草木灰、河泥）、播种蔬菜 |
| 夏 | 四月 | 移栽插秧——全年最忙，全家出动甚至雇短工 |
| 夏 | 五月 | 第一次耘田除草，保持水位，早稻区追肥 |
| 夏 | 六月 | 二耘三耘，"越二十日而耘，再耘三耘而止" |
| 秋 | 七月 | 早稻收割；双季稻区翻耕插晚稻，须赶在立秋前 |
| 秋 | 八月 | 单季稻灌浆成熟，排水晒田 |
| 秋 | 九月 | 收割打谷晒谷入仓；播种冬小麦或油菜 |
| 冬 | 十月 | 晚稻收尾，捞河泥沤肥 |
| 冬 | 十一月 | 农闲——修缮房屋、打柴、妇女纺纱织布 |
| 冬 | 十二月 | 继续纺织，准备年货、祭祖、过年 |

据明代《温州府志》记载："春分时平地，浸种，下秧，通田。春夏之交，分早秧插田……初秋收早谷毕……至冬初而获，名曰双收。"

#### 北方旱作区（以华北平原为例）

| 季节 | 月份 | 主要农事 |
|------|------|----------|
| 春 | 正月 | 农闲尾声，整修农具、清畜圈积肥、"顶凌耙地"保墒 |
| 春 | 二月 | 惊蛰春耕——犁地耙地、施底肥、播春小麦 |
| 春 | 三月 | 播种谷子、高粱、大豆、棉花，需等"透雨" |
| 夏 | 四月 | 冬小麦灌浆，间苗除草 |
| 夏 | 五月 | "麦秋"——抢收冬小麦、打场晒粮；立即翻耕播种秋粮 |
| 夏 | 六月 | 锄草追肥，"锄禾日当午" |
| 秋 | 七至八月 | 秋粮管理，棉花多次采摘 |
| 秋 | 九月 | 秋粮收割入仓，播种冬小麦 |
| 冬 | 十月 | 冬小麦出苗，浇越冬水，缴纳秋粮 |
| 冬 | 十一至十二月 | 农闲——男子打短工打柴，妇女纺线织布 |

### 2.2 主要作物

**南方主粮：水稻**

- 明代水稻品种已十分丰富，包括早稻、中稻、晚稻，粳稻、籼稻等
- 双季稻在明代得到推广，但比例仍较小，约占南方总耕地的3%
- 稻麦复种（冬种小麦/油菜，夏种水稻）在长江流域广泛实行，提高了土地利用率

**北方主粮：小麦与粟**

- 冬小麦是华北地区最重要的粮食作物，秋播夏收
- 粟（谷子，脱壳后为小米）是传统北方主粮，耐旱，春播秋收
- 高粱、大豆、黍等作为辅助粮食作物
- 华北流行二年三熟制：第一年种秋粮，收后播冬小麦，次年夏收后接种豆类或杂粮

**新引进作物**

嘉靖至万历年间，多种美洲作物传入中国：
- **番薯（地瓜）**：约1560年代由吕宋传入福建，后推广至南方各省。耐旱高产，成为贫苦农民的重要口粮
- **玉米**：嘉靖年间传入，最初在西南山区种植，后逐步推广
- **花生**：万历年间在沿海地区开始种植
- **烟草**：万历年间从吕宋传入福建，迅速推广为经济作物

**经济作物**

- **棉花**：明中叶后，松江府（今上海地区）棉花种植占耕田的六七成。棉花是最重要的经济作物，与纺织业密切相关
- **桑树**：江南蚕桑区（湖州、嘉兴等地）桑园面积扩大，出现"桑争稻田"现象
- **茶叶**：福建、浙江、安徽等地的山区农民兼种茶叶

### 2.3 农具

明代农具在继承宋元基础上有所改进，铁制农具全面普及。主要农具包括：

**耕地工具**：犁（南方曲辕犁可调深浅，北方直辕犁适合深翻）、耙（竖齿耙、伐耙等，碎土整地）、耖（南方水田专用，平整田面）

**播种与中耕**：耧车（北方播种器，同时开沟、下种、覆土）、锄头、镰刀

**灌溉工具**：龙骨水车/翻车（手摇或脚踏驱动链板提水，最重要的灌溉工具，明代已有畜力驱动型）、戽斗（简易提水）、筒车（水力自动提水）、龙尾车（徐光启仿制西洋水车，"省物而不烦，用力少而得水多"）

**加工工具**：石磨（磨面）、砻（去稻壳）、碓/舂臼（舂米）、风车/扬谷器（去糠）

明清之际，"生铁麻口"技术发明后，"擦生铁"农具广泛使用，耐磨性和锋利度进一步提高。

### 2.4 灌溉

- **南方水田**：依赖河流、湖泊、池塘等天然水源，通过沟渠引水入田，龙骨水车是核心工具
- **北方旱地**：主要靠天降雨，辅以井灌。华北平原凿井灌溉较普遍但覆盖有限
- **水利工程**：嘉靖至万历年间多次兴修水利，但崇祯年间因财政困难而年久失修

### 2.5 亩产量数据

根据经济史学者的研究，明代中后期的粮食亩产量（折合今市制）如下：

| 作物 | 地区 | 亩产（市斤/市亩） | 备注 |
|------|------|-------------------|------|
| 水稻（单季） | 江南 | 约 488 | 明后期常年平均 |
| 水稻（双季合计） | 华南 | 约 600-700 | 早晚稻合计 |
| 小麦 | 华北 | 约 150-200 | 旱地常年产量 |
| 粟（谷子） | 华北 | 约 80-150 | 耐旱但低产 |
| 全国南北平均 | — | 约 346 | 各类粮食平均 |

与宋代相比，明代稻谷亩产提高约70%，麦粟亩产提高约57%，这主要得益于品种改良、施肥技术进步和复种制度的推广。

**亩产的换算关系**

明代1石约合今120-150斤（各地不一）。江南稻谷亩产一般为2-3石（折稻谷240-450斤），脱壳后得米约60%-70%，即约150-300斤米。

## 三、赋税与徭役负担

### 3.1 田赋制度与税率

明代田赋制度经历了从实物征收到折银征收的重大转变。

**基本税率**

洪武年间定制：
- **官田**：每亩税五升三合五勺
- **民田**：每亩税三升五合（即每亩约交粮3.5升）
- **重租田**（原张士诚旧地，主要在苏松地区）：每亩税八升五合，税负是普通民田的2.4倍

以一个拥有20亩民田的自耕农为例，正额田赋为 20 x 3.5升 = 70升 = 7斗，折合约0.7石粮食。

但实际上，明代各地田赋差异极大。苏松地区的"重赋"是全国闻名的沉重负担——苏州府的田赋甚至是其他府县的数倍乃至十倍。

**实物征收的弊端**

在一条鞭法推行之前，农民需将实物粮食运送至指定仓库。运输过程中的损耗由农民自行承担。更严重的是官吏在验收时的各种盘剥：

- **淋尖踢斛**：官员验收公粮时猛踢量斗（斛），使堆在斛顶的粮食洒落，洒下的粮食美其名曰"损耗"，归入官吏私囊。据记载，一次踢斛可从每石粮中多得数升
- **耗羡（火耗）**：折银后，零碎银两需熔铸成银锭，官府借口"熔铸损耗"额外加征。火耗通常加征10%-30%不等，有的甚至超过正税本身

### 3.2 一条鞭法的折银改革

**改革背景与内容**

正统元年（1436），朝廷在长江以南部分地区开始将田赋折银征收，称"金花银"，折率为每石米折银二钱五分。

万历九年（1581），张居正在全国推行一条鞭法，核心内容为：

1. **赋役合并**：将田赋和各种徭役合并为一条，统一征收
2. **折银征收**：除苏杭等少数地区仍征实物外，其余一律折为白银
3. **按田摊派**：将部分役银摊入田亩，按"丁粮"比例分摊，各地有"丁一粮三""丁六粮四"等不同比例
4. **官收官解**：由官府统一征收解运，取消里甲代征

**折银对自耕农的实际影响**

一条鞭法在制度设计上有简化税制、减少中间盘剥的积极意义，但对自耕农产生了一个致命的问题——**农民必须先卖粮换银，再用白银纳税**。

时人记载："农无银，贱其粟以易银"。农民集中在完税期限前卖出粮食，导致谷价被压低、银价被抬高，形成"银贵谷贱"的恶性循环。农民实际承受的税负因此加重——名义税率不变，但换银过程中的损失可达20%-30%。

这一问题在北方尤为严重。北方经济落后，白银流通量少，银钱兑换不便，农民往往要跑到几十里外的市镇才能兑换到银两。万历后期至崇祯年间，全球白银输入锐减（欧洲三十年战争导致明欧贸易萎缩），白银购买力飙升（通货紧缩），农民卖粮换银时实际税负翻倍，"纯靠种地卖粮、换银子完税的西北农民纷纷破产"。

### 3.3 徭役种类与时间

明代徭役体系复杂，对自耕农构成沉重负担。主要分为三大类：

#### 里甲正役

以里甲为单位承担，每年由里长1人、甲首10人应役，10年轮一次。职责包括：
- 催征钱粮、勾摄公事
- 攒造黄册（户口登记）
- 出办上供物料、地方公费
- 裁决民间纠纷、维护地方治安
- 管理民间水利设施

里甲正役轮到时，里长须垫付各种公费，负担极重。一个中等自耕农若被佥为里长，往往要倾家荡产。

#### 均徭

正统八年（1443）创立的均徭法，将经常性杂役按户等高下和丁粮多寡分派。主要役目包括：
- **粮长**：负责催征本地田赋并押运至指定仓库
- **解户**：押解赋税银两或物资到京师或省城
- **马船头**：管理驿站马匹或官船
- **馆夫**：在驿站服役，接待过往官员
- **库子、斗级**：在官仓管理粮食出入

以解户为例，农民须自备盘缠，长途押运银两或物资至京师，路途数百甚至上千里，花费数月时间，途中的食宿、运输费用全部自理。若物资在途中丢失或损坏，还须赔偿。一次解运可使一个中等自耕农家庭多年积蓄化为乌有。

#### 杂泛差役

非经常性的临时差役，包括：
- 修河、修路、修城墙
- 斫薪、抬柴（为官府提供燃料）
- 修仓、运料
- 站夫、铺兵（驿站杂役）

杂泛差役的特点是征发不定期、持续时间不确定，严重干扰农业生产。

**徭役不公的实际状况**

制度规定按户等和丁粮公平摊派，但实际执行中弊端丛生。豪强富户与官吏胥吏勾结，通过以下手段转嫁负担：
- **诡寄**：将自己的田产伪报到官绅名下，享受免役特权
- **飞洒**：将自己应纳的税粮虚报到他人名下
- **改册**：篡改黄册上的户等记录

结果是"富户反得逃避，贫民下户独受其苦"，大量贫困自耕农不堪徭役重压而流亡。

### 3.4 三饷加派的致命冲击

崇祯年间的三饷加派是压垮自耕农的最后一根稻草。

#### 辽饷

- **起始**：万历四十六年（1618），为应对后金（努尔哈赤）的军事威胁
- **加征额度**：分三次加征——第一次每亩加三厘五毫，第二次再加三厘五毫，第三次加二厘，合计每亩加征**九厘银**
- **总额**：约520万两
- **崇祯追加**：崇祯四年（1631），每亩提高到**一分二厘**，总额增至约660万两，加上关税盐课杂项，合计**740万余两**

#### 剿饷

- **起始**：崇祯十年（1637），兵部尚书杨嗣昌为镇压农民起义而奏请
- **总额**：每年加派约**280万两**（一说330余万两）
- **持续至**：崇祯十三年（1640）被迫停止

#### 练饷

- **起始**：崇祯十二年（1639），清兵入关劫掠后加征
- **加征额度**：田赋每亩再加**一分**
- **总额**：每年约**730万两**

#### 三饷对自耕农的具体冲击

以一个拥有20亩田地的北方自耕农为例计算（崇祯十二年后）：

| 项目 | 每亩额度 | 20亩合计 |
|------|----------|----------|
| 原额田赋（折银） | 约1-2分 | 约0.2-0.4两 |
| 辽饷加派 | 1.2分 | 0.24两 |
| 练饷加派 | 1分 | 0.20两 |
| **三饷合计加派** | **2.2分** | **0.44两** |
| **总税负** | **约3-4分** | **约0.6-0.8两** |

三饷加派使田赋总额增加了一倍以上。加上火耗、淋尖踢斛等额外盘剥，实际负担更重。三饷合计全国加派总额高达约**2000万两白银**，超过正额赋税数倍。

对于年收入仅数两银子的自耕农家庭而言，这意味着全部收入的三分之一乃至一半要交给官府，加上天灾导致的歉收，许多家庭根本无力承受。

## 四、日常生活

### 4.1 饮食

明代农民的日常饮食因地区和家庭经济状况差异很大，但总体以粗粮为主，肉食稀少。

**进餐频次**

- **南方经济发达地区**（江南）：一日三餐已基本普及。据弘治年间《温州府志》记载，普通农民"早晚两顿都是喝粥，中午才吃得上干饭"
- **北方农村**：很多地方仍保持一日两餐的传统，"习以为风"。农忙时节可能加一顿简餐

**主食**

- **北方**：以小米（粟米）、小麦面食为主粮。小米熬粥是最常见的早餐，午餐为杂粮饼、馒头或面条。穷困家庭以糠麸、豆饭、高粱、黍米充饥
- **南方**：以稻米为主粮。普通农户日常以稀粥为主，只有干重活时才吃干饭。贫困时掺入番薯、芋头等

**副食**

- **蔬菜**：自家菜园种植的白菜、萝卜、葱蒜、韭菜、豆角、黄瓜等应季蔬菜
- **腌制品**：咸菜、腌萝卜、酱菜是最重要的下饭菜，冬季尤其依赖
- **豆腐**：自家磨制或从小贩处购买，是主要的蛋白质来源
- **肉食**：普通自耕农家庭平日极少吃肉，只有在节日、婚丧、待客时才杀鸡宰鹅。猪肉在年节时食用，鱼在水乡地区较为常见
- **食油**：北方用菜籽油或麻油，南方用菜籽油或茶油，但用量极省

**饮品**

- 白开水或粗茶是日常饮料。农忙时喝凉开水或绿豆汤解暑
- 自酿米酒（南方）或黍酒（北方），多在节日或待客时饮用

**嘉靖以后的变化**

番薯（地瓜）、玉米等美洲作物的引入，逐渐丰富了贫苦农民的食物来源。尤其是番薯，耐旱高产，可在贫瘠山地种植，成为灾荒年份的救命粮。但这些新作物的大规模推广主要在清代。

### 4.2 服饰

明代对农民服饰有严格的等级规定。

**洪武定制**：农民只许穿绌（粗绸）、纱、绢、布；商人止许绢、布，"农家有一人为商贾者，亦不得衣绌、纱"。禁穿红色、深绿、黄色、玄色、紫色，允许蓝色、赭色等"杂色"。不许穿靴子戴官帽，妇女不得佩戴金玉珍珠饰品。

**实际穿着**：男子常服为皂布巾（黑头巾）、青布棉袄、蓝布裤、白布袜、青布鞋，平日穿青布"直身"（长袍），劳动时短衣短裤。女子穿布裙布衫，蓝布裙最常见，发髻用布巾或荆钗固定。

**棉布普及**：明代棉花推广后，棉布取代麻布成为平民主要衣料。松江布"衣被天下"。嘉靖后服饰禁令渐松弛，但多数自耕农仍以粗布衣为主，一件棉袄"新三年，旧三年，缝缝补补又三年"。

### 4.3 日常起居

- **北方**：以火炕为生活中心，炕下烧柴或秸秆取暖。**南方**：堂屋为活动中心，冬季火盆取暖
- **照明**：菜籽油灯为主，贫困家庭用松明或竹篾。日出而作、日落而息
- **用水**：北方井水为主，南方河水、塘水为主

### 4.4 节令活动

明代农村的节令活动是农民为数不多的娱乐和社交场合。明清时期传统节日体系已完善定型，伦理交往与世俗娱乐的特点突出。

**主要节日与农民活动**

- **春节（正月初一至十五）**：贴春联、放爆竹、祭祖、拜年、吃团圆饭。这是农民一年中最隆重的节日，即使贫困家庭也要设法准备一顿像样的年夜饭。正月十五元宵节观灯
- **清明节（三月）**：扫墓祭祖、踏青。同时也是提醒春耕的重要节气——"清明前后，种瓜点豆"
- **端午节（五月初五）**：包粽子、饮雄黄酒、挂艾草菖蒲、赛龙舟（南方水乡）
- **中元节（七月十五）**：祭祀亡灵，焚烧纸钱。明清祭祖文化盛行，"人无贫富，皆祭其先"
- **中秋节（八月十五）**：吃月饼、赏月、祭月。恰逢秋收时节，是庆祝丰收的节日
- **重阳节（九月初九）**：登高、饮菊花酒
- **冬至**：北方吃饺子，南方吃汤圆。"冬至大如年"

**庙会与社戏**

明代农村庙会是集市、宗教和娱乐的综合体。每逢佳节或神灵诞辰，村镇举办庙会，农民可以：
- 买卖农产品和日用品
- 观看戏班演出（社戏）
- 参与宗教祭祀活动
- 进行社交往来

庙会也是节日消费的重要场合，随着商品经济发展，"每逢佳节大型集市、商业庙会兴盛"。

### 4.5 婚丧开支

婚丧嫁娶是自耕农家庭最大的非生产性开支，往往导致一个家庭多年的积蓄一朝耗尽，甚至负债累累。

**婚嫁开支**

明代婚礼遵循"六礼"（纳采、问名、纳吉、纳征、请期、亲迎）的基本程序，但农村有所简化。主要开支包括：
- **聘礼（彩礼）**：各地标准不一，通常包括银两、布匹、食品等。中等自耕农家庭娶一门亲事，聘礼约需银1-3两
- **酒席**：宴请亲友邻里，需猪肉、酒、米等，花费约1-2两银
- **嫁妆**：女方家庭准备的嫁妆，包括被褥、衣物、家具等，也是一笔不小的开支
- **合计**：一场中等规模的农村婚事总花费约3-6两银，相当于自耕农家庭一年甚至数年的净收入

**丧葬开支**

明清丧仪讲究礼制，即使贫困家庭也要尽力操办。主要费用包括：
- **棺木**：最大开支之一。普通松木棺材约需银1-2两
- **丧服**：全家及亲属需制作白色丧服（麻布或白布）
- **祭品**：香烛纸钱、供品
- **酒席**：招待前来吊唁的亲友
- **风水先生**：选择墓地、确定下葬日期
- **合计**：一场中等丧事约需银2-5两

婚丧开支是导致自耕农借贷的重要原因之一。许多家庭为操办婚丧不得不借高利贷，从此陷入债务泥潭。

### 5.3 自给自足与市场参与

明代中后期自耕农已非纯粹自给自足，而是不同程度参与市场交换。**必须购买**的商品包括食盐（官府专卖）、铁制农具、针线染料、纳税用白银；**可出售**的商品包括余粮、棉布、生丝、茶叶、家禽家畜等。

市场化程度存在显著地区差异：江南最高，"松江一府，专藉棉布一事"，农妇纺织收入甚至超过种田；北方较低，自给自足成分更大；沿海沿江活跃，内陆偏远则封闭。

### 5.4 副业

**纺织**是最重要的副业。松江府为全国棉纺织业中心，"所出布匹，日以万计"，"织机十室必有"，农民"全家衣食全都依赖妇女的纺织补贴"。山东临清州"阖境桑麻，男女纺绩，以给朝夕"。广东农民养蚕年得丝40斤，"可充八口之食矣"。江南棉织区农家全年棉织200天，收入可达20石粮食的价值，远超种田。

**其他副业**：养猪养鸡鸭（猪粪兼作肥料）、编织竹器草鞋、山区采集药材山货、水乡捕鱼、农闲打短工或小规模贩运。

## 六、风险与困境

### 6.1 天灾：旱涝蝗灾的频率和影响

明代中后期天灾频率显著上升，尤其是嘉靖至崇祯年间，几乎无年无灾。

**旱灾**

旱灾是华北地区最致命的灾害。以崇祯年间为例：

- **崇祯大旱（1637-1643）**：持续时间和受灾范围为近五百年罕见，波及南北方23个省份。核心旱区河南省连旱7年，1640年达灾情顶峰
- **陕西连年大旱**：《汉南续郡志》记载："崇祯元年，全陕天赤如血。五年大饥，六年大水，七年秋蝗、大饥，八年九月西乡旱，略阳水涝，民舍全没。九年旱蝗，十年秋禾全无，十一年夏飞蝗蔽天……十三年大旱……十四年旱"。几乎年年灾害不断
- **河南惨状**："大旱遍及全省，禾草皆枯，洛水深不盈尺，草木兽皮虫蝇皆食尽，人多饥死，饿殍载道"

**蝗灾**

蝗灾常与旱灾伴随出现：
- 河南于崇祯十年、十一年、十二年、十三年连续蝗旱，"人相食，草木俱尽，土寇并起"
- 崇祯十四年（1641）六月，"两京、山东、河南、浙江大旱蝗……野无青草，十室九空"

**水灾**

南方以水灾为主要威胁。长江、黄河、淮河流域洪水频发，一次大水可淹没数县甚至数府的农田。

**对自耕农的影响**

一次严重的旱灾或水灾，意味着当年颗粒无收或大幅减产。但赋税不因灾减免（蠲免政策执行不力），农民须照常纳税。具体影响链条为：

1. 灾年减产或绝收 → 无粮缴税
2. 被迫借贷或典卖田产凑银纳税
3. 若来年仍然歉收 → 债务累积
4. 最终无力偿还 → 田产被债主或地主吞并
5. 沦为佃农或流民

### 6.2 小冰期

明末处于"明清小冰期"的最寒冷阶段，这一全球性气候异常对农业生产造成了毁灭性打击。

**气温变化**

- 1600年至1644年间，气温骤然下降到千年以来最低点，平均气温较正常时期下降约**2℃**
- 华北地区降水量下降**11%至47%**
- 1640年和1641年，年降水不足300毫米，每年5月至9月旱情加重，"禾苗尽枯，庄稼绝收"

**对农业的影响**

- 生长季缩短，北方作物无法正常成熟
- 旱灾频率和强度急剧增加
- 1637至1643年的干旱可能是中国近五百年最严重的干旱，"寸草不生，赤地千里"
- 粮食大幅减产，部分地区连续数年绝收

**连锁效应**

气候灾变引发了一系列连锁反应：
- 粮食歉收 → 粮价暴涨 → 饥荒
- 饥荒 → 人口流移 → 传播鼠疫
- 鼠疫在陕西、山西、河北三省蔓延，死亡千万以上
- 饥民造反 → 军事镇压 → 加征三饷 → 更多农民破产 → 更多人加入起义

### 6.3 借贷：利率与还款压力

当自耕农遭遇天灾、疾病、婚丧等意外开支时，借贷是唯一的选择。然而高利贷的陷阱往往使他们永远无法翻身。

**法定利率上限**

明朝法律规定：
- 月息不得超过**三分**（即月利率3%，年利率36%）
- "一本一利"原则：利息总额不得超过本金，即累计利息等于本金时停止计息
- 违者"笞四十"

**实际利率**

法律规定形同虚设，实际民间借贷利率远高于此：
- 常见的"九出十三归"：借10两银只到手9两，还款时要还13两
- 年息50%-100%的高利贷十分普遍
- 灾年借贷利率更高，甚至达到年息100%-200%
- 以粮食借贷的"青苗钱"：春借一石秋还二石，利率100%

**还款压力的恶性循环**

以一个自耕农因灾年借银3两为例（年息60%）：
- 第一年应还：3 + 1.8 = 4.8两
- 若无力偿还，利滚利第二年：4.8 + 2.88 = 7.68两
- 两年后债务增至本金的2.56倍

实际上，许多农民在借贷时即以田产作抵押（"典田"）。若到期无力偿还，抵押的田产即归债主所有。这是自耕农丧失土地、沦为佃农的最常见路径。

### 6.4 土地兼并：被迫卖田

明代中后期土地兼并是一个不断加速的过程，自耕农的土地通过多种途径被蚕食。

**兼并的主要主体**

1. **皇室宗藩**：明朝皇室和藩王大肆侵占民田建立"皇庄""王庄"。福王朱常洵（万历帝之子）就藩洛阳时，一次获赐庄田2万顷，占据河南大量良田。藩王通过奏乞、纳献、强买等方式不断扩张庄田
2. **缙绅地主**：官绅享有免除部分赋税徭役的特权，大量农民将田产"诡寄"到官绅名下以逃避赋役，实质上形成了依附关系
3. **富商大贾**：明中后期商业资本向土地投资，富商购买大量田产
4. **寺庙**：佛寺道观占有大量庙产田

**自耕农失地的典型过程**

一个自耕农家庭失去土地通常经历以下阶段：

1. **初始**：拥有20亩自有田，勉强温饱
2. **遭灾**：一年天灾导致歉收，但赋税照征
3. **借贷**：向地主或富户借银纳税，以部分田产（如5亩）作为抵押
4. **典田**：第二年仍无力偿债，将抵押田产"典"出（典卖保留回赎权），得银还部分旧债
5. **再灾**：若再遇灾年或家庭变故（如疾病、丧事），被迫继续典卖或"绝卖"田产
6. **半自耕农**：田产减至不足维持生计（如仅剩5-10亩），须同时租佃他人土地
7. **完全失地**：最终全部田产典卖殆尽，完全沦为佃农

这一过程可能持续数年，也可能因一场大灾在一两年内急剧完成。

**诡寄与飞洒的危害**

除了直接的土地买卖，"诡寄""飞洒"等手法也在蚕食自耕农的权益：
- **诡寄**：豪强将自己的田产虚报在自耕农名下，使小农替其承担赋税
- **飞洒**：将应纳的赋税凭空分摊到其他农户的税额中
- 通过"田主、地痞、豪绅、胥吏间的勾结"，使"官册与实种之间产生了极大的紊乱"，最终受害的是无权无势的小农

### 6.5 沦为佃农的过程

当自耕农失去全部或大部分土地后，摆在他们面前的首要选择就是成为佃农。

**佃农的处境**

- **地租比例**：明代地租率普遍在收成的**50%以上**，甚至有达到**70%-80%**的记录
- **定额租与分成租**：明代就全国而言，两种租制大致各占一半。定额租不论年景好坏须交固定租额；分成租按比例分成
- **押租制度**：万历年间开始出现押租制。佃农须向地主预交一定数额的银两（多半高于一年地租）作为抵押，才能获得承佃资格。这对刚失去土地的贫困农民而言又是一道门槛
- **租佃契约**：须写明租佃亩数、租额、坐落位置，有"挽中人"或"托保人"担保，并注明"不敢拖欠"等字样

**佃农的权益**

明代佃农的法律地位较前代有所改善——佃农不再被视为农奴，理论上有人身自由和迁徙权。但实际上，佃农对地主的经济依附极深，抗租的后果可能是被驱逐、丧失押租金，甚至遭受人身暴力。

## 七、从自耕农到流民/起义者

### 7.1 失地后的选择

当自耕农丧失全部土地时，面临以下出路：

1. **成为佃农**：最普遍的选择，留在本地租佃土地（详见6.5节）
2. **流入城镇做工**：在织坊、矿场做工，或当搬运工、建筑工、店铺伙计。但城镇就业容量有限
3. **沦为流民**：离开故土四处漂泊——沿官道乞讨、流入山区开荒（"棚民"）、聚集城市外围。嘉靖以后流民数量持续增加，崇祯年间达到高峰
4. **参军**：明中后期军队缺员严重，招募流民充军，但军中待遇恶劣、军饷常被克扣
5. **加入起义军**：当所有合法生存途径堵死时，武装反抗成为最后出路。崇祯年间陕西饥民"皮骨已尽，救死不赡"，官府反而加征"新饷""均输"，饥民遂揭竿而起

### 7.2 明末农民起义中自耕农的角色

#### 起义的爆发

**天启七年（1627）白水起义——序幕**

陕西白水县农民王二、种光道等人，聚集灾民数百人揭竿起义。他们首战攻入澄城县城，王二疾呼："谁敢杀知县？"众齐声应道："我敢杀！"连呼三次后攻入县城，诛杀了催征苛税的知县张斗耀。

这次起义的参与者主要是因连年旱灾、苛捐杂税而破产的自耕农和佃农。他们的直接诉求是减免赋税、赈济灾民。

**崇祯元年（1628）——全面爆发**

王二首义后各地纷纷响应。陕西府谷王嘉胤、汉南王大梁、安塞高迎祥等领导饥民起义，"全陕响应"。参加者的成分包括：
- 破产自耕农
- 无地佃农
- 流民饥民
- 失业驿卒（如李自成——崇祯初年因裁撤驿站而失业）
- 逃亡军士
- 矿工

#### 从流民到起义军

**李自成的崛起**

李自成原为陕西米脂县驿卒，因崇祯帝裁撤驿站而失业。约1630年前后加入高迎祥领导的起义军，任"闯将"。高迎祥战死后，李自成被推为"闯王"。

**"均田免赋"口号的提出**

崇祯十三年（1640）前后，李自成在河南提出"均田免赋"的口号。这一口号直击自耕农和佃农最核心的诉求——要田种、不交税。民间广泛传唱："迎闯王，不纳粮"。

这个口号之所以产生巨大号召力，正是因为它精准回应了破产自耕农的两大痛点：
1. **土地**：因兼并而失去的土地
2. **赋税**：压垮骆驼的三饷加派

#### 起义的社会基础

明末农民起义的参与者中，破产自耕农是最重要的社会基础。他们的典型经历是：

1. 原本拥有少量土地，勉强维持温饱
2. 嘉靖至万历年间，赋税加重、土地兼并加剧，生活日益困难
3. 崇祯年间，小冰期导致连年旱灾、三饷加派使税负翻倍
4. 被迫卖田、借贷、沦为佃农
5. 灾情持续恶化，佃农也无法维持生存
6. 最终沦为流民，加入起义军

这一从"自耕农 → 半自耕农 → 佃农 → 流民 → 起义者"的下沉轨迹，是明末数百万起义军参与者的共同生命路径。

#### 大顺政权与自耕农的命运

**短暂的希望**

崇祯十七年（1644）正月，李自成在西安建立大顺政权，年号"永昌"。三月攻入北京，崇祯帝自缢，明朝灭亡。

大顺政权的核心政策——"均田免赋"——反映了破产自耕农恢复自耕地位的渴望。但这一政策在实际执行中面临根本性矛盾：

- 免赋意味着政府没有固定财政收入
- 无法维持军队和行政机构的运转
- 只能依靠"追赃助饷"——向明朝官绅追缴财产充作军费

**"追赃助饷"的失控**

大顺军进入北京后，对明朝官员实行追赃政策，从自愿到强迫，最终发展为全城拷掠，"人人皆得用刑，处处皆可施刑"，满城"哀号欲绝"。

这一政策使刚刚投降的士绅阶层迅速转向反对大顺政权。最终缙绅地主与清军合流，吴三桂引清兵入关，大顺政权在短短42天后撤离北京。

**悲剧的循环**

大顺政权的失败标志着明末自耕农群体的彻底悲剧——他们以血的代价推翻了明朝的暴政，却未能建立起能够保障自身利益的新秩序。清朝入关后，虽然宣布废除三饷，但很快又重新征收辽饷，土地兼并问题也未从根本上解决。

从宏观历史视角看，明代自耕农的职业生命周期，实质上是一部从相对独立到逐步沦丧的衰落史。制度性的赋税压榨、结构性的土地兼并、周期性的自然灾害，加上晚明小冰期的极端气候，共同编织了一张越收越紧的网，将这个庞大群体从土地上连根拔起，最终化为席卷天下的滔天洪水。
