# 明朝皇帝的职业生命周期

> 聚焦嘉靖至崇祯（1521-1644），以皇帝个体的"生命周期"为主线，详列每个阶段的制度规则、具体流程与实际案例。

---

## 一、成为皇帝

### 1.1 继承制度：嫡长子继承制

**法律依据**：朱元璋《皇明祖训》明确规定皇位继承的核心原则——"有嫡立嫡、无嫡立长"，特殊情况下以"兄终弟及"为补充。

**继承顺序**（按优先级排列）：
1. 嫡长子
2. 嫡长孙（嫡长子已故时）
3. 嫡次子
4. 庶长子
5. 庶长孙
6. 其余庶子（按年龄排序）

**实际执行情况**：明朝十六帝中，真正以嫡长子身份继位者约6人（仁宗朱高炽、宣宗朱瞻基、英宗朱祁镇、宪宗朱见深、武宗朱厚照等）。嘉靖至崇祯六帝中，无一人以嫡长子身份即位：

| 皇帝 | 继位方式 | 说明 |
|------|---------|------|
| 嘉靖（世宗） | 旁支入继 | 以堂弟身份继武宗 |
| 隆庆（穆宗） | 庶三子 | 嘉靖第三子，前两子早夭 |
| 万历（神宗） | 庶长子 | 隆庆第三子，但为实际长子 |
| 泰昌（光宗） | 庶长子 | 万历长子，国本之争的焦点 |
| 天启（熹宗） | 庶长子 | 泰昌长子 |
| 崇祯（思宗） | 庶五子/兄终弟及 | 天启异母弟，天启无嗣 |

### 1.2 即位仪式的完整流程

明代皇帝即位称"登极"，仪式在奉天殿（后改称皇极殿）举行，流程如下：

**前期准备**：
1. 大行皇帝驾崩或禅位后，遗诏颁布，确定继位人选
2. 礼部与翰林院拟定登极仪注
3. 拱卫司陈列卤簿仪仗于午门外，列甲士、旗仗
4. 设五辂（五种等级的皇家马车）于奉天门外

**仪式当日流程**：
1. **告祀天地**：继位者先至南郊祭告天地，宣示受命
2. **鼓严集官**：鼓初严，百官着朝服立于午门外；鼓三严，丞相以下依次入内
3. **升御座**：皇帝着衮冕升御座，大乐鼓吹齐奏
4. **鸣鞭开帘**：乐止后，将军卷帘，尚宝卿将御玺置于案上，拱卫司鸣鞭
5. **百官朝拜**：引班导百官入丹墀拜位，行四拜礼，乐作乐止交替进行
6. **捧表宣读**：捧表官由殿西门入，跪捧贺表；宣表官跪读贺词
7. **三呼万岁**：百官搢笏、鞠躬三次、舞蹈、拱手加额，高呼万岁三声
8. **颁诏天下**：遣官册拜皇后、册立皇太子，以即位诏告天下
9. **拟定年号**：由内阁拟定新年号，自次年正月初一启用

**特殊案例——嘉靖入城之争**：正德十六年（1521），武宗驾崩无嗣，内阁首辅杨廷和按"兄终弟及"原则迎立湖北安陆兴献王朱厚熜（年仅14岁）。杨廷和要求朱厚熜从东华门入宫，先居文华殿"以皇太子礼"即位。朱厚熜坚持以"天子"身份从大明门（皇帝专用正门）入宫，直赴奉天殿登极。双方僵持不下，最终杨廷和让步，朱厚熜从大明门入，于奉天殿即皇帝位。这一入城路线之争，已为此后三年半的"大礼议"埋下伏笔。

### 1.3 遗诏与顾命大臣制度

**遗诏的产生**：
- 皇帝临终前口述遗诏，由司礼监太监手录，送内阁票拟后颁布
- 实例：天顺八年（1464）正月十六日，英宗预感不久于人世，召皇太子朱见濡及司礼监太监牛玉，口授遗命，牛玉在侧手录，写毕即送内阁票拟。次日英宗驾崩，遗诏随即颁行
- 很多"遗诏"实际上是内阁大臣在皇帝死后代拟，并非皇帝真实口述

**遗诏的标准内容**：
1. 回顾在位功业（多为美化套语）
2. 指定继位人
3. 指定辅政大臣
4. 丧服制度安排：通常规定"以日易月，二十七日释服"
5. 对朝政的最后嘱托

**顾命大臣制度**：
- 明代无正式"顾命大臣"官衔，但实践中大行皇帝指定内阁大学士或司礼监太监辅佐新帝
- 司礼监在永乐时期开始承担托孤顾命职责，形成内阁大臣与司礼监太监双重辅政的格局

**嘉靖至崇祯的顾命案例**：

| 时间 | 新帝 | 顾命安排 | 结局 |
|------|------|---------|------|
| 1521 | 嘉靖 | 杨廷和主持迎立 | 大礼议后杨廷和致仕 |
| 1567 | 隆庆 | 徐阶、高拱、张居正辅政 | 徐阶主导初政 |
| 1572 | 万历 | 张居正+司礼监冯保联合辅政 | 张居正实际掌政十年 |
| 1620.8 | 泰昌 | 方从哲为首辅 | 泰昌仅在位29天 |
| 1620.9 | 天启 | 刘一燝、韩爌辅政 | 魏忠贤+客氏迅速掌权 |
| 1627 | 崇祯 | 黄立极、施凤来等 | 崇祯迅速清除阉党，亲政 |

### 1.4 非正常继位案例：嘉靖大礼议

**背景**：正德十六年（1521年）三月，武宗朱厚照驾崩，年30岁，无子嗣，亦无同父兄弟。内阁首辅杨廷和依据《皇明祖训》"兄终弟及"条款，选定武宗堂弟、兴献王朱祐杬之子朱厚熜继位。

**核心争议**：朱厚熜即位后，杨廷和等大臣要求他过继为明孝宗（武宗之父）的嗣子，改称生父兴献王为"皇叔父"。朱厚熜拒绝，坚持追尊生父为皇帝。

**大礼议进程**：
1. **嘉靖元年（1522）**：杨廷和援引汉代定陶王、宋代濮王旧例，要求嘉靖称孝宗为"皇考"、生父为"皇叔父兴献大王"
2. **嘉靖二年（1523）**：新科进士张璁上疏支持嘉靖，主张"继统不继嗣"
3. **嘉靖三年（1524）七月**：左顺门哭谏事件——以杨慎（杨廷和之子）为首的230余名官员集体跪于左顺门放声大哭。嘉靖震怒，命锦衣卫逮捕为首者，随后对134人实施廷杖，当场杖毙16人
4. **嘉靖十七年（1538）**：嘉靖追尊生父为"睿宗知天守道洪德渊仁宽穆纯圣恭简敬文献皇帝"，神位升祔太庙排在武宗之上。改兴献王墓为"显陵"。大礼议至此彻底结束

**影响**：大礼议是嘉靖朝的政治分水岭。嘉靖通过此事确立了绝对皇权，清洗了以杨廷和为首的旧臣集团，扶植了张璁、桂萼等"议礼派"新贵，从此牢牢掌握朝政主导权。

## 二、日常政务

### 2.1 皇帝一天的时间表

**标准作息**（以明初至中期勤政皇帝为模型）：

| 时辰 | 现代时间 | 活动 |
|------|---------|------|
| 寅时末 | 约凌晨4:00 | 起床，洗漱 |
| 卯初 | 约凌晨5:00 | 早膳（乾清宫），喝茶汤、吃饼饵 |
| 卯正 | 约6:00-7:00 | **早朝**（御门听政），春夏卯正（6:00），秋冬辰初（7:00） |
| 辰时 | 约8:00-9:00 | 退朝后批阅奏章 / 召见大臣 |
| 巳时 | 约9:00-11:00 | **经筵日讲**（逢日讲日）/ 继续处理政务 |
| 午时 | 约11:00-13:00 | 午膳 |
| 未-申时 | 约13:00-17:00 | 批阅奏章 / 休息 / 后宫活动 |
| 酉时 | 约17:00-19:00 | 晚膳 |
| 戌-亥时 | 约19:00-23:00 | 燕居（休息、娱乐） |
| 子时前 | 约23:00 | 就寝 |

**大臣的时间线**：百官须午夜起床（约凌晨1:00-2:00），凌晨3:00（丑时末）到达午门外等候，凌晨5:00（寅时末）宫门开启，按品位依次进入。路途远的京官甚至需要半夜出发。

### 2.2 御门听政的具体流程

明代早朝（常朝）在奉天门（后改皇极门）举行，称"御门听政"。

**完整流程**：
1. **击鼓**：凌晨鼓声响起，文武百官分列午门左右掖门外排队
2. **鸣钟开门**：钟声响后，掖门开启，百官依次从左右掖门入，过金水桥
3. **站班**：至奉天门（皇极门）丹墀前，东西相向站立
4. **皇帝升座**：皇帝到来，升宝座
5. **鸣鞭**：净鞭响，全场肃静
6. **行礼**：鸿胪寺官赞唱，百官入班，面北行一跪三叩头礼
7. **分班侍立**：朝拜后，百官分班站好
8. **奏事**：各衙门依次奏事。谢恩、见辞人员由鸿胪寺官宣念后，在午门外行礼
9. **纠仪**：御史、序班纠察百官仪容举止违规者
10. **退朝**：鸣鞭，皇帝还宫，百官依次退出

**制度演变**：
- **永乐七年（1409）**：永乐帝在北京冬天早朝，百官久立不堪。遂将仪式与奏事分开——简短仪式后，大臣到右顺门内便殿依次奏事，无事者退还衙门办公
- **宣德十年（1435）**：英宗即位年仅9岁，大学士"三杨"（杨士奇、杨荣、杨溥）规定：每次早朝议题不超过八件；各部门须提前报题并沟通内阁拟定意见
- **正统以后**：早朝日益流于形式，实质性政务讨论转移到文华殿的"便殿奏事"和内阁票拟系统

**缺朝记录**：明代早朝缺席现象严重，有记录显示某次文武官员缺席者多达1,160人。

### 2.3 批阅奏章的方式

明代奏章处理是皇帝行使权力的核心渠道，流程分四个阶段：

**第一阶段：进奏**
- 六部等衙门堂官在早朝时面奏
- 通政使司、鸿胪寺等机构收集各地奏章后随时入奏
- 官员也可径直至会极门投递
- 通政司是明代奏章的"总枢纽"，负责分类登记

**第二阶段：呈阅**
- 奏章进呈后，由司礼监分类拣选，将要务呈送皇帝
- 皇帝阅后可做三种处理：
  - **留中不报**：搁置不理（万历帝大量使用此法）
  - **中旨**：皇帝直接下令，绕过内阁
  - **发内阁票拟**：最常规的处理方式

**第三阶段：票拟**
- 内阁大学士以皇帝名义拟写批答草稿
- 用小票墨书，贴于奏折上，供皇帝审定
- 首辅通常承担最重要奏章的票拟工作
- 票拟要反映外廷舆论并维护国家成规，有相当独立性

**第四阶段：批红**
- 皇帝用朱笔批答（批红），同意则照批，不同意可改票或留中
- 勤政皇帝亲自批红（如嘉靖"章奏批答，疾如风雨"）
- 怠政皇帝则由司礼监秉笔太监代为"照阁票批红"
- 批红完成后发还内阁，六科校对后下发各部执行

**日常奏章数量**：明中后期，每日进呈的各类奏章可达数十件到上百件，重大事务时更多。

### 2.4 与内阁的互动方式

**制度框架**：
- 明代废丞相后，内阁由"秘书咨询机构"逐渐演变为事实上的行政中枢
- 内阁大学士品级仅正五品（后加六部尚书衔提升地位），但票拟权使其拥有极大影响力
- 皇帝通过司礼监制衡内阁，形成"外廷（内阁）-内廷（司礼监）"双轨体制

**互动渠道**：
1. **面奏**：内阁大臣可在早朝或便殿单独面见皇帝
2. **密揭**：内阁首辅可通过密封奏疏直接与皇帝沟通，不经通政司
3. **票拟往返**：皇帝对内阁票拟不满意，可发回重拟（"改票"）
4. **中旨**：皇帝绕过内阁直接下达诏令，内阁有权封还拒绝执行

### 2.5 嘉靖/万历不上朝时的替代运作方式

**嘉靖模式（嘉靖二十一年后，约24年不上朝）**：

嘉靖在壬寅宫变（1542年被宫女行刺）后移居西苑永寿宫，此后二十余年不再临朝，但始终通过以下方式把控朝政：
- **亲批奏章**：嘉靖虽不上朝，但奏章必亲自批阅，"章奏批答，疾如风雨"，事必躬亲
- **密疏沟通**：通过与严嵩（首辅任期长达15年）、徐阶等重臣的密揭往来遥控朝政
- **驭臣之术**：善于利用大臣之间的矛盾互相制衡，如利用徐阶制衡严嵩，又以严嵩压制其他大臣
- **青词考核**：嘉靖崇道，要求内阁大学士撰写"青词"（道教祭祀用骈文），以此作为亲近程度和升迁的重要标准。严嵩之所以能长期专权，与其善写青词密切相关

**评价**：嘉靖不上朝但干活（且很勤勉），只是将大量精力投入修道和驭术。朝政并未瘫痪，反而运转有序。

**万历模式（万历十五年后，约28年不上朝）**：

万历在张居正去世（1582年）后逐渐亲政，但自万历十五年（1587年）起日益怠政，至万历二十年代后几乎完全罢工：
- **极少召见大臣**：与嘉靖不同，万历连奏章都懒得批
- **大量留中不发**：将奏章搁置不理，石沉大海。官员任命奏请长期无回复
- **缺官严重**：万历中后期，中央和地方大量官职空缺。六部尚书经常有数席空悬，地方督抚亦长期无人
- **国本之争**：万历因立储问题（偏爱郑贵妃之子朱常洵，不愿立长子朱常洛）与群臣对峙数十年，以怠政作为消极对抗

**评价**：万历与嘉靖的关键区别在于——嘉靖不上朝但批奏章，万历连奏章也不批。但万历时期亦未出现严重的宦官、外戚干政局面，明朝的官僚系统依靠"票拟-批红"机制在低效率状态下仍勉强运转。

## 三、生活与起居

### 3.1 饮食

**御膳管理的三驾马车**：

| 机构 | 性质 | 职能 |
|------|------|------|
| 光禄寺 | 外廷衙门 | 食材采购、菜单制定、筹备宴席 |
| 尚膳监 | 内廷宦官衙门 | 烹调制作 |
| 尚食局 | 女官系统 | 试膳（验毒）、侍膳 |

流程：光禄寺采购食材并制定菜单 → 连同食材交给尚膳监 → 尚膳监烹制 → 尚食局女官先试膳确认安全 → 呈给皇帝

**御膳规格**：
- 明朝后期宫廷饮食花销约每月1.1万两白银
- 主要食材：猪肉、面食、香油、牛奶、豆类、禽蛋、干果、糖碱、香辛料等
- 明初宫廷饮食较简朴（朱元璋崇尚节俭），以家畜家禽为主
- 永乐迁都北京后，面食（花卷、馒头、面饼）比重增大，肉菜做法多样化

**各帝饮食偏好**：
- 永乐帝：喜食鲥鱼
- 隆庆帝：嗜食驴肠
- 崇祯帝：标榜节俭，实则偏好燕窝

**用膳场景**（以崇祯为例）：
- 早起洗漱后，先在外室喝茶汤、吃饼饵
- 然后在乐队伴奏下进入乾清宫明间，面南而坐，享用早膳
- 天启年间，司礼太监体乾、魏忠贤、李永贞连同乳母客氏四家轮流办膳，每家管事官数十员、厨役数百人，乾清宫内领膳、请膳、管果酒的近侍每家四五十人

### 3.2 服饰

明代皇帝服饰按场合严格区分：

**衮冕（最高等级礼服）**：
- 十二旒冕冠 + 十二章纹衮服
- 玄衣黄裳：日、月、星辰、山、龙、华虫六章织于上衣；宗彝、藻、火、粉米、黼、黻六章绣于下裳
- 使用场合：祭祀天地、宗庙、社稷、先农、册拜、正旦、冬至、圣节等重大典礼
- 明代只保留衮冕一种，废除了先前的其余五种冕制

**皮弁服**：
- 使用场合：朔望日（每月初一、十五）视朝、降诏、降香、进表、外藩朝贡

**常服（龙袍，使用最广）**：
- 洪武三年定制：乌纱折角向上巾 + 盘领窄袖袍 + 金琥珀透犀腰带
- 永乐三年改制：盘领窄袖黄袍 + 玉带 + 皮靴
- 前后及两肩各织金盘龙一条（"四团龙袍"）
- 使用场合：常朝视事、日讲、省牲、谒陵、献俘、大阅——日常政务几乎都穿常服

**燕弁服（便服）**：
- 皇帝在宫中燕居时穿着
- 参酌古制制作，寓意"深宫独处，以燕安为戒"

### 3.3 居住

**紫禁城内的核心宫殿**：

- **乾清宫**：皇帝的正式寝宫。明代乾清宫是皇帝日常起居、召见大臣、批阅奏章的核心场所。暖阁设有龙床，两侧设书案。嘉靖以前的皇帝多居于此
- **西苑（今中南海至北海一带）**：嘉靖在壬寅宫变后移居西苑永寿宫，修道炼丹二十余年，直至驾崩
- **文华殿**：经筵日讲场所，亦用于皇帝接见大臣
- **武英殿**：偶用于皇帝处理军务

**嘉靖的西苑生活**：
- 嘉靖二十一年（1542年）壬寅宫变后，嘉靖搬离紫禁城正殿区域
- 在西苑永寿宫中设坛修道、炼丹采药
- 日常不穿龙袍，改穿道袍
- 身边围绕道士陶仲文等人
- 但仍通过密揭系统处理朝政

### 3.4 出行

**銮驾制度**：明代皇帝出行的卤簿仪仗分三个等级：

| 等级 | 名称 | 使用场合 | 规模 |
|------|------|---------|------|
| 一等 | 大驾卤簿 | 祭祀天地、太庙等重大典礼 | 规模最大，含全套仪仗、车驾、乐舞 |
| 二等 | 法驾卤簿 | 一般出巡 | 中等规模 |
| 三等 | 銮驾卤簿 | 日常出入 | 较简 |

**卤簿组成**：护卫甲士、兵器仪仗、马车（辂）、乐器、旗帜、扇、伞、盖、拂尘、服饰、动物等。整套仪仗人员可达数千人。

**实际情况**：嘉靖至崇祯时期，皇帝极少出宫。嘉靖移居西苑后几乎不出西苑大门；万历晚年行动不便，更不出宫；崇祯在位十七年，几乎没有离开过北京。明中后期皇帝最大规模的出行是祭祀天坛和先农坛。

### 3.5 后宫生活

**后宫制度**：
- 明代后宫设皇后一人、皇贵妃一人、贵妃二人、妃四人、嫔若干
- 后妃多从民间良家选取，不同于唐代的功臣之女，有利于防止外戚干政
- 皇帝侍寝由敬事房太监管理记录

**嘉靖至崇祯的后宫概况**：
- **嘉靖**：先后册封四位皇后（陈皇后、张皇后、方皇后、杜皇后未正式册封）。壬寅宫变中方皇后救驾有功。嘉靖后期沉迷修道和"采阴补阳"之术，征召大量少女入宫，采集她们的经血（"红铅"）炼丹
- **隆庆**：好女色，后宫充盈，据说纵欲过度导致早逝
- **万历**：因"国本之争"（偏爱郑贵妃、冷落长子生母王恭妃）与群臣对峙近三十年。王恭妃在冷宫中度过大半生，双目近乎失明
- **泰昌**：即位前长期被万历冷落。即位后旋即接受郑贵妃进献的美女八人，纵欲后身体垮塌，仅29天即驾崩
- **天启**：乳母客氏把持后宫，与魏忠贤结为"对食"（宫中男女配偶关系）。客氏受封"奉圣夫人"，在无皇太后主持后宫的情况下横行无忌
- **崇祯**：与周皇后感情深厚。北京城破之夜，崇祯逼皇后自尽，亲手砍伤长平公主

### 3.6 娱乐与个人爱好

**嘉靖——修道炼丹**：
- 崇信道教贯穿一生，在位四十五年中后二十余年几乎全部投入修道
- 身边长期围绕道士：邵元节、陶仲文等
- 宫中设坛炼丹，丹药含砒霜等重金属，微量砒霜使人面色红润、精神亢奋
- 命内阁大臣撰写"青词"（道教祭祀骈文），以此考核大臣
- 嘉靖二十一年壬寅宫变：宫女杨金英等十余人因不堪虐待，深夜以绳索试图勒死嘉靖，因绳结打成死结未成。事后嘉靖大肆株连，更加沉迷修道

**天启——木匠皇帝**：
- 朱由校自幼缺乏正规教育（父亲泰昌被万历冷落，朱由校亦不受重视），却对木工展现出非凡天赋
- 具体事迹：
  - 打造御用戗金双龙小铁斧
  - 雕刻护灯小屏八幅，上刻"寒鸟争鸣"等精美图案
  - 设计可折叠的轻巧木床——亲自选材、锯木、组装
  - 发明中国最早的喷泉装置——在大木桶中装满水，放置十余木球浮于水面，桶下钻孔设机关，水托木球旋转不止
- 做木工时废寝忘食，"朝夕营造，制作失败则弃之又来，毫不厌倦"
- 魏忠贤正是利用天启沉迷木工的时机，趁其全神贯注时呈送奏章请批，天启随口应答："朕已悉矣！汝辈好为之。"由此逐步专权

## 四、权力与制衡

### 4.1 皇帝实际能做什么、不能做什么

**皇帝能做的**：
- 最终决定权：所有政令须经皇帝批红方生效
- 人事任免：高级官员（四品以上）的任命须皇帝批准
- 发动战争与和平：军事决策的最终裁决权
- 赦免与恩典：大赦天下、赐爵、赏赐
- 廷杖：可以直接下令杖打大臣（明代特色）
- 中旨：绕过内阁直接下令（但可能被封驳）

**皇帝不能（轻易）做的**：
- 不能轻易废立太子：须有正当理由并获群臣认可（万历国本之争即为明证）
- 不能随意更改祖制：《皇明祖训》具有准宪法地位
- 不能无视六科封驳：六科给事中有权封还诏书
- 不能强迫内阁票拟违心意见：内阁可以拒绝执行中旨
- 不能完全绕过官僚系统：明代的行政运作高度依赖文官体系

### 4.2 与内阁/六部/言官/宦官的权力博弈

**皇帝 vs 内阁**：
- 内阁通过票拟权掌握政策建议的主动权
- 首辅的权力大小取决于皇帝的信任程度
- 强势首辅（如张居正）可以压制皇帝意志；弱势首辅则沦为橡皮图章
- 皇帝可通过更换首辅来调整政策方向

**皇帝 vs 六部**：
- 六部直接向皇帝负责（废丞相后），但实际运作中需与内阁协调
- 六部尚书掌握具体行政执行权
- 吏部尚书（"天官"）掌握中低级官员考核任免，权力极大

**皇帝 vs 言官（科道官）**：
- 六科给事中 + 十三道监察御史，合称"科道"
- 拥有封驳权（封还诏书）、弹劾权、监察权
- 言官以"死谏"为荣，敢于直面皇帝。越被处罚，声望越高
- 嘉靖、万历都曾因言官的持续上疏而苦恼不已，但无法根除这一制度

**皇帝 vs 宦官**：
- 司礼监通过批红权成为"内相"，与内阁首辅"对柄机要"
- 东厂（宦官掌管的特务机构）为皇帝提供独立于文官系统的情报
- 皇帝利用宦官制衡文官集团，但也可能被宦官反噬（如天启朝魏忠贤专权）
- 嘉靖朝宦官势力被压制；万历朝冯保权大但被清算；天启朝魏忠贤达到宦官权力巅峰；崇祯初年清除阉党但后期仍重用太监

### 4.3 中旨（绕过内阁直接下令）

**定义**：皇帝不经内阁票拟，直接以手诏形式下达的命令，称为"中旨"。

**制度规定**：
- 中旨在制度上是合法的——皇帝是最终权力来源
- 但明代形成了一个惯例：内阁有权对中旨提出异议、封还甚至拒绝执行
- 六科给事中亦可封驳中旨

**嘉靖至崇祯中旨的使用情况**：
- **嘉靖**：频繁使用中旨，尤其在大礼议期间多次绕过内阁直接下令，内阁难以抗衡
- **万历**：在国本之争中，万历试图通过中旨立朱常洵为太子，遭内阁和言官联合抵制而失败
- **天启**：魏忠贤借天启名义大量发出中旨，打压东林党人
- **崇祯**：崇祯性格急躁，频繁以中旨任免官员、调整政策，导致政令朝令夕改

**官员对中旨的态度**：以中旨任命的官员被称为"传奉官"，受到文官集团的普遍鄙视，视为"不由正途"。

### 4.4 廷杖（皇帝对官员的暴力手段）

**制度概况**：
- 廷杖即在朝廷上以荆条或竹板杖打大臣，始于朱元璋
- 明代前期（建文、永乐、洪熙、宣德）均无廷杖记录
- 从成化朝开始频繁使用，至嘉靖朝达到高峰

**行刑方式的演变**：
- **成化朝**：宪宗虽多次施杖，但刑前在臀部垫上厚垫，减轻伤害
- **正德朝**：太监刘瑾蛊惑武宗撤去垫子，要求受刑者脱裤（"去衣"），暴露私处受杖
- **嘉靖朝**：延续"去衣"制度，且频率极高

**具体案例**：

**正德元年（1506）——王阳明受杖**：
- 时任兵部主事的王守仁（王阳明）上书救援因得罪太监刘瑾而下狱的同僚戴铣等人
- 被"去衣"廷杖四十，后发配贵州龙场充军
- 王阳明在龙场悟道，创立心学

**正德十四年（1519）——南巡谏阻事件**：
- 武宗执意巡游江南，文官集团群起进谏
- 146名官员遭受廷杖，其中11人当场杖毙

**嘉靖三年（1524）——左顺门事件（大礼议廷杖）**：
- 230余名官员跪于左顺门放声大哭，抗议嘉靖追尊生父
- 嘉靖下令锦衣卫廷杖，134人受杖，16人当场杖毙
- 这是明代规模最大的一次集体廷杖

**嘉靖朝统计**：据《明史》记载，嘉靖四十八年间有名有姓的受杖官员88人，其中杖死11人。

**廷杖的政治意义**：
- 廷杖是皇帝直接对文官施加肉体暴力的制度化手段
- 对文官而言，受廷杖反而可能提升声望——"以直言获杖"被视为忠臣的标志
- 部分官员甚至主动上疏激怒皇帝，以求一杖成名

## 五、健康与死亡

### 5.1 太医院对皇帝的医疗服务

**机构设置**：
- 太医院设院使一人（正五品）、院判二人（正六品），下辖御医数十人
- 分科设置：大方脉（内科）、小方脉（儿科）、妇人科、疮疡科、针灸科、眼科、口齿科、伤寒科、正骨科等
- 院使和院判须与御医轮班在内府御药房值班供事

**诊治流程**：
- 皇帝身体不适时，太医院值班御医入诊
- 诊治方案须由多名御医会诊确定
- 药方需经司礼监审核，药材由御药房专人煎制
- 皇帝服药前须有专人试药
- 帝王出巡时，太医院官随从侍奉

**风险与困境**：
- 御医诊治皇帝压力极大：治好了是本分，治不好可能掉脑袋
- 嘉靖朝有"团灭太医院"之说——嘉靖崇信道士炼丹，对太医院的正规医疗持怀疑态度，多名御医因诊治不合心意而获罪

### 5.2 丹药服食

**嘉靖帝的丹药迷信**：
- 嘉靖一生迷恋道教炼丹，尤其是壬寅宫变（1542年）后更加沉迷
- 在道士陶仲文、邵元节等人引导下，宫中长期设坛炼丹
- 丹药成分包含铅、汞、砒霜等重金属——极低量的砒霜可使人面色红润、精神亢奋，但长期服用导致慢性中毒
- "红铅"：征召宫女采集经血炼制丹药（这也是壬寅宫变的重要原因之一）
- 嘉靖最终于嘉靖四十五年（1567年）驾崩，享年60岁，学者多认为与长期服食丹药导致慢性中毒有关

**泰昌帝的红丸案**：
- 万历四十八年（1620年）七月，万历驾崩，皇太子朱常洛即位，是为泰昌帝
- 即位后郑贵妃进献美女八人，朱常洛纵欲后身体迅速恶化
- 八月初十日起卧床不起
- 御药房提督太监崔文升用大黄、芒硝（泻药）为其治疗，导致腹泻不止，一夜如厕三四十次
- 鸿胪寺丞李可灼进献"红铅丸"，朱常洛服用后暂时好转，命再进一丸
- 八月三十日（服第二丸后数时辰），泰昌帝驾崩，在位仅29天
- 此为明末三大案之一"红丸案"，至今争议不断

### 5.3 驾崩流程

**临终与遗诏**：
1. 皇帝病危时，司礼监和内阁大臣入宫守候
2. 皇帝口述遗诏（或由大臣代拟），司礼监太监手录
3. 遗诏送内阁审定后正式颁布
4. 确认继位人选（通常是已册立的皇太子）

**丧礼流程**（大丧礼）：
1. **讣告天下**：颁布遗诏和讣告
2. **小殓**：整理遗体，更换殓服
3. **大殓**：将遗体放入棺椁（梓宫）
4. **停灵**：梓宫停于乾清宫（或其他指定宫殿），百官轮班哭临
5. **丧服**：以日易月制——以一天代一月，服丧27天后释服，不禁止民间音乐嫁娶
6. **新帝即位**：通常在丧期内举行登极仪式
7. **上谥号庙号**：由内阁和礼部拟定大行皇帝的谥号和庙号

**葬礼与陵寝**：
1. 出殡日（通常在驾崩后数月），梓宫从紫禁城出发，经北京城至昌平天寿山陵区
2. 棺柩到达陵地后，先停放在隆恩殿，遣官告祭祖陵、后土、陵寝山神
3. 在地宫内铺设木制轨道，将梓宫徐徐引上棺床
4. 主管王公大臣验视后，撤去木轨，放置陪葬品（衣物珍宝、两口装满麻油的大缸，盖上点燃"万年灯"）
5. 封闭墓门，焚烧仪仗
6. 在隆恩殿举行"题主礼"——大学士二人行一跪三叩礼后题写神位牌
7. 神牌由皇帝和大臣护卫回京，安放在太庙
8. 治丧活动宣告结束

### 5.4 嘉靖至崇祯六帝的死因和寿命

**嘉靖帝（明世宗朱厚熜，1507-1567）**：
- 在位45年，享年60岁
- 死因：长期服食丹药导致慢性中毒，加之晚年体弱多病
- 嘉靖四十五年（1566年）十二月驾崩于西苑
- 葬于永陵（十三陵中规模第二大，仅次于长陵，耗银800万两，建造7年）

**隆庆帝（明穆宗朱载坖，1537-1572）**：
- 在位6年，享年35岁
- 死因：纵欲过度导致体虚，最终中风而亡
- 隆庆六年（1572年）五月驾崩于乾清宫
- 葬于昭陵

**万历帝（明神宗朱翊钧，1563-1620）**：
- 在位48年（明朝在位最长），享年57岁
- 死因：长期酗酒、沉湎酒色，晚年腿疾严重无法行走（1956年定陵发掘证实其遗骨右腿明显短于左腿）
- 万历四十八年（1620年）七月驾崩于弘德殿
- 葬于定陵（十三陵中唯一被考古发掘的帝陵，1956-1958年发掘，出土文物3,000余件，包括金冠、衮服、金锭103枚、银锭65枚等）

**泰昌帝（明光宗朱常洛，1582-1620）**：
- 在位29天（明朝在位最短），享年38岁
- 死因：红丸案——即位后纵欲、误服泻药、再服"红铅丸"后暴毙
- 万历四十八年（1620年）九月初一驾崩于乾清宫
- 葬于庆陵

**天启帝（明熹宗朱由校，1605-1627）**：
- 在位7年，享年22岁（明朝最短命皇帝之一）
- 死因：天启五年（1625年）在西苑游船时落水，此后身体一直不好。天启七年（1627年）病情恶化，据说服用"仙药灵露饮"后病情加重，八月驾崩
- 葬于德陵

**崇祯帝（明思宗朱由检，1611-1644）**：
- 在位17年，享年33岁
- 死因：甲申之变——崇祯十七年（1644年）三月十九日凌晨，李自成农民军攻入北京内城，崇祯出逃未果，返回宫中令皇后自尽、砍伤公主后，与太监王承恩至煤山（景山）寿皇亭，以发覆面，自缢而死
- 遗言写于衣襟："朕凉德藐躬，上干天咎，致逆贼直逼京师，皆诸臣误朕。朕死，无面目见祖宗，自去冠冕，以发覆面。任贼分裂，无伤百姓一人。"
- 葬于思陵（十三陵中最小，面积仅6,500平方米。崇祯本无陵寝，思陵原为其田贵妃墓，后由李自成和清朝先后扩建）
