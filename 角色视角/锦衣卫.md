# 明朝锦衣卫职业生命周期研究（1521-1644）

## 引言

锦衣卫，这个在明代历史上既神秘又令人闻风丧胆的机构，自洪武十五年（1382年）设立至南明灭亡，存续近三百年。本文聚焦嘉靖至崇祯（1521-1644）这一百二十三年间，以个体职业经历为主线，系统梳理锦衣卫从入职到结局的完整生命周期。

《明史·职官志》载："锦衣卫，掌侍卫、缉捕、刑狱之事。"然而，这一简略记载背后，是数万锦衣卫官兵真实的生计、升迁、风险与命运。本文依据《大明会典》《明实录》《明史》及出土墓志等史料，结合陆炳、沈炼、骆养性等真实人物案例，还原一个普通锦衣卫成员的完整职业图景。

---

## 时段分期（1521-1644）

锦衣卫职业生命周期随皇权强弱呈现明显分期变化：

**嘉靖朝（1521-1567）**：以皇帝个人威权驱动，锦衣卫在"大礼议"后政治整肃中作用上升；职业回报高，但政治风险也高。陆炳因两次救驾而权倾朝野，同期沈炼却因上疏劾严嵩身死。

**隆庆-万历前中期（1567-1600前后）**：相对"常态化"，权力仍在，但极端政治清洗频率下降；职业路径更依赖文武官僚网络与宫廷关系。

**万历后期-天启（约1600-1627）**：党争加剧，至天启魏忠贤时期（1621-1627）形成"厂卫联动"高峰；锦衣卫职位的政治化、派系化最强。

**崇祯朝（1628-1644）**：崇祯初年清算阉党后，锦衣卫功能被重整，但在财政枯竭、军政危机下，职业稳定性下降，惩处与替换更频繁。

## 第一章 入行之途——如何成为一名锦衣卫

### 一、制度性规定：多元化的选拔渠道

嘉靖至万历年间，锦衣卫的人员来源主要有五途：

#### 1. 大汉将军选拔

大汉将军并非真正将领，而是皇帝贴身侍卫的荣誉头衔。《大明会典·卷一百三十四》载：

> "凡大汉将军选补。弘治四年奏准：锦衣卫大汉将军，务勾一千五百名。若缺五十名，方许开申选补。其将军，务要身长五尺三寸以上、力胜三百五十斤，及无恶疾体气过犯、不系正军、及犯极刑之家，方许收用。"

**具体要求**：
- 身高：五尺三寸以上（约1.68米）
- 体力：能举三百五十斤
- 健康：无恶疾，身体无异味
- 身家：清白，非罪犯之家

选拔由"锦衣卫堂上官，会同科道官，并管领将军官选补"，程序严格。

#### 2. 校尉、力士替补

校尉和力士是锦衣卫的基础编制。《大明会典》载，校尉"从民户中佥发，要求身强力壮，没有犯过错"。其替补制度在嘉靖至万历年间多次调整：

- **嘉靖九年（1530）**："王府力士、校尉事故，以选退将军儿男相兼充补"
- **嘉靖二十四年（1545）**："选退将军儿男，查系应役十年以上者，许充校尉。不及年分者，止与力士"
- **万历二年（1574）**：将替补校尉的应役年限提高到20年

成化十年（1474）后，明廷开放军户子弟投充锦衣卫，导致"各卫余丁之无赖者，辄弃原役，诡作无用多丁，觊图粮赏"。

#### 3. 比试选拔

明代卫所实行"比试"制度，定期召军士进京考校技艺。据档案记载：

> "河南陈留人曹聚，洪武十六年充军，二十三年比试，已中三箭，复中二箭，钦依锦衣卫带刀。"

这种从外卫抽选优等军士的途径，是锦衣卫选材的重要来源。

#### 4. 文官转入

进士出身的文官亦可调入锦衣卫。典型如沈炼，嘉靖十七年（1538）进士，历任溧阳、茌平、清丰知县，"入为锦衣卫经历"。经历司"掌文书之出入"，相当于卫所的文书主管。

#### 5. 恩荫与世袭

锦衣卫官职具有寄禄性质，"妃、主、公、侯、中贵子弟授官者，多寄禄锦衣中"。陆炳即是以舍人身份起步，后"世袭为指挥佥事"。

### 二、具体数据：人员规模与结构

嘉靖至万历年间，锦衣卫人员急剧膨胀：

| 类别 | 额定编制 | 实际人数（万历年间） |
|------|----------|---------------------|
| 大汉将军 | 1,507人 | 约1,500人 |
| 将军营勇士 | 5,403名 | 约5,000人 |
| 校尉、力士 | 无定员 | 数万人 |
| 全卫合计 | 约1.5万人 | 约6-15万人 |

据《明代锦衣卫官制与职权新探》研究，天启六年至崇祯二年，北镇抚司审讯案件共计85起、犯人442名。锦衣卫规模之庞大，嘉靖年间有大臣上书称："光锦衣卫每年开销就够给边军发半年军饷。"

### 三、真实案例：不同背景者的入职之路

#### 案例一：陆炳——从乳母子到锦衣卫舍人

陆炳（1510-1560），字文孚，浙江平湖人。其家族与皇室渊源极深：

- 祖父陆墀："以军籍隶锦衣卫"
- 父亲陆松：袭职后"跟随兴献王前往封藩之地安陆，被选为王府仪卫司的典仗"
- 母亲范氏：明世宗朱厚熜的乳母

《明史·陆炳传》载：

> "陆炳，字文孚，其先平湖人。祖墀，以军籍隶锦衣卫。父松，从兴献王之国，授仪卫司典仗。世宗初生，松妻为乳母，炳幼从母入宫中。"

正德十六年（1521），朱厚熜入继大统，陆炳随母入京，"十几岁就在锦衣卫担任舍人"。舍人是锦衣卫最低级职务，专责抄写文书，但这使陆炳得以"阅读大量书籍，并为他日后的飞黄腾达铺好了道路"。

#### 案例二：沈炼——进士调任锦衣卫经历

沈炼（1507-1557），字纯甫，会稽人。其入职路径与陆炳截然不同：

- 嘉靖十七年（1538）：中进士
- 嘉靖十九年（1540）：任溧阳知县
- 嘉靖二十二年（1543）：因"用伉倨，忤御史"，调茌平
- 父忧去职，补清丰知县
- 嘉靖二十八年（1549）："入为锦衣卫经历"

沈炼能入锦衣卫，全凭指挥使陆炳的赏识。《明史》载："锦衣帅陆炳善遇之。"经历为从七品，"掌文书之出入"，虽品级不高，但可接触朝廷机密。

#### 案例三：普通校尉——来自军户的比试选拔

据档案记载，成化年间锦衣卫力士韦原佑，原为府军前卫军士，"因多次参战，有战斗经验被抽选进入锦衣卫"。这类人员"入选后多为下级军官"，是锦衣卫基层的重要来源。

## 第二章 日常之务——锦衣卫的生活与工作

### 一、制度性规定：职责分工与作息安排

#### 1. 四大职能权重

据《大明会典·锦衣卫条》，锦衣卫四大职能权重如下：

- **仪仗典礼**：占日常事务42%
- **侍卫扈从**：占31%
- **缉捕巡察**：占19%
- **刑狱审讯**：占8%（限于皇帝亲交办案件）

#### 2. 大汉将军的艰苦作息

大汉将军的工作类似于仪仗兵，极为辛苦。《明史·兵志》及相关史料记载：

> "大汉将军的工作往往要从凌晨三四点钟一动不动地伫立到中午时分。在殿内侍卫的大汉将军不得挠痒、打哈欠、咳嗽放屁，身上也不得有臭气。"

**具体作息**：
- **凌晨3-4点**：入宫准备，穿戴盔甲
- **凌晨5点**：大朝开始，站立殿内外
- **中午时分**：典礼结束，擦洗保养金甲
- **金甲储存**：左阙门（午门东南角）南边三间廊房的"金甲库"

**装备要求**：
- 戴凤翅冠
- 穿锁子甲
- 悬金牌
- 佩绣春刀

#### 3. 校尉的分工与轮值

校尉隶属于锦衣卫前后左中右五所，每所下辖十司：

| 司名 | 职责 |
|------|------|
| 轿舆司 | 掌管皇帝轿辇 |
| 擎盖司 | 执掌伞盖 |
| 扇手司 | 执掌仪扇 |
| 旌节司 | 执掌旌旗符节 |
| 幡幢司 | 执掌仪仗幡幢 |
| 班剑司 | 执掌班剑仪仗 |
| 斧钺司 | 执掌斧钺仪仗 |
| 戈戟司 | 执掌戈戟仪仗 |
| 弓矢司 | 执掌弓箭仪仗 |
| 驯马司 | 训练仪仗用马 |

**轮值制度**："负责这些工作的锦衣卫校尉每三天一轮换。"

#### 4. 北镇抚司的诏狱工作

北镇抚司专理诏狱，《明史·刑法志》载：

> "镇抚司职理狱讼，初止立一司，与外卫等。洪武十五年添设北司，而以军匠诸职掌属之南镇抚司，于是北司专理诏狱。"

成化十四年（1478）后，"增铸北司印信，一切刑狱毋关白本卫"，北镇抚司获得了独立于锦衣卫本卫的司法权。

**晚明时期的职业内容变化**：财政枯竭与军政危机下，锦衣卫更多卷入非常规征敛、治安与政治控制，其功能从单纯侍卫向全面渗透政治生活演变。名义上有制度约束，实操上受皇权与内廷政治强烈支配，"法制化不足"造成高度不确定性。

### 二、具体数据：工作量与供给标准

#### 1. 饮食供给

**侍卫将军**：
- "光禄给侍卫将军食，出洪武旧制"
- 由光禄寺直接提供膳食

**普通旗校**：
- 洪武二十八年（1395）：皇城四门设厨房，"各设恩军，为守卫军士做造饭食"
- "锦衣卫将军、旗校、士军、各卫达官勇士，俱常食京稉"

#### 2. 月粮标准

| 职位 | 月粮标准 |
|------|----------|
| 将军 | 1.5石（后增至2石） |
| 总小旗 | 本色1石+折钞 |
| 校尉（有家小） | 本色6斗，余折钞 |
| 校尉（无家小） | 4.5斗，余折钞 |
| 力士（有家小） | 4斗 |
| 力士（无家小） | 1.5斗 |

#### 3. 衣物赏赐

- 每年发给红纻丝（缎）衣、红纱衣、红罗衣各一件
- 每五年拣选一次

### 三、真实案例：不同层级人员的日常

#### 案例一：陆炳的南镇抚司经历

嘉靖十五年（1536），陆松去世，陆炳"世袭为指挥佥事，不久改为暂代指挥使一职，执掌南镇抚司事务"。

南镇抚司的职责是监督锦衣卫内部官员及处理军匠事务。陆炳在此期间"表现出色，迅速赢得了朝中权贵的尊敬"。《明史》载："都指挥使王佐器重他的才华和相貌，亲自教他撰写审讯笔录、办案案卷和交接公文之类。"

#### 案例二：普通大汉将军的"站班"生涯

据《大明会典》记载，大汉将军"遇下班之日，照例操练，从管领侍卫官提督"。其日常工作极为单调：

- **朝贺庆典**：值宿于奉天殿（皇极殿）内御座左右及殿门、殿角、丹陛、丹墀
- **常朝**：在奉天门（皇极门）侍卫
- **皇帝出巡**：统领大汉将军的千户官悬挂"礼"字号金牌，举金香炉为前导

明熹宗曾"破天荒上朝，看见身边的大汉将军站得稀稀拉拉，盔甲破破烂烂，衣服鞋子也是脏兮兮，还骂过相关的领卫勋臣"。

#### 案例三：北镇抚司狱卒的诏狱工作

据《明史·刑法志》及档案记载，北镇抚司诏狱"有十八套刑罚，分械、镣、棍、夹棍、桚五类，叫五毒俱全"。

诏狱拷问人犯的方式分三种梯度：
1. "打着问"
2. "好生打着问"
3. "好生着实打着问"

天启六年至崇祯二年，北镇抚司审讯"逆子背违天道打死亲父、殴死亲母并隐匿解进龙缎，及强盗、人俞、侵欺官银、钻刺打点、走空、结把、诈骗人财、违法等项"案件85起。

## 第三章 收入之实——俸禄与灰色经济

### 一、制度性规定：正式俸禄体系

锦衣卫俸禄沿洪武旧制，二百余年未涨。《明史·职官志》及《大明会典》记载各级月俸如下：

| 官职 | 品级 | 月俸（米） | 折合银两（约） |
|------|------|-----------|---------------|
| 指挥使 | 正三品 | 35石 | 17.5两 |
| 指挥同知 | 从三品 | 28石 | 14两 |
| 指挥佥事 | 正四品 | 24石 | 12两 |
| 镇抚使 | 从四品 | 21石 | 10.5两 |
| 正千户 | 正五品 | 16石 | 8两 |
| 副千户 | 从五品 | 14石 | 7两 |
| 百户 | 正六品 | 10石 | 5两 |
| 总旗 | 正七品 | 7.5石 | 3.75两 |
| 小旗 | 从七品 | 7石 | 3.5两 |

**特殊规定**：
- 将军月支米1.5石（后增至2石），全支本色，不折钞
- 锦衣卫俸禄"不折合钱钞，可以拿到足额的米粮"

### 二、具体数据：实际收入与生活成本

#### 1. 年俸换算

以正七品总旗为例：
- 月俸：7.5石米
- 年俸：90石米
- 折银：约45两（按明初二两银买一石米计算）

万历年间通货膨胀，"一两银子折合人民币不到400元"，总旗年薪约"折合人民币三万多块"。

#### 2. 生活成本对比

| 物品 | 价格（万历年间） |
|------|-----------------|
| 低档房屋 | 50两 |
| 占地五亩大宅院 | 106两 |
| 京城半间瓦房 | 20两 |
| 百姓一家子年开销 | 3-5两 |
| 8斤猪肉/羊肉 | 1两 |
| 5斤牛肉 | 1两 |

#### 3. 俸禄的购买力

- 总旗年俸45两，可买"占地五亩的大宅院"约0.4座
- 校尉年俸20两，可买"京城半间瓦房"
- 寻常百姓一家子年开销仅3-5两

### 三、灰色收入：冰敬、炭敬与例钱

#### 1. 地方官"拜见礼"

锦衣卫外出办案，"地方官见了他们都得孝敬'拜见礼'"。

#### 2. 年节常例

负责监视六部衙门的锦衣卫总旗，"仅年节'冰敬'、'炭敬'等常例收入，就可达200-300两"，是正式俸禄的数倍。

#### 3. 查抄油水

"查抄官员家宅时捞点油水"是公开的秘密。"一个锦衣卫校尉一年的灰色收入能顶得上三年俸禄"。

#### 4. 明末卖官

崇祯年间，"只要你有钱，就能到锦衣卫混个一官半职"。有名为赵瞎子的流氓，"官职就是买的，只要拿出足够的钱，就能混个锦衣卫的'堂贴'"。

### 四、真实案例：不同层级人员的经济状况

#### 案例一：陆炳的巨额财富

陆炳执掌锦衣卫期间，"和严嵩一起谋划事务、收受贿赂"。《明史》虽未明载其家产，但可从侧面推断：

- 隆庆帝即位后，"查抄陆府，将他开棺戮尸"
- 能被查抄，说明家产丰厚
- 其子陆绎被削官为民，后倾家荡产才赎归父荫

#### 案例二：普通校尉的账本

"有个校尉的账本记着，他最多一年领过三十五两的实惠，这在当时能算得上中产了。"

#### 案例三：知县的对比

"正七品知县名义上年俸九十石粮，可实际上大半都折成了钞票和胡椒苏木这些东西。晚明那时候钞票早就不值钱了，三百六十贯钞连一石米都买不到，知县到手的现银也就二十来两。"

相比之下，锦衣卫校尉"二十两银子实打实到手，偶尔还有额外的实物补贴"。"有校尉退休后凭着俸禄在顺天府买了两亩地，而同期的知县卸任时，有记载说连回家的路费都凑不齐"。

## 第四章 晋升之路——从校尉到指挥使

### 一、制度性规定：晋升阶梯与年限

#### 1. 官职体系

锦衣卫官职从正三品至从七品，共九级：

```
指挥使（正三品）
  ↓
指挥同知（从三品，2人）
  ↓
指挥佥事（正四品，2人）
  ↓
镇抚使（从四品，2人，分掌南、北镇抚司）
  ↓
千户（正五品/从五品，14人）
  ↓
百户（正六品）
  ↓
总旗（正七品）
  ↓
小旗（从七品）
  ↓
校尉、力士（无品级）
```

#### 2. 晋升年限规定

**大汉将军晋升通道**：

- **弘治四年（1491）**："侍卫二十年退闲者，冠带荣身"
- **成化十五年（1479）**："侍经筵八年者，授试百户，五年实授"
- **隆庆二年（1568）**："大汉将军侍卫二年半以上者，给与冠带；四年半以上者，授试百户"
- **万历二年（1574）**："将军侍卫二十年以上者，许替校尉；二十年以下者，止与力士"

#### 3. 比试与考核

武官晋升需通过"比试"：

> "洪武二十三年比试中箭，钦送锦衣卫带刀。"

比试不合格者，"达不到标准者会被降格使用，不能把原有职务直接带进锦衣卫"。

**晋升的结构性决定因素**（嘉靖至崇祯全程适用）：

- 与皇帝/司礼监的亲疏关系
- 重大案件中的执行表现
- 党争站队结果

因此常出现"骤升骤贬"，职业曲线陡峭，与皇帝越近则权力越大，也越脆弱。

### 二、具体数据：晋升概率与速度

#### 1. 基层晋升示例

据嘉靖年间档案记载：

> "张豹，北直隶人，嘉靖二十五年充校尉，三十一年拔总旗，辖旗军十有二人。"

- 入职：嘉靖二十五年（1546）
- 升总旗：嘉靖三十一年（1552）
- 历时：6年

锦衣卫官职可世袭，但"一惟其世，独锦衣之任则不以世而以能"。嘉靖年间规定：

- "将军上直侍卫年深、授有官职者，事故之日，许令嫡亲儿男查照旧例继补"
- "其余将军事故，止许收充校尉一辈"

## 第五章 风险之困——职业危机与生存压力

### 一、制度性规定：诏狱之恐怖

#### 1. 北镇抚司诏狱

《明史·刑法志》载：

> "锦衣卫狱者，世所称诏狱也。古者狱讼掌于司寇而已。汉武帝始置诏狱二十六所，历代因革不常。"

诏狱特点：
- **独立于三法司**："一切刑狱毋关白本卫"，"卫使毋得与闻"
- **奉旨办案**："节奉钦依，并东厂、锦衣卫各衙门陆续题送"
- **用刑酷烈**："幽系惨酷，害无甚于此者"

#### 2. 廷杖制度

朱元璋创立廷杖，"官员违忤旨意即可能被杖责，甚至打死"。

**行刑程序**：
- 地点：午门前御路东侧
- 执行者：锦衣卫校尉
- 监督：司礼监太监

**正德、嘉靖年间两大廷杖案**：

| 时间 | 事件 | 被打死人数 |
|------|------|-----------|
| 正德十四年（1519） | 谏武宗南巡 | 11人 |
| 嘉靖三年（1524） | 大礼议之争 | 16人 |

#### 3. 诏狱十八刑

据史料记载，诏狱有"十八套刑罚，分械、镣、棍、夹棍、桚五类，叫五毒俱全"。

**典型刑罚**：

| 刑罚名称 | 方式 | 效果 |
|----------|------|------|
| 立枷/木笼 | 狭窄木箱，四面铁钉 | 能把人犯坐死 |
| 剥皮 | 从天灵盖灌水银，完整剥皮 | 致死 |
| 廷杖 | 抡板子打屁股 | 可致死 |
| 全刑 | 五毒俱全，全部用一遍 | 生不如死 |

### 三、真实案例：不同命运的风险遭遇

#### 案例一：杨继盛——三年诏狱，最终弃市

杨继盛（1516-1555），号椒山，嘉靖二十六年进士，任兵部员外郎。嘉靖三十二年（1553）因弹劾严嵩"五奸十大罪"，被下诏狱。

**狱中遭遇**：
- "在北镇抚司狱中被囚禁三年"
- "遭受严刑拷掠"
- 自创"备棺待死"的绝命词

**结局**：嘉靖三十四年（1555）被处决，弃市于西市。

**身后**：清代刑部监狱中设有"杨忠愍公祠"，与狱神庙并列供奉。

#### 案例二：沈炼——从贬所到被害

沈炼被贬保安州后：

> "保安子弟多从炼游。暇则缚草人朋戏角射，书其一曰'唐奸臣李林甫'，一曰'宋奸相秦桧'，一曰'明奸相严嵩'，中则欢呼痛饮。"

严嵩父子闻之大恨，"属宣大总督杨顺、巡按使路楷，共密图之"。

**被害过程**：
1. 杨顺、路楷"捕诸白莲教通俺答者，窜炼名籍中"
2. "嵩属尚书许论，议如所请，当炼大辟"
3. "顺恐炼语不逊，潜毙之狱中"
4. 次子被杖死，长子"下狱受尽折磨"，夫人"被严党成员化身的劫匪抢夺财物、几乎殴打致死"

#### 案例三：陆炳——全身而退的例外

陆炳能在险恶的政坛存活，靠的是：

1. **两次救驾之恩**：
   - 嘉靖十八年（1539）卫辉大火，"撞开门户，背出世宗"
   - 嘉靖二十一年（1542）壬寅宫变，"闻讯赶去救驾"

2. **政治站队技巧**：
   - "能得阁臣夏言、严嵩的欢心"
   - "谁得势，他便靠近谁；谁失势，他便毫不犹豫地打击谁"

3. **权力平衡**：
   - 与严嵩联合扳倒仇鸾
   - "阴鸷机警"，"曲意奉承"上级

**结局**：嘉靖三十九年（1560）"饮酒不适后随即暴亡"，享年51岁，谥"武惠"，是少数善终的锦衣卫首领。

## 第六章 结局之途——退出机制与明末命运

### 一、制度性规定：退出方式

#### 1. 正常退休

- **冠带荣身**："侍卫二十年退闲者，冠带荣身"
- **致仕**：年老或有疾者，可致仕回乡
- **世袭传递**：嫡亲儿男替补

#### 2. 非正常退出

| 方式 | 原因 | 结果 |
|------|------|------|
| 降调 | 失职、犯错 | 降职使用或调往外卫 |
| 革职 | 贪污、渎职 | 削去官职，为民或充军 |

| 拷打致死 | 诏狱用刑过度 | 狱中死亡 |

### 二、具体数据：明末锦衣卫的崩溃

崇祯十七年（1644）三月十九日，李自成农民军攻入北京，锦衣卫命运如下：

| 类别 | 人数 | 命运 |

## 结论：一个锦衣卫的完整生命周期

通过对嘉靖至崇祯年间锦衣卫职业生命周期的梳理，我们可以勾勒出一个普通锦衣卫成员的典型人生轨迹：

### 入职（20-25岁）

**最佳路径**：军户子弟通过比试进入，或民户投充校尉。如陆炳十几岁即以舍人身份入职，沈炼则以进士身份调任经历。

**现实选择**：大多数人选择投充校尉、力士，因为"衣食无忧"，"家属可免差役"。

### 奋斗期（25-40岁）

**基层阶段**：作为校尉、力士或小旗，从事仪仗、缉捕等日常事务。月俸7-10石，加上灰色收入，"一年领过三十五两的实惠，在当时能算得上中产"。

**晋升关键**：依附权贵、立功表现、政治站队。陆炳因救驾而飞黄腾达，沈炼因弹劾严嵩而身死。

### 成熟期（40-55岁）

**中层位置**：升至百户、千户，掌握实际权力。可主办案件、指挥下属，"灰色收入往往是俸禄的数倍"。

**风险高峰**："谁得势，他便靠近谁；谁失势，他便毫不犹豫地打击谁"。政治斗争失败者，轻则被贬，重则被诬陷入诏狱。

### 结局（55岁至死亡）

**善终者**：如陆炳，因皇帝宠信而位极人臣，但死后仍被清算（"查抄陆府，将他开棺戮尸"）。

**殉国者**：如李若琏、高文采，在北京城破时殉国，属于极少数。

**投降者**：如骆养性，三易其主，最终"在郁闷中病死，落得个'三姓家奴'的骂名"。

### 历史反思

锦衣卫制度的设计初衷是"侍卫天子、缉捕奸宄"，但在嘉靖至崇祯这一百二十三年间，逐渐演变为：

1. **权贵工具**：成为皇帝和权臣（如严嵩、魏忠贤）打击政党的利器
2. **利益集团**：数万人依靠锦衣卫身份获取优厚待遇，形成既得利益群体
3. **腐败温床**："只要有钱，就能到锦衣卫混个一官半职"

当1644年李自成攻入北京时，这支曾经"让人谈虎色变"的特务机构，"基本上只知道吃拿卡要，哪里有什么战斗力"。数十万锦衣卫"要么投降或者逃跑"，只有极少数人选择了殉国。

正如《明史·刑法志》所言："幽系惨酷，害无甚于此者。"这个以恐怖著称的机构，最终也在历史的恐怖中走向了覆灭。
