# 一条鞭法

## 概述

一条鞭法是明朝中后期最重要的赋役制度改革，核心内容是将田赋与徭役合并为一条，统一折银征收，由官府直接收解（"官收官解"），取代此前的实物征收和力役征发。该法经嘉靖年间地方试点，到万历九年（1581）由张居正推广为全国通行制度。

---

## 一、改革的具体内容

### 核心要点

1. **合并赋役**：将原来分别征收的田赋（夏税、秋粮）和各种徭役（里甲正役、均徭、杂泛）合并为一条
2. **折银征收**：赋役一律折算为白银缴纳（"计亩征银"），部分地区保留少量本色
3. **官收官解**：取消"民收民解"制度，改由官府直接征收、解运
4. **力役改银**：原需亲身服役的力差，改为出银代役，官府以银雇人承应
5. **计亩摊派**：将部分丁役摊入田亩，按田亩面积分摊

### 丁粮比例

各地丁役与田粮的折算比例不一：
- 有的地方"丁六粮四"（丁占六成、粮占四成）
- 有的地方"粮占四分之三"
- 具体比例由各地因地制宜决定

## 四、关键人物的贡献

### 桂萼（？-1531）

- 嘉靖九年以大学士身份首次提出"编审徭役"方案
- 为一条鞭法的理论奠基者

### 庞尚鹏（1524-1581）

- 嘉靖四十年后任浙江巡按御史，多次改革赋役制度
- 万历四年任福建巡抚，力行一条鞭法
- 是一条鞭法从试点走向推广的关键推动者

### 海瑞（1514-1587）

- 任应天巡抚期间在南直隶大力推行一条鞭法
- 以刚直著称，推行有力

### 张居正（1525-1582）

- 万历六年（1578）下令清丈全国土地
- 万历九年（1581）将一条鞭法确立为全国通行制度
- 在其执政期间（万历元年至十年），一条鞭法推广至全国大部分省份
- 清丈结果：全国新增田地约2.8亿亩

### 其他人物

- **陶谐**：嘉靖十年在江西首行类似改革
- **梁材**：嘉靖十年提出革除赋役弊病方案
- **傅汉臣**：嘉靖十年首次使用"一条编法"名称
- **刘光济**：隆庆二年在江西全省推行

## 五、执行效果的数据

### 太仓银库收入变化

| 时期 | 太仓银库收入（两） |
|------|------------------|
| 正德初年 | 149万 |
| 嘉靖十一年（1532） | 243万 |
| 嘉靖二十八年（1549） | 295.7万 |
| 隆庆元年（1567） | 201.4万 |
| 万历五年（1577） | 435.94万 |
| 天启时期 | 327万 |

万历五年较隆庆元年增长116%，效果显著。

### 清丈成效

- 清丈前：全国田地约500余万顷
- 清丈后：约700余万顷
- 新增：约2.8亿亩（约300万顷）

### 张居正改革的综合效果

据《明史纪事本末》："力筹富国，太仓粟可支十年，冏寺积金至四百余万。"

## 六、各地执行差异

### "南橘北枳"现象

一条鞭法在各地的执行程度参差不齐：

1. **南方推行较早较彻底**：南直隶、浙江、江西等赋役繁重地区，改革动力强
2. **北方推行较晚较粗放**：北方省份多在万历十五年以后才推行
3. **丁粮比例各地不一**：有的丁六粮四，有的粮占四分之三
4. **折银程度不一**：部分偏远地区仍保留较多本色征收

### 张居正去世后的回潮

万历十年（1582）张居正去世后：
- 部分已清丈的田地数据被地方豪绅推翻
- 一些地方出现赋役分征的回潮
- 但总体框架（折银征收）未被根本推翻

### 持续影响

- 一条鞭法确立了白银作为主要赋税缴纳媒介的地位
- 推动了明代社会的白银货币化进程
- 清代的"摊丁入亩"是其思路的进一步延伸

## 七、桂萼首倡的背景

### 桂萼其人

桂萼（?-1531），字子实，安仁（今江西余江）人。嘉靖初年以支持"大礼议"获世宗信任，累迁至大学士。

### 首倡动机

嘉靖九年（1530），桂萼以大学士身份向世宗提出"编审徭役"方案，其核心思路包括：
- 痛感各地徭役编审不均，里甲长从中舞弊
- 主张按丁粮实际数目统一编审，杜绝编审中的上下其手
- 提出将各类杂役合并折银，简化征收手续
- 此方案虽非完整的一条鞭法，但奠定了"合并折银"的理论基础

### 学术争议

学界对桂萼是否为一条鞭法"首创者"存在争议：
- 支持者认为桂萼最早系统提出了合并赋役、折银征收的设想
- 反对者认为此前已有地方官在局部试行类似做法（如弘治年间的"征一法"）
- 多数学者倾向于认为一条鞭法是多地实践逐步汇聚的产物，桂萼的贡献在于将地方经验上升为制度性方案

## 八、一条鞭法的三大要点详述

### 要点一：合并赋役

改革前，田赋与徭役分别征收，各有名目：
- 田赋含夏税、秋粮，又分本色、折色
- 徭役含里甲正役、均徭银差、均徭力差、杂泛差役
- 各项名目不下数十种，征收手续繁琐

一条鞭法将上述所有名目合并为**一条**，统一编入赋额，不再逐项分征。

### 要点二：统一折银

- 所有赋役项目一律折算为白银缴纳（极少数如白粮、漕粮仍征本色）
- 统一了计量单位，便于核算和监督
- 从根本上改变了明初以来实物缴纳为主的格局
- 推动了白银成为国家财政的基础货币

### 要点三：官收官解

- 改革前：粮长（里长）负责催征，将税粮运送至指定仓库（"民收民解"）
- 改革后：地方官府直接设柜征收白银，由官府负责解运（"官收官解"）
- 减少了里甲长层层盘剥的空间
- 但也增加了地方官吏借机加征"火耗"的机会

## 九、南北执行差异分析

### 南方特点

- 折银比例高，一般达80-90%以上
- 丁役入田比例较大，田亩承担更多赋役
- 商品经济发达，白银流通充足，折银容易执行
- 苏松地区因税负极重，一条鞭法对减轻繁琐手续有实际帮助

### 北方特点

- 折银比例较低，部分地区仅50-60%
- 丁银仍按人丁单独征收的比例较大
- 银钱交换不便，部分贫困地区出现"折色不便"问题
- 有些州县直到崇祯年间才正式实行

### 边远地区

- 云贵、甘肃等地推行最晚（万历十五至十七年）
- 实际执行中保留较多本色和力役
- 部分少数民族地区基本未实行

---

## 存疑

1. 各省推行一条鞭法的精确时间，文献记载不一，尤其是北方省份的具体起始年份有待进一步考证
2. 桂萼嘉靖九年的方案与后来实行的一条鞭法之间的关系，学界存在争议
3. 海瑞在南直隶推行一条鞭法的具体政策措施和效果，相关史料较少
4. 一条鞭法推行后实际执行率（真正折银的比例）的量化研究尚不充分
5. "征一法"等前驱改革与一条鞭法的源流关系有待更深入梳理
