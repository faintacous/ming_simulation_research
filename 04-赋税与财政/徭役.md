# 徭役

## 概述

明朝徭役制度以里甲制为基础，分为里甲正役、均徭和杂泛差役三大类。此外还有以户籍为依据的"配户当差"制度，将全国人户按职业编为民户、军户、匠户、灶户等不同名色，分别承担对应的专门差役。明朝徭役的总趋势是从力役到银役、从人身依附到货币化，最终汇入一条鞭法的改革洪流。

---

## 一、里甲制

### 编制方式

洪武十四年（1381）在全国推行里甲制度：

- **基本单位**：每110户编为1里
- **里长**：由丁粮最多的10户轮流担任
- **甲**：其余100户编为10甲，每甲10户
- **甲首**：每甲选1人为甲首

### 十甲轮役的具体方式

- **轮值周期**：10年一轮，称"排年"
- **年度安排**：每年由1名里长率领当值那一甲的10名甲首应役
- **排序标准**：先后顺序根据各户丁粮多寡预先编排，丁粮最多者先轮到
- **称谓**：当年在役者称"见年里长"，休歇年份称"排年"

### 里甲正役内容

里长与甲首的核心职能为"催征钱粮，勾摄公事"，具体包括：

1. **赋税催征**：催促本里各户按时完纳田赋
2. **徭役编派**：协助官府编派各类徭役
3. **户籍管理**：登记人口、田产变动
4. **地方事务**：调解纠纷、维护治安
5. **官差应付**：官员过境时供应食宿、车马等（后演化为沉重负担）

此外设"里老人"（又称耆老），负责教化、劝农及审理民间轻微案件。

## 二、均徭法

### 产生背景

明初里甲正役之外的杂役，原为"临期量力差遣"，由里长编佥，弊端丛生。成化年间开始推行均徭法加以改革。

### 三则编审（上中下户）

均徭法的核心是按户等编审：

1. 依照实在丁粮重新调查，将各户分为**上户、中户、下户**三等
2. 按户等高低分派轻重不同的差役
3. 编佥之权由里长改归有司（官府）
4. 依据实际丁粮制订《均徭文册》，取代原先仅依据黄册的方式

### 银差与力差

均徭法将差役分为两类：

- **力差**：需亲身服役的差役，如皂隶、马夫、门子、库子、斗级、禁卒等衙门差役
- **银差**：纳银代役的差役，由官府用所收银两雇人应役

力差一般派上户，银差派中下户。后来发展趋势是力差也逐步折银。

## 三、杂泛差役

### 配户当差制的种类

明朝实行"配户当差"制度，根据户籍编定将人户配属不同差役。据王毓铨研究，明代全国役户超过80种，主要4种为：

| 户籍类型 | 承担差役 | 说明 |
|---------|---------|------|
| 民户 | 里甲正役、均徭、杂泛 | 最普遍的户籍，承担一般赋役 |
| 军户 | 军役 | 每户出一丁赴指定卫所当军，世袭 |
| 匠户 | 匠役 | 分轮班匠和住坐匠，为官府服手工劳役 |
| 灶户 | 灶役 | 专门煮盐，隶属于各盐场 |

### 匠户的两类

- **轮班匠**：归工部主管，每三年赴京服役三个月（后改为四年一班），嘉靖四十一年（1562）改为一律纳银代役，每名每班征银银四钱五分
- **住坐匠**：长期在京师各监局服役，每月上工十天

### 其他杂泛差役

杂泛差役种类繁多，各地因行政需要而千差万别。综合各类文献，主要可分为以下几大类：

**一、衙门差役类**：
- 皂隶（衙门杂役）、门子（看门）、禁卒（看守囚犯）
- 库子（仓库管理）、斗级（量粮）
- 膳夫（官厨）、仵作（验尸）

**二、治安巡防类**：
- 弓兵（巡逻缉捕）、民壮（地方武装，嘉靖年间各县编额30-100名不等）
- 总甲（地方治安负责人）、巡检司弓兵

**三、驿递交通类**：
- 铺兵（传递公文）、驿夫（驿站服务）
- 马夫（喂养驿马）、水夫（驿站船只）
- 递运所夫（运输物资）、脚夫（搬运）
- 站坝夫（运河沿线维护）

**四、基层管理类**：
- 坊长、厢长（城市基层管理）
- 粮长（催征田赋）、塘长（水利管理）
- 耆老/里老人（劝农教化、调解纠纷）

**五、官府采购类**：
- 铺户买办（为官府采购物资的商户）
- 祗应人户（供应官府日常物品）

**六、其他专项差役**：
- 养马户、养象户
- 机兵（操作机械、守城器具）
- 纸户（造纸供应）、柴薪夫（伐薪供应）

### 嘉靖至万历的杂泛差役数据

据沈榜《宛署杂记》记载，万历年间北京宛平县的杂役负担：
- 每年实征铺行银（代役银）**10,641.31两**
- 万历十六年（1588）审编时，宛平、大兴两县共编银**5,136.3两**

嘉靖四十五年（1566），京城铺户买办之役改革，将铺户按资产分为三等九则：
- 上上则：征银9钱
- 中则：每则各递减1钱
- 下下则：征银1钱

### 驿递负担的恶化

驿递系统是明中后期最沉重的杂泛差役之一：
- 全国驿站约**1,936处**（含水驿、马驿、递运所等）
- 每站需配备驿夫、马夫、水夫等数十人
- 过境官员的食宿、车马、人夫费用均由地方里甲承担
- 嘉靖年间，驿递负担在部分地区已超过正役数倍
- 万历年间裁撤部分驿站（后因李自成起义而被后人议论）

## 四、黄册编造

### 编造依据

洪武十四年（1381）颁布诏令，在全国范围内编造赋役黄册。

### 十年一造的规定

- 每十年大造一次
- 首次编造：洪武十四年（1381）
- 此后每逢辛年（洪武二十四年、永乐九年……）大造

### 具体流程

1. **发单**：政府颁发《清册供单》（又称"户帖"）表格
2. **填报**：甲首将表格分发给各户填写，登记户内人口、田宅及资产情况
3. **逐年更新**：每年填报一次，由地方官核实丁口、田宅的变动
4. **甲首汇总**：甲首将本甲各户清册汇总
5. **里长汇总**：里长将本里各户清册汇总成册
6. **逐级上报**：依次上报至县、府、布政使司
7. **呈送户部**：布政使司汇总后上呈户部

### 登载格式

采用"四柱式"记录法：

| 项目 | 说明 |
|------|------|
| 旧管 | 上次大造时的原有数目 |
| 新收 | 十年间新增的数目（细分为"正收""转收"） |
| 开除 | 十年间减少的数目（细分为"正除""转除"） |
| 实在 | 本次大造的实有数目 |

### 册籍份数与保管

- 一式四份：一份上呈户部，布政司、府、县各存一份
- 户部的黄册存于南京玄武湖（后湖）黄册库
- 至万历三十年（1602），后湖黄册库有册库667间，收藏黄册达153万余册

### 黄册的衰败

嘉靖以后，黄册编造流于形式：
- "攒造不实"，抄袭旧册
- 户口脱漏严重
- 后湖黄册多有虫蛀腐烂

## 五、徭役负担的量化数据

### 里甲正役负担

- 每里110户，每年轮10户（1里长户+10甲首户）应役
- 10年轮一次，每户约每10年服正役1年

### 均徭负担

各地差异极大，以下为部分记载：

- 明中期以后，各地均徭银差折银数额从每丁数分到数钱不等
- 嘉靖年间苏州府等地均徭银每丁约银1-3钱

### 匠班银

- 轮班匠每名每班（四年一班）纳银4钱5分
- 住坐匠每月上工十天，无银代替

### 驿递负担

- 明中后期驿递系统负担极重
- 各地里甲需轮流供应过境官员食宿、车马、人夫
- 此项负担往往远超正役

## 存疑

1. 明代全国80余种役户的完整名录和各类具体义务，尚待系统整理
2. 均徭法推行的确切时间，各省不一，通说为成化至弘治年间，但各地起始年份的精确编年有待考证
3. 黄册编造质量下降的具体时间节点，学界尚有争议
4. 里甲正役中"官差应付"（供应过境官员）的实际负担量化数据缺乏系统统计
