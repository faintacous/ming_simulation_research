# 明朝白银货币化

## 概述

明代货币史经历了"铜钱时代""纸钞时代"和"白银时代"的转变。白银货币化发端于民间，而非来自官方的制度设计，是一个自下而上的过程。从洪武末年白银在民间非法流通，到隆庆元年（1567）法定确认"银钱兼使"，再到万历九年（1581）一条鞭法全面推行完成赋役折银，历时近两百年。白银货币化深刻重塑了明代的财政体制、经济结构和社会形态，使明朝经济深度嵌入全球贸易网络，也埋下了晚明"银荒"导致经济崩溃的隐患。

---

## 一、从实物经济到白银经济的五阶段演变

### 1.1 阶段划分

| 阶段 | 时间 | 特征 | 关键事件 |
|------|------|------|---------|
| **第一阶段：白银民间崛起** | 洪武末年—永乐（约1390s–1420s） | 宝钞制度下白银在民间非法流通 | 洪武三十年（1397）禁银令名存实亡；民间交易中白银已盛行 |
| **第二阶段：宝钞崩溃与银进** | 永乐—正统（1403–1449） | 宝钞信用瓦解，白银交易急增 | 正统元年（1436）南方田赋折银（金花银制度开始）；正统三年（1438）后白银交易占契约总数的**64%** |
| **第三阶段：白银全面取代宝钞** | 正统—成化（1436–1487） | 白银从非法货币变为官方认可的合法货币 | 成化年间90件契约**无一例外使用白银**；宝钞在民间流通中彻底绝迹 |
| **第四阶段：白银主币地位确立** | 成化—嘉靖（1487–1566） | 白银成为社会流通领域及国家实际认可的主币 | 嘉靖年间（16世纪40年代）白银货币化过程基本完成 |
| **第五阶段：法定化与全面渗透** | 隆庆—万历（1567–1620） | 白银获得法定货币地位，全面渗透国家财政 | 隆庆元年（1567）颁令"银钱兼使"；万历九年（1581）一条鞭法全面推行 |

[来源：中国经济史论坛-宝钞崩坏、白银需求与海外白银流入；光明日报-明代货币白银化问题认识误区辨正]

### 1.2 关键时间节点

- **洪武八年（1375）**：发行大明通行宝钞，禁止金银流通
- **正统元年（1436）**：南方田赋折银征收，金花银制度开始——这是白银进入国家财政体系的标志
- **正统年间**：宝钞贬值至千贯抵银一两（官定一贯抵一两），事实上崩溃
- **成化十七年（1481）**：规定每8文铜钱折银一分（银1两 = 铜钱800文）
- **隆庆元年（1567）**：明穆宗颁令"凡买卖货物，值银一钱以上者，银钱兼使；一钱以下止许用钱"——**首次以法权形式确认白银合法货币地位**
- **万历九年（1581）**：张居正全面推行一条鞭法，"银始独重于天下，百物皆取银为准矣"

### 1.3 驱动机制分析

白银货币化的驱动力是多元的：

| 驱动因素 | 说明 |
|---------|------|
| 宝钞崩坏 | 纸币信用丧失后的货币真空，白银作为替代品自然填补 |
| 财政体制内生需求 | 赋役折银化是行政效率的内在要求 |
| 私钱泛滥 | 明中叶私铸铜钱恶化，劣币驱逐良币，白银成为可靠替代 |
| 民间自发选择 | 白银具有稳定性、可分割性和保值性，民间长期偏好 |
| 海外白银涌入 | 16世纪中叶后日本和美洲白银大量流入，提供了货币化的物质基础 |

## 二、宝钞的发行和崩坏过程

### 2.1 发行制度

- **设立机构**：洪武七年（1374）设立**宝钞提举司**，下辖抄纸局、印钞局和宝钞库、行用库 [来源：维基百科-大明宝钞]
- **发行时间**：洪武八年（1375）三月，朱元璋下诏印制"大明通行宝钞"
- **印造周期**：规定每年三月至十月印造，所造纸钞由宝钞提举司解送内库收贮备用
- **核心缺陷**：**无准备金**、**无限额**、**强制推行**

### 2.2 面额种类

| 面额 | 发行时间 |
|------|----------|
| 壹贯 | 洪武八年（1375） |
| 伍百文、肆百文、叁百文、贰百文、壹百文 | 洪武八年（1375） |
| 伍拾文、肆拾文、叁拾文、贰拾文、壹拾文 | 洪武二十二年（1389）增发 |

共**十一种**面额。

### 2.3 法定比价

壹贯 = 白银一两 = 铜钱一千文 = 黄金0.25两 = 米一石 [来源：百度百科-大明宝钞]

### 2.4 发行量

| 时期 | 年均发行量 | 备注 |
|------|-----------|------|
| 洪武年间（24年） | 约**515万锭/年** | 24年总发行约**1.2–1.3亿锭** |
| 洪武二十三年（1390） | **1,500万锭**（一说1,400万锭） | 单年最高峰 |
| 回笼量 | 约**4,000万锭** | 仅占发行量的约1/3 |
| 市场留存 | 约**8,000–9,000万锭** | 未能回收，持续推动贬值 |

对比：元代纸币发行初期仅约20万锭/年，后来多数也控制在150万锭以下。

[来源：知乎-朱八八制造的明朝先天性财政问题]

### 2.5 贬值过程

| 时间 | 宝钞与白银比价 | 贬值程度 | 备注 |
|------|---------------|---------|------|
| 洪武八年（1375） | 1贯 = 1两 | 官方定价 | 发行之初 |
| 洪武十八年（1385） | 约3–5贯 = 1两 | 贬至约**1/3–1/5** | 发行仅约10年 |
| 洪武二十三年（1390） | 约4贯 = 1两 | 贬至官价的**1/4** | 发行仅15年 |
| 洪武二十七年（1394） | 约6贯 = 1两 | 贬至**1/6** | |
| 洪武三十年（1397） | 约14贯 = 1两 | 贬至约**1/14** | |
| 朱元璋在位30年末 | 壹贯仅值**100文** | 贬至面值的**1/10** | |
| 朱棣在位22年末 | 壹贯仅值**12文** | 贬至面值的约**1/83** | |
| 宣德七年（1432） | 100贯 = 1两 | 贬至**1/100** | 湖广等地商税折钞价 |
| 宣德十年（1435） | 1,000贯 = 1两 | 贬至**1/1,000** | |
| 正统元年（1436） | 1,000余贯 = 1两 | 不足**1/1,000** | "今银一两当钞千余贯" |
| 弘治—正德年间 | "积之市肆，过者不顾" | 名存实亡 | 宝钞制度事实终结 |

[来源：中新网-明代的纸币超发；知乎-大明通行宝钞]

### 2.6 崩溃原因

1. **无准备金**：朱元璋发行纸币时没有"准备金"概念
2. **无限制超发**：投放量远超社会商品流通需求（洪武24年发行1.2亿锭以上）
3. **只出不收**：政府大量印钞支付开支，但税收中不回收宝钞（或回收极少）
4. **倒钞法不健全**：洪武九年（1376）制定《倒钞法》设行用库调换破损纸钞，但有限额，且官府收税只要新钞
5. **禁金银令无法执行**：民间继续广泛使用白银
6. **制作简陋**：桑皮纸材质易破损伪造，进一步削弱信用

### 2.7 退出流通时间线

| 时间 | 事件 |
|------|------|
| 宣德三年（1428） | 因纸币贬值，钞法不通，**明宣宗令停造新钞** |
| 正统—成化年间（约1436–1487） | 宝钞在民间流通中**彻底绝迹** |
| 弘治—正德年间 | 钞法已形同虚设 |
| 嘉靖以后 | 白银正式成为主币，宝钞仅在极少数场合以象征性面额出现 |

### 2.8 对官员俸禄的影响

宝钞贬值直接冲击了官员收入——官员俸禄中有大量以宝钞折支的部分：

| 年份 | 一品官员月俸折合大米（公石） |
|------|--------------------------|
| 1380年 | 约116 |
| 1433年 | 约46 |
| 1444年 | 约35 |
| 1471年 | 约21 |

官员实际购买力在一个世纪内下降了约**82%**。

## 三、白银的国内来源

### 3.1 总体产量

明代国内白银产量总体偏低，不足以支撑白银货币化的巨大需求。

- 明朝平均银课收入约**10万两/年**，后期不足10万两 [来源：中国经济史论坛]
- 银课约占银矿产额的**30%或以上**，故实际银产量约为银课的3倍，即约**30万两/年**
- 宋应星（崇祯十年前后）："合八省所生，不敌云南之半" [来源：《天工开物》]

### 3.4 银课的局限性

需要注意：《明实录》中的银课数字并非全部来自实际矿产征收。明中后期征课数字激增，实际开采量并未同步激增，而是"陪纳"（强制摊派）增多，矿区百姓被迫补足朝廷设定的课额。

## 四、海外白银流入

### 4.1 流入背景与驱动力

#### 金银比价套利

| 地区 | 16世纪初金银比价 | 16世纪末金银比价 |
|------|-----------------|-----------------|
| **中国** | **1:6** | 1:5.5–1:7（广东） |
| 欧洲（西班牙） | 1:12 | 1:12.5–1:14 |
| 波斯 | 1:10 | — |
| 印度 | 1:8 | — |

- 中国白银价格约为西班牙的**两倍**
- 巨大价差吸引全球白银流向中国："正是中国的巨大需求引起了中国白银价值两倍于世界其他地方" [来源：中国经济史论坛]

#### 金银比价的长期变动

| 时段 | 中国金银比价 | 趋势说明 |
|------|------------|---------|
| 16世纪30年代前 | 约1:6 | 白银相对黄金极贵 |
| 16世纪30年代–17世纪初 | 从1:6渐升至1:7–8 | 海外白银开始流入，白银相对贬值 |
| 17世纪初–30年代 | 约1:7–8 | 相对稳定 |
| 17世纪30年代后 | 升至1:10–13 | 白银大量流入后进一步贬值 |

[来源：中国经济史论坛-明朝中后期世界白银为何大量流入中国]

### 4.2 日本渠道

#### 日本白银产量

- **石见银山**：16世纪世界最大银矿之一，1533年引入灰吹法（精炼技术）后产量大增
- 1560–1601年：日本每年白银出口**33.75–48.75吨**
- 17世纪初年：达**130–160吨/年**，占世界银产量**30%–40%** [来源：岩生成一]

#### 贸易路线

- **中日直接贸易**：中国商人（走私为主）直接赴日本交易
- **葡萄牙人中转贸易**：以澳门为中心，建立澳门—长崎航线
- **荷兰人参与**：17世纪初荷兰东印度公司加入

- 从1550年到1645年，海外流入白银约**14,000吨**，是同期中国自产白银的近**10倍**
- 这说明明代白银货币化在物质基础上**严重依赖海外供给**

[来源：中国经济史论坛-明清时期白银流入量分析]

### 6.3 变动分析

#### 长期趋势（洪武至万历）：白银贬值

- **原因**：海外白银大量流入增加供给，白银相对铜钱贬值
- 1两银子能换到的铜钱数减少（从1,000文降至600–800文）

#### 崇祯末年逆转：白银急剧升值

- **原因**：
  1. 17世纪30年代全球白银产量下降
  2. 日本德川幕府限制白银出口（1639年锁国）
  3. 国际白银流入锐减
  4. 战乱导致白银窖藏，流通中白银减少
  5. 劣质铜钱泛滥进一步推高银价
- 1两银子兑换铜钱数从600–800文暴增至2,000–2,500文

## 七、白银的流通形式

### 7.1 称量银货币体系

明朝白银并非铸币，而是**称量货币**——以重量为交易单位，而非面值。明朝政府**从未将白银铸造成标准化银币**，也未设立统一的成色标准和度量衡标准。

### 7.2 银锭（银元宝）

| 类型 | 重量 | 特征 |
|------|------|------|
| 大元宝 | **50两**（约1,850克） | 两端翘起的船形（圆首束腰形），上有铸造地点、重量和银匠姓名 |
| 中锭 | 约10–20两 | 形制与大元宝相似但较小 |
| 小锭 | 约1–5两 | 文字较少，有时铸印年号 |
| 最大规格 | 可达**500两** | 极少见 |

明代一两约等于**37克**。

**成色名称**：花银、金花银（早期）；纹银、雪花银、细丝、松纹、足纹（晚期）。

### 7.3 碎银与交易方式

- 日常小额交易使用碎银（从银锭上剪下的碎块）
- 购物者随身携带**钢剪**（剪银用）和**戥子**（精密小秤，称重用）
- 交易时需用**试银石**检验成色
- 各地"平"（衡量标准）不统一，存在库平、市平、漕平等不同标准
- 火耗（熔铸损耗）成为地方官加征的重要名目

## 八、白银货币化对一条鞭法的支撑关系

### 8.1 互为因果的关系

白银货币化与一条鞭法的关系是**互为因果、相互支撑**的：

| 方向 | 关系说明 |
|------|---------|
| 白银货币化 → 一条鞭法 | 白银在民间的广泛流通，为赋役折银提供了**物质基础和社会条件** |
| 一条鞭法 → 白银货币化 | 赋役折银的制度性要求，反过来**加速和巩固**了白银的主币地位 |

### 8.2 具体支撑机制

1. **物质前提**：白银货币化使大量白银在民间流通，使"赋役折银"在技术上成为可能
2. **制度推力**：一条鞭法将田赋、徭役统一折银征收，从国家制度层面强制推广白银使用
3. **财政效果**：折银征收简化了征税流程，减少了中间盘剥，中央太仓库岁入从隆庆元年23万两增至万历五年435.94万两（增幅近**20倍**）
4. **需求放大**：赋役折银使全国产生了对白银的巨大刚性需求，拉动海外白银持续流入

### 8.3 万明的核心观点

万明等学者指出："一条鞭法的实行是白银货币化完成的标志，又是白银货币化的一个结果。""一条鞭法改革之后，明代中国对白银有一种无限渴求，因为明代对白银货币的需求很大程度上是由赋税货币化引起的。" [来源：万明-中国的"白银时代"与国家转型]

### 8.4 风险与后果

白银货币化与一条鞭法的深度绑定带来了系统性风险：
- 明朝政府对白银**无铸币权**（非铸造货币），无法通过货币政策调节经济
- 白银供给完全依赖市场和海外贸易，政府丧失了货币调控能力
- 当17世纪30年代全球白银供应锐减时，明朝经济直接受到冲击
- 万明等学者认为：**明朝覆亡与全球白银供应变化存在深层关联**

## 九、白银货币化对社会经济的影响

### 9.1 财政制度变革

- **田赋折银**：正统元年（1436）开始，南方部分田赋折银征收
- **一条鞭法**：万历九年（1581）全面推行，赋役折银计算
- 中央财政收入显著增长：从嘉靖末年约243万两增至万历五年435.9万两，增幅**79%**
- 标志着中国从实物税制向货币税制的历史性转变

### 9.2 商品经济发展

- 促进了市场繁荣与专业化分工
- 产业非农化程度加深，出现更多独立的手工业门类
- 东南沿海丝绸、瓷器、茶叶大量出口
- 城镇化加速，江南市镇经济繁荣

### 9.3 社会分化加剧

- 贫富差距拉大
- 商人阶层崛起，传统士农工商的等级秩序受冲击
- 土地兼并加速：白银交易使土地买卖更便利

### 9.4 对农民的影响

- **赋役折银**增加了农民实际负担：必须先将粮食出售换取白银
- 粮价低迷时低价抛售，实际税负加重
- 北方白银流通不如江南普及，"以银代役"对北方农民尤为不利
- "平静的乡村生活起了波澜"

### 9.5 区域经济差异

| 地区 | 白银充裕程度 | 一条鞭法推行效果 |
|------|------------|-----------------|
| 东南沿海 | 充裕 | 效果好，商品经济发达 |
| 北方内陆 | 匮乏 | 效果差，百姓难以获银缴税 |

南北经济差距因白银货币化而进一步扩大。

### 9.6 晚明"银荒"与经济危机

**17世纪30年代起**，明朝从"通货膨胀"进入"通货紧缩"：

**外因**：
- 日本锁国（1639年），限制白银出口
- 美洲银产量下降
- 全球白银流入中国锐减

**内因**：
- 战乱导致白银大量窖藏
- 三饷加派导致白银从民间向军事领域单向流动
- 商业萎缩减少白银流通

**后果**：
- 物价下跌，银贵钱贱
- 农民卖粮换银愈加困难
- 税收征缴更加艰难
- 形成"白银短缺 → 经济萎缩 → 财政困难 → 加派 → 民变"的恶性循环

### 9.7 与全球贸易体系的关联

白银货币化使明朝经济深度嵌入全球贸易网络：
- 中国成为全球白银的最大"吸纳器"
- 形成了以中国丝瓷茶换取全球白银的"丝-银对流"格局
- 当全球白银供应出现波动时，明朝经济直接受到冲击
- 万明等学者认为：明朝覆亡与全球白银供应变化存在深层关联
