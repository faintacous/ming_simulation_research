# 明朝漕粮

## 概述

漕运是明朝将南方各省征收的税粮（漕粮）通过大运河运送至京师（北京）的国家物资运输制度。自永乐迁都北京后，漕运成为维系京师百万军民口粮和北方边防军需的经济命脉。明成化八年（1472年）正式确定漕粮定额为每年 **400 万石**，由专门的**漕运总兵官**和**漕运总督**统领**运军**（漕军）负责运输。漕运制度庞大而复杂，运输成本极高，是明朝财政的一大负担。

---

## 一、漕运定额（400万石）的历史演变

### 1.1 运河漕运的确立

| 年份 | 事件 | 漕运规模 |
|-----|------|---------|
| 永乐元年（1403） | 朱棣决定迁都北京 | 开始大规模南粮北运 |
| 永乐九年（1411） | 工部尚书**宋礼**主持疏浚会通河（运河山东段） | 运河全线贯通 |
| 永乐十三年（1415） | 运河漕运正式取代海运成为主要运粮方式 | 年运约 300–350 万石 |
| 宣德年间 | 漕运体制逐步完善 | 年运约 350 万石 |
| 成化八年（1472） | **正式确定漕粮年额为 400 万石** | 成为定额 |
| 正德至嘉靖 | 实际完运量波动，常不足额 | 实际约 300–380 万石 |
| 万历年间 | 制度运转相对稳定 | 大致完成定额 |
| 天启–崇祯 | 运河淤塞、兵荒马乱、漕运日益困难 | 实际大幅缩减 |

### 1.2 漕粮来源省份

漕粮主要从以下省份征调（即"有漕八省"）：

| 省份 | 年漕运定额（约） | 占比（约） | 备注 |
|------|----------------|-----------|------|
| 南直隶（江苏、安徽） | 约 180 万石 | 45% | 最大漕粮来源，含苏松常等重赋区 |
| 浙江 | 约 63 万石 | 16% | |
| 江西 | 约 50 万石 | 12.5% | |
| 湖广（湖北、湖南） | 约 50 万石 | 12.5% | |
| 山东 | 约 25 万石 | 6% | |
| 河南 | 约 20 万石 | 5% | |
| 北直隶 | 少量 | — | 部分地区有漕 |

（注：以上数据为概数，各朝定额略有调整）

### 1.3 漕粮的性质

漕粮是田赋的一部分，不是额外加征：
- 各省田赋分为**起运**和**存留**两部分
- **起运**部分中需运往京师的粮食即为漕粮
- **存留**部分供本地官府支用
- 漕粮在田赋总额中的比例因省而异

## 二、运军编制

### 2.1 总体规模

| 项目 | 数据 |
|------|------|
| 漕运总兵官 | 1 员（驻淮安） |
| 漕运总督 | 1 员（文官，总理漕务） |
| 漕运十二总 | 12 个卫所总编制 |
| 运军（漕军）总数 | **约 127,600 人**（12.76 万人） |
| 运船（漕船）定额 | **11,770 只** |

### 2.2 十二总的编制

运军按卫所编制，分为十二个"总"（总运单位），分驻沿运河各卫所：

| 总名 | 驻地（约） | 辖运军/船 |
|------|----------|----------|
| 第一总至第十二总 | 南京至通州沿运河各卫 | 各总约 1 万余军、900–1000 只船 |

各总下设卫、所、旗，按编制逐级管理。每只漕船额配旗军约 **10–12 名**。

### 2.3 运军的来源和身份

- 运军属于**军户**，世代当差
- 每户须出一名正军服漕运之役
- 正军阵亡或病故，由其家族中其他男丁替补（"勾丁"）
- 运军待遇：
  - 月粮：按月发放口粮
  - 行粮：运漕期间额外发放的口粮，每名运军最高约 2 石 8 斗行粮
  - 另有少量"帮费"银两

## 三、漕船数量和规格

### 3.1 漕船定额

| 时期 | 漕船数量 | 备注 |
|------|---------|------|
| 永乐初 | 各类船只约 14,000 艘 | 包括多种船型 |
| 天顺朝以后 | **定额 11,770 艘** | 标准化后的额定数 |
| 万历年间 | 维持约 11,000–12,000 艘 | |

### 3.2 漕船规格

| 项目 | 规格 |
|------|------|
| 船型 | 平底浅吃水河船 |
| 标准长度 | 约 **16 米**（52 尺） |
| 标准宽度 | 约 **2.8 米**（9 尺余） |
| 额定载重 | **300–500 石** |
| 船材 | 楠木、杉木为佳，次用松木 |
| 使用年限 | 约 10 年，到期按例修造 |
| 修造费用 | 每只约 **100–150 两白银** |

### 3.3 遮洋船

在部分路段需经海路运输的区间（如辽东），使用**遮洋船**：
- 较一般漕船更大更坚固
- 每船额定旗军 **12 名**
- 遮洋船定额约 **47 只**（部分地区）

## 四、运输成本

### 4.1 直接成本

漕运的总成本远超运输粮食本身的价值：

| 成本项目 | 估计数额 | 备注 |
|---------|---------|------|
| 运军行粮 | 每名约 2.8 石 | 12.76 万人 × 2.8 石 ≈ 36 万石 |
| 运军月粮 | 每名约 12 石/年 | 12.76 万人 × 12 石 ≈ 153 万石 |
| 漕船修造 | 每年数十万两 | 每10年全面换新 |
| 闸坝维修 | 每年约 30–50 万两 | 运河沿线数十座闸坝 |
| 河道疏浚 | 每年数万至数十万两 | 黄河夺淮后尤其昂贵 |
| 运丁沿途费用 | 难以估算 | 纤绳、码头费、过闸费等 |

### 4.2 综合运输比率

据研究估计：
- 运送 400 万石漕粮到京师的**全部成本**约需 **1,200–1,600 万石粮食**的等价支出
- 即运费成本约为正粮的 **3–4 倍**
- 每运 1 石粮到京师，综合成本约 **1.5–1.7 石粮食加 0.5–0.7 两白银**

### 4.3 与海运的成本对比

| 方式 | 单位成本（每石） | 时间 | 风险 |
|------|---------------|------|------|
| 河运（大运河） | 约 0.5–0.7 两 | 数月 | 低（但成本高） |
| 海运 | 约 0.1–0.2 两 | 数周 | 较高（风浪） |

海运成本仅为河运的 1/3 至 1/5，但明朝始终以河运为主，原因包括：运河沿线数十万运军的生计问题、漕运利益集团的阻力、海禁政策等。

## 五、沿途损耗率

### 5.1 正式耗米（加耗）

各省漕粮除正粮外，需额外缴纳**耗米**以弥补运输途中的自然损耗：

| 省份/地区 | 每石正粮加耗 | 备注 |
|----------|-------------|------|
| 湖广 | **4 斗**（40%） | 路途最远，加耗最高 |
| 浙江 | **4 斗**（40%） | |
| 江西 | **4 斗**（40%） | |
| 应天及江南直隶诸府 | **3 斗**（30%） | |
| 江北直隶诸府 | **2 斗 5 升**（25%） | |
| 徐州 | **2 斗**（20%） | |
| 山东 | **1 斗 5 升**（15%） | 距京师最近 |
| 河南 | **1 斗 5 升**（15%） | |

（据成化九年户部规定）

### 5.2 实际损耗

实际损耗远高于正式加耗：
- **自然损耗**：霉变、虫蛀、渗水，约占 5%–10%
- **人为损耗**：运军盗卖（"使漏"）、沿途被勒索
- **船难损失**：翻船、搁浅导致的粮食损失
- **实际抵仓率**：最终运抵京师通州仓的粮食约为起运量的 **八成**左右
- 综合损耗率约 **20%–30%**

## 六、折银比率

### 6.1 漕粮折银

部分漕粮可折银代缴，称为"折色漕粮"或"折漕"：

| 类型 | 折银率 | 备注 |
|------|-------|------|
| 白粮折银 | **每石折银 0.5 两** | 终明一代不变 |
| 一般漕粮折征 | **每石折银 0.5–0.7 两** | 因地因时而异 |
| 金花银折率 | **每石折银 0.25 两** | 正统初年定制 |

### 6.2 折色与本色的比例

- 明初以本色（实物）为主，折色极少
- 正统以后折色比例逐渐增大
- 一条鞭法推行后，折色进一步增加
- 但漕粮始终保留了大量本色征收（因京师和北方军镇确实需要实物粮食）
- 万历年间，漕粮折色约占总额的 20%–30%

### 6.3 折色的影响

- 折银减轻了远途运输的负担
- 但折银比率往往对百姓不利（官方折率低于市场粮价）
- 同时丧失了部分实物供应保障功能

## 七、三种运输方式的演变（支运→兑运→长运）

据《明史·食货志》记载："自成祖迁燕后，运法凡三变。初支运，次兑运、支运相参。至支运悉变为长运而制定。"

### 7.1 支运（民运）

- **时期**：永乐至宣德初年
- **方式**：各地百姓（粮户）将漕粮亲自运送至指定仓库
- **流程**：
  1. 南方各省农户将税粮运至淮安、瓜州等中转仓
  2. 再由官府组织民夫转运至北京通州仓
  3. 分段转运，百姓承担巨大运输劳力和费用
- **弊端**：百姓往返耗时数月，严重影响农业生产；途中损耗极大

### 7.2 兑运（军民合运）

- **时期**：宣德五年（1430）至成化七年（1471）
- **方式**：百姓将漕粮运至指定水次（岸边交接点），兑交给运军，由运军负责转运至京师
- **改革者**：漕运总兵官**陈瑄**首倡
- **流程**：
  1. 百姓将粮食运至淮安、瓜州等江边水次
  2. 运军在水次接收粮食
  3. 运军负责从水次转运至通州
- **加耗规定**：百姓需额外缴纳"过江费"和"加耗米"以补偿运军
- **效果**：减轻了百姓长途运输的负担，但兑运与支运长期并行

### 7.3 长运（改兑法/军运）

- **时期**：成化七年（1471）至明末
- **方式**：运军直接赴江南各州县水次交兑，百姓不再远运
- **改革者**：漕运都御史**滕昭**推行
- **流程**：
  1. 运军过江，直接到江南各州县水次（码头）接受粮食
  2. 百姓只需将粮运至本地附近的水次即可
  3. 运军负责全程运输至北京通州
- **加耗调整**：成化九年（1473）定"加耗则例"，按距离远近计算运费
- **效果**：进一步减轻百姓运输负担，但运军负担加重，且需额外征收过江费用

### 7.4 三种方式对比

| 项目 | 支运 | 兑运 | 长运 |
|------|------|------|------|
| 百姓运输距离 | 全程 | 至水次（数百里） | 至本地水次（数十里） |
| 运军责任 | 无/少 | 水次至京师 | 江南至京师全程 |
| 百姓负担 | 极重 | 中等 | 较轻（但需缴过江费） |
| 运军负担 | 轻 | 中等 | 极重 |
| 实行时期 | 永乐至宣德 | 宣德至成化 | 成化至明末 |

## 八、遮洋总与海运

### 8.1 遮洋总（海军）编制

除运河漕运外，明朝还保留了少量海运力量：

| 项目 | 数据 |
|------|------|
| 遮洋总军士 | 约 **7,000 人** |
| 遮洋海船 | 约 **350 只** |
| 运输路线 | 主要从南直隶沿海至天津 |
| 运粮量 | 占总漕运量的极小部分 |

### 8.2 海运的兴废

- 元代以海运为主，永乐初年仍有海运
- 永乐十三年（1415）运河全线贯通后，海运逐渐被取代
- 嘉靖年间曾有恢复海运的讨论，但因"百万漕工衣食所系"等阻力而未能实现
- 明末始终未恢复大规模海运

## 存疑 & 待查

1. **各省漕粮起运定额的精确数据**：上文提供的各省定额为概数，需查《大明会典·漕运》和《万历会计录》获取精确数值
2. **十二总的具体驻地和辖区**：各总的名称、驻地、辖运军/船数据需查漕运专志
3. **漕船修造的具体费用和周期**：不同时期、不同地区的造船成本差异较大
4. **运军的实际生存状况**：运军逃亡率和缺额率的历年数据待查
5. **崇祯朝漕运实际完运量**：崇祯年间因战乱和运河问题，实际运达的漕粮数据极不完整
6. **漕粮品质标准**：京仓验收的品质标准和拒收/扣减规则待查
