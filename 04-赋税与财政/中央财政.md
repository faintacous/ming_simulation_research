# 明朝中央财政

## 概述

明代中央财政以"内库—外库"二元体系为核心。外库以户部太仓库为中枢，负责国家公共财政的收支运转；内库以内承运库为核心，存储皇室私有财富。嘉靖以后太仓库地位急剧上升，从正统七年（1442）仅存储马草折银的小库，发展为岁入数百万两、承担边饷京运的财政枢纽。军费在中央支出中占比60%–90%，是压垮明末财政的主因。崇祯末年太仓库实际征收率仅9.3%，户部留贮不满二千两，财政体系全面崩溃。

## 一、太仓库

### 1.1 设立与沿革

- **设立时间**：正统七年（1442），添设户部主事一员专管 [来源：百度百科-太仓库]
- **初始职能**：收纳南直隶苏、常等府解纳的**马草折银**，规模极小，财政地位低微 [来源：《太仓库与明代财政制度演变研究》]
- **别称**：太仓银库，简称"银库"，隶属户部 [来源：百度百科-太仓库]
- **设立背景**：永乐以后，内库（内承运库等）日益成为皇室私库，不再承担国家财政的主要储蓄功能。正统年间宝钞崩溃、白银货币化加速，朝廷需要一个专门的外库来管理日益增多的白银财政收入，太仓库由此应运而生
- **收入类项**：凡各直省派剩麦米、内库中绵丝绢布、马草、盐课之折银者，籍没家财变卖田产、追收店钱之援例上交者，皆由此库贮存

### 1.2 职能演变

| 阶段 | 时间 | 职能特征 |
|------|------|----------|
| 草创期 | 正统七年（1442） | 仅存储马草折银，规模微小 |
| 积累期 | 弘治末—正德初（约1505–1521） | 开始接纳中央财政体系的收支盈余银，成为"财政储备"机构，收入主要用于应急军情和重大灾情救济 |
| 扩张期 | 嘉靖—隆庆（1521–1572） | 岁入类项迅速增长，岁入预算银额**200余万两**；嘉靖末隆庆初代替内库承担京运年例银 |
| 鼎盛期 | 万历前期（1573–1600） | 万历五年岁入达**435.94万两**；万历会计录载岁额**400万两**，其中专属边饷**389万余两** |
| 膨胀崩溃期 | 万历后期—崇祯（1600–1644） | 预算岁额升至**2,000余万两**（含加派），但逋欠严重，实际征收率急剧下降 |

[来源：《太仓库与明代财政制度演变研究》；从太仓库岁入类项看明代财政制度的变迁]

**关键转折**：嘉靖末、隆庆初，太仓库代替内库从主体上承担起发放京运年例银的职责。嘉靖中期，北边军饷白银供给量占总岁额的比重升至**82%以上**，太仓库由"储备库"转变为"运营库"。

## 二、支出结构

### 2.1 军费（最大支出项）

军费是明朝财政支出中最大的一项，也是太仓库存在的核心理由。

- **总体占比**：明中叶以后，军费开支占中央政府支出的**60%–90%** [来源：清华大学-明代GDP试探]
- **万历会计录所载**：太仓岁额400万两中，专属边饷者**389万余两**（占97%） [来源：《万历会计录》]

#### 北边军饷的演变

| 时期 | 太仓库京运银占北边军饷比例 | 说明 |
|------|--------------------------|------|
| 嘉靖初 | 约10% | 民运为主，太仓库仅补充 |
| 万历初 | 约30%以上 | 太仓库地位上升 |
| 万历二十九年（1601） | 超过50% | 太仓库京运银约400万两，首次超过民运，成为北边军饷核心供应 |
| 崇祯十六年（1643） | — | 户部兵饷岁入预算额总计达**1,584万两** |

[来源：《太仓库与明代财政制度演变研究》]

- **万历二十九年数据**：北边十四镇年例岁费京运、民运银共**720万两**，其中太仓库京运银约**400万两**
- **军饷白银化**：嘉靖中期，北边军饷中白银供给量占总岁额比重升至**82%以上**

#### 三饷加派

| 名称 | 开征时间 | 初始额度 | 最终额度 | 用途 |
|------|----------|---------|---------|------|
| **辽饷** | 万历四十六年（1618） | 亩加三厘五毫 → 九厘 → 一分二厘 | 田赋**660万两**，合盐课关税杂项共**740.83万两** | 辽东防御后金 |
| **剿饷** | 崇祯十年（1637） | 总额**280万两** | 280万两（原议一年，实征至崇祯十三年） | 镇压农民起义 |
| **练饷** | 崇祯十二年（1639） | 亩加一分 | 总额**730余万两**（田赋部分约481万两） | 训练边兵 |

- **三饷总额超过明朝正赋一倍以上**，每年加派超过**2,000万两** [来源：百度百科-三饷；维基百科-三饷]
- 辽饷加派经过：万历四十六年（1618）加三厘五毫，四十七年（1619）复加三厘五毫，四十八年（1620）再加二厘，前后合计每亩加征银九厘，共**520.06万两**；崇祯四年（1631）提高至一分二厘，共银**660万两**
- 崇祯四年十二月设立考核制度：地方官员若无法征收足够正税、加派和金花银，不能参加考选晋升 [来源：维基百科-崇祯帝]

#### 实发方式

俸禄分"本色"与"折色"两种支取：
- **月米**：每位官员每月支取1石（实物）
- **折绢米**：每年2个月以绢折支，绢1匹折银7钱
- **折银米**：每年10个月以银折支

折色制度大幅降低了官员的实际收入。

- 万历二十四年（1596）坤宁宫失火后的重建工程，是开征矿税的直接诱因

## 三、内帑（皇帝私库）

### 3.1 内承运库

**设立沿革**：
- 洪武二年（1369）设"内府库"
- 洪武八年（1375）改称"承运库"
- 洪武二十年（1387）分设"内承运库"（皇城内，宦官管理）和"外承运库"（户部管辖）

**管理机构**：设掌印太监及佥书太监，属内府宦官系统。户部可派员巡视登记，但无权管理。

**核心地位**：内承运库在内府十库中地位最高，专储金银，金花银为其最大收入项。

**职能演变**：
- 洪武时期：兼具国库与皇室库房的双重功能（"公私一体"）
- 永乐至正德：职能转向君主私库
- 嘉靖三十七年（1558）：明确"每年进银100万两，外加预备钦取银"的原则，皇室财政专属化正式确立

#### 金花银详情

- 正统元年（1436），南直隶、浙江、江西、湖广、福建、广东、广西七省夏税秋粮**400余万石**折银征收
- 米麦每石折银**二钱五分**，共折银**101.27万两**
- 每季分进内承运库**25万余两**
- 据《万历会计录》，内承运库岁入"金花银共壹百壹万贰千余两"
- 扣除武臣俸禄约**十余万两**后，其余约**78%**皆为御用
- 占全国税粮总数约**9%**的折粮银转移至皇宫内库 [来源：百度百科-金花银；维基百科-内承运库]

#### 矿税进奉详情（万历朝）

- **开征时间**：万历二十四年（1596）六月起，以修建宫殿为名
- **执行方式**：派遣大批太监为矿监、税使到各地"开矿榷税"
- **进奉数据**（万历二十五年至三十四年，1597–1606）：
  - 文秉《定陵注略·内库进奉》载：十年进奉白银**569万余两**，黄金约**1.2万两**（年均白银约57万、黄金约1,000两）
  - 《明神宗实录》核校后：实际征收约为进奉数字的**3倍**，估算年均白银**171万两**，十年间达**1,710万两**
- **贪污与损耗**：矿监税使及亲随贪污瓜分的白银数额，远大于进奉内库的数额
- **废止时间**：万历四十八年（1620）神宗去世后废止

[来源：万明-全球视野下的明朝覆没；维基百科-矿税之弊]

| 万历末年（1620） | 约**700万两** | 万历在位48年积蓄，神宗驾崩时"帑如山积" |
| 万历四十八年—天启四年（1620–1624） | 发帑银**1,938万两** | 五年间接济辽东前线 |
| 万历四十六年—天启七年（1618–1627） | 协济总额**2,645.33万两** | 十年间，年均264.53万两，为金花银正额岁入的2.5倍 |
| 崇祯初年（1628） | 所剩无几 | 经天启朝大量拨发后 |
| 崇祯十七年（1644） | **争议极大** | 崇祯自称"内帑如洗"；学界分歧——一说仍有存银，一说已耗尽 |

[来源：故宫博物院-明宫藏银之谜（蒿峰）]

#### 万历四十六年至天启七年内库协济明细

- 十年间内库共计协济支银**2,645.33万两**
- 年均约264.53万两，为内库金花银正额岁入的**2.5倍**
- 流向构成：助饷（1,387万余两）、犒赏、助工三项占协济总额的**95.92%** [来源：从明代内库与皇室财政看明代国运兴衰]

### 3.5 内帑与外帑的互动案例

#### （一）外库 → 内库（皇帝提取）

| 时间 | 内容 |
|------|------|
| 成化、弘治朝 | 内库多次从太仓库支取银两 |
| 嘉靖时期 | 皇帝经常从太仓库中支取银两补贴皇室 |
| 万历八年—二十九年（1580–1601） | 22年间内库对部寺库银直接提取不少于**14次**，侵夺总额不低于**206.96万两** |
| 万历二十七年（1599） | 神宗拒绝户部支取内库银用于边饷的请求（反向拒绝的典型案例） |

#### （二）内库 → 外库（皇帝拨发）

| 时间 | 事件 | 金额 |
|------|------|------|
| 萨尔浒之战后（1619） | 兵部太仆寺、户部太仓、工部节慎库全空，钱只能从内帑出 | — |
| 万历四十八年—天启四年（1620–1624） | 五年发帑银接济辽东前线 | **1,938万两** |
| 天启七年十一月（1627） | 崇祯登极赏赉 | 先后两次发御前银**130万两** |
| 崇祯元年三月（1628） | 济陕西军饷 | 发帑银**50万两** |
| 崇祯十五年八月（1642） | 买嘉谷备赈济 | 发御前银**10万两** |

#### （三）崇祯末年的内帑困局

崇祯在位期间多次拒绝户部尚书的"乞帑"请求，或仅拨发少量银两（数万两级别）。学界对明亡时内库存银有两种截然不同的看法：
- **有存银说**：皇帝始终不愿拨发，明亡后内库仍有大量积存
- **已耗尽说**：历次协济已消耗一空，"内帑如洗"确为实情

[来源：故宫博物院-明宫藏银之谜；知乎-崇祯内帑]

太仆寺本职为管理马政，成化四年建立常盈库专门存放备用马价银和草场租银。因养马地方灾伤或马匹不合格而折征银两，积累日丰。

## 五、户部的财政管理职能

### 5.1 组织架构

户部为明朝中央财政的最高管理机构，户部尚书（正二品）掌管全国户籍人口、土地、税赋（含盐税、船税、茶税等）、徭役、宝钞、仓储等事务。

户部下设**十三清吏司**，按地域分省管理各地钱粮：浙江、江西、湖广、福建、山东、山西、河南、陕西、四川、广东、广西、云南、贵州。

### 5.2 清吏司的专项职能

各清吏司除分省管理钱粮外，还兼管特定中央事务：

| 清吏司 | 专管事项 |
|--------|---------|
| 陕西清吏司 | 百官、勋贵、宗室俸禄 |
| 山东清吏司 | 盐课 |
| 贵州清吏司 | 关税 |
| 云南清吏司 | 漕运 |

### 5.3 财政运作流程

1. 户部每年通过十三清吏司向对应的十三布政司分配当年税额
2. 各布政司将税额分配到辖下各府
3. 各府再分配到各州县
4. 州县以里甲为单位进行征收
5. 征收后按"起运/存留"比例，留足地方所需支出数额
6. 将应上缴部分运送至京城

### 6.3 关键数据节点

- **洪武二十六年（1393）**：官民田总850.7万余顷；夏税米麦471.7万余石；秋粮2,472.9万余石

### 6.4 崇祯末年的财政崩溃

- **崇祯十六年**（1643）：户部赋税逋欠率达**85%**
- **崇祯十七年**（1644）正月：户部赋税实际征收率仅**9.3%**
- 倪元璐明言："饷银解到**八万余两**，而京边之索饷者数百万"
- 政府已基本丧失为边镇军饷提供中央资助的能力 [来源：故宫博物院-明宫藏银之谜]
