# 明末三饷加派

## 概述

三饷是明朝末年在正常田赋之外加征的三项军事附加税的总称，包括**辽饷**、**剿饷**和**练饷**。三饷从万历四十六年（1618年）开始陆续开征，到崇祯末年合计加派总额超过 **2,000 万两白银**，远超明朝正常年份的全部田赋收入。三饷的沉重负担是激化社会矛盾、引发农民大起义、最终导致明朝灭亡的直接原因之一。

---

## 一、辽饷

### 1.1 开征背景

- 万历四十六年（1618年），后金努尔哈赤以"七大恨"誓师，发兵攻明
- 辽东军事形势骤然紧张，急需大量军费
- 国库空虚，太仓存银不足以应对战争开支
- 户部尚书**李汝华**提议加征田赋以筹措辽东军费

### 1.2 历次加征

| 时间 | 加征内容 | 亩加额度 | 合计年加征总额 |
|------|---------|---------|--------------|
| **万历四十六年（1618）** | 首次加派 | 每亩加 **3 厘 5 毫** | **201 万 0,031 两** |
| **万历四十七年（1619）** | 第二次加派 | 再加 **3 厘 5 毫**（累计 7 厘） | 约 400 万两 |
| **万历四十八年（1620）** | 第三次加派 | 再加 **2 厘**（累计 9 厘） | **520 万 0,062 两** |
| **天启年间** | 扩展至盐课、关税、杂项 | — | 约 620–650 万两 |
| **崇祯四年（1631）** | 田赋从 9 厘加至 1 分 2 厘 | 每亩 **1 分 2 厘** | 田赋 **667 万 9,208 两** |
| **崇祯四年合计** | 含盐课 93.9 万、关税 27.1 万、杂项 | — | **1,029 万 9,602 两** |

### 1.3 征收范围

- 全国除贵州等极少数边远贫困地区外，各省均须承担
- 按在册田亩面积统一加征
- 不分官田民田，一律适用

### 1.4 辽饷的演变

- 最初定性为"暂行"加征，战事结束即停
- 但随着辽东战事持续恶化（萨尔浒之败、沈阳失陷等），辽饷非但未停反而不断加码
- 从万历末到崇祯亡国，辽饷征收长达 **26 年**，成为事实上的永久性加税

## 二、剿饷

### 2.1 开征背景

- 崇祯初年，农民起义在陕西爆发并蔓延全国
- 李自成、张献忠等各路义军纵横中原
- 朝廷需要大量军费镇压起义

### 2.2 开征时间与税额

| 项目 | 内容 |
|------|------|
| 开征时间 | **崇祯十年（1637年）** |
| 提议者 | 兵部尚书**杨嗣昌** |
| 名义用途 | 增兵十二万镇压农民起义 |
| 年征收总额 | **280 万两** |
| 征收方式 | 按亩加派 + 关税、盐课等附加 |
| 停征时间 | 崇祯十三年（1640年）名义停征 |

### 2.3 剿饷的四大来源

| 来源 | 额度（万两） | 征收方式 |
|------|-----------|---------|
| 因粮（均输） | 192.9 | 亩输六合，石折银八钱 |
| 溢地（清查隐匿田亩） | 40.6 | 查出隐瞒未报田亩后征收 |
| 驿递（裁减驿站） | 20.0 | 裁撤冗余驿站节省经费 |
| 事例（寄学监生纳银） | 约 26.0 | 纳银入监生资格 |
| **合计** | **约 280.0** | |

### 2.4 杨嗣昌的"四正六隅、十面张网"方案

- 杨嗣昌提出以十面合围的军事方案消灭农民军
- 需要增兵十二万人，年军饷约 280 万两
- 崇祯帝批准，下令加征剿饷
- 实际征收（崇祯十一年）：均粮征银 180 万余两 + 溢地银 45 万余两 + 裁站银 20 万 + 盐课 16 万余 + 其他 10 万 = 合计约 **271 万两**

### 2.5 征收问题

- 剿饷名义上在崇祯十三年停征，但实际上很多地方仍在继续征收
- 停征后其负担并入了练饷或以其他名目继续征收
- 实际征收年限约 **3–4 年**

## 三、练饷

### 3.1 开征背景

- 崇祯十一年（1638年）九月至十二年三月，清军绕道蒙古大举入关
- 清兵深入京畿、山东，烧杀掳掠（"己巳之变"后的再次入关）
- 九边防线岌岌可危，杨嗣昌提议加练边兵

### 3.2 开征时间与税额

| 项目 | 内容 |
|------|------|
| 开征时间 | **崇祯十二年（1639年）** |
| 提议者 | 兵部尚书**杨嗣昌** |
| 名义用途 | 训练边防军队、加强九边防御力量 |
| 年征收总额 | **730 万两** |
| 征收来源 | 地亩加征（每亩加**1分**）+ 关税 + 盐课 + 契税 + 典税 + 官吏赃罚银 + 公费节省等 |
| 停征时间 | 未停征，直至明亡 |

### 3.3 练饷的征收结构

练饷 730 万两的来源构成：

| 来源 | 金额（万两） | 备注 |
|------|------------|------|
| 地亩加征（每亩 1 分） | **481.2** | 约占总额 66%，覆盖全国十三省 572 万余顷 |
| 赋役核实 | **70.0** | 清查应征未征赋役 |
| 裁减驿站 | **50.0** | 裁撤冗余驿站节省经费 |
| 盐课加征 | **40.0** | 各盐区分摊 |
| 契税 | **24.0** | |
| 赃罚 | **20.0** | 官吏罚没银两 |
| 关税加征 | **20.0** | 各钞关加征 |
| 典税 | **3.0** | |
| 公费节省 | **1.0** | 削减地方公费 |
| **合计** | **约 730** | |

练饷的实际支出逐年膨胀：崇祯十五年（1642），年用练饷达 **870 万 9000 余两**，超原定加派 **140 多万两**。

## 四、三饷合计

### 4.1 总额汇总

| 饷名 | 开征时间 | 年征总额（万两） | 持续年数 |
|------|---------|----------------|---------|
| 辽饷 | 万历四十六年（1618） | 520 → 660 → 740 | 26年（至明亡） |
| 剿饷 | 崇祯十年（1637） | 280 | 约3–4年 |
| 练饷 | 崇祯十二年（1639） | 730 | 5年（至明亡） |
| **三饷合计（崇祯末年）** | — | **约 1,670–1,750 万两** | — |

实际上，加上各种附加费和地方加派，百姓实际负担的总额可能超过 **2,000 万两**。

### 4.2 与正常田赋的对比

| 项目 | 金额/数量 |
|------|----------|
| 明朝正常年份田赋总额 | 约 2,600 万石（折银约 1,300 万两） |
| 三饷加派总额 | 约 1,670–2,000 万两 |
| 加派占正赋比例 | **约 130%–150%** |

也就是说，三饷加派几乎等于在原来的田赋基础上**再征一倍多**。

- 但北方省份如**陕西、河南、山东**虽然分摊绝对额较低，但这些地区**旱灾、蝗灾频发**，百姓更加无力负担
- 陕西是农民起义的发源地，正是因为天灾人祸叠加、三饷加派将百姓逼上绝路

## 六、对百姓的实际负担

### 6.1 一个普通农户的赋税清单（崇祯末年示例）

以一个拥有 20 亩中等良田的自耕农为例：

| 项目 | 金额/数量 | 备注 |
|------|----------|------|
| 正赋（田赋折银） | 约 0.4–0.6 两 | 一条鞭法后统一折银 |
| 辽饷加派 | 0.24 两（20亩 × 1.2分） | 崇祯四年标准 |
| 练饷加派 | 0.20 两（20亩 × 1分） | 崇祯十二年起 |
| 剿饷加派 | 约 0.10 两 | 名义停征但实际继续 |
| 各种火耗、加成 | 约 0.15–0.30 两 | 地方额外加征 |
| **合计** | **约 1.1–1.4 两** | |

### 6.2 负担加重的倍数

- 一条鞭法时期（万历初年），20 亩田的赋税约 0.4–0.6 两
- 崇祯末年同样 20 亩田的实际负担约 1.1–1.4 两
- 赋税负担增加了 **2–3 倍**
- 加上天灾导致的粮食减产和银贵谷贱问题，实际负担更重

### 6.3 预征制度

- 每年预征次年加派的十分之三
- 集中在正月、二月征收——恰好是农村"青黄不接"最困难的时期
- 等于提前透支民间余粮，使百姓在最困难时节被迫卖粮纳银

### 6.4 具体案例

- **山东**：部分地方百姓负担增加 **六倍** 之多
- **南直隶苏松地区**：一亩民田价值七八两银子，但每年需纳饷至十两——一年赋税已超过土地本身价值
- 各地贪官加派幅度极不均匀，"暗为加派者，不知几百千万"，"私派多于正赋"

### 6.5 社会后果

- **流民激增**：无力缴税的农民弃田逃亡，加入流民大军
- **起义蔓延**：陕西、河南、湖广等地农民纷纷起义
- **恶性循环**：逃亡者越多，留下的百姓需分摊更多的赋税（"带征"），"赋愈重则逃愈多，逃愈多则赋愈重"
- **正税萎缩**：三饷加派最终也无法足额征收，北方战乱省份实际征收率不足正额的 50%
- **军心涣散**：军饷不足导致欠饷，士兵哗变或投降
- **起义号召**：李自成以"三年免征""均田免赋"为号召，获得广泛民众支持

## 七、历史评价

### 7.1 "饮鸩止渴"

三饷加派被后世普遍视为"饮鸩止渴"之策：
- 短期内筹集了部分军费
- 但从根本上动摇了统治基础
- 将原本可以争取的中间力量（自耕农）推向了对立面

### 7.2 三饷逐步加派的完整编年

| 年份 | 事件 | 亩征额度变化 | 年征总额（万两） |
|------|------|------------|--------------|
| 万历四十六年（1618） | 李汝华首议辽饷，每亩加三厘五毫 | 3.5厘 | 约200 |
| 万历四十七年（1619） | 萨尔浒之败，再加三厘五毫 | 累计7厘 | 约400 |
| 万历四十八年（1620） | 沈辽失陷，再加二厘 | 累计9厘 | 520 |
| 天启年间 | 辽饷维持不变 | 9厘 | 520 |
| 崇祯四年（1631） | 辽饷加至一分二厘 | 1分2厘 | 660（含杂项达740） |
| 崇祯十年（1637） | 杨嗣昌议增剿饷 | 另加 | 剿饷280 |
| 崇祯十二年（1639） | 杨嗣昌议增练饷，每亩加一分 | 另加1分 | 练饷730 |
| 崇祯十三年（1640） | 剿饷名义停征 | — | 实际并入其他名目 |
| 崇祯十七年（1644） | 明亡 | — | 三饷合计约1670-1750 |

### 7.3 征收方式的变化

- **初期**（万历四十六年）："因粮输饷"，按各地原有税粮数额加派
- **崇祯四年**：改为按田亩统一加征，不再按原税粮比例
- **崇祯十二年**：杨嗣昌主张"一律按亩均输"，不分税粮多少，按田亩面积统一加征
- 四川地区通过"行税银""牙行银"等名目变相增税
- 实际征收中胥吏额外索取达正赋数倍

### 7.4 实际征收与预期的差距

三饷虽名义税额巨大，但实际到手的军饷远低于预期：

| 项目 | 名义额度（万两） | 实际征收（万两，估） | 差距原因 |
|------|----------------|-------------------|---------|
| 辽饷 | 740 | 约400-500 | 逃亡、荒废、拖欠 |
| 剿饷 | 280 | 约150-200 | 征期短，阻力大 |
| 练饷 | 730 | 约300-400 | 大面积抛荒，北方战区尤甚 |

### 7.5 清朝的态度

- 清朝入关后，顺治元年（1644年）即宣布**废除三饷**
- 以此收买人心，争取汉族民众支持
- 但实际上，清初不久即恢复了辽饷的征收（改名"新增地丁银"等）
- 顺治朝后期恢复征收，实质上延续了明末的加税格局

## 八、三饷对民间的影响数据

### 8.1 银贵谷贱问题

三饷加派恰逢明末"银荒"（白银流入减少），导致银贵谷贱：
- 崇祯初年：1两银可买米约**2石**
- 崇祯末年：1两银可买米约**4-5石**
- 百姓需卖出更多粮食才能换得缴税所需的白银
- 实际赋税负担因银价上涨而翻倍

### 8.2 逃亡与抛荒

- 陕西：崇祯年间大面积抛荒，部分州县"十室九空"
- 河南：崇祯十三年（1640年）以后，大量农民逃入起义军
- 湖广：崇祯中期已出现"除衙蠧"民变，反抗加派

### 8.3 "带征"恶性循环

逃亡户的赋税被转嫁给剩余住户（"带征"），形成恶性循环：
1. 赋税过重 → 部分农户逃亡
2. 逃亡户的税额由留存户分摊 → 留存户负担加重
3. 更多农户不堪重负逃亡 → 分摊面进一步缩小
4. 最终剩余农户负担增至不可承受 → 大面积逃亡或投入起义军

## 存疑 & 待查

1. **各省辽饷的精确分摊数据**：上文估算基于每亩 9 厘的标准，实际各省可能有调整和减免，需查《崇祯实录》和户部档案
2. **剿饷的实际停征情况**：崇祯十三年"停征"后各地是否真正停止征收，争议很大
3. **练饷 730 万两的来源结构明细**：地亩加征与其他来源的精确比例需查原始奏疏
4. **三饷的实际征收率**：名义加派金额与实际到手军饷之间的差距，各省各年数据极不完整
5. **崇祯朝各省逃亡和抛荒的田亩数据**：这直接影响三饷的实际征收基数
6. **"带征"制度的具体运作**：逃亡户的赋税如何转嫁给留存户的具体规则
