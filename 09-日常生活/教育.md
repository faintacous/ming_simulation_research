# 教育

> 聚焦嘉靖至崇祯（1521-1644），以《大明会典》《明史·选举志》及地方志为主要依据。

---

## 一、私塾

### 1. 类型

明代私塾主要分三类：

| 类型 | 说明 |
|------|------|
| 家塾 | 富裕家庭请先生到家中教授子弟 |
| 村塾/族塾 | 一村或一族合资延请先生，在祠堂或专设学舍中教学 |
| 散馆 | 先生自行设馆招收学生 |

### 2. 入学年龄

- 一般五六岁至八岁入学开蒙
- 最常见的入学年龄为六至七岁
- 也有晚至十二三岁才入塾者
- 学生年龄差异很大，同一塾中自五六岁至二十岁均有，以十二三岁以下居多
- 开蒙日通常选在正月或二月的吉日

### 3. 教材与课程

#### 启蒙阶段（识字）
- **"三百千"**：《三字经》《百家姓》《千字文》——识字入门的基本教材
- 《神童诗》《千家诗》——诗歌启蒙
- 《增广贤文》《幼学琼林》——通识读物
- 此阶段以认字、描红、习字为主，兼背诵

#### 经学阶段
- 学习顺序通常为：
  1. 《孝经》——开篇即讲孝道
  2. 《大学》——四书之首，纲领性文本
  3. 《中庸》
  4. 《论语》
  5. 《孟子》
  6. 四书学完后进入五经
- 以朱熹《四书集注》为标准注本（科举指定用书）
- 教学方法：先逐句教读（先生读一句，学生跟读一句），再背诵，然后讲解义理

#### 作文阶段
- 学习八股文写作（为科举应试准备）
- 练习对联、诗词
- 习字（楷书为主）

### 4. 学费（束脩）

#### "束脩"的含义
- 原义为一束干肉（十条干肉），出自《论语·述而》："自行束脩以上，吾未尝无诲焉"
- 明代"束脩"已演变为泛指学费的统称，不限于实物

#### 学费构成
- **入门礼（拜师礼）**：
  - 入学第一天行拜师礼，供奉孔子牌位
  - 学生向先生行叩拜礼，奉上束脩
  - 入门礼包括：干肉、酒、布帛等实物，或折合银两
- **常年束脩**：
  - 按年或按季支付
  - 形式多样：银钱、粮食、布帛等均可
- **节敬**：
  - 每逢端午、中秋、春节等节日，另送礼物或银钱
  - 是束脩的重要补充

#### 学费水平
- 差异极大，取决于先生水平、地域、家庭经济
- **低端**：乡村散馆，年束脩仅银1-3两（或折合等量粮食）
- **中端**：中等私塾，年束脩约银5-10两
- **高端**：延请名师到家中教学，年束脩可达银20-50两以上
- 据明清徽州文书，康熙年间婺源县塾师詹元相的年收入仅银1两6钱至1两9钱，说明底层塾师收入极低

### 5. 先生（塾师）的收入与地位

#### 收入来源
- 束脩为主要收入
- 节敬为重要补充
- 部分塾师兼做抄写、代笔书信等工作补贴收入
- 收入并不完全来自执教授徒

#### 社会地位
- 塾师多为科举落第的秀才或老年生员
- 社会地位不高不低：高于农工商，但远低于进士举人
- "穷酸秀才"教私塾为明代社会的常见现象
- 部分程度差的先生"只能教教识字课本'三百千'"

#### 教学日程
- 每日清晨入塾，下午放学
- 冬季天短早放，夏季天长晚放
- 每月一般休息两三天
- 春节、端午、中秋等大节放假
- 农忙时乡村私塾多放假（"农隙入学"）

## 二、社学

### 1. 制度规定

#### 设立诏令
- 洪武八年（1375），朱元璋下诏天下设立社学
- 规定：**每五十家设社学一处**
- 社学为官办性质，免收学费
- 目的是普及教化，使基层百姓子弟"知书达理"

#### 分布规模
- 洪武年间社学数量极大：
  - 仅南京一地即设小学（含社学）**2919所**
  - 南京上元、江宁两县，每个坊厢都设有社学
  - 溧水县有11所社学
  - 句容县在洪武到永乐年间有16所社学
- 这些数据说明洪武时期社学覆盖面极广

### 2. 教官与师资

- "择通晓经书者为社师"
- 社师由地方官选拔任命
- 待遇由官府提供（免除徭役、供给口粮）
- 社师水平参差不齐

### 3. 课程内容

| 类别 | 内容 |
|------|------|
| 经典教育 | 《大诰》《御制大明律令》等法律文书（必修） |
| 识字读书 | 《三字经》《百家姓》《千字文》等 |
| 伦理教化 | 《孝经》《小学》等 |
| 实用知识 | 农桑耕种等农业知识 |
| 礼仪规范 | 冠婚丧祭之礼 |

- 社学教育特别注重宣讲《大诰》和法律，这是与私塾的重要区别
- 朱元璋要求社学学生能背诵《大诰》中的条目

### 4. 实际运行

- 洪武至永乐年间社学最为兴盛
- 宣德以后逐渐衰落：
  - 许多社学有名无实
  - 社师空缺无人补充
  - 经费无着
  - 学舍破败
- 嘉靖以后，社学多已名存实亡
- 部分地区地方官有零星恢复之举

## 三、书院

### 1. 明代书院的兴衰

#### 发展阶段
| 时期 | 状况 |
|------|------|
| 明初（洪武至正统） | 注重官学而抑制书院，书院数量极少 |
| 弘治以后（1488-） | 书院复起 |
| 正德至嘉靖（1506-1566） | 书院蓬勃发展，嘉靖一朝新建及修复书院达 **596所**，远超以往各朝 |
| 嘉靖末至隆庆 | 嘉靖四十四年（1565）下令禁毁书院，短暂压制 |
| 万历至天启 | 书院再度复兴，但天启五年（1625）魏忠贤下令拆毁天下书院 |
| 崇祯 | 魏忠贤倒台后书院再度恢复 |

#### 禁毁书院的原因
- 朝廷认为书院讲学可能形成朋党
- 嘉靖禁书院与张璁等人打击王阳明学派有关
- 天启禁书院则是魏忠贤打击东林党的手段

### 2. 主要书院列表

| 书院名称 | 所在地 | 创建/复兴时间 | 特点 |
|----------|--------|--------------|------|
| 白鹿洞书院 | 江西庐山 | 南唐始建，明代多次修复 | 朱熹制定学规，为天下书院楷模 |
| 岳麓书院 | 湖南长沙 | 北宋始建，明代持续运营 | 王阳明心学的重要传播地 |
| 东林书院 | 江苏无锡 | 万历三十二年（1604）修复 | 晚明影响最大的书院 |
| 嵩阳书院 | 河南登封 | 北宋始建，明代修复 | 理学传统 |
| 首善书院 | 北京 | 万历年间建 | 京城讲学重地 |
| 江右王门书院群 | 江西各地 | 嘉靖年间大量涌现 | 王阳明弟子讲学基地 |
| 甘泉书院 | 广东增城 | 嘉靖年间 | 湛若水学派 |
| 证人书院 | 浙江绍兴 | 万历年间 | 刘宗周讲学之所 |

### 3. 东林书院运作方式（典型案例）

#### 修复过程
- 万历三十二年（1604）四月动工，九月完工
- 由顾宪成、高攀龙等人发起
- 地方官府同意在原址修复，提供部分支持

#### 讲会制度
- **大会**：每年春秋各举办一次大讲会，会期三天
- **小会**：除天寒酷暑外，每月九日、十日、十一日举办小会
- **会主**：每次讲会设会主一名，主持讲学
- **讲学内容**：
  - 通常主讲"四书"，以某一篇为开头
  - 兼及先秦诸子、历代儒释道及哲学、文学、史学等
  - 讲完后听讲人提问、辩论
  - 重大理论成果编辑刊印，如《会语》《东林商语》等

#### 《东林会约》
- 依据朱熹《白鹿洞书院学规》订立
- 规范讲学制度
- 规定"愿赴者至，不必遍启"——自由参加，不强制通知

### 4. 经费来源

| 来源 | 说明 |
|------|------|
| 学田 | 书院拥有的田产，租金为主要经费 |
| 官府拨款 | 地方官员不定期拨付 |
| 士绅捐资 | 地方绅士、商人捐赠 |
| 刊刻收入 | 书院刻书出售的收入 |

- 经费来源不稳定，随政治形势和地方官态度而变化
- 学田是最稳定的经费来源

## 四、府州县学（地方儒学）

### 1. 学校设置

- 明代在各级行政单位设立官学：
  - **府学**：设于各府治所
  - **州学**：设于各州治所
  - **县学**：设于各县治所
  - **卫学**：设于军事卫所（供军户子弟就读）

### 2. 教官编制

| 学校 | 教官设置 |
|------|----------|
| 府学 | 教授一人（从九品），训导四人 |
| 州学 | 学正一人，训导三人 |
| 县学 | 教谕一人，训导二人 |

- 教官品级极低（从九品或未入流），社会地位不高
- 教授/学正/教谕为正职，训导为辅助
- 教官由朝廷选派或地方推荐

### 3. 生员名额

#### 三类生员

| 类别 | 说明 | 待遇 |
|------|------|------|
| 廪膳生员（廪生） | 最高等，由官府供给廪食 | 每月廪米六斗 |
| 增广生员（增生） | 次等，名额在廪生基础上扩充 | 无廪食供给 |
| 附学生员（附生） | 新入学者，名额不定 | 无廪食供给 |

#### 各级名额

| 学校 | 廪膳生 | 增广生 |
|------|--------|--------|
| 京府学 | 60人 | 60人 |
| 外府学 | 40人 | 40人 |
| 州学 | 30人 | 30人 |
| 县学 | 20人 | 20人 |

- 洪武年间初定生员名额为：府学40人，州学30人，县学20人
- 宣德年间扩大名额，增设增广生，与廪膳生同数
- 附学生名额不定，实际招收人数远超定额
- 明中后期，各地生员总数大幅膨胀

#### 升降制度
- 初入学者皆为附学生
- 考试合格后可递升为增广生、廪膳生
- 廪生出缺则增生递补，增生出缺则附生递补

### 4. 课程与日常

- 以《四书五经》为核心课程
- 兼习《大明律》《大诰》
- 练习八股文写作
- 每月有月课，定期考核
- 学生须住校（理论上），实际多走读
- 生员享有一定特权：免徭役、见官不跪等

## 五、岁考六等制度

### 1. 考试体系

明代对生员的考核分两种：

| 考试 | 主持者 | 频率 | 目的 |
|------|--------|------|------|
| 岁考 | 提学官 | 每年一次 | 考核学业，决定升降 |
| 科考 | 提学官 | 乡试前一年举行 | 选拔参加乡试资格 |

### 2. 岁考六等评定

岁考成绩分为六等，各等有不同的奖惩：

| 等级 | 评定 | 结果 |
|------|------|------|
| **一等** | 优秀 | 附生可升增生，增生可升廪生；取得参加科考资格 |
| **二等** | 良好 | 同一等，可升补、可参加科考 |
| **三等** | 平常 | 维持原等，不升不降 |
| **四等** | 差劣 | 受责处（体罚警告） |
| **五等** | 很差 | 廪生降为增生，增生降为附生，附生降为青衣（半开除状态） |
| **六等** | 极差 | **黜革**——开除生员学籍，失去一切生员特权 |

### 3. 考试后果的严重性

- 被黜革（六等）意味着：
  - 失去"秀才"身份
  - 失去免徭役特权
  - 失去见官不跪特权
  - 终身无法再参加科举（除非重新通过县试、府试、院试）
  - 社会地位急剧下降
- 五等的"降为青衣"亦是严重处分：
  - "青衣"即穿青色衣服，区别于正式生员的蓝色方巾儒服
  - 相当于"留校察看"

### 4. 实际执行

- 岁考为生员最大的心理压力来源
- 提学官有很大的自由裁量权
- 部分提学官严苛，一次岁考黜革数十人
- 部分提学官宽厚，极少给六等
- 作弊、请托之风在晚明日趋严重

## 六、教育制度的整体特点

### 1. 科举导向

- 从私塾到府州县学，教育的核心目标是培养科举人才
- 课程设置、考核标准都围绕科举考试
- 八股文训练占据大量时间

### 2. 阶层分化

| 阶层 | 教育途径 |
|------|----------|
| 贵族官宦 | 家塾（名师教授）→ 国子监 → 科举 |
| 中等之家 | 私塾 → 府州县学（生员）→ 科举 |
| 贫困之家 | 社学（免费但质量差）→ 少数进入府州县学 |
| 最贫困者 | 无受教育机会 |

### 3. 教育覆盖率

- 明代教育覆盖率远超前代
- 洪武年间社学网络理论上可覆盖绝大多数农村
- 但实际运行中覆盖率大打折扣
- 据估算，明中后期男性识字率约为10%-20%
- 女性基本被排除在正规教育之外（但富家女子有家庭教育）
