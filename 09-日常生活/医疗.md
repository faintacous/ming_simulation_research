# 医疗

> 聚焦嘉靖至崇祯（1521-1644），以《大明会典》《明代医事制度》《本草纲目》及相关研究为主要依据。

---

## 一、太医院

### 1. 机构沿革

- 洪武元年（1368）正式设立太医院，为中央医药管理机构
- 分南京、北京两院（永乐迁都后）
- 隶属于六部之外，地位独特

### 2. 官制

#### 历代变化

| 时期 | 长官设置 |
|------|----------|
| 洪武元年 | 设太医令（正五品），太医丞（正六品） |
| 洪武十四年（1381） | 改设太医令1人，太医丞1人，吏目1人，御医4人 |
| 洪武二十二年（1389） | 改太医令为院使（正五品），太医丞为院判（正六品） |
| 隆庆五年（1571） | 最终确定编制：院使1人，院判2人，御医10人，吏目10人 |

#### 嘉靖至崇祯年间的编制（隆庆五年定制）

| 职位 | 品级 | 人数 | 职责 |
|------|------|------|------|
| 院使 | 正五品 | 1人 | 太医院最高长官 |
| 院判 | 正六品 | 2人 | 副长官 |
| 御医 | 正八品 | 10人 | 为皇帝及宫廷诊治 |
| 吏目 | 从九品 | 10人 | 协助御医 |
| 医士 | 无品级 | 若干 | 基层医疗人员 |
| 医生（学生） | 无品级 | 若干 | 在院学习的学生 |

### 3. 十三科分科表

明代太医院下设十三个临床专科，是中国古代医学分科最细的时期之一：

| 序号 | 科名 | 对应现代科室 | 御医/吏目配置 |
|------|------|-------------|--------------|
| 1 | 大方脉 | 内科 | 5人 |
| 2 | 小方脉 | 儿科 | 2人 |
| 3 | 妇人 | 妇产科 | 2人 |
| 4 | 疮疡 | 外科 | 1人 |
| 5 | 针灸 | 针灸科 | 1人 |
| 6 | 眼 | 眼科 | 1人 |
| 7 | 口齿 | 口腔科 | 1人 |
| 8 | 咽喉 | 耳鼻喉科 | 1人 |
| 9 | 伤寒 | 传染病/热病科 | 4人 |
| 10 | 接骨（正骨） | 骨伤科 | 1人 |
| 11 | 金镞 | 军事外伤科 | （后期渐废） |

- 十三科中以大方脉（内科）和伤寒科配置最多
- 金镞、按摩、祝由三科在嘉靖以后逐渐被合并或裁撤
- "医官、医生、医士，专科肄业"——每个医务人员只学一个专科

### 4. 附属机构

| 机构 | 职能 |
|------|------|
| 生药库 | 收储各地上贡的生药材 |
| 御药房 | 配制宫中用药 |
| 御药局 | 管理药物调配 |

## 二、医术考核制度

### 1. 入院途径

明代进入太医院有三个主要途径：

#### 医户子弟选拔
- 明代沿袭元制，将户籍分为民、军、医、儒、灶等类
- 医户子弟世代业医，子承父业
- 太医院定期从医户子弟中选拔优秀者入院学习

#### 医学考试
- 弘治五年（1492）明确规定：
  - "凡医家子弟，查照旧例，选入本院"
  - 推选二三名堪任教师教习医术
  - **每季考试**一次
  - "通晓本科者，收充医士，食粮当差"
  - "未通晓者，仍令习学一年再试"
  - **"三试不中者，黜之"**——三次考试不及格者开除

#### 地方推荐
- 各地方官可推荐医术精湛者入太医院
- 需经太医院复核考试

### 2. 在职考核

- **三年一小考，五年一大考**
  - "凡医家子弟，择师而教之。三年、五年一试、再试、三试，乃黜陟之"
- 考核内容包括理论和实践
- 考核结果决定升降

### 3. 嘉靖年间太医院的危机

- 嘉靖帝沉迷修道炼丹
- 多次因方药问题处罚太医
- 嘉靖年间太医院医官屡遭廷杖、下狱
- 导致御医人人自危，不敢用重药

## 三、惠民药局

### 1. 设立与分布

- **洪武三年（1370）**在南京、北京及各府、州、县均设惠民药局
- 是明代最重要的官办医疗机构
- 理论上覆盖全国各级行政单位

#### 管理体制

| 层级 | 主管 | 人员配置 |
|------|------|----------|
| 两京惠民药局 | 太医院统辖 | 大使1人、副使1人 |
| 府惠民药局 | 府衙管辖 | 提领1人 |
| 州惠民药局 | 州衙管辖 | 医官1人 |
| 县惠民药局 | 县衙管辖 | 医官1人 |

### 2. 职能

- 为平民诊病卖药
- 贮备药物、调制成药
- **服务对象**："军民工匠贫病者"均可在惠民药局求医问药
- 遇疫病流行时免费提供药物

### 3. 药品与收费

- 惠民药局掌管常用成药的贮备和调制
- 常备药物包括：
  - 伤寒方剂（麻黄汤、桂枝汤等）
  - 疮疡外用药
  - 瘟疫方（如大黄、苍术等避秽药物）
  - 妇人产后方
  - 小儿惊风方
- **收费原则**：
  - 贫民免费或半价
  - 一般民众低价购买
  - 实际收费标准因时因地而异

### 4. 实际运行状况

- **问题突出**：
  - "许多药局有名无实"——设而不运营
  - "有医无药"——有医官但无药物储备
  - "局舍破败"——建筑年久失修
  - 经费无着落，药材供应不稳定
- 洪武至永乐年间运行较好，中后期逐渐荒废
- 嘉靖以后多数府州县惠民药局已名存实亡
- 万历十五年（1587）京城暴发瘟疫时，太医院奉旨"选良医分拨五城诊视给药"——说明紧急情况下仍依赖太医院而非惠民药局

## 四、民间医生

### 1. 培养途径

#### 世医（家传）
- 明代沿元制设"医户"，子孙世代业医
- 世医家庭将医术代代相传，"或父子相继，或翁婿相传"
- 许多世医将经验编写成医学读物作为家传教材
- 17世纪有多种这类医书刊行
- 世医极有利于医学专门化发展

#### 师徒传授
- 拜入名医门下学习
- 学徒期通常三至五年
- 跟师学习理论、观摩诊疗、练习配药
- 出师后独立行医

#### 自学成医
- 通过研读医书自学成才
- 明代医书刊印量大增，降低了自学门槛
- 但"非正统"自学医生常受正统世医批评
- 被批评为"缺乏学术训练的庸医"

#### 儒医（弃儒从医）
- 科举不第的读书人转而学医
- "不为良相，便为良医"——儒生从医有正当性
- 儒医通常文化素养高，善于著书立说
- 明代许多重要医学著作出自儒医之手

### 2. 医户制度的困境

- 医户社会地位低下，"与乐工、厨师、班匠等同"
- 许多医户衣食不得温饱
- 常有**逃户**现象——医户逃避身份管束
- 这是明代特有的制度性问题

### 3. 民间医生的收入

- 收入差异极大：
  - 名医收入可观，诊金从银数钱至数两不等
  - 普通医生收入微薄
  - 游方医生（走街串巷行医者）收入最不稳定
- 部分医生兼营药铺以增加收入
- "医不叩门"——医生一般不主动上门，等病人上门求诊
- 出诊费另算，路程远则费用高

## 五、药铺经营

### 1. 药铺类型

| 类型 | 说明 |
|------|------|
| 前店后坊 | 前面卖药、后面制药的小型药铺，最为常见 |
| 大药铺 | 拥有较大规模的药材批发兼零售铺面 |
| 专科药铺 | 专营某类药物（如眼药铺、膏药铺） |

### 2. 经营模式

- 药铺通常兼具诊病和卖药功能
- 坐堂医生为药铺招徕顾客
- 药铺自行采购原药材，炮制加工后出售
- 成药和丸散膏丹为重要利润来源
- 名药铺有自己的秘方和招牌药

### 3. 药材贸易

- 明代药材贸易网络发达
- 重要药材集散地：
  - 安徽亳州
  - 河北安国（祁州）
  - 河南禹州
  - 江西樟树
- 长途贩运药材的商人称为"药商"或"药客"

## 六、常见疾病

### 1. 主要疾病谱

| 疾病类别 | 具体病种 |
|----------|----------|
| 传染病 | 伤寒（泛指发热性疾病）、瘟疫、疟疾、痢疾、天花 |
| 寄生虫病 | 血吸虫病（南方）、蛔虫病 |
| 皮肤病 | 疥疮、癣 |
| 消化系统 | 腹泻、腹痛、食积 |
| 呼吸系统 | 咳嗽、哮喘 |
| 妇科 | 产褥热、难产 |
| 小儿科 | 惊风、麻疹、天花 |
| 外伤 | 骨折、刀伤、烧伤 |
| 眼病 | 白内障、翳膜 |

### 2. 死亡率最高的疾病
- **瘟疫**（鼠疫、霍乱等传染病）
- **天花**：明代无有效预防手段（种痘术尚在早期探索中）
- **产褥热**：产妇死亡率极高
- **疟疾**：南方湿热地区常见

## 七、李时珍与《本草纲目》

### 1. 李时珍生平

| 项目 | 内容 |
|------|------|
| 姓名 | 李时珍，字东璧，号濒湖 |
| 生卒 | 1518-1593（正德十三年至万历二十一年） |
| 籍贯 | 湖广蕲州（今湖北蕲春） |
| 家世 | 世代医家，祖父为铃医，父亲李言闻为当地名医 |
| 科举经历 | 三次参加乡试未中，遂弃儒从医 |
| 仕途 | 嘉靖年间曾被推荐入太医院任职，约一年后辞归 |

### 2. 《本草纲目》编纂过程

- **起因**：李时珍发现前代本草著作多有错误、遗漏和混淆
- **编纂时间**：自嘉靖三十一年（1552）开始，至万历六年（1578）定稿
  - 前后历时**27年**
  - 经过**三次改写**
- **工作方法**：
  - 参考古今医学文献**800余种**
  - 多次实地考察：走遍湖广、安徽、河南、河北等地
  - 亲自采集药物标本
  - 向药农、猎人、渔民等采访实际用药经验
  - 亲身试验部分药物

### 3. 《本草纲目》内容

| 项目 | 数据 |
|------|------|
| 全书字数 | 约190万字 |
| 收录药物 | **1892种** |
| 其中新增 | 374种（李时珍新发现或新收录） |
| 收集方剂 | **11096个** |
| 插图 | **1160幅** |
| 分类 | 16部、60类 |
| 卷数 | 52卷 |

#### 十六部分类
水、火、土、金石、草、谷、菜、果、木、服器、虫、鳞、介、禽、兽、人

### 4. 刊刻与传播

- 万历六年（1578）定稿后，长期未能刊印
- 万历十八年（1590）在南京藏书家兼出版商**胡承龙**资助下得以刻印
- 万历二十一年（1593）李时珍去世，**未及看到自己的著作正式出版**
- 万历二十四年（1596）在南京正式刊行（"金陵本"）
- 此后迅速传播，被译为日文、拉丁文、英文等
- 2011年入选联合国教科文组织"世界记忆名录"

## 八、明末大鼠疫

### 1. 爆发与传播

| 时间 | 事件 |
|------|------|
| 崇祯六年（1633） | 鼠疫始发于山西 |
| 崇祯六年至十三年 | 在山西蔓延扩散 |
| 崇祯十四年（1641） | 传播至河北（北直隶） |
| 崇祯十四年 | 传入北京，造成大规模死亡 |
| 崇祯十六年（1643） | 北京疫情达到高峰 |
| 崇祯十七年（1644） | 李自成攻破北京，明亡 |

### 2. 死亡规模数据

#### 北京地区
- 据台湾明史专家**邱仲麟**估算：
  - 自鼠疫始至李自成攻破北京，约**20万**北京人死亡
  - 占当时北京总人口的**五分之一至四分之一**
- 崇祯十六年（1643）8月至12月，死亡人数高达全城人口的五分之一
- 每天死亡上万人
- 城门口被运出的棺材堵塞

#### 北京驻军损失
- 驻守京师的明军原有约10万人
- 大疫过后仅余5万多人
- 军队减员近半，严重削弱了北京的防御能力
- 这是李自成能够轻易攻破北京的重要原因之一

#### 全国影响
- **华北三省（陕西、山西、河北）**：死亡人数至少在**千万人以上**（据曹树基等学者统计）
- 北京城"近60%的人死于鼠疫"（部分学者估计）
- 苏州23万户减至仅5万户
- "全国人口死亡过半"——这一说法可能有夸大，但死亡规模确实极为惊人

### 3. 疫情细节

#### 山西（1633年始）
- 崇祯六年（1633）山西大疫
- "兴县死者枕藉"
- "大同瘟疫大作，十室九病"
- 死者相枕，十室九空

#### 北京（1643-1644）
- 症状描述："疙瘩瘟"——患者腋下或颈部出现硬块（淋巴结肿大，典型鼠疫症状）
- "朝染夕亡，日每不下数百人"
- 沿街的死尸无人收殓
- "有棺无力买者委弃通衢"
- 城中哀号之声不绝

### 4. 疫情的军事影响

- 鼠疫随明军与李自成军队的交战而扩散
- 驻京明军战斗力因疫病大幅下降
- 崇祯十七年（1644）三月，李自成攻北京时几乎未遇有效抵抗
- **鼠疫被认为是明朝灭亡的重要推手之一**

### 5. 医学应对

- 明末吴有性（吴又可）于崇祯十五年（1642）撰写**《温疫论》**
  - 这是中国医学史上第一部传染病学专著
  - 提出"戾气"学说——认为瘟疫由天地间的特殊致病因子（"戾气"/"杂气"）引起
  - 已接触到每种传染病有其特殊致病因子这一实质问题
  - 主张用达原饮等方剂治疗
  - 在理论上远超同时代的认知水平
- 但面对如此大规模的鼠疫，医疗手段基本无能为力

## 九、地方医学机构

### 1. 府州县医学

- 各府设医学正科1人
- 各州设医学典科1人
- 各县设医学训科1人
- 职责：管理地方医生、监督惠民药局、处理医疗纠纷
- 实际影响力有限

### 2. 军事卫所医疗

- 各卫所设医官
- 负责军队的伤病救治
- 药材由军方供应
- 战时临时征调民间医生
