# 明代物价（嘉靖至崇祯，1521–1644）

## 概述

明代物价体系以白银为核心计量单位（嘉靖以后白银成为事实上的主要货币），辅以铜钱。从正统到崇祯的二百年间，物价总体呈上升趋势，尤其是米价——从正统年间每石约0.3两上涨至崇祯末年每石3–4两，涨幅超过十倍。这一轮长期通胀与白银大量流入、人口增长、土地兼并和自然灾害等因素密切相关。崇祯末年的物价崩溃是明朝灭亡的重要经济背景。

### 1.2 地域差异规律

1. **边镇 > 内地**：北方边镇米价长期高于内地，嘉靖三十八年边镇米价约为内地的**10倍**
2. **北方 > 南方**：南方产粮区米价低于北方
3. **灾年 > 丰年**：灾年米价可暴涨数倍至十倍
4. **战区 > 后方**：天启至崇祯间辽东战区米价极端高企

### 1.3 米价长期趋势

以北方边镇50年均值为例，明代米价的长期上升趋势清晰可见：

```
1440–1489:  0.489 两/石（基准）
1490–1539:  1.752 两/石（↑ 3.6 倍）
1540–1589:  2.658 两/石（↑ 5.4 倍）
1590–1625:  4.477 两/石（↑ 9.2 倍）
```

## 三、盐价

### 3.1 盐引制度与盐价

明代实行盐引制度，每一纸盐引代表**200斤**（一小引）盐的官方凭证 [来源：维基百科"盐引法"；百度百科"盐引"]。

**开中法**（洪武四年/1371定制）：商人赴边塞纳粮，按里程远近以1–5石粮食换取一小引盐引 [来源：同上]。

### 5.4 生活成本与工资对照

万历四十七年（1619），徐光启奏疏记载 [来源：知乎]：

> "都下贫民佣工，一日得钱二十四五文，仅足给食，三冬之月，衣不蔽体。"

按银钱比价（万历间约1两=600文），日薪24–25文约折银**0.04两**。这意味着一个北京普通佣工的日工资**仅够当天食物**，冬季连衣服都穿不起。

## 存疑 & 待查

1. 明代各地区茶叶、药材的具体价格数据缺乏系统整理
2. 明代书籍的价格（刻本、抄本）需从《万历会计录》和书坊记录中进一步挖掘
3. 白银流入量的逐年变化与物价的相关性需更精确的计量分析
4. 明代不同地区米价的年度序列数据（如有完整地方志记载）待补充
5. 明代城市与农村的物价差异缺乏系统比较
6. 崇祯年间物价崩溃的各地时间线（非同步崩溃）需进一步厘清
7. 明代手工业品（如瓷器、漆器）的价格分级数据待查
