# 丧葬制度

> 聚焦嘉靖至崇祯（1521-1644），以《大明会典》《大明律》《仪礼》及明人笔记为主要依据。

---

## 一、五服制度

五服是中国丧葬礼制的核心，以服丧之轻重区分亲属远近。按丧服由重至轻，依次为斩衰、齐衰、大功、小功、缌麻五等。

### 1. 斩衰（最重，服期三年/实际27个月）

#### 丧服特征
- 用最粗的生麻布制成，不缝边（"斩"即不缝缉）
- 衣裳左右和下边毛边外露
- 冠用粗麻绳束之
- 腰绖（腰带）用麻绳，粗如拇指
- 竹杖（丧杖）

#### 适用亲属关系
| 服丧者 | 为何人服 |
|---------|----------|
| 子（含嗣子） | 为父 |
| 子 | 为母（明代改制后） |
| 未嫁女 | 为父、为母 |
| 妻 | 为夫 |
| 承重孙 | 为祖父母（父已亡时） |

#### 明代重大变革：为母服斩衰三年
- **传统规定**：儒家经典《仪礼》规定，父在为母服齐衰杖期（一年），理由是"一家不可有二至尊"
- **唐《开元礼》调整**：不论父在与否，为母服齐衰三年（提升服期但未升级丧服等级）
- **洪武七年（1374）改制**：
  - 朱元璋下令：无论父在与否，子为母皆服**斩衰三年**
  - 这一改革将母丧的丧服等级从齐衰提升至斩衰，与父丧完全相同
  - 打破了"一家不可有二至尊"的传统原则
  - 朱元璋的理由是"教天下以孝"，认为母亲生育哺养之恩与父亲等同
- **历史意义**：这是中国丧服制度史上的重大变革，使母丧地位首次在制度上与父丧完全对等

### 2. 齐衰（次重，服期分四等）

#### 丧服特征
- 用次粗生麻布制成，缝边整齐（"齐"即缝缉整齐）
- 较斩衰稍细致，但仍为粗布

#### 四等细分

| 等级 | 服期 | 适用关系 |
|------|------|----------|
| 齐衰三年 | 三年（27个月） | 母为长子（明以前的传统） |
| 齐衰杖期 | 一年（持杖） | 夫为妻 |
| 齐衰不杖期 | 一年（不持杖） | 为祖父母、为伯叔父母、为兄弟、为未嫁姊妹 |
| 齐衰三月 | 三个月 | 为曾祖父母、为高祖父母 |

### 3. 大功（服期九个月）

#### 丧服特征
- 用熟麻布（经加工的较细麻布）制成
- "功"指缝制的工夫，"大功"即做工较粗

#### 适用亲属关系
| 服丧者 | 为何人服 |
|---------|----------|
| 本人 | 为已嫁姑（父亲的姐妹） |
| 本人 | 为已嫁姊妹 |
| 本人 | 为堂兄弟（同祖父的兄弟） |
| 本人 | 为未嫁堂姊妹 |
| 本人 | 为众子妇（非长子之妻） |

### 4. 小功（服期五个月）

#### 丧服特征
- 用较精细的熟麻布制成
- "小功"即做工较精细

#### 适用亲属关系
| 服丧者 | 为何人服 |
|---------|----------|
| 本人 | 为伯叔祖父母（祖父的兄弟及其妻） |
| 本人 | 为从祖伯叔父母（父亲的堂兄弟及其妻） |
| 本人 | 为从祖兄弟（父亲堂兄弟之子） |
| 本人 | 为已嫁从姊妹 |
| 本人 | 为外祖父母 |
| 本人 | 为母舅、母姨 |

### 5. 缌麻（最轻，服期三个月）

#### 丧服特征
- 用最精细的熟麻布制成，质地接近普通布料
- 是五服中最轻的丧服

#### 适用亲属关系
| 服丧者 | 为何人服 |
|---------|----------|
| 本人 | 为族曾祖父母 |
| 本人 | 为族祖父母 |
| 本人 | 为族父母 |
| 本人 | 为族兄弟 |
| 本人 | 为外甥、外甥女 |
| 本人 | 为表兄弟 |
| 本人 | 为岳父母 |
| 本人 | 为女婿 |

### 6. 出五服

- 超出缌麻三月范围的亲属即为"出五服"
- 出五服意味着在法律和礼制上不再有丧服义务
- 但仍可自愿穿素服吊唁

## 二、丁忧守制

### 1. 制度规定

- **丁忧**：官员父母去世后，必须离职回籍守丧
- **守制期限**：二十七个月（实际执行）
  - 理论上为三年，但按"以日易月"计算为二十七个月
  - 从闻丧之日起算
  - 期间不得为官、不得应举、不得嫁娶、不得宴饮作乐
- **适用范围**：
  - 在职官员（文武百官）
  - 已取得功名尚未入仕者
  - 明制规定父丧和母丧均须丁忧二十七个月（因洪武改制为母服斩衰三年）

### 2. 丁忧期间的行为规范

- 辞去官职，回到原籍守墓
- 居丧期间服粗布麻衣
- 不饮酒、不食肉（前三月尤严）
- 不得婚娶嫁女
- 不得参加宴会、听戏、游乐
- 不得与妻妾同房
- 居处简朴，寝苫枕块（睡草席、枕土块），三月后可稍改善
- 违者有严厉处分：
  - "匿丧不举"者杖六十
  - 居丧期间作乐者杖八十
  - 居丧嫁娶者杖一百

### 3. 夺情起复

#### 定义
- "夺情"即国家因特殊需要，不令官员去职守丧，以素服办公
- 是丁忧制度的例外条款

#### 明代规定
- 明朝明文规定："内外大小官员丁忧者，不许保奏夺情起复"
- 这意味着明代原则上**禁止夺情**，比前代更严格

#### 张居正夺情事件（万历五年，1577）

这是明代最著名也最具争议的夺情案：

- **背景**：万历五年九月，内阁首辅张居正之父张文明去世
- **经过**：
  - 14岁的万历帝本拟批准张居正丁忧
  - 张居正暗示希望被"夺情"留任
  - 万历帝下旨夺情，令张居正"在官守制"
- **争议**：
  - 翰林院编修吴中行、赵用贤等上疏反对，遭廷杖
  - 多名官员上书谴责张居正"忘亲贪位"
  - 被弹劾的官员或被杖责、或被贬谪
- **后果**：
  - 张居正穿吉服办公，引起更大不满
  - 此事成为张居正身后被清算的罪状之一
  - 此案使"夺情"在明代更加臭名昭著

#### 其他夺情案例
- 明代夺情案例极少，除张居正外几乎无成功先例
- 武将偶有夺情之例，争议较文官小

## 三、丧葬仪式流程

汉族传统丧礼分**殓、殡、葬**三大阶段，明代基本沿袭此制。

### 1. 初终（临终至断气）

- 将病危者移至正寝（堂屋正中）
- 家人侍候左右
- 断气后，举家号哭
- 以新棉置于口鼻前验证是否真正断气

### 2. 招魂（复）

- 由家人持死者衣服登屋顶
- 面朝北方呼唤死者姓名三次
- 将衣服抛下，覆盖于死者身上
- 象征呼唤灵魂归来

### 3. 沐浴

- 为死者净身
- 以温水擦洗全身
- 梳理头发
- 修剪指甲

### 4. 小殓

- 为死者穿寿衣
- 寿衣层数为单数（三、五、七层不等）
- 品官寿衣按品级穿戴朝服或常服
- 庶民穿平日最好的衣服
- 口中含珠或含玉（含饭/饭含）：
  - 品官含珠玉
  - 庶民含铜钱

### 5. 大殓（入棺）

- 择吉时将死者放入棺材
- 棺材材质按等级：
  - 品官多用楠木、柏木
  - 庶民用杉木、松木
  - 棺木厚度：品官四寸以上，庶民二三寸
- 棺内放置随葬物品
- 封棺后举家大哭

### 6. 殡（停灵）

- 停灵于家中正堂
- 设灵位、香案
- 亲友前来吊唁
- 停灵时间：
  - 品官停灵七日至四十九日不等
  - 庶民通常三至七日
  - 风水先生择葬日

### 7. 出殡（送葬）

- 择吉日送灵柩至墓地
- 送葬队伍：
  - 前导：开路神、纸扎仪仗
  - 乐队：吹鼓手
  - 灵柩：由杠夫抬行
  - 孝子：披麻戴孝，执杖步行，扶柩而哭
  - 亲友：按远近排列
  - 后随：纸钱、纸扎物品
- 品官出殡可用鼓乐、仪仗，品级越高排场越大

### 8. 安葬

- 灵柩入穴
- 孝子跪拜
- 填土封墓
- 立墓碑
- 焚烧纸扎物品
- 行"安土"之礼

### 9. 葬后礼仪

- **虞祭**：安葬当日或次日举行
- **卒哭**：百日后结束日常哭泣
- **小祥**：周年祭
- **大祥**：二周年祭
- **禫祭**：二十七月除服

## 四、墓葬形制（按品级）

### 1. 《大明会典》坟茔规制

洪武元年（1368）定制，按品级严格规定坟墓规格：

| 品级 | 坟茔面积 | 坟丘高度 | 石像生 |
|------|----------|----------|--------|
| 一品 | 九十步 | 一丈八尺 | 石人、石马、石虎、石羊各一对 + 望柱一对 |
| 二品 | 八十步 | 一丈四尺 | 同一品 |
| 三品 | 七十步 | 一丈二尺 | 减石人一对 |
| 四品 | 六十步 | 八尺 | 石望柱一对、石马、石虎各一对 |
| 五品 | 六十步 | 八尺 | 石望柱一对 |
| 六品 | 四十步 | 六尺 | 不许设石像生 |
| 七品以下 | 二十步 | 六尺 | 不许设石像生 |

### 2. 碑碣规制

| 品级 | 碑制 |
|------|------|
| 五品以上 | 石碑，螭首龟趺 |
| 六品以下 | 石碣，方趺（不用龟趺螭首） |

### 3. 墓室结构

- **皇陵**：地下宫殿，多室结构（如定陵地宫）
- **品官墓**：
  - 一至三品：可用砖室墓，有墓道、甬道、墓室
  - 四至五品：砖室或石室，规模缩小
  - 六品以下：多为砖圹或土圹
- **庶民墓**：土坑竖穴，棺木直接入土

### 4. 实际考古数据

- 万历帝定陵地宫：总面积1195平方米，由前殿、中殿、后殿及左右配殿组成
- 品官墓实际规格多超过定制，晚明尤甚
- 考古发现的明代中期以后品官墓，随葬品日趋丰厚

## 五、明代丧葬的特殊问题

### 1. 火葬与土葬之争

- 明初严禁火葬：《大明律》规定"尊长已死而烧其尸者，杖一百"
- 但实际上在江浙、福建等地火葬仍然流行
- 原因：土地紧张、棺木昂贵、佛教影响
- 万历年间有官员上书禁止火葬，效果有限

### 2. 厚葬之风

- 晚明厚葬之风盛行
- 丧葬费用高昂，中等之家一场丧事花费银数十两至百余两
- 有"生不起、死不起"之叹
- 奢侈者棺木、衣衾、随葬品远超定制

### 3. 风水信仰

- 明人极重风水择地
- 聘请风水先生看墓地，费用不菲
- 因风水争端引发的诉讼在地方档案中屡见不鲜
- 有人因等待"吉壤"而长期停柩不葬

### 4. 僧道参与

- 请僧人念经超度为普遍做法
- "做七"（每七天一次法事，共七七四十九天）
- 请道士做道场
- 这些活动大大增加了丧葬费用

## 六、丧葬与法律

### 1. 居丧违法行为处罚

| 违法行为 | 处罚 |
|----------|------|
| 匿丧不举 | 杖六十 |
| 居丧嫁娶 | 杖一百 |
| 居丧作乐 | 杖八十 |
| 居丧生子 | 不处罚但遭社会非议 |
| 居丧应试 | 取消成绩 |
| 弃亲之丧出家 | 杖一百 |

### 2. 丧服与刑法

- 五服制度直接关联刑法量刑
- 亲属间犯罪，服制越近则处罚越重（卑犯尊）或越轻（尊犯卑）
- 例：子杀父斩立决，缌麻亲杀人则按普通杀人从重
