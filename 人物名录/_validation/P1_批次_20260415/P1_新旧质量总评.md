## 5. 最终结论：今日修订整体是否带来质量提升？

**不是。6 人中仅 2 人（33%）显著提升，4 人（67%）持平或退化，其中 3 人明确退化。**

### 5.1 修订成功的模式（王喜姐、李时珍）

这两人的共性：
- **有具体、可操作的 04-12 问题**（蓬溪知县归子、128 万两珠宝来源、原姓黄定论）
- **问题本质是"删/改一个具体断言"**，而非"重构整篇叙事风格"
- **新版在古籍来源数上提升**（王喜姐 7→15、李时珍 5→6）
- **新版在百科依赖上下降**（王喜姐虽总数上升但结构化；李时珍 8→4）

### 5.2 修订失败的模式（张居正、朱翊钧、郑贵妃、海瑞）

这四人的共性：
- **旧版问题偏"风格/倾向"层面**（戏剧化、政治概念硬套、野史争议）
- **修订采用"Kimi 重写"而非"针对性改订"**——整篇推倒导致新错引入、已修旧错回滚
- **一手来源数下降**（张居正 11→10、海瑞 27→12、郑贵妃 20→13）
- **新增二手/自媒体来源**（朱翊钧增 3 个低学术源、海瑞引入"海南日报"）

### 5.3 信度评估

**Codex 评审本身的信度：高。** 判断依据：
1. **确实读了文件**：6 份评审全部给出具体行号、具体引文、具体卷次，不是泛泛而谈
2. **判断方向不一致**：6 人中同时出现"显著提升"与"退化"，说明不是机械评优或评差
3. **量化数据可复核**：卷次引用数、来源数等都可用 grep 二次验证
4. **第六次调用曾报"读不到文件"然后重试成功**，说明 Codex 不会编造通不过自检的答案——这反而是信度证据

**唯一保留**：Codex 的卷次统计口径（如"只计《明史》本身不含《明史纪事本末》"）与用户原始要求可能略有差异，但每份报告都显式声明了口径，不影响相对比较。

## 6. 未来改进建议

基于本批次观察：

1. **修订策略区分问题类型**。具体断言错误（如"蓬溪知县"、"128 万两"）适合"定点修订"；风格问题（戏剧化、政治概念）适合"风格改写 + 源头约束"。**不应把两类都喂给 Kimi 重写**。

2. **Kimi 重写必须挂诊断单**。本次 4 个退化案例的共性是"整篇推倒"。按 CLAUDE.md"工作流 C：修订已有传记"规范，Kimi 重写前必须有 Codex 的修订诊断单约束范围，不能无约束 Kimi 重写。

3. **强化"不得回滚已修硬伤"的 prompt 约束**。张居正案例中 5 条回滚说明 Kimi 没有"延续性意识"，重写时需要在 prompt 里明确列出"以下旧错已修，禁止回滚"。

4. **限制二手来源类型**。朱翊钧新增腾讯新闻、知乎、网易号说明 Kimi 的网搜不分层。建议 prompt 增加"禁用自媒体/问答社区类来源"的硬约束。

5. **引入退化拦截点**。本批次若在 commit 前对比新旧"一手来源数"、若下降超过 15% 则自动触发二次校验，可拦截本批次 3 个退化案例中的至少 2 个。

6. **王喜姐/李时珍模式可复用**。对剩余传记的修订，优先采用"小动作、定点改"路线，避免整篇重写。
