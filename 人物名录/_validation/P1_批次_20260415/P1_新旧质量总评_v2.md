## 6. 海瑞卷次数仍低于旧版（56→39）的 Codex 判定

Codex 明确判定此项为**"内容质量大升，但引文显式化退步"**：
- 一手古籍来源数 10→23（超旧版 2.3 倍），证据面显著拓宽
- 进入正文的史料从"少数正史+百科/二手"升级为"明史、实录、国榷、文集、行状、方志并用"
- 族属矛盾已从旧版"前文定论后文存疑"的分裂状态，修正为通篇统一的"存疑"表述
- 但卷次下降**并非完全因为去重或精度提升**，确有部分是 B 版少标、漏标

**Codex 最终判定仍为"显著提升"**，理由是来源层级与古籍密度的增益远超卷次显式化的损失。这表明卷次数并非唯一衡量指标，证据链广度更关键。

## 7. 最终结论：合并补救是否成功？可否 commit？

### 7.1 整体结论：**合并补救成功，质量净提升可以 commit**

支持证据：
1. **主观判断**：v1 的"2 提升 / 4 退化"改善到 v2 的"5 提升 / 1 退化"，有 4 人方向翻转
2. **客观数据**：一手来源均值从 v1→v2 回升 6.8%，卷次均值超旧版 9.3%
3. **问题修复率**：完全修复从 50% 提升到 67%
4. **合并路径机理验证**：对张居正/海瑞/郑贵妃而言，"旧版优质内容合并"是确实扭转退化的关键步骤

### 7.2 保留意见与建议

**必须在 commit 前处理的两项**（从 v1 延续或 v2 新增的硬问题）：
1. **王喜姐 v2 逆转**——一手来源从 v1 的 15 个降到 7 个，反而低于旧版（7 vs 旧版相同），且新增"同居二十余年"（旧版为 16 年）等事实退化。**建议 commit 前对王喜姐做一次定点回补**，把 v1 被合并掉的优质一手来源回填。
2. **海瑞一条鞭法争议**——v1→v2 从"部分修复"退到"未修复"，需要用户/下一轮 Codex 做定点改订。

**建议延后处理的三项**（非阻断性，可在下一批次迭代中清理）：
- 张居正家世/祖籍/日期回滚错
- 朱翊钧残余网媒来源
- 海瑞/王喜姐英文夹杂

### 7.3 可否 commit 的判定

**可以 commit，但建议在 commit message 中记录"王喜姐待回补"作为未闭环问题**。若用户对王喜姐的质量退化零容忍，建议先做一次定点修订再 commit；若接受"4 人显著提升、1 人轻微提升换 1 人退化"的净效果，现状可以进入版本库。
