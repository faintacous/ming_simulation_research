## 技术备忘

- 通过 stdin pipe（`cat prompt.txt | kimi --print --input-format text`）传递中文 prompt 可靠
- `--print` 模式下 kimi 会同时流式输出 UI 与最终内容，但 `--print` + Write 工具模式下直接写盘为最终稿（不含 UI 流）
