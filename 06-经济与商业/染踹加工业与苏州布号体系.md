# 染踹加工业与苏州布号体系

## 概述
明清时期，江南棉布产业形成了"织在松江、染踹在苏州、销往全国"的产销分工格局。苏州凭借全国商贸交通枢纽地位和高超的染踹加工技术，在清初逐渐取代松江成为全国最大的棉布加工与贩售中心。以阊门为中心的棉布字号体系，通过"放料制"（putting-out system）将染坊、踹坊纳入商业资本支配之下，形成了中国早期工业化最具代表性的生产组织形态之一。

## 一、棉布后整理加工的工艺流程

### 1.1 总体工序
棉布从坯布到成品需经过以下后整理加工环节：
- **浆布**：以米浆等浆料处理坯布，增加布面硬挺度
- **染色**：根据颜色需求送入不同染坊（蓝坊、红坊、杂色坊等）
- **漂洗**：染好的布料清水漂洗三次去浮色，晾晒
- **踹光**（碾布）：以元宝石碾压布面，使之平整光洁、色泽亮丽
- **看布**：质检环节，检查布面质量
- **行布**：包装销售环节

[来源：邱澎生《由放料到工厂》；《天工开物·乃服》]

### 1.2 染色工艺
- 明代苏州染坊已有高度分工，分为**蓝坊、红坊、红漂坊、杂色坊**等不同类型 [来源：明代文献]
- 苏州一地染匠有**数千人** [来源：综合文献]
- 蓝靛是与棉布结合最好的染料，蓝草种植和蓝靛制造成为明清两代重要产业 [来源：界面新闻"蓝染布与清代的兴衰"]
- 基本染色技法包括：直接染、媒染、还原染、防染、套色染 [来源：古代染色工艺文献]
- 蓝靛可染出粉蓝、天蓝、海蓝、毛蓝、深蓝等多种色阶 [来源：染色工艺文献]
- 康熙五十九年（1720年），苏州城中专业染坊多达**64家** [来源：碑刻史料]

### 1.3 踹光工艺
- **工具**：元宝石（又称碾石、踹石），形如大元宝，长约1.5米，高约0.7米
- **材质选择**：《天工开物》记载"取江北性冷质腻者"，好的碾石"每块能值十多两银子"，碾压时"石不发烧"，不因摩擦生热损坏织物 [来源：宋应星《天工开物·乃服》]
- **操作方法**：踹匠站在元宝石上，以双腿协调控制旋转角度与速度，借自身体重和石头重力碾压布匹，使之"缕紧不松泛" [来源：搜狐文史]
- **技术要求**：需优秀平衡感、精确力道控制、极强体力，至少需十年经验 [来源：搜狐文史]
- **图像记载**：清乾隆三十年（1765年），直隶总督方观承绘制棉花种植纺织全过程十六幅工笔画，第十六幅"练染图"中有踹布工劳作场面 [来源：方观承《棉花图》]

### 1.4 踹坊从染坊中分离
- 康熙末年，随着棉布业兴盛和市场扩大（尤其是每年**数十万匹**政府采购订单的刺激），踹布业从染坊中陆续**分离**出来，成为独立行业 [来源：经济史文献]
- 这一专业化分工是苏州棉布加工业成熟的重要标志

## 二、苏州布号的组织形式

### 2.1 字号（布号）
- **定义**："本客"非指店铺，而指棉布字号。字号商人从事棉布收购、委托踹染加工和大宗批销，是较大规模的商业资本 [来源：经济史论坛]
- **集中地点**：字号集中开设于苏州城西北郊**阊门**一带，主要分布在**上塘街、下塘街** [来源：碑刻、方志]
- **经营描述**："苏布名称四方，习是业者在阊门外上下塘，谓之字号，漂布、染布、看布、行布各有其人，一字号常数十家赖以举火" [来源：清代方志]
- **资本门槛**：开设字号所需资本巨大，"惟富人乃能办此" [来源：邱澎生论文]
- **数量变化**：
  - 康熙初年：约40-70家 [来源：邱澎生]
  - 康熙三十九年（1700年）：69家棉布商人（罢工碑文所记） [来源：碑刻史料]
  - 康熙五十九年（1720年）：染坊64家 [来源：碑刻史料]

### 2.2 包头（作头）
- 包头是踹坊的名义老板，备有生产设备（踹石、场地等）
- 但学者基本上**不**将踹坊视为"独立的手工业工场"，理由是踹匠工资按匹计价**由布号发给**，包头更像是字号商人的"工资领放人"而非真正的老板 [来源：邱澎生]
- 碑文记载："苏郡出产布货，所用踹匠，盈万成千，俱责**包头钤束**。工价有例，食用有条，原自相安"（康熙四十年碑文） [来源：碑刻史料]
- 包头负责：盘查踹匠来历、建立循环簿、维持坊内秩序

### 2.3 踹匠
- 基本上是**已离开农村的外来人口**，进行全年劳动 [来源：邱澎生]
- 以精壮青年男性为主
- 苏州平均一座踹坊内有**20-30人**集体劳动 [来源：邱澎生]
- 在同一场所、同一资本支配下按件计酬

### 2.4 三方关系结构
```
字号（商业资本）
  ├── 预发布匹 → 染坊（染色加工）→ 收回
  ├── 预发布匹 → 踹坊（踹光加工）→ 收回
  │        ├── 包头（管理、领放工资）
  │        └── 踹匠（实际劳动者，按匹计酬）
  └── 看布、行布（质检与销售）
```

## 三、布号从松江向苏州的转移

### 3.1 时间线
- **明代**：布号集中于松江府的**枫泾、朱泾**等棉布产地镇，"数百家布号皆在松江、枫泾、朱泾乐业" [来源：综合文献]
- **清初**：布号开始向苏州阊门一带集中，形成"**布店在松，发卖在苏**"的产销格局 [来源：邱澎生]
- **康熙-乾隆年间**：松江地区愈来愈以苏州城为销售中心，苏州完全确立全国棉布加工与贩售中心地位 [来源：邱澎生]
- **咸丰十年（1860年）**：太平天国攻打苏州，阊门至枫桥商业区被烧毁，棉布贸易中心开始向上海转移 [来源：阊门维基百科]

### 3.2 迁移原因
1. **交通枢纽**：苏州处于全国商贸交通网络中心，水运发达，辐射全国
2. **加工技术**：苏州在"青蓝布"染踹加工方面技术领先
3. **市场集聚**：苏州已有丝织业等商业基础设施，商人网络成熟
4. **地理邻近**：苏州地近松江府棉布产区，原料获取便利

## 四、踹匠的劳动条件与抗争

### 4.1 规模数据
- **雍正年间**官方调查：
  - 踹坊**450余处** [来源：邱澎生引雍正年间档案]
  - 包头**340余人** [来源：邱澎生]
  - 踹石**一万九百余块** [来源：邱澎生]
  - 踹匠、染匠合计**至少两万人以上**（另一说踹匠两万多人） [来源：邱澎生；网易文史]
- 这批踹匠是苏州成为全国第一大工商业城市过程中最典型的**低技术移民群体**

### 4.2 工价与收入
- **康熙九年（1670年）**工价标准：每匹纹银**一分一厘** [来源：碑刻]
- **康熙三十九年（1700年）大罢工后**：每匹提升至**一分一厘三毫** [来源：碑刻]
- 强壮熟练踹匠的收入估算：
  - 日踹布量：约**10匹**
  - 日收入：一钱一分
  - 月收入：约**3两3钱**（按30天计）
  - 年收入：约**40两银子**
- 对比：《红楼梦》中刘姥姥称"庄家人过一年"约需二十多两，踹匠名义收入看似较高 [来源：网易文史]
- 但踹匠须偿还包头预支的食用开销，并按月缴纳住房和工具使用费（每月**三分六厘**），实际到手收入大打折扣 [来源：碑刻史料]

### 4.3 劳动管控
- 踹匠白天做工，夜间**关闭在坊**，不得外出
- 五人**连环互保**，官府与军队协助包头监管
- 实施"一家有事，九家连坐"的保甲管理 [来源：碑刻]
- 禁止夜间行走、酗酒赌博
- 进入行业需有保人担保，出事保人全家连坐（雍正年间李卫强化） [来源：网易文史]

### 4.4 罢工事件年表

| 时间 | 领导者 | 规模与经过 | 结果 |
|------|--------|------------|------|
| 康熙九年（1670年） | 窦桂甫 | 年荒米贵，包头短扣工银，齐行叫歇 | 窦桂甫杖责驱逐；立碑核定工价"每匹一分一厘"，规定"店家无容短少，工匠不许多勒" |
| 康熙三十二年（1693年） | 罗贵等16人 | 要求增加工资，撕毁官府告示、驱逐布商、捣毁踹房、聚众殴抢 | 罗贵等16人逃匿，其他踹匠枷责 |
| 康熙三十九年（1700年） | 刘如珍等 | **上千人响应**，"变乱之势比诸昔年尤甚"，持续近一年，69家棉布商遭重创；踹匠建立"罢工准备金"制度，"积少成多，已盈千万" | 工价提升至"一分一厘三毫"；苏州府以保甲约束踹匠 |

[来源：网易文史系列文章；碑刻史料]

### 4.5 碑刻记录
自康熙九年至乾隆六十年，苏州府、县官府前后**八次**应字号请求而刻立禁止踹匠"把持"碑，竖立在阊门外**广济桥堍** [来源：碑刻史料新编]

## 五、"放料制"（Putting-Out System）的运作机制

### 5.1 基本定义
放料制是一种由商人（字号）向分散的加工作坊预发原料（布匹），加工完成后按件计酬收回成品的生产组织方式。在中国语境中也称为"包买制"。

### 5.2 苏州棉布放料制的具体运作
1. 字号商人从松江等地收购坯布（白布）
2. 将坯布分发（"放料"）给染坊进行染色
3. 染好的布匹再分发给踹坊进行踹光
4. 收回成品后由字号统一包装、品检、销售
5. 踹匠按匹计酬，工资由字号通过包头发放

### 5.3 制度特征
- **商业资本支配生产**：整个加工过程在商业资本支配下进行，字号控制原料和成品
- **空间分散**：加工作坊在空间上相对分散，非集中厂房
- **按件计酬**：作坊工匠完成产品后按件领取工资
- **长期契约**：字号通过"一体放料与预先确立品质规格"与加工坊建立长期关系，"减少商人每次与对手价格谈判时尔虞我诈的损失" [来源：邱澎生]
- **两种经营形式**：字号之下的染坊、踹坊，或由字号直接投资经营，或由字号将棉布交外部坊加工 [来源：经济史文献]

### 5.4 放料制 vs 工厂制
邱澎生核心论点：**放料制并未演变为工厂制**，原因不在于技术或资源，而在于**法律制度调整的局限**：

- **交易成本方面**：长期契约降低了信息和谈判成本，但增加了执行成本（工匠偷卖原料、踹坊违约等需通过诉讼解决）
- **法律责任困境**：踹匠罢工时，政府既要维护"禁止把持垄断"的市场原则，又要保护"失其生计"的弱势群体，难以清晰划分法律责任
- **理性选择**：对商人而言，转向工厂制将承担更大政治和法律风险，放料制"对投入生产事业的商人财产较有保障"
- **结论**："工厂制不必然优于放料制"——不应将两者视为"落后/进步"的线性演化关系 [来源：邱澎生《由放料到工厂？》]

## 六、"资本主义萌芽"争论的学术背景

### 6.1 争论起源
- **1935年**：邓拓提出"鸦片战争前中国已经产生新的社会经济系统的苗芽"
- **1952年**：《毛泽东选集》第二卷正式纳入"资本主义萌芽"相关论述
- **1954年**：关于《红楼梦》的大讨论引发广泛学术对话；傅衣凌提出江浙纺织业已向工场手工业发展
- **1957年、1960年**：两部讨论集出版
- **1985年**：许涤新、吴承明主编《中国资本主义发展史》第一卷出版，成为该领域重要总结，定义萌芽产生时间为"明代后期"
- **1990年代后**：讨论逐渐沉寂

### 6.2 棉纺织业的特殊地位
- **比较严谨的看法**：明中叶在苏州、杭州丝织业和佛山冶铁业中已见资本主义性质的手工作坊
- **棉纺织业的悖论**：当时中国产值最大的棉纺织手工业仍属**农家副业**，学者"可肯定没有资本主义萌芽"
- **但在棉布加工环节**：染坊和踹坊被认为存在资本主义萌芽形式（雇佣劳动、按件计酬、集体劳动）
- **关键区别**：纺织环节（农户分散进行）vs 染踹加工环节（集中化、雇佣化）

### 6.3 主要学术立场

| 学者 | 主要观点 |
|------|----------|
| 吴承明 | 萌芽是"资本主义生产关系的发生过程"，非"一厂一店"，是新的先进生产关系 |
| 许涤新 | 与吴承明合编《中国资本主义发展史》，定萌芽于明代后期 |
| 傅衣凌 | 江浙纺织业已向工场手工业发展，出现接近资本主义家内作业的最初生产形态 |
| 李伯重 | 直言"资本主义萌芽"是"伪命题"，本质是学者的"争气"情结 |
| 黄宗智（Philip Huang） | 质疑仅基于欧洲经验的理论框架 |
| 黄仁宇 | "把小孩子叫做预备成人"无实际意义 |
| 邱澎生 | 放料制 vs 工厂制并非线性演化，而是不同制度环境下的理性选择 |

### 6.4 国际学术界
- 西方经济史学家普遍倾向否认这些发展预示着资本主义转变
- 但罗兹·墨菲评价"中国马克思主义历史学家的见解是非常合理的"
- 日本学者（20世纪50年代）曾得出与中国学者相似结论
