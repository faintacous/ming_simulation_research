# 冶铁业

## 概述

明代冶铁业是当时世界上规模最大的冶铁产业，铁产量自明初到嘉靖时代不仅是中国历史的巅峰，也居同期世界第一位。嘉靖年间全国铁产量估计达9,000万斤（约4.5万吨）。从洪武年间的官营13所铁冶到洪武二十八年（1395）仅30年即开放民营，此后民营铁冶主导产业发展。广东佛山崛起为全国最大的铁器生产和贸易中心，明人霍与瑕言"两广铁货所都，七省需焉"。遵化铁厂则是北方最重要的官营铁冶。

## 一、佛山铁冶规模

### 1.1 发展历程

| 时期 | 状况 |
|------|------|
| 明初 | 佛山已有约**5,000人**从事铸铁业 |
| 成化、弘治年间 | 居民"大率以铁冶为业"，时有"三千余家" |
| 天启年间 | 冶铁业分为至少**7个**专业行当 |
| 明末 | 单个炒铁炉场规模已达**200余工人** |
| 清代乾隆 | 从事铸铁业者约**2万人**，炒铁炉**40余所** |

[来源：中国经济史论坛《广东佛山冶铁和铁器铸造业中的资本主义萌芽》](http://economy.guoxue.com/?p=4569)

### 1.2 佛山铁冶的特殊地位

- 明清两代官府规定"两广铁版**统运于佛山一埠铸造**"
- 佛山聚集了大量优秀铸铁工匠
- 佛山炉户获得了**长期铸铁权**
- 形成了从生铁冶炼到铁器铸造的完整产业链

### 1.3 炒铁炉场的经营规模

明末佛山一个典型炒铁炉场的数据：
- 工人：**200余人**
- 盈亏标准：每天出铁**20版**可赚钱；不到**10版**就要赔钱
- 这一规模已具有相当的工业化特征

## 二、遵化铁冶

### 2.1 基本信息

## 三、官营与民营的比例变化

### 3.1 制度演变时间线

| 时间 | 政策 | 说明 |
|------|------|------|
| **洪武七年（1374）** | 设官铁冶**13所** | 南方6所、北方7所，禁民间开采 |
| **洪武二十八年（1395）** | **"诏罢各处铁冶，令民得自采炼"** | 政策大转向，开放民营 |
| 永乐-宣德 | 官民并存 | 民营快速发展 |
| 正统以后 | 民营为主 | 官营铁冶逐渐萎缩 |
| 嘉靖-万历 | 民营占**绝对主导** | 佛山等地民营冶铁业繁荣 |

> **关键事实**：铁的矿禁不过**30年**（1374-1395），即开放民营。《明实录》载洪武二十八年九月："各处续开炉冶，今已三年，而内库见贮铁凡**三千七百四十三万余斤**。上以库内储铁已多，诏罢各处铁冶。"

[来源：知乎《浅谈明代的冶铁业》](https://zhuanlan.zhihu.com/p/56438349)

### 3.2 官营铁冶的特点

| 方面 | 说明 |
|------|------|
| 管理 | 由工部管理，派官员和太监监督 |
| 劳动力 | 使用征调的匠户 |
| 产品 | 主要用于军器、农具等国家需要 |
| 问题 | 效率低下、管理腐败、工匠逃亡严重 |

### 3.3 民营铁冶的优势

| 方面 | 说明 |
|------|------|
| 经营 | 灵活高效 |
| 产品 | 面向市场，品种丰富 |
| 劳动力 | 雇佣关系，积极性高 |
| 技术 | 利润驱动下不断改进 |

## 六、"七省需焉"的市场网络

### 6.1 原文出处

明万历佛山人**霍与瑕**记述：

> "富国强兵之术以盐铁为首务。两广铁货所都，**七省需焉**。每岁浙、直、湖、湘客人腰缠过梅岭者数十万，皆置铁货而北。"

### 6.2 市场辐射范围

| 方向 | 目的地 | 说明 |
|------|--------|------|
| **北上** | 浙江、南直隶、湖广等 | 经梅岭（大庾岭）北运 |
| **国内沿海** | 福建、浙江等沿海各省 | 经海路 |
| **东南亚** | 越南、东南亚各国 | 海上贸易，广锅出口 |
| **日本** | 长崎等港口 | 经海路 |
| **南亚** | 印度、尼泊尔、不丹 | 经海路/陆路 |
| **朝鲜** | — | 经海路 |

### 6.3 贸易规模

- 每年翻越梅岭购铁的外省客商达**"数十万"**
- 郑芝龙集团每年运货到台湾的船只200-300艘，主要商品中包括**铁锅**
- 佛山铁器（尤其是广锅）通过澳门出口海外

## 七、冶铁技术

### 7.1 主要冶铁方法

| 方法 | 产品 | 说明 |
|------|------|------|
| **高炉炼铁** | 生铁 | 使用木炭或焦炭（华北部分地区）作燃料 |
| **炒铁** | 熟铁 | 将生铁在炒铁炉中反复翻搅去碳 |
| **灌钢法** | 钢 | 将生铁浇灌到熟铁上，使碳均匀分布 |

### 7.2 佛山的炒铁-铸造分工

佛山形成了完整的产业链：

1. **炒铁炉户**：将铁矿石炼成生铁，再炒炼为熟铁
2. **铸造炉户**：将生铁或熟铁铸造为各类铁器成品
3. 两个环节分属不同行当，明确的产业分工

### 7.3 主要冶铁聚集地的劳动力规模

| 地点 | 人数 | 时期 |
|------|------|------|
| 佛山 | 冶铸工人**二三万人** | 成化弘治至清代 |
| 福建漳州龙溪 | 单炉多至**五七百人** | — |
| 山西夏县温峪山 | 采矿冶铁聚集**六七千人** | 万历年间 |
| 北直隶蓟州之西 | 采矿冶铁聚集**近万余人** | 嘉靖年间 |
