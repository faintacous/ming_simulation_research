# 其他商帮

## 概述

除徽商、晋商、闽商、粤商外，明代还有多个区域性商帮活跃于全国各地。本文涵盖陕商（茶马贸易/边镇供应）、江右商帮（瓷器/纸张）、龙游商帮（纸/木材）、洞庭商帮（丝绸/大米）和宁波商帮（海运/钱庄）。各商帮各据一方优势资源，形成了差异化的经营模式。

## 关键事实

### 一、陕商（关中商帮/秦商）

#### 1.1 基本概况

- 以陕西关中地区商人为主体，尤其西安府、凤翔府、同州、华州等地 [来源：陕商研究]
- 活跃时间：明初即崛起，与晋商同为最早受益于开中法的商帮 [来源：盐法史研究]
- 与晋商活动区域高度重叠，常被合称"山陕商人" [来源：商帮研究]

#### 1.2 茶马贸易

**茶马互市制度**
- 明代在西北设茶马司（陕西茶马司、甘肃茶马司），管理以茶换马的官方贸易 [来源：《明史·食货志》]
- 茶马比价：洪武间官方定价约为上等马一匹最多换茶叶120斤，但实际比价波动很大 [来源：茶马古道研究]
- 明政府对茶叶实行专卖（"榷茶"），严禁私茶出境——但走私始终存在 [来源：茶法研究]
- 明政府实行"食盐开中""茶马交易""棉布征实""布马交易"等一系列特殊经济政策 [来源：MBA智库百科]

**陕商在茶马贸易中的角色**
- 陕西为茶马互市的核心区域：汉中地区（陕南）产茶，甘肃/青海为茶马交易地点 [来源：茶马贸易研究]
- 陕商从汉中、四川采购茶叶，运至兰州、河州（今临夏）等地进行茶马交易 [来源：茶马贸易研究]
- 茶叶运输路线：汉中→凤县→天水→兰州/河州 [来源：商路研究]
- 运输方式：骡马驮运为主（山路崎岖，不利于车运） [来源：交通史研究]
- 嘉靖至万历间，私茶贸易规模远超官方茶马互市 [来源：茶法研究]

**贸易规模数据**
- 陕商每年经营的布匹达2700万匹 [来源：陕商研究]
- 茶叶1125多万斤 [来源：陕商研究]
- 皮革150多万张 [来源：陕商研究]
- 羊毛2500多万斤 [来源：陕商研究]
- 药材1500万斤 [来源：陕商研究]
- 水烟700万斤 [来源：陕商研究]

**茶引与茶税**
- 官方发放茶引，每引允许贩运茶叶100斤 [来源：《明史·食货志》]
- 陕西年发茶引数量：据万历间数据约数千引 [来源：茶法研究]
- 实际贩运量远超引额，走私茶叶利润丰厚 [来源：茶法研究]

#### 1.3 边镇供应

- 陕商与晋商类似，承担西北边镇（延绥/榆林、宁夏、甘肃、固原）的物资供应 [来源：边镇经济研究]
- 供应品种：粮食、布匹、铁器、茶叶等 [来源：边镇研究]
- 明初亦通过开中法获取盐引（主要为解州盐和灵州盐） [来源：盐法史研究]

#### 1.4 活动区域

| 区域 | 经营内容 | 备注 |
|------|----------|------|
| 泾阳、三原 | 大本营 | 陇海线通车前一直是陕商总部 |
| 汉中 | 茶叶采购地 | 陕南产茶区 |
| 甘肃（兰州/河州） | 茶马互市地点 | 茶马司所在 |
| 四川/贵州 | 盐和茶叶 | "贩盐于川黔" |
| 西北边镇 | 军需物资供应 | 延绥/宁夏/固原 |
| 苏湖地区 | 棉布采购 | "鬻布于苏湖"，垄断江南标布运销达300年 |
| 陇青/蒙藏 | 茶叶运销 | "输茶于陇青" |
| 各地 | 同晋商合建山陕会馆 | 见晋商.md |

#### 1.5 特点

- 宋应星《天工开物·野议·盐政议》评价："商之有本者，大抵属秦、晋与徽郡三方之人" [来源：《天工开物》]
- 陕西商帮被认为是中国按地域亲缘关系最早出现的商帮 [来源：陕商研究]
- 以泾阳、三原为中心，以西北、川、黔、蒙、藏为势力范围 [来源：MBA智库百科]
- 与晋商高度重叠但有区分：陕商更偏重茶马贸易和四川方向，晋商更偏重蒙古方向 [来源：商帮比较研究]
- 资本规模总体不及晋商和徽商 [来源：商帮研究]
- "山陕会馆"遍布中原和长江流域，反映两帮的紧密合作 [来源：会馆史研究]

### 二、江右商帮（赣商）

#### 2.1 基本概况

- 以江西商人为主体，"江右"为江西的古称（长江之西/右岸） [来源：江西方志]
- 主要籍贯：抚州府、吉安府、南昌府、饶州府等 [来源：赣商研究]
- 明代谚语："无江右人不成市"——反映江右商人分布之广 [来源：民间谚语]
- 与徽商的显著区别：江右商以中小资本和零散经营为特色，不如徽商资本集中 [来源：商帮比较研究]

#### 2.2 经营范围

**瓷器**
- 景德镇（饶州府）为明代全国最大瓷器产地 [来源：景德镇瓷器史]
- 江右商人控制了景德镇瓷器的外销网络 [来源：瓷器贸易研究]
- 景德镇瓷器产量：万历间年产数十万件至上百万件（含民窑） [来源：景德镇窑业研究]
- 销售路线：经赣江→鄱阳湖→长江至全国，或经赣江→庾岭→广东出口海外 [来源：商路研究]

**纸张**
- 江西为明代重要造纸区：铅山（广信府）的连四纸、竹纸驰名 [来源：造纸史研究]
- 纸张用途：书写、印刷、包装 [来源：纸张贸易研究]
- 铅山纸的品质极高，为官府公文和书籍刻印的指定用纸之一 [来源：《铅山县志》]

**茶叶**
- 江西亦为产茶区（庐山云雾茶等） [来源：茶叶史]
- 部分江右商人经营茶叶贸易 [来源：赣商研究]

**药材**
- 江西樟树镇（后为"中国药都"）为南方最大的药材集散中心之一 [来源：药材贸易研究]
- 药材行业中江右商人占有重要地位 [来源：赣商研究]

**米谷**
- 江西为产粮大省，部分粮食经长江外销 [来源：粮食贸易研究]

#### 2.3 活动区域

- 分布极广，几乎遍及全国——"无江右人不成市"；在西南地区出现"非江右商贾居之不成地，无江右商贾买卖不成市"的景象 [来源：赣商研究]
- 明代江西会馆占北京会馆总数的34% [来源：刘上洋/澎湃新闻]
- 江西会馆占湖南地区会馆总数的46%，占云南地区会馆总数的38%，占贵州地区的34% [来源：澎湃新闻]
- 许多城镇还建起了"江西街""江西路" [来源：赣商研究]
- 在云南、贵州、两广、湖广等开发较晚的地区尤其活跃 [来源：赣商研究]
- 在云南的矿业开发中江右商人参与显著 [来源：云南矿业研究]
- 在湖广的城镇商业中江右商人是重要力量 [来源：湖广商业史]
- 各地设有江西会馆（万寿宫/许真君庙），全国各地万寿宫数量至少曾达1500座以上 [来源：刘上洋/澎湃新闻]

#### 2.4 组织特点

- 以中小资本的个体经营和小合伙为主 [来源：赣商研究]
- 宗族组织不如徽商严密，但同乡网络发达 [来源：商帮比较研究]
- 经营方式灵活：从行商（走贩）到坐商（店铺）均有 [来源：赣商研究]
- 擅长在新开发地区抢占先机（如西南边疆地区） [来源：赣商研究]

#### 2.5 江西会馆（万寿宫）

| 城市 | 备注 |
|------|------|
| 北京 | 江西会馆 |
| 汉口 | 万寿宫 |
| 昆明 | 万寿宫/江西会馆 |
| 贵阳 | 万寿宫 |
| 广州 | 江西会馆 |
| 成都 | 万寿宫 |
| 南京 | 江西会馆 |

- 万寿宫以祭祀许逊（许真君，晋代江西人，道教"净明忠孝道"祖师）为核心 [来源：会馆史研究]

### 三、龙游商帮

#### 3.1 基本概况

- 以浙江衢州府龙游县商人为主体 [来源：龙游商帮研究]
- 明代谚语将龙游商帮与徽商并称："遍地龙游" [来源：民间谚语]
- 虽来自一个小县，但商业影响力不可小觑 [来源：龙游商帮研究]

#### 3.2 经营范围

**纸张与书业**
- 龙游及周边衢州山区盛产竹纸（毛边纸、连史纸等） [来源：造纸史研究]
- 龙游商人控制了浙西竹纸的生产和销售网络 [来源：纸张贸易研究]
- 纸张通过水运（衢江→钱塘江→长江）销往全国 [来源：商路研究]
- 纸商远销江苏一带，书商销售范围包括浙江、江苏、河北及山东、京师 [来源：龙游县政府]
- 明代龙游书商、藏书家童佩是代表人物之一 [来源：龙游商帮研究]

**木材**
- 衢州山区有丰富林木资源 [来源：衢州方志]
- 木材扎筏沿衢江而下 [来源：木材贸易研究]

**珠宝与古玩**
- 龙游商人中有专门从事珠宝、古玩鉴赏和买卖者 [来源：龙游商帮研究]
- 珠宝商最大市场在京师 [来源：龙游县政府]
- 在北京和南京的古玩市场中有一定活跃度 [来源：龙游商帮研究]

**矿业（商业资本向产业资本转化的典型）**
- 龙游商人敏锐意识到要获更大利润须转向手工业生产和工矿产业 [来源：中国数字科技馆]
- 果断投入纸业、矿业的商品生产，使商业资本转化为产业资本 [来源：中国数字科技馆]
- 在福建、江西、云南等地的银矿、铜矿开发中有参与 [来源：龙游商帮研究]

**海外贸易**
- 海外贸易商涉及日本、东南亚各国 [来源：龙游县政府]
- 边贸商远销缅甸、印度，并在云南姚安地区垦荒经商 [来源：龙游县政府]

#### 3.3 活动区域

- 纸商远销江苏，书商销售至浙江、江苏、河北、山东、京师 [来源：龙游县政府]
- 粮食商毗邻各省，山货商销售到全国及东南亚 [来源：龙游县政府]
- 在云南、贵州等西南边疆也有龙游商人足迹 [来源：龙游商帮研究]
- 在北京有一定活动 [来源：龙游商帮研究]
- 万历年间（1573—1602），龙游商帮与徽商、晋商以及江右商人在商场中角逐，称雄一时，故有"遍地龙游"之谚 [来源：维基百科/百度百科]

#### 3.4 特点

- 经营理念：投资上的"敢为天下先"精神和"海纳百川"的包容性 [来源：中国数字科技馆]
- 规模小但精干，善于经营边缘市场和冷门行业 [来源：龙游商帮研究]
- 有"钻天洞庭遍地龙游"之说——龙游商人善于深入偏远地区寻找商机 [来源：民间谚语]

### 四、洞庭商帮

#### 4.1 基本概况

- 以苏州府太湖中洞庭东山和洞庭西山（今苏州市吴中区东山镇、金庭镇）的商人为主体 [来源：洞庭商帮研究]
- "钻天洞庭"——形容洞庭商人善于钻营、精明过人 [来源：民间谚语]
- 洞庭两山面积不大，人口有限，但商人辈出 [来源：苏州方志]

#### 4.2 经营范围

**丝绸**
- 苏州为明代全国最大的丝绸生产和贸易中心 [来源：苏州方志]
- 洞庭商人在丝绸贸易中占有重要地位 [来源：丝绸贸易研究]
- 从湖州、嘉兴采购生丝→苏州加工→销往全国和海外 [来源：丝绸贸易研究]
- 部分洞庭商人投资丝织作坊（机户） [来源：苏州织造研究]

**大米/粮食**
- 洞庭商人在长江流域的粮食贸易中非常活跃 [来源：粮食贸易研究]
- 从湖广采购大米运销江南（"湖广熟天下足"） [来源：粮食贸易研究]
- 粮食贸易路线：湖广→长江→苏州/南京/杭州 [来源：商路研究]

**棉布**
- 参与松江棉布的贸易 [来源：棉布贸易研究]
- 贩运的货物品种还有棉花、棉纱、蓝靛、药材、南北特产以至书籍，"无物不营、无所不包" [来源：MBA智库百科]

**金融业**
- 洞庭商人擅长金融操作 [来源：洞庭商帮研究]
- 经营钱庄（银钱兑换）、放贷 [来源：金融史研究]
- 为后来苏州钱庄业的发展奠定基础 [来源：金融史研究]

#### 4.3 活动区域

**东山商人与西山商人的分工**
- **东山商人家族**主要活跃于以山东临清为中心的华北地区，以经营布匹贸易为主 [来源：MBA智库百科/洞庭商帮研究]
- **西山商人家族**则主要活跃在以长沙、汉口为中心的长江中游地区，以经营米粮、绸布贸易为主 [来源：MBA智库百科/洞庭商帮研究]

| 区域 | 经营内容 |
|------|----------|
| 苏州 | 大本营，丝绸/棉布贸易中心 |
| 湖广（汉口/长沙） | 粮食采购（西山商人主导） |
| 山东（临清） | 布匹贸易（东山商人主导） |
| 南京 | 丝绸/粮食销售 |
| 北京 | 丝绸和布匹贸易 |
| 长江沿线 | 粮食转运 |

**主要家族**
- 东山：王氏、叶氏、席氏、严氏、翁氏等 [来源：洞庭商帮研究]
  - 翁氏的"翁百万"翁笾，其父辈即已经商到临清 [来源：洞庭商帮研究]
  - 席氏在席左源、席右源时已"北走齐燕，南贩闽广" [来源：洞庭商帮研究]
  - 王鏊高中探花，历仕户部尚书、文渊阁大学士 [来源：洞庭商帮研究]
- 西山：秦氏、徐氏、沈氏、叶氏等 [来源：洞庭商帮研究]

#### 4.4 组织特点

- 资本规模中等，但经营手段精明 [来源：洞庭商帮研究]
- 注重信息收集和市场判断 [来源：洞庭商帮研究]
- 家族内部分工协作：常一家之中兄弟分驻不同城市 [来源：洞庭商帮研究]
- 冯梦龙描写："两山之人，善于货殖，八方四路，去为商为贾，所以江湖上有个口号，叫做'钻天洞庭'" [来源：冯梦龙小说/MBA智库百科]
- 与徽商合称"钻天洞庭遍地徽" [来源：洞庭商帮研究]

### 五、宁波商帮

#### 5.1 基本概况

- 以浙江宁波府（鄞县、慈溪、镇海、象山等）商人为主体 [来源：宁波商帮研究]
- 宁波自古为海运要港，南宋时设市舶司 [来源：宁波方志]
- 明初设宁波市舶司（专对日本），嘉靖二年（1523）争贡之役后停废 [来源：《明史·食货志》]

#### 5.2 经营范围

**海运**
- 宁波商人有深厚的航海传统 [来源：宁波航海史]
- 明初至嘉靖间，宁波为合法对日朝贡贸易（"勘合贸易"）的指定港口 [来源：中日贸易研究]
- 嘉靖后虽市舶司停废，宁波商人仍通过走私和间接渠道参与海上贸易 [来源：海禁研究]
- 沿海运输（北至天津/辽东，南至福建/广东）中宁波船商占有重要份额 [来源：沿海航运研究]

**钱庄**
- 宁波商人在金融领域非常活跃 [来源：宁波商帮研究]
- 在各地开设钱庄，从事银钱兑换和放贷 [来源：金融史研究]
- 宁波钱庄的运作较为规范，为清代宁波帮钱庄的前身 [来源：金融史研究]

**南北货贸易**
- 宁波商人经营"南北货"——将南方产品（丝绸、茶叶、糖）运至北方，北方产品（药材、皮毛）运至南方 [来源：宁波商帮研究]
- 利用海运航线实现长距离贸易 [来源：沿海航运研究]

**鱼盐**
- 宁波沿海有丰富的渔业和盐业资源 [来源：宁波方志]
- 鱼货和盐是宁波商人的传统经营品种 [来源：宁波商帮研究]

**药材**
- 宁波慈溪等地有药材贸易传统 [来源：药材贸易研究]
- 明崇祯年间，宁波药材商在北京建立鄞县会馆，标志着"宁波商帮"的初步形成 [来源：宁波商帮研究]
- 主要活动地域在北京等北方地带，主要经营行业是药材业和成衣行 [来源：宁波商帮研究]

**双屿港走私贸易**
- 16世纪上半叶，舟山双屿港发展为全国最具规模的海外贸易走私港口 [来源：维基百科]
- 双屿港在1520—1540年代约20多年间迅速发展为沟通亚洲、欧洲的中国最大私人海外贸易基地 [来源：维基百科]
- 1538年，福建海商金子老"据宁波之双屿"，招来李光头、许栋、王直等人共同经营，并引来日本私商和葡萄牙商人 [来源：知乎]

#### 5.3 活动区域

| 区域 | 经营内容 |
|------|----------|
| 宁波 | 大本营，港口和钱庄中心 |
| 杭州 | 丝绸/茶叶贸易 |
| 南京 | 商业贸易 |
| 北京/天津 | 海运终点，南北货贸易 |
| 辽东 | 海运贸易（豆/粮） |
| 日本（长崎） | 嘉靖前为合法贸易港，嘉靖后走私 |
| 福建沿海 | 沿海航运中转 |

#### 5.4 特点

- 海洋性格突出：善于航海，对海上风险有成熟的应对经验 [来源：宁波商帮研究]
- 金融意识强：较早发展钱庄业务 [来源：金融史研究]
- 与日本的历史联系深厚（宁波—博多/长崎航线为东亚最古老的海上贸易线路之一） [来源：中日贸易研究]
- 嘉靖间因海禁加严和市舶司停废受到重大打击，部分宁波商人转入走私或转行 [来源：海禁研究]

## 时间线

- 洪武三年（1370）：开中法创设，陕商与晋商同时受益 [来源：《明史·食货志》]
- 洪武间：茶马互市制度确立，陕商介入茶马贸易 [来源：茶马司档案]
- 嘉靖二年（1523）：争贡之役，宁波市舶司停废，宁波商帮海外贸易受阻 [来源：《明实录》]
- 嘉靖至万历间：各商帮活动达到高峰 [来源：商业史研究]
- 崇祯间：战乱影响各商帮经营 [来源：明末史料]
