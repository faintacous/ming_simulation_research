# 典当与民间信用

## 概述

明代民间金融体系包括当铺、钱庄、印局等多层次机构，以及各种非正式的借贷关系。随着商品经济发展和白银货币化进程加速，这些金融机构在嘉靖至崇祯年间（1521-1644）日益活跃，形成了覆盖从官僚士绅到城市贫民各阶层的多层次信用网络。《大明律》规定月利上限三分（3%），累计"不过一本一利"，但实际中高利贷普遍存在。万历五年（1577）钱庄获得法定地位，会票在明末萌芽。

---

## 一、当铺的运作方式

### 1.1 基本制度

当铺是明代最主要的民间金融机构，核心业务是以物质押放款。

| 要素 | 具体规定 |
|------|---------|
| **当期** | 通常为**18-24个月**（一年半到两年），过期不赎即为"绝当" |
| **折价率** | 当物估价一般**不超过原价的一半**（五折），贵重物品可能更低 |
| **赎回** | 当期内可随时赎回，需归还本金并支付利息 |
| **绝当处理** | 过期未赎的当物归当铺所有，可自行出售获利 |
| **当票** | 当铺开具当票作为凭证，赎回时须持票 |

### 1.2 利率

当铺利率因地区、铺本大小和当物价值而异，实行差别定价：

| 条件 | 月利率 | 年化利率 | 说明 |
|------|--------|---------|------|
| 福建当铺（铺本少） | **3%-4%/月** | 36%-48% | 铺本小、风险高 |
| 徽州当铺（铺本大） | **1%-3%/月** | 12%-36% | 资本雄厚 |
| 湖州府·十两以上 | **1.5%/月** | 18% | 大额当物 |
| 湖州府·一两以上 | **2%/月** | 24% | 中等当物 |
| 湖州府·一两以下 | **3%/月** | 36% | 小额当物 |

> **规律**：当物价值越高、铺本越大，利率越低；小额典当利率最高——这与现代金融中"风险溢价"的逻辑一致。

[来源：北京大学《明清山西商人多层次金融体系的创新及其局限性》](https://fem.pku.edu.cn/docs/20190528102708913925.pdf)

### 1.3 当铺的经营者

| 群体 | 特征 |
|------|------|
| **徽商** | 最大的当商群体，资本雄厚，遍布各省，"无徽不成典" |
| **晋商** | 山西商人大量经营典当业 |
| **官僚士绅** | 利用权势和资本开设当铺，或委托他人经营 |
| **寺庙** | 佛教寺院自南北朝即有"寺库"（长生库/质库），明代仍有延续 |

### 1.4 当铺的数量

- 明代后期当铺遍布城乡
- 徽商经营的当铺"遍天下"，从京师到边远府县都有分布
- 仅北京一城的当铺就有数十家以上

## 二、《明律》对利率的规定

### 2.1 法定上限

《大明律》明确规定：

> "凡私放钱债及典当财物，每月取利，并不得过三分，年月虽多，不过一本一利，违者笞四十。"

核心规则：

| 规则 | 内容 | 意义 |
|------|------|------|
| **月利上限** | 每月不超过**三分**（3%，年利率36%） | 限制暴利 |
| **累计利息上限** | "不过一本一利"——累计利息不超过本金100% | 防止利滚利 |
| **违规处罚** | 违者**笞四十** | 刑事处罚 |

### 2.2 "不过一本一利"的实际含义

- 借1两银，无论借期多长，利息最多收1两
- 例：月利3分借1两，约33个月后利息累计达1两，此后不再收息
- 此规则旨在防止长期借贷中利息无限膨胀

### 2.3 实际执行情况

法定利率上限在实际中经常被突破：

| 现象 | 说明 |
|------|------|
| 民间高利贷 | 月利五分至一角（5%-10%/月），甚至更高 |
| "驴打滚" | 利滚利做法虽违法但普遍存在 |
| 灾年利率 | 偏远地区和灾荒时期利率飙升 |
| 借粮利率 | 灾年借粮常"借一还二"（100%）甚至"借一还三" |

[来源：中国新闻网《古代借贷那些事儿》](https://www.chinanews.com.cn/m/cul/2018/08-17/8602941.shtml)

## 三、官僚和寺庙参与典当

### 3.1 官僚参与

- 不少官僚利用职权便利从事放贷活动
- 高级官员或退休官僚以家族名义开设当铺，或向当铺投资
- 官府有时也直接参与放贷：军营放贷、漕运衙门放贷等
- 官员放贷往往利用权势压低风险，实际利率可能高于市场

### 3.2 寺庙参与

- 佛教寺院自南北朝以来即有金融传统
- "寺库"（又称"长生库""质库"）管理寺院财产
- 寺院金融活动包括：典当、存钱保管、汇兑
- 明代寺庙仍参与典当和放贷：既有慈善性低息借贷，也有逐利性高息放贷
- 寺院借贷与民间借贷、官方借贷并行，成为三大借贷方式之一

## 四、钱庄的起源与法定化

### 4.1 起源背景

| 时间 | 事件 |
|------|------|
| 正统年间（1436-1449） | 大明宝钞严重贬值，朝廷放松用银禁令 |
| 正统之后 | 银钱公开流通，但私钱庞杂、铜钱轻重不一 |
| 正统-隆庆 | 市场产生强烈的货币兑换需求，专营兑换的"钱铺""钱肆"出现 |

### 4.2 万历五年法定化

- **万历五年（1577）**，两广总督**庞尚鹏**奏准在各地设立钱铺
- 这是钱铺/钱庄的**法定之始**
- 此后钱铺的设立有了官方许可，地位获得正式承认

[来源：维基百科《钱庄》](https://zh.wikipedia.org/zh-hans/%E9%8C%A2%E8%8E%8A)；[MBA智库百科《钱庄》](https://wiki.mbalib.com/wiki/%E9%92%B1%E5%BA%84)

### 4.3 钱庄的功能

| 功能 | 说明 |
|------|------|
| **货币兑换** | 核心业务——银两与铜钱之间的兑换，不同成色银两的换算 |
| **银两鉴定与熔铸** | 替客户鉴定银两成色、熔铸白银为元宝（"倾银"业务） |
| **存款** | 接受客户存放银两 |
| **放款** | 向商户和个人发放短期贷款 |
| **汇兑** | 后期逐渐发展出异地汇兑功能 |

## 五、会票雏形

### 5.1 历史渊源

- 唐代"飞钱"——商人在京师存钱入各地进奏院，换取凭证后到各地取钱
- 顾炎武《日知录》记载："唐宪宗之飞钱，即如今之会票也。"

### 5.2 明代的会票

| 方面 | 说明 |
|------|------|
| **出现时间** | 明末清初已经出现 |
| **用途** | 长途贸易中的异地汇兑 |
| **运作方式** | 商人在某地钱庄/银号存银，换取会票，异地凭票取银 |
| **优势** | 避免携带大量白银的风险和运输成本 |
| **使用范围** | 尚处萌芽阶段，主要由晋商、徽商等大商帮内部使用 |

### 5.3 银票与票号

- 严格意义上的银票要到**清代中期**才出现
- 日升昌票号（1823年）是中国第一家票号
- 明代所谓"银票"更接近私人借据或存单，不具备市场广泛流通功能

[来源：百度百科《银票》](https://baike.baidu.com/item/%E9%93%B6%E7%A5%A8/960172)；[维基百科《票号》](https://zh.wikipedia.org/zh-hans/%E7%A5%A8%E5%8F%B7)

## 六、印局（小额信贷）

### 6.1 起源与定义

印局是明清时期山西商人创办的专业小额贷款机构，主要服务于**城市底层市民**（小商小贩、手工业者、贫民等）。

### 6.2 运作方式

| 特征 | 说明 |
|------|------|
| **目标客户** | 城市贫民、小商贩——资产少、缺少抵押物 |
| **贷款方式** | 以**无抵押担保贷款**为主，需有担保人 |
| **还款方式** | **频繁还贷**——每日或每数日还款一次（非按月/年） |
| **利率水平** | **高利率**——因客户风险高，利率远超当铺 |
| **风险控制** | 担保人制度 + 高频还贷降低违约风险 |

### 6.3 与现代小额贷款的相似性

印局的运作模式与现代 microfinance 有显著相似之处：
- 面向无抵押物的低收入人群
- 高频还款降低风险
- 以高利率覆盖高坏账率

[来源：北京大学《明清山西商人多层次金融体系的创新及其局限性》](https://fem.pku.edu.cn/docs/20190528102708913925.pdf)

## 七、民间借贷利率数据汇总

### 7.1 各类借贷利率比较

| 借贷类型 | 月利率 | 年化利率 | 说明 |
|---------|--------|---------|------|
| 法定上限 | 3分（3%） | 36% | 《大明律》规定 |
| 当铺（大额） | 1-2分 | 12-24% | 徽州大当铺 |
| 当铺（小额） | 3-4分 | 36-48% | 福建小当铺 |
| 钱庄放款 | 1.5-3分 | 18-36% | 视客户信用 |
| 民间私贷 | 3-5分 | 36-60% | 有抵押 |
| 高利贷 | 5分-1角 | 60-120%+ | 无抵押或紧急借贷 |
| 印局小额贷 | 高于3分 | 36%+ | 面向贫民 |

### 7.2 灾荒时期的利率

- 灾荒年份粮食短缺，借贷利率飙升
- 借粮常按"借一还二"（100%利息）甚至"借一还三"
- 灾年高利贷成为农民破产、土地兼并的重要推手

## 八、明代多层次金融体系总结

| 机构类型 | 主要客户 | 核心业务 | 利率水平 | 准入门槛 |
|---------|---------|---------|---------|---------|
| 当铺 | 各阶层 | 质押放款 | 月利1-4分 | 需有当物 |
| 钱庄/钱铺 | 商人 | 兑换、存放、放贷 | 月利1.5-3分 | 商业信用 |
| 印局 | 城市贫民 | 小额无抵押贷款 | 高于月利3分 | 需有担保人 |
| 寺庙 | 各阶层 | 质押放款、存保 | 不等 | 因寺院而异 |
| 民间私贷 | 农民、小商人 | 个人间借贷 | 月利3分-1角 | 人际关系 |
| 官府借贷 | 军户、漕运人员等 | 特定人群放贷 | 不等 | 特定身份 |
