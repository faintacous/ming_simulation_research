# 长途贸易与商路

## 概述

明代国内长途贸易依赖五大主要商路：大运河（南北纵贯）、赣江庾岭路（连接长江与珠江流域）、湘江路（连接长江与广西/贵州）、西北丝路（连接中原与西域/中亚）、闽粤海路（东南沿海海上贸易线路）。这五条商路构成了明代商品流通的骨架，支撑起跨区域的大宗商品贸易。嘉靖至崇祯间，白银货币化和商品经济发展使长途贸易量达到前所未有的规模。

## 关键事实

### 一、大运河（京杭大运河）

#### 1.1 路线

- 全长约1800公里（明代运河线路，含各段迂回） [来源：运河史研究]
- 南起杭州，北至北京（通州），纵贯南北
- 主要河段：
  - **杭州至镇江段**（江南运河）：经嘉兴、苏州、常州至镇江，全程约330公里 [来源：运河志]
  - **镇江至扬州段**：渡长江（瓜洲渡/京口渡） [来源：运河志]
  - **扬州至淮安段**（里运河）：约190公里 [来源：运河志]
  - **淮安至徐州段**（中运河）：经宿迁至徐州，约300公里 [来源：运河志]
  - **徐州至济宁段**（鲁运河南段）：经微山湖区 [来源：运河志]
  - **济宁至临清段**（鲁运河北段/会通河）：约200公里，为运河最困难河段（地势高，水源不足） [来源：运河志]
  - **临清至天津段**（卫运河/南运河）：约400公里 [来源：运河志]
  - **天津至通州段**（北运河）：约150公里 [来源：运河志]
  - **通州至北京**：陆运约40里 [来源：运河志]

#### 1.2 主要节点城市

| 城市 | 位置 | 商业功能 |
|------|------|----------|
| 杭州 | 运河南端 | 丝绸/茶叶集散地，设北新钞关 |
| 苏州 | 江南运河 | 丝绸/棉布中心，设浒墅关 |
| 镇江 | 长江/运河交汇 | 南北货物转运 |
| 扬州 | 运河与长江交汇 | 盐业中心，徽商大本营 |
| 淮安 | 运河中段 | 漕运管理中心（设漕运总督），设淮安钞关 |
| 济宁 | 运河中段 | 北方最大内河港口之一 |
| 临清 | 运河与卫河交汇 | 北方最大商业城市之一，设临清钞关 |
| 天津 | 运河北段 | 海运/河运交汇点 |
| 通州 | 运河北端 | 北京的外港，漕粮仓储地 |

#### 1.3 运输方式

- **漕船**：官方漕运专用船只
  - 漕船标准载量：约400石（约24吨）至500石 [来源：漕运研究]
  - 年运漕粮约400万石（成化至万历间的平均数） [来源：《明史·食货志》]
  - 漕船数量：约一万至一万二千艘 [来源：漕运研究]
- **商船**：民间商船
  - 大型商船载量可达数百石至千石以上 [来源：内河航运研究]
  - 商船数量远超漕船，但缺乏精确统计 [来源：内河航运研究]
- 动力：人力拉纤（逆水段）、风帆（顺风段）、水流（顺水段） [来源：航运史研究]

#### 1.4 运输时间

- 从杭州至北京（通州）全程：
  - 顺利情况下约40至60天 [来源：漕运研究]
  - 受水位、天气、过闸排队等影响，实际常需两三个月甚至更长 [来源：漕运研究]
  - 漕船因需等候过闸和沿途检查，通常比商船更慢 [来源：漕运研究]
- 反向（北至南）因多为顺流，速度更快：约30至40天 [来源：航运研究]

| 南向北 | 丝绸/棉布 | 苏杭产品北销 |
| 南向北 | 茶叶 | 供应北方市场 |
| 南向北 | 瓷器 | 景德镇产品经长江转运河北上 |
| 南向北 | 食盐（两淮盐） | 北销引地 |
| 北向南 | 棉花 | 北方棉花南运至松江等纺织中心 |
| 北向南 | 皮毛/药材 | 北方/蒙古特产南销 |

- 大运河沿线设有多座钞关（商税征收站） [来源：钞关研究]
- 主要钞关及万历六年（1578年）税收数据：

| 钞关 | 位置 | 年税收（银两） |
|------|------|---------------|
| 临清关 | 临清 | 约83,000两（定额；万历二十四年实征117,861两） |
| 河西务关 | 河西务 | 约42,000两 |
| 淮安关 | 淮安 | 约43,000两 |
| 扬州关 | 扬州 | 约25,000两 |
| 浒墅关 | 苏州 | 约52,000两 |

### 二、赣江庾岭路

#### 2.1 路线

- 连接长江流域与珠江流域（华南）的最重要商路，庾岭路上"商贾如云，货物如雨，万足践履，冬无寒土" [来源：商路研究/中国经济史论坛]
- 全程路线：
  - **长江至鄱阳湖**：由九江入鄱阳湖 [来源：交通史研究]
  - **鄱阳湖至赣江**：经南昌、吉安溯赣江而上 [来源：交通史研究]
  - **赣州**：赣江上游终点，陆路转运起点 [来源：赣州方志]
  - **庾岭（大庾岭/梅岭）**：赣粤交界的山岭，海拔约300余米 [来源：地理志]
    - 翻越庾岭的陆路约60至80里 [来源：商路研究]
    - 唐代张九龄开凿梅关古道后成为南北交通要道 [来源：梅关古道研究]
    - 山路以石阶铺设，骡马可行 [来源：商路研究]
  - **庾岭至北江/浈江**：翻过庾岭后进入广东浈江（北江上游） [来源：交通史研究]
  - **韶州（今韶关）**：浈江与武江汇合为北江 [来源：韶州方志]
  - **北江至广州**：顺北江而下至广州 [来源：交通史研究]

#### 2.2 主要节点城市

| 城市 | 位置 | 功能 |
|------|------|------|
| 九江 | 长江/鄱阳湖入口 | 长江与赣江水路的转运点 |
| 南昌 | 赣江中游 | 江西省城，物资集散中心 |
| 吉安 | 赣江中游 | 粮食和瓷器转运 |
| 赣州 | 赣江上游 | 水陆转运关键节点 |
| 南雄 | 庾岭南麓 | 陆路至水路转运点 |
| 韶州（韶关） | 北江上游 | 南北货物汇集 |
| 广州 | 路线终点 | 对外贸易中心 |

#### 2.3 运输方式

- **水运**（赣江段和北江段）：帆船和纤拉，载重从数十石到数百石 [来源：内河航运研究]
- **陆运**（庾岭段）：
  - 骡马驮运为主 [来源：商路研究]
  - 人力挑运（挑夫/"脚夫"） [来源：商路研究]
  - 庾岭山路设有多处歇脚的驿站和店铺 [来源：驿站研究]
- 水陆联运的时间：
  - 赣江水运段约10至15天（视水位） [来源：航运研究]
  - 庾岭陆运段约2至3天 [来源：商路研究]
  - 北江水运段约5至7天 [来源：航运研究]
  - 全程（九江至广州）约需20至30天 [来源：商路研究]

#### 2.4 大宗商品

| 方向 | 主要商品 | 备注 |
|------|----------|------|
| 北向南 | 丝绸/棉布 | 江浙产品南销广东 |
| 北向南 | 瓷器 | 景德镇产品经赣江南下 |
| 北向南 | 茶叶 | 供应广东和出口 |
| 北向南 | 铁器/日用品 | 各种工业品 |
| 南向北 | 白银 | 海外贸易回流的白银 |
| 南向北 | 胡椒/香料 | 进口东南亚香料 |
| 南向北 | 象牙/犀角 | 进口奢侈品 |
| 南向北 | 广东特产 | 水果干、药材等 |

### 三、湘江路

#### 3.1 路线

- 连接长江流域与广西/贵州的商路 [来源：商路研究]
- 全程路线：
  - **长江至洞庭湖**：由岳州（今岳阳）入洞庭湖 [来源：交通史研究]
  - **洞庭湖至湘江**：溯湘江而上 [来源：交通史研究]
  - **长沙**：湘江中游，湖广南部商业中心 [来源：长沙方志]
  - **衡州（今衡阳）**：湘江中上游 [来源：衡州方志]
  - **永州**：湘江上游 [来源：永州方志]
  - 分两条支路：
    - **灵渠路**：经灵渠（秦代开凿）至桂江，再至梧州、西江 [来源：灵渠研究]
    - **陆路至贵州**：经湘西至贵州/云南 [来源：西南商路研究]

#### 3.2 主要节点城市

| 城市 | 位置 | 功能 |
|------|------|------|
| 岳州（岳阳） | 洞庭湖/长江 | 长江与湘江水路转运点 |
| 长沙 | 湘江中游 | 湖广南部商业中心 |
| 衡州（衡阳） | 湘江中上游 | 南下商品中转站 |
| 永州 | 湘江上游 | 通往广西的跳板 |
| 桂林 | 灵渠终点/桂江上游 | 广西省城 |
| 梧州 | 桂江/西江交汇 | 广西最大商业城市 |

#### 3.3 运输方式

- **湘江水运**：帆船和纤拉 [来源：内河航运研究]
  - 水深季节性变化大：夏秋水深通大船，冬春水浅需用小船 [来源：航运研究]
- **灵渠**：
  - 全长约34公里，设陡门（船闸）36座 [来源：灵渠研究]
  - 可通行小型船只（载重约10至20石） [来源：灵渠研究]
  - 古代唯一连接长江水系和珠江水系的人工运河 [来源：灵渠研究]
- **陆运**（至贵州段）：骡马驮运和人力挑运 [来源：西南商路研究]

#### 3.4 运输时间

- 岳州至长沙：约3至5天（顺水快，逆水慢） [来源：航运研究]
- 长沙至桂林（经灵渠）：约15至20天 [来源：商路研究]
- 长沙至贵阳（经陆路）：约20至30天 [来源：商路研究]

#### 3.5 大宗商品

| 方向 | 主要商品 | 备注 |
|------|----------|------|
| 北向南 | 盐（两淮盐/川盐） | 湖广/广西食盐供应 |
| 北向南 | 布匹/丝绸 | 江浙产品南销 |
| 北向南 | 铁器/日用品 | 供应西南边疆 |
| 南向北 | 大米 | "湖广熟天下足" |
| 南向北 | 木材 | 湖南/广西山区林木 |
| 南向北 | 矿产 | 广西/贵州的铜、锡等 |
| 南向北 | 药材 | 西南地区药材 |

### 四、西北丝路（陆上丝绸之路）

#### 4.1 路线

- 明代西北丝路已远不如汉唐繁荣，但仍有一定规模贸易 [来源：丝绸之路研究]
- 受限因素：
  - 明朝与西域关系不如汉唐密切 [来源：明代西域关系研究]
  - 嘉峪关以西为蒙古/吐鲁番势力范围，安全不可靠 [来源：西北边防研究]
  - 海上贸易路线兴起，部分替代了陆上丝路 [来源：贸易史研究]
- 路线：
  - **西安** -- **兰州** -- **凉州（武威）** -- **甘州（张掖）** -- **肃州（酒泉）** -- **嘉峪关** [来源：交通史研究]
  - 嘉峪关以西：**哈密** -- **吐鲁番** -- **中亚** [来源：丝绸之路研究]
  - 明朝实际控制范围至嘉峪关 [来源：明代边防研究]

#### 4.2 主要节点城市

| 城市 | 功能 |
|------|------|
| 西安 | 西北贸易起点和管理中心 |
| 兰州 | 茶马互市重要据点 |
| 凉州（武威） | 河西走廊东端重镇 |
| 甘州（张掖） | 河西走廊中段，军事/商业重镇 |
| 肃州（酒泉） | 河西走廊西端 |
| 嘉峪关 | 明朝西部边关，出入境检查站 |
| 哈密 | 嘉峪关以西第一个重要城镇（明初一度设卫） |

#### 4.3 运输方式

- **骆驼商队**：穿越沙漠和戈壁的主要方式 [来源：丝绸之路研究]
  - 每头骆驼可载约150至200公斤 [来源：交通史研究]
  - 商队规模从数十头到数百头骆驼不等 [来源：丝绸之路研究]
- **骡马驮运**：河西走廊段（有水草供应）也使用骡马 [来源：交通史研究]
- **车运**：路况较好的关中至河西段使用大车 [来源：交通史研究]

#### 4.4 运输时间

- 西安至嘉峪关：约30至40天（约2500里） [来源：驿站研究]
- 嘉峪关至哈密：约15至20天 [来源：丝绸之路研究]
- 嘉峪关至撒马尔罕（中亚）：约三至四个月以上 [来源：丝绸之路研究]

#### 4.5 大宗商品

| 方向 | 主要商品 | 备注 |
|------|----------|------|
| 东向西 | 丝绸/缎匹 | 中国丝绸仍为西域最受欢迎商品 |
| 东向西 | 茶叶 | 供应西北游牧民族和中亚 |
| 东向西 | 瓷器 | 因陆运易碎，量不大 |
| 东向西 | 铁器 | 农具和日用品 |
| 西向东 | 玉石（和田玉） | 新疆特产，价值极高 |
| 西向东 | 马匹 | 西域马/中亚马 |
| 西向东 | 宝石/珊瑚 | 奢侈品 |
| 西向东 | 香料/药材 | 中亚/西亚产 |

#### 4.6 朝贡使团贸易

- 哈密、吐鲁番、撒马尔罕等地使团以朝贡名义入境，实际携带大量商品交易 [来源：朝贡贸易研究]
- 使团人数从数十人到数千人不等 [来源：《明实录》]
- 明政府对使团规模和频率多有限制 [来源：朝贡制度研究]
- 嘉靖后朝贡贸易频率下降，丝路贸易进一步萎缩 [来源：丝绸之路研究]

### 五、闽粤海路

#### 5.1 路线

- 沿中国东南沿海的海上贸易线路，向外延伸至日本、东南亚、印度乃至欧洲 [来源：海上贸易研究]
- 国内段：
  - **宁波** -- **温州** -- **福州** -- **泉州/月港** -- **潮州/汕头** -- **广州/澳门** [来源：沿海航运研究]
- 国际段（东洋线）：
  - 月港/泉州 -- **台湾（大员/鸡笼）** -- **琉球（那霸）** -- **日本（长崎/平户/萨摩）** [来源：东洋贸易研究]
- 国际段（西洋线/南洋线）：
  - 月港/广州 -- **占城** -- **暹罗（大城）** -- **马六甲** -- **爪哇（万丹）** [来源：南洋贸易研究]
  - 月港 -- **马尼拉（吕宋）** [来源：马尼拉贸易研究]
  - 广州/澳门 -- **果阿（印度）** -- **里斯本（葡萄牙）** [来源：葡萄牙贸易研究]

#### 5.2 主要节点港口

| 港口 | 角色 | 备注 |
|------|------|------|
| 宁波 | 对日朝贡贸易港（嘉靖前） | 争贡之役后停废 |

#### 5.3 运输方式

- **船型**：
  - **福船**：福建/浙江产，尖底、深吃水、善于远洋航行，载重数十至400吨 [来源：造船史研究]
  - **广船**：广东产，较福船更大更坚固，载重可达500吨以上 [来源：造船史研究]
  - **沙船**：江苏/浙江产，平底，适合沿海浅水航行 [来源：造船史研究]
- **航行动力**：风帆（中式硬帆），利用季风 [来源：航海史研究]
- **导航**：罗盘（指南针）、针路（航海指南，如《顺风相送》）、牵星术 [来源：航海技术研究]

### 七、商路上的基础设施

#### 7.1 驿站

- 明代全国设驿站约1900余处 [来源：《明会典》]
- 驿站主要服务于官方通讯和官员出行，但商人也可使用（需付费） [来源：驿站研究]
- 大运河沿线驿站最为密集 [来源：驿站研究]

#### 7.2 钞关/关卡

- 除大运河沿线钞关外，各商路沿线也设有税收关卡 [来源：商税研究]
- 税率通常为货值的1%至3%（各关不同） [来源：商税研究]
- 万历二十四年（1596）后矿监税使大幅增加商路课税负担 [来源：矿监税使研究]

#### 7.3 会馆与商栈

- 各商帮在商路沿线主要城市设有会馆 [来源：会馆史研究]
- 会馆提供住宿、仓储、信息交流等服务 [来源：会馆史研究]
- 私人客栈、货栈也是商路基础设施的组成部分 [来源：商业史研究]

## 时间线

- 永乐九年（1411）：重新疏浚会通河（山东段），大运河全线贯通 [来源：《明史·河渠志》]
- 永乐十三年（1415）：停止海运漕粮，全面转为河运 [来源：漕运研究]
- 嘉靖二年（1523）：争贡之役，宁波市舶司停废 [来源：《明实录》]
- 嘉靖三十二年（1553）前后：葡萄牙人进入澳门 [来源：澳门史]
- 隆庆元年（1567）：隆庆开海，月港开放 [来源：《明实录》]
- 万历六年（1578）：钞关税收数据（各关合计约33万两） [来源：《明史·食货志》]
- 万历二十四年（1596）：设矿监税使，商路税负大增 [来源：《明实录》]
- 天启至崇祯间：战乱破坏大运河漕运和各商路 [来源：明末史料]
