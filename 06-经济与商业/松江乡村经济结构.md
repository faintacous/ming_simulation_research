# 松江府乡村经济结构

> 聚焦时段：嘉靖至崇祯（1521-1644）
> 核心议题：种植结构、土地关系、农户家计、赋税负担、城乡流动、社会分化

---

## 概述

松江府（辖华亭、上海、青浦三县）是明代棉纺织业的绝对中心，其乡村经济呈现"稻棉转换 + 家庭手工业 + 市镇网络 + 税役货币化压力"的四层结构。嘉靖至崇祯年间，松江乡村从传统稻作区演变为"棉七稻三"的商品化农业区，农户普遍以"耕+纺"复合经营维持生计。该地区承担全国最重的田赋（亩均税率约为全国平均的7-10倍），叠加官布征解和明末三饷加派，赋税负担构成乡村经济的核心压力。深度商品化的小农手工业体系既是松江经济活力的来源，也使其高度依赖市场价格和白银流通，抗风险能力脆弱。

## 一、种植结构：从稻作主导到"棉七稻三"

### 1.1 棉稻比例与演变

嘉靖到晚明，松江地方文献已见"东乡大半植棉"的格局，多数田地种棉而非种稻。这一转变并非单纯农业变化，而是为纺织供给原料的产业化农业。

| 时段 | 种植格局 | 特征 |
|------|---------|------|
| 嘉靖至隆庆（1521-1572） | 棉作扩张，"大半植棉" | 家织普及，粮食自给率下降 |
| 万历（1572-1620） | "棉七稻三"格局定型 | 市镇网络和布业分工成熟 |
| 天启至崇祯（1620-1644） | 棉作维持但收益恶化 | 白银流通冲击与财政危机叠加 |

[来源：正德《松江府志》；嘉靖《上海县志》；崇祯《松江府志》]

### 1.2 各县差异

- **华亭县**（松江府附郭县）：棉田占比最高，是三梭细布的主要产地，承担府级布解的最大份额（约49%）
- **上海县**：棉田广布，东部沿海地区尤甚，"东乡大半植棉"主要指此区域
- **青浦县**（嘉靖二十一年从华亭分出）：面积最小，布解份额最低（约19%），但同样以棉作为主

[来源：天启元年布解分配数据（范金民论文）]

### 1.3 棉田产量

| 指标 | 数据 | 来源 |
|------|------|------|
| 一般棉田亩产（子花） | 约50-60斤 | 综合估算 |
| 松江府年产棉布 | 约2,000万匹 | 范金民估算 |
| 日产量 | "日以万计" | 正德《松江府志》 |

### 1.4 棉作扩张的影响

- 棉区农户需购买外地粮食（主要来自湖广），形成"湖广熟，天下足"的粮棉互补格局
- 农户粮食自给率大幅下降，生存高度依赖市场交换
- 棉花种植扩大了明代松江的商品化农业比例

[来源：松江报《松江棉纺织业的兴衰》；正德《松江府志》]

## 二、土地关系：自耕农萎缩与永佃权发展

### 2.1 自耕农与佃农比例

明代中后期土地集中趋势加剧，江南地区的自耕农比例持续下降：

| 指标 | 数据 | 来源 |
|------|------|------|
| 自耕农占比 | 约1/10 | 《明代农业生产力的发展和生产关系的演变》 |
| 佃户占比 | 约9/10 | 同上 |
| 江苏出租田地比例 | 略高于50% | 学术估算 |

这一比例在嘉靖至万历间因土地兼并加剧而进一步恶化。松江地区因赋税极重，中小自耕农更难维持，向佃农转化的趋势尤为突出。

[来源：中国经济史论坛]

### 2.2 地租制度

松江属于江南定额租（包租制）的典型区域：

| 租制类型 | 地租率 | 特点 |
|---------|--------|------|
| 定额租 | 约50%-60% | 主流租制，不随收成变化 |
| 分成租 | 约50%（对半分） | 少数地区存在 |

地租形式以实物（米粮）为主，明中后期在商品经济发达地区逐渐出现货币地租。佃农丰年有余而歉年亏损，风险由佃农自行承担。

[来源：知乎《中国经济史笔记》；知乎《地主到底占多少土地》]

### 2.3 永佃权与"一田两主"

明代中后期松江地区逐渐发展出永佃制：

```
一块土地
├── 田底权（田骨） → 属于田底主（原地主）
│   └── 收取"大租"
└── 田面权（田皮） → 属于田面主（佃农）
    └── 可世代相传，可转让买卖
```

- 佃农通过长期耕种或支付押金获得永久耕作权
- 田底权和田面权可分别转让，形成"准产权"
- 部分地区出现田面权再转租的"一田三主"现象
- 永佃权给予佃农一定的土地使用安全感，有利于农业投入

[来源：中国土地法制研究网；南京农业大学学报]

## 三、农户家计：耕纺复合经营

### 3.1 典型自耕农年收支（松江，以10亩为例）

松江农户并非纯农业户，而是"农忙务田、农隙纺织"的双重收入结构。

**收入端**：

| 项目 | 估算 | 说明 |
|------|------|------|
| 水稻收入（3亩稻田） | 约6石米 | 亩产约2石，"棉七稻三"比例 |
| 春花收入（3亩） | 约1.5石 | 小麦/油菜轮作 |
| 棉花收入（7亩棉田） | 约350-420斤子花 | 亩产50-60斤 |
| 纺织收入 | 约3-5两白银 | 家庭妇女纺纱织布，年产布约30-50匹（标布） |
| **毛收入合计** | 约7.5石米 + 3-5两银 | 折银约10-12两 |

**支出端**：

| 项目 | 估算 | 说明 |
|------|------|------|
| 口粮（四口之家） | 约12石米 | 成人年需约3石 |
| 购粮补缺 | 约4.5石（折银约2-3两） | 自产不足，需购买外地粮 |
| 田赋正赋 | 约2.4两 | 亩均约24升，10亩共240升，折银约2.4两 |
| 官布摊派 | 约0.3-0.5两 | 折入各种名目 |
| 种子、农具 | 约0.5两 | 包括棉种、稻种 |
| **支出合计** | 约5-6两 | 不含三饷加派 |

**结余**：正常年份约4-6两，遇灾年或加派则亏损。

[来源：综合《农民.md》收支数据与《棉纺业.md》产量数据推算]

### 3.2 典型佃农年收支（松江，佃种8亩）

| 项目 | 收入/支出 | 说明 |
|------|----------|------|
| 毛收入 | 约8-10两 | 农业+纺织 |
| 地租（50%） | -4-5两 | 定额租制 |
| 口粮购补 | -2-3两 | 棉区需外购粮食 |
| 杂费 | -0.5两 | 种子、工具等 |
| **结余** | 约0-2两 | 极度脆弱，几无储蓄 |

### 3.3 纺织收入的关键补充作用

- "以织助耕，女红有力焉"——纺织收入是松江农户维持生计的关键补充 [来源：正德《松江府志》]
- 男女性别分工被重组：女性参与农作、男性也参与纺织，家庭劳动被更充分商品化
- 松江的"乡村"本质上已嵌入区域手工业分工，而非封闭自然经济
- 标布约2-3钱/匹，三梭布价格为普通白布的二倍以上
- 棉布以出售为目的，商品化程度极高

[来源：正德《松江府志》；棉纺业.md]

### 3.4 生产周期

| 月份 | 农事 | 纺织 |
|------|------|------|
| 正月-二月 | 整地、春花播种 | 纺纱织布（农闲旺季） |
| 三月-四月 | 棉花播种、春花收割 | 持续纺织 |
| 五月-六月 | 棉花管理、插秧 | 减少（农忙） |
| 七月-八月 | 收割早稻 | 减少 |
| 九月-十月 | 采摘棉花、收割晚稻 | 开始轧棉弹花 |
| 十一月-十二月 | 农闲 | 纺纱织布高峰 |

## 四、赋税负担：正赋 + 官布 + 三饷的三重压力

### 4.1 松江田赋的特殊高税率

松江因明初惩罚性税率，长期承担远超全国平均水平的赋税：

| 指标 | 数据 | 来源 |
|------|------|------|
| 松江田地面积 | 约440万亩 | 万历清丈数据 |
| 松江税粮定额 | 约120万石 | 《万历会计录》 |
| 松江亩均税率 | 约2.7斗/亩（27升） | 综合计算 |
| 万历六年亩均税额 | 24.29升 | 综合研究 |
| 全国平均亩均税率 | 约3.5升/亩 | 综合计算 |
| **松江与全国平均之比** | **约7-8倍** | 综合计算 |
| 与北直隶每府州之比 | 约23倍 | 综合研究 |

"苏松赋税半天下"——松江面积仅为苏州的十分之三，赋税却约为苏州的二分之一。

[来源：《万历会计录》；黄仁宇《十六世纪明代中国之财政与税收》；叶梦珠《阅世编》]

### 4.2 官布征解负担

松江府每年须向朝廷贡解大量棉布，是独特的实物赋税：

| 指标 | 数据 | 时期 |
|------|------|------|
| 松江府年解布 | 约132,774匹 | 天启元年定额 |
| 其中三梭细布 | 33,000匹 | 含御用布5,000匹 |
| 其中阔白棉布 | 99,774匹 | 粗布 |
| 布价银（细布） | 每匹7-8钱1分 | 万历年间 |
| 布价银（粗布） | 每匹3-4钱3分 | 万历年间 |
| 全国官布中苏松常占比 | 约80% | 弘治十五年数据 |

官布征解被称为"最苦差役"——解户"每年费至千金以外，虽巨室大姓，往往立破"。解运过程经历验布、印布、包装、运输、京中交纳等环节，每匹除铺垫外赔银4钱2分。

[来源：范金民《明清江南官布之征解》]

### 4.3 三饷加派（崇祯年间）

松江在正赋和官布之上，还须承担崇祯朝的三饷加派：

| 饷名 | 亩加额度 | 松江年增负担（约440万亩） |
|------|---------|------------------------|
| 辽饷 | 每亩1分2厘 | 约5.3万两 |
| 练饷 | 每亩1分 | 约4.4万两 |
| 剿饷 | 每亩约数厘 | 约1-2万两 |
| **三饷合计** | 约2分2厘-2分5厘 | **约10-12万两** |

### 4.4 农户赋税总负担示例

以崇祯末年拥有10亩田的自耕农为例：

| 项目 | 金额（两） | 说明 |
|------|----------|------|
| 正赋（折银） | 约2.0-2.4 | 亩均约24升折银 |
| 官布摊派 | 约0.3-0.5 | 折入赋额 |
| 辽饷 | 0.12 | 10亩×1.2分 |
| 练饷 | 0.10 | 10亩×1分 |
| 剿饷 | 约0.05 | 名义停征但仍征 |
| 火耗等附加 | 约0.3-0.5 | 地方额外加征 |
| **合计** | **约3.0-3.5两** | 占毛收入约30-35% |

南直隶苏松地区最极端时，"一亩民田价值七八两银子，每年需纳饷至十两"——一年赋税已超过土地本身价值。

[来源：田赋.md；三饷加派.md；一条鞭法.md]

### 4.5 赋税时间线

| 时段 | 亩均实际税负（升/银） | 变化原因 |
|------|---------------------|---------|
| 嘉靖初（1521-1545） | 约24升（折银约1.5-2两/10亩） | 继承洪武惩罚性高税率 |
| 一条鞭法后（1581-1618） | 约2-2.5两/10亩 | 折银征收，略有均平 |
| 辽饷开征后（1618-1637） | 约2.5-3两/10亩 | 亩加9厘至1.2分 |
| 三饷叠加后（1639-1644） | 约3-3.5两/10亩（不含火耗） | 正赋翻倍 |
| 极端情况 | 超过土地价值 | "赋愈重则逃愈多" |

## 五、城乡流动：棉布外销与粮食外购

### 5.1 棉布从乡村到市镇

松江乡村的棉布产品经市镇网络集散流通：

```
农户家庭纺织 → 乡村集市/牙行 → 市镇棉布行 → 外地布商 → 全国各地
```

- 晚明朱泾、朱家角等镇成为花布交易节点
- "布商挟重资而来，白银动以数万两，多则数十万两"
- 牙行经纪人充当中介，组织从乡村收购棉布
- 组织形态长期停留在家庭生产 + 经纪人/牙人收购网络，未转向工场制

[来源：棉纺业.md；Codex校验稿]

### 5.2 国内贸易流向

| 方向 | 路线 | 说明 |
|------|------|------|
| 北上 | 松江→南京→北京及北方各省 | 经运河北运，最主要流向 |
| 西行 | 松江→湖广→四川 | 沿长江西运 |
| 南下 | 松江→浙江→福建→广东 | 海路或陆路 |
| 外销 | 松江→广州/月港→东南亚/日本 | "南京布"出口 |

### 5.3 粮食从外地到棉区

松江后期需大量输入粮食，形成跨区域的粮棉交换链：

```
湖广/江西（产粮区） → 运河/长江 → 松江（棉布区）
                                  ↕
                        松江棉布 → 全国各地
```

- 松江后期需输入粮食，同时输出棉布并吸纳外来原棉
- 形成"粮食外购—纺织外销"的区域分工链
- 这意味着松江乡村对市场价格、交通和商人中介高度依赖

[来源：Codex校验稿；棉纺业.md]

### 5.4 原棉流入

松江本地棉花产量无法完全满足纺织需求，需从外地输入原棉：

- 外来原棉主要来自苏州太仓、嘉定等周边产棉区
- 更远处的河南、山东等华北产棉区也有输入
- 松江成为原棉的净输入、棉布的净输出区域

## 六、社会分化：士绅避税与流民问题

### 6.1 士绅避税转嫁

松江作为江南士绅密集区，各种避税手段使赋税负担向中小农户转移：

**投献**：百姓将田产名义上转移到有功名的士绅名下，以借用优免特权逃避赋役。百姓向士绅缴纳远低于应缴税额的"挂名费"，实际耕种和收益仍归原百姓。嘉靖至万历间尤为普遍。

**诡寄**：大地主将名下田产分割，假托在多个小户名下，降低户等以减少赋役。变体包括"铁脚诡寄"（固定分散假名）和"通天诡寄"（形成地方惯例）。

**飞洒**：富户勾结里书，在编制黄册时将自己名下的田产减少，将赋税"飞洒"到其他无力抗辩的小户名下。

**其他手段**：
- 拖欠：直接拖延不缴，依靠士绅身份使催征困难
- 隐匿田亩：在黄册和鱼鳞图册中隐瞒实际田产
- 买富差贫：贿赂里书和差役，将差役转派给贫户

[来源：士绅阶层.md]

### 6.2 万历优免新例的制度漏洞

万历三十八年（1610）《优免新例》大幅扩大了士绅的优免田亩：

| 身份 | 优免田亩 |
|------|---------|
| 一品京官 | 10,000亩 |
| 未仕进士 | 最高3,350亩 |
| 未仕举人 | 1,200亩 |
| 生员、监生 | 80亩 |

优免范围的扩大为士绅的土地兼并提供了更大的制度空间，加速了松江自耕农的消亡。

[来源：士绅阶层.md]

### 6.3 赋税转嫁的社会后果

- "田多者反居下役"——拥有大量田产的士绅反而承担最轻的赋役
- 中小自耕农成为赋税的主要承担者，负担日益加重
- 松江地区嘉靖三十年至三十九年金花银逋欠率达34.8%
- 崇祯年间"赋愈重则逃愈多，逃愈多则赋愈重"的恶性循环

[来源：田赋.md；三饷加派.md]

### 6.4 流民问题

- 无力缴税的农民弃田逃亡，加入流民大军
- 留下的百姓需分摊逃亡户的赋税（"带征"），负担进一步加重
- 崇祯末年北方多省实际征收率不足正额的50%
- 陕西、河南等地流民成为农民起义的主力

### 农户经济参数

以下参数可用于NPC经济行为设计：

**自耕农（10亩田，四口之家）**：

| 参数 | 基准值 | 波动范围 | 说明 |
|------|--------|---------|------|
| 年毛收入 | 10两 | 7-14两 | 含农业+纺织 |
| 年固定支出 | 5.5两 | 5-6.5两 | 口粮+种子+农具 |
| 正赋 | 2.4两 | 2-3两 | 松江高税率 |
| 三饷加派（崇祯） | 0.25两 | 0.2-0.3两 | 按亩计征 |
| 年净结余 | 2-4两 | -2至+6两 | 灾年可为负 |
| 破产阈值 | 连续亏损2年 | — | 被迫卖田或投献 |
| 日纺纱量 | 4两纱 | 3-5两 | 一个妇女 |
| 日织布量 | 1匹 | 0.5-1.5匹 | 标布，含经纬工序 |

**佃农（佃种8亩，四口之家）**：

| 参数 | 基准值 | 波动范围 | 说明 |
|------|--------|---------|------|
| 年毛收入 | 8两 | 5-11两 | 含农业+纺织 |
| 地租 | 4两 | 3.5-5两 | 约50%定额租 |
| 口粮+杂费 | 3两 | 2.5-4两 | 需外购粮食 |
| 年净结余 | 0-1两 | -2至+2两 | 极度脆弱 |
| 破产阈值 | 单年大灾 | — | 沦为雇农或流民 |
| 纺织收入占比 | 约40% | 30%-50% | 纺织是维持线 |

**关键经济行为触发条件**：
- 当结余连续为负时 → 尝试投献士绅名下 / 卖田 / 逃亡
- 当棉布价格下跌超过20% → 收入显著下降，触发困难状态
- 当粮价上涨超过30% → 购粮支出激增，挤压纺织利润
- 当白银相对粮食升值 → "银贵谷贱"，折银缴税更加困难
