# 匠户制度

## 概述

匠户制度是明代官府对手工业者实行的户籍管理制度，属工部管辖。所有被编入匠籍的手工业者及其家庭世代承袭工匠身份，不得脱籍或分户，需无偿为朝廷服役。匠户分为住坐匠（常驻京师，每月上工10日）和轮班匠（分布各省，四年一班赴京服役三月）两大类。制度经历了从严格人身控制到逐步松弛的过程：成化二十一年（1485）首允纳银代役（南匠月银九钱，北匠六钱），嘉靖四十一年（1562）全面推行班匠银（每名年银四钱五分），清顺治二年（1645）正式废除匠籍。

---

## 一、住坐匠与轮班匠的分类

### 1.1 两类匠户对比

| 对比项 | 住坐匠 | 轮班匠 |
|--------|--------|--------|
| **驻地** | 京师（南京/北京） | 分布全国各地 |
| **服役方式** | 在所在机构**常年服役** | 每隔数年轮流**赴京服役三个月** |
| **服役频率** | 每月服役**10天**，歇20天（年服役120天，闰年130天） | 每班服役**3个月**，间隔数年 |
| **路费** | 无需长途跋涉 | 赴京路费**自行承担**，往返常达三四个月 |
| **人数** | 较少（集中于京师） | **绝大多数**（分布各省） |
| **管辖** | 民匠隶内府**内官监**；军匠隶都司卫所 | 隶属**工部** |
| **报酬** | 上工期间领少量月粮（一般**三斗**，多者至一石） | 服役期间仅领少量月粮和直米 |

### 1.2 住坐匠的细分

住坐匠按户籍分为两种：

| 类型 | 说明 |
|------|------|
| **民匠** | 从民间征集，隶于匠籍，属工部管辖，主要服务于皇族 |
| **军匠** | 隶于军籍，属都司卫所管辖 |

- 住坐匠须连家带小迁到工场所在地，住在固定坊厢之内
- 按分工编定排甲
- 每月上工10日，休息20日——"住坐之匠，例月上工十日"

### 1.3 轮班匠的服役规则

| 规则 | 内容 |
|------|------|
| 服役时间 | 每班三个月 |
| 劳动性质 | **无偿劳动**，无报酬 |
| 路费 | 全部**自行承担** |
| 实际耗时 | 路途遥远者往返加服役共需**六七个月** |
| 困苦程度 | "疲于奔命，苦不堪言" |

## 二、轮班周期的演变

### 2.1 周期变化时间线

| 时间 | 轮班周期 | 说明 |
|------|---------|------|
| **洪武十九年（1386）** | **三年一班** | 轮班制正式确立 |
| **洪武二十六年（1393）** | 定为**五种班次** | 按工种和距离分为不同班次 |
| 洪武二十六年-景泰四年 | 五种班次沿用**60余年** | 基本稳定 |
| **景泰五年（1454）** | 统一改为**四年一班** | 减轻工匠负担 |
| 此后至嘉靖 | **四年一班** | 但实际执行日渐松弛 |

### 2.2 五种班次的含义

洪武二十六年所定的五种班次，根据工匠的**工种**和与京师的**距离远近**确定不同的轮班周期和服役时间：
- 距离近的工匠轮班较频繁（如二年一班、三年一班）
- 距离远的工匠间隔较长（如四年一班、五年一班）
- 景泰五年的改革将五种班次**统一为四年一班**，简化了管理

## 三、匠户人数

### 3.1 各类匠户的数量

| 类别 | 人数 | 时期 | 说明 |
|------|------|------|------|
| 北京地区匠户 | 约**27,000户** | — | 含各部会工匠 |
| 军匠（卫所） | **26,000户** | 宣德年间 | 正匠26,000人，加帮贴丁超过50,000人 |
| 轮班匠 | 占总匠户**绝大多数** | — | 分布各省 |
| 住坐匠 | 较少 | — | 集中于京师 |

### 3.2 各匠正匠与帮贴丁

- 每户正匠做工，免除杂役
- 另免家内一丁以帮贴应役
- 以军匠为例：正匠26,000人 + 帮贴丁约26,000人 = 超过**50,000人**

## 四、纳银代役改革时间线

### 4.1 完整改革进程

| 时间 | 改革内容 | 意义 |
|------|---------|------|
| **正统初年** | 英宗令南方工匠出银代役，由官府雇觅在京工匠供应 | 最早的纳银代役尝试 |
| **成化二十一年（1485）** | 首次正式允许轮班匠纳银代役：**南匠月银九钱，北匠月银六钱** | 纳银代役的制度化开端 |
| 弘治年间 | 纳银代役继续实行但尚未全面推广 | 过渡期 |
| 正德年间 | 部分地区推行纳银代役 | 逐步扩大 |
| **嘉靖四十一年（1562）** | 全面推行"班匠银"制度：**每名轮班匠每年纳银四钱五分** | 轮班制名存实亡 |
| 嘉靖四十一年令文 | "自本年秋季为始，将改年班匠**通行征价类解**，不许私自赴部投当" | 彻底改为征银 |

[来源：中国法学网《中国古代对于手工业的管理及立法》](http://iolaw.cssn.cn/gdfls/200411/t20041104_4593789.shtml)；[维基百科《轮班匠》](https://zh.wikipedia.org/zh-hans/%E8%BD%AE%E7%8F%AD%E5%8C%A0)

### 4.2 清初正式废除

| 时间 | 事件 | 意义 |
|------|------|------|
| **顺治二年（1645）** | 清政府颁令"**各省俱除匠籍为民**"，免征班匠银 | 匠户制度**正式废除** |
| 顺治以后 | 实施"计工给值"/"按工给值"制度 | 工匠改为雇佣制 |
| **雍正四年（1726）** | 各省将班匠银**归并田亩或地丁带征** | 班匠银作为独立税种消失 |

[来源：趣历史《清朝政府为何宣布废除匠籍制度》](http://m.qulishi.com/article/202104/499648.html)；[维基百科《匠籍》](https://zh.wikipedia.org/zh-hans/%E5%8C%A0%E7%B1%8D)

### 4.3 改革的驱动力

| 因素 | 说明 |
|------|------|
| **工匠大量逃亡** | 严苛制度导致消极怠工或逃亡，在册工匠日益减少 |
| **商品经济发展** | 白银货币化使以银代役成为可能 |
| **行政管理困难** | 轮班制管理成本高、效率低 |
| **财政需要** | 征收银两比征调劳役更便于使用 |

## 五、班匠银数额

### 5.1 成化二十一年（1485）的初始标准

| 类别 | 月银 | 年银 | 说明 |
|------|------|------|------|
| **南匠** | **九钱**（0.9两） | 10.8两 | 南方各省工匠 |
| **北匠** | **六钱**（0.6两） | 7.2两 | 北方各省工匠 |

> **南北差异原因**：南匠赴京路途更远，若亲自服役花费更大，因此代役银也更高。此阶段为**自愿纳银**，价格需有足够吸引力让工匠主动选择。

原文："输班工匠有愿出银价者，南匠出银**九钱**免赴京，北匠出银**六钱**，到部随即批放。"

### 5.2 嘉靖四十一年（1562）的统一标准

| 类别 | 年银 | 说明 |
|------|------|------|
| 所有轮班匠（统一） | **四钱五分**（0.45两） | 大幅降低，**强制征收** |

### 5.3 数额变化的分析

| 对比 | 成化标准（年） | 嘉靖标准（年） | 变化 |
|------|--------------|--------------|------|
| 南匠 | 10.8两 | 0.45两 | 降幅**96%** |
| 北匠 | 7.2两 | 0.45两 | 降幅**94%** |

变化原因：
- 成化时是**可选的**（愿纳者纳），价格需有吸引力
- 嘉靖时改为**强制征收**，需所有工匠都能承担
- 0.45两/年对普通手工业者是可承受的负担

## 六、匠户的实际生活状况

### 6.1 服役之苦

| 困境 | 说明 |
|------|------|
| **无偿劳动** | 仅在上工期间领取少量月粮和直米，无其他报酬 |
| **路途奔波** | 轮班匠往返常达三四个月，路费全部自担 |
| **耽误生计** | 服役加路途共需六七个月，家中正常生计中断 |
| **世代束缚** | 全家须世代承袭工匠身份，不得脱籍或分户 |
| **社会歧视** | 匠户地位低于民户 |

### 6.2 逃匠现象

严苛的制度导致了大规模逃亡：

| 时期 | 逃匠人数 | 说明 |
|------|---------|------|
| 正统三年（1438） | **4,255人** | 有记录的逃亡数 |
| 正统十年（1445） | 约**10,000人** | 逃亡加剧 |
| 景泰元年（1450） | **34,800余人** | 达到高峰 |

- 《大明律》规定对逃匠予以追捕和惩处
- 但随着逃匠人数增多，实际难以执行
- 逃亡的普遍化是推动纳银代役改革的重要因素

### 6.3 匠官制度

- 少数技艺超群的工匠可被授予官职，称为"**匠官**"
- 匠官制度是匠户改善地位的少数途径之一
- 但数量极少，对大多数匠户而言遥不可及
- 仅可通过科举等极少数途径改变身份

## 七、制度的完整生命周期

### 7.1 制度演变总表

| 阶段 | 时间 | 变化 | 说明 |
|------|------|------|------|
| **元代奠基** | 1271-1368 | 将民间工匠强行编入匠籍 | 明承元制 |
| **严格执行期** | 洪武-正统 | 轮班制严格执行，逃匠受罚 | 约80年 |
| **松弛期** | 景泰-成化 | 逃匠日增，轮班周期延长至四年 | — |
| **纳银代役开始** | 成化二十一年（1485） | 允许以银代役 | 自愿性质 |
| **全面推行班匠银** | 嘉靖四十一年（1562） | 强制征银，轮班制名存实亡 | 匠籍名义保留 |
| **匠籍名义存续** | 嘉靖-崇祯 | 匠籍仍在，但实际服役基本消失 | — |
| **正式废除** | 顺治二年（1645） | "各省俱除匠籍为民" | — |
| **班匠银归并** | 雍正四年（1726） | 班匠银归并地丁 | 税种消失 |

### 7.2 废除的历史意义

| 意义 | 说明 |
|------|------|
| **人身解放** | 手工业者获得身份自由，不再世代束缚 |
| **劳动力市场化** | 工匠可自由选择雇主和职业，促进劳动力合理流动 |
| **手工业发展** | 改为雇佣制后，工匠积极性和技术水平提高 |
| **社会结构变化** | 手工业者地位提升，不再被视为"贱役" |
