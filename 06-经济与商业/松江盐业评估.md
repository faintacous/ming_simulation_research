# 松江盐业评估（下沙盐场体系）

## 概述

松江府下沙盐场是两浙盐运司浙西27个盐场之冠，元初定额年产6,683吨，明正统年间峰值达8,450吨/年，灶丁约15,000人。盐场以三场九团形式组织，沿海岸绵延约50公里。嘉靖年间遭倭寇破坏后持续衰落，万历年间产量降至约2,820吨/年。随着海岸线东移导致盐水浓度下降，盐场逐步萎缩，退出的盐碱地转为棉花种植，推动了松江棉纺织业的繁荣——这一"盐→棉"产业转型链是松江经济史上最具特色的结构性变迁之一。经评估，下沙盐场体系值得作为独立游戏节点纳入模拟。

## 关键事实

- 下沙盐场为浙西27个盐场产量之冠 [来源：下沙的厚重 - 长江文化网]
- 元初定额年产6,683吨，明正统年间峰值约8,450吨/年 [来源：南汇盐业兴衰 - 段炼]
- 洪武年间灶丁约15,000人 [来源：南汇盐业兴衰 - 段炼]
- 组织形式为三场九团，沿海岸绵延约50公里 [来源：下沙的厚重 - 长江文化网]
- 嘉靖三十二年（1553年）倭寇入侵，严重破坏盐场设施 [来源：海上记忆 - 上观新闻]
- 万历年间产量降至约2,820吨/年，为峰值的三分之一 [来源：南汇盐业兴衰 - 段炼]
- 海岸东移 → 盐水浓度下降 → 盐场萎缩 → 退出盐碱地种棉 → 棉纺织业繁荣，形成完整的产业转型链 [来源：南汇盐业兴衰 - 段炼；海上记忆 - 上观新闻]
- "水乡灶户"经历制度性转化，从盐业生产者转为棉花种植与纺织从业者 [来源：官退商进 - 深圳大学学报]

## 时间线

- 元初：下沙盐场定额6,683吨/年，为浙西诸场之首 [来源：段炼]
- 洪武年间：灶丁约15,000人，三场九团体制运行 [来源：段炼]
- 正统年间（1436-1449）：产量达峰值约8,450吨/年 [来源：段炼]
- 嘉靖三十二年（1553）：倭寇入侵破坏盐场 [来源：上观新闻]
- 嘉靖至万历：海岸线持续东移，盐水浓度下降，盐场逐步萎缩 [来源：段炼]
- 万历年间：产量降至约2,820吨/年，大量盐碱地转为棉田 [来源：段炼]
- 明末：盐→棉产业转型基本完成，松江成为全国棉纺织业中心 [来源：综合]

## 关联主题

- → 06-经济与商业/棉纺业：盐场萎缩后退出的盐碱地转为棉田，直接推动棉纺织业繁荣
- → 06-经济与商业/松江市镇经济特色：盐业衰落与棉业兴起改变了市镇的经济结构
- → 06-经济与商业/棉布市镇集散体系：产业转型后的市镇集散网络
- → 05-军事与边防/金山卫：嘉靖倭寇破坏盐场与沿海防御体系密切相关
- → 04-赋税与财政：灶户的盐课负担与制度性转化
- → 07-社会阶层：灶户作为特殊户籍群体的身份变迁

## 存疑 & 待查

- 三场九团的具体空间分布和各团产量差异，需进一步查证地方志
- 灶丁15,000人的统计口径（是否含家属？是在册灶丁还是实际从业人口？）
- "水乡灶户"制度性转化的具体时间节点和法律依据，灶户户籍何时正式解除？
- 盐→棉转型过程中是否存在过渡期的盐棉兼营阶段？
- 元初定额与明正统峰值之间的产量变化曲线是否平稳？
- 万历以后至明末盐场是否完全停产，还是维持少量生产？
- 下沙盐场与周边其他盐场（如袁浦场、青村场）的关系和产量对比

## 设计对接参考

- **地方类别**：盐田/矿场
- **空间类型组合**：盐场、工棚、称量处、灶户聚落
- **核心游戏价值**：盐业兴衰→棉业兴起的产业转型链——玩家可体验从煮盐到植棉的经济结构变迁，这一转型链在时间跨度和因果逻辑上都具有高度可玩性
- **与下沙/新场镇的关系**：下沙为盐场核心区域，新场镇因盐场贸易兴起而得名（"新盐场"简称），两者共同构成盐业生产与交易的空间双核。随着盐业衰落，新场镇逐步转型为棉布集散市镇
