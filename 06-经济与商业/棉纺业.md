# 棉纺业

## 概述

明代是中国棉纺织业的黄金时期。棉花在元代黄道婆改良纺织技术后迅速普及，到明代已成为全国最重要的衣着原料。松江府（今上海地区）是棉纺织业的绝对中心，明末年产棉布约2,000万匹，"衣被天下"。嘉靖至崇祯年间（1521-1644），松江棉布不仅满足国内需求，还作为"南京布"（Nankeen）出口海外。民谚"买不尽松江布，收不尽魏塘纱"生动反映了其市场规模之巨。

| 其中松江府 | 约**2,000万匹** | 范金民估算 |

| 日产量 | "日以万计" | 正德《松江府志》 |

[来源：清华大学《明代GDP试探》](https://www.sem.tsinghua.edu.cn/__local/A/EA/CD/D2CF2AC419259F603A93B571197_744A52E0_824B0.pdf)

### 1.2 产量的基础

- 松江府下辖华亭、上海、青浦等县，几乎**家家纺织**
- "比户纺织，家家出布"——棉纺织是普通农户的重要副业乃至主业
- 分散经营但总量惊人，是典型的家庭手工业模式
- 正德《松江府志》载：乡间"尤为精敏，所出布匹，日以万计"

### 1.3 税收指标

| 指标 | 数据 | 时期 |
|------|------|------|
| 全国征收棉布 | **176万匹** | 万历六年（1578） |
| 山东/河南/河北/山西四省占比 | 全国的**65.8%** | 万历六年 |
| 松江府赋税额 | 全国最高 | "苏松赋税半天下" |

## 二、棉花种植区域分布

### 2.1 主要种植区

| 地区 | 种植情况 | 特点 |
|------|---------|------|
| **松江府（上海地区）** | 东乡"大半植棉"，**"棉七稻三"**格局 | 核心产区，品质最优 |
| **苏州府** | 太仓、嘉定等地广种棉花 | 与松江相连 |
| **河南** | "中州沃壤，半植木棉" | 华北最大产棉区 |
| **山东** | 东昌等地植棉较多 | "贸易四方，为利甚溥" |
| **河北（北直隶）** | 种植面积广 | 华北重要来源 |
| **山西** | 有一定种植 | 面积最小 |
| **湖广** | 明代开始扩大 | 新兴产区 |
| **两广、福建** | 有种植 | 以热带棉（草棉）为主 |

### 2.2 种植格局的特点

**长江流域产棉区**：以江苏、湖北、安徽、江西、湖南为主，光照充足、热量丰富、水热协调。

**黄河流域产棉区**：以河南、河北、山东、陕西、山西为主，光照充足、土壤肥力适中、纤维品质较好。

[来源：松江报《松江棉纺织业的兴衰》](http://m.shsjb.com/index.php/home/article/index/appkey/258/date/2020-08-19/aid/5721892.html)

### 2.3 种植政策与产量

| 指标 | 数据 |
|------|------|
| 洪武年间政策 | 规定农户按田亩种植一定比例棉花和桑树 |
| 棉花折米比价 | "以棉花一斤折米二斗"（鼓励种棉） |
| 一般棉田亩产 | 子花约**50-60斤** |

### 2.4 种植格局的影响

- 松江等地"棉七稻三"——多数田地种棉而非种稻
- 棉区农户需购买外地粮食（主要来自湖广）
- 形成了"**湖广熟，天下足**"的粮棉互补格局
- 棉花种植扩大了明代的商品化农业比例

## 三、纺织工艺

### 3.1 黄道婆的贡献

- **黄道婆**（约1245-?），松江乌泥泾人，被尊为"布业始祖"
- 从海南黎族学习先进纺织技术后回到松江，传授给当地百姓
- 改良了棉花的**轧籽、弹花、纺纱、织布**全套工艺
- 使松江布从粗糙、单一、稀松变为精致、牢固、美观

### 3.2 主要纺织工具

| 工序 | 工具名称 | 功能 |
|------|---------|------|
| **轧棉** | 搅车（脱籽机） | 将棉花与棉籽分离 |
| **弹花** | 弹弓（椎弓）、弹花车 | 将脱籽后的棉花弹松 |
| **纺纱** | 纺车（纬车） | 将棉花纺成棉纱 |
| **整经** | 经车、经床 | 将纱线整理为经线 |
| **绕线** | 绕锭架、卷筳 | 将纱线绕成团或管 |
| **织布** | 织布机（踏板织机） | 将经纬线交织成布 |
| **后加工** | 轩床、线架 | 纱线后加工整理 |
| **踹布** | 踹布石 | 压光整理，使布匹光滑平整 |

### 3.3 技术改进

- 明中叶后出现**搅车、卷筳、轩床、线架**等改良工具
- **脚踏纺车**取代手摇纺车，效率大幅提高
- 脚踏式织机由黄道婆时代发展至明代已相当成熟
- 弹棉花技术精进："花皆飞起，用以织布，尤为精软"

### 4.3 市场交易规模

- "布商挟重资而来，白银动以数万两，多则数十万两"
- 外地商人纷纷到松江收布，形成大规模的批发市场
- 松江形成了棉花种植→纺纱→织布→踹布→销售的完整产业链

## 五、"买不尽松江布"的市场规模

### 5.1 谚语出处

万历《嘉兴府志》记载：

> "贫者无资本不能织，日卖数两线以糊口，故有'买不尽松江布，收不尽魏塘纱'之谣。"

此谚语生动反映了松江棉布产量之巨大和供给之源源不断。

### 5.2 经济背景

| 因素 | 说明 |
|------|------|
| **全民纺织** | 松江府几乎家家户户从事棉纺织 |
| **商品化程度高** | 棉布以出售为目的，非自给自足 |
| **专业分工** | 已形成种棉→纺纱→织布→踹布→销售的完整产业链 |
| **市场需求旺盛** | 全国人口增长，棉布是最基本的衣着材料 |
| **贸易网络发达** | 通过运河、长江和海路覆盖全国乃至海外 |

### 5.3 国内贸易流向

| 方向 | 路线 | 说明 |
|------|------|------|
| **北上** | 松江→南京→北京及北方各省 | 经运河北运，最主要流向 |
| **西行** | 松江→湖广→四川 | 沿长江西运 |
| **南下** | 松江→浙江→福建→广东 | 海路或陆路 |
| **外销** | 松江→广州/月港→东南亚/日本 | 海外贸易 |

### 5.4 社会影响

- "苏松赋税半天下"——松江因棉纺织业成为国家最重要的财税来源地
- 棉纺织业改变了松江的社会结构：从传统农业转变为商品经济主导
- 农户生计从依赖粮食生产转向依赖棉布生产和市场交换
- 松江的人口密度和城镇化水平远高于其他农业地区

## 六、棉布的对外出口

### 6.1 "南京布"（Nankeen）的由来

- "Nankeen"是西方商人对中国棉布的通称
- 实际产地并非南京，而是松江府等江南地区
- 葡萄牙语和西班牙语中称"松江布"，英国商人后改称"南京布"

[来源：浙江大学学报《丝绸之路续篇："南京布"的外销》](https://www.zjujournals.com/soc/article/2017/1008-942X-47-3-5.html)

### 6.2 明代出口情况

- 明末松江棉布已开始通过广州和月港出口到东南亚
- 通过马尼拉贸易出口到美洲
- 晚明江南棉布在墨西哥"挤占了西班牙货"（樊树志考证）
- 英国东印度公司发现苏松棉布不易褪色，品质优于广州产品

### 6.3 清代出口高峰（可作参考）

| 年份 | 出口量 | 说明 |
|------|--------|------|
| 1786年 | 372,020匹 | 早期出口 |
| 1795年 | 1,005,000匹 | 增长迅速 |
| 1819年 | 超过**3,300,000匹** | 出口最高峰 |
| 1786-1833（48年） | 累计超过**4,400万匹** | 英法荷瑞丹等国商船购买 |

[来源：北京大学郭卫东《"南京布"的外销》](http://ayacd.yrdcpcn.org.cn/upload/2023/0726/b8dc86e1-67bc-4c77-9c5b-8accbe16e6f9.pdf)；[樊树志《晚明江南棉布在墨西哥挤占了西班牙货》](https://news.qq.com/rain/a/20241024A04PUN00)

## 存疑 & 待查

- 明代松江棉布的确切年产量数据——"2,000万匹"为范金民估算，其他学者的估算数据范围
- 明代棉布的更详细价格数据（不同品种、不同地区的差价）
- 松江棉布踹布业（后加工）的具体规模和组织形式
- 明代棉布出口的精确数量——目前明代数据较少，多引用清代数据
- 棉纺织业对松江地区粮食供应的影响程度
- "棉七稻三"格局的形成时间和地域差异
