# 白银流通

## 概述

明代中后期（1521-1644），白银从非法流通的民间货币转变为事实上的法定货币，成为帝国财政和商业活动的核心。嘉靖至崇祯年间，中国国内银矿产出远不能满足经济需求，海外白银大规模涌入——日本和美洲白银通过贸易网络流入中国，使明朝成为全球白银的"终极吸纳者"。白银以称量货币形式流通，没有标准铸币，交易依赖戥子称重和成色鉴定，由此衍生出火耗等附加成本。

## 一、国内银矿产出

### 1.1 主要银矿分布

明代银矿主要分布在云南、浙江、福建、四川等省，其中云南银产量长期居全国首位。

| 省份 | 主要矿区 | 说明 |
|------|---------|------|
| **云南** | 大理、楚雄等地 | 全国银课之半出于云南 |
| **浙江** | 处州（今丽水）等地 | 永乐年间银课一度达82,070两 |
| **福建** | 建宁、延平等地 | 永乐年间银课32,800两 |
| **四川** | — | 天顺年间银课13,517两 |

[来源：中国经济史论坛《明代的银课与银产额》](http://economy.guoxue.com/?p=6571)

### 1.2 各时期银课数据

"银课"是政府从银矿中征收的税银，按课税比例可以推算银产量。

| 时期 | 浙江银课 | 福建银课 | 云南银课 | 说明 |
|------|---------|---------|---------|------|
| 洪武年间（1368-1398） | 2,870两 | 2,670两 | — | 初期规模小 |
| 永乐年间（1403-1424） | 82,070两 | 32,800两 | — | 产量高峰 |
| 宣德年间（1426-1435） | 94,040两 | 40,270两 | — | 继续增长 |
| 天顺年间（1457-1464） | 38,930两 | 28,250两（实际13,400余两） | 102,380两 | 浙闽下降，云南上升 |
| 万历八年（1580） | — | — | 5-6万两 | 明显下降 |

[来源：中国经济史论坛《明代的银课与银产额》](http://economy.guoxue.com/?p=6571)

### 1.3 云南银矿的核心地位

- 自天顺二年（1458）后的一百二十余年中，云南每年银课收入多时超过十万两
- 在全国银课总额中，云南每年约占百分之五十以上
- 万历八年（1580），云南银课降至5-6万两，宋应星言云南银课"占有全国银产量之半"
- 以此推算，全国银课最多时每年不过10-12万两

[来源：中国经济史论坛《明清时代云南的银课与银产额》](http://economy.guoxue.com/?p=6618)

### 1.4 国内银产的严重不足

明代国内银产远不能满足财政和市场需求：

- 嘉靖至万历时期，朝廷每年财政开支300-600余万银两
- 国产矿课仅占国库白银来源的约13%
- 其余87%以上来自田赋和工商税收——而这些税银本身大量源自海外白银流入

[来源：中国经济史论坛《明朝中后期世界白银为何大量流入中国》](http://economy.guoxue.com/?p=7414)

## 二、海外白银流入

### 2.1 两条主要渠道

| 渠道 | 起始时间 | 路线 | 说明 |
|------|---------|------|------|
| **日本白银** | 约1540年代 | 日本→中国沿海（直接贸易）/ 日本→东南亚→中国（转口） | 石见银山1533年引入灰吹法后产量激增 |
| **美洲白银** | 约1570年代 | 美洲→马尼拉→中国（太平洋航线）/ 美洲→欧洲→印度→中国 | 1571年马尼拉大帆船贸易开始 |

### 2.2 日本白银流入量

日本白银流入中国的估算，各学者数据差异较大：

| 学者/来源 | 估算时段 | 流入量 | 说明 |
|---------|---------|--------|------|
| 多数学者 | 1540-1644 | 4,000-8,000万两 | 共识区间 |
| 按年均75吨计算 | 1540-1644 | 约6,075万两 | — |
| 整个明清时代 | 1540-约1800 | 约20,000万两 | 其中1/3直接贸易，2/3转口 |

- 石见银山1526年大规模开采，1533年引入灰吹法后飞跃发展
- 1600年前后鼎盛期，石见银山产量接近全球银产量的1/3

[来源：中国经济史论坛《明清时期白银流入量分析》](http://economy.guoxue.com/?p=4935)

### 2.3 美洲白银流入量

| 学者 | 估算时段 | 流入量 | 说明 |
|------|---------|--------|------|
| 多数学者 | 1571-1644 | 4,000-8,000万两 | 主流估算区间 |
| 万明 | 1571-1644 | 约2.04亿两 | 偏高估计 |
| 弗兰克（Gunder Frank） | 1550-1650 | 7,000-10,000吨（约1.9-2.7亿两） | 含日本和美洲合计 |

- 仅1631年一年，由菲律宾输入澳门的白银就达1,400万两
- 该数字约为永乐至宣德（1403-1434）30年官银矿总产量的2.1倍
- 约为万历年间国库岁入的3.8倍

[来源：中国经济史论坛《明朝中后期海外白银输入的三条主要渠道》](http://economy.guoxue.com/?p=7514)；[弗兰克《白银资本》](https://www.pbc.gov.cn/eportal/fileDir/yanjiuju/resource/cms/2020/12/2020120211342856173.pdf)

### 2.4 白银流入的全球背景

- 1550-1644年间，从中国专业市镇走出约236种出口产品
- 明朝每年输入白银约占全世界白银产量的60%-80%
- 出口商品主要为丝绸、瓷器、茶叶、棉布等
- 贸易顺差巨大，白银净流入

[来源：光明日报《明代货币白银化问题认识误区辨正》](https://news.gmw.cn/2018-08/27/content_30795654.htm)

### 2.5 崇祯末年白银流入的中断

- 1630年代后，日本实行锁国政策，限制白银出口
- 1639年马尼拉发生屠杀华人事件，太平洋贸易萎缩
- 海外白银流入锐减，加剧了国内通货紧缩
- 银价暴涨，铜钱急贬，崇祯末年1两银兑2,000-2,500文钱

## 三、银锭形制规格

### 3.1 官银

| 特征 | 说明 |
|------|------|
| **来源** | 赋税、课银上缴后由地方熔铸为银锭 |
| **铭文** | 刻有年份、省份、熔铸机构、银匠姓名等信息 |
| **重量** | 多为五十两或十两 |
| **成色** | 有严格的成色标准 |
| **形制** | 嘉靖以后：锭体增高，两端起翅（元宝形） |

### 3.2 民银（商银）

| 特征 | 说明 |
|------|------|
| **来源** | 商铺、银楼、炉户铸造 |
| **铭文** | 银楼名号、炉户名称，有时附吉语 |
| **重量** | 不统一，从数两到数十两不等 |
| **成色** | 参差不齐，需要鉴定 |

### 3.3 碎银

- 日常交易中广泛使用碎银
- 碎银来源：大锭剪切而成，或零星收入积累
- 使用时需称量——出门必备**戥子**（精密小秤）和**剪子**

[来源：百度百科《明清银锭》](https://baike.baidu.com/item/%E6%98%8E%E6%B8%85%E9%93%B6%E9%94%AD/5534755)；[首席收藏网《简析银锭形制的历史演变》](http://coin.shouxi.com/news/opinion/2023/0517/9913.html)

### 3.4 银锭形制的演变

- 明代前期（洪武、永乐、宣德）：沿用元代平板束腰器型，稍有起翅
- 约嘉靖时期起：锭体增高，两端明显起翅，形成标志性的"元宝"形
- 明代银锭长度较元代变短，厚度增加，束腰缩小，周缘增高

[来源：首席收藏网《简析银锭形制的历史演变》](http://coin.shouxi.com/news/opinion/2023/0517/9913.html)

## 四、流通方式：称量与火耗

### 4.1 称量货币的本质

明代白银不是铸币制，而是称量制：

- **无统一面值**：每次交易都需称重
- **成色不一**：不同来源的白银含银量不同，需鉴定
- **重量标准**：明代一两约37克（与唐42克、宋39克、元36.6克有别）
- **换算体系**：1两 = 10钱 = 100分

### 4.2 银两鉴定

- 银铺/钱庄提供银两成色鉴定服务（"倾银"业务）
- 鉴定方法：看色泽、听声音、称重量
- 成色表述：纹银（高纯度，约93.5%）、足银等

### 4.3 火耗

**火耗**是将碎银熔铸为标准银锭时的损耗，也是地方官府征收的附加费：

| 方面 | 说明 |
|------|------|
| **起源** | 嘉靖八年（1529）始，各地解交户部税银需熔铸为特定银锭 |
| **实际损耗** | 正常熔铸损耗约1%-3% |
| **加征比例** | 地方官以火耗为名额外加征，远超实际损耗 |
| **弊端** | 火耗成为地方官合法"创收"的渠道，百姓负担加重 |
| **后续** | 清雍正帝实行"火耗归公"改革，试图规范 |

[来源：知乎《明清白银有火耗的问题》](https://www.zhihu.com/question/4987054695)

## 五、购买力数据

### 5.1 米价变动

米价是衡量白银购买力的核心指标：

| 时期 | 米价（每石） | 说明 |
|------|------------|------|
| 洪武年间 | 约0.25两 | 物价低廉 |
| 正德年间 | 约0.5两 | 开始上涨 |
| 嘉靖年间 | 约0.6-0.8两 | 持续上涨 |
| 万历年间 | 约0.6-1.0两 | 平均约0.638两/石（江南） |
| 崇祯初年 | 约1.0-1.2两 | 进一步上涨 |
| 崇祯末年 | 1.6-1.8两甚至更高 | 战乱导致暴涨 |

> 注：1石约为72公斤（明代官斗容量计算）。

### 5.2 万历年间主要物价

| 商品 | 价格 | 说明 |
|------|------|------|

| 上等猪肉 | 0.02两/斤 | 二分银 |
| 上等羊肉 | 0.015两/斤 | 一分五厘 |
| 活肥鸡 | 0.04两/只 | 四分银 |
| 活鹅 | 0.2两/只 | 二钱银 |
| 上等白绵 | 0.88两/斤 | 万历九年 |
| 阔白布 | 0.28两/匹 | 万历九年 |

[来源：知乎《明朝时一两银子的购买力》](https://www.zhihu.com/question/30575324)；[澎湃新闻](https://www.thepaper.cn/newsDetail_forward_8311183)

### 5.3 白银购买力的现代折算

| 估算方法 | 1两银约等于人民币 | 依据 |
|---------|-------------|------|
| 以米价折算（万历） | 约700元 | 万历0.638两/石，现代米价约5元/斤 |
| 综合物价折算 | 约200-400元 | 考虑多种商品的平均 |

> 注：不同估算方法差异较大，主要因为不同商品的相对价格在四百年间变化巨大。

## 存疑 & 待查

- 嘉靖至万历年间的具体年份白银国内产量数据仍有争议，不同学者估算差距较大
- 万历矿税太监开矿实际产出多少白银，缺乏可靠数据
- 崇祯末年白银流入中断的精确时间节点和数量变化需要进一步研究
- 不同地区（京师/江南/华南）的银价差异和地域流通特征待细化
