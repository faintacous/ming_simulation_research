# 闽商与粤商

## 概述

闽商以福建漳州、泉州商人为核心，粤商以广州、潮州商人为主体，两者均以海外贸易为主要特色。嘉靖至崇祯间，闽商以月港为基地建立了覆盖东南亚、日本、马尼拉的海上贸易网络；粤商则以广州和澳门为窗口参与对外贸易。两者在经营路线、贸易对象和组织方式上既有重叠又有差异，共同构成了明代海洋贸易的主力。

## 关键事实

### 一、闽商

#### 1.1 月港海商群体

**月港的地位**
- 月港位于福建漳州府海澄县（今龙海市），是明代中后期最重要的民间对外贸易港口 [来源：《海澄县志》]
- 嘉靖间海禁松弛前，月港已是走私贸易的核心据点 [来源：明代海禁研究]
- 隆庆元年（1567年）部分开放海禁（"隆庆开海"）后，月港被指定为唯一合法的民间出海贸易港 [来源：《明实录》]
- 万历间月港贸易量达到鼎盛 [来源：海外贸易研究]

**月港的贸易规模**
- 隆庆开海后，朝廷在月港设"督饷馆"征收出海商税（饷税） [来源：《海澄县志》]
- 年发放出海许可证（"船引"）数量：
  - 隆庆初年：约50引 [来源：《明实录》]
  - 万历初年增至约100引 [来源：海关税收研究]
  - 万历中期：约110至137引 [来源：海关税收研究]
- 每船载货量从数十吨到数百吨不等 [来源：造船史研究]
- 月港年饷税收入：
  - 万历初年约银2万两 [来源：万历《漳州府志》]
  - 万历中期增至约2.9万两至3.5万两 [来源：海关税收研究]
  - 这仅是官方征收部分，实际贸易额远大于此
- 据估计，万历间月港年贸易总额可能达数十万两至上百万两白银 [来源：学界估算]

**船引与航线限制**
- 船引分为"东洋"和"西洋"两类 [来源：《海澄县志》]
  - 东洋引：前往日本、琉球方向（但官方禁止直接赴日贸易，实际上大量船只经由中转进行对日贸易）
  - 西洋引：前往东南亚（吕宋、暹罗、占城、爪哇、马六甲等）
- 每张船引规定了目的地、往返期限和船只大小 [来源：海禁法规研究]

#### 1.2 闽商的经营特点

**主要出口商品**
- 生丝和丝绸：最大宗的出口商品，从江浙产区采购运至月港出海 [来源：丝绸贸易研究]
- 瓷器：从景德镇和福建本地窑口（德化窑、漳州窑）采购 [来源：陶瓷贸易研究]
  - 漳州窑（"汕头器"）专为外销而生产，产品以青花瓷为主 [来源：考古研究]
  - 德化白瓷（"中国白"）在欧洲市场广受欢迎 [来源：外销瓷研究]
- 铁器/铁锅 [来源：东南亚贸易研究]
- 棉布 [来源：纺织品贸易研究]
- 糖（福建产糖区） [来源：制糖史研究]
- 纸张、书籍（福建建阳为刻书中心） [来源：出版史研究]
- 药材（中药材在东南亚有稳定需求） [来源：药材贸易研究]

**主要进口商品**
- 白银（最大宗进口品）：来自日本银矿和西班牙美洲殖民地 [来源：白银流通研究]
  - 日本银：通过直接或中转贸易获得
  - 西班牙银元（"佛银"）：通过马尼拉大帆船贸易获得
  - 据估计，隆庆至崇祯间（1567—1644），通过闽商渠道流入中国的白银达数千万两 [来源：全汉昇等学者估算]
- 胡椒和香料（东南亚产） [来源：香料贸易研究]
- 苏木（染料） [来源：东南亚贸易研究]
- 鹿皮（日本/台湾产，用于制甲） [来源：鹿皮贸易研究]
- 檀香木、象牙、犀角等奢侈品 [来源：奢侈品贸易研究]

**船型与航行**
- 主要船型：福船（"鸟船"的一种），载重从数十吨到400吨不等 [来源：造船史研究]
- 大型商船：长约30至40米，宽约8至10米，可载数百吨货物 [来源：造船史研究]
- 利用季风航行：
  - 冬季（东北季风）出航前往东南亚
  - 夏季（西南季风）返航
  - 单次往返约4至8个月 [来源：航海史研究]

#### 1.3 海外贸易网络

**马尼拉华侨社区（涧内/八连）**

- 西班牙于1571年占领马尼拉后，中国商人（以闽商为主）迅速涌入 [来源：马尼拉华侨史研究]
- 华侨社区称"涧内"（Parian），位于马尼拉城外的指定居住区 [来源：西班牙殖民档案]
- 人口规模：
  - 万历初（1580年代）：约一万至两万人 [来源：西班牙殖民档案]
  - 万历三十年前后：可能达两万至三万人 [来源：学界估算]
- 华侨从事行业：贸易（大宗商品买卖）、手工业（木匠、铁匠、裁缝）、零售、食品加工 [来源：华侨史研究]
- 马尼拉大帆船贸易（"太平洋丝绸之路"）：
  - 路线：马尼拉→太平洋→阿卡普尔科（墨西哥）
  - 中国丝绸和瓷器经此路线销往美洲和欧洲
  - 美洲白银经此路线回流至马尼拉，再由闽商带回中国
  - 每年1至2艘大帆船往返，每船载银约100万至300万比索 [来源：大帆船贸易研究]
- 排华事件：
  - 万历三十一年（1603）：西班牙殖民当局屠杀华侨，死亡约两万人 [来源：《明实录》、西班牙档案]
  - 此后华侨社区重建，但类似屠杀在后续年代多次发生 [来源：华侨史研究]

**日本贸易**

- 明朝官方禁止对日贸易（因倭寇问题），但闽商通过多种方式规避：
  - 经由琉球中转 [来源：琉球贸易研究]
  - 直接走私至日本九州（萨摩、平户、长崎等） [来源：中日贸易研究]
  - 嘉靖间海商/海盗首领如汪直（王直，歙县人但以福建为活动基地）、李旦等在日本建立贸易据点 [来源：倭寇研究]
- 对日主要出口：生丝、丝绸、书籍、药材、瓷器 [来源：中日贸易研究]
- 对日主要进口：白银（日本石见银山等银矿产银量极大）、铜、硫磺、刀剑 [来源：中日贸易研究]
- 据估计，万历间中日贸易年额可达白银数十万至上百万两 [来源：学界估算]

**东南亚网络**

- 暹罗（泰国）：大城（阿瑜陀耶）设有华人社区，闽商在此贸易 [来源：暹罗华侨史]
- 爪哇：万丹（Banten）、巴达维亚（后为荷兰据点）有华商活动 [来源：爪哇华侨史]
- 占城（越南中部）、柬埔寨：传统贸易伙伴 [来源：东南亚贸易研究]
- 马六甲：1511年被葡萄牙占领后，闽商仍在此活动 [来源：马六甲贸易研究]
- 文莱、苏禄：闽商有零星贸易 [来源：东南亚贸易研究]

#### 1.4 闽商代表人物

| 人物 | 时期 | 活动 | 备注 |
|------|------|------|------|

| 李旦 | 万历—天启间 | 日本/台湾贸易 | 泉州人，在日本平户建立贸易帝国，被称为"Captain China" |
| 颜思齐 | 天启间 | 台湾开发/海上贸易 | 海澄人，率众移居台湾北港 |
| 郑芝龙 | 天启—崇祯间 | 海上贸易/海上武装 | 南安人，李旦养子，建立海上霸权，控制东亚海域贸易 |
| 林凤 | 万历初 | 海盗/贸易 | 潮州人，率船队攻击马尼拉（1574年） |

**李旦海商集团数据**
- 1614—1625年间，李旦海商集团派遣商船达**23艘**，包括李旦本人船18艘、其弟华宇船5艘，自日本平户长崎往返东南亚各国 [来源：厦门大学研究]
- 李旦通常一船货物价值约**10万两白银** [来源：厦门大学研究]

**郑芝龙海商帝国数据**
- 鼎盛时期拥兵数万，船**700艘** [来源：维基百科]
- 荷兰人统计：当时中国人一年自中国输出丝织品及其他价值黄金**45吨**以上的货品，其中三分之二为郑芝龙及其盟党所有 [来源：郑芝龙研究]
- 以"十八芝"为核心，在吞并李旦、颜思齐、刘香等海商集团后发展壮大 [来源：维基百科]
- 学者指出"从16世纪到19世纪初的亚洲海域，确曾存在过一个虚拟的福建人的海上商业帝国" [来源：中国科学院大学人文学院]

### 二、粤商

#### 2.1 广州对外贸易

**广州的地位**
- 广州自唐代起即为中国最重要的对外贸易港口 [来源：广州外贸史]
- 明初设广州市舶司（与泉州、宁波并列三市舶司） [来源：《明史·食货志》]
- 嘉靖间（1523年"争贡之役"后），泉州、宁波市舶司相继停废，广州市舶司成为唯一保留的市舶司 [来源：《明实录》]
- 广州成为朝贡贸易中唯一的合法对外窗口（至隆庆开海前） [来源：朝贡贸易研究]

**广州贸易规模**
- 朝贡贸易：暹罗、占城、爪哇、马六甲等国的朝贡船只在广州入港 [来源：朝贡贸易研究]
- 附搭私货贸易：朝贡使团船只携带大量私货进行交易，这才是贸易实体 [来源：朝贡贸易研究]
- 民间走私贸易：海禁期间广州沿海走私从未断绝 [来源：海禁研究]
- 广州市舶司年抽分收入：万历间约银数千两至万余两（仅为官方部分） [来源：市舶司研究]

**广州的贸易对象**
- 东南亚诸国（暹罗、占城、爪哇、马六甲等） [来源：广州外贸史]
- 葡萄牙（1517年来华后逐步建立贸易关系） [来源：中葡关系史]
- 后期与荷兰、英国等也有间接接触 [来源：中欧贸易史]

#### 2.2 澳门贸易

**澳门的建立与发展**
- 嘉靖三十二年（1553）前后，葡萄牙人获准在澳门居住和贸易（具体过程有多种说法） [来源：澳门史研究]
- 澳门逐步发展为葡萄牙在东亚的贸易基地 [来源：澳门史研究]
- 万历间澳门常住葡萄牙人约800至1000人，另有大量华人居民和奴仆 [来源：葡萄牙档案]

**澳门贸易网络**
- 澳门—果阿（印度）航线：
  - 中国丝绸、瓷器、茶叶→果阿→欧洲
  - 印度棉布、胡椒、宝石→澳门→中国
  - 每年1至2艘大帆船（"黑船"），航程约6至8个月 [来源：葡萄牙海上贸易研究]
- 澳门—长崎航线：
  - 中国生丝→长崎→日本市场（此为利润最高的航线）
  - 日本白银→澳门→中国
  - 每年1至2艘"南蛮船"，单船载银可达数十万至上百万两 [来源：中日葡三角贸易研究]
  - 利润率极高：中国生丝在日本的售价约为采购价的2至3倍 [来源：生丝贸易研究]
- 澳门—马尼拉航线：
  - 与西班牙马尼拉的贸易 [来源：澳门贸易研究]

**粤商与澳门的关系**
- 粤商为澳门贸易的重要参与者：向葡商供货（丝绸、瓷器等）、采购洋货 [来源：澳门商业史]
- 广州—澳门之间的"广州期市"（每年定期开放的贸易集会） [来源：广州外贸史]
- 买办（"comprador"）制度的雏形：华商充当中间人，为葡商代理采购和销售 [来源：买办制度研究]

#### 2.3 粤商的经营特点

**主要商品**
- 出口：生丝、丝绸、瓷器、铁器、糖、药材（与闽商类似） [来源：广州外贸史]
- 进口：白银、胡椒、香料、檀香木、象牙、西洋钟表和火器（少量） [来源：广州外贸史]
- 特色：广东本地产品如铁器（佛山产）、糖（广东产糖区）也是重要出口品 [来源：广东手工业研究]

**佛山铁器**
- 佛山为明代南方最大的铁器制造中心 [来源：佛山方志]
- 产品包括：铁锅、农具、兵器、铁钉 [来源：佛山铁器研究]
- 万历间佛山铁器从业者数千人 [来源：佛山方志]
- 铁器大量出口东南亚 [来源：东南亚贸易研究]

**潮州商人**
- 潮州商人在粤商中占有特殊地位 [来源：潮商研究]
- 主要活动于东南亚（暹罗、柬埔寨、爪哇） [来源：潮州华侨史]
- 经营范围：大米、糖、海产品、瓷器 [来源：潮商研究]
- 在暹罗大城有较大的潮州华侨社区 [来源：暹罗华侨史]

### 三、闽商与粤商的对比

| 维度 | 闽商 | 粤商 |
|------|------|------|
| 核心港口 | 月港（漳州） | 广州/澳门 |
| 主要贸易方向 | 东洋（日本）和西洋（东南亚/马尼拉） | 南洋（东南亚）和西洋（印度/欧洲，经澳门） |
| 合法性 | 隆庆开海后月港为合法出海港 | 广州市舶司为官方窗口；走私同样普遍 |
| 商人来源 | 以漳州、泉州人为主 | 以广州、潮州人为主 |
| 海外华侨 | 马尼拉涧内社区为最大据点 | 暹罗、柬埔寨等地华侨社区 |
| 与欧洲人关系 | 与西班牙人在马尼拉互动 | 与葡萄牙人在澳门互动 |
| 船只规模 | 中小型福船为主，也有大型商船 | 广船（"广东帆船"）较福船更大 |
| 武装程度 | 较高（海商/海盗界限模糊） | 相对较低（依托官方体系较多） |
| 贸易组织 | 私人资本主导，宗族/同乡网络 | 私人资本与官方中间人（买办雏形）并存 |

## 时间线

- 洪武初：设广州、泉州、宁波三市舶司 [来源：《明史·食货志》]
- 嘉靖二年（1523）：争贡之役，此后泉州、宁波市舶司停废 [来源：《明实录》]
- 嘉靖三十二年（1553）前后：葡萄牙人获准居住澳门 [来源：澳门史研究]

- 万历三十一年（1603）：马尼拉排华事件，约两万华侨被杀 [来源：《明实录》]
