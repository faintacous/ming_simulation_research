# 瓷器业

## 概述

明代景德镇是世界制瓷业的中心。官窑（御器厂）代表了当时全球最高的制瓷水平，嘉靖一朝官窑烧造量近百万件；民窑约900座，从业人口数万人，是瓷器产量的主体。嘉靖至崇祯年间（1521-1644），景德镇瓷器品种和产量均达新高峰，"官搭民烧"制度促进了官民窑的互动。瓷器外销在新航路开辟后大幅增长，1602-1682年间仅荷兰东印度公司就贩运中国瓷器1,600万件以上。万历年间的"克拉克瓷"风靡欧洲，成为中国外销瓷的代名词。

## 二、官窑——御器厂

### 2.1 设立与沿革

| 时间 | 事件 |
|------|------|
| **洪武二年（1369）** | 在景德镇设立**御窑厂**（又称"御器厂"） |
| 明初 | 御窑29座，分烧不同品种 |
| 宣德以后 | 增至**58座** |
| 嘉靖-万历 | 产量最高峰，实行"官搭民烧" |

### 2.2 管理制度

| 要素 | 内容 |
|------|------|
| **督陶官** | 朝廷派遣，明代多以**宦官**（中官）充任 |
| **生产方式** | 明初实行匠役制，嘉靖后逐渐向雇佣制过渡 |
| **品质管理** | 极为严格，不合格品一律**打碎掩埋**（考古发掘证实） |
| **成本管理** | **不计成本**，代表最高水平 |

### 2.3 御器厂的组织结构

**二十三作**（生产车间）：大碗作、酒钟作、碟作、盘作等

**六种窑型共92座**：风火窑、色窑、大龙缸窑、匣窑等

### 2.4 御器厂编制与人员

| 时期 | 编制 | 说明 |
|------|------|------|
| 正嘉时期 | 官匠**334人**，其中成坯工**197人**（占59%） | — |
| 隆庆间 | 官匠**500人**，成坯工约**295人** | — |
| 隆庆间 | 工匠和役夫共**1,500人** | 占全部佣工的3%-12% |

[来源：中国经济史论坛《景德镇制瓷业中的资本主义萌芽》](http://economy.guoxue.com/?p=5419)

### 2.5 官窑年产量

| 时期 | 烧造数据 | 说明 |
|------|---------|------|
| 宣德八年（1433） | 一次烧造**443,500件** | 据《大明会典》 |
| 嘉靖八年（1529） | **2,570件** | 产量低谷 |
| 嘉靖二十三年（1544） | **50,000余件** | 恢复增长 |
| 嘉靖三十四年（1555） | **73,735件** | 继续增长 |
| 嘉靖一朝总计 | 有文字记载的近**60万件** | — |

### 2.6 万历大龙缸事件

- 万历十三年（1585）：烧造大龙缸"属难成"，阁臣请求停止
- 万历二十七年（1599）：太监**潘相**督造青花大龙缸，久未烧成
- 潘相对瓷工进行鞭笞甚至杀害，**激起景德镇窑工民变**
- 据传有窑工投身窑火以示抗议
- 反映了官窑制度对工匠的残酷压迫

## 三、民窑

### 3.1 规模

| 指标 | 数据 | 时期 |
|------|------|------|
| 窑口数量 | 约**900座** | 隆庆、万历年间 |
| 从业人数 | **数万人** | 万历年间 |
| 嘉靖十九年记载 | "民以陶为业，聚佣至**万余人**" | — |

- 民窑数量远超官窑，是瓷器产量的主体
- 民窑产品面向国内市场和海外出口

### 3.2 官窑与民窑比较

| 对比项 | 官窑 | 民窑 |
|--------|------|------|
| 品质 | 最高，不计成本 | 参差不齐，注重成本 |
| 品种 | 以御用器型为主 | 品种丰富，适应市场 |
| 纹饰 | 规范严格，龙凤等受限 | 较为自由，题材广泛 |
| 销售 | 全部供宫廷 | 国内市场 + 外销 |
| 款识 | 官窑年号款 | 多无款或堂号款 |

### 3.3 "官搭民烧"制度

- 嘉靖以后因官窑产能不足，实行**"官搭民烧"**
- 将部分御用瓷器烧造任务分配给民窑完成
- 促进了民窑技术水平的提高
- 但也加重了民窑的负担
- 嘉靖、万历时期成为重要的生产模式

## 四、主要瓷器品种

### 4.1 各时期代表品种

| 时期 | 代表品种 | 特点 |
|------|---------|------|
| 洪武 | 釉里红 | 铜红料发色，技术难度极高 |
| 永乐-宣德 | **青花** | 进口"苏麻离青"钴料，发色深沉浓艳，青花瓷巅峰 |
| 成化 | **斗彩** | 釉下青花 + 釉上彩，"鸡缸杯"最著名 |
| 弘治 | 黄釉 | 娇黄釉，色泽淡雅 |
| 正德 | 阿拉伯文瓷器 | 反映与伊斯兰文化交流 |
| **嘉靖-万历** | **五彩** | 釉上多色彩绘，红绿黄蓝黑多色并用，色彩对比强烈 |
| 万历 | 青花五彩 | 青花与五彩结合，极为华丽 |
| 天启-崇祯 | 转变期青花 | 风格自由奔放，为清代奠基 |

### 4.2 青花瓷

- 明代景德镇**最主要**的瓷器品种
- 钴料使用：永宣用进口"苏麻离青"，成化后多用国产"平等青""石子青"
- 万历时期青花瓷大量外销，成为欧洲贵族竞相收藏的珍品
- 嘉靖、万历两朝民窑青花存世量居明代各朝之首

### 4.3 五彩瓷

- 嘉靖至万历时期的标志性品种
- 万历朝五彩瓷制作达到明代顶峰，数量多、质量精
- 色彩强烈（红、绿、黄、蓝、黑等），视觉效果突出
- 在欧洲和日本大受欢迎

## 五、瓷器出口

### 5.1 外销渠道的演变

| 时期 | 主要渠道 | 特点 |
|------|---------|------|
| 明初-正德 | 朝贡贸易、私人海商 | 以东亚、东南亚为主 |
| 正德-嘉靖 | 葡萄牙人加入 | 欧洲市场开始打开，1553年澳门开埠 |
| 隆庆-万历 | 马尼拉贸易、荷兰东印度公司 | 大规模外销欧洲 |
| 天启-崇祯 | 荷兰VOC主导 | 克拉克瓷大量外销 |

### 5.2 外销数量数据

| 时间范围 | 外销数量 | 承运者 | 来源 |
|---------|---------|--------|------|
| 1602-1682（80年） | **1,600万件**以上 | 荷兰东印度公司 | T·沃尔克《瓷器与荷兰东印度公司》 |
| 万历三十八年（1610） | 一艘荷兰商船购入瓷器**1万余件** | 荷兰商船 | — |
| 万历四十二年（1614） | 一艘荷兰商船载运**7万余件**至阿姆斯特丹 | 荷兰商船 | — |
| 万历时期 | 占葡萄牙环球贸易景德镇瓷器的**69.86%** | 葡萄牙人 | 345件中241件 |

[来源：故宫博物院《中国瓷器的大量外销》](https://www.dpm.org.cn/Uploads/File/pdf/d8/01/e4/d801e45fc4538262c9aad5ae3041e366.pdf)

### 5.3 克拉克瓷

| 方面 | 说明 |
|------|------|
| **名称由来** | 1602年荷兰东印度公司截获葡萄牙"Santa Catharina"号商船，船上大量青花瓷在阿姆斯特丹拍卖引发轰动 |
| **特征** | 开光图案为主要装饰，以青花为主 |
| **产地** | 景德镇及福建若干窑场 |
| **年代** | 主要为万历至天启（1573-1627） |
| **影响** | 欧洲市场上最常见的明朝外销瓷，催生了荷兰代尔夫特蓝瓷等仿制品 |

### 5.4 主要出口目的地

| 区域 | 主要市场 | 主要品种 |
|------|---------|---------|
| **东南亚** | 越南、泰国、菲律宾、印尼 | 青花、五彩 |
| **日本** | 长崎 | 青花、五彩 |
| **南亚** | 印度 | 青花 |
| **中东** | 波斯、阿拉伯 | 青花（伊斯兰文化偏好） |
| **欧洲** | 荷兰、葡萄牙、西班牙、英国 | 克拉克瓷、五彩 |
| **美洲** | 墨西哥（经马尼拉转运） | 青花 |
| **东非** | 肯尼亚等地 | 青花 |

### 5.5 出口瓷器价格

| 品种 | 价格 | 说明 |
|------|------|------|
| 弘治年间铁锅折还物价中的参考 | 三尺广锅每口150贯 | 以宝钞计价 |
| 万历青花瓷在欧洲 | 极高，贵族竞相收藏 | 无统一价格 |
| 日本市场铁锅 | 每口约银1两 | 可作参照 |

> 瓷器在产地价格较低，经海路运至欧洲后价格可涨数倍至数十倍。

## 六、景德镇窑工的生存状况

### 6.1 窑工人数

| 时期 | 人数 | 来源 |
|------|------|------|
| 嘉靖十九年（1540） | "聚佣至**万余人**" | 文献记载 |
| 万历年间 | "不下**数万人**" | 文献记载 |
| 御器厂（隆庆间） | 工匠和役夫**1,500人** | — |

### 6.2 窑工的分工

景德镇制瓷业的社会分工极为精细，据清代记载（可参考明末状况）：

- 制瓷从原料开采到成品出窑需经过**72道工序**
- "共计一坯之力，过手七十二，方克成器"
- 各工序均有专门的工匠

### 6.3 万历窑工民变

- 太监潘相督造，对工匠鞭笞杀害
- 窑工反抗，据传有人投身窑火
- 反映了官窑制度的残酷和工匠的抗争
