# 铜钱与宝钞

## 概述

明代货币体系经历了从铜钱到宝钞到铜钱白银并行再到白银为主的演变过程。铜钱和宝钞作为明初的法定货币，在白银货币化浪潮中逐渐边缘化。宝钞因无准备金、无节制超发，百年间贬值千倍以上，正统年间实质退出流通，隆庆初年正式废止。铜钱虽始终存在，但明代铸量远逊宋代，仅约400-600万贯，嘉靖至崇祯间私铸泛滥、钱法混乱。

---

## 一、铜钱铸造体系

### 1.1 铸造机构

| 机构 | 隶属 | 职能 | 说明 |
|------|------|------|------|
| **宝源局**（京师） | 工部 | 中央主要铸币机构 | 南京宝源局铸钱纵贯明朝始终 |
| **各地宝泉局**（省局） | 各省布政司 | 地方铸钱 | 洪武年间设立 |

- 明代铸钱机构数量和规模远不及宋代
- 铸钱总量也远逊于宋代（宋代年铸数百万贯，明代总计约400-600万贯）

[来源：首席收藏网《明朝钱币史》](http://coin.shouxi.com/news/opinion/2016/0711/2530.html)

### 1.2 铸币能力

- 洪武二十六年（1393）定则例：除南京外，全国各地炉座一年共可铸钱**189,414贯零800文**
- 实际执行中并非年年铸造，时常停铸
- 明前期推行宝钞，一度禁止铜钱流通

### 1.3 铸钱总量

从洪武元年（1368）到隆庆六年（1572），明朝铸币机构累计铸造铜钱约**400万-600万贯**。

[来源：中国集币在线《明朝铜钱各时期多寡悬殊》](http://www.jibi.net/News/gbzs/133156140.html)

## 二、各朝年号钱

### 2.1 铸钱概况

明代从太祖洪武至思宗崇祯，历十六帝（含南明以前），年号钱铸造情况如下：

| 年号 | 在位年份 | 铸钱情况 | 备注 |
|------|---------|---------|------|
| **洪武通宝** | 1368-1398 | 大量铸造 | 有小平、折二、折三、折五、折十五等 |
| 建文 | 1399-1402 | 未铸或极少 | 存世极罕 |
| **永乐通宝** | 1403-1424 | 大量铸造 | 郑和下西洋大量携带，用于海外贸易 |
| 洪熙 | 1425 | 未铸 | 在位仅10月 |
| **宣德通宝** | 1426-1435 | 有铸造 | 量不大 |
| 正统-天顺 | 1436-1464 | **未铸** | 推行宝钞，禁用铜钱 |
| 成化 | 1465-1487 | **未铸** | 同上 |
| **弘治通宝** | 1488-1505 | 弘治十六年（1503）开铸 | 量少 |
| 正德 | 1506-1521 | **未铸** | 正德通宝多为后世伪作 |
| **嘉靖通宝** | 1522-1566 | 铸量较多 | 质量较好，制式规范 |
| **隆庆通宝** | 1567-1572 | 有铸造 | 量少 |
| **万历通宝** | 1573-1620 | 大量铸造 | 万历年间增加铸钱炉 |
| 泰昌通宝 | 1620 | 少量 | 在位仅1月，钱为光宗后铸 |
| **天启通宝** | 1621-1627 | 大量铸造 | 但私铸泛滥，质量低劣 |
| **崇祯通宝** | 1628-1644 | 大量铸造 | 版别极多（数百种），品质参差 |

### 2.2 特殊时期

#### 正统至成化（1436-1487）：约50年未铸铜钱

- 原因：朝廷大力推行宝钞，禁止民间使用铜钱
- 正统年间更明令禁止铜钱交易
- 但民间仍暗中使用铜钱和白银

#### 万历至崇祯：铸钱量大增但质量堪忧

- 万历年间大量增设官铸炉
- 铜价上涨导致官铸无利可图
- 私铸横行，劣钱充斥市场
- 崇祯朝铸钱版别多达**数百种**，反映了混乱的货币状况

[来源：首席收藏网《明朝钱币史》](http://coin.shouxi.com/news/opinion/2016/0711/2530.html)；[中国政府网《明朝錢幣的鑄行》](http://big5.www.gov.cn/gate/big5/www.gov.cn/test/2006-05/10/content_277336.htm)

## 三、银钱比价变化数据

### 3.1 官定与市场比价

| 时期 | 官定/市场比价（1两银兑铜钱） | 说明 |
|------|-------------------------|------|
| 洪武初年 | 1两 = 1,000文 | 官定标准 |
| 成化年间 | 1两约800文 | 银价渐贵 |
| 弘治年间 | 1两约700文 | |
| 隆庆元年（1567） | 1两 = 800文（制钱） | 官方规定 |
| 万历六年（1578） | 嘉靖/隆庆/万历制钱8文=银1分；旧钱12文=银1分 | 区分新旧钱 |
| 万历年间 | 1两约400-600文（金背钱） | 市场波动大 |
| 天启年间 | 1两约500文 | |
| 崇祯初年 | 1两约800-1,000文 | |
| 崇祯末年（1643-1644） | 1两 = **2,000-2,500文** | 钱法大坏，铜钱急贬 |

### 3.2 比价变动的原因

1. **白银供给变动**：海外白银流入增加时银价下降（中期）；崇祯末年流入锐减时银价暴涨
2. **铜钱质量恶化**：私铸泛滥，含铜量下降
3. **战争与财政危机**：崇祯朝军费激增，大量铸造劣质铜钱
4. **信用崩溃**：政府货币信用不断损害，市场拒收劣钱

### 3.3 私钱问题

| 时期 | 状况 |
|------|------|
| 万历年间 | 增设官铸炉，但铜价上涨使官铸无利 |
| 天启年间 | 私铸开始泛滥，劣钱名称繁多 |
| 崇祯年间 | 私钱问题极为严重，版别数百种 |

[来源：澎湃新闻《明清时期的货币、民生与国用》](https://www.thepaper.cn/newsDetail_forward_20248132)

## 四、大明宝钞

### 4.1 基本信息

| 项目 | 说明 |
|------|------|
| **始造时间** | 洪武八年（1375）制造，次年以中书省名义发行 |
| **材质** | 桑皮纸，灰青色 |
| **尺寸** | 一贯面额：高约34厘米、宽约22厘米 |
| **特点** | 中国历史上面积最大的纸币 |

### 4.2 十一种面额

| 面额 | 折合铜钱 | 折合白银 |
|------|---------|---------|
| **一贯** | 1,000文 | 1两 |
| 五百文 | 500文 | 5钱 |
| 四百文 | 400文 | 4钱 |
| 三百文 | 300文 | 3钱 |
| 二百文 | 200文 | 2钱 |
| 一百文 | 100文 | 1钱 |
| 五十文 | 50文 | 5分 |
| 四十文 | 40文 | 4分 |
| 三十文 | 30文 | 3分 |
| 二十文 | 20文 | 2分 |
| 十文 | 10文 | 1分 |

另有折合黄金的比价：宝钞**四贯 = 黄金一两**。

[来源：首席收藏网《解析大明通行宝钞》](http://coin.shouxi.com/news/opinion/2024/0205/11026.html)

### 4.3 发行制度的致命缺陷

| 制度缺陷 | 具体表现 |
|---------|---------|
| **无准备金** | 没有贵金属或实物作为准备金支撑 |
| **无发行上限** | 朝廷随意增发以弥补财政支出 |
| **回笼机制薄弱** | 洪武九年设"行用库"收换旧钞，但新钞发出远多于回收 |
| **强制流通** | 多次禁止金银交易，强制使用宝钞，违者治罪 |
| **无兑换承诺** | 不承诺以金银兑换宝钞（与宋代交子不同） |

### 4.4 宝钞贬值历程（关键数据）

| 时间 | 贬值程度 | 具体表现 |
|------|---------|---------|
| 洪武八年（1375） | **基准** | 1贯钞 = 1两银 = 米1石 |
| 洪武十八年（1385） | 开始贬值 | 贬值初显 |
| 洪武二十三年（1390） | 贬至**1/4** | |
| 洪武二十七年（1394） | 贬至**1/6** | |
| 永乐二年（1404） | 贬至**1/30** | 米1石 = 宝钞30贯 |
| 宣德七年（1432） | 贬至**1/100** | |
| 正统元年（1436） | 贬至**约1/1,000** | 朝廷弛用银之禁 |
| 正统九年（1444） | 继续贬值 | 米1石 = 宝钞100贯 |
| 正统十三年（1448） | 贬至**约1/500** | 1贯钞折铜钱仅2文 |
| 成化年间（1465-1487） | 贬至**1/1,000至1/数千** | 1贯纸钞仅值1文钱 |

> 从洪武八年到成化年间（约100年），宝钞贬值超过**1,000倍**。

[来源：首席收藏网《探大明宝钞失败之谜》](http://coin.shouxi.com/news/opinion/2018/0822/4729.html)；[AllHistory《大明宝钞：如何百年内贬值千倍》](https://m2.allhistory.com/ah/article/5f586ba8231c390001a07449)

### 4.5 退出流通的时间线

| 时间 | 事件 |
|------|------|
| 正统年间（1436-1449） | "积之市肆，过者不顾"，实质退出流通 |
| 正统元年（1436） | 弛用银之禁，允许白银缴纳赋税——宝钞政策实际失败的标志 |
| 成化至弘治 | 仅在某些官方场合象征性使用 |
| 嘉靖初年 | 有短暂恢复宝钞流通的讨论，未能实施 |
| 隆庆初年（约1567） | 大明宝钞**正式废止** |

此后明朝再未发行纸币，直至清末才重新出现官方纸币。

### 4.6 宝钞失败的原因分析

| 原因 | 详细说明 |
|------|---------|
| **缺乏准备金** | 无金银储备支撑纸币信用，与现代法币不同，也与宋代交子有别 |
| **超发无度** | 朝廷将印钞视为免费财政收入，发行量远超经济需要 |
| **回笼机制缺失** | 无有效的旧钞回收和销毁机制，单向投放 |
| **强制流通适得其反** | 禁银令无法执行，反而损害政府信用 |
| **无兑换承诺** | 持有人无法兑换金银，一旦信心动摇即无底线贬值 |
| **通货膨胀螺旋** | 贬值导致百姓拒收，进而更大贬值，恶性循环 |

## 五、嘉靖至崇祯时期的铜钱状况

### 5.1 嘉靖通宝

- 嘉靖六年（1527）开铸
- 质量较好，制式规范
- 是正统至正德约50年停铸后的恢复
- 铸有**金背**（经过打磨抛光处理）品种

### 5.2 万历通宝

- 大量铸造，增加铸钱炉
- 市场流通中新旧钱并存，官方规定不同折价
- 制钱8文 = 银1分（嘉靖/隆庆/万历制钱）
- 旧钱（洪武等）12文 = 银1分

### 5.3 天启通宝

- 天启年间铸量大但私铸泛滥
- 官铸与私铸难以区分
- 含铜量参差不齐
- 开始出现严重的劣币驱逐良币现象

### 5.4 崇祯通宝

- 崇祯年间铸钱量最大
- 版别多达数百种——全国各地竞相铸造
- 品质极不统一
- 军费紧张迫使降低铸钱成本
- 铜价高企，私铸利润丰厚，屡禁不止
