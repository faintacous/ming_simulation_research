# 官员铨选

## 概述

明朝官员铨选制度是将科举功名转化为实际官职的核心环节，由吏部主管。进士依据殿试甲次（一甲、二甲、三甲）获得不同的初授官职，其中最优者入翰林院，其余分授京官或外官。未中进士的举人、贡生另有授官通道，但起点更低、升迁更难。铨选制度在嘉靖至崇祯年间基本稳定运行，但晚明候缺壅塞、冗官日增等问题日益严重。

## 一、进士初授官规则

### 1.1 殿试三甲与授官对应关系

| 甲次 | 称号 | 名额 | 初授官职 | 品级 |
|------|------|------|----------|------|
| 一甲第一名 | 状元，赐进士及第 | 1人 | 翰林院修撰 | 从六品 |
| 一甲第二名 | 榜眼，赐进士及第 | 1人 | 翰林院编修 | 正七品 |
| 一甲第三名 | 探花，赐进士及第 | 1人 | 翰林院编修 | 正七品 |
| 二甲 | 赐进士出身 | 约50–150人 | 庶吉士（馆选入选者）、六部主事（正六品）、中书舍人，或外授知州（从五品） | 正六品–从五品 |
| 三甲 | 赐同进士出身 | 约200–300人 | 庶吉士（少数）、评事、中书、行人（正七品–从七品），或外授推官、知县（正七品） | 正七品为主 |

**嘉靖至崇祯各科录取人数统计：**

| 时期 | 科数 | 一甲总人数 | 二甲总人数 | 三甲总人数 | 合计 | 平均每科 |
|------|------|-----------|-----------|-----------|------|---------|
| 嘉靖（1521–1566） | 15科 | 45 | 1,407 | 3,469 | 4,921 | 约328人 |
| 隆庆（1567–1572） | 2科 | 6 | 约190 | 约510 | 约706 | 约353人 |
| 万历（1573–1620） | 16科 | 48 | 985 | 4,050 | 5,083 | 约318人 |
| 天启（1621–1627） | 2科 | 6 | — | — | 约700 | 约350人 |
| 崇祯（1628–1644） | 5科 | 15 | — | — | 约1,600 | 约320人 |

典型单科数据举例：
- 嘉靖二年癸未科：一甲3名，二甲142名，三甲265名，共410名
- 嘉靖五年丙戌科：一甲3名，二甲90名，三甲208名，共301名
- 万历二年甲戌科：一甲3名，二甲70名，三甲226名，共299名
- 万历五年丁丑科：一甲3名，二甲57名，三甲241名，共301名

**注意：** 二甲与三甲的比例在不同年份差异很大，但三甲人数始终远多于二甲。万历年间二甲人数明显少于嘉靖，三甲人数则更多。

[来源：维基百科"明朝进士列表"、百度百科"庶吉士"条目；知乎"通往权力之路：明清科举进士的上岸史"；故宫博物院"翰林院"词条]

**关键规则：**
- 一甲三人直接授翰林院官职，无需经过观政或馆选
- 修撰一职专为状元设置，他人不能升任此职
- 二甲第一名俗称"传胪"，有时亦可直接入翰林院
- 二、三甲进士须先经"观政"实习，再正式授官

[来源：新浪"大名鼎鼎的庶吉士"；故宫博物院"翰林院"词条]

### 1.2 观政进士制度

观政进士是明代特有的授官前实习制度，由洪武十八年（1385）正式确立。

| 项目 | 内容 |
|------|------|
| 适用对象 | 二甲、三甲进士中未入选庶吉士者 |
| 实习机构 | 六部、都察院、大理寺、通政司等中央官署；永乐后扩至五军都督府 |
| 实习期限 | 三至六个月，最长一年 |
| 实习待遇 | 参照八品官俸禄 |
| 期满程序 | 由实习部门考核，合格后由吏部铨选授官 |
| 制度演变 | 宣德七年（1432）起允许刑部观政进士参与案件审理；万历后逐渐流于形式 |

[来源：百度百科"观政进士"条目；中国反腐倡廉发展史编纂委员会"明朝授官程序的特点"]

## 二、庶吉士制度

### 2.1 制度沿革

| 时间 | 事件 |
|------|------|
| 洪武十八年（1385） | 朱元璋首次设置庶吉士 |
| 永乐二年（1404） | 庶吉士改为专隶翰林院管理和培养，就学于文渊阁 |
| 弘治四年（1491） | 馆选的时间、程序、主持衙门正式"立为定例" |
| 此后 | 成为定制，每科殿试后进行馆选 |

[来源：维基百科"庶吉士"条目；知乎"明清庶吉士制度"]

### 2.2 馆选标准

| 标准项 | 具体要求 |
|--------|----------|
| 文章水平 | 提交平日所作文字（论、策、诗、赋、序、记等），限十五篇以内 |
| 年龄 | 弘治四年后要求"年二十五岁以下，果有过人资质"；年龄过大者原则上不入选 |
| 书法（楷法） | 最过硬的标准，科举时代特色，楷书水平是关键筛选条件 |
| 仪表风度 | "观其仪度"，要求仪态端正 |
| 主持部门 | 弘治四年后由内阁会同吏部、礼部共同考选 |

**破例案例：** 马一龙登科时已49岁，触及年龄红线，朝廷破例准予其"减年十岁"入馆。

[来源：知乎"明清庶吉士制度"；新浪"大名鼎鼎的庶吉士"]

### 2.3 在馆学习

| 项目 | 内容 |
|------|------|
| 学习地点 | 文渊阁（翰林院附设庶常馆） |
| 督课者 | 由阁臣（内阁大学士）亲自督课 |
| 学习期限 | 三年 |
| 学习内容 | 经史、诗文、书法、时务策论等 |
| 实质 | 高级官员储备培训，兼有政务见习性质 |

[来源：百度百科"庶吉士"条目]

### 2.4 散馆分流规则

三年期满进行"散馆"考试，按成绩分为三等：

| 等次 | 分流去向 | 具体官职 |
|------|----------|----------|
| 优等（留馆） | 留翰林院任职 | 二甲出身者授编修（正七品），三甲出身者授检讨（从七品） |
| 中等 | 授京官 | 分授给事中、御史等科道官 |
| 下等（不合格） | 外放 | 出任知县等地方官 |

[来源：百度百科"庶吉士"条目；知乎"明清庶吉士制度"；故宫博物院"翰林院"词条]

**散馆考试严格程度：** 清康熙四十二年的案例可作参照——散馆时仅杨尤其等十四人授职编检，而涂天相等二十人因"文义荒疏"被罚令再学习三年。明代亦有类似严格执行的记录。

## 三、举人授官

### 3.1 举人入仕的三条途径

| 途径 | 说明 |
|------|------|
| 考中进士 | 最佳出路，走进士授官路线 |
| 直接授教职 | 分授府、州、县学教授、学正、教谕等 |
| 入国子监候选 | 进入国子监学习后待选，可授小京官或外官 |

[来源：百度百科"举人"条目；知乎"明代举人的出路和去向"]

### 3.2 举人直接授官的具体规则

| 项目 | 内容 |
|------|------|
| 授官上限 | 最高可选至知县（正七品），但实际机会较少 |
| 常见授职 | 府州县学教职、州县佐贰官（如县丞、主簿） |
| 候选方式 | 乡试三考不中者可赴吏部选官 |
| 考试要求 | 举人选官需经吏部考试，内容为策论和判语各一篇 |
| 升迁限制 | 举人出身官员升迁天花板低于进士，通常难以升至省级主官 |
| **嘉靖八年变化** | 此后举人铨选只选学正、教谕，与副榜举人一律三月一选 |

**教职升迁路径：**
- 举人出身教官任职**三年以上**：学行优者 → 国子监监丞、助教
- 有从政才能者 → 推官、知县
- 弘治后补充规定：举人出身教官须历任**六年以上**、才行出众方可出任科道

[来源：知乎"明代举人的出路和去向"；百度百科"举人"条目；国家人文历史"明代知县升迁"]

### 3.3 大挑制度

大挑制度是举人入仕的补充途径，主要在清代制度化（每六年举行一次），明代已有类似实践但尚未完全制度化。连续三次以上落榜的举人中挑选，被挑者最高授知县，大部分仍担任地方学校教官。等待时间漫长，竞争激烈。

[来源：百度百科"举人"条目；知乎"清代举人做官的大挑制度"]

## 四、贡生授官

### 4.1 贡生的类型（明代）

| 类型 | 选拔方式 | 说明 |
|------|----------|------|
| 岁贡 | 府州县学每年选拔食廪最久者 | 最常见的贡生类型 |
| 选贡 | 由提学官考选优秀生员 | 质量较岁贡高 |
| 恩贡 | 遇皇帝登极等庆典特选 | 额外名额 |
| 纳贡（例贡） | 出资捐纳入监 | 明中后期渐多 |

### 4.2 贡生授官规则

| 项目 | 内容 |
|------|------|
| 入监学习 | 贡生先入国子监（太学）肄业 |
| 授官范围 | 可授府佐（同知、通判）、州县正官、教职 |
| 考试要求 | 需经吏部考试（策论、判语），按成绩授官 |
| 授官上限 | 一般不超过知县，多为教谕、训导等教职或县丞、主簿等佐贰官 |
| 升迁前景 | 较举人更低，通常限于中低级官职 |

[来源：知乎"科举制度中什么是贡生、举人、贡士、进士"；百度百科"举人"条目]

## 五、铨选的主管机构与流程

### 5.1 吏部铨选

| 项目 | 内容 |
|------|------|
| 主管部门 | 吏部文选清吏司 |
| 铨选方式 | 京官五品以下、外官五品以下用"类推"之法——各类官缺一并开出，每缺推一人 |
| 高级官员 | 四品以上京官、三品以上外官由廷推或会推产生 |
| 特旨授官 | 皇帝直接以圣旨升授官员，明代称"传奉"，属于绕过吏部的非常规渠道 |

[来源：故宫博物院赵晶论文"传奉官研究"]

### 5.2 候缺与候补

| 项目 | 内容 |
|------|------|
| 产生原因 | 官缺有限，新进官员（进士、举人、贡生）和丁忧起复官员均需候缺 |
| 等待时间 | 短则数月，长则数年；晚明壅塞严重 |
| 候补困境 | 京城居住开销大，俸年不计入候补期，长期等待经济压力巨大 |
| 万历改革 | 万历二十六年（1598），因吏部司官"营干滥入、作弊多端"，神宗谕令各部院掌印官择举堪任之人 |

[来源：爱思想"明中后期吏部司官分省与官僚政治"；知乎"丁忧制度"]

## 六、铨选的主管机构与核心规则

### 吏部铨选

| 项目 | 内容 |
|------|------|
| 主管部门 | 吏部文选清吏司 |
| 铨选方式 | 京官五品以下、外官五品以下用"类推"之法——各类官缺一并开出，每缺推一人 |
| 高级官员 | 四品以上京官、三品以上外官由廷推或会推产生 |
| 铨选四要素 | **资**（出身资格）、**年**（年资）、**俸**（俸禄等级）、**次**（候选顺序） |
| 特旨授官 | 皇帝直接以圣旨升授官员，明代称"传奉"，属绕过吏部的非常规渠道 |

**铨选四要素说明：**
- "资"（出身）是最重要因素，进士出身在铨选中享有绝对优先权
- "年"决定了候选的先后，入仕越久、排名越前
- "俸"反映现任品级，影响升迁的起点
- "次"是候选名册中的具体顺序

### 候缺与候补

| 项目 | 内容 |
|------|------|
| 产生原因 | 官缺有限，新进官员和丁忧起复官员均需候缺 |
| 等待时间 | 短则数月，长则数年；晚明壅塞严重 |
| 候补困境 | 京城居住开销大，俸年不计入候补期，经济压力极大 |
| 候缺优先序 | 进士优先于举人和贡生 |

[来源：故宫博物院赵晶论文"传奉官研究"、国家人文历史"明代知县升迁"]

## 七、捐纳制度

### 6.1 制度起源与演变

| 时间 | 事件 | 说明 |
|------|------|------|
| 正统十四年（1449） | 土木堡之变后军费急需 | 首开捐纳先例 |
| 景泰元年（1450） | "凡生员纳粟上马者，许入监，限千人而止" | 首次大规模开放 |
| 景泰四年（1453） | 向灾区捐800担米的生员可成贡生 | 为赈济灾荒而扩大范围 |
| 成化年间 | 正式定规：生员纳米百石以上入国子监 | 制度化 |
| 正德年间 | 非儒生身份者也可获"例监生"身份 | 门槛进一步降低 |
| 嘉靖以后 | 捐纳成为常规化的财政手段 | 规模扩大 |

[来源：维基百科"捐纳"、维基百科"官职买卖"、百度百科"监生"]

### 6.2 捐纳标准（成化年间定规）

| 身份 | 捐纳数额 | 所获身份/官品 |
|------|---------|-------------|
| 生员 | 纳米百石以上 | 入国子监，称"例监" |
| 军民 | 纳米二百五十石 | 正九品散官 |
| 军民 | 每加五十石 | 增二级 |
| 军民 | 最高可至 | 正七品止 |

**捐纳身份类别：**
- **例监**：捐纳钱粟获监生身份，可入国子监
- **例贡**：秀才通过捐纳获贡生身份

### 6.3 制度影响

**积极面：** 为朝廷提供重要财政补充，在边疆军需紧迫时发挥关键作用

**消极面：**
- 冲击科举取士的公平性
- 国子监生员质量下降，"例监"占比逐渐超过正途监生
- 催生大量候补官员，加剧官缺紧张
- 间接助长官场腐败

[来源：维基百科"捐纳"、华东师范大学"明代捐纳与官学教育的衰败"]
