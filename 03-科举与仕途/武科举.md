# 武科举

## 概述

明代武举制度起步较晚，远逊于文科举的完善程度。明初实行世袭武官制和卫所屯兵制，对武举需求不大。直到天顺八年（1464年）才颁布《武举法》，嘉靖元年（1522年）议准《武举条格》后武举才成为定制。武举分乡试、会试两级（崇祯四年增设殿试），考试内容强调"先之以谋略，次之以武艺"，文武并重。

---

## 一、武举的设立与沿革

### 1.1 时间线

| 时间 | 事件 |
|------|------|
| **天顺八年（1464年）** | 明宪宗即位，颁布《武举法》，首次制度化武举选拔 |
| **成化六年（1470年）** | 首次举行武举会试，取中**刘良**等6名武进士。刘良为明朝**第一位武状元** |
| 成化十四年（1478年） | 武举正式纳入三年一科的定期考试 |
| 弘治六年（1493年） | 进一步完善武举考试内容 |
| 弘治十七年（1504年） | 定武举会试录取名额为**35名** |
| **嘉靖元年（1522年）** | 议准《武举条格》，武举遂成定制，分乡试、会试 |
| 嘉靖至万历 | 武举规模逐步扩大 |
| **崇祯四年（1631年）** | 增设武举**殿试**，武举始有三级完整体系 |
| 崇祯十年（1637年） | 武举会试录取人数增至约**200名** |

## 二、考试结构与内容

### 2.1 考试原则

明代武举考试遵循**"先之以谋略，次之以武艺"**的原则：
- 如考生在**策论笔试**中不合格，直接淘汰，不得参加武艺考试
- 这与唐宋武举重武艺轻文化形成鲜明对比

### 2.2 武乡试（三场）

| 场次 | 考试内容 | 具体要求 |
|------|----------|----------|
| **第一场** | 马上骑射 | 以**三十五步**为射程标准 |
| **第二场** | 步下射箭 | 以**八十步**为射程标准 |
| **第三场** | 策论笔试 | 考策论**一道**，内容涉及兵法和时局 |

### 2.3 武会试（三场）

| 场次 | 考试内容 | 具体要求 |
|------|----------|----------|
| **第一场** | 马上骑射 | 标准同乡试 |
| **第二场** | 步下射箭 | 标准同乡试 |
| **第三场** | 策论笔试 | 考策**二道**、论**一道** |

### 2.4 武殿试（崇祯四年后）
- 崇祯四年（1631年）始设
- 由皇帝亲自主持
- 考试内容：骑射及策问
- 设一甲三名：武状元、武榜眼、武探花

## 三、骑射步射的具体标准

### 3.1 天顺八年《武举法》的初始标准

| 等级 | 策论 | 骑射（马上） | 步射（步下） | 待遇 |
|------|------|-------------|-------------|------|
| **上等** | 答策二道合格 | 中**四箭以上**（九箭中） | 中**二箭以上**（九箭中） | 官加署职二级；旗军舍余授试所镇抚；民人授卫经历，月支米三石 |
| **次等** | 答策二道合格 | 中**二箭以上** | 中**一箭以上** | 官加署职一级；旗军舍余授冠带总旗；民人授试卫知事，月支米二石 |

### 3.2 嘉靖以后的标准演变

嘉靖《武举条格》对骑射步射标准进行了调整：

| 项目 | 具体要求 |
|------|----------|
| 骑射 | 九箭射中**三箭以上**为合格 |
| 步射 | 九箭射中**五箭以上**为合格 |
| 弓力 | 有具体拉力要求（按"石"计算），一般要求**一石以上** |

> 明代一石弓约相当于现代的**50-60公斤**拉力（约110-130磅）。能拉一石弓已属不易，拉二石弓者极为罕见。

### 3.3 策论考题示例

武举策论从**武经七书**等军事典籍中出题，也涉及时事。万历年间的策论题目示例：
- 山东乡试："明主内修外治"
- 湖广乡试："王者必有股肱羽翼以成威神"

> 文官阅卷时"尤喜能附会儒家思想的题目"，反映了明代武举中文化素养的重要性。

## 四、录取标准和人数

### 4.1 录取人数的历史变化

#### 武举会试

| 时期 | 录取人数 | 说明 |
|------|---------|------|
| 成化六年（1470年）首科 | **6人** | 刘良等6名武进士 |
| 弘治十七年（1504年） | **35人** | 定为常制 |
| 嘉靖年间 | 约50-100人 | 逐步增加 |
| 万历年间 | 约100-150人 | 继续增长 |
| 崇祯十年（1637年） | 约**200人** | 明末最多 |

#### 武举乡试

| 时期 | 全国总录取人数 |
|------|-------------|
| 正德十三年（1518年）前后 | 约**970人** |
| 明末（崇祯年间） | 近**2,000人** |

### 4.2 考生构成变化

| 时期 | 考生来源 |
|------|----------|
| 明初至嘉靖前期 | 以**世职武官子弟**为主（世袭军官家庭） |
| 嘉靖后期至崇祯 | **军民各阶层**参与比例上升，"世职、庆袭舍人以外，旗舍军民等有43人"比例显著提高 |

## 五、武进士的初始任命

### 5.1 天顺至弘治时期

武举中式者根据原有身份获得不同级别的任命：

| 原有身份 | 授职 |
|----------|------|
| 现任武官 | 在本职基础上**加署职二级**（上等）或一级（次等） |
| 旗军舍余（军户子弟） | 授**试所镇抚**（上等）或冠带总旗（次等） |
| 民人（平民出身） | 授**卫经历**（上等）或试卫知事（次等） |

### 5.2 嘉靖以后

| 任命方式 | 说明 |
|----------|------|
| 署职制度 | 署副千户以实授百户为基础 |
| 不支俸、不世袭 | 署职仅为荣衔，不领实际俸禄 |

### 5.3 崇祯四年后（增设殿试）

| 甲次 | 初授官职 |
|------|----------|
| 武状元 | — |
| 武榜眼 | — |
| 武探花 | — |
| 其他武进士 | 根据殿试名次授职 |

> 崇祯四年后，武进士获得实授官职的待遇大幅提升，反映了明末对武将人才的迫切需求。

## 六、与文科举的地位差异

### 6.1 制度差异

| 对比项 | 文科举 | 武科举 |
|--------|--------|--------|
| 设立时间 | 洪武三年（1370年） | 天顺八年（1464年） |
| 成熟度 | 高度完善 | 长期不完善 |
| 考试级数 | 四级（童试→乡试→会试→殿试） | 二级（乡试→会试），崇祯四年后三级 |
| 每科录取 | 约300-400人 | 约35-200人 |
| 社会地位 | 极高 | 远低于文科举 |
| 任官前景 | 可至内阁大学士 | 通常止于中下级武官 |

### 6.2 社会认知差距

- 明代社会**重文轻武**，武举出身者在官场上长期受到歧视
- 文进士可以担任巡抚甚至总督等统兵大员，武进士则难以获得独立统兵权
- 谚语"好铁不打钉，好男不当兵"反映了社会对武职的普遍轻视
- 许多武官职位仍由世袭武官或文官转任把持

### 6.3 嘉靖至崇祯时期的变化

- **嘉靖时期**：倭寇侵扰东南、北方蒙古威胁加剧，武举的重要性开始上升
- **万历时期**：三大征（朝鲜之役、宁夏之役、播州之役）凸显武将缺乏的问题
- **崇祯时期**：后金崛起、农民起义频发，朝廷急需武将人才
  - 崇祯四年增设武举殿试
  - 录取人数大幅增加至约200人
  - 武进士的初始任命待遇提升
  - 但为时已晚，武举改革未能挽救明朝的军事颓势
