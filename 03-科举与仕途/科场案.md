# 科场案

## 概述

明代科举制度虽然设有糊名、誊录、锁院、搜检等多重防弊措施，但科场舞弊案仍层出不穷。舞弊方式主要包括关节（通关节）、夹带（怀挟）、冒籍（顶名冒籍）、贿赂考官、替考（枪替）、越房调卷等。本文详述嘉靖至崇祯期间（及影响深远的此前案例）的重大科场舞弊案。

明代对科场舞弊的处罚相对清代较为温和：嘉靖以前多为杖责、贬官、废黜功名；嘉靖以后加重为"身负重枷示众一月，褫革为民"；万历以后更重至"重枷三个月，发配边远地区充军"。

---

## 一、常见舞弊方式

| 方式 | 说明 |
|------|------|
| **通关节** | 考生与考官事先约定暗号（如在试卷中使用特定字词或句式），考官据此识别并录取 |
| **夹带（怀挟）** | 考生将抄写的缩微版经文或范文藏于衣物、文具、食物中带入考场 |
| **替考（枪替）** | 雇请他人代替自己入场考试 |
| **冒籍** | 冒充他省（录取率较高省份）的籍贯参加考试 |
| **贿赂考官** | 直接向考官行贿，换取录取 |
| **越房** | 同考官跨越自己负责的阅卷区域，到其他房间搜索特定考生的试卷 |
| **调卷** | 在弥封或誊录环节动手脚，替换或篡改试卷 |

## 二、弘治十二年唐寅案（1499年）

> 虽在嘉靖之前，但影响深远，是明代最著名的科场案之一。

### 基本信息

| 项目 | 内容 |
|------|------|
| 时间 | 弘治十二年（1499年） |
| 考试 | 己未科**会试** |
| 涉案人物 | 主考官**程敏政**、考生**唐寅**（唐伯虎）、考生**徐经** |
| 举报人 | 给事中**华昶** |

### 案情经过

1. **背景**：程敏政为弘治十二年会试主考官，唐寅和徐经均为此科考生
2. **举报**：给事中华昶弹劾程敏政"泄漏试题"给唐寅和徐经
3. **怀疑根据**：会试前，唐寅曾在考前扬言自己必中会元，引发猜疑；徐经曾以重金拜访程敏政府邸
4. **调查**：弘治帝大怒，暂停程敏政阅卷，立即逮捕徐经和唐寅下狱审讯
5. **复核结果**：李东阳等人复核试卷后，认为**并无泄题的确凿证据**

### 处罚结果

| 涉案人 | 处罚 |
|--------|------|
| **程敏政** | 以"临财不廉，不避嫌疑"为由**令致仕**（强制退休），出狱后不久病逝 |
| **唐寅** | **革除功名**，"身贯三木"（受刑具折磨），后被发往浙江充吏，唐寅不就 |
| **徐经** | 革除功名，充吏，后英年早逝 |
| **华昶** | 因举报"言事不实"被调任南京太仆寺主簿 |

### 后世评价
- 多数学者认为此案很可能是**冤案**——程敏政并未真正泄题
- 唐寅因此案终身未能入仕，转而成为著名画家和文人
- 此案反映了明代科场的政治敏感性：一旦卷入舞弊嫌疑，即使无确证也难以全身而退

**来源**：[弘治十二年唐伯虎舞弊案 - 知乎](https://zhuanlan.zhihu.com/p/92662994)；[深度揭秘唐伯虎科场舞弊案 - 知乎](https://zhuanlan.zhihu.com/p/527535227)；[1499年明朝科举考试悬案 - 知乎](https://zhuanlan.zhihu.com/p/69209379)

## 三、嘉靖二十三年甲辰科场案（1544年）

### 基本信息

| 项目 | 内容 |
|------|------|
| 时间 | 嘉靖二十三年（1544年） |
| 考试 | 甲辰科**会试** |
| 涉案人物 | 主考官**张潮**（病逝）、副主考**江汝璧**、同考官**沈坤、彭凤、欧阳㬇、高节**等、首辅**翟銮**之子**翟汝孝、翟汝俭** |

### 案情经过

1. **异常事件**：甲辰科会试主考官、礼部尚书兼翰林学士**张潮**在入贡院后，三场笔试刚结束即**突然病逝**
2. **接替主持**：副主考官**江汝璧**在同考官茅瓒的协助下主持阅卷
3. **舞弊行为**：江汝璧串通同考官沈坤、彭凤、欧阳㬇、高节等人，**徇私舞弊**，录取了首席大学士（少傅衔）**翟銮**的两个儿子——**翟汝孝**和**翟汝俭**
4. **殿试结果**：经殿试，翟汝孝位列**一甲第三名**（探花）
5. **举报**：有人举报翟氏兄弟去年一起通过乡试、今年又一起通过会试，"大有蹊跷"

### 舞弊方式
- **通关节**：江汝璧与翟銮府邸事先有联络
- **考官串通**：多名同考官共同参与，形成舞弊网络

### 处罚结果

| 涉案人 | 处罚 |
|--------|------|
| **翟汝孝、翟汝俭** | **废黜功名**，革除进士身份 |
| **翟銮**（首辅） | 因此案受牵连，被**勒令致仕** |
| **江汝璧**等考官 | **杖责、贬官** |

> 此案被认为是嘉靖朝**处罚最重的科场案**，但以清代标准来看仍属温和。

**来源**：[明甲辰科场案 - 维基百科](https://zh.wikipedia.org/zh-hans/%E6%98%8E%E7%94%B2%E8%BE%B0%E7%A7%91%E5%9C%BA%E6%A1%88)；[明朝三大科场舞弊案件 - 腾讯](https://ac.qq.com/Ask/detail/aid/zfpezfzftu)

## 四、万历十六年顺天乡试案（1588年）

### 基本信息

| 项目 | 内容 |
|------|------|
| 时间 | 万历十六年（1588年） |
| 考试 | 戊子科**顺天乡试** |
| 涉案人物 | 主考官**黄洪宪**、考生**王衡**（宰相王锡爵之子）、**沈璟**、**陈与郊** |

### 案情经过

1. **争议焦点**：此科录取的举人中，有多名权贵子弟中式，引发朝野哗然
2. **主要嫌疑**：宰相申时行的女婿、大学士王锡爵之子**王衡**中式，被质疑为考官徇私
3. **舆论压力**：科道官纷纷弹劾主考官黄洪宪

### 处罚结果

| 涉案人 | 处罚 |
|--------|------|
| **王衡** | 被列为嫌疑人，后通过复试及殿试证明才学（万历二十九年中**榜眼**），但终身难消阴影 |
| **沈璟** | 万历十七年迫于舆论引咎辞职 |
| **陈与郊** | 万历十八年被迫告假回乡，两年后免职 |
| **高桂** | 降两级 |
| **饶伸** | 革职为民 |

> 此案争议较大：王衡后来中了榜眼，证明其确有才学。案件实质可能更多是**党争借题发挥**，而非纯粹的科场舞弊。

## 五、万历三十八年韩敬案（1610年）

### 基本信息

| 项目 | 内容 |
|------|------|
| 时间 | 万历三十八年（1610年） |
| 考试 | 庚戌科**会试** |
| 涉案人物 | 同考官**汤宾尹**、考生**韩敬**、知贡举**吴道南**（王图） |

### 案情经过

1. **背景**：汤宾尹为南京国子监祭酒，此科担任会试同考官（分校官）。韩敬为其门生
2. **越房搜卷**：韩敬的试卷被分配到**徐銮**考房，徐銮将其刷掉。汤宾尹得知后，**越房**到徐銮考房中搜出韩敬的试卷
3. **涂改批阅**：汤宾尹将徐銮的批阅意见**涂去**，重新签上自己的批阅意见，强行将韩敬的试卷纳入推荐名单
4. **引发效仿**：其他同考官见状，纷纷效仿，从被刷掉的试卷中找出各自的"关系户"，一下"搜检"出**18位**
5. **与主考争吵**：知举官吴道南发现异常，汤宾尹居然与吴道南**公开争吵**，为韩敬"据理力争"
6. **结果**：汤宾尹强行使韩敬被录取为**第一名（会元）**。殿试后韩敬更获**状元**

### 举报与处理

| 时间 | 事件 |
|------|------|
| 万历三十八年 | 韩敬中状元，朝野哗然 |
| 万历三十九年 | 京察，东林党揭发汤宾尹舞弊 |
| 万历四十年十月 | 给事中**孙振基**弹劾汤宾尹 |

### 处罚结果

| 涉案人 | 处罚 |
|--------|------|
| **汤宾尹** | **夺官**，被罢免一切职务 |
| **韩敬** | 令**冠带闲住**（保留功名但不得任官），此后处处受鄙视排挤，没几年即辞官回家 |

### 后续
- 韩敬服阕后曾上疏申冤
- 万历四十七年三月被**降二级**，调南京别衙门用
- 因此案，韩敬虽保住了状元头衔，但在官场上实际无法立足
- 此案与当时的**东林党争**密切相关

> 韩敬被讽为"四万两状元"，暗示其状元地位是用金钱买来的。

**来源**：[明代文人的命运与科场舞弊](https://html.rhhz.net/WNSFXYXB/html/e91bc852-5501-49bb-8ac6-e7635ba687e8.htm)；[明清两代的科场案 - 新浪博客](https://blog.sina.com.cn/s/blog_4fcd5e5e0102yju8.html)

## 六、万历四十四年沈同和案（1616年）

### 基本信息

| 项目 | 内容 |
|------|------|
| 时间 | 万历四十四年（1616年） |
| 考试 | 丙辰科**会试** |
| 涉案人物 | **沈同和**（吴江人）、**赵鸣阳**（沈同和同乡） |

### 案情经过

1. **背景**：沈同和为万历五年进士沈季文之子，以仪容出众、善诗赋闻名，但**不擅长八股文写作**
2. **乡试阶段**：万历四十三年（1615年），沈同和在同乡赵鸣阳的协助下通过了乡试，成为举人
3. **会试舞弊**：
   - 沈同和**贿赂礼部官员**，安排自己与赵鸣阳**同住一间号舍**
   - 会试第一场的七篇文章中，**除一篇抄自刊刻文集外，其余全部由赵鸣阳代写**
4. **放榜结果**：沈同和高中**第一名（会元）**，赵鸣阳列第六名
5. **引发怀疑**：京城士子哗然——一个不善八股文的人怎么可能中会元？

### 复试与真相

1. **覆试**：吏部知贡举吴道南下令礼部进行**覆试**
2. **当堂出题**：令沈同和以"节俭之君"为题作八股文一篇
3. **当场露馅**：沈同和**无法作答**
4. **诗赋测试**：有人为其辩护说他有诗才。万历帝令其当场作**咏梅花诗百首**，本想为其开脱
5. **彻底失败**：沈同和**一整天连一首诗都写不出来**

### 处罚结果

| 涉案人 | 处罚 |
|--------|------|
| **沈同和** | **遣戍**（发配戍边） |
| **赵鸣阳** | **杖责**，褫革功名，革职为民 |

> 万历帝本人对此案的处理有一句名言："国家以制义取士，非诗赋也"——意思是科举考的是八股文，不是诗赋，沈同和的"诗才"不能救他。

**来源**：[沈同和 - 维基百科](https://zh.wikipedia.org/zh-hans/%E6%B2%88%E5%90%8C%E5%92%8C)；[明代文人的命运与科场舞弊](https://html.rhhz.net/WNSFXYXB/weixin/e91bc852-5501-49bb-8ac6-e7635ba687e8.htm)

## 七、复社操纵科场（崇祯年间，1628-1644年）

### 基本信息

| 项目 | 内容 |
|------|------|
| 时间 | 崇祯年间（约1628-1644年） |
| 性质 | 非单一科场案，而是长期的**系统性操纵** |
| 核心人物 | **张溥**（复社领袖） |

### 背景
- 天启四年（1622年），张溥开始结社评论时政，初名"应社"，后改为**"复社"**
- 复社成员最多时高达**三千余人**
- 复社以科举文章为纽带，将全国士子联结为一个庞大的学术—政治网络

### 操纵科场的方式

| 方式 | 具体操作 |
|------|----------|
| **公荐** | 复社领袖公开向考官推荐特定考生 |
| **独荐** | 张溥等人以个人信誉担保，向考官推荐门生 |
| **转荐** | 通过复社网络层层转介绍，将推荐信送到考官手中 |
| **编选时文** | 编选《国表》《国衡》等八股文选集，引领文风走向，使复社文人的写作风格成为考场主流 |
| **控制首辅** | 张溥甚至控制了首辅**周延儒**，给周延儒一个册子写满人名，"有的要大用，有的要罢官"，周延儒尽数照办 |

### 影响程度
时人评价：
- "春秋两试，孰元孰魁，孰先孰后，庶常（张溥）已编定无遗人矣"
- "赖其奖擢成名者数十百人"
- "中式者皆复社之人"

### 反弹
- 崇祯六年（1633年），温体仁继任首辅后，首开攻讦复社之端
- 温体仁之弟育仁欲求入复社被拒，遂指使人写《绿牡丹传奇》讽刺复社
- 崇祯十四年（1641年），张溥猝死（年仅40岁），复社势力迅速衰落

### 定性
- 复社操纵科场并非传统意义上的"舞弊"——没有替考、夹带、通关节等行为
- 但其**系统性地影响考官决策和考试结果**，实质上扭曲了科举的公平性
- 复社的影响力说明：晚明科举已不仅是个人才学的竞争，更是**社会网络和政治力量**的角逐

**来源**：[崇祯时期的复社人何以称霸科场 - 新浪博客](https://blog.sina.com.cn/s/blog_485dcc670102xzc3.html)；[张溥 - 百度百科](https://baike.baidu.com/item/%E5%BC%A0%E6%BA%A5/33423)

### 8.3 天顺七年贡院火灾（1463年）

| 项目 | 内容 |
|------|------|
| 时间 | 天顺七年（1463年） |
| 考试 | 会试第一场 |
| 事件 | 巡查贡院的士兵**生火取暖**引发火灾 |

## 九、科场案的历史意义

### 9.1 与党争的深层关联
明代中后期的科场案，往往不只是单纯的考试舞弊，而是与**朝廷党争**密不可分：

| 案件 | 相关党争 |
|------|----------|
| 韩敬案（1610年） | 东林党 vs 宣党 |
| 顺天乡试案（1588年） | 与申时行、王锡爵等阁臣的政治对抗 |
| 复社操纵科场 | 复社 vs 温体仁一系 |

### 9.2 制度反思
- 每次重大科场案后，朝廷都会加强防弊措施
- 但新措施往往催生新的舞弊方式
- 科举作为选拔制度的公信力在晚明遭到严重削弱
- 复社操纵科场等现象说明：制度性腐败已超出了个案处理的范畴

### 9.3 明清处罚力度对比
- 明代科场案的处罚以**杖责、贬官、废黜功名、充军**为主
- 清代则大幅加重：顺治丁酉科场案中，主考官**被处斩**，多人被杀
- 明代的相对温和可能与文官集团的强大有关——官僚系统内部有足够的力量抵制过于严厉的处罚
