# 功名与特权

## 概述

明代科举功名分为生员（秀才）、举人、贡生、进士四个主要层级，另有庶吉士作为进士中的精英选拔。各级功名持有者享有不同程度的免役特权、司法特权、社会特权和经济特权。嘉靖二十四年（1545年）的《优免则例》和万历三十八年（1610年）的《优免新例》是两个关键的制度节点，后者大幅扩大了士绅的优免范围，加剧了赋役负担向底层百姓的转移。

---

## 一、生员（秀才）的权利清单

| 权利项目 | 具体内容 |
|----------|----------|
| **免除杂役** | 免粮2石、人丁2丁对应的杂泛差役（嘉靖二十四年定制） |
| **见官不跪** | 见知县行揖礼（作揖），有座椅可坐；普通百姓须跪拜 |
| **免受笞刑** | 地方官不得直接对生员施加笞杖等肉体刑罚 |
| **司法保护** | 若生员犯罪，须先由学政**褫革功名**，方可审理处罚 |
| **穿戴特权** | 可穿**襕衫**、戴**方巾**，与普通百姓衣着有明显区分 |
| **廪膳补贴** | 廪生每月领取廪米六斗（约折银3-4两/年） |
| **科考资格** | 有权通过科考获取乡试资格 |
| **社会尊称** | 称"相公" |

### 免刑的具体规定
- 地方官若未经褫革程序直接杖责秀才，属**"凌辱斯文"**
- 此行为将被上级严查，可能导致地方官受处分
- 严重时甚至引发当地生员**集体罢考抗议**（"闹考"）
- 学政对生员有直接惩罚权：岁考成绩列五六等者可予以降等或黜革

### 万历优免新例下的生员
- 万历三十八年改按田亩计算：生员、监生优免田**80亩**

## 二、贡生的权利清单

贡生是从生员中选拔送入国子监的人。明代分四种：

| 类型 | 选拔方式 | 说明 |
|------|----------|------|
| **岁贡** | 廪生按资历论资排辈（"挨贡"） | 最常见 |
| **选贡** | 嘉靖时增设，通过考试选拔 | 每三至五年一次 |
| **恩贡** | 逢皇室庆典加选 | 人数不定 |
| **纳贡** | 援例捐纳取得资格 | 万历以后日渐泛滥 |

| 权利项目 | 具体内容 |
|----------|----------|
| **见官礼仪** | 与生员同：见县官不跪，行揖礼 |
| **做官资格** | 可任"儒学训导"（县学副长官）等基层教职 |
| **会试资格** | 可直接参加会试（免乡试），但实际中式率极低 |
| **优免** | 与生员同：免粮2石、人丁2丁对应的杂役 |
| **月银补贴** | 有生活补助 |

**来源**：[贡生 - 百度百科](https://baike.baidu.com/item/%E8%B4%A1%E7%94%9F/1384662)；[贡生——明清大多数知识分子的选择 - 知乎](https://zhuanlan.zhihu.com/p/315582529)

## 三、举人的权利清单

| 权利项目 | 具体内容 |
|----------|----------|
| **做官资格** | 可直接选授知县、学正、主簿、教谕等基层官职 |
| **见官平等** | 可与知县**平起平坐** |
| **免除杂役** | 嘉靖二十四年：免粮2石、人丁2丁对应的杂役 |
| **优免扩大** | 万历三十八年：未仕举人优免田**1,200亩**对应的杂役 |
| **司法特权** | 犯罪须先**革去举人功名**再审理；死罪可三次减免 |
| **会试资格** | 可赴京参加会试 |
| **终身有效** | 举人功名终身有效，多次会试不第仍保留 |
| **社会地位** | 称"老爷"或"举人老爷"，在地方具有显著影响力 |
| **蓄奴** | 可使用奴仆（普通百姓不得使用） |

> 举人虽然在嘉靖二十四年的优免额度与生员相同（免粮2石、丁2人），但其实际社会地位和做官资格远高于生员。万历优免新例更是将举人的优免田亩从微不足道提升到1,200亩。

**来源**：[明清时期的举人有何特殊性 - 3233](http://m.3233.cn/n/u285-105508.html)；[古代科举制中功名的实际待遇 - 知乎](https://www.zhihu.com/question/23593962)

## 四、进士的权利清单

| 权利项目 | 具体内容 |
|----------|----------|
| **立即授官** | 一甲直授翰林院官职；二三甲经朝考后分发 |
| **仕途起点高** | 最低从正七品知县起步 |
| **入阁资格** | 进士出身是入阁的前提——"非进士不入翰林，非翰林不入内阁" |
| **优免** | 按实授官品级享受优免（见下文则例） |
| **未仕优免** | 万历三十八年：未仕进士优免田最高**3,350亩** |
| **宗族荣耀** | 可立旗杆、建牌坊，光耀门楣 |
| **社会地位** | 称"进士老爷""大人"，地方最高层士绅 |

### 进士初始授官

| 甲第 | 初始官职 | 品级 |
|------|----------|------|
| 状元 | 翰林院修撰 | 从六品 |
| 榜眼、探花 | 翰林院编修 | 正七品 |
| 二甲庶吉士 | 翰林院学习三年 | 无品级 |
| 二甲非庶吉士 | 六部主事或外放知县 | 正六品或正七品 |
| 三甲 | 知县、推官等 | 正七品 |

## 五、庶吉士的权利与前景

| 权利项目 | 具体内容 |
|----------|----------|
| **翰林院学习** | 在翰林院读书三年，由资深翰林教导 |
| **储相身份** | 号称"储相"——未来内阁大学士的后备军 |
| **散馆后授官** | 优异者授翰林院编修或检讨；较差者外放知县 |
| **快速晋升** | 翰林出身者晋升速度远快于外放官员 |

- 始于洪武十八年（1385年）
- 每科从二三甲进士中选取约20-30人
- 明英宗后形成惯例："非翰林不入内阁"
- 张居正即为嘉靖二十六年（1547年）庶吉士出身

## 六、嘉靖二十四年（1545年）优免则例

### 背景
嘉靖二十四年，明廷首次正式颁布有明确限额的《优免则例》。此前官绅的优免缺乏统一标准，各地执行差异极大。

### 京官优免额度

| 品级 | 免粮（石） | 免丁（人） |
|------|-----------|-----------|
| 一品 | 30石 | 30丁 |
| 二品 | 24石 | 24丁 |
| 三品 | 20石 | 20丁 |
| 四品 | 16石 | 16丁 |
| 五品 | 14石 | 14丁 |
| 六品 | 12石 | 12丁 |
| 七品 | 10石 | 10丁 |
| 八品 | 8石 | 8丁 |
| 九品 | 6石 | 6丁 |

### 其他身份

| 身份 | 免粮（石） | 免丁（人） |
|------|-----------|-----------|
| 外官 | 京官的**一半** | 一半 |
| 致仕官 | 本品的**七成** | 七成 |
| 教官、监生、举人、生员 | **2石** | **2丁** |
| 杂职、省祭官、承差、知印吏典 | **1石** | **1丁** |

### 关键辨析

> **所谓"免粮三十石、人三十丁"并非免除赋税本身，而是免除附着在这些人丁田亩上的杂泛差役的银两。** 正赋（田赋正税）该交还是要交。优免从始至终只涉及徭役中的杂役部分，不涉及赋税。里甲正役不能免。

**来源**：[明代的举人、秀才可以免除多少赋税和徭役 - 知乎](https://www.zhihu.com/question/290134594)；[明代徭役优免类型概说 - 故宫博物院 PDF](https://www.dpm.org.cn/Uploads/File/2020/02/16/u5e4921428f4e4.pdf)

## 七、万历三十八年（1610年）优免新例

### 重大变化
万历三十八年《优免新例》将优免额度**大幅提高**，从按"粮石/人丁"计算改为按**田亩**计算。

### 京官优免（按田亩）

| 品级 | 优免田亩 |
|------|---------|
| 一品 | **10,000亩** |
| 二品 | 8,000亩 |
| 三品 | 6,000亩 |
| 四品 | 5,000亩 |
| 五品 | 4,000亩 |
| 六品 | 3,500亩 |
| 七品 | 3,000亩 |
| 八品 | 2,700亩 |

### 其他身份

| 身份 | 优免田亩 |
|------|---------|
| 外官 | 京官的**一半** |
| 致仕官 | 本品的**六成** |
| 未仕进士 | 最高**3,350亩** |
| 未仕举人 | **1,200亩** |
| 生员、监生 | **80亩** |

### 嘉靖与万历优免的对比

| 对比项 | 嘉靖二十四年 | 万历三十八年 |
|--------|-------------|-------------|
| 计算单位 | 粮石 + 人丁 | **田亩** |
| 一品京官 | 30石 + 30丁 | **10,000亩** |
| 举人 | 2石 + 2丁 | **1,200亩** |
| 生员 | 2石 + 2丁 | **80亩** |
| 致仕减免 | 本品七成 | 本品**六成** |

> 万历新例的优免额度远超嘉靖旧例。这一变化被认为是明末士绅势力膨胀、赋役负担向底层转移的重要制度根源之一。

**来源**：[杂文：关于明代优免问题小议 - 知乎](https://zhuanlan.zhihu.com/p/107802407)；[在明朝有功名的读书人需要交税吗 - 知乎](https://zhuanlan.zhihu.com/p/106898629)

## 八、监生（国子监学生）的入学途径和待遇

### 8.1 监生的四种类型

| 类别 | 入学途径 | 说明 |
|------|---------|------|
| **举监** | 会试落榜的举人由翰林院选入国子监 | 唯一具有举人资格的监生，可直接参加下次会试 |
| **贡监** | 贡生（岁贡、选贡、恩贡、纳贡）入监 | 岁贡按资历挨次入监；选贡经考试选拔；恩贡遇庆典加选 |
| **荫监** | 高官子弟或殉难者子孙入监 | 京官三品以上可荫一子入监；因公殉职、守土死节者亦可荫子 |
| **例监** | 捐纳银粮取得监生资格 | 正统年间始设，不需生员身份即可入监 |

### 8.2 监生待遇
- 免收学费（例监除外，须缴纳捐资）
- 监生家属**免除丁役**
- 可参加乡试和会试
- 肄业期满后可直接**选授官职**（教谕、训导、县丞、主簿等）
- 明初规定监生在监期间有**廪饩**供给

### 8.3 捐纳制度（例监）

| 时间 | 捐纳标准 | 说明 |
|------|---------|------|
| **正统十二年（1447年）** | 捐献**800石粮食** | 赈济灾荒之需，首次开例。可成为纳贡生，入国子监 |
| **成化年间** | 生员纳米**100石以上** | 正式定规，须有生员身份 |
| 成化年间 | 军民纳**250石**得正九品散官 | 每加50石增二级，至正七品止 |
| 正德年间 | 非生员亦可捐纳入监 | 称"例监生" |
| 嘉靖至崇祯 | 捐纳规模扩大 | 成为朝廷重要财源 |

## 九、见官不跪与司法特权详述

### 见官礼仪等级

| 身份 | 见知县 | 见知府 | 见学政 |
|------|--------|--------|--------|
| 普通百姓 | **跪拜** | 跪拜 | — |
| 生员 | **揖礼**（有座椅） | 揖礼 | 行弟子礼 |
| 贡生 | 揖礼 | 揖礼 | 揖礼 |
| 举人 | **平起平坐** | 揖礼 | 揖礼 |
| 进士/官员 | 按品级行礼 | 按品级行礼 | — |

### 司法特权

| 功名 | 司法保护 |
|------|----------|
| 生员 | 知县不得直接杖责；须先报学政褫革功名 |
| 举人 | 须先革去举人功名（由礼部备案）再审理 |
| 进士/官员 | 按品级享有更高级别的司法保护 |
| 有功名者犯死罪 | 可获**三次减免**机会 |
| 非死罪 | 减轻处罚 |

## 九、投献、诡寄等利用特权避税的方式

### 9.1 投献

| 项目 | 说明 |
|------|------|
| **定义** | 普通百姓将田地、人丁"献"给有功名的士绅名下 |
| **目的** | 借助士绅的优免特权逃避赋役 |
| **操作** | 百姓名义上将田产过户给士绅，实际仍自己耕种；向士绅缴纳比官府赋税更少的"租子" |
| **士绅获益** | 无需购买即获得大量田产名义所有权，坐收"租差" |
| **危害** | 大量田产从赋役册籍中"消失"，负担转嫁给小民 |

### 9.2 诡寄

| 项目 | 说明 |
|------|------|
| **定义** | 将自己的田地**虚报**在有功名者名下 |
| **变体** | "贴脚诡寄"——浙江一带的特有形式 |
| **操作** | 富户将大量田产拆分登记在多个有功名者名下 |
| **范围** | 嘉靖至崇祯时期在江南地区极为普遍 |

### 9.3 飞洒

| 项目 | 说明 |
|------|------|
| **定义** | 将赋税负担**强行转嫁**到他人田地上 |
| **操作** | 在赋役册籍中虚增某些小户的田亩数 |
| **受害者** | 无权无势的小农 |

### 9.4 花分

| 项目 | 说明 |
|------|------|
| **定义** | 将大块田产拆分为许多小块，分别登记在不同人名下 |
| **目的** | 使每个名下的田亩不超过优免限额，全部获得优免 |

### 累积效应
1. 大量田产逃脱赋役登记
2. 在册田亩减少 → 亩均赋税增加 → 小民负担加重
3. 小民不堪重负 → 弃田流亡 → 在册田亩进一步减少
4. 形成恶性循环，是明末财政崩溃的重要原因

> 万历以后，投献和诡寄规模急剧膨胀。张居正一条鞭法试图通过清丈田亩解决此问题，但张居正死后改革成果很快被逆转。

**来源**：[诡寄 - 百度百科](https://baike.baidu.com/item/%E8%AF%A1%E5%AF%84/2611579)；[明清士绅的快速发财法：投献 - 知乎](https://zhuanlan.zhihu.com/p/580516376)；[官绅不纳粮 - 知乎](https://www.zhihu.com/question/62116661)
