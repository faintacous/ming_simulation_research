# 考核制度

## 概述

明朝官员考核制度分为"考满"和"考察"两大体系。考满侧重政绩，三年一考、六年再考、九年通考，按称职/平常/不称职三等决定升降；考察侧重德行操守，分为京察（针对京官，六年一次）和大计（针对外官，三年一次），以"八法"为标准淘汰不合格者。万历元年（1573）张居正推行考成法，以"三本账"核销制度强化行政效率，是明代考核制度最重要的改革。嘉靖至崇祯年间，京察逐渐异化为党争工具，制度效力大打折扣。

---

## 一、考满制度

### 1.1 基本框架

| 项目 | 内容 |
|------|------|
| 考核对象 | 全体在职文武官员 |
| 考核周期 | 三年初考、六年再考、九年通考 |
| 考核等次 | 称职、平常、不称职（上中下三等） |
| 考核主体 | 由本衙门正官考核，监察御史复核，吏部考功司覆考 |
| 升降规则 | "九年考满例升二级"为明前中期惯例 |

[来源：中国法学网陈国平"明代官员考核制度述论"；中国历史研究院相关文章]

### 1.2 考满流程

**京官考满：**
1. 三年期满 → 本衙门正官"察其行能，验其勤惰"，写出考语
2. 考语分称职/平常/不称职三等
3. 送监察御史考核
4. 吏部考功司覆考，作最终结论
5. 六年、九年再依此流程累计评定

**外官考满：**
1. 州、县官按月统计成绩，上报知府
2. 知府每年考察一次，上报布政司
3. 第三年由巡抚、巡按通核各官事状，造册上报
4. 中央考功部门作最终结论

[来源：中国法学网陈国平"明代官员考核制度述论"；参考网"明代地方官考满制度"]

### 1.3 考满标准——"六事"考核

地方官的考核以《到任须知》规定的"六事"为核心：

| 六事 | 考核内容 | 具体指标 |
|------|----------|----------|
| 学校 | 兴学化民 | 学校是否兴办、儒学生员数量 |
| 田野 | 农业开发 | 田野是否辟荒、耕地面积是否增加 |
| 户口 | 人口管理 | 户口是否增加（正统八年后"以户口增减为其殿最"） |
| 赋役 | 赋税征收 | 赋役是否均平、是否如期完纳 |
| 讼狱 | 司法治安 | 词讼是否简少、判案是否公正 |
| 盗贼 | 社会治安 | 盗贼是否平息、地方是否安宁 |

正统八年（1443）吏部议定："今后守令功绩，必须考验六事，以户口增减为其殿最，庶任斯职者，用心抚字。"

[来源：中国历史研究院相关文章；中国法学网陈国平论文]

### 1.4 考满的升降规则

| 考满结果 | 处理方式 |
|----------|----------|
| 三考称职 | 升二级 |
| 两称职一平常 | 升一级 |
| 称职与不称职相抵 | 视具体情况，一般维持原职 |
| 三考平常 | 不升不降 |
| 两平常一不称职 | 降一级 |
| 三考不称职 | 降二级或罢黜 |

**注意：** 明中后期，考满逐渐向年资倾斜。有学者研究发现，高官职位通常由任职二十余年至三十年的官员担任，任职十四年以下者几乎不可能获得高官。

[来源：中国历史研究院相关文章；腾讯新闻"通往权力之路"]

## 二、京察制度

### 2.1 基本框架

| 项目 | 内容 |
|------|------|
| 考察对象 | 京城各衙门官员 |
| 考察周期 | 六年一次（弘治年间定制） |
| 主持者 | 吏部会同都察院 |
| 考察标准 | 八法（见下表） |
| 核心特征 | 京官四品以上自陈（上疏自述功过），五品以下由吏部和都察院直接考察 |

[来源：维基百科"京察"条目；百度百科"京察"条目]

### 2.2 八法标准详解

| 法名 | 含义 | 处置方式 |
|------|------|----------|
| 年老 | 年事已高，精力不济，不堪任事 | 致仕（强制退休） |
| 有疾 | 身患疾病，无法正常履职 | 致仕（强制退休） |
| 罢软 | 为政疲软无为，缺乏担当，不能决断 | 闲住（免职闲居，不授新职） |
| 不谨 | 行为不检点，操守有亏，但未达贪污程度 | 闲住或降调 |
| 贪 | 贪污受贿，侵吞公帑 | 为民（削职为民，最严厉处分之一） |
| 酷 | 滥用刑罚，残暴害民 | 为民 |
| 浮躁 | 性情轻率，办事不稳重，好大喜功 | 降三级调用 |
| 才力不及 | 能力不足，难胜其任 | 降二级调用 |

[来源：百度百科"八法"条目；百度百科"京察"条目；维基百科"京察"条目]

### 2.3 京察的处置方式汇总

| 处置类型 | 含义 | 适用情况 |
|----------|------|----------|
| 致仕 | 强制退休，给予一定体面 | 年老、有疾 |
| 降调 | 降级调任他职 | 浮躁（降三级）、才力不及（降二级） |
| 闲住 | 免去现职，居家闲住，不再授官 | 罢软、不谨 |
| 为民 | 削去官身，贬为庶民 | 贪、酷 |

### 2.4 京察四品以上自陈制度

- 四品以上京官在京察时须上疏"自陈"——自述任职以来的功过得失
- 皇帝根据自陈内容和吏部意见决定去留
- 实际操作中，自陈多为形式，皇帝通常"温旨慰留"
- 但政治斗争中，自陈可能成为被政敌利用的机会
- 自陈制度正式确定于正德四年（1509年）

**自陈罢黜数据统计：**

| 京察年份 | 四品以上自陈被罢黜人数 | 说明 |
|---------|----------------------|------|
| 正德四年（1509） | 8人 | 两京百余名四品以上京官中仅8人被罢 |
| 正德十年（1515） | 0人 | 无一人因自陈被罢黜 |
| 隆庆三年（1569） | 2人 | 仅2人被罢黜 |
| 万历十五年（1587） | 0人 | 无一人被黜落 |
| 天启三年（1623） | 1人 | 仅1人被罢黜 |

以上数据表明，自陈制度在实际运作中效力极为有限，高级官员除非涉入政治斗争，极少因自陈被罢黜。

[来源：中华文史网"政治"栏目]

### 2.5 京察作为政治斗争工具的典型案例

#### 嘉靖六年京察
"大礼议"余波未平，户部尚书秦金、工部尚书赵璜因自陈被罢黜，但给予极高荣休礼仪。体现了皇帝借京察进行权力微调的手段。

#### 嘉靖三十年京察
兵部尚书王邦瑞、兵部右侍郎翁万达同时因自陈被黜。同一部门尚书、侍郎被同时黜落，在整个明代京察史上极为罕见，反映的是嘉靖帝借京察清除特定人物。

#### 万历二十一年"癸巳京察"（1593）
- 吏部尚书孙鑨主持
- 引发朝中严重的门户之争
- 被视为东林党兴起的发端
- 此后京察日益成为党争工具

#### 万历三十九年"辛亥京察"（1611）
- 主持者多为东林党人
- 齐、楚、浙诸党遭到打击
- 京察结果引起各方强烈反弹

#### 万历四十五年"丁巳京察"（1617）
- 齐、浙、楚三党势力大盛
- 东林党人在此次京察中几乎被驱逐殆尽
- 标志着党争全面控制京察制度

[来源：中华文史网"政治"栏目；豆瓣"万历癸巳乙巳辛亥丁巳京察始末"；维基百科"京察"条目]

### 2.6 京察制度的衰败

- 高级京官除非政治斗争，绝少因京察自陈而去位
- 每六年仅罢黜极个别官员，制度对官员的约束力有限
- 万历以后京察完全异化为党派角力的工具，失去原有的考核功能

## 三、大计制度

### 3.1 基本框架

| 项目 | 内容 |
|------|------|
| 考察对象 | 外官（地方各级官员） |
| 考察周期 | 三年一次 |
| 正式名称 | 又称"朝觐考察"（外官进京朝觐时接受考察） |
| 朝觐年份 | 丑、辰、未、戌年（洪武二十九年起改为三年一次） |
| 主持者 | 巡抚、巡按通核，布政司、按察司协同 |
| 考察标准 | 与京察八法相同 |
| 报送流程 | 州县 → 府 → 布政司 → 巡抚巡按覆核 → 造册报送吏部 |

[来源：中国法学网陈国平"明代官员考核制度述论"；共产党员网"明代吏治思想制度之得"]

### 3.2 大计的处分标准

| 类别 | 处分 |
|------|------|
| 年老、有疾 | 五品以下致仕（强制退休） |
| 浮躁、才力不及 | 降调 |
| 罢软、不谨 | 闲住 |
| 贪、酷 | 贬为庶民 |

### 3.3 大计的实际效果

- 大计在明前期执行较为认真，有一定惩戒效果
- 明中后期，地方考核日趋形式化，"考语千人一面"
- 巡抚、巡按的考核意见往往受到人情关系和政治因素影响
- 但相比京察，大计因涉及中下级地方官，政治化程度相对较低

## 四、张居正考成法

### 4.1 背景

万历元年（1573），张居正出任首辅后，针对官场"姑息之政"——公文拖延、政令不行、官员怠惰——推行考成法，核心理念是"综核名实"。

### 4.2 三本账制度

六部和都察院将所属官员应办事项设定期限，分别登记在三本账簿上：

| 账簿持有者 | 职责 |
|------------|------|
| **六部/都察院**（底册） | 留作本部门的底册，逐月检查：每完成一件销出一件，未完成须如实申报，否则以违罪处罚 |
| **六科**（稽核册） | 根据账簿登记，要求六部每半年上报一次执行情况，逾期未完成事项由六科纠举议处 |
| **内阁**（总核册） | 对六科的稽查工作进行查实，确保六科认真履行监督职责 |

[来源：维基百科"考成法"条目；百度百科"考成法"条目；最高人民检察院"张居正改革中的考成法考论"]

### 4.3 核销流程

```
事项登记 → 六部/都察院按月自查
           ↓
       六科每半年稽核一次
           ↓
       内阁年终总核查
           ↓
       未完成者 → 逐级追责
```

**关键机制：**
- "月有考、岁有稽"——每月检查、每年汇总
- 每半年和年终要"类奏汇缴"——通盘清查完成与未完成事项
- 连续的、累积式稽核是考成法的精义所在

[来源：最高人民检察院"张居正改革中的考成法考论"；共产党员网"天下之事，不难于立法"]

### 4.4 考成法的处分措施

处罚力度递进：镌俸（扣薪）→ 住俸（停薪）→ 夺俸（罚薪）→ 降级 → 革职

### 4.5 考成法的实际效果——数据

| 方面 | 具体数据 |
|------|----------|
| 行政效率 | 大幅提升，公文拖延现象显著减少 |
| 赋税征收 | 完纳率从此前的50%–60%提高至90%以上 |
| 万历三年（1575） | 查出各省抚按官名下未完成事件**237件**，涉及抚按诸臣**54人** |
| 万历三年处分 | 凤阳巡抚王宗沐、巡按张更化、广东巡按张守约、浙江巡按肖廪——罚停俸三月 |
| 万历四年（1576） | 山东**19名**、河南**12名**地方官因征赋不足九成受降级和革职处分 |
| 万历五年十一月 | **48名**各级官吏被一次性勒令致仕闲住 |
| 万历八年至九年 | 一年时间内裁官**595员** |
| 万历元年至九年累计 | 中央和地方共裁汰冗员达**1,321名** |
| 张居正主政全期 | 裁掉冗官约占官吏总数的**十分之三** |
| 持续时间 | 万历元年推行，万历十年（1582）张居正去世后逐渐废弛 |
| 争议 | 被批评者认为过于苛严，"催科太急"，但对扭转嘉靖后期至隆庆年间的行政松弛效果显著 |

[来源：维基百科"考成法"条目；百度百科"考成法"条目；最高人民检察院"张居正改革中的考成法考论"]

## 五、考核制度的总体评价

### 制度设计的优点

| 层面 | 评价 |
|------|------|
| 考满 | 以量化指标（六事）为基础，注重政绩，流程完整 |
| 京察 | 八法涵盖品德、能力、健康三个维度，较为全面 |
| 大计 | 逐级考核、层层覆核，理论上可防止上级独断 |
| 考成法 | 三级监督体系（六部→六科→内阁）设计精巧，以事考人 |

### 实际运作中的问题

| 时期 | 问题 |
|------|------|
| 明中期 | 考满逐渐流于形式，"考语千人一面"，年资重于政绩 |
| 万历后 | 京察完全沦为党争工具，失去考核功能 |
| 天启间 | 阉党操纵京察，排斥异己，制度公正性荡然无存 |
| 崇祯间 | 朝政混乱，考核制度名存实亡 |
| 考成法废弛后 | 行政效率迅速回落，赋税完纳率大幅下降 |

### 外官考核的特殊困难

- "到任须知"中的六事指标在实践中难以精确量化
- 巡抚、巡按的考核意见受人情关系和政治因素影响
- 地方官考核结果往往"因人就事"而非"因事考人"
- 相比京察，大计因涉及中下级地方官，政治化程度相对较低
