# 俸禄与待遇

## 概述

明朝官员俸禄制度以"薄俸"著称，清人修《明史》时评价"自古官俸之薄，未有若此者"。俸禄标准定型于洪武二十五年（1392），正一品月俸87石，从九品月俸5石。但由于"折色"制度——将部分俸禄折为宝钞、胡椒、苏木等贬值物品发放——官员实际到手的收入远低于名义标准。正式俸禄不足以维持体面生活，导致"常例""规费"等灰色收入成为官僚体系运转的隐性基础。明朝不设养廉银，这一制度为清雍正朝所创。柴薪银、皂隶折银等额外补贴在明中后期逐渐制度化，成为官员合法收入的重要组成部分。

### 1.2 与历代俸禄的对比

| 朝代 | 正一品年俸参考 | 知县级年俸参考 | 特点 |
|------|---------------|---------------|------|
| 唐（贞观） | 约700石（禄米） + 职田收入 | — | 另有职田、公廨钱等 |
| 宋 | 远高于明代（正一品料钱折合约数千贯） | 知县年俸约米千余石+各项补贴 | 高俸制，另有服装费、茶汤钱等 |

## 二、本色与折色制度

### 2.1 基本概念

| 概念 | 含义 |
|------|------|
| 本色 | 直接发放实物（米粮）的部分 |
| 折色 | 将部分俸禄折换为其他物品或货币发放 |
| 折色是明朝"低俸制"的根源 | 朱元璋初定的俸禄标准并不低，但折色制度使实际到手大幅缩水 |

### 2.2 本色俸禄的组成

| 类别 | 发放方式 | 说明 |
|------|----------|------|
| 月米 | 每位官员每月支取1石米 | 不论品级高低，一律1石 |
| 折绢米 | 每年有2个月的俸禄以绢支取 | 通常1匹绢折银7钱 |
| 折银米 | 每年有10个月的俸禄以银折米支取 | 折价往往低于市价 |

[来源：知乎"明朝官员的真实俸禄是多少"；儒家网吴钩"明代官员的工资是高还是低"]

### 2.3 折色的具体物品

| 折色物品 | 说明 |
|----------|------|
| 大明宝钞 | 最主要的折色手段；洪武末年已严重贬值，至明中期几乎废纸 |
| 胡椒 | 永乐十二年（1414）首次出现胡椒折俸；郑和下西洋带回大量胡椒，充入内库折发官俸 |
| 苏木 | 另一种常见的折俸物资，来自南洋贸易 |
| 布帛 | 偶尔以布帛折俸 |
| 白银 | 明中后期逐渐增加以白银折俸的比例 |

[来源：知乎"大明俸禄折支胡椒苏木罪魁祸首考"；澎湃新闻"倒卖俸禄的黄牛党"；搜狐"揭秘明朝实行的胡椒苏木折俸"]

### 2.4 折色导致的实际收入缩水

**以万历时期正一品官员为例：**

| 名义年俸 | 折色后实际发放 |
|----------|---------------|
| 1,044石禄米 | 禄米12石 + 银215两5钱1分2厘 + 宝钞7,128贯 |

宝钞7,128贯在万历时期实际购买力极低，形同废纸。因此正一品官员的实际年收入约折合白银200余两+12石米。

**以正七品知县为例：**

| 名义年俸 | 折色后实际所得 |
|----------|---------------|
| 90石禄米 | 实得米约27石，余下折钞630贯 |
| 折合白银 | 约27.49两/年 |

[来源：北京日报"明代官俸之薄堪称历代之最"；人民论坛"明代官员的收入与生活水平"]

### 2.5 胡椒苏木折俸的背景

- 永乐年间郑和下西洋带回大量南洋香料（胡椒、苏木等），存于内库
- 朝廷用这些囤积物资折抵官俸，解决财政压力
- 官员领到胡椒、苏木后须自行在市场上变卖换钱
- 市场上胡椒、苏木价格因大量抛售而不断下跌，官员实际所得进一步缩水
- 滋生了"倒卖俸禄"的黄牛行业

[来源：知乎"大明俸禄折支胡椒苏木罪魁祸首考"；东方头条"明朝时期胡椒苏木珍贵"]

## 三、官邸制度

### 3.1 官员住房规制

| 品级 | 正堂间数 |
|------|----------|
| 公侯至亲王 | 正堂7至11间（后改为7间） |
| 五品以上 | 正堂5至7间 |
| 六品以下及平民 | 正堂3间 |

### 3.2 地方公廨（衙署）

| 部分 | 功能 |
|------|------|
| 正堂（大堂） | 发布政令、举办庆典、审理案件的公共空间 |
| 二堂/三堂 | 穿过屏门之后的内室，为官员日常起居之所 |
| 附属建筑 | 吏房、库房、牢房等 |

地方官在任期间住在衙署内，公廨既是办公场所也是住所。

[来源：中国社会科学网"明清时期府县衙门的政治构成"]

### 3.3 官舍与私宅

- 京官一般无专门官邸，需自行租赁或购买住房
- 地方官住在衙署内，离任后须腾退
- 住房开支是京官的重要负担之一

## 四、养廉银问题

| 项目 | 内容 |
|------|------|
| 明朝是否有养廉银 | **没有**。养廉银是清雍正元年（1723）创立的制度，为清朝特有 |
| 明朝的替代补贴 | 有三项额外津贴：①道里费（差旅津贴）②柴薪银 ③廪给银 |
| 皂隶折银 | 明中后期制度化的补贴之一，官员原可使用官方差役，后改为折银发放 |
| 补贴目的 | 虽无"养廉银"之名，但这些津贴意在补贴官员生活，有"养廉"之实 |
| 补贴数额 | 数额有限，远不足以弥补薄俸的缺口 |

[来源：维基百科"养廉银"条目；人民论坛"明代官员的收入与生活水平"]

## 五、灰色收入

### 5.1 灰色收入的定义

"常例钱"是官僚体系中名副其实的灰色收入——不是彻底的贪污受贿（黑色收入），但也不是合法的正式俸禄。它是整个官僚体系从上到下心知肚明、人人都拿的"潜规则"收入。

### 5.2 常见灰色收入形式

| 名称 | 说明 |
|------|------|
| 常例钱 | 各种公务处理中按惯例收取的费用，如审案结案费、办理文书费等 |
| 规费 | 各级衙门办理公务时向当事人收取的费用 |
| 陋规 | 非正式的惯例性收费，种类繁多 |
| 火耗 | 征收赋税时以"熔铸损耗"为名加征的额外银两 |
| 赃罚 | 将罚款收入归入私囊 |
| 迎接银/送别银 | 官员上任时的"迎接银"和离任时的"送别银" |
| 说事过钱 | 中间人引介办事的佣金 |
| 润笔 | 为人撰写文章的酬劳（此项属合法业余收入） |
| 投献与诡寄 | 接受非法的土地投献和赋税诡寄 |

[来源：人民论坛"明代官员的收入与生活水平"；儒家网吴钩"明代官员的工资是高还是低"；明朝"低俸制"的真相]

### 5.3 常例钱的具体数额

以正七品知县为例：

| 收入类型 | 年收入（两银） |
|----------|---------------|
| 正式俸禄（折色后） | 约27.49两 |
| 常例钱 | 约2,700–3,000两 |
| 迎接银（上任一次） | 约2,000两 |
| 送别银（离任一次） | 约2,000两 |

**关键数据：** 一个知县即使不贪污不受贿，仅靠常例收入每年约有3,000两——几乎是正一品官员名义年俸的十倍。

有史料记载，一位知县致仕后共积累了6,000两银子的额外收入，比其10年官俸多出123倍。

[来源：人民论坛"明代官员的收入与生活水平"；儒家网吴钩"明代官员的工资是高还是低"]

### 5.4 灰色收入的制度逻辑

- 朝廷心知肚明但默许，因为正式俸禄根本不足以维持官员及其家庭、幕僚、差役的开支
- 常例钱实质上是"官僚体系的运营成本"由民间承担
- 清廉官员（如海瑞）拒绝收取常例，反而被视为异类
- 灰色收入的存在使得反腐标准模糊——"合理的常例"与"过分的贪婪"之间缺乏清晰界限

## 六、典型案例：海瑞的清贫生活

海瑞（1514–1587），历任知县、户部主事、应天巡抚、南京右都御史（正二品），是明朝最著名的清官。

| 项目 | 内容 |
|------|------|
| 在任表现 | 拒绝一切常例和灰色收入 |
| 正二品实际月收入 | 禄米1石 + 银约12两 + 宝钞412贯（实际购买力约折合4石米） |
| 生活状况 | 无力经常食肉，生活清贫 |
| 去世时 | 正三品（南京右都御史），家中极度贫困，连棺材都买不起 |
| 丧事 | 副手前来吊唁时发现其家境，最终由同事凑钱购置棺材 |
| 遗产 | 身上仅有白银8两 |

海瑞的案例极端但真实地反映了：如果一个明朝官员完全依靠正式俸禄生活，即使做到正二品高官，也难以维持基本体面。

[来源：知乎"作为正部级干部的海瑞死的时候身上只有8两银子"；北京日报"明代官俸之薄堪称历代之最"]
