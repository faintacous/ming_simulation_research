# 八股文

## 概述

八股文是明代科举考试的核心文体，因文章中有四个段落各由两股相对的排比文字组成（共八股），故名"八股文"。又称"制义""制艺""时文""时艺""经义"。八股文由破题、承题、起讲、入题、起股、中股、后股、束股八部分组成，要求考生以四书五经为题，依据朱熹《四书集注》"代圣人立言"，不得阐发个人见解。

---

## 一、八股文的完整格式

### 1.1 八部分结构详解

| 部分 | 位置 | 具体要求 |
|------|------|----------|
| **破题** | 开篇第一部分 | 只能用**两句话**，点破题目含义，概括全篇主旨。须用第三人称，不直接引用原文 |
| **承题** | 紧接破题之后 | 最多**三句话**，承接破题，进一步说明题意，将破题的含义展开 |
| **起讲** | 过渡段 | 上结承题、下启正文。明代一般**三至五句**即可（清代增至十句左右）。开始进入"代圣人立言"的口吻，以"意谓""且夫""尝思"等虚词起头 |
| **入题** | 引入正论 | 过渡到正式论述，承上启下，点出论述方向。篇幅较短 |
| **起股** | 正论第一段 | 两股相对的排比文字（即两段对偶文），初步展开论述 |
| **中股** | 正论第二段 | 两股相对，为全篇**核心部分**，论述最为充分，篇幅最长 |
| **后股** | 正论第三段 | 两股相对，进一步深化论述，或从另一角度阐发 |
| **束股** | 正论第四段 | 两股相对，收束全篇论述，呼应题目 |

#### 补充说明
- 起股、中股、后股、束股四部分，每部分各有两股相对的文字，合计**八股**
- "股"即排比对偶的段落，两两相对，如同骈文
- 八股之后还有**大结**（收结），总结全篇，不计入八股

### 1.2 各部分的文法规则

| 部分 | 文法要求 |
|------|----------|
| 破题 | 须**暗点题义**，不可明引原文，须用"意在言外"的手法 |
| 承题 | 可小作发挥，但仍须紧扣题目，不可游离 |
| 起讲 | 须入**口气**（即代圣人立言，模仿孔孟语气），从此处开始以古人身份说话 |
| 入题 | 由虚转实，将抽象义理引入具体论述 |
| 起股至束股 | 必须**两两相对**（排比对偶），上下股字数相等、句式相对、意义相关 |

## 二、出题范围

### 2.1 四书五经题目

| 考试类型 | 出题来源 | 备注 |
|----------|---------|------|
| 乡试第一场 | 四书义3道 + 本经义4道 | 四书题最重要 |
| 会试第一场 | 四书义3道 + 本经义4道 | 同上 |
| 童试（县试、府试、院试） | 四书义为主，多用小题 | 小题出自四书 |

### 2.2 四书出题

| 书名 | 说明 |
|------|------|
| 《大学》 | 篇幅较短，可出题目有限 |
| 《中庸》 | 篇幅较短 |
| 《论语》 | 篇幅较长，出题较多 |
| 《孟子》 | 篇幅最长，出题最多 |

### 2.3 五经出题
考生须选定一经（"专经"）：

| 经 | 指定注疏 |
|----|----------|
| 《易》 | 程颐《伊川易传》、朱熹《周易本义》 |
| 《书》 | 蔡沈《书集传》及古注疏 |
| 《诗》 | 朱熹《诗集传》 |
| 《春秋》 | 《左传》《公羊传》《穀梁传》及胡安国传、张洽传 |
| 《礼记》 | 古注疏 |

### 2.4 题目类型

| 题型 | 说明 |
|------|------|
| 大题 | 乡试、会试多用。题意比较完整 |
| 小题 | 童试多用。题目较短 |
| 连章题 | 取一整章或数章 |
| 全章题 | 取完整一章 |
| 数节题 | 取数节经文 |
| 一节题 | 取一节 |
| 数句题 | 取数句 |
| 单句题 | 仅取一句 |

### 2.5 截搭题
- 随着科举进行数百年，四书中完整的句子不够用
- 考官将**上下两章、两节互不相关**的文句合为一题，或各取半句凑成一题
- 称为**截搭题**，亦有冒上题、承下题、上全下偏题、上偏下全题等变体
- 截搭题被批评为割裂经义，但明中后期已成常态

## 三、评判标准

### 3.1 核心原则
- **"代圣人立言"**：必须体现儒家思想，用儒家观点阐释四书五经中的义理
- 对四书的理解必须以朱熹《四书集注》为**唯一准绳**
- 须**"入口气"**：模仿孔子、孟子等圣贤的口吻和语气
- 不得阐发个人见解，不得引用**六经以外**的书籍

### 3.2 文章优劣标准

| 标准 | 说明 |
|------|------|
| **理** | 义理精当，合乎经义，不违朱子之说 |
| **气** | 文气充沛，贯通全篇 |
| **法** | 格法严谨，八股对仗工整，结构完备 |
| **辞** | 辞藻雅正，用语清醇 |

- 王鏊的八股文被评价为"**理实、气舒、神完、法备**"，被视为典范
- 整体文风追求**"清真雅正"**

### 3.3 禁忌
- 不得使用佛道语词
- 不得引用怪僻书籍
- 不得违反程朱理学正统解释
- 不得在文中表达不满或讽刺时政

## 四、字数限制的历史变化

| 时期 | 字数限制 | 备注 |
|------|---------|------|
| 明初（洪武至宣德） | 无严格限制 | 文章篇幅不一 |
| 成化年间（1464-1487） | 逐步形成规范 | 八股格式日趋严格 |
| 明中期 | 五经文以**500字**为满篇，四书文以**300字**为满篇 | |
| 嘉靖至万历 | 逐步放宽 | 文章篇幅有所增长 |
| 万历以后 | 约**550-600字** | |
| 清康熙 | 四书文须满**600字** | |
| 清乾隆以后 | 增至**700字** | |

**说明**：成化二十三年（1487年）始由"经义"正式变为开考八股文，规定按八股方式作文，格式严格，限定字数。

## 五、嘉靖至崇祯的文风演变

### 5.1 各时期文风特征

| 时期 | 文风特征 | 代表人物 |
|------|---------|---------|
| **成化年间**（1464-1487） | 八股文格式正式确立，讲究格律、步骤 | 王鏊、谢迁、章懋 |
| **弘治至正德**（1488-1521） | 追求"理实气舒"，文风醇正 | 王鏊 |
| **嘉靖年间**（1522-1566） | 唐宋派古文家以古文为时文，文气灵巧变化 | 唐顺之、归有光 |
| **隆庆至万历前期**（1567-1600） | 讲究机法，灵变而不逾矩，借题发挥 | 赵南星、顾宪成、汤显祖 |
| **万历后期至天启**（1600-1627） | 文风趋于浮靡，好用僻典，追求奇崛 | 文风渐衰 |
| **崇祯年间**（1628-1644） | 试图矫正浮靡之风，但总体文风已难回复 | — |

### 5.2 代表人物与名篇

| 人物 | 生卒年 | 身份 | 八股文特色 |
|------|--------|------|-----------|
| **王鏊** | 1450-1524 | 成化十一年（1475）殿试第三名（探花），官至户部尚书 | "理实、气舒、神完、法备"，代表作《百姓足君孰与不足》 |
| **唐顺之** | 1507-1560 | 嘉靖八年（1529）会试第一 | 以古文为时文，文气灵巧 |
| **归有光** | 1506-1571 | 嘉靖十九年举人，六十岁始中进士 | 以古文笔法入八股，情理兼备 |
| **赵南星** | 1550-1627 | 万历二年（1574）进士 | 内容不墨守成规，借题发挥 |
| **汤显祖** | 1550-1616 | 万历十一年（1583）进士 | 文辞优美，才气纵横 |

## 六、八股文格式完整示例结构

以一篇标准八股文为例（结构示意）：

```
一、破题（2句）
  ——点明题旨

二、承题（2-3句）
  ——承接破题，进一步阐发

三、起讲（3-5句，明代）
  ——过渡，入圣人口气

四、入题（数句）
  ——引入正式论述

五、起股
  【上股】排比对偶段落
  【下股】与上股对仗

六、中股（篇幅最长）
  【上股】核心论述
  【下股】与上股对仗

七、后股
  【上股】深化论述
  【下股】与上股对仗

八、束股
  【上股】收束论述
  【下股】与上股对仗

九、大结（不计入八股）
  ——总结全篇
```
