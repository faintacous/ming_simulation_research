# 会试与殿试

## 概述

会试和殿试是明代科举制度的最高两级考试。会试在京师礼部贡院举行，由礼部主持，三年一科；殿试由皇帝亲自主持，在宫中举行。会试中式者称"贡士"，殿试后赐进士出身，分一甲、二甲、三甲三等。殿试不淘汰，仅重新排定名次。

---

## 一、会试

### 1.1 时间
- 逢丑、辰、未、戌年举行（乡试次年）
- 三场考试日期：
  - 第一场：二月初九日
  - 第二场：二月十二日
  - 第三场：二月十五日
- 因在春季举行，又称**"春闱"**
- 如遇恩科，另行安排

### 1.2 地点
- **礼部贡院**（京师，北京）
- 永乐十九年（1421年）迁都北京后，会试在北京举行
- 贡院位于北京东南，号舍数千间

### 1.3 考官选任

| 官职 | 人数 | 选任方式 |
|------|------|----------|
| **主考官（总裁）** | 2人 | 由皇帝钦命，从内阁大学士或礼部尚书/侍郎中选任 |
| **同考官** | 8-17人 | 由内阁会同礼部从翰林院编修、检讨、六部司官中选任 |
| **提调** | 1人 | 礼部尚书或侍郎 |
| **监试** | 2人 | 都察院御史 |

### 1.4 三场考试内容

与乡试相同，分三场：

| 场次 | 内容 | 详细 |
|------|------|------|
| 第一场 | 四书义三道、经义四道 | 从四书五经出题，八股文格式 |
| 第二场 | 论一道、判五道、诏诰表各一道 | 公文写作能力考核 |
| 第三场 | 经史时务策五道 | 对时政见解 |

### 1.5 录取人数

#### 明代历科会试录取人数（各时期平均）

| 时期 | 科数 | 一甲 | 二甲 | 三甲 | 合计 | 平均每科 |
|------|------|------|------|------|------|----------|
| 洪武（1368-1398） | 6科（发榜7次） | 21 | 223 | 686 | 930 | 约155人 |
| 建文-永乐 | — | — | — | — | — | 约100-200人 |
| 宣德（1426-1435） | 3科 | 9 | 105 | 186 | 300 | 100人 |
| 正统（1436-1449） | 5科 | 15 | 220 | 413 | 648 | 约130人 |
| 成化（1465-1487） | 8科 | 24 | 755 | 1670 | 2449 | 约306人 |
| 嘉靖（1522-1566） | 15科 | 45 | 1407 | 3469 | 4921 | 约328人 |
| 崇祯（1628-1644） | 6科 | 18 | 383 | 1595 | 1996 | 约333人 |

- 明初录取人数较少，成化以后稳定在**约300名**左右
- 最多一科录取472人
- 成化十一年（1475年）后，会试录取约以**300名**为常额
- 明代共取进士约**24,866人**（有不同统计口径）

### 1.6 副榜
- 会试亦设副榜
- 永乐年间，会试副榜可给予落第举人做官机会
- 副榜举人可授教职或入国子监肄业

## 二、殿试

### 2.1 时间
- 会试放榜后约一个月举行
- 明初在三月初一日
- 后改为**三月十五日**举行
- 次日读卷，再次日放榜

### 2.2 地点
- **奉天殿**（后称皇极殿，今太和殿）
- 阅卷在**华盖殿**（后称中极殿）或**文华殿**进行

### 2.3 策问出题
- 殿试只考**时务策一道**（策问）
- 策问由皇帝出题，或由内阁拟题呈皇帝钦定
- 策文分为**制策**（皇帝出题）和**对策**（考生回答）
- 对策一般须在**一千字以上**
- 内容涉及当朝政治、经济、军事、礼法等重大问题

### 2.4 阅卷流程

| 步骤 | 说明 |
|------|------|
| 1. 答卷 | 贡士在殿内试桌答策，当日交卷 |
| 2. 收卷 | 受卷官收卷 |
| 3. 弥封 | 弥封官将试卷中考生信息折叠弥封、编号 |
| 4. 分卷 | 将弥封试卷分配给各读卷官 |
| 5. 初评 | 读卷官将所分试卷分为三等——上等（一甲备选）、中等（二甲）、下等（三甲） |
| 6. 读卷 | 次日，读卷官齐赴文华殿（或华盖殿），向皇帝朗读评定的上等试卷 |
| 7. 钦定 | 皇帝以**御笔亲定**一甲三名（状元、榜眼、探花）的名次 |
| 8. 拆封 | 拆开弥封，填写金榜 |

#### 读卷官选任
- 读卷官从**内阁大学士、六部尚书/侍郎、都察院、通政司、大理寺正官、詹事府、翰林院堂上官**中选任
- 提调由**礼部尚书或侍郎**担任

### 2.5 三甲划分规则

| 甲第 | 名号 | 人数 | 赐名 |
|------|------|------|------|
| **一甲** | 状元、榜眼、探花 | 固定3人 | 赐**进士及第** |
| **二甲** | 传胪（第一名称传胪） | 约占总数1/3 | 赐**进士出身** |
| **三甲** | — | 约占总数2/3 | 赐**同进士出身** |

- 殿试**不淘汰**，所有参加殿试的贡士均获进士功名
- 只是重新排定名次和甲第
- 一甲固定3人，二甲、三甲人数由皇帝酌定

### 2.6 传胪仪式

传胪是殿试放榜唱名的隆重典礼，一般在殿试后两天举行。

#### 具体仪式流程

| 步骤 | 内容 |
|------|------|
| 1. 设金榜 | 金榜用黄纸书写，故称"黄甲"或"金榜" |
| 2. 皇帝升殿 | 皇帝御奉天殿（皇极殿），文武百官朝服列班 |
| 3. 新进士入列 | 新科进士身着崭新公服，头戴三枝九叶冠，分左右两班站立于文武百官之后 |
| 4. 宣读金榜 | 传胪官（鸿胪寺官）宣读一甲三名姓名、籍贯 |
| 5. 唱名传呼 | 阁门承接，传于阶下，卫士齐声传名高呼，声震殿庭 |
| 6. 一甲出列 | 状元、榜眼、探花依次出列，跪受敕命 |
| 7. 二三甲唱名 | 依次宣读二甲、三甲名次 |
| 8. 颁赐 | 赐状元朝服、冠带 |
| 9. 金榜张贴 | 金榜张贴于长安左门外，供士民观看 |
| 10. 御街夸官 | 状元率新进士骑马游街，自长安左门出，经大明门，赴国子监行释菜礼 |

#### 金榜题名的具体仪式
- "金榜"用黄纸书写，张贴于长安左门外
- 中进士者称**"金榜题名"**
- 状元居榜首，依次排列
- 金榜由礼部制作，皇帝钦定

## 三、朝考与庶吉士馆选

### 3.1 庶吉士制度概述
- 庶吉士是明代翰林院内的短期职位，称"储相"
- 从新科进士中选拔优秀者入翰林院学习
- 制度草创于洪武、永乐时期
- 弘治四年（1491年）趋于健全和完备

### 3.2 馆选（选拔流程）

| 步骤 | 内容 |
|------|------|
| 1. 呈送文章 | 新科进士于一月之内，将平日所作论、策、诗、赋、序、记等文字（限15篇），赴礼部呈献 |
| 2. 初审 | 礼部阅视编号后送翰林院考订，词藻文理可取者按号行取 |
| 3. 复试 | 由礼部与内阁共同出题考核 |
| 4. 比对 | 将复试答卷与先前送上的文章比对水平 |
| 5. 钦点 | 皇帝在名单上圈点，选入者即为庶吉士，称**"点翰林"** |

- 一甲三名（状元、榜眼、探花）**直接授翰林院官职**，不经馆选
- 馆选对象为**二甲、三甲**进士中的优秀者
- 每科馆选人数不定，一般**20-30人**

### 3.3 在馆学习

| 项目 | 内容 |
|------|------|
| 学习地点 | 翰林院附设**庶常馆**（文渊阁） |
| 管理 | 由阁臣督课 |
| 馆课 | 日常课业，由馆师每日检查督促 |
| 阁试 | 相当于月考，每月两次，题目由内阁大学士出，成绩登记造册 |
| 学习内容 | 诗文、史学、典章制度、公文写作等 |
| 学习期限 | **三年** |

### 3.4 散馆（毕业考试）

| 成绩 | 授官 |
|------|------|
| 优异（二甲出身） | 授**翰林院编修**（正七品） |
| 优异（三甲出身） | 授**翰林院检讨**（从七品） |
| 合格 | 授科道官（给事中、御史）或知县 |
| 不合格 | 外放科道或知县等职官 |

### 3.5 进士初授官职总表

| 功名等级 | 初授官职 |
|----------|----------|
| 一甲第一名（状元） | 翰林院**修撰**（从六品） |
| 一甲第二名（榜眼） | 翰林院**编修**（正七品） |
| 一甲第三名（探花） | 翰林院**编修**（正七品） |
| 二三甲选庶吉士 | 入庶常馆学习三年，散馆后授编修/检讨 |
| 二三甲非庶吉士 | 先**观政**（实习），后授六部主事、行人司行人、中书舍人，或外放府推官、知州、知县等 |

### 3.6 观政制度
- 肇始于洪武十八年（1385年）
- 新科进士分赴六部、都察院、大理寺等中央官署"观政"（实习）
- 永乐年间扩展至五军都督府
- 实习期限：约三个月至半年
- 宣德七年起，刑部观政进士可参与审案
- 观政期满后正式授官
