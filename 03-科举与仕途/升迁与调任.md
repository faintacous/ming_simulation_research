# 升迁与调任

## 概述

明朝文官升迁遵循"论资排辈"与"考满升迁"并行的原则，三年初考、六年再考、九年通考，"九年考满例升二级"。不同出身的进士走不同的升迁路线，大致可归纳为三条主干：翰林路线（通往内阁的快车道）、地方路线（由知县层层升至督抚）、科道路线（以言官身份跃升至部院高层）。嘉靖至崇祯年间，升迁制度整体沿袭旧制，但资历在升迁中的权重不断加大，破格提拔虽有但属例外。

[来源：维基百科"明朝官职表"；知乎"明朝翰林院详解"；故宫博物院"大学士"词条]

### 1.2 翰林路线的关键特征

- **速度最快**：翰林出身者升迁速度显著快于其他路线，通常15–20年可入阁
- **"非翰林不入阁"**：自永乐朝以后形成惯例，内阁大学士几乎全部出自翰林院
- **兼充经筵讲官**：翰林官员经常充任经筵日讲官，与皇帝建立近距离关系
- **修史修书**：翰林官员负责修《实录》等重要典籍，是表现机会

### 1.3 内阁大学士的品级与加衔

| 项目 | 内容 |
|------|------|
| 殿阁大学士本职品级 | 正五品（终明之世未变） |
| 殿阁名称 | 华盖殿（后改中极殿）、谨身殿（后改建极殿）、文华殿、武英殿、文渊阁、东阁 |
| 实际品级 | 通过加衔提升：加六部尚书衔（正二品）、加太子太保/太师等衔（正一品） |
| 署衔格式 | "某部尚书兼某殿阁大学士"——本衔在下而兼衔在上 |
| 首辅 | 阁臣中居首者，票拟之权最大，得宠者可比拟宰相 |
| 次辅/群辅 | 其余阁臣 |
| 嘉靖后 | 殿阁大学士的朝位班次排在六部尚书前面 |

[来源：知乎"明朝的内阁大学士品级"；维基百科"大学士"条目；百度百科"内阁首辅"条目]

| 序号 | 官职 | 品级 | 通常升迁来源 | 管辖范围 |
|------|------|------|-------------|----------|

| 2 | 知州 | 从五品 | 知县考满升任 | 一州（直隶州或散州） |
| 3 | 知府 | 正四品 | 知州升任/京官外转 | 一府 |
| 4 | 按察司副使/佥事 | 正四品–正五品 | 知府升任 | 分巡道/兵备道 |
| 5 | 布政司参议/参政 | 从四品–从三品 | 按察司副使升任 | 分守道 |
| 6 | 按察使 | 正三品 | 参政升任 | 一省司法监察 |
| 7 | 布政使 | 从二品 | 按察使/参政升任 | 一省行政财赋 |
| 8 | 巡抚 | 加右副都御史衔（正三品）或右佥都御史衔（正四品） | 布政使/按察使/京官出任 | 一省或数府军政 |
| 9 | 总督 | 加兵部尚书衔（正二品）+ 右都御史衔 | 巡抚/侍郎/尚书出任 | 数省军政 |

[来源：维基百科"明朝官职表"；国家人文历史官网"明代成为知县后怎么规划职业道路"；知乎"知府巡抚府尹布政使品级区别"]

### 2.2 地方路线的关键特征

- **品级跨度大**：从知县（正七品）到总督（加衔正二品），跨越十余级
- **"知县三级跳"**：知县到知州差3级，知州到知府差3级，知府到布政使也差约3级
- **升迁耗时长**：按正常年资，从知县到布政使通常需要20–30年
- **转京优于留外**：在知县任上如能转入京师任给事中、御史或部主事，升迁前景更好
- **方面官转迁**：按察副使、参议等"方面官"在本省内有固定的"小转法"迁转惯例

### 2.3 巡抚与总督的特殊性

| 项目 | 巡抚 | 总督 |
|------|------|------|
| 性质 | 差遣（非正式官职） | 差遣（非正式官职） |
| 加衔 | 通常加都察院右副都御史或右佥都御史 + 兵部右侍郎 | 通常加兵部尚书 + 都察院右都御史 |
| 管辖范围 | 一省或数府 | 跨省（如两广总督、蓟辽总督） |
| 设置时间 | 宣德以后逐渐常设 | 嘉靖以后逐渐固定 |
| 来源 | 布政使、按察使、京官侍郎等 | 巡抚升任或京官尚书/侍郎出任 |

[来源：维基百科"都御史"条目；百度百科"都察院右副都御史"条目]

[来源：维基百科"明朝官职表"；国家人文历史官网；知乎"明朝科道官升迁"]

### 3.2 科道路线的关键特征

- **品卑权重**：给事中仅从七品，但有封驳、纠察、弹劾之权，职权远超品级
- **"给事中优于其他"**：在晋升前景上，给事中被认为优于同品级其他京官
- **内升外转两条路**：内升一般升太常寺少卿等四品京官；外转则为道员和知府
- **升迁快速**：科道官若表现出色，可较快升至侍郎甚至尚书

### 3.3 典型案例：张学颜的科道升迁

张学颜（万历初年理财名臣）的升迁历程：

| 阶段 | 职务 | 品级 |
|------|------|------|
| 起步 | 三甲进士 → 曲沃知县 | 正七品 |
| 转科道 | 工科给事中 | 从七品 |
| 外转 | 山西参议 → 副使 | 从四品→正四品 |
| 军政要职 | 右佥都御史巡抚辽东 | 正四品加衔 |
| 入部 | 兵部侍郎 | 正三品 |
| 最高 | 户部尚书 | 正二品 |
| 用时 | 从三甲进士到六部尚书仅约25年 | — |

## 四、升迁的通则与特例

### 4.1 升迁通则

| 规则 | 内容 |
|------|------|
| 年资优先 | 九年考满升二级为基本规则，资历是升迁的首要依据 |
| 考满关联 | 称职升、平常不动、不称职降，三次考满综合评定 |
| 京官优势 | 京师任职者升迁机会多于地方 |
| 散官阶制 | 每品分初授阶、升授阶、加授阶三阶，按资历功劳稽考 |
| 丁忧扣年 | 丁忧期间和候补等待期不计入俸年 |

### 4.2 候补制度

| 项目 | 内容 |
|------|------|
| 产生原因 | 官缺少于待选官员；官员丁忧、致仕、降罚等产生缺位 |
| 等待方式 | 候缺官员在京城或原籍等待吏部分配 |
| 经济压力 | "京城居之不易"，候补期间无俸禄，开销全靠自费 |
| 晚明恶化 | 万历以后官缺壅塞严重，候补时间越来越长 |

### 4.3 破格提拔的案例

| 案例 | 说明 |
|------|------|
| 洪武年间荐举 | 朱元璋时期通过荐举科目（聪明正直、贤良方正等）由地方官举荐人才破格录用 |
| 张居正的快速崛起 | 嘉靖二十六年进士 → 庶吉士 → 编修 → 经筵讲官 → 入阁，升迁速度远超常规 |
| 军功升迁 | 边疆战事中因军功获得跳级提拔（如杨嗣昌以兵部侍郎加尚书衔） |
| 传奉官 | 皇帝绕过吏部直接以圣旨升授，属于非常规但合法的通道 |

[来源：故宫博物院赵晶论文"传奉官研究"；维基百科"张居正"条目]

## 七、品级与实权不对应的情况

明代官制中存在显著的"品级与实权不对应"现象，一些品级较低的官职拥有远超其品级的实际权力：

| 官职 | 品级 | 实际权力 | 说明 |
|------|------|---------|------|
| 内阁大学士 | 正五品（本官） | 实际上的宰相 | 通过"票拟"掌握国家决策权，通常加尚书衔（正二品）以提升地位 |
| 六科给事中 | 从七品 | 可封驳圣旨、弹劾百官 | "位卑权重"的典型，参与官员选拔、御前会议、审理案件 |
| 监察御史 | 正七品 | 可弹劾包括王公贵族在内的任何官员 | 十三道巡按代天巡狩，权力极大 |
| 巡抚 | 无固定品级 | 统管一省军政 | 差遣官，以加都御史衔确定地位，实际权力远超布政使（从二品） |
| 总督 | 无固定品级 | 统管数省军政 | 加兵部尚书/侍郎兼都御史衔 |

**典型案例：**
- 嘉靖至崇祯间，夏言、严嵩、张居正等内阁首辅的实际权力相当于宰相，但大学士本官品级仅正五品
- 天顺二年（1458）后形成"非进士不入翰林，非翰林不入内阁，南北礼部尚书侍郎及吏部右侍郎非翰林不任"的惯例
- 袁崇焕以三甲进士出身，不到十年便升至督师尚书——这是战时体制下的特例

[来源：维基百科"明朝官职表"、知乎"明朝官职"、澎湃新闻"翰林院为何成为大明官员进步的快车道"]

## 八、方面官"小转法"

"小转法"是明代中后期方面官（布政司、按察司的参政、参议、副使、佥事）的逐级迁转惯例：

| 原职 | 升迁目标 | 品级变化 |
|------|---------|---------|
| 正五品佥事 | 从四品参议 | 升一级 |
| 从四品参议 | 正四品副使 | 升一级 |

- 此惯例打破了之前"方面官每升二级"的做法
- 大量出现始于正德初年刘瑾当政时期
- 原因：方面官处于内外中低级官员晋升高级京堂官的枢纽位置，官缺有限但官员众多
- 明中后期保举制下，部分方面官不满九年即得保举升迁，甚至超升三四级
- 为防止夤缘速升，明廷多次规定保举官员的在任时长——以两考（六年）或一考（三年）为限

[来源：参考网/余璐"明代方面官的迁转惯例——以小转法为中心的考察"]

## 九、升迁年限的具体数据

### 各路线升迁估计年限

| 路线段 | 估计年限 | 说明 |
|--------|---------|------|
| 庶吉士 → 编修 | 3年 | 散馆后直接留馆 |
| 编修 → 侍读/侍讲 | 6-9年 | 需经考满 |
| 侍读学士 → 入阁 | 视皇帝信任 | 可能很快，也可能数十年 |
| **翰林路线总计** | **约15-25年至入阁** | 快者如张居正约20年 |
| 知县 → 知府 | 9-15年 | 通常需经同知/通判过渡 |
| 知府 → 布政使 | 9-20年 | 部分需经分守/分巡道 |
| 布政使 → 巡抚 | 数年 | 需廷推 |
| **地方路线总计** | **约25-35年至巡抚** | 张学颜25年从知县升至户部尚书，属较快 |
| 御史/给事中 → 少卿 | 6-9年 | 科道官升迁相对较快 |
| 少卿 → 侍郎 | 6-12年 | — |
| 侍郎 → 尚书 | 3-9年 | — |
| **科道路线总计** | **约20-30年至尚书** | — |

**关键结论：**
- 高官职位（侍郎以上）通常由任职**二十至三十年**的官员担任
- 任职**十四年以下**的官员几乎不可能获得高官职位（翰林出身的极少数例外）
- 方面官任职往往超过九年方能考满升迁，有的布政使、知府达**二十余年**才升迁

### 典型案例

| 人物 | 出身 | 起始官职 | 终点官职 | 总年限 | 说明 |
|------|------|---------|---------|--------|------|
| 张学颜 | 三甲第156名 | 知县 | 户部尚书 | 约25年 | 嘉靖至万历，属较快 |
| 袁崇焕 | 三甲 | 知县 | 督师尚书 | 不到10年 | 战时特例，非常规升迁 |
