# 回避与丁忧

## 概述

明朝建立了系统的任官回避制度，包括地域回避（籍贯回避）、亲属回避和职务回避三大类，目的是防止官员利用地缘、血缘关系徇私舞弊，强化中央集权。丁忧制度要求官员在父母去世后离职守丧27个月，是礼法体系的核心制度之一。夺情（即在特殊情况下免除或缩短丁忧）则是丁忧制度的例外条款，万历五年（1577）张居正夺情案是明朝政治史上最具争议的事件之一。

---

## 一、地域回避（籍贯回避）

### 1.1 基本原则

明朝实行"自学官外，不得官本省"的根本原则——除学校教官外，官员一律不得在本省任职。

[来源：百度百科"古代回避制度"条目；人民论坛网"明清任官要遵循哪些回避制度"]

### 1.2 制度沿革

| 时间 | 规定内容 |
|------|----------|
| 洪武四年（1371） | 吏部铨选"南北更调"，定为常例——"南人官北，北人官南" |
| 洪武十三年（1380） | 将全国地方官任用划分三大互调区域（详见下表） |
| 洪武二十六年（1393） | 重新界定籍贯回避：除僧道、阴阳医士原籍任用外，其余官吏必须避籍 |
| 此后 | 更加明确，"自学官外，不得官本省" |

[来源：腾讯评论"籍贯回避：中国古代的异地为官"；湘潮网"中国古代任官回避制度"]

### 1.3 洪武十三年的三大互调区域

| 区域 | 官员籍贯 | 任职地域 |
|------|----------|----------|
| 第一区 | 北平、山西、陕西、河南、四川 | 浙江、江西、湖广、直隶（南直隶） |
| 第二区 | 浙江、江西、湖广、直隶 | 北平、山东、山西、陕西、河南、四川 |
| 第三区 | 广东、广西、福建 | 山东、山西、陕西、河南、四川 |

**惩罚性任用规则：** 考核不称及降谪者，不分南北，一律发往广东、广西、江西龙南和安远、湖广郴州之地任用，以示惩戒。

[来源：湘潮网"中国古代任官回避制度"；人民论坛网"明清任官要遵循哪些回避制度"]

### 1.4 五百里回避

北直隶（京畿地区）官员的特殊规定：任官须离本籍五百里以上，称为"避籍"。这一规定主要适用于京畿周边地区，防止官员在距家乡过近的地方任职。

[来源：人民论坛网"明清任官要遵循哪些回避制度"]

## 二、亲属回避

### 2.1 基本规则

| 回避类型 | 具体规定 |
|----------|----------|
| 父子/兄弟回避 | 父子、兄弟不得同时在同一省为官 |
| 翁婿回避 | 翁婿不得在同一省为官 |
| 同衙门回避 | 有亲属关系者不得在同一衙门任职 |
| 京官与王府 | 京官严禁与王府结亲；若已有姻亲关系，京官须外调地方 |
| 大臣科道限制 | "大臣之族不得任科道"——重臣的亲属不得担任给事中和监察御史 |

[来源：人民论坛网"明清任官要遵循哪些回避制度"；湘潮网"中国古代任官回避制度"；新浪博客"明代任官回避制度"]

### 2.2 制度目的

- 防止同族官员在同一辖区内相互包庇
- 防止科道言官因亲属关系而不敢弹劾高官
- 防止京官通过王府姻亲关系介入藩王政治

## 三、职务回避

### 3.1 具体规定

| 回避类型 | 具体规定 |
|----------|----------|
| 户部钱粮回避 | 户部官员不得管理本省钱粮事务 |
| 巡回监察回避 | 巡回监察官须回避原籍地、曾任官之地、寓居之地 |
| 铨选回避 | 吏部铨选官员时，与候选人有亲属关系者须回避 |
| 司法回避 | 审理案件涉及本籍或亲属时须回避 |

[来源：人民论坛网"明清任官要遵循哪些回避制度"；人民网"中国古代的人事回避制度"]

### 3.2 回避制度的执行与限制

- 明代回避制度在前期执行较为严格
- 中后期因官缺紧张、人才匮乏，执行有所松弛
- 但基本框架始终维持，是明清两代共同遵循的任官原则
- 学官（教授、教谕等）和阴阳、医术人员是明确的例外——允许原籍任用

## 四、丁忧制度

### 4.1 基本规则

| 项目 | 内容 |
|------|------|
| 适用范围 | 全体文武官员 |
| 丁忧对象 | 父母去世（洪武规定：仅限父母及祖父母承重者，"其余期年服制不许奔丧"） |
| 守丧期限 | 27个月（三年之丧，实际按27个月计算） |
| 丁忧期间 | 离职回籍守丧，不得参与政事，不得着华服、赴宴会、嫁娶 |
| 隐匿不报 | 匿丧不报者，一经查实即予革职，且永不叙用 |

[来源：百度百科"丁忧"条目；知乎"宋、明、清代丁忧制度"]

### 4.2 丁忧期间的待遇

| 项目 | 内容 |
|------|------|
| 俸禄 | 丁忧期间停发俸禄 |
| 年资 | 丁忧期间不计入俸年，直接影响升迁速度 |
| 复职 | 丧期满后须经吏部重新铨选候补，称"服阕起复" |
| 候补问题 | 起复后仍需候缺，等待时间不定 |

### 4.3 武官的特殊规定

| 项目 | 内容 |
|------|------|
| "金革之事不避" | 古制中武官在战时可不丁忧，称"墨绖从戎" |
| 明代实践 | 武官丁忧制度较文官灵活，边防将领在战时常获夺情 |
| 边疆巡抚丁忧 | 嘉靖中期以后，因"南倭北虏"，边疆巡抚丁忧须正式交代才可离任 |

[来源：知乎"宋、明、清代丁忧制度"；百度百科"丁忧"条目]

## 五、夺情制度

### 5.1 基本含义

"夺情"即"夺去父子之情"——国家因特殊需要，命丁忧中的官员提前复职或根本不令离职，免除其守丧义务。

### 5.2 夺情的适用条件

| 条件 | 说明 |
|------|------|
| 军事紧急 | 边疆战事紧迫，武将或统帅不可缺位 |
| 政务特殊需要 | 皇帝认为某重臣不可或缺，政务不能离开 |
| 皇帝特旨 | 夺情必须由皇帝亲自下旨，不能由官员自行决定 |

### 5.3 夺情案例

#### 案例一：万历五年张居正夺情案（1577）

这是明朝最著名的夺情案，也是万历朝政治史上的标志性事件。

**事件经过：**

| 时间 | 事件 |
|------|------|
| 万历五年九月 | 张居正父亲张文明病逝 |
| 第一次上疏 | 张居正上奏请求回乡守制，万历帝以"深切依赖"为由拒绝 |
| 第二次上疏 | 张居正再次请求，表示"臣以二十七月报臣父，以终身事皇上"；万历帝再次拒绝 |
| 第三次上疏 | 张居正第三次上书；万历帝回复："连日不得面卿，朕心如有所失，七七之期犹以为远，矧回三年" |
| 支持方观点 | "皇帝冲龄，天下不可一日无首辅" |
| 反对方观点 | "孝道为首"——"国而非此不可以为国，家而非此不可以为家，人而非此则禽兽矣" |

**反对夺情的上疏者及其结局：**

| 姓名 | 官职 | 上疏内容 | 处罚 |
|------|------|----------|------|
| 吴中行 | 翰林院编修 | 首先上疏谏止夺情 | 廷杖六十，逐出京城 |
| 赵用贤 | 翰林院编修 | 次日上疏谏止 | 廷杖六十，逐出京城 |
| 艾穆 | — | 与沈思孝联名上疏 | 廷杖八十，镣梏收监三日，遣戍凉州 |
| 沈思孝 | — | 与艾穆联名上疏 | 廷杖八十，镣梏收监三日，遣戍神电卫 |
| 邹元标 | 进士 | 四人受杖次日为其力争 | 廷杖（一腿被打断，终身残疾） |

**廷杖的惨烈程度：** 吴中行、赵用贤受杖后"腐肉数十脔，大者盈掌，深至寸"。

**营救未果：** 翰林院侍讲赵志皋、张位、于慎行、张一桂、田一俊、李长春，修撰习孔教、沈懋学等均上疏求解，但均被阻隔未能呈上。学士王锡爵会同词臣数十人向张居正求情，未获接受。

**万历帝的定性：** 万历帝颁下圣旨，指责反对夺情者是"借纲常之说，肆为排挤之意"。

**事件的政治影响：**
- 五人正直之名震动天下
- 张居正虽得以留任继续改革，但政治形象受到严重损害
- 此事成为张居正死后被清算的重要口实之一
- 反映了礼法与政治实用主义之间的深刻矛盾

[来源：知乎"以张居正夺情为切入点，说说明朝大臣丁忧那些事儿"；知乎"明代万历五年的一场夺情风波"；维基百科"张居正"条目；新浪博客"张居正主政因夺情一事而愤怒，廷杖五人"]

#### 案例二：崇祯朝杨嗣昌夺情案（1636）

| 项目 | 内容 |
|------|------|
| 时间 | 崇祯九年十月（1636） |
| 背景 | 后金入寇、兵部尚书张凤翼畏罪自杀，兵部无人主持 |
| 当事人 | 杨嗣昌，时正在母丧守制 |
| 夺情 | 崇祯帝下旨夺情，命杨嗣昌接任兵部尚书 |
| 杨嗣昌态度 | 三疏请辞，崇祯帝不许 |
| 到任 | 崇祯十年（1637）三月抵京赴任 |
| 反对者 | 少詹事黄道周攻讦最力，崇祯帝于七月五日召御前辩论 |
| 辩论焦点 | 黄道周以纲常伦理反对夺情；杨嗣昌则抓住黄道周曾为"杖母烝妾"的郑鄤辩护一事反击 |
| 结果 | 崇祯帝袒护杨嗣昌，贬斥黄道周等弹劾者 |

杨嗣昌夺情的真正争议并非单纯的礼法问题——反对者实际上反对的是他主导的对清和议政策。

[来源：百度百科"杨嗣昌"；维基百科"杨嗣昌"；知乎"兵部尚书：晚明最危险的官职"]

#### 明朝可夺情的几类人

| 类别 | 说明 |
|------|------|
| 武将 | 军事将领因军务紧急可夺情，"金革之事不避" |
| 特殊岗位人才 | 御医、钦天监官员等不可替代的技术官员 |
| 与战事相关的文官 | 非武将但参与军事指挥的巡抚、总督 |
| 百姓请命者 | 主政一方的父母官，百姓自发为其请命夺情（极少见） |

**钦天监官员的特殊规定：**
- 丁忧亦不开缺（不注销原职）
- 仅给假百日
- 穿孝二十七个月，停其升转，扣资俸三个月
- 原因：天文历法事务不可中断，人才稀缺

[来源：百度百科"夺情起复"；知乎"以张居正夺情为切入点"]

#### 匿丧的法律后果

| 违规行为 | 处分 |
|----------|------|
| 匿丧不报 | 发原籍为民，永不叙用（最严厉） |
| 居丧嫁娶 | 杖一百，革职 |
| 居丧作乐杂戏 | 杖八十 |
| 释服从吉（提前脱丧服） | 杖八十 |

匿丧的高风险使其成为极少数官员的冒险行为。正统七年（1442）定制："凡官吏匿丧者，俱发原籍为民。"

[来源：知乎"宋明清代丁忧制度"、百度百科"丁忧"]
