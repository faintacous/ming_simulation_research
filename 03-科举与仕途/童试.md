# 童试（县试、府试、院试）

## 概述

童试是明代科举制度的最基层考试，也称"童生试"，包括县试、府试、院试三个阶段。通过全部三阶段考试者获得"生员"（秀才）资格，入府、州、县学读书。童试不限年龄，凡未取得生员资格者，无论老幼均称"童生"。

---

## 一、县试

### 1.1 时间
- 每年二月举行
- 黎明前点名入场，当日交卷

### 1.2 地点
- 本县县衙或县学考棚

### 1.3 考官
- **知县**主持，为主考官

### 1.4 报名与保结
- 童生须向本县衙署**礼房**报名
- 填写父母、祖父母、曾祖父母三代存殁、已仕未仕之履历
- **互结**：五名童生互相担保，保证身家清白、无冒籍、匿丧、顶替、假捏姓名等弊，若一人舞弊则"五人连坐"
- **保结**：须有本县**廪生一名**作保，保证考生不属于优倡隶皂之子孙

### 1.5 考试场次与内容

| 场次 | 内容 |
|------|------|
| 第一场（正场） | 四书文二篇、五言六韵试帖诗一首 |
| 第二场 | 四书文一篇 |
| 第三场 | 经论（五经文） |
| 第四场 | 律赋或诗 |
| 第五场 | 策论一道（有的县只考四场） |

- **第一场为"正场"**，最为关键，第一场录取者即准许应府试
- 后续各场是否参加由考生自便
- 每日一场，黎明入场，即日交卷

### 1.6 录取
- 县试录取名额无统一定制，由知县视报考人数酌定
- 录取者称"县试案首"（第一名）
- 通过县试方可参加府试

---

## 二、府试

### 2.1 时间
- 一般在县试之后，约在四月举行

### 2.2 地点
- 各府府城

### 2.3 考官
- **知府**主持

### 2.4 报名
- 通过县试者方可报名
- 报名流程与县试类似，须增加一名廪生作保（即需两名廪生保结）
- 仍需五人互结

### 2.5 考试内容
- 场次与县试类似，亦为四至五场
- 内容同样包括四书文、试帖诗、经论等
- 第一场仍为正场，考四书文

### 2.6 录取
- 通过府试者取得**"童生"**正式资格
- 府试第一名称"府案首"
- 取得童生资格后，方可参加院试

---

## 三、院试

### 3.1 时间
- 学政到任后按临各府、州，巡回考试
- 一般一年或两年举行一次

### 3.2 地点
- 各府府城，学政巡回至该府时举行

### 3.3 考官
- **提学官**（学政/提学御史/提学副使）主持
- 明代由提学官主持，学政品级为按察司副使或佥事（正四品或正五品）

### 3.4 考试内容

| 场次 | 内容 |
|------|------|
| 经古（加试，正场之前） | 经解、史论、诗赋 |
| 正场 | 四书文二篇、五经文一篇 |
| 复试 | 视情况加试 |

### 3.5 录取与学额

通过院试者取得**生员（秀才）**资格，入府、州、县学。

#### 明代学额（府、州、县学生员定额）

| 学校等级 | 廪膳生名额 | 增广生名额 |
|----------|-----------|-----------|
| 京府学 | 60名 | 60名 |
| 外府学 | 40名 | 40名 |
| 州学 | 30名 | 30名 |
| 县学 | 20名 | 20名 |

- **附学生**无名额限制，成化以后大量扩招
- 初入学者皆为**附生**，在岁考、科考中按成绩递补增生、廪生

---

## 四、生员等级：廪生、增生、附生

### 4.1 三等分类

| 等级 | 全称 | 说明 |
|------|------|------|
| 廪生 | 廪膳生员 | 诸生之首，由官府供给膳食，每年发**廪饩银四两**（明后期折银）；月给米六斗（明初） |
| 增生 | 增广生员 | 定额增加的生员，不享受膳食供给 |
| 附生 | 附学生员 | 于廪生、增生外再增名额，附于诸生之末，无名额限制 |

### 4.2 递补规则
- 初入学者均为**附生**
- 通过岁考成绩优异者，依次递补为增生、廪生
- 廪生有缺，由增生中优异者递补；增生有缺，由附生递补

### 4.3 廪生的特殊职责
- 须为应考的童生具结保证（保结制度）
- 保证童生无身家不清及冒名顶替等弊

---

## 五、岁考六等制度

### 5.1 考试性质
- 岁考（岁试）由**学政**主持
- 约每**两年**举行一次（学政任期三年，巡回各府考试）
- 目的：考核在学生员的学业水平，据以升降

### 5.2 六等评定与待遇差异

| 等级 | 待遇 |
|------|------|
| **一等**（前列） | 视廪膳生有缺，依次充补廪生；一等给赏（赏银、纸笔等） |
| **一等**（后列） | 补增广生；给赏 |
| **二等** | 给赏；具备参加科考资格 |
| **三等** | 如常，不升不降 |
| **四等** | 挞责（笞罚） |
| **五等** | 廪生降为增生，增生降为附生，附生降为**青衣**（罚穿青衣以示惩戒） |
| **六等** | **黜革**（革除生员资格，沦为平民） |

### 5.3 科考
- 科考是乡试前的**甄别性资格考试**，由学政主持
- 同样采用六等法评定，但实际操作中三等以下极少
- 科考**一、二等者**方准送乡试，称为"科举生员"
- 考试内容：四书义、五经义，兼有论、策、表等

## 六、考试的具体防弊措施

### 6.1 报名阶段的防弊

| 措施 | 具体内容 |
|------|---------|
| **五人互结** | 同考童生五人互相担保，保证身家清白、无冒籍、匿丧、顶替等弊，若一人舞弊则五人连坐 |
| **廪生作保** | 县试须一名廪生具结，府试须两名廪生具结，保证考生不属于优倡隶皂之子孙 |
| **亲供核验** | 考生须填写三代履历、体貌特征，入场时由门吏核对，防止冒名顶替 |
| **排斥贱籍** | 娼优、皂吏子孙、从事"贱业"者一律不准报考 |
| **禁止冒籍** | 考生须为本籍良民，冒充他县他省籍贯者查出严惩 |

### 6.2 考场的防弊

| 措施 | 具体内容 |
|------|---------|
| **搜检入场** | 入场前搜身检查衣物文具，防止夹带缩微书籍或纸条 |
| **对号入座** | 考前张榜公布座位编排，考生按号入座 |
| **巡逻** | 差役在考场内巡逻，禁止交头接耳、传递纸条 |
| **限时交卷** | 日落前须交卷，未完成者给烛三支，烛尽扶出 |

### 6.3 院试的特殊防弊

| 措施 | 具体内容 |
|------|---------|
| **学政外派** | 学政由朝廷从京官中选任，三年一任，与本地无利益勾连 |
| **巡回考试** | 学政巡回各府主持院试，不在某地久留 |
| **加试经古** | 正场前加试经古一场（解经、史论、诗赋），增加多维度考核 |
| **六等黜革** | 已入学生员须接受岁考，六等者革除功名，确保淘汰 |

## 七、应试人数规模

- 明代中后期，各县应童试的童生通常在**数百至数千人**不等
- 大县可达二三千人，小县亦有四五百人
- 生员总数：据研究者估算，明代全国生员总数约在**3万至5万人**之间

## 存疑 & 待查

1. 明代各府县具体的院试录取人数（每次院试实际录取多少人）缺乏系统数据，各地差异较大
2. 县试、府试的具体场次安排在明代与清代是否完全一致，部分文献未严格区分
3. 岁考"给赏"的具体赏赐物品和金额，各时期有变化，需进一步查证
4. 附生数量在成化以后急剧膨胀的具体数据待查
5. 明代廪饩银的折算标准和实际发放情况需更多地方志数据
