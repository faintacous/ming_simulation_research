# 士绅阶层

> 聚焦时段：嘉靖至崇祯（1521-1644）
> 核心议题：功名特权体系、优免制度演变、地方社会角色、避税手段

---

## 一、功名等级与对应特权

### 1.1 明代功名层级

明代科举体系形成三级功名制度，每一级对应不同的社会特权：

| 功名等级 | 获取方式 | 社会称谓 | 特权概览 |
|---------|---------|---------|---------|
| 生员（秀才） | 通过院试 | 相公、秀才 | 免本人徭役、见官不跪、不受刑讯 |
| 举人 | 通过乡试 | 老爷 | 免徭役、可直接做官（候缺）、社会地位显著提升 |
| 进士 | 通过会试殿试 | 老爷/大人 | 授官、全面优免特权 |

**来源**：[明代的举人，秀才，分别可以免除多少赋税和徭役？ - 知乎](https://www.zhihu.com/question/290134594)

### 1.2 功名特权具体清单

**生员特权**：
- 免本身徭役及户内二丁差役（洪武年间定例）
- 见官不行跪拜礼
- 不受笞杖刑（须先革去功名方可用刑）
- 穿戴儒巾、青色襕衫（身份标识）
- 可使用"生员之家"门牌，享受社会尊重

**举人特权**：
- 享有做官资格，可候补知县等中低级官职
- 优免徭役范围扩大
- 在地方上可以直接面见知县，参与地方事务
- 家族整体社会地位提升，产生"中举效应"

**进士特权**：
- 立即授官（一甲翰林院，二三甲分配各部或地方）
- 全面享受官员级别的优免特权
- 致仕后仍保留大部分优免

**来源**：[明朝官绅是否真有不纳税的特权？- 腾讯新闻](https://news.qq.com/rain/a/20250501A0624P00)

### 1.3 重要澄清：优免的实际范围

明代士绅优免的对象是**杂役**（徭役中的差役部分），而非田赋。士绅仍然需要缴纳田赋（税粮），但可以免除各种差遣和劳役。里甲正役原则上也不在优免范围之内。

**来源**：[明朝徭役简述，官绅是否可以免役 - 腾讯新闻](https://news.qq.com/rain/a/20250911A055XO00)

## 二、嘉靖二十四年《优免则例》

### 2.1 则例全文主要内容

嘉靖二十四年（1545年），明廷正式颁布《优免则例》，对各级官员和功名持有者的优免标准作出系统规定：

**京官优免标准**：
| 品级 | 免粮（石） | 免丁（丁） |
|------|-----------|-----------|
| 一品 | 30石 | 30丁 |
| 二品 | 24石 | 24丁 |
| 三品 | 20石 | 20丁 |
| 四品 | 16石 | 16丁 |
| 五品 | 14石 | 14丁 |
| 六品 | 12石 | 12丁 |
| 七品 | 10石 | 10丁 |
| 八品 | 8石 | 8丁 |
| 九品 | 6石 | 6丁 |

**外官**：各减京官的一半。

**教官、监生、举人、生员**：各免粮二石、人丁二丁。

**杂职省祭官承差知印吏典**：各免粮一石、人丁一丁。

**致仕官**：以礼致仕者，免本品十分之七；闲住者，免一半。犯赃革职者不在优免之例。

**来源**：[明代的举人，秀才，分别可以免除多少赋税和徭役？ - 知乎](https://www.zhihu.com/question/290134594)；[明朝官绅是否真有不纳税的特权？- 腾讯新闻](https://news.qq.com/rain/a/20250501A0624P00)

### 2.2 "免粮"与"免丁"的含义

此处的"免粮"并非免除田赋本身，而是免除与该粮额对应的**杂役负担**。"免丁"同理，指免除该人丁数对应的差役派征。明代的徭役派征既按人头也按粮额计算，因此优免以粮和丁两个维度同时规定。

**来源**：[明代徭役优免类型概说 - 故宫博物院](https://www.dpm.org.cn/Uploads/File/2020/02/16/u5e4921428f4e4.pdf)

## 三、万历三十八年《优免新例》的变化

### 3.1 新例核心内容

万历三十八年（1610年），朝廷颁布《优免新例》，以**田亩数**取代旧例的粮石和人丁标准，反映了一条鞭法推行后的赋役制度变化：

**现任京官优免田亩**：
| 品级 | 优免田（亩） |
|------|------------|
| 一品 | 10,000亩 |
| 二品 | 8,000亩 |
| …… | 递减 |
| 八品 | 2,700亩 |

**外官**：减京官一半。

**致仕官**：免本品十分之六。

**未仕功名持有者**：
| 身份 | 优免田（亩） |
|------|------------|
| 未仕进士 | 最高3,350亩 |
| 未仕举人 | 1,200亩 |
| 生员、监生 | 80亩 |

### 3.2 与嘉靖则例的关键变化

1. **计量单位转变**：从"粮石+人丁"改为"田亩数"，适应一条鞭法后以银代役的新体制
2. **优免额度大幅扩大**：万历新例的优免范围远超嘉靖旧例，未仕进士从2石2丁扩大到3,350亩
3. **制度性漏洞加深**：优免田亩数的大幅增加，为士绅阶层的土地兼并提供了更大的制度空间

**来源**：[在明朝，有功名的读书人需要交税吗？ - 知乎](https://zhuanlan.zhihu.com/p/106898629)；[明代的举人，秀才，分别可以免除多少赋税和徭役？ - 知乎](https://www.zhihu.com/question/290134594)

## 四、士绅在地方社会的角色

### 4.1 调解与司法

- 士绅是地方纠纷的主要调解者，虽然从国家法角度不具备正式司法权，但在实际操作中承担了大量民间调解职能
- 乡约制度是士绅教化和调解的重要工具，其中王守仁（正德年间）制定的《南赣乡约》影响深远
- 绝大多数乡村纠纷通过家法族规在宗族内部解决，士绅族长往往是最终裁决者

**来源**：[第二章 绅权对国家权力的制度分享 - 知乎](https://zhuanlan.zhihu.com/p/680665632)

### 4.2 慈善与公益

- **义庄义田**：士绅家族设立义庄、义田，用于救济贫困族人和乡邻
- **灾荒赈济**：地方遭遇灾荒时，士绅常组织或主导赈济活动，包括开设粥棚、平粜粮食
- **社仓制度**：以乡约为中心的保甲、社仓制度成为明代中后期基层治理和慈善救助的主要内容

**来源**：[明代乡村慈善模式的制度设计和基层组织慈善功能的演变](https://www.sky.yn.gov.cn/UpLoadFiles/File/2024-03-05/YY2MPOUE9T%E5%91%A8%E3%80%80%E8%8D%A3.pdf)

### 4.3 修桥铺路与公共工程

- 士绅主导地方的**修桥铺路、兴修水利、筑城卫民**等公共事务
- 兴建书院、善堂、义庄等文教和慈善设施
- 修缮路桥、组织庙会、修建公祠
- 福建科举家族在地方公共事务中尤为活跃

**来源**：[漫谈江南士绅 - 知乎](https://zhuanlan.zhihu.com/p/578280438)；[旌表义民与明代的乡村治理](https://xbgjxt.swu.edu.cn/article/id/5f44d655ee641d795a4ed5ab)

### 4.4 士绅与地方官的互动

- 士绅是地方官推行政令的重要中介和合作者
- 地方官新到任时通常需要拜访当地有影响力的士绅
- 士绅可以直接面见知县提出建议或反映问题
- 但也存在士绅阻挠政令、与地方官对抗的情况

## 五、避税手段的具体操作方式

### 5.1 投献

**定义**：百姓将自己的田产名义上转移到有功名的士绅名下，以借用士绅的优免特权逃避赋役。

**具体操作**：
1. 百姓与士绅达成私下协议
2. 百姓在官府的田产登记中将自己的田地改为士绅名下
3. 百姓向士绅缴纳一定的"挂名费"（远低于应缴税额）
4. 士绅名下田产增加但不超过优免额度，或即使超过也因身份特权而能减免
5. 实际耕种和收益仍归原百姓，但赋役负担大为减轻

**规模**：士绅中举后，往往引来大量百姓前来投献，使士绅名下田产急剧膨胀。这种现象在嘉靖至万历间尤为普遍。

**来源**：[明清士绅们的快速发财法——投献 - 知乎](https://zhuanlan.zhihu.com/p/580516376)

### 5.2 诡寄

**定义**：富户将名下的田产分割，假托在多个小户名下，从而降低自己的户等，减少赋役负担。

**具体操作**：
1. 大地主将大量田产拆分成小份
2. 分别登记在亲属、佃户或虚拟人名之下
3. 每个"小户"的田产不多，被评为下等户，赋役负担轻
4. 实际控制权仍在大地主手中

**变体**：
- **铁脚诡寄**：将田产固定分散在多个假名下
- **通天诡寄**：形成地方性的惯例和风俗，官府难以追查

**来源**：[官绅不纳粮只是明朝的制度还是说从古至今都是如此？ - 知乎](https://www.zhihu.com/question/62116661)

### 5.3 飞洒

**定义**：将自己应缴的赋税负担转嫁到其他无力抗辩的小户名下。

**具体操作**：
1. 富户勾结里书（负责编制赋役册籍的基层人员）
2. 在编制黄册时，将自己名下的田产或丁口数目减少
3. 将减少的部分"飞洒"到其他小户名下
4. 小户突然发现自己的赋役负担增加，但难以查清原因
5. 受害者往往是无力抗辩的普通农户

### 5.4 花分与影射

- **花分**：将田产在册籍上做细碎的拆分和重组，使追查困难
- **影射**：利用同姓或相似地名的混淆，将赋役转嫁他人

### 5.5 其他手段

- **拖欠**：直接拖延不缴税赋，依靠士绅身份使催征困难
- **隐匿田亩**：在黄册和鱼鳞图册中隐瞒实际拥有的田产
- **买富差贫**：通过贿赂里书和差役人员，将本该由富户承担的差役转派给贫户

### 5.6 避税手段的社会影响

- 导致"田多者反居下役"的荒谬现象
- 赋役负担日益集中在无特权的中小农户身上
- 加剧了贫富分化和社会矛盾
- 国家财政收入持续萎缩
- 成为明末社会危机的重要根源之一

**来源**：[中国明末的阶级斗争与财政史（上） - 知乎](https://zhuanlan.zhihu.com/p/103978984)；[大明朝廷真的给士绅们免除赋税吗？ - 知乎](https://www.zhihu.com/question/57749523)

## 六、士绅阶层的规模与分布

### 6.1 数量估算

明代生员总数约在数十万人的规模：
- 全国府州县学生员名额合计约3-5万人
- 加上历年累积的未仕举人和致仕官员，士绅阶层总人数可达数十万
- 若计入其优免覆盖的家庭成员，影响人口可达数百万

### 6.2 地域分布

- 江南地区（南直隶、浙江、福建）士绅密度最高
- 科举发达省份的士绅对地方社会的影响更为深远
- 北方士绅数量相对较少，但在地方社会中同样扮演重要角色

## 七、制度层面 vs 实际执行

| 维度 | 制度规定 | 实际情况 |
|------|---------|---------|
| 优免范围 | 仅限杂役 | 通过各种手段事实上免除了大部分赋役 |
| 优免额度 | 有明确限额 | 限额成为摆设，实际优免远超规定 |
| 投献禁令 | 朝廷多次下令禁止 | 屡禁不止，愈演愈烈 |
| 里甲正役 | 不在优免范围 | 士绅家庭常以各种方式规避 |
| 田赋缴纳 | 士绅仍须缴纳 | 拖欠严重，催征困难 |
