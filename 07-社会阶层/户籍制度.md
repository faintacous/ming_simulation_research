# 户籍制度

> 聚焦时段：嘉靖至崇祯（1521-1644），兼及制度建设背景（洪武）
> 核心议题：四大户籍义务、军户逃亡、灶户计丁办课、黄册编制程序、里甲编制、鱼鳞图册

---

## 一、四大户籍概述

### 1.1 制度框架

明代户籍按"本等名色占籍"的原则划分，不同职业的人占有不同的户籍，对国家承担不同的差役。主要分为四大类：

| 户籍类别 | 核心义务 | 管理机构 | 比例 |
|---------|---------|---------|------|
| 民户 | 纳粮当差（赋税和徭役） | 户部、府州县 | 占绝大多数 |
| 军户 | 出丁从军、屯田 | 兵部、都司卫所 | 约占总户数的20-25% |
| 匠户 | 承担造作（手工业劳役） | 工部 | 约30万户 |
| 灶户 | 煮盐办课 | 户部、盐运司 | 约4-5万户 |

### 1.2 户籍的世袭性

- 户籍身份**世代相传**，原则上不得改变
- "役皆永充"——一旦编入某类户籍，子孙必须继承
- 死若逃者则**勾补**——逃亡或死亡后从族内勾取亲属补充
- 这一制度在明代中后期逐渐松弛，但法律框架始终存在

## 二、民户的义务详述

### 2.1 赋税义务

**田赋（税粮）**：
- 按拥有的田亩数和田地等级征收
- 分为夏税（小麦等）和秋粮（稻谷等）
- 一条鞭法后折银征收
- 明末加派三饷，负担急剧加重

**杂税**：
- 各种名目的附加税
- 地方官员额外摊派
- "火耗"等征收损耗

### 2.2 徭役义务

**里甲正役**：
- 110户编为一里，十甲轮流应役
- 十年一轮，每年一甲当值
- 当值甲内十户和当年里长负责催征赋税、处理公务

**均徭**：
- 按丁粮多少派征
- 包括各种差遣：解运物资、充当皂隶、看管仓库等
- 分为"银差"和"力差"

**杂泛差役**：
- 临时性的劳役差遣
- 修筑城墙、堤坝、道路等
- 运送物资等

## 三、军户的义务详述

### 3.1 核心义务

**出丁从军**：
- 军户须**户出一丁**赴卫所当兵，称为"正军"（旗军）
- 须出一余丁随正军到营佐助，称为"军余"
- 户下须以一丁**供给在营正军**的日常需求
- 正军死亡或退役后，由军余或族内适龄男丁替补

**军事屯田**：
- 明初"几乎无军不屯"
- 军队屯田自养，是明初军饷的主要来源
- 屯军与守军的比例各地不同，通常为七分屯三分守或六分屯四分守

### 3.2 军户逃亡问题

**逃亡数据**：

| 时期 | 逃亡情况 | 来源 |
|------|---------|------|
| 正统三年（1438年） | 逃亡官军**1,633,664人** | 官方统计 |
| 嘉靖年间 | 部分卫所逃亡率达**70%** | 各地报告 |
| 16世纪初 | 一些卫所逃亡率达**80%** | 学者统计 |
| 嘉靖八年（1529年） | 桂萼公布全国兵籍仅**97万** | 与初期311万相比缺额巨大 |

**逃亡原因**：
1. 卫所劳役繁重，待遇恶劣
2. 军官侵占军屯田地，正军沦为私人佃户
3. 军官私役军士，军士不堪其苦
4. 军屯生产关系落后，收入微薄
5. 战争伤亡后勾补困难，族人逃避补役

**逃亡后果**：
- 卫所实际兵力远低于编制
- 军事战斗力急剧下降
- 明代中后期不得不以**募兵制**补充，增加军费支出
- 军户制度名存实亡

### 3.3 军屯的衰落

- 宣德以后，卫所官侵占军屯田地已成常态
- 正统年间，军屯收入大幅减少
- 军官私役军士耕种私田
- 朱元璋"养兵百万不费百姓一粒米"的理想彻底破灭
- 到崇祯年间，国家无力支付军饷，成为明亡的重要原因

## 四、灶户的义务详述

### 4.1 计丁办课制度

**核心含义**：灶户按照户内**丁口数量**承担食盐生产任务，完成规定的盐课（盐税）。

**具体标准**（以两淮盐场为例）：

| 指标 | 数值 |
|------|------|
| 每丁岁办盐 | 小引盐16引（另一标准为20引） |
| 每引重量 | 200斤 |
| 两淮29个盐场年产 | 705,180引 |

### 4.2 团煎制度

- 一场分**几团**，一团分**几户**
- 灶户轮流煎纳
- 按团组织生产，集中管理

### 4.3 灶户的免役

- 洪武十七年（1384年），两淮都运盐使司上言：灶户既验丁煎盐，若再承担其他徭役，会妨碍食盐年课
- 明太祖批示：**蠲其杂役，盐额如故**
- 灶户免除其他杂役，专注于煮盐
- 但实际中灶户仍常被摊派额外劳役

### 4.4 灶户的困境

- 盐场环境恶劣，劳动强度大
- 盐课标准固定，但自然条件变化影响产量
- 灶户世代为灶，不得脱籍
- 私自逃亡者受到严厉追惩
- 灶户的生活水平普遍低于普通民户

**来源**：[中华文史网 - 经济](http://www.historychina.net/jj/430116.shtml)；[明代灶户盐课标准 - 盐引与资本市场](http://economy.guoxue.com/?p=2832)

## 五、匠户的义务详述

### 5.1 匠户的分类

| 类别 | 管理机构 | 服役方式 |
|------|---------|---------|
| 轮班匠 | 工部 | 定期赴京轮值服役 |
| 住坐匠（民匠） | 内府内官监 | 在本地定期服役 |
| 住坐匠（军匠） | 各卫所 | 在军事机构服役 |

### 5.2 轮班匠的义务

**洪武十九年（1386年）定制**：
> 命工部"籍诸工匠，验其丁力，定以**三年为班**，更番赴京轮作三月，如期交代"

- 每三年轮一次，每次在京服役**三个月**
- 自行承担往返路费和在京期间的生活费用
- 服役期间无工资
- 全国轮班匠约**23万**人

### 5.3 住坐匠的义务

- 每月服役**十天**（"住坐之匠，例月上工十日"）
- 属内府内官监管理
- 除完成本职造作外，还需承担解运任务
- 工作繁重且无报酬

### 5.4 以银代役的制度变革

| 时期 | 变革内容 |
|------|---------|
| 成化二十一年（1485年） | 开始对轮班匠实行**输银代服役** |
| 嘉靖四十一年（1562年） | 普遍实行**"匠班银"**制度 |
| 制度影响 | 匠户以缴纳银两代替亲身服役，人身束缚松弛 |

**匠班银标准**：
- 轮班匠每年缴纳匠班银约**4钱5分**银
- 缴银后免除赴京服役
- 但匠户对政府的劳役义务名义上仍旧存在

### 5.5 匠户的实际处境

- 匠户地位在明代各户类中**最低**
- 匠役"永充"制度使其世代不得脱籍
- 明代中后期，随着以银代役的推行，人身依附有所松弛
- 但匠户身份的社会歧视仍然存在

**来源**：[轮班匠 - 维基百科](https://zh.wikipedia.org/zh-hans/%E8%BD%AE%E7%8F%AD%E5%8C%A0)；[匠籍 - 百度百科](https://baike.baidu.com/item/%E5%8C%A0%E7%B1%8D/1910528)；[匠户制 - 百度百科](https://baike.baidu.com/item/%E5%8C%A0%E6%88%B7%E5%88%B6/4327903)

## 六、黄册十年一造的编制程序

### 6.1 制度建立

- 洪武十四年（1381年），朱元璋命令全国各府县编制**赋役黄册**
- 从洪武十四年至崇祯十五年（1642年），共攒造**27次**
- 黄册送至南京**后湖黄册库**（玄武湖）保存
- 后湖黄册库近260年间共贮藏黄册**170多万册**，是当时世界上最伟大的档案馆之一

### 6.2 编制程序

#### 第一步：基层登记

- 以**户**为单位登记
- 甲首负责本甲十户的信息采集
- 里长汇总本里110户的信息

#### 第二步：分类登记

**户类分为**：
| 类别 | 定义 |
|------|------|
| 正管户 | 有赀产、应役者 |
| 畸零户（带管） | 鳏寡孤独、不应役者 |

#### 第三步：户头登记内容

每户首载：
1. 户长**姓名**
2. 所属**都图乡贯**
3. **户籍**类别（民/军/匠/灶）
4. **户等**
5. 应役**年份**

#### 第四步：四柱式记载

每户下分**人丁**和**事产**（田地房屋等）两大部分，按"四柱式"登载：

| 栏目 | 含义 |
|------|------|
| 旧管 | 上次造册时的原有数目 |
| 新收 | 本次新增的数目（新生人口、新购田产等） |
| 开除 | 本次减少的数目（死亡人口、出卖田产等） |
| 实在 | 当前实际数目（旧管 + 新收 - 开除） |

#### 第五步：汇总上报

```
甲首 → 里长 → 县衙 → 府衙 → 布政使司 → 户部
                                          ↓
                                    后湖黄册库（存档）
```

每级逐层汇总审核，最终送达南京后湖保存。

### 6.3 黄册的形制

- 以黄色封面装订，故称"黄册"
- 每里编为一册
- 册之首总为一图（里的总概况）
- 分配十年轮流应役的安排

### 6.4 黄册制度的衰败

- 明代中后期，黄册编造日益敷衍
- 里书（基层编造人员）作弊严重
- 人口和田产的实际变动得不到如实反映
- 后湖黄册库管理不善，大量黄册霉烂损毁
- 黄册逐渐失去实际管理功能

**来源**：[赋役黄册 - 百度百科](https://baike.baidu.com/item/%E8%B5%8B%E5%BD%B9%E9%BB%84%E5%86%8C/1050709)；[黄册 - 维基百科](https://zh.wikipedia.org/zh-hans/%E9%BB%83%E5%86%8A)；[探秘明朝"国家档案馆" - 澎湃新闻](https://m.thepaper.cn/baijiahao_16213556)

## 七、里甲110户编制

### 7.1 里甲制度的核心

**编制方式**：
- 以**110户**为一里
- 推选丁粮最多的**10户**为里长
- 其余**100户**分为**10甲**，每甲10户
- 每甲选一人为**甲首**

### 7.2 轮役制度

- 10个里长**十年一轮**，每年一人当值
- 当值里长与当年应役一甲的10户共同管理一里之事
- 里长和甲首的安排以**丁粮多少**为标准
- 出丁交粮最多者先轮到，最少者最后轮到

### 7.3 里长与甲首的职责

| 角色 | 主要职责 |
|------|---------|
| 里长 | 催征赋税、维持治安、调解纠纷、上传下达 |
| 甲首 | 协助里长、管理本甲十户 |

### 7.4 里甲制度的演变

- 洪武年间制度严密，执行有力
- 明代中期以后，里甲制度逐渐松弛
- 人口流动和土地兼并使原来的110户编制被打破
- 里长之役成为沉重负担，富户设法逃避
- 一条鞭法后，里甲正役逐渐被折银征收

## 八、鱼鳞图册与黄册的关系

### 8.1 鱼鳞图册的基本概念

- 鱼鳞图册是以**土地**为纲的土地登记簿册
- 因田地形状排列如鱼鳞而得名
- 详细记录每块田地的编号、面积、四至（东南西北邻界）、土质等级、所属业主

### 8.2 黄册与鱼鳞图册的互补关系

| 维度 | 黄册 | 鱼鳞图册 |
|------|------|---------|
| 以何为纲 | 以**人户**为母 | 以**田地**为母 |
| 以何为辅 | 以田为子 | 以人户为子 |
| 主要功能 | 定徭役、征赋税 | 稽查田亩编号四至 |
| 登记重点 | 户口、丁口、事产 | 田地形状、面积、等级 |
| 比喻 | 经 | 纬 |

### 8.3 互相印证

- 两册起着"一经一纬"的作用
- 如同编织成的一张大网，把百姓的人口和土地全部网罗其中
- "相须而不可无"——互相补充、缺一不可
- 黄册上的田产记录可用鱼鳞图册核实
- 鱼鳞图册上的业主信息可用黄册核对

### 8.4 鱼鳞图册的内容

每块田地的登记信息包括：
1. **字号**：田地的唯一编号
2. **亩分**：田地面积
3. **四至**：东南西北的邻界
4. **土质等级**：上、中、下等
5. **税额**：应缴税粮数目
6. **业主**：所有者姓名
7. **图形**：绘制田地形状（因此得名"鱼鳞"）

### 8.5 两册制度的衰败

- 明代中后期，两册都面临严重的失实问题
- 土地兼并导致实际业主频繁变动，图册更新不及时
- 人口流动使黄册中的户口信息与实际不符
- 基层编造人员（里书）的作弊和腐败
- 两册逐渐沦为形式，失去了核实户口和土地的实际功能

## 九、户籍制度的总体评价

### 9.1 制度设计的初衷

- 确保国家获得稳定的赋税收入和劳役供给
- 通过世袭分工实现社会的有序管理
- 控制人口流动，维持社会稳定

### 9.2 制度与实际的落差

| 维度 | 制度规定 | 嘉靖至崇祯实际 |
|------|---------|----------------|
| 户籍世袭 | 不得改变 | 大量逃户、脱籍 |
| 军户出丁 | 每户出一正军 | 逃亡率高达70-80% |
| 匠户服役 | 亲身赴京轮作 | 以银代役普遍化 |
| 黄册编制 | 十年精确造册 | 编造敷衍、数据失实 |
| 人口固定 | 不得擅自迁移 | 流民遍布各省 |
| 里甲编制 | 110户整齐编排 | 户口变动使编制混乱 |

### 9.3 制度演变的趋势

- **人身依附逐渐松弛**：以银代役使匠户、灶户的人身束缚减轻
- **户籍界限模糊化**：一条鞭法后，各类户籍的义务差异缩小
- **流动性增大**：尽管法律禁止，人口的实际流动不断增加
- **制度失效**：黄册和鱼鳞图册的管理功能逐渐丧失
