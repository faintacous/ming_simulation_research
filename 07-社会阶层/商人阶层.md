# 商人阶层

> 聚焦时段：嘉靖至崇祯（1521-1644）
> 核心议题：服饰限制、法律限制、捐纳制度、商籍、士商合流、商人实际社会地位变化

---

## 一、洪武服饰限制条文

### 1.1 核心条文

洪武十四年（1381年），朱元璋颁布明确的服饰等级规定：

> **"农衣绌、纱、绢、布。商贾止衣绢、布。农家有一人为商贾者，亦不得衣绌、纱。"**

### 1.2 条文解读

| 身份 | 可穿面料 | 禁止面料 |
|------|---------|---------|
| 农民 | 绌（绸）、纱、绢、布 | 锦绣 |
| 商人 | 绢、布 | 绌（绸）、纱及以上 |
| 农家有商人者 | 绢、布 | 绌（绸）、纱及以上 |

### 1.3 其他服饰限制

- 洪武三年（1370年）规定：商人不得穿锦绣，不得用金绣
- 商人及其家属不得乘马、不得穿靴
- 商人戴帽颜色有限制
- 违反者处以杖刑

### 1.4 执行与松弛

- 洪武至永乐年间执行较严
- 宣德以后逐渐松弛
- 嘉靖至万历年间，富商服饰已与士大夫无异，禁令名存实亡
- 晚明"服妖"之风盛行，等级服饰制度全面崩溃

## 二、商人法律限制清单

### 2.1 政治限制

| 限制内容 | 具体规定 | 说明 |
|---------|---------|------|
| 科举限制 | 洪武年间禁止商人子弟参加科举 | 后来逐渐放宽 |
| 仕途限制 | 商人不得为官 | 通过捐纳制度可绕开 |
| 参政限制 | 商人不得干预地方政事 | 实际中常被违反 |

### 2.2 社会限制

| 限制内容 | 具体规定 |
|---------|---------|
| 服饰限制 | 只许穿绢布 |
| 出行限制 | 不许乘马 |
| 居住限制 | 不得建造逾制房屋 |
| 社交限制 | 社会地位排在士农工之后 |

### 2.3 经济限制

| 限制内容 | 具体规定 |
|---------|---------|
| 商税 | 三十税一（洪武定制） |
| 关卡 | 各地设立税关，征收过境税 |
| 盐茶专卖 | 重要商品由国家专营 |
| 海禁 | 禁止私人海外贸易（隆庆开海前） |

### 2.4 法律保护

值得注意的是，明代法律在限制商人的同时，也保护商人的合法权益：
- 在涉商案件审理中，商人法律地位与庶民中其他人群是**平等的**
- 官吏不得无故骚扰商人经营
- 禁止在官道设卡额外征税

**来源**：[明代商人户籍问题初探 - 中国法学网](http://iolaw.cssn.cn/gdfls/200504/t20050409_4595629.shtml)；[抑商、恤商与通商：明代商业政策简论 - 人民论坛网](https://www.rmlt.com.cn/2019/0919/557269.shtml)

## 三、捐纳制度

### 3.1 制度起源与发展

| 时期 | 关键事件 |
|------|---------|
| 景泰元年（1450年） | 例监始设——"以边事孔棘，令天下纳粟纳马者入监读书" |
| 成化年间 | 正式定规，确立捐纳标准 |
| 嘉靖以后 | 捐纳日益普遍化 |
| 万历年间 | 规模进一步扩大 |

### 3.2 具体标准

**成化年间定制**：

| 捐纳内容 | 标准 | 获得资格 |
|---------|------|---------|
| 纳米入监 | 100石以上 | 国子监监生 |
| 纳粟授官 | 250石 | 正九品散官 |
| 每加50石 | 递增 | 增二级 |
| 上限 | — | 至正七品止 |

### 3.3 捐纳的类型

- **捐监**：捐纳获得监生资格（"例监"），可直接参加乡试
- **捐贡**：捐纳获得贡生资格（"例贡"）
- **捐官**：直接捐纳散官品级
- **纳粟/纳马**：以实物（粮食或马匹）换取资格

### 3.4 捐纳的意义

- 为商人子弟提供了进入仕途的"杂途"
- 购得的是各种**资格**，能否实际任官仍须经过铨选
- 客观上打破了士商之间的壁垒
- 国家财政困难时期的重要收入来源

**来源**：[捐纳 - 维基百科](https://zh.wikipedia.org/zh-hans/%E6%8D%90%E7%B4%8D)；[纳粟 - 百度百科](https://baike.baidu.com/item/%E7%BA%B3%E7%B2%9F/2509762)

## 四、商籍制度

### 4.1 背景

明初户籍制度将人户分为"民、军、匠"三大类，**未为商人专门设籍**。商人附于民籍之下，其户籍身份与普通编户无异。

### 4.2 商籍的设立

- 由于商人长期在外经商，子弟无法回到原籍参加科举考试
- 官府遂在部分地区设立**"商籍"**，允许商人子弟在经商所在地附籍应考
- 这解决了因户籍限制导致的商人子弟教育权利问题

### 4.3 典型案例

- **两淮盐商**：在扬州等地设有商籍，盐商子弟可在当地参加科举
- **徽商**：徽州商人在各地经商聚居，子弟可附商籍应试
- **晋商**：山西商人在各地也有商籍设立

### 4.4 商籍的意义

- 承认了商人长期流动的现实
- 为商人子弟提供了教育和仕进的渠道
- 促进了士商融合
- 但商籍的名额通常有限，竞争激烈

**来源**：[明代商人户籍问题初探 - 中国法学网](http://iolaw.cssn.cn/gdfls/200504/t20050409_4595629.shtml)

## 五、士商合流现象

### 5.1 思想基础

**心学的推动**：
- 王阳明心学完成了儒家思想对"天理""人欲"的突破
- 肯定了欲望的先天性，将儒家"修己"之路向商人敞开
- 嘉靖初，李梦阳在王文显墓志铭中记述："**商与士，异术而同心**"

**"四民同道"观念**：
- 海瑞认为士、农、工、商、军五民各有其功，"**商贾通焉而资于天下**"
- 王守仁提出"四民异业而同道"
- 商人被纳入儒家道德框架之中

### 5.2 弃儒从商现象

**原因**：
- 科举竞争日益激烈，中举概率极低
- 商业利润丰厚，吸引力增强
- 生员数量膨胀，大量读书人无法通过科举出仕
- 家庭经济压力迫使部分士人转入商业

**表现**：
- 明代中后期出现大量"弃儒从商"的案例
- 部分家族采取"亦儒亦商"的策略——部分子弟读书应举，部分子弟经商
- 徽商中尤为典型，形成"贾而好儒"的传统
- 出现"儒商"这一新的社会身份认同

### 5.3 士商互动的具体形式

| 形式 | 说明 |
|------|------|
| 商人子弟读书应举 | 通过科举或捐纳进入士绅阶层 |
| 士人投入商业 | 失意士人或致仕官员从事商业活动 |
| 商人与士人联姻 | 通过婚姻关系实现阶层融合 |
| 商人资助文化事业 | 商人出资修建书院、刊刻书籍 |
| 商人模仿士人生活方式 | 收藏书画、参与文会 |

**来源**：["四民同道"与"士商合流" - 汉斯出版社](https://www.hanspub.org/journal/paperinformation?paperid=74101)；[从"是官当敬"到"弃儒从贾" - 第一财经](https://www.yicai.com/news/100516426.html)

## 六、商人的实际社会地位变化

### 6.1 明初（洪武至宣德）：严格抑商

- 服饰、出行、居住等全面限制
- 商人被排在"四民"最末
- 但朱元璋同时也注意保护商业秩序
- 开中法使盐商与国家形成互利关系

### 6.2 明中期（正统至嘉靖）：限制松动

- 经济发展使商人财富急剧增长
- 商人通过捐纳开始进入官场
- 服饰限制逐渐松弛
- 商人在地方社会的影响力增大

### 6.3 明后期（隆庆至崇祯）：士商融合

- 隆庆元年（1567年）开放海禁，海外贸易合法化
- "抑商"政策落下帷幕，"通商"与"恤商"成为新方向
- 大批商人及其子弟通过捐纳进入仕途
- 商人生活方式与士大夫趋同

### 6.4 万历矿监税使事件

- 万历皇帝派出大量矿监税使搜刮商人
- 商人遭受沉重打击
- 引发多地民变和商人抗议
- 反映了商人在政治上仍然脆弱的一面

### 6.5 主要商帮

| 商帮 | 特点 | 主营行业 |
|------|------|---------|
| 徽商 | "贾而好儒"，文化水平高 | 盐业、典当、茶叶、木材 |
| 晋商 | 重信义，组织严密 | 盐业、票号、长途贸易 |
| 闽商 | 善于海外贸易 | 海外贸易、丝绸、瓷器 |
| 粤商 | 依托广州口岸 | 对外贸易 |
| 龙游商帮 | 浙江龙游，精明务实 | 纸张、木材 |

## 七、制度规定与实际状况的对比

| 维度 | 制度规定 | 嘉靖至崇祯的实际状况 |
|------|---------|-------------------|
| 服饰 | 商人只许穿绢布 | 富商服饰与士大夫无异 |
| 科举 | 初期禁止商人子弟应试 | 通过商籍、捐监等渠道广泛参与 |
| 社会地位 | 四民之末 | 与士大夫并驾齐驱 |
| 政治参与 | 不得干预政事 | 通过联姻和捐纳深度介入 |
| 居住建筑 | 不得逾制 | 富商建筑极尽奢华 |
