# 妇女地位

> 聚焦时段：嘉靖至崇祯（1521-1644）
> 核心议题：一夫一妻多妾制、七出三不去、聘金嫁妆、妇女财产权、贞节旌表、女性教育、寡妇再嫁

---

## 一、一夫一妻多妾制

### 1.1 制度规定

明代实行严格的**一夫一妻多妾制**：一个男子只能有一个正妻（嫡妻），但可以合法纳妾。妻与妾之间存在严格的等级差异。

### 1.2 妻妾的法律区别

| 维度 | 正妻（嫡妻） | 妾（侧室） |
|------|-------------|-----------|
| 法律地位 | 与丈夫地位平等 | 低于正妻 |
| 聘娶程序 | 须经六礼（纳采、问名、纳吉、纳征、请期、亲迎） | 无需完整六礼 |
| 子女地位 | 嫡子，优先继承 | 庶子，继承权次于嫡子 |
| 家庭权力 | 主持家务、管理内宅 | 受正妻管辖 |
| 离异后待遇 | 有较多保障 | 保障极少 |

### 1.3 纳妾的具体规则

**《大明律》规定**：
- 民年四十以上无子者，方许奏选一妾，不得蓄妾过多
- 官员纳妾另有规定
- 不得以妾为妻，违者杖九十
- 有妻更娶者杖九十，离异

**实际执行**：
- 制度上限制纳妾条件，实际中富户和官员普遍多妾
- 明代中后期纳妾之风盛行
- 部分大户蓄养十数名甚至数十名侍妾

### 1.4 妻妾等级的维护

- 正妻地位受法律保护，丈夫殴妻与殴妾处罚不同
- 正妻有权管束侍妾
- 妾不得在正妻面前失礼
- 但正妻若虐待侍妾至死，也会受到法律制裁

## 二、七出三不去

### 2.1 七出条文

"七出"是丈夫（及其家族）可以休弃妻子的七种法定理由：

| 条目 | 内容 | 理由解释 |
|------|------|---------|
| 一 | 不顺父母 | 不孝公婆 |
| 二 | 无子 | 未能传宗接代 |
| 三 | 淫 | 通奸或行为不端 |
| 四 | 妒 | 嫉妒（主要指阻碍丈夫纳妾） |
| 五 | 有恶疾 | 患有严重疾病 |
| 六 | 多言 | 搬弄是非，离间亲属 |
| 七 | 盗窃 | 偷窃财物 |

### 2.2 三不去条文

即使妻子有"七出"之情，若符合以下三种情况之一，丈夫**不得休弃**：

| 条目 | 内容 | 理由解释 |
|------|------|---------|
| 一 | 有所取无所归 | 妻子娘家已无亲人可依靠 |
| 二 | 与更三年丧 | 曾为公婆守丧三年 |
| 三 | 前贫贱后富贵 | 共患难，不可弃 |

### 2.3 "三不去"的保护意义

- 限制了丈夫无条件休妻的权力
- 保护了在特定困境中的妇女
- 但实际执行中，"三不去"常被忽视
- "七出"中的某些条款（如"无子"、"妒"）定义模糊，容易被滥用

## 三、聘金与嫁妆

### 3.1 聘金（彩礼）

**制度规定**：
- 明代对不同社会等级的聘金有大致的礼制规范
- 官员和士绅家庭的聘金通常以银两计算
- 普通百姓以实物（布帛、牲畜等）为主

**实际数据**（明代中后期）：
- 士绅家庭：聘金可达数十两至数百两白银
- 富商家庭：可能更高，有时以奢侈品（珠宝、绸缎）折算
- 普通百姓：数两至十余两白银，或等值实物
- 贫困家庭：有时仅为象征性的少量礼物

### 3.2 嫁妆

**嫁妆的社会功能**：
- 嫁妆是女子出嫁时从娘家带到夫家的财物
- 为新婚妇女提供独立的私产
- 增强妇女在夫家的地位和议价能力
- 嫁妆通常包括：衣物、首饰、家具、妆匣、被褥，富户还包括田产、店铺

**厚嫁风俗**：
- 明代中后期"厚嫁"之风盛行
- 殷实人家嫁女儿，从马桶到棺材都会准备
- 嫁妆穷富悬殊，富户嫁妆可价值数百至数千两
- 高额嫁妆产生了严重社会问题：
  - **溺女婴**行为增多（养女花费太大）
  - 人口性别比例失衡
  - 贫困男子成婚困难

### 3.3 嫁妆的法律归属

- 嫁妆在法律上逐渐被视为**夫家财产的一部分**
- 《大明律》规定，妇女改嫁时："夫家财产及**原有嫁妆**并听前夫之家为主"
- 这意味着改嫁妇女连嫁妆都可能保不住
- 但在日常生活中，嫁妆仍被视为妻子的"私产"，由妻子实际管理

## 四、妇女财产权

### 4.1 嫁妆管理权

- 在日常生活中，嫁妆通常由妻子管理
- 丈夫动用妻子嫁妆需要妻子同意（至少在礼制上如此）
- 但法律上，嫁妆的最终归属权在改嫁时归夫家

### 4.2 家庭财产管理权

- 作为妻子或母亲的妇女，可以**管理家庭财产**
- 在家庭财产的支配上有一定的权利
- 特别是在丈夫外出经商或为官时，妻子实际掌管家务和财产
- 寡妇在子女未成年时可以代管家产

### 4.3 继承权的限制

**无子寡妻**：
- 明代法律规定，无子寡妻在理论上**不再有权继承**亡夫的财产
- 有义务为亡夫**立继**（从族内过继嗣子）
- 一旦立嗣，财产由嗣子继承

**有子寡妻**：
- 可以作为监护人管理家产直到儿子成年
- 但不能自行处分家产

**改嫁妇女**：
- 改嫁时连嫁妆都可能被前夫家族扣留
- 财产权几乎完全丧失

### 4.4 女儿的继承权

- 明代法律中，女儿的继承权远弱于儿子
- 有子之家，女儿一般不参与家产分配（以嫁妆代替）
- 无子之家，女儿可能获得部分遗产（但需要与宗族协调）
- 未嫁女有一定的遗产份额

**来源**：[《明清时代妇女的地位与权利》- 豆瓣](https://book.douban.com/reading/10597732/)；[纵横 - 北京政法网](https://www.bj148.org/wh/bl/zh/202102/t20210204_1597796.html)

## 五、贞节旌表数据

### 5.1 历代旌表数据对比

根据学者**董家遵**对《古今图书集成》中资料的统计：

| 时代 | 贞节烈女人数 | 说明 |
|------|------------|------|
| 周至五代 | 92人 | 含唐代数据 |
| 宋代 | 152人 | |
| 元代 | 359人 | |
| **明代** | **27,141人** | 远超此前所有朝代 |
| 清代（至统计时止） | 9,482人 | |

### 5.2 明代贞节旌表制度

**制度建设**：
- 明太祖朱元璋在位期间**详细制定**关于旌表的规定
- 其后历朝皇帝遵循其重视旌表制度的政策
- 旌表制度远比前代受到重视

**旌表标准**：
| 类别 | 定义 |
|------|------|
| 节妇 | 三十岁之前丧夫，守节到五十岁以上的妇女 |
| 烈女 | 为保全贞操而死亡的女子 |
| 贞女 | 未婚夫死后终身不嫁的女子 |

**旌表方式**：
- 由地方官府上报，经礼部审核
- 获旌表者可建**贞节牌坊**
- 家族可享受一定的赋税优免
- 事迹载入地方志

### 5.3 数据背后的社会意涵

- 明代27,141人的旌表数对比唐代（包含在"周至五代92人"中）增长了数百倍
- 反映了明代对妇女贞节的**制度性强化**
- 理学（程朱理学）对社会风气的深刻影响——"饿死事小，失节事大"
- 国家通过旌表制度**主动引导**妇女守节
- 明代中后期出现了大量**未婚守节**（贞女）的极端案例

## 六、女性教育

### 6.1 女四书

明末清初，山东琅琊人**王相**将四部女教经典汇编为《闺阁女四书集注》，成为明清女子教育的核心教材：

| 书名 | 作者 | 朝代 | 篇章数 | 核心内容 |
|------|------|------|--------|---------|
| 《女诫》 | 班昭 | 东汉 | 7篇 | 卑弱、夫妇、敬顺、妇行、专心、曲从、叔妹 |
| 《内训》 | 徐皇后（仁孝文皇后） | 明永乐 | 20章 | 德行、修身、慎言、谨行、勤励、节俭等 |
| 《女论语》 | 宋若萃 | 唐代 | 12章 | 仿《论语》体例论述女德 |
| 《女范捷录》 | 刘氏（王相之母） | 明末 | 11篇 | 统论、后德、母仪、孝行、贞烈等 |

### 6.2 《内训》的特殊地位

- 《内训》是**仁孝文皇后**（明成祖朱棣的徐皇后）所作
- 专为教育宫中妇女编写
- 20章涵盖：德行、修身、慎言、谨行、勤励、节俭、警戒、积善、迁善、崇圣训、景贤范、事父母、事君、事舅姑、奉祭祀、母仪、睦亲、慈幼、逮下、待外戚
- 在明代宫廷和士大夫家庭中影响深远

### 6.3 女子教育的实际状况

**正面**：
- "女四书"的普及提高了当时女子的文化水平
- 士大夫家庭中，女子多能读书识字
- 明代出现了不少有文学才华的女性（如叶小鸾、柳如是等）

**限制**：
- 女子教育的核心目的是"妇德"而非知识
- "女子无才便是德"的观念在部分群体中流行
- 普通百姓家庭的女子很少有受教育机会
- 女子不得参加科举考试

**来源**：[明清女性必读书《女四书》- 澎湃新闻](https://www.thepaper.cn/newsDetail_forward_24424337)；[《女四书》：古代的女子教科书 - 知乎](https://zhuanlan.zhihu.com/p/628117356)

## 七、寡妇再嫁的社会态度

### 7.1 制度层面

**法律规定**：
- 明代法律并未**绝对禁止**寡妇再嫁
- 但《大明律》对再嫁有具体限制：
  - 守丧期内不得再嫁（违者杖一百）
  - 收继婚（娶父辈或兄弟寡妻）严格禁止
  - 娶父、伯、叔寡妇者**判斩首**
  - 娶寡兄嫂或弟媳者**判绞死**
- 改嫁妇女须归还前夫家嫁妆和财产

**旌表制度的导向**：
- 国家通过旌表守节妇女，引导社会崇尚贞节
- 守节可获得旌表、建牌坊、减免赋税等荣誉和实利
- 再嫁则被视为"失节"，受到道德谴责

### 7.2 社会风气的变化

**明代中后期的宽容趋势**：
- 明代中叶以后，夫丧改嫁已**为社会舆论所认可**
- 特别是在商品经济发达的江南地区
- 底层百姓家庭中寡妇再嫁较为普遍
- 经济因素（寡妇独力难以维持生计）促进了再嫁的实际发生

**士大夫阶层的矛盾**：
- 理学家倡导"饿死事小，失节事大"
- 但许多理学家大儒在著作中对妻子表达了敬重
- 认为夫妇之间在人格上是**对等的**
- 实际中，士大夫家庭的寡妇再嫁也并非罕见

### 7.3 再嫁的经济后果

- 改嫁妇女面临的最大经济风险是**丧失所有财产**
- 嫁妆归前夫家所有
- 子女通常留在前夫家
- 再嫁后的社会地位可能降低

## 八、制度与实际的落差

| 维度 | 制度/理念 | 实际社会状况 |
|------|---------|------------|
| 纳妾限制 | 四十无子方许纳妾 | 富户普遍多妾 |
| 七出休妻 | 有"三不去"保护 | "三不去"常被忽视 |
| 贞节要求 | 理学极力倡导 | 民间再嫁普遍 |
| 庶民蓄妾 | 有条件限制 | 限制形同虚设 |
| 女子教育 | "妇德"为主 | 士绅家庭女子文化水平不低 |
| 嫁妆归属 | 改嫁归夫家 | 日常由妻子管理 |
| 财产继承 | 寡妻无权继承 | 实际中寡妻常管理家产 |
