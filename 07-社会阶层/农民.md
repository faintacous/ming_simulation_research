# 农民

> 聚焦时段：嘉靖至崇祯（1521-1644）
> 核心议题：自耕农/佃农/雇农比例、租佃关系、地租率、皇庄王庄、永佃权与一田两主、农民收入与生产周期

---

## 一、农民阶层的构成与比例

### 1.1 三类农民的界定

| 类别 | 定义 | 特征 |
|------|------|------|
| 自耕农 | 拥有自己土地并亲自耕种 | 需缴纳田赋和承担徭役 |
| 佃农 | 租种他人土地 | 需缴纳地租，田赋通常由地主承担 |
| 雇农 | 被雇佣为他人劳作 | 领取工资或口粮，无土地 |

### 1.2 比例估算

根据明代文献记载和学者研究：

- 明代中后期，**自耕农仅占约1/10**，其余**9/10为佃户**（据《明代农业生产力的发展和生产关系的演变》的估算）
- 江苏地区约有**略高于50%的田地用于出租**，租佃率与20世纪相近
- 土地集中趋势在嘉靖至万历间加剧，自耕农比例持续下降
- 不同地区差异显著：江南佃农比例高于北方，山区自耕农比例相对较高

**来源**：[明代农业生产力的发展和生产关系的演变 - 中国经济史论坛](http://economy.guoxue.com/?p=4496)

### 1.3 雇农群体

- 雇农在法律上被称为"雇工人"，法律地位低于普通良民
- 明代中后期，随着土地兼并加剧，雇农群体不断扩大
- 雇农又分为长工（年雇）和短工（季节性雇佣）
- 长工与雇主之间有时存在类似主仆的人身依附关系

## 二、租佃关系类型

### 2.1 两大租制

#### 定额租（包租制）

- 地租数额在租约中事先约定，不随收成变化
- 丰年时佃农获利，歉年时风险由佃农承担
- 明代中后期逐渐成为主要租制，尤其在江南地区
- 佃农有更大的生产积极性，因为增产部分归自己

#### 分成租

- 地租按收成的一定比例分配，随年景浮动
- 丰年时地主获利多，歉年时风险共担
- 常见的分成比例为"对半分"（地主与佃农各得50%）
- 在经济较落后的地区和新开垦地区更为常见

### 2.2 地租形式

| 形式 | 说明 | 流行程度 |
|------|------|---------|
| 实物地租 | 以粮食等实物缴纳 | 最普遍，占主导地位 |
| 货币地租 | 以银两或铜钱缴纳 | 明中后期在商品经济发达地区增多 |
| 劳役地租 | 以劳动服务代替租金 | 较少见，主要存在于落后地区 |

**来源**：[中国经济史笔记（四）中国传统社会的土地制度与租佃关系 - 知乎](https://zhuanlan.zhihu.com/p/112810566)

## 三、地租率数据

### 3.1 制度层面的规定

明代官方曾规定官田租率，但民间租率缺乏统一规制。

### 3.2 实际数据

- 明代地租率**大都在收成的50%以上**
- 部分地区地租率达到**70%甚至80%**
- 江南地区的典型地租率约为**50%**（对半分）
- 皇庄王庄的地租相对较轻，通常按每亩征银三分（远低于民间地租）

### 3.3 各地区地租率比较

| 地区 | 地租率范围 | 说明 |
|------|-----------|------|
| 江南（苏松） | 50%-60% | 定额租为主，实物缴纳 |
| 闽粤 | 50%-70% | 分成租与定额租并存 |
| 北方 | 50%-60% | 分成租较多 |
| 皇庄王庄 | 每亩三分银 | 制度规定较轻，但额外摊派较重 |

**来源**：[地主到底占多少土地，佃户究竟交多少地租？ - 知乎](https://zhuanlan.zhihu.com/p/24443316)

## 四、皇庄与王庄

### 4.1 皇庄的发展

| 时期 | 数量/面积 | 说明 |
|------|-----------|------|
| 弘治二年（1489） | 5处，12,800顷 | 京畿地区 |
| 正德年间 | 300多处，37,595顷46亩 | 武宗大肆扩张 |
| 嘉靖初 | 36处，37,595顷46亩 | 经成化、弘治、正德三朝发展 |

- 武宗朱厚照即位一个月内即在大兴县设皇庄7所
- 十年内扩展至昌平、真定、保定等地
- 皇庄面积占京师八府官民田总数的**1/7**

### 4.2 皇庄的经营

- 按每亩三分征税，御马监每年所收皇庄子粒银约**10万两**
- 由宦官（太监）代管，管理腐败严重
- 皇庄佃户受宦官盘剥，处境常不如普通佃农

### 4.3 王庄规模

| 时期 | 规模 | 占比 |
|------|------|------|
| 万历年间 | 约20.9万顷 | 占官田12.6%，官民田1.8% |
| 各类庄田合计（万历） | 约58.1万顷 | 占官田35.03%，官民田5% |

- 王庄占庄田总量的**35.97%**，是庄田的最大组成部分
- 明代宗室藩王封地遍布各省，对地方土地格局影响深远
- 弘治十五年（1502年），全国官田面积达民田的**1/7**

**来源**：[皇庄 - 百度百科](https://baike.baidu.com/item/%E7%9A%87%E5%BA%84/1715155)；[崇祯时期皇帝手里的皇庄有多少？ - 知乎](https://www.zhihu.com/question/462894766)

## 五、永佃权与"一田两主"制度

### 5.1 永佃权的形成

- 永佃制是明代中后期逐渐发展起来的租佃制度
- 佃农通过长期耕种、垦荒或支付押金，获得对土地的**永久耕作权**
- 地主不得随意收回土地，但保留田底权（所有权）
- 佃农获得田面权（耕作权），可以世代相传

### 5.2 一田两主制度

**制度结构**：

```
一块土地
├── 田底权（田骨） → 属于田底主（原地主）
│   └── 收取"大租"
└── 田面权（田皮） → 属于田面主（佃农）
    └── 收取"小租"（若再转租）
```

**核心特征**：
- 田底权和田面权可以**分别转让和买卖**
- 田底主向佃农收取"大租"
- 佃农若将田面权转租，向次佃收取"小租"
- 形成"一田三主"的更复杂关系

### 5.3 法律与社会意义

- 永佃权给予佃农一定的土地使用安全感
- 佃农对土地的投入（改良、水利等）受到保障
- 田面权成为一种可交易的"准产权"
- 在明清法律中，田面权有时被视为独立的"业权"

**来源**：[从一田二主到永佃权 - 中国土地法制研究网](https://illss.gdufs.edu.cn/info/1024/4174.htm)；[明清地权契约中"业"的表述与田骨田皮的"业权"属性](https://html.rhhz.net/njnydxxbskb/20170515.htm)

## 六、农民的年收入估算

### 6.1 粮食产量基础数据

**南方稻作区**：
| 类型 | 亩产（石） | 说明 |
|------|-----------|------|
| 双季稻 | 约5石 | 约占南方耕地3% |
| 稻麦连作 | 约4石 | 约占20% |
| 单季稻 | 约3石 | 约占47% |
| 南方平均 | 2.72石 | 综合估算 |

**北方旱作区**：
- 上地：约1石/亩
- 中下地：0.5-0.7石/亩
- 两年三熟上地可超过1石/亩

### 6.2 农民收支计算（以江南佃农为例）

**收入端**：
- 佃种面积：典型约5-10亩
- 亩产（秋熟中等）：约2石米
- 春花产量：约0.5石/亩
- 毛收入：约12.5-25石

**支出端**：
- 地租（50%）：约6.25-12.5石
- 种子、农具等生产成本
- 家庭口粮（成人年需约3石米）
- 余粮极少，甚至不足

### 6.3 地区差异

- 浙江种桑养蚕的农民收入**最多**（兼营经济作物）
- 江苏纺棉织布的农民**次之**
- 四川种植粮食的农民收入**最少**

**来源**：[明清时期的粮食生产与农民生活水平 - 中国经济史论坛](http://economy.guoxue.com/?p=5300)；[明清时期的江南农民，究竟处于怎样生活水平 - 澎湃新闻](https://www.thepaper.cn/newsDetail_forward_27270947)

## 七、农业生产周期

### 7.1 南方水稻区（以江南为例）

| 月份 | 活动 |
|------|------|
| 正月-二月 | 整地、准备秧苗 |
| 三月 | 播种春花（小麦、油菜等） |
| 四月-五月 | 插秧（早稻），收割春花 |
| 六月-七月 | 田间管理，除草灌溉 |
| 八月-九月 | 收割早稻，播种晚稻（双季区） |
| 十月-十一月 | 收割晚稻/单季稻，播种冬小麦 |
| 十二月 | 农闲，修缮农具，从事副业 |

### 7.2 北方旱作区

| 月份 | 活动 |
|------|------|
| 二月-三月 | 春耕，播种小麦、粟、高粱 |
| 四月-五月 | 间苗、锄草 |
| 六月 | 田间管理 |
| 七月-八月 | 收割小麦，播种秋粮 |
| 九月-十月 | 收割秋粮（粟、高粱等） |
| 十一月-正月 | 农闲，从事手工业或打零工 |

### 7.3 复种制度

- 江南地区推行**稻麦连作**或**稻油菜连作**
- 部分地区实现**双季稻**（约占南方耕地3%）
- 北方多为**一年一熟**或**两年三熟**
- 明代中后期复种指数提高，推广了番薯、玉米等新作物

**来源**：[明代农业生产力的发展和生产关系的演变 - 中国经济史论坛](http://economy.guoxue.com/?p=4496)；[明清粮食亩产 - 知乎](https://zhuanlan.zhihu.com/p/709330156)

## 八、农民的赋役负担

### 8.1 田赋

- 田赋是农民最基本的税收负担，按田亩和粮额征收
- 一条鞭法后，田赋折银征收
- 明末"三饷"加派使农民负担急剧加重

### 8.2 徭役

- 里甲正役：110户编为一里，十年一轮
- 均徭：按丁粮多少分派
- 杂泛差役：临时性的差遣和劳役

### 8.3 额外摊派

- 地方官吏的"火耗"（征银时的损耗加成）
- 各种杂税和摊派
- 赋役不均导致中小农户负担尤重
