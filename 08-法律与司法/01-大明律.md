# 大明律

## 概述

《大明律》是明朝的基本法典，历经吴元年（1367）至洪武三十年（1397）前后三十年反复修订定型，此后"终明之世未再修订"。全律以《名例律》冠首，下按六部分为吏、户、礼、兵、刑、工六律，共七篇三十卷三十门四百六十条。其法律精神上承《唐律疏议》，结构上借鉴《元典章》按六部分类的编排方式，是中国古代最后一部以"律"命名的综合性法典。[来源：维基百科-大明律；百度百科-大明律]

## 一、编纂沿革

### 1.1 四次修律过程

| 时间 | 版本 | 篇目体例 | 条数 | 主持者/背景 |
|------|------|----------|------|------------|
| 吴元年（1367） | 《律令》 | 律285条 + 令145条 | 430条 | 左丞相李善长等 |
| 洪武七年（1374） | 《大明律》初版 | 仿唐律十二篇（卫禁、斗讼、诈伪、杂律、捕亡、断狱、名例等） | 606条（30卷） | 刘惟谦等 |
| 洪武二十二年（1389） | 修订版 | **改为六律体例**：名例 + 吏户礼兵刑工 | 460条（30卷） | 大幅度改定 |
| 洪武三十年（1397） | 定本 | 同上，微调 | 460条（30卷） | 最终颁行，"终明之世未再修订" |

### 1.2 体例革新的历史意义

- 洪武二十二年的改革**打破了自北齐以来沿用八百年的十二篇旧例**
- 改为按六部（吏户礼兵刑工）职掌分律，以《名例律》冠首
- 这一体例被清代《大清律例》完全沿用

## 二、篇目结构与各篇条数

### 2.1 总体结构：7篇30卷460条

| 篇名 | 卷数 | 条数 | 内容概要 |
|------|------|------|----------|
| **名例律** | 1卷 | 47条 | 五刑、十恶、八议、自首、收赎、老幼废疾、共犯、累犯等总则性规定 |
| **吏律** | 2卷 | 33条 | 职制（15条）、公式（18条）：官员选任、考课、文书管理 |
| **户律** | 7卷 | 95条 | 户役（14条）、田宅（10条）、婚姻（18条）、仓库（24条）、课程（19条）、钱债（3条）、市廛（7条） |
| **礼律** | 2卷 | 26条 | 祭祀（6条）、仪制（20条） |
| **兵律** | 5卷 | 75条 | 宫卫（17条）、军政（20条）、关津（8条）、厩牧（11条）、邮驿（19条） |
| **刑律** | 11卷 | 171条 | 贼盗（28条）、人命（20条）、斗殴（22条）、骂詈（8条）、诉讼（12条）、受赃（11条）、诈伪（12条）、犯奸（10条）、杂犯（11条）、捕亡（8条）、断狱（29条） |
| **工律** | 2卷 | 13条 | 营造（9条）、河防（4条） |

### 2.2 与唐律的对比

| 对比项 | 《唐律疏议》 | 《大明律》 |
|--------|-------------|-----------|
| 篇目数 | 12篇 | 7篇（1名例 + 6律） |
| 条数 | 502条 | 460条 |
| 组织原则 | 按法律事项分类 | 按行政职能（六部）分类 |
| 经济立法 | 较少 | 扩大，增设《课程》《钱债》《市廛》 |
| 工律 | 无独立篇 | **首设工律** |
| 量刑原则 | "一准乎礼" | "世轻世重"——礼仪事项从轻，贪污重罪加重 |

## 三、十恶

### 3.1 十恶列表与具体定义

十恶源自北齐"重罪十条"，经隋唐定型，明律沿用，属**"常赦所不原"**之罪。

| 序号 | 罪名 | 定义 |
|------|------|------|
| 一 | **谋反** | 谋危社稷 |
| 二 | **谋大逆** | 谋毁宗庙、山陵及宫阙 |
| 三 | **谋叛** | 谋背本国，潜从他国 |
| 四 | **恶逆** | 殴及谋杀祖父母、父母，杀伯叔父母、姑、兄姊、外祖父母、夫之祖父母、父母 |
| 五 | **不道** | 杀一家非死罪三人及肢解人，造畜蛊毒、魇魅 |
| 六 | **大不敬** | 盗大祀神御之物、乘舆服御物，盗及伪造御宝，合和御药误不依本方，御膳误犯食禁，御幸舟船误不牢固 |
| 七 | **不孝** | 告言、咒骂祖父母父母，祖父母父母在而别籍异财，供养有缺，居父母丧身自嫁娶，若作乐、释服从吉，闻祖父母父母丧匿不举哀，诈称祖父母父母死 |
| 八 | **不睦** | 谋杀及卖缌麻以上亲，殴告夫及大功以上尊长小功尊属 |
| 九 | **不义** | 杀本属府主、刺史、县令、见受业师，吏卒杀本部五品以上长官，及闻夫丧匿不举哀、若作乐释服从吉及改嫁 |
| 十 | **内乱** | 奸小功以上亲、父祖妾，及与和者 |

### 3.2 十恶的法律后果与量刑

| 罪名 | 本人刑罚 | 连坐范围 |
|------|---------|---------|
| **谋反** | 凌迟处死，不分首从 | 祖父、父、子、孙、兄弟及同居之人，十六岁以上皆斩；十五岁以下及母女妻妾给付功臣之家为奴 |
| **谋大逆** | 同谋反 | 同谋反 |
| **谋叛** | 本人斩首 | 父母、妻子流二千里 |
| **恶逆** | 凌迟处死 | 视情节而定 |
| **不道** | 斩首 | 妻子流二千里 |
| **大不敬** | 依情节从斩到杖 | — |
| **不孝** | 斩或绞 | — |
| **不睦** | 依情节笞杖至斩 | — |
| **不义** | 斩 | — |
| **内乱** | 绞 | — |

**法律效力要点**：
- **常赦所不原**：即便遇到大赦，犯十恶者也不在赦免之列
- **不适用八议**：犯十恶者即使具有八议身份，也不能援引八议减罪
- **不适用官当、纳赎**等减刑优待
- 十恶制度集中体现了维护皇权和宗法伦理的双重目的

## 四、八议

### 4.1 八议列表与适用范围

八议制度源自《周礼》"八辟"，经曹魏入律，明律沿用。凡应议之人犯罪，**须先奏请，议定后再拟罪**。

| 序号 | 议类 | 适用对象 |
|------|------|----------|
| 一 | **议亲** | 皇帝袒免以上亲属及太皇太后、皇太后缌麻以上亲 |
| 二 | **议故** | 皇帝故旧（特旨者） |
| 三 | **议功** | 有大功勋者 |
| 四 | **议贤** | 有大德行者 |
| 五 | **议能** | 有大才能者 |
| 六 | **议勤** | 有大勤劳者 |
| 七 | **议贵** | 爵一品及文武官三品以上者 |
| 八 | **议宾** | 前朝国君之后裔（承先代之后为国宾者） |

### 4.2 八议的处理程序

1. 应议之人犯死罪：**实封奏闻**，取旨裁决
2. 犯流罪以下：**减一等**处理
3. **例外**：犯十恶者不得援引八议

## 五、名例律中的其他重要制度

### 5.1 自首制度

- 犯罪未发而自首者，免其罪
- 已发觉而自首者，减罪二等
- 但对于杀伤人命、已得赃物者有限制

### 5.2 老幼废疾收赎

| 年龄/身体条件 | 处理方式 |
|--------------|----------|
| 七十以上、十五以下、废疾 | 犯流罪以下收赎（铜钱折抵） |
| 八十以上、十岁以下、笃疾 | 犯杀人应死者奏闻取裁，余罪收赎 |
| 九十以上、七岁以下 | 虽有死罪不加刑 |

### 5.3 明律的特有条款

- **奸党律**：禁止朝臣交结朋党，为前代律典所无
- **交结近侍官员律**：禁止外朝官员与内廷宦官交结
- **"重其所重，轻其所轻"原则**：贼盗及有关帑项钱粮等加重，典礼及风俗教化事宜从轻

---

## 六、大明律的法律地位

- 洪武三十年定本颁行后，**终明之世再未修改**律文本身
- 但通过《问刑条例》（弘治至万历三次修订）弥补律文之不足
- 万历十三年（1585）编成《大明律集解附例》，"律为正文，例为附注"
- 其体例结构直接影响清代《大清律例》的编纂

---

## 七、主要罪名分类与量刑原则

### 7.1 "轻其轻罪，重其重罪"

《大明律》采用"世轻世重"原则：事关典礼及风俗教化等事定罪较轻；贼盗及有关帑项钱粮等事定罪较重。[来源：百度百科-大明律]

| 罪名类别 | 代表性罪名 | 刑罚特征 | 与唐律比较 |
|---------|-----------|---------|-----------|
| **谋反大逆** | 谋反、谋大逆 | 凌迟处死，亲族连坐 | 继承唐律但连坐范围更广 |
| **贼盗** | 强盗得财 | 不分首从皆斩 | 比唐律加重 |
| **受赃** | 官吏受财枉法 | 赃满八十贯绞 | 唐律十五匹绞，明律起点更低 |
| **人命** | 谋杀、故杀 | 斩；过失杀收赎 | 基本继承 |
| **犯奸** | 强奸 | 绞 | 比唐律加重 |

### 7.2 明律独创的罪名

- **奸党罪**：《吏律》"奸党"条为前代所无，文武官员交结朋党紊乱朝政者斩，妻子为奴，财产入官
- **交结近侍官员罪**：禁止外朝官员与内廷宦官交结
- **官吏出入人罪**：故意错判的官吏与所出入之罪同等处罚

## 八、嘉靖至崇祯时期的律法实况

### 8.1 律典的僵化与补救

- 洪武三十年定本后律文不再修改
- 面对社会变迁，通过《问刑条例》补充（弘治279条 → 嘉靖385条 → 万历382条）
- 至晚明，条例实际上已取代律典成为司法实践的主要依据

### 8.2 万历《大明律集解附例》

万历十三年（1585），以"律为正文，例为附注"的体例，将382条《问刑条例》附于相关律条之后，编成《大明律集解附例》颁行，此后至明末未再修改。这是中国古代"律例合编"体例的开端。
