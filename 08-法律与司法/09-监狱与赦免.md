# 监狱与赦免制度

## 概述

明代首次在法律文献中正式将"狱"称为"监"，"监狱"一词此后沿用至今。明代监狱分为中央监狱（刑部、都察院）、地方监狱（府州县）、军事监狱（卫所）和特种监狱（锦衣卫诏狱）四大系统。在囚犯管理方面，洪武元年即颁布了关于衣食医药的具体规定。赦免方面，明朝276年间共进行约52-55次大赦（平均5年一次），频率远低于前代，体现了"严赦慎赦"的政策取向。《大明律》首次系统规定了"常赦所不原"的范围，远超传统"十恶不赦"。[来源：中国反腐倡廉发展史；中国新闻网]

## 一、监狱建制

### 1.1 监狱体系

| 系统 | 设置层级 | 管理机构 | 性质 |
|------|----------|----------|------|
| **中央监狱** | 刑部、都察院 | 刑部提牢厅 | 关押重大案犯 |
| **地方监狱** | 府、州、县 | 府设司狱司，州县由主官兼管 | 关押辖区案犯 |
| **军事监狱** | 卫所 | 都指挥使司司狱 | 关押军事犯 |
| **特种监狱** | 锦衣卫北镇抚司 | 锦衣卫直管 | **诏狱**，独立于三法司 |

### 1.2 中央监狱

#### 刑部监狱

| 项目 | 内容 |
|------|------|
| **管理机构** | 提牢厅 |
| **提牢官** | 刑部每月轮派十三清吏司一名主事提点刑部大狱 |
| **下属人员** | 司狱6名、看监、门差、供事、捕役等狱卒 |
| **监狱分区** | 设内监和外监：内监收押强盗人命死罪犯，外监收押轻罪犯 |

**提牢厅的具体职责**：

| 职责 | 说明 |
|------|------|
| 查验关防出入 | 监控监狱进出人员 |
| 稽查收支月粮煤油 | 确保囚犯供给到位 |
| 修理狱具 | 维护监狱设施 |
| 查理病囚医药 | 保障囚犯医疗 |
| 审查久押不决 | 清理"淹禁日久"的案件 |
| 点视囚犯 | 定时巡查核实在押人数 |

#### 都察院监狱

- 设于都察院，关押该院纠察弹劾的重大囚犯

### 1.3 地方监狱

| 层级 | 监狱管理 | 狱官品级 |
|------|----------|---------|
| **府** | 设司狱司，有专职司狱 | 从九品 |
| **州** | 不设专职狱官，由知州兼管 | — |
| **县** | 不设专职狱官，由知县及佐贰官兼管 | — |

**地方监狱的基本设施标准**（洪武年间规定）：
- 须有围墙、大门
- 分设男监、女监
- 设有病房
- 须有放风场地

### 1.4 诏狱（锦衣卫北镇抚司）

| 项目 | 内容 |
|------|------|
| **管辖** | 皇帝钦定案件的犯人 |
| **管理** | 锦衣卫北镇抚司直管 |
| **性质** | 独立于三法司的特种监狱 |
| **条件** | 极为恶劣，刑讯酷虐 |
| **典型囚犯** | 杨涟、左光斗、魏大中等东林党人 |
| **特殊权力** | 锦衣卫自行逮捕、侦讯、行刑、处决，不经三法司 |

[来源：中国反腐倡廉发展史-明朝的监狱建制；中国法学网-明清州县的监狱]

## 二、囚犯分类管理

### 2.1 分类原则

明代监狱实行**分类杂居制**：

| 分类标准 | 具体规定 |
|---------|---------|
| **性别** | 男女分监 |
| **身份** | 贵贱异监——官吏犯公罪单独关押 |
| **罪行轻重** | 内监（重罪）/ 外监（轻罪） |
| **年龄** | 七十岁以上、十五岁以下单独关押 |
| **残疾** | 有残疾者单独关押 |
| **散收/锁禁** | 官吏犯公罪依情节散收或锁禁 |

### 2.2 女犯的特殊规定

- 妇女犯罪并非一概收监
- 只有犯**奸罪**和**私罪**才收监
- 其他罪行可交由丈夫或家属管领

## 三、囚犯待遇

### 3.1 洪武元年的诏令

明太祖洪武元年颁布诏令，规定了囚犯待遇的基本标准：

> "枷纽常须洗涤，席苇常须铺置，冬设暖匣，夏备凉浆。无家属者，日给仓米一升，冬给棉衣一件，夜给油灯，病给医药。"

### 3.2 具体待遇规定

| 待遇项目 | 具体规定 | 备注 |
|----------|----------|------|
| **饮食** | 无家属者日给仓米一升（约1.035升） | 天顺元年始正式确认 |
| **衣物** | 冬季给棉衣一件 | 嘉靖六年令每年冬天发棉衣裤一套 |
| **卫生** | 枷纽常须洗涤，席苇常须铺置 | 定期清洁刑具和铺设垫褥 |
| **照明** | 夜间给油灯 | — |
| **防暑御寒** | 冬设暖匣，夏备凉浆 | — |
| **医疗** | 病给医药 | 成化十二年广设惠民药局治疗囚人 |
| **家属探视** | 定期准许家属探监 | — |
| **节庆优待** | 节庆之日家属可带黄酒肉果等食物入监 | — |

### 3.3 医疗保障

| 时期 | 措施 |
|------|------|
| 洪武年间 | 规定"病给医药" |
| 成化十二年（1476） | 广设惠民药局治疗囚人 |
| 日常制度 | 提牢厅职责包括"查理病囚医药" |

### 3.4 禁止虐囚的规定

《大明律》专设**"凌虐罪囚"**和**"狱囚衣粮"**条款：

| 违规行为 | 处罚 |
|----------|------|
| 狱官禁卒凌虐殴伤罪囚 | 依法惩处 |
| 使用超重超规格刑具 | 依法惩处 |
| 非法使用匣床 | 处罚相关官役 |
| 苛扣囚粮 | 惩处 |
| 狱卒苦楚囚人 | 严行断罪 |

**点视制度**：提牢官须定时巡查监狱，核实在押囚犯状况。违反者：禁子严行断罪，狱官申牒上司究治。

[来源：中国反腐倡廉发展史-明朝监狱系囚管理制度；中国法学网-明清州县的监狱]

## 四、实际监狱条件

| 时期 | 实际状况 |
|------|----------|
| 洪武至永乐 | 制度执行较好 |
| 正统以后 | 开始松弛 |
| 成化至万历 | 监狱条件逐渐恶化 |
| 万历至崇祯 | "狱政开始腐败，直至明亡" |

**实际存在的问题**：
- 狱吏索贿勒索囚犯及家属
- 克扣囚粮
- 监狱拥挤，瘟疫时有发生（热审制度的存在即因此需求）
- 诏狱条件极其恶劣，"五毒"刑讯常态化
- 久押不决现象普遍
- 万历怠政时期积案如山

## 五、赦免制度

### 5.1 明代赦免的总体数据

| 统计项 | 数据 |
|--------|------|
| 明朝国祚 | **276年**（1368-1644） |
| 大赦总次数 | **52-55次**（统计口径略异） |
| 平均频率 | 约每**5.02-5.31年**一次 |

**与其他朝代对比**：

| 朝代 | 存续年数 | 大赦次数 | 平均周期 |
|------|---------|---------|---------|
| 西汉 | 约200年 | 约80次 | 2.5年 |
| 三国两晋南北朝 | 369年 | 428次 | **0.86年** |
| 唐朝 | 289年 | 184次 | 1.57年 |
| 宋朝 | 319年 | 203次 | 1.57年 |
| 元朝 | 97年 | 45次 | 2.16年 |
| **明朝** | **276年** | **52-55次** | **5.02-5.31年** |
| 清朝 | 276年 | 19次 | 14.53年 |

明代大赦频率在历代王朝中**倒数第二**（仅高于清朝），体现了"严赦慎赦"的政策取向。

[来源：中国新闻网-古今中外的大赦与特赦；江苏检察网]

### 5.2 赦免的类型

| 类型 | 说明 | 效力 |
|------|------|------|
| **大赦** | 范围最广，免除绝大多数罪犯的罪行和刑罚 | 免除刑罚执行，消灭犯罪记录 |
| **曲赦** | 赦免特定地区的罪犯 | 限定地域 |
| **别赦** | 赦免特定个人 | 限定个人 |
| **减等** | 将罪犯刑罚降低一等或数等 | 减轻而非免除 |
| **赎刑** | 允许以钱物赎罪 | 以财代刑 |

### 5.3 大赦的施行场合

| 场合类型 | 具体情形 | 频率 |
|---------|---------|------|
| **登极赦** | 新皇即位时颁布大赦 | 最为常见，每位皇帝即位必赦 |
| **改元赦** | 改元时颁赦 | 较常见 |
| **册封赦** | 册封皇后或立太子时颁赦 | 时有发生 |
| **庆典赦** | 重大军事胜利等庆典 | 较少 |
| **灾异赦** | 遭遇大的自然灾害时颁赦 | 时有发生 |
| **祥瑞赦** | 出现吉兆时颁赦 | 较少见 |

**嘉靖帝的赦免频率**：嘉靖帝在位45年，使用了约10次大赦，已算明朝皇帝中使用次数较多者。

### 5.4 大赦的法律效力

- **免除刑罚执行**：已判未执行的刑罚一律免除
- **消灭犯罪记录**：经大赦之人，刑事责任完全归于消灭
- 已执行完毕的刑罚不受影响
- 大赦令中通常会明确列出**不赦之罪**

## 六、不赦之罪（"常赦所不原"）

### 6.1 十恶不赦

"十恶"为最基本的不赦之罪：

| 序号 | 罪名 |
|------|------|
| 1 | 谋反 |
| 2 | 谋大逆 |
| 3 | 谋叛 |
| 4 | 恶逆 |
| 5 | 不道 |
| 6 | 大不敬 |
| 7 | 不孝 |
| 8 | 不睦 |
| 9 | 不义 |
| 10 | 内乱 |

### 6.2 《大明律》"常赦所不原"条款

《大明律》**首次系统规定**了"常赦所不原"的范围，远超"十恶"：

> "凡犯十恶、杀人、盗系官财物及强盗、窃盗、放火、发冢、受枉法不枉法赃、诈伪、犯奸、略人略卖、和诱人口，若奸党，及谗言左使杀人、故出入人罪，若知情故纵、听行藏匿……一应真犯，虽会赦并不原宥。"

**不赦罪类完整清单**：

| 不赦罪类 | 包含具体罪名 |
|----------|-------------|
| **十恶** | 谋反、谋大逆、谋叛、恶逆、不道、大不敬、不孝、不睦、不义、内乱 |
| **杀人** | 故杀、谋杀等人命案件 |
| **盗官财物** | 盗窃官府财物 |
| **强盗** | 武力劫掠 |
| **窃盗** | 偷窃 |
| **放火** | 故意纵火 |
| **发冢** | 盗掘坟墓 |
| **受赃** | 官吏受贿（枉法赃、不枉法赃） |
| **诈伪** | 诈骗、伪造 |
| **犯奸** | 强奸等性犯罪 |
| **略人** | 拐卖人口 |
| **奸党** | 交结朋党紊乱朝政 |
| **谗言左使杀人** | 进谗言导致他人被杀 |
| **故出入人罪** | 司法官故意错判 |
| **知情故纵** | 知情不报，纵容犯罪 |

### 6.3 "常赦所不原"的制度意义

- 中国古代法律中**首次以成文法系统规定不赦之罪的完整范围**
- 不赦范围远超此前历代的"十恶不赦"
- 体现明太祖"重典治国"思想
- 也是明代大赦频率远低于前代的重要原因——不赦范围扩大，大赦实际效力受限

### 6.4 "非常赦"的例外

> 赦书中"临时定罪名特免及减降从轻者"不受"常赦所不原"限制。

即皇帝可在赦书中**特别指名**赦免"常赦所不原"之罪，但这属于皇帝个人意志的行使，非常规操作。

## 七、特殊赦免情形

### 7.1 老幼废疾的减免

| 条件 | 处理 |
|------|------|
| 七十岁以上、十五岁以下、废疾 | 流罪以下收赎 |
| 八十岁以上、十岁以下、笃疾 | 杀人应死者奏闻取裁，余罪收赎 |
| 九十岁以上、七岁以下 | "虽有死罪不加刑" |

### 7.2 "留养承祀"制度

| 项目 | 内容 |
|------|------|
| **含义** | 死刑犯若为独子、父母年老无人赡养，可奏请"留养" |
| **条件** | 犯人为独子，父母年迈或重病，无其他子嗣赡养 |
| **结果** | 减死刑为流刑或徒刑，令其在家赡养父母 |
| **限制** | 犯十恶及"常赦所不原"者不得留养 |

### 7.3 朝审中的减免

朝审（每年霜降后）对监候死囚的处理结果分类：

| 结果类型 | 含义 | 后续 |
|----------|------|------|
| **情实** | 罪证确凿 | 奏请皇帝批准处决 |
| **缓决** | 暂缓执行 | 等下次朝审再议 |
| **可矜** | 情有可原 | 减等处罚 |
| **留养承祀** | 家有老亲需赡养 | 减免死罪 |

## 八、热审与清理监狱

### 8.1 热审制度

| 项目 | 内容 |
|------|------|
| **创立** | 永乐二年（1404） |
| **时间** | 每年小满后十日开始 |
| **目的** | 暑热将至，清理监狱防止瘟疫 |
| **对象** | 在京各监狱未决轻罪犯 |
| **处理** | 笞罪释放、徒流减等、枷号暂免、久押无据者释放 |

### 8.2 监狱拥挤的周期性问题

- 明代监狱经常出现积压大量未决犯的问题
- 热审、朝审等制度本身就说明监狱管理困境
- 万历怠政时期刑部积案如山，监狱更加拥挤
- 提牢厅专列"审查淹禁日久而不结案的罪犯"职责
