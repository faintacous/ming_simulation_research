# 审判流程

## 概述

明代司法审判实行逐级审转覆核制度，由州县初审逐级上报至中央三法司。核心特征是"审判权与决断权分离"——基层州县虽有审理权，但对徒刑以上案件无终审权，须逐级上报直至有权机关作出终审判决。死刑案件须经朝审特殊审核程序，最终由皇帝裁决。嘉靖至崇祯期间，巡按御史的司法权力不断扩张，逐渐取代按察使司成为地方司法的实际主导者。[来源：江苏法院网；汉斯学术出版社]

## 一、四级审判体系

### 1.1 总体结构

```
第一审级：州县（知县/知州）
    ↓ 徒刑以上案件上报
第二审级：府（知府/推官）
    ↓ 复核后上报
第三审级：省按察司（按察使）/ 巡按御史
    ↓ 死刑案件上报
第四审级：中央三法司（刑部/大理寺/都察院）
    ↓ 死刑奏请
皇帝裁决
```

### 1.2 各级审判权限

| 审级 | 审判机关 | 长官/品级 | 审判权限 | 决断权 |
|------|---------|----------|---------|--------|
| **第一审级** | 州、县 | 知县（正七品）/知州（从五品） | 审理一切案件 | 县决笞五十以下；州决杖八十以下 |
| **第二审级** | 府 | 知府（正四品）/推官（正七品） | 复核州县上报案件 | 决杖一百以下；徒以上继续上报 |
| **第三审级** | 按察司/巡按 | 按察使（正三品）/巡按御史（正七品） | 复审徒流案件 | 徒刑以下有终审权；死刑须上报 |
| **第四审级** | 三法司 | 刑部尚书（正二品） | 审理死刑及重大案件 | 流刑以下终审；死刑奏请皇帝 |

[来源：江苏法院网；陕西地方志]

## 二、州县初审

### 2.1 州县的司法角色

| 项目 | 内容 |
|------|------|
| **审判官** | 知县（正七品）或知州（从五品） |
| **辅助人员** | 县丞、主簿、典史 |
| **管辖范围** | 辖区内一切民事和刑事案件的初审 |
| **核心特征** | "行政与司法合一"——知县既是行政长官也是审判官 |

### 2.2 起诉方式

| 方式 | 说明 |
|------|------|
| **告诉** | 当事人向衙门递交诉状（"呈词"） |
| **举告** | 第三人举报犯罪 |
| **觉举** | 官府主动发现犯罪 |
| **登闻鼓** | 百姓击鼓直诉皇帝（户婚田土不许） |

### 2.3 刑事案件初审程序

```
报案/举报
    ↓
知县差人勘察犯罪现场（"相验"）
    ↓
缉捕嫌犯
    ↓
审讯（堂审），使用法定刑讯工具
    ↓
招供画押
    ↓
拟定判决
    ↓
笞杖刑：知县直接执行
徒刑以上：上报府级复核
```

### 2.4 民事案件初审程序

```
原告于放告日递交诉状
    ↓
知县审查诉状，决定是否受理
    ↓（受理）
签发堂票，传唤被告
    ↓
双方到堂，知县当堂审理
    ↓
查验证据（契约、文书、证人）
    ↓
当堂判决（"批词"或"堂谕"）
    ↓
不服者可上诉至府
```

### 2.5 放告日制度

| 项目 | 内容 |
|------|------|
| **含义** | 州县衙门仅在特定日期接受民事诉讼 |
| **典型安排** | 每月逢三、六、九日（或逢三、八日） |
| **非放告日** | 不受理民事案件，但重大刑案不受此限 |
| **农忙限制** | 各地官府于农忙季节减少或暂停受理民事案件 |

### 2.6 越诉的处罚

- **"若越本管官司辄赴上司称诉者，笞五十"**——即使所告属实
- 但朱元璋特许：百姓遇贪官案件可直接绑缚赴京

## 三、府级复核

### 3.1 府的司法角色

| 项目 | 内容 |
|------|------|
| **审判官** | 知府（正四品） |
| **专职司法官** | 推官（正七品），"掌推勘刑狱" |
| **管辖范围** | 复核辖区内州县上报的徒刑以上案件 |
| **权限** | 决杖一百以下；徒刑以上继续上报按察司 |
| **实际运作** | 知府不常亲审刑案，多由推官代为主持 |

## 四、提刑按察使司

### 4.1 组织编制

| 职位 | 品级 | 人数 | 职责 |
|------|------|------|------|
| **按察使** | 正三品 | 1人 | 掌一省刑名按劾 |
| **副使** | 正四品 | 无定员 | 分道巡察，设兵备、提学、巡海等专职 |
| **佥事** | 正五品 | 无定员 | 分道巡察 |

**内部下属机构**：

| 机构 | 长官 | 职能 |
|------|------|------|
| 经历司 | 经历、知事各1人 | 掌收纳文书与勘察刑名 |
| 照磨所 | 照磨、检校各1人 | 掌照刷案卷 |
| 司狱司 | 司狱1人 | 掌检察监狱 |

### 4.2 按察使的司法职能

| 职能 | 说明 |
|------|------|
| **复审权** | 复审州县、府上报的徒流死罪案件 |
| **终审权** | 徒刑以下案件有终审权 |
| **死刑上报** | 死刑案件审核后上报中央三法司 |
| **纠举** | "纠官邪，戢奸暴，平狱讼，雪冤抑，以振扬风纪" |
| **照刷案卷** | 定期检查下属机关案卷有无延迟、错误 |
| **审录罪囚** | 定期会同三法司审录在押罪囚 |

### 4.3 副使与佥事分道

| 道职 | 职掌 |
|------|------|
| 兵备道 | 兵备防务 |
| 提学道 | 教育考试 |
| 巡海道 | 海防事务 |
| 清军道 | 军籍清理 |
| 驿传道 | 驿站邮传 |
| 水利道 | 水利工程 |
| 屯田道 | 屯田管理 |

[来源：知乎-详解明朝提刑按察使司；维基百科-提刑按察使司；百度百科-按察使]

## 五、巡按御史

### 5.1 基本信息

| 项目 | 内容 |
|------|------|
| **品级** | 正七品（监察御史） |
| **派遣机构** | 都察院 |
| **巡按区域** | 每省一人（或数人） |
| **法律地位** | **"代天子巡狩"** |
| **任期** | 一年一换 |

### 5.2 司法职能

| 职能 | 具体内容 |
|------|---------|
| **审录罪囚** | "按临所至，必先审录罪囚" |
| **吊刷案卷** | 检查官府案卷有无延迟、错误、遗漏 |
| **受理上诉** | 受理百姓对地方判决不服的上诉 |
| **纠举官员** | "大事奏裁，小事立断" |
| **初审权** | 宣德以后，巡按成为初审机关之一 |
| **复核死刑** | 对地方死刑案件行使复核权 |

### 5.3 权力与品级的反差

- 仅正七品，但权势远超正三品按察使
- "代天子巡狩，所按藩服大臣、府州县官诸考察，举劾尤专"
- 州县官员迎接巡按时"迎跪道旁，倘遇风雨，即知府亦陷膝泥中"
- 正七品巡按可裁断六品官员的不法行为

### 5.4 与按察使的关系演变

| 对比项 | 按察使 | 巡按御史 |
|--------|--------|---------|
| **品级** | 正三品 | 正七品 |
| **任职性质** | 常设地方官 | 临时差遣 |
| **权力来源** | 制度规定 | "代天子巡狩" |
| **明前期** | 一省司法主官 | 巡查监督 |
| **明后期** | 权力缩减，配合巡按 | 实际主导地方司法 |

> 《大明会典》："内外大小衙门官员但有不公不法等事，在内从监察御史，在外从按察司纠举。"但到明后期，巡按御史权力不断扩张，按察使的监察职能日益缩减。[来源：汉斯学术出版社]

## 六、死刑的特殊审核程序

### 6.1 死刑复核全流程

```
州县初审 → 府复核 → 按察司/巡按复审
    ↓
上报中央三法司
    ↓
刑部审理（十三清吏司分省审理）
    ↓
大理寺复核（驳正）
    ↓
都察院监督
    ↓
死刑立决：奏请皇帝核准后执行
死刑监候：等待朝审
    ↓
朝审（每年霜降后）
    ↓
结果分类：情实/缓决/可矜/留养承祀
    ↓
情实者：奏请皇帝批准后执行
```

### 6.2 朝审制度

| 项目 | 内容 |
|------|------|
| **创立时间** | 天顺三年（1459） |
| **创立者** | 明英宗 |
| **创立缘由** | "人命至重，死者不可复生" |
| **审理时间** | 每年**霜降后**至冬至前 |
| **参与者** | 三法司长官 + 公、侯、伯等勋贵 |
| **审理对象** | 在京的死刑监候囚犯 |
| **历史意义** | 清代"秋审"制度的直接前身 |

**朝审结果分类**：

| 结果 | 含义 | 后续处理 |
|------|------|---------|
| **情实** | 罪证确凿，应予执行 | 奏请皇帝批准后处决 |
| **缓决** | 暂缓执行 | 等下次朝审再议 |
| **可矜** | 情有可原 | 建议减等处罚 |
| **留养承祀** | 家有老亲需赡养 | 减免死罪 |

### 6.3 大理寺的驳正程序

```
刑部审理完毕 → 送大理寺复核
    ↓
合规 → 执行
不当 → "照驳"（驳回刑部重审）
    ↓（三次驳回不改）
"参驳"（弹劾审理官员）
```

### 6.4 "三覆奏"与"五覆奏"

- 《大明律》规定死刑执行前须**"三覆奏"**（三次请示皇帝确认）
- 在京处决者须**"五覆奏"**
- 实际操作中多有简化

[来源：江苏法院网；维基百科-秋审制度]

## 七、番异与追驳程序

| 术语 | 含义 |
|------|------|
| **番异** | 犯人对判决不服，提出申辩 |
| **改调审理** | 番异后调换其他衙门重新审理 |
| **圆审** | 二次番异不服，由九卿会审 |
| **追驳** | 九卿评议后案情仍不明，退回原机构再审 |
| **制决** | 多次审理仍无法定案，请旨皇帝最终裁决 |

> "若亭疑谳决，而囚有番异，则改调隔别衙门问拟。二次番异不服，则具奏，会九卿鞫之，谓之圆审。至三四讯不服，而后请旨决焉。"

## 八、刑讯制度

### 8.1 法定刑讯工具

| 工具 | 规格 | 用途 |
|------|------|------|
| **讯杖** | 大头径四分五厘，小头径三分五厘，长三尺五寸 | 拷讯用，打臀腿 |
| **夹棍** | 杨木制，二根长三尺余，去地五寸，贯以铁条 | 夹腿骨 |
| **拶子** | 五根圆木，各长七寸，径围各四分五厘 | 夹手指（主要用于女犯） |

### 8.2 刑讯限制

- **拷讯次数**：总数不得超限
- **间隔要求**：两次拷讯不得相距过近
- **禁止非法刑具**：脑箍、烙铁等为"非法之具"
- **实际执行**：明中后期非法刑具屡禁不止

### 8.3 吕坤《刑戒》

万历年间官员吕坤著《刑戒》，试图从技术层面控制拷讯过度：
- 规定行刑力度标准
- 限制拷讯次数和间隔
- 强调"审当以理服人，不得专恃刑讯"

## 九、特殊审判程序

### 9.1 军职人员

- 军职人员由卫所自行审理
- 五军都督府参与军事案件审判
- 重大军事犯罪仍须报送刑部

### 9.2 僧道犯罪

- 犯罪僧道先由僧录司/道录司审理
- 重案移送有司

### 9.3 宗室犯罪

- 宗室犯罪由宗人府审理
- 重大案件奏请皇帝裁决
- 一般不适用普通刑罚

## 十、嘉靖至崇祯时期的审判实况

### 10.1 审判效率下降

| 时期 | 问题 |
|------|------|
| 嘉靖中后期 | 皇帝长期不朝，积案增多 |
| 万历怠政 | "二十八年不出宫门"，刑部积案如山 |
| 天启年间 | 厂卫绕过三法司直接处置政敌 |
| 崇祯年间 | 频繁更换刑部尚书，司法稳定性受损 |

### 10.2 巡按御史权力膨胀

- 嘉靖至崇祯期间巡按御史逐渐掌握地方司法主导权
- 按察使实际沦为巡按的配合者
- 巡按一人总揽弹劾、审案、监察多项职能
- 部分巡按以权谋私，导致地方司法腐败

### 10.3 党争对司法的影响

- 万历至天启间东林党与阉党斗争严重干扰司法
- 天启年间魏忠贤专权，诏狱成为政治迫害工具
- 崇祯帝虽勤政但疑心过重，频繁干预个案
