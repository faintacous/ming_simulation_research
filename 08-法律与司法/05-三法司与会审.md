# 三法司与会审制度

## 概述

明代中央司法机构由刑部、大理寺、都察院三个机关组成，合称"三法司"。其分工原则为"刑部受天下刑名，都察院纠察，大理寺驳正"。三者互不隶属、各向皇帝负责，形成"鼎足而三"的制衡格局。在三法司之上，明代还发展出朝审、热审、大审、圆审等多种会审制度，是中国古代司法史上的重要创新。[来源：维基百科-三法司；故宫博物院]

## 一、三法司的组成与分工

### 1.1 刑部（主审）

| 项目 | 内容 |
|------|------|
| **长官** | 刑部尚书（正二品），左右侍郎各一人（正三品） |
| **下属机构** | 十三清吏司（按省份分设：浙江、江西、湖广、陕西、广东、山东、福建、河南、山西、四川、广西、贵州、云南） |
| **各司编制** | 郎中1人（正五品）、员外郎1人（从五品）、主事1-2人（正六品） |
| **总编制** | 正七品以上官员约45人，八品以下约9人 |
| **核心职能** | 受理全国刑名案件、审理地方重案要案、审理中央各部门案件 |
| **审判权限** | 流刑以下有终审权；死刑须送大理寺复核后奏请皇帝 |
| **特殊机构** | 提牢厅：管理刑部大狱，每月轮派十三司一名主事提点 |

[来源：维基百科-三法司；知乎-明朝官职]

### 1.2 大理寺（复核）

| 项目 | 内容 |
|------|------|
| **长官** | 大理寺卿（正三品），左右少卿各一人（正四品） |
| **下属机构** | 左右二寺，各设寺丞（正六品） |
| **核心职能** | "审谳平反刑狱之政令"——复核刑部审理的案件 |
| **驳正权** | 认为刑部判决有误时可**驳回重审**（"照驳"） |
| **参驳权** | 三次驳回拟议不符，可弹劾审理官员（"参驳"） |
| **地位** | 不直接审案，专司复核，相当于"上诉法院" |

### 1.3 都察院（监察）

| 项目 | 内容 |
|------|------|
| **长官** | 左右都御史（正二品），左右副都御史（正三品），左右佥都御史（正四品） |
| **下属机构** | 十三道监察御史（正七品），每道7-11人不等，共约110人 |
| **核心职能** | "纠劾百官，辨明冤枉" |
| **司法监督** | 与刑部尚书、大理寺卿共审重案；纠举刑部、大理寺审理不当的案件；巡按监察御史审录罪囚、吊刷案卷 |

### 1.4 三法司的工作关系

```
案件审理流程：
地方案件 → 刑部审理（初审/终审流刑以下）
              ↓
         大理寺复核（驳正）
              ↓（如有不当）
         驳回刑部重审
              ↓
         都察院全程监督
              ↓（死刑案件）
         奏请皇帝裁决
```

**相互制约关系**：
- 刑部审判 → 大理寺复核 → 都察院监督
- 三者互不隶属，各向皇帝负责
- "鼎足而三"的制衡格局

[来源：维基百科-三法司；知乎-在明朝刑部都察院大理寺的职能区别]

## 二、会审制度

### 2.1 三司会审（常规会审）

| 项目 | 内容 |
|------|------|
| **参与者** | 刑部尚书、大理寺卿、左都御史 |
| **审理对象** | 重大疑难案件 |
| **法律依据** | "凡遇有大狱重囚，三法司会审" |
| **程序** | 三法司长官共同审理，合议定罪 |
| **频率** | 随案进行 |

### 2.2 朝审

| 项目 | 内容 |
|------|------|
| **创立时间** | 天顺三年（1459），英宗下令 |
| **制度缘起** | "人命至重，死者不可复生" |
| **审理时间** | 每年**霜降后**至冬至前 |
| **参与者** | 三法司 + 公、侯、伯等勋贵 |
| **审理对象** | 在京的死刑监候囚犯 |
| **性质** | 死刑复核的核心制度，兼具**缓刑**和**减刑**功能 |
| **历史意义** | 清代"秋审"制度的直接前身 |

**结果分类**：

| 结果 | 含义 | 后续处理 |
|------|------|---------|
| **情实** | 罪证确凿，应予执行 | 奏请皇帝批准处决 |
| **缓决** | 暂缓执行 | 等下次朝审再议 |
| **可矜** | 情有可原 | 减刑 |
| **留养承祀** | 为赡养父母或延续香火 | 减免死罪 |

### 2.3 热审

| 项目 | 内容 |
|------|------|
| **创立时间** | 永乐二年（1404） |
| **审理时间** | 每年**小满后十日**开始 |
| **参与者** | 刑部主导 + 都察院 + 锦衣卫 + 大理寺 |
| **审理对象** | 在京各监狱中**未决轻罪犯** |
| **制度目的** | 暑热将至，清理监狱以防瘟疫流行 |
| **处理方式** | 笞罪者释放、徒流减等发落、枷号暂免、久押无实据者释放 |
| **性质** | 兼具恤刑与疏通监狱之功能 |

### 2.4 大审

| 项目 | 内容 |
|------|------|
| **创立时间** | 成化十七年（1481） |
| **审理周期** | **每五年一次** |
| **参与者** | **司礼监太监一员**（奉旨主持） + 三法司堂上官 |
| **审理地点** | 大理寺 |
| **审理对象** | 南北两京未决案件 |
| **仪式规格** | "赍敕张黄盖于大理寺，为三尺坛，中坐"——太监居中高坐，三法司左右坐，御史、郎中以下捧牍立 |
| **实际权力** | "三法司视成案，有所出入轻重，俱视中官意，不敢忤也"——须看太监脸色 |
| **南京** | 命内守备行之 |
| **性质** | 皇帝通过太监直接干预司法的制度化体现 |

### 2.5 圆审（九卿会审）

| 项目 | 内容 |
|------|------|
| **参与者** | 三法司长官 + 吏、户、礼、兵、工五部尚书 + 通政使，合称"九卿" |
| **审理对象** | **特别重大疑难案件**或囚犯反复翻供（"番异"）的案件 |
| **启动条件** | 案件经两次审理犯人仍"番异不服" |
| **程序** | "若亭疑谳决，而囚有番异，则改调隔别衙门问拟。二次番异不服，则具奏，会九卿鞫之" |
| **后续** | "至三四讯不服，而后请旨决焉"——请皇帝最终裁决（"制决"） |

### 2.6 各类会审对比

| 会审类型 | 创立时间 | 频率 | 主持者 | 对象 | 核心特征 |
|----------|----------|------|--------|------|----------|
| 三司会审 | 明初 | 随案 | 三法司长官 | 重大案件 | 常规会审 |
| 朝审 | 天顺三年（1459） | 每年秋冬 | 三法司 + 勋贵 | 死刑监候 | 死刑复核 |
| 热审 | 永乐二年（1404） | 每年夏季 | 刑部主导 | 轻罪未决 | 恤刑清狱 |
| 大审 | 成化十七年（1481） | 每五年 | **太监**主持 | 南北京未决 | 皇权干预 |
| 圆审 | 制度渐成 | 随案 | 九卿 | 疑难翻供 | 最高级别 |

## 三、驳正与追驳程序

### 3.1 驳正程序

```
刑部审理 → 大理寺复核
              ↓
         合规 → 执行
         不当 → "照驳"（驳回重审）
              ↓（三次驳回不改）
         "参驳"（弹劾审理官员）
```

### 3.2 番异与追驳

- **番异**：犯人对判决不服，提出申辩
- **改调审理**：番异后调换其他衙门重新审理
- **圆审**：二次番异不服，九卿会审
- **追驳**：九卿评议后案情仍不明，退回原机构再审
- **制决**：多次审理仍无法定案，请旨皇帝裁决

## 四、嘉靖至崇祯时期的会审实践

### 4.1 制度运行的异变

| 时期 | 问题 |
|------|------|
| 嘉靖中后期 | 皇帝长期不朝，会审流于形式 |
| 万历年间 | "万历怠政"，刑部积案如山，会审无法及时举行 |
| 天启年间 | 厂卫绕过三法司直接处置政敌，会审制度被架空 |
| 崇祯年间 | 频繁更换刑部尚书，司法稳定性受损 |

### 4.2 大审中太监权力的膨胀

- 大审制度本身就是皇权干预司法的产物
- "三法司视成案……俱视中官意"——三法司须看太监脸色
- 太监居中高坐，三法司左右侍坐，御史郎中以下捧牍而立
- 这一制度深刻体现了明代宦官干政的特征

### 4.3 党争对司法的影响

- 万历至天启间东林党与阉党斗争严重干扰司法
- 天启年间魏忠贤专权，"钩党之狱"大肆迫害东林党人
- 崇祯时期虽清除阉党，但党争余波仍影响司法公正
