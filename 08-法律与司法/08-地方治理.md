# 地方治理

## 概述

明代基层治理体系由里甲制、保甲制（十家牌法）、乡约制度和申明亭/旌善亭四大支柱构成。里甲制解决赋役征发（经济管理），保甲制解决社会治安（安全控制），乡约解决道德教化（思想引导），申明亭/旌善亭则是三者的实体空间载体。嘉靖至崇祯期间，里甲制因一条鞭法改革和人口流动而趋于瓦解，保甲和乡约在部分地区仍有活力，宗族力量则在地方治理中日益重要。[来源：求是网-明代乡村治理体系研究]

## 一、里甲制度

### 1.1 创设与基本编制

| 项目 | 内容 |
|------|------|
| **创设时间** | 洪武十四年（1381） |
| **法律依据** | 《大明令》及相关诏令 |
| **编制原则** | 以**110户**为一里 |
| **直接目的** | 征发赋役 |
| **户籍登记** | 每十年编造一次黄册 |

#### 编制结构

```
一里（110户）
├── 里长 × 10户（丁粮最多的10户轮充）
└── 十甲 × 100户
    └── 每甲 10户，设甲首1人
```

#### 各角色职责

| 角色 | 选任标准 | 人数 | 职责 |
|------|----------|------|------|
| **里长** | 丁粮最多的10户 | 10人（轮流） | 总管一里赋税催征、差役编派 |
| **甲首** | 每甲10户中推选 | 10人 | 管理本甲事务、催征本甲赋税 |

### 1.2 里长轮役制度

| 项目 | 内容 |
|------|------|
| **轮役周期** | 10年为一周期 |
| **轮役顺序** | 按丁粮多少排序，丁粮最多者先当值 |
| **当值职责** | 每年1名里长率领10名甲首应当差役 |
| **核心任务** | "追征钱粮"——催缴本里税赋 |

### 1.3 畸零户制度

各里中无力承担差役的鳏寡孤独人户，编为**畸零户**，带管于110户之外：
- 年老、残疾者
- 十岁以下幼童
- 寡妇
- 外郡寄庄人户

### 1.4 里甲的多重职能

| 职能类别 | 具体内容 |
|----------|----------|
| **赋税催征** | 首要职能，里长甲首督促百姓缴纳税款 |
| **户籍编造** | 协助编造黄册（户口册），十年一造 |
| **治安维持** | 维护本里社会秩序 |
| **教化推行** | 宣讲圣谕、传达政令 |
| **纠纷调处** | 配合老人调处户婚田土等民事纠纷 |
| **差役征发** | 编排里内人户应役 |

### 1.5 里甲制的衰落

| 时期 | 问题 |
|------|------|
| 明中叶 | 人口流动加剧，原编110户逐渐不足 |
| 嘉靖后 | 赋役不均导致里长甲首负担过重，多有逃亡 |
| 万历 | 一条鞭法改革后，里甲征收赋役功能逐渐被取代 |
| 明末 | 里甲作为基层组织形式延续，但实质功能大减 |

[来源：百度百科-里甲制度；维基百科-里甲；求是网]

## 二、保甲制度（十家牌法）

### 2.1 创设背景

| 项目 | 内容 |
|------|------|
| **创设者** | 王守仁（王阳明） |
| **创设时间** | 正德十二年（1517） |
| **创设地点** | 南赣（今江西赣州、南安一带） |
| **创设背景** | 王阳明任南赣巡抚期间，为平定山区匪患、区分"盗"与"民"而推行 |

### 2.2 基本编制

| 层级 | 单位 | 户数 | 负责人 |
|------|------|------|--------|
| 牌 | 十户为一牌 | 10户 | 牌长 |
| 甲 | 十牌为一甲 | 100户 | 甲长 |
| 保 | 十甲为一保 | 1000户 | 保正 |

### 2.3 门牌登记内容

每户门前须悬挂一面小牌，查实造册报官备用：

| 登记项目 | 说明 |
|----------|------|
| 人丁数量 | 本户男女丁口总数 |
| 身份类别 | 某丁为官吏/生员/当差役 |
| 职业 | 习某技艺、作某生理 |
| 婚姻变动 | 过继、入赘等情况 |
| 残疾情况 | 有无残疾 |
| 田粮户籍 | 户籍类别及田粮数额 |

### 2.4 巡查与连坐

**巡查制度**：
- 牌长**每日**挨家挨户核实人口信息
- 轮流巡查监察
- 发现可疑情况**立即报官**

**连坐制度**：
- "如有隐匿者，十家连坐"
- "一家为匪不法及窝藏盗贼，九家公同出首，毋得容隐"
- 十家中一家犯法而其余九家不举报，则九家连坐

### 2.5 十项禁令

| 序号 | 禁令内容 |
|------|----------|
| 1 | 严禁拜盟结党 |
| 2 | 行踪诡秘 |
| 3 | 窝盗藏娼 |
| 4 | 赌博 |
| 5 | 酗酒斗殴 |
| 6 | 不务正业 |
| 7 | 私藏兵器 |
| 8 | 聚众闹事 |
| 9 | 传播谣言 |
| 10 | 容留来路不明之人 |

### 2.6 推广与影响

- 王阳明南赣推行后效果显著，山区匪患大为减少
- 后被推广至其他地区
- 清代沿用并发展：雍正四年（1726）确立"十户一牌头"体系

[来源：数字王阳明资源库-十家牌法；知乎-王阳明的十家牌法]

## 三、乡约制度

### 3.1 历史渊源

| 时期 | 乡约发展 |
|------|----------|
| 北宋 | 吕大钧《吕氏乡约》——最早的乡约（1076年） |
| 明正德 | 王阳明《南赣乡约》——将乡约与保甲结合 |
| 明万历 | 吕坤《乡甲约》——进一步制度化 |
| 清代 | 圣谕宣讲 + 乡约合一 |

### 3.2 王阳明《南赣乡约》

#### 基本信息

| 项目 | 内容 |
|------|------|
| **颁布时间** | 正德十三年（1518） |
| **条文数** | **十六条** |
| **施行地区** | 南赣地区 |
| **核心理念** | 通过集体监督和自我反省改造民众品格 |

#### 三部分结构

| 部分 | 条文 | 内容 |
|------|------|------|
| 第一部分 | 前5条 | 组织建设：人员构成、文簿设置、会费、聚会日期、约所选择 |
| 第二部分 | 第6-15条 | 对同约人、寄庄人户、客商、亲族、军民等各类人员及情况的要求 |
| 第三部分 | 第16条 | 聚会仪式 |

#### 组织架构

| 职位 | 人数 | 选任标准 | 职责 |
|------|------|----------|------|
| **约长** | 1人 | 年高有德，为众所敬服 | 掌管全局，调度善恶簿册 |
| **约副** | 2人 | 同约长条件 | 辅助约长 |
| **约正** | 4人 | 公直果断者 | 质证事实，裁定善恶 |
| **约史** | 4人 | 通达明察者 | 记录善行与过失 |
| **知约** | 4人 | 精健廉干者 | 管理名册和日常行为记录 |
| **约赞** | 2人 | 礼仪习熟者 | 主持仪式程序 |

#### 彰善与纠过机制

| 机制 | 方法 |
|------|------|
| **彰善** | 用明确措辞表扬善行，公开张扬 |
| **纠过** | 用隐晦措辞记录过失，避免激化矛盾 |
| **改过程序** | 先私下劝阻 → 集体劝导 → 正式纠正 |

#### 处罚体系

| 级别 | 处罚 |
|------|------|
| 轻微 | 无故缺会罚银一两 |
| 自首改正 | 仅记录，不追究 |
| 不改 | 书入纠过簿册 |
| 严重 | 呈报官府 |
| 恶劣 | 执送官府 |
| 极恶 | 请兵剿杀 |

**不参加乡约者罚银1两**——强调全覆盖。

[来源：百度百科-南赣乡约；数字王阳明资源库；儒家网]

### 3.3 吕坤《乡甲约》（万历年间）

| 项目 | 内容 |
|------|------|
| **创新** | 将乡约与保甲制度正式结合 |
| **编制** | "以一里为率，各立约正一人，约副一人" |
| **特点** | "约"成为一个组织单元，设约正、约副、约讲、约史等 |

## 四、申明亭与旌善亭

### 4.1 制度创设

| 项目 | 内容 |
|------|------|
| **创设时间** | 洪武五年（1372） |
| **创设者** | 明太祖朱元璋 |
| **设置范围** | 城乡各坊里厢，"设申明亭处，也必设旌善亭" |

### 4.2 申明亭

| 功能 | 内容 |
|------|------|
| **公示惩戒** | "凡境内之人民有犯者，书其过，名榜于亭上" |
| **纠纷调解** | 里中老人在申明亭调处户婚田土等民事纠纷 |
| **法律宣讲** | 定期宣读并讲解《大诰》《大明律》《教民榜》 |
| **教化** | 宣讲圣谕六事："孝顺父母，尊敬长上，和睦乡里，教训子弟，各安生理，毋作非为" |

### 4.3 旌善亭

| 项目 | 内容 |
|------|------|
| **功能** | 张贴善人善事，表彰嘉行 |
| **与申明亭配合** | "彰善瘅恶"——申明亭惩恶，旌善亭扬善 |

### 4.4 管理人员

| 角色 | 选任 | 职责 |
|------|------|------|
| **老人** | 每里推选年高有德者3-10人 | 掌管申明亭事务，调处纠纷，宣讲教化 |
| **里长** | 丁粮最多之户 | 襄助老人，协管亭务 |

### 4.5 老人的具体职能

| 职能 | 详细内容 |
|------|---------|
| **法律教化** | 定期宣读《大诰》《大明律》《教民榜》，宣讲圣谕六事 |
| **纠纷调解** | "民间户婚田土斗殴相争一切小事，不许辄便告官，务要经由本管里甲老人理断" |
| **品行约束** | 酷似秦汉"三老"，尊为"方巾御史" |

### 4.6 衰落过程

| 时期 | 状况 |
|------|------|
| 洪武至永乐 | 制度运行较好 |
| 宣德以后 | 开始出现弊端 |
| 弘治至嘉靖 | "亭宇不修，善恶不书" |
| 万历以后 | 大多形同虚设 |

衰落原因：地方官玩忽职守、里老以权谋私、亭宇年久失修、民事纠纷转向官府诉讼。

[来源：百度百科-申明亭；百度百科-旌善亭；人民论坛]

## 五、宗族在地方治理中的角色

### 5.1 宗族组织的兴起

| 时期 | 发展状况 |
|------|---------|
| 明初 | 宗族力量较弱，国家直接控制基层 |
| 明中叶 | 士大夫开始宗族组织建设 |
| 嘉靖至万历 | 修族谱、建祠堂、制定族规的高峰期 |
| 明末 | 宗族成为地方治理的重要力量 |

### 5.2 宗族的治理功能

| 功能 | 内容 |
|------|------|
| **纠纷调处** | 族长依族规裁断族内纠纷 |
| **赋税协助** | 族田收入用于代缴族人税赋 |
| **教育** | 设立族学，资助族人应举 |
| **赈济** | 族田收入用于赈济困难族人 |
| **治安** | 族规约束族人行为，维护秩序 |
| **祭祀** | 祠堂祭祀凝聚族人认同 |

### 5.3 族规的法律效力

| 项目 | 内容 |
|------|------|
| **法律地位** | 非官方法律，但得到官府默认甚至支持 |
| **内容** | 包括孝悌规范、婚丧礼仪、财产分配、惩戒措施 |
| **执行** | 由族长在祠堂执行，可施加体罚（鞭笞）、逐出宗族等惩罚 |
| **与官府关系** | 嘉靖至万历间，官府多次批准族规族训以支持宗族控制 |

### 5.4 宗族的经济基础

| 类型 | 用途 |
|------|------|
| **族田** | 收入用于税赋、祭祀、赈济 |
| **义田** | 资助贫困族人和科举备考 |
| **祠产** | 维护祠堂和祭祀活动 |

[来源：求是网；中国社会科学网-明代家族社会转型]

## 六、各项制度的协同关系

```
明代基层治理体系：

里甲制（行政基础）
├── 赋税催征
├── 户籍管理
└── 差役编派

保甲制/十家牌法（治安保障）
├── 户口核查
├── 巡查监察
└── 连坐互保

乡约（道德教化）
├── 彰善纠过
├── 集体约束
└── 风俗改良

申明亭/旌善亭（宣教与调处）
├── 公示善恶
├── 法律宣讲
└── 纠纷调解

宗族（社会自治）
├── 族内纠纷调处
├── 经济互助
└── 教化约束
```
