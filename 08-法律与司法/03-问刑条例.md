# 问刑条例

## 概述

《问刑条例》是明代为弥补《大明律》的僵化与滞后而制定的补充性法规。自弘治十三年（1500）首次正式编定以来，经嘉靖二十九年（1550）、万历十三年（1585）两次大修，条数从279条增至385条再至382条。万历版以"律为正文，例为附注"的形式编入《大明律集解附例》，开创了中国古代"律例合编"的体例。问刑条例在司法实践中逐渐取代律典成为主要裁判依据，"前后达一百四十余年之久"（1500-1644）。[来源：百度百科-问刑条例；百度百科-大明律]

## 一、条例的产生背景

### 1.1 律典僵化的困境

- 洪武三十年（1397）《大明律》定本后，朱元璋下令"一字不可更改"
- 社会经济在明中叶发生巨大变化，律典无法应对新型犯罪和社会问题
- 各级司法机关在实践中援引先例、皇帝谕旨、榜文等作为裁判依据
- 这些散见各处的"例"缺乏系统整理，司法标准不一，"中外问刑衙门引例配比，多失本意"

### 1.2 从散例到成编

| 阶段 | 时间 | 说明 |
|------|------|------|
| 散例期 | 洪武至弘治前 | 皇帝诏谕、部院则例散见各处，未成体系 |
| 首次动议 | 弘治五年（1492） | 刑部尚书彭韶等谏言删定条例 |
| 正式编成 | 弘治十三年（1500） | 刑部尚书白昂主持删定，编成279条 |

[来源：百度百科-问刑条例；中国法学网-明代法律概述]

## 二、三次重大修订

### 2.1 弘治《问刑条例》（279条）

| 项目 | 内容 |
|------|------|
| **编定时间** | 弘治十三年（1500） |
| **主持人** | 刑部尚书白昂 |
| **条数** | **279条** |
| **编纂方式** | 从累积的散例中删除重复矛盾者，整理编定 |
| **施行时长** | 约50年，至嘉靖二十九年修订 |
| **法律效力** | 与《大明律》并行，作为审判的补充依据 |

**重要内容方向**：
- 扩大充军刑适用范围：文武官吏及军民百姓皆可充军
- 确立口外为民的闰刑制度
- 规范枷号的期限和执行方式
- 涵盖盗贼、人命、斗殴、户婚田土、赋役、军政等方面

[来源：百度百科-问刑条例；中国法学网-明代法律概述]

### 2.2 嘉靖《问刑条例》（385条）

| 项目 | 内容 |
|------|------|
| **时间** | 嘉靖二十七年至二十九年（1548-1550），三十四年续增9条 |
| **主持人** | 刑部尚书顾应祥（初修）、喻茂坚（续修）、何鳌（续增） |
| **条数** | 嘉靖二十九年定**385条**（一说376条），嘉靖三十四年续增9条 |
| **修例背景** | 弘治条例施行已久，"问刑衙门于律例之外又增比附" |
| **增删情况** | 净增百余条 |

**嘉靖修例中主要增改方向**：

| 领域 | 新增条例内容 | 背景 |
|------|-------------|------|
| **倭寇** | 通倭罪加重处罚，充军或斩 | 嘉靖年间东南沿海倭患严重 |
| **科举** | 科场舞弊夹带加重处罚 | 科举腐败问题加重 |
| **边防** | 边关走私、私通外族的处罚 | 北虏南倭的安全压力 |
| **宗室** | 宗室犯罪的特别处理规定 | 宗室人口膨胀、问题增多 |
| **赋税** | 拖欠赋税的强化处罚 | 财政压力增大 |
| **盗贼** | 结伙抢劫、杀人越货加重 | 社会治安恶化 |

[来源：百度百科-问刑条例；CSDN-法制史学习笔记]

### 2.3 万历《问刑条例》（382条）

| 项目 | 内容 |
|------|------|
| **时间** | 万历二年至十三年（1574-1585） |
| **主持人** | 刑部尚书舒化等 |
| **条数** | **382条** |
| **重大创新** | 以**"律为正文，例为附注"**形式，将条例附于律条正文之后 |
| **编入** | 《大明律集解附例》 |
| **充军条例占比** | 约189条，占全部条例的**49.5%** |
| **施行** | 至明末（1644）未再修改，约60年 |
| **历史意义** | 开创中国古代**律例合编**体例，为清代沿用 |

**编排方式**：
- 将382条例文逐条附编在相关律条正文之后
- 例文对律文起**补充**、**解释**和**修正**作用
- 律条中未涉及的新型犯罪由条例填补
- 律条量刑过轻或过重者由条例调整

[来源：百度百科-大明律；北京政法网]

### 2.4 三次修订对比

| 对比项 | 弘治（1500） | 嘉靖（1550） | 万历（1585） |
|--------|-------------|-------------|-------------|
| **条数** | 279条 | 385条 | 382条 |
| **间隔** | — | 50年 | 35年 |
| **编排形式** | 单独成书 | 单独成书 | **律例合编** |
| **增幅** | — | +106条（+38%） | -3条（微调） |
| **主要方向** | 整理散例 | 应对嘉靖新情况 | 体例创新 |
| **施行至** | 嘉靖二十九年 | 万历十三年 | 明末（1644） |

```
弘治十三年（1500）：279条
      ↓ +106条
嘉靖二十九年（1550）：385条
      ↓ -3条
万历十三年（1585）：382条
      ↓ 此后至明末未改
```

## 三、律例关系

### 3.1 "律为正文，例为附注"

万历十三年后律例关系的基本原则：

| 维度 | 律（大明律） | 例（问刑条例） |
|------|------------|--------------|
| **性质** | 基本法、正文 | 补充法、附注 |
| **稳定性** | 一字不改（240余年不变） | 可修订增删（约50年一修） |
| **地位** | 法律效力的根本来源 | 对律的补充、解释和修正 |
| **编排** | 正条 | 附于相关律条之后 |
| **司法适用** | 基本准则 | 优先适用（新法优于旧法） |

### 3.2 例对律的三种功能

| 功能 | 说明 | 举例 |
|------|------|------|
| **补充** | 律中未规定的新型犯罪由例补充 | 通倭罪、科场舞弊 |
| **解释** | 对律文模糊处进行具体化 | 明确"强盗"的具体构成要件 |
| **修正** | 实际调整律文的量刑轻重 | 加重或减轻特定罪名的刑罚 |

### 3.3 实践中的"以例破律"

- 明中后期条例在司法实践中的重要性逐渐超过律典
- 司法官习惯先查条例、后查律文
- 条例虽名为"附注"，实质上成为审判主要依据
- 律与条例矛盾时**条例优先**（"例可破律"）
- 律例均无规定时**比附援引**最相近条文

[来源：百度百科-问刑条例；北京政法网；中国法学网]

## 四、条例的类型与典型内容

### 4.1 条例来源分类

| 来源类型 | 说明 |
|----------|------|
| **皇帝敕令** | 皇帝对具体案件的裁决被整理为通行条例 |
| **部院议覆** | 刑部等对案件处理意见经皇帝批准后形成条例 |
| **审判惯例** | 长期适用的判例升格为成文条例 |

### 4.2 充军类条例（占比最大）

充军条例在三次修例中持续膨胀，反映明中后期社会治安恶化与刑罚加重趋势：

| 时期 | 充军罪名条数 |
|------|-------------|
| 洪武初年 | 22款 |
| 嘉靖年间 | 213款 |
| 万历年间 | 243款 |
| 万历条例382条中 | 约189条（**占49.5%**） |

典型充军条例：
- **窝藏逃军**：知情窝藏逃军者充军
- **私出外境**：私出外境及违禁下海者充军
- **聚众斗殴致死**：为首者充军

### 4.3 贪赃类条例

- 官吏受财枉法：赃满一定数额加重处罚
- 科敛害民：官吏于额外科取民财者加重处罚

### 4.4 盗贼类条例

- 强盗、窃盗的具体量刑标准
- 累犯加重的具体规定

## 五、万历至崇祯的司法实况

### 5.1 条例的再度僵化

万历十三年后382条例文未再正式修订，至崇祯末年已沿用近60年。这导致：
- 旧例不能应对明末社会巨变（流民、民变等）
- 皇帝通过诏令、谕旨对个案作特别裁决
- 刑部等机构自行制定部院则例
- 无明确律例依据时"比附援引"

### 5.2 条例之外的法律渊源

| 法源 | 说明 |
|------|------|
| **皇帝谕旨** | 对个案的特别指示，最高法律效力 |
| **部院则例** | 各部门自行制定的操作规范 |
| **《大明令》** | 洪武年间颁布的行政法令 |
| **《大明会典》** | 行政法规汇编，含大量司法行政规范 |
| **《大诰》** | 洪武年间颁布，后渐废弛但未正式废除 |
| **《教民榜文》** | 基层治理的道德和法律规范 |

### 5.3 修例程序

```
刑部尚书上疏请求修例
    ↓
皇帝批准
    ↓
刑部主持，会同大理寺、都察院参与
    ↓
删除过时条款、增补新条款、调整量刑
    ↓
上奏审批
    ↓
皇帝批准颁行
```

## 六、历史影响

### 6.1 对清代的影响

- 清代《大清律例》直接沿用明代律例合编体例
- 清律以《大明律》为蓝本，条文大量继承
- 清代条例更加频繁修订，五年一小修、十年一大修
- "律例合编"体例延续至清末修律

### 6.2 律例体系的得失

| 优点 | 缺点 |
|------|------|
| 弥补律典僵化的不足 | 条例过多可能造成法律繁杂 |
| 灵活应对社会变化 | "以例破律"破坏法律稳定性 |
| 为司法官提供具体指引 | 条例之间可能相互矛盾 |
| 开创律例合编先例 | 修例受政治博弈影响 |
