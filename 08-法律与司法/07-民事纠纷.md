# 民事纠纷

## 概述

明代将民事纠纷统称为"户婚田土"或"细事"，与刑事案件（"人命""盗贼"等"重事"）严格区分。民事纠纷实行"先民间调处、后官府审理"的双重管辖体制：朱元璋在《教民榜文》中明确规定户婚田土等小事须先经里甲老人调处，调处不成方可赴官告状。官府设有放告日制度，仅在特定日期接受民事案件。民间的讼师虽为百姓维权所需，却被官方贬为"讼棍"，处于法律打压之下。[来源：最高人民检察院；百度百科-讼师]

## 一、民事案件的分类与管辖

### 1.1 "户婚田土"的范畴

| 类别 | 包含内容 |
|------|----------|
| **户** | 户籍争议、分家析产、继承纠纷、过继争议 |
| **婚** | 婚约效力、退婚、妆奁归属、义绝（离婚）、重婚 |
| **田** | 土地买卖纠纷、田产边界争议、租佃关系、典当回赎 |
| **土** | 宅基纠纷、坟地争议、水利灌溉分配 |
| **斗殴** | 轻微人身冲突（未致伤残） |
| **钱债** | 借贷纠纷、商业欠款 |

### 1.2 双重管辖体制

| 层级 | 管辖机构 | 法律依据 |
|------|----------|----------|
| **第一层：里甲调解** | 里甲老人 | 《教民榜文》（洪武三十一年） |
| **第二层：州县审理** | 知县/知州 | 《大明律》 |

## 二、基层调解：里甲老人制度

### 2.1 《教民榜文》的核心规定

朱元璋在《教民榜文》中明确规定：

> "民间户婚、田土、斗殴、相争，一切小事，须要经由本里老人、里甲断决。若系奸盗诈伪、人命重事，方许赴官陈告。"

这意味着：
- 户婚田土等民事纠纷**不许直接告官**
- 必须先由里甲老人调解裁断
- 只有涉及奸盗诈伪、人命的重案才能直接赴官
- 未经老人调处即赴官告状者，**即使有理亦要处罚**

[来源：中国民俗学网-《教民榜文》与明太祖基层社会治理方略]

### 2.2 老人制度

| 项目 | 内容 |
|------|------|
| **选任条件** | 年五十以上、有德望、处事公正、为众所敬服 |
| **选任方式** | 每里推选三名、五名或十名老人 |
| **地位** | 类似秦汉的"三老"，掌教化与纠纷调处，尊为"方巾御史" |
| **权力** | 独立裁断户婚田土等民事和部分轻微刑事案件 |
| **场所** | 申明亭（设于各里社） |

### 2.3 调解程序

1. 当事人向本里老人陈述纠纷
2. 老人召集双方在申明亭调处
3. 调处成功则结案，记录在册
4. 调处不成，老人出具文书准许当事人赴官告状
5. 未经老人调处即赴官告状者，**即使有理亦要处罚**

### 2.4 老人制度的衰落

| 时期 | 状况 |
|------|------|
| 洪武至永乐 | 制度运行较好 |
| 宣德以后 | 开始出现弊端 |
| 弘治至嘉靖 | "里老也不自尊重，以权谋私" |
| 万历以后 | 大多形同虚设，民事纠纷转向直接告官 |

[来源：求是网-明代乡村治理体系研究；人民论坛]

## 三、官府审理

### 3.1 放告日制度

| 项目 | 内容 |
|------|------|
| **含义** | 州县衙门仅在特定日期接受民事案件的告状 |
| **法律依据** | 各地官府自行规定，无统一立法 |
| **典型安排** | 每月逢三、六、九日（或逢三、八日）为放告日 |
| **操作方式** | 放告日当天知县升堂，受理百姓递交的诉状 |
| **非放告日** | 不受理民事案件，但重大刑案不受此限 |
| **农忙限制** | 各地官府于农忙季节减少或暂停受理民事案件（清代正式制度化为"农忙止讼"） |

### 3.2 诉状要求

| 项目 | 规定 |
|------|------|
| **格式** | 须按规定格式书写，载明原告、被告、事由 |
| **字数限制** | 诉状文字不得过长 |
| **代书** | 不识字者可请人代书，但代书者须实名 |
| **禁止事项** | 不得"增减情节"，否则代书者与当事人同罪 |

### 3.3 审理程序

```
原告递交诉状（放告日）
    ↓
知县/推官审查诉状，决定是否受理
    ↓（受理）
签发传票（"堂票"），传唤被告
    ↓
双方到堂，知县/推官当堂审理
    ↓
查验证据（契约、文书、证人）
    ↓
当堂判决（"批词"或"堂谕"）
    ↓
不服者可上诉至府
```

### 3.4 越诉的处罚

《大明律》明确禁止越级告状：
- **"若越本管官司辄赴上司称诉者，笞五十"**——即使所告属实
- 但朱元璋特许百姓在贪官案件中可直接绑缚赴京

## 四、各类民事纠纷的处理

### 4.1 田土纠纷

| 类型 | 处理方式 | 法律依据 |
|------|---------|---------|
| **田产买卖纠纷** | 查验契约（"红契"即官府钤印契约），以契约为准 | 《大明律·户律·田宅》 |
| **边界争议** | 查验鱼鳞图册（田亩册）确认四至 | 鱼鳞图册 + 实地勘量 |
| **租佃纠纷** | 依约定处理，佃户拖欠租金可追索 | 习惯法 + 律例 |
| **典当回赎** | 典期内允许回赎，过期则归典主 | 《大明律·户律》 |

### 4.2 婚姻纠纷

| 类型 | 处理方式 | 法律依据 |
|------|---------|---------|
| **退婚** | 须有法定理由（如疾病隐瞒、身份不符等） | 《大明律·户律·婚姻》 |
| **重婚** | 有夫之妇再嫁者杖九十，后夫离异归前夫 | 《大明律》 |
| **义绝** | 夫妻双方符合义绝条件者须离婚 | 律例 |
| **妆奁争议** | 妆奁归女方所有，离婚或夫亡后可带走 | 习惯法 |

### 4.3 债务纠纷

| 类型 | 处理方式 | 法律依据 |
|------|---------|---------|
| **借贷** | 以借据为凭，到期不还可告官追索 | 《大明律·户律·钱债》 |
| **利息限制** | 月利不得超过三分 | 《大明律》 |
| **违禁取利** | 超过法定利率者杖四十 | 《大明律》 |

## 五、讼师

### 5.1 讼师的角色

| 项目 | 内容 |
|------|------|
| **职业内容** | 代写诉状、教当事人应对审讯、提供法律建议 |
| **社会评价** | 官方视为"讼棍""哗徒"，属"下九流" |
| **民间需求** | 实际不可或缺，百姓依赖讼师维权 |
| **矛盾地位** | 官方贬为"讼棍"，民间奉为"貔貅"（守护者） |

### 5.2 《大明律》对讼师的惩罚

| 罪名 | 刑罚 |
|------|------|
| 教唆词讼 | 视同诬告论处 |
| 代书增减情节 | 与当事人同罪 |
| 投递匿名文书告人 | 官府不得受理，投递者杖八十 |
| 私自承揽词讼 | 杖八十 |

### 5.3 讼师的实际状况

- 明中后期讼师大量涌现，与商品经济发展和科举落第知识分子增多有关
- 讼师秘本广泛流传，教授诉讼技巧
- 官府反复严禁但屡禁不止
- 群体中确有借机敲诈勒索者，但也有帮助弱者维权者

[来源：百度百科-讼师；中国法律文化网]

## 六、刑名师爷（幕友）

### 6.1 产生原因与角色

| 项目 | 内容 |
|------|------|
| **产生原因** | 科举以四书五经和八股文取士，官员缺乏法律实务知识 |
| **兴起时间** | 明朝中后期出现，清代鼎盛 |
| **聘请方式** | 地方官员**私人聘请**，非正式官职，费用自掏 |
| **法律地位** | 非官非吏，属"幕宾"（私人顾问） |

### 6.2 刑名师爷的职责

| 职责 | 说明 |
|------|------|
| **审阅诉状** | 代官员审查民事和刑事诉状 |
| **拟定判词** | 代官员撰写判决书（"批词"） |
| **提供法律建议** | 引用律例为官员审判提供法律依据 |
| **复核案卷** | 检查案卷是否合规，避免被上级驳回 |
| **主导审案** | 实际中常**主导案件审理并代拟判词** |

### 6.3 师爷的分类

| 类型 | 职能 |
|------|------|
| **刑名师爷** | 掌管律法，协助断案（最重要） |
| **钱谷师爷** | 掌管税务、财政（次重要） |
| **书启师爷** | 起草公文书信 |
| **帐房师爷** | 管理账务 |
| **教读师爷** | 教育官员子弟 |

### 6.4 绍兴师爷

| 项目 | 内容 |
|------|------|
| **地域特征** | 来自绍兴府八县的师爷特别多，形成"绍兴师爷"品牌 |
| **专长** | 熟悉刑名律法，精通文书制作 |
| **师承关系** | 师爷之间有师徒传承，形成庞大的"师爷帮" |
| **收入** | 刑名师爷和钱谷师爷收入最高，合称"刑钱" |

[来源：百度百科-刑名师爷；维基百科-幕友；维基百科-绍兴师爷]

## 七、民事审判的法律依据

### 7.1 成文法依据

- 《大明律·户律》：户役、田宅、婚姻、仓库、课程、钱债、市廛各篇
- 《问刑条例》中的相关条例
- 《大明令》中的民事规范

### 7.2 非成文法依据

| 类别 | 说明 |
|------|------|
| **情理** | 司法官审理时大量依据"天理""人情""国法"三者并重 |
| **习惯** | 各地民间交易习惯、婚姻习俗 |
| **典制** | 《大明会典》中的行政规范 |
| **教化** | 圣谕六事、《教民榜文》等道德规范 |

## 八、嘉靖至崇祯的民事司法实况

### 8.1 诉讼量增加

- 明中后期商品经济发展，田土买卖频繁，纠纷增多
- 老人调解制度形同虚设，百姓直接告官
- 放告日制度在部分地区执行不严

### 8.2 讼师活跃

- 嘉靖至崇祯间讼师群体持续壮大
- 讼师秘本如《讼师秘笈》等广泛流传
- 官府持续打压但收效甚微

### 8.3 刑名师爷的重要性提升

- 知县审判民事案件时越来越依赖刑名师爷
- 师爷实际主导判决，知县往往仅在判词上署名
- 这一现象反映了明代科举制度与司法实务之间的脱节
