# 五刑与法外之刑

## 概述

明代法定正刑沿用隋唐以来的笞、杖、徒、流、死五刑体系，共分二十等，载于《大明律·名例律》。在正刑之外，明代还大量使用凌迟、枭首、充军、枷号、刺字、廷杖、口外为民等"法外之刑"（亦称"闰刑"），其中充军刑因使用广泛而事实上成为独立刑种。明代刑罚的总体特征是"重其重罪，轻其轻罪"——对贼盗、受赃等重罪的量刑在唐宋基础上大幅加重。[来源：百度百科-五刑；维基百科-五刑；中国反腐倡廉发展史]

## 一、正刑：五刑二十等

### 1.1 笞刑（五等）

笞刑为最轻之刑，以小荆条笞打臀部。

| 等级 | 笞数 | 折赎铜钱 |
|------|------|----------|
| 第一等 | 笞一十 | 钞六百文 |
| 第二等 | 笞二十 | 钞一贯二百文 |
| 第三等 | 笞三十 | 钞一贯八百文 |
| 第四等 | 笞四十 | 钞二贯四百文 |
| 第五等 | 笞五十 | 钞三贯 |

**刑具规格**：

| 项目 | 规格 |
|------|------|
| 材质 | 小荆条制成，须削去节目 |
| 大头径 | 二分七厘（约0.86厘米） |
| 小头径 | 一分七厘（约0.54厘米） |
| 长度 | 三尺五寸（约112厘米） |
| 禁止事项 | 不得以筋胶诸物装钉 |
| 行刑部位 | 臀部，应决者用小头 |
| 核验方式 | 用官降较板如法较勘 |

[来源：中国反腐倡廉发展史-明时期的法定刑；百度百科-五刑]

### 1.2 杖刑（五等）

杖刑重于笞刑，以荆条击打臀部。

| 等级 | 杖数 | 折赎铜钱 |
|------|------|----------|
| 第六等 | 杖六十 | 钞三贯六百文 |
| 第七等 | 杖七十 | 钞四贯二百文 |
| 第八等 | 杖八十 | 钞四贯八百文 |
| 第九等 | 杖九十 | 钞五贯四百文 |
| 第十等 | 杖一百 | 钞六贯 |

**刑具规格**：

| 项目 | 规格 |
|------|------|
| 材质 | 荆条制成 |
| 大头径 | 三分二厘（约1.02厘米） |
| 小头径 | 二分二厘（约0.70厘米） |
| 长度 | 三尺五寸（约112厘米） |
| 行刑部位 | 臀部 |
| 板折杖比 | 亦有以板折杖者，五板折十杖 |

**讯杖**（审讯用）规格更粗：
- 大头径：四分五厘（约1.44厘米）
- 小头径：三分五厘（约1.12厘米）
- 行刑部位：臀部和腿部

[来源：中国反腐倡廉发展史；儒家网-古代杖刑究竟怎么个杖法]

### 1.3 徒刑（五等）

徒刑为强制劳役刑，每等附加杖刑。

| 等级 | 徒期 | 附加杖 |
|------|------|--------|
| 第十一等 | 一年 | 杖六十 |
| 第十二等 | 一年半 | 杖七十 |
| 第十三等 | 二年 | 杖八十 |
| 第十四等 | 二年半 | 杖九十 |
| 第十五等 | 三年 | 杖一百 |

**执行方式**：
- 服役地点：盐场煎盐或铁冶炼铁
- 劳役定额："每日煎盐三斤"或"每日炒铁三斤"
- 服役期满后放还原籍
- 徒刑为最高三年，再重则升为流刑

[来源：百度百科-五刑；中国反腐倡廉发展史]

### 1.4 流刑（三等）

流刑为强制迁徙刑，将犯人发配至远方，各等均附杖一百。

| 等级 | 流距 | 附加杖 |
|------|------|--------|
| 第十六等 | 二千里 | 杖一百 |
| 第十七等 | 二千五百里 | 杖一百 |
| 第十八等 | 三千里 | 杖一百 |

**特殊规定**：
- 流犯发配实行**南北易置**：南方犯人发北方边塞，北方犯人发南方烟瘴之地
- 《大明律》规定"二死三流同为一减"——流罪三等若减一等，均按徒三年处置
- 流刑犯若持有《大诰》可减一等
- 流放地编入当地户籍，居住一年
- 明中后期流刑逐渐被充军刑取代，实际使用趋于废弛

[来源：百度百科-流刑；中国法学网-明代流行考]

### 1.5 死刑（二等）

| 等级 | 方式 | 执行特点 |
|------|------|----------|
| 第十九等 | **绞** | 以绳索勒毙，保全尸体，为较轻的死刑 |
| 第二十等 | **斩** | 以刀斩首，身首分离，为较重的死刑 |

**执行分类**：

| 类别 | 说明 |
|------|------|
| **立决**（决不待时） | 案情确凿、罪行极重者，判决后即行处决 |
| **监候**（秋后处决） | 须经朝审复核后方可执行，于霜降后至冬至前行刑 |

**行刑时间限制**：
- 遵循"秋冬行刑"传统，监候死囚于每年霜降后执行
- 立春后至秋分前不得执行死刑（立决者例外）
- 行刑须避开国忌日、斋戒日和正月等

[来源：知乎-古人犯罪的死刑有哪些；上海交大中国法与社会研究院]

## 二、法外之刑（闰刑）

法外之刑指不在《大明律》五刑正条中规定、但在司法实践中广泛使用的刑罚。[来源：维基百科-大明律]

### 2.1 凌迟

凌迟为最残酷的极刑，即民间所称"千刀万剐"。

| 项目 | 内容 |
|------|------|
| **法律依据** | 《大明律·刑律·贼盗》："谋反大逆……凌迟处死" |
| **适用罪名** | 谋反、谋大逆、恶逆（杀祖父母/父母）、杀一家非死罪三人以上、采生折割人、造畜蛊毒杀人 |
| **刀数标准** | 例该三千三百五十七刀（3357刀），每十刀一歇、一吆喝 |
| **分日执行** | 头一日先剐三百五十七刀 |
| **刀片大小** | "如大指甲片" |
| **出血情况** | "初动刀则有血流寸许，再动刀则无血矣" |

**下刀次序**：先胸部（割去乳头、胸肌）→ 生殖器（男性）→ 大腿 → 双臂 → 腹肌 → 臀部 → 耳、鼻、眼、唇 → 最后枭首

**著名案例**：

| 案例 | 时间 | 刀数 | 详情 |
|------|------|------|------|
| 刘瑾 | 正德五年（1510） | 3,357刀 | 大太监谋反罪，割三日；第一日割357刀后押回牢中，"喝粥两大碗"；第二日割数十刀后气绝 |
| 郑鄤 | 崇祯十二年（1639） | 3,600刀 | 翰林院庶吉士，被指"杖母不孝" |
| 袁崇焕 | 崇祯三年（1630） | — | 以"通敌叛国"罪凌迟于西市 |

[来源：维基百科-凌迟；知乎-明朝的凌迟处死]

### 2.2 枭首（枭示）

| 项目 | 内容 |
|------|------|
| **执行方式** | 斩下首级悬挂于高杆或城门上示众 |
| **适用罪名** | 强盗、杀人等重罪犯人 |
| **与凌迟关系** | 凌迟之后通常枭首示众 |
| **目的** | 威慑民众，以儆效尤 |
| **展示期** | 通常示众数日至数月 |

[来源：维基百科-中国酷刑史]

### 2.3 充军（五等）

充军是明代最重要的法外之刑，轻于死刑、重于流刑，将犯人发配至军事卫所从事劳役。

#### 五等分类

| 等级 | 名称 | 距离 | 典型发配地 |
|------|------|------|-----------|
| 第一等 | **附近充军** | 二千里 | 沿海附近卫所 |
| 第二等 | **边卫充军** | 二千五百里 | 边境卫所 |
| 第三等 | **边远充军** | 三千里 | 偏远边境 |
| 第四等 | **极边充军** | 四千里 | 最远边境卫所 |
| 第五等 | **烟瘴充军** | 四千里 | 南方烟瘴之地（两广、云贵） |

#### 刑期分类

| 类别 | 说明 |
|------|------|
| **终身充军** | 犯人本人终身在卫所服役，死后子孙不再继续 |
| **永远充军** | 犯人死后由子孙继续充军，世代相传不绝 |

#### 充军条数的增长（数据层面）

| 时期 | 充军罪名条数 | 说明 |
|------|-------------|------|
| 洪武初年 | 22款 | 主要针对军官军人 |
| 嘉靖年间 | 213款 | 适用对象扩大至文武官吏和百姓 |
| 万历年间 | 243款 | 持续膨胀 |
| 万历《问刑条例》382条中 | 约189条（占49.5%） | 充军条例占全部条例近一半 |

[来源：百度百科-充军；中国反腐倡廉发展史-明朝时期的充军刑和枷号刑；中国法学网]

### 2.4 枷号

枷号是明代创设的**耻辱刑**，令犯人戴枷立于衙门前或闹市示众。

#### 枷的法定规格

| 项目 | 规定 |
|------|------|
| 长度 | 五尺五寸（约176厘米） |
| 两端宽度 | 一尺五寸（约48厘米） |
| 材质 | 干木制成 |
| 死刑犯用枷重量 | 三十五斤 |
| 徒罪/流罪犯用枷重量 | 二十斤 |
| 杖罪犯用枷重量 | 十五斤 |

#### 枷号执行方式

| 项目 | 内容 |
|------|------|
| 标识 | 枷上标明犯人姓名和所犯罪状 |
| 期限 | 通常一个月，也有更长者 |
| 执行地点 | 犯罪原地方枷号示众 |
| 与充军搭配 | "枷号一个月，然后照前发遣" |

**制作管理**（洪武二十六年令）：
- 在京各衙门所用刑具须经检查合规方可使用
- 笞杖由应天府采办
- 枷枢由龙江提举司制造
- 铁索铁镣由宝源局打造

**滥用情况**：刘瑾当政时期，曾使用重达一百五十斤的大枷，远超法定标准。

[来源：中国反腐倡廉发展史-明朝时期的充军刑和枷号刑]

### 2.5 刺字（墨刑/黥面）

| 项目 | 内容 |
|------|------|
| **执行方式** | 在犯人面部或手臂刺字 |
| **刺字内容** | 刺所犯罪名，如"窃盗"等 |
| **适用范围** | 明律主要限于**抢夺**和**窃盗**两项罪名 |
| **特点** | 明律在刺字适用范围上较为节制，比《大诰》时期大幅收窄 |
| **目的** | 永久标记犯人身份，防止再犯、便于识别 |

[来源：中国法学网-明代流行考；知乎-清代刺字]

### 2.6 廷杖

廷杖是明代独有的**对朝臣施行的法外体罚**，在午门外公开执行。

#### 执行规格

| 项目 | 内容 |
|------|------|
| **执行地点** | 午门之外，公开行刑 |
| **执行者** | 锦衣卫校尉执杖 |
| **监刑者** | 司礼监太监 |
| **刑具** | 大荆条，大头径三分二厘，小头径二分二厘，长三尺五寸 |

#### 执行步骤

1. 皇帝（或权宦）下令廷杖
2. 司礼监出帖加盖印信（"驾帖"）
3. 刑科给事中签批
4. 锦衣卫拿人并执行
5. 以麻布自犯人肩脊下束缚，一人缚其两足，四面牵曳，仅露臀部受杖
6. 犯人头面触地，口中满尘

#### 暗语系统

| 监刑太监暗号 | 含义 | 结果 |
|-------------|------|------|
| 脚尖张开呈外八字 | "用心打" | **轻打**，受杖者可活 |
| 脚尖闭合站立 | "着实打" | **重打**，受杖者必死 |

#### 制度化过程

| 时期 | 情况 |
|------|------|
| 洪武年间 | 偶有施行，尚非制度 |
| 永乐年间 | 频率增加 |
| 正统以后 | 逐渐制度化 |
| 正德年间 | 刘瑾改为"去衣"（脱裤）行刑，致死率大增 |
| 嘉靖年间 | 达到高峰 |

#### 重大廷杖事件

| 时间 | 事件 | 被杖人数 | 死亡人数 |
|------|------|----------|----------|
| 正德元年（1506） | 谏阻刘瑾乱政 | 21人 | 戴铣等人杖死 |
| 正德十四年（1519） | 谏阻武宗南巡 | 146人 | 11人杖死 |
| 嘉靖三年（1524） | **左顺门事件**（大礼议） | 134人 | 16人杖死 |

嘉靖在位48年间，有名可查被廷杖者共88人，其中11人杖死。[来源：百度百科-廷杖；知乎；澎湃新闻]

### 2.7 口外为民

| 项目 | 内容 |
|------|------|
| **性质** | 将犯人发往长城以外地区为民 |
| **适用对象** | 主要针对有司（文官系统）犯罪者 |
| **法律依据** | 弘治《问刑条例》规定 |
| **执行方式** | 先在原地方枷号一个月，然后发口外为民 |
| **与充军区别** | 充军须从军服役；口外为民则为普通编民 |

[来源：中国法学网-明代流行考]

## 三、《大诰》中的酷刑（洪武时期特例）

明太祖于洪武十八年至二十年间颁布《大诰》四编，共236个条目（案例156个，无案例条目80个），规定了大量超出《大明律》的残酷刑罚，但在朱元璋去世后逐渐废弛。[来源：百度百科-明大诰]

**《大诰》中超出明律的主要酷刑**：

| 酷刑名称 | 说明 |
|---------|------|
| 族诛 | 灭族，株连亲族 |
| 剥皮 | 将犯人皮肤剥下，充草悬挂示众（贪官特用） |
| 抽肠 | 从腹部取出肠子 |
| 挑筋去指 | 剔除手指筋骨 |
| 挑筋去膝盖 | 剔除膝盖筋骨 |
| 断手、刖足 | 砍手砍脚 |
| 斩趾枷令 | 砍去脚趾后枷号 |
| 墨面文身 | 在面部大面积刺字 |
| 阉割为奴 | 阉割后发为奴仆 |
| 枷项游历 | 戴枷在各地游行示众 |

**量刑对比示例**：

| 罪名 | 《大明律》刑罚 | 《大诰》刑罚 |
|------|---------------|-------------|
| 有司滥收无籍之徒 | 杖一百，徒三年 | **族诛** |

《大诰》条目80%以上针对贪官污吏。

## 六、嘉靖至崇祯时期的刑罚实况

### 6.1 充军刑的泛滥

- 嘉靖至崇祯期间充军刑适用范围持续扩大
- 条例不断增加新的充军罪名（从22款增至243款）
- 终身充军与永远充军并行
- 充军事实上取代流刑成为仅次于死刑的重刑

### 6.2 廷杖的政治化

- 嘉靖朝"大礼议"中廷杖群臣最为惨烈（1524年杖死16人）
- 万历朝"争国本"期间多次廷杖
- 天启朝魏忠贤专权期间廷杖成为打击异己工具

### 6.3 凌迟的使用

明代凌迟使用频率远高于前代：
- 嘉靖二十一年（1542），宫女杨金英等谋弑嘉靖帝，凌迟处死
- 崇祯三年（1630），袁崇焕凌迟于西市
