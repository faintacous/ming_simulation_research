# 厂卫司法

## 概述

明代"厂卫"是皇帝直接控制的特务和司法机构，独立于正规三法司体系之外，是明代政治的显著特征。厂卫体系包括锦衣卫（1382年设）、东厂（1420年设）、西厂（1477年设，两度设废）和内行厂（约1506年设，存续约5年）。厂卫的司法权来自皇帝个人授权，不在正式法律体系之中——《大明律》和《大明会典》中均无厂卫司法的明文规定，属于法外特权。[来源：维基百科-锦衣卫；维基百科-东厂]

## 二、锦衣卫

### 2.1 组织架构

| 层级 | 机构 | 长官 | 品级 | 职能 |
|------|------|------|------|------|
| 指挥层 | 锦衣卫指挥使司 | 指挥使 | 正三品 | 总揽全卫事务 |
| 南镇抚司 | — | 镇抚 | 从四品 | 管理本卫军匠、军人军纪 |
| **北镇抚司** | — | 镇抚 | 从四品 | **专理诏狱**，掌侦察逮捕、审讯刑罚 |
| 千户所 | 十四个千户所 | 千户 | 正五品 | 下辖百户所 |

### 2.2 北镇抚司与诏狱

**诏狱的性质**：

| 项目 | 内容 |
|------|------|
| **含义** | 由皇帝下诏书直接指定案件的监狱 |
| **管理者** | 北镇抚司 |
| **管辖范围** | 皇帝钦定案件，不限罪名和身份 |
| **独立权力** | 自行逮捕、侦讯、行刑、处决，不经三法司 |
| **成化改制** | 成化元年（1465）增铸北镇抚司印信，一切刑狱**直呈皇帝**，无需经锦衣卫指挥使转达 |
| **不受干预** | 锦衣卫官员中不掌诏狱者"亦不得干预其事" |

**诏狱的刑讯**：

拷问三档次：

| 档次 | 口令 | 严酷程度 |
|------|------|----------|
| 轻 | "打着问" | 一般拷讯 |
| 中 | "好生打着问" | 加重拷讯 |
| 重 | "好生着实打着问" | 最严酷拷讯 |

**"五毒"刑具**（每次用刑一套）：

| 刑具 | 说明 |
|------|------|
| 械 | 木制桎梏 |
| 镣 | 铁制脚镣 |
| 棍 | 行刑棍杖 |
| 拶 | 夹手指（五根圆木，各长七寸） |
| 夹棍 | 夹腿骨（杨木制，长三尺余，贯以铁条） |

**亲历记述**（天启六年，东林六君子之一魏大中供述）：
> "一拶敲一百，穿梭一夹，敲五十杠子，打四十棍，惨酷备至"

### 2.3 锦衣卫的其他司法职能

| 职能 | 说明 |
|------|------|
| **侦查缉捕** | 侦查皇帝指定案件，缉捕嫌犯 |
| **廷杖执行** | 锦衣卫校尉负责在午门执行廷杖 |
| **会审参与** | 参加三法司会审（特别是热审），提供案件情报 |
| **监督行刑** | 监督死刑执行过程 |
| **监狱管理** | 管理诏狱及其囚犯 |

[来源：维基百科-锦衣卫；百度百科-镇抚司]

## 三、东厂

### 3.1 设立背景与组织

| 项目 | 内容 |
|------|------|
| **设立背景** | 明成祖靖难夺位后，对建文旧臣和锦衣卫均不完全信任，设东厂以亲信宦官侦缉 |
| **全称** | 东缉事厂 |
| **首领** | 东厂掌印太监（"厂公"/"督主"），全称"钦差总督东厂官校办事太监" |
| **首领地位** | 宦官序列中**仅次于司礼监掌印太监**的第二号人物 |
| **人员来源** | 掌班、领班、司房约四十余人由锦衣卫拨给；役长和番役从锦衣卫精干者中挑选 |

### 3.2 东厂的职权

| 职权 | 说明 |
|------|------|
| **侦缉** | "访谋逆妖言大奸恶等"——侦缉谋反、妖言惑众等重大案件 |
| **监视** | 监视政府官员、社会名流、学者等各种政治力量 |
| **直报皇帝** | 监视结果直接向皇帝汇报，无需经过任何中间环节 |
| **逮捕** | 对地位较低的政治反对派可直接逮捕 |
| **审讯** | 初期无审讯权，须交北镇抚司；**明末逐渐获得自有监狱和审讯权** |

### 3.3 东厂与锦衣卫的关系

| 对比项 | 锦衣卫 | 东厂 |
|--------|--------|------|
| 人员 | 武官和校尉 | 太监为首，借用锦衣卫人员 |
| 首领 | 锦衣卫指挥使（武官） | 掌印太监（宦官） |
| 设立目的 | 皇帝亲军 + 侦缉 | **监督锦衣卫** + 侦缉 |
| 权力等级 | 初期高于东厂，后低于东厂 | "东厂权力在锦衣卫之上" |
| 审讯权 | 有（北镇抚司） | 初期无，明末获得 |
| 监狱 | 有（诏狱） | 初期无，明末有 |

[来源：维基百科-东厂；百度百科-东厂]

## 四、西厂与内行厂

### 4.1 西厂

| 项目 | 内容 |
|------|------|
| **第一次设立** | 成化十三年（1477），太监**汪直**任提督 |
| **第一次撤废** | 成化十八年（1482） |
| **第二次设立** | 正德元年（1506），太监**谷大用**任提督 |
| **第二次撤废** | 正德五年（1510/1511） |
| **职权范围** | 与东厂相同但规模更大，侦缉范围更广 |
| **特点** | 人员编制比东厂多一倍，权势一度在东厂之上 |

### 4.2 内行厂

| 项目 | 内容 |
|------|------|
| **设立** | 正德初年（约1506），太监**刘瑾**主掌 |
| **撤废** | 正德五年（1510），随刘瑾倒台废除 |
| **侦缉范围** | 比东厂、西厂、锦衣卫**更广** |
| **特殊权力** | 负有**监视东厂、西厂、锦衣卫**三个机构的职责 |
| **存续时间** | 仅约5年 |

## 五、厂卫与三法司的关系

### 5.1 制度层面

- 厂卫的司法权**来自皇帝个人授权**，不在正式法律体系之中
- 三法司的司法权来自《大明律》和《大明会典》
- 两者并行运作，但权力来源和法律基础不同
- 《大明律》中**无厂卫司法的明文规定**

### 5.2 实际运作

| 情形 | 厂卫与三法司的互动 |
|------|-------------------|
| **常态** | 厂卫负责侦缉、逮捕，三法司负责正式审判 |
| **诏狱案件** | 厂卫独立审判，三法司无权过问 |
| **热审、朝审** | 锦衣卫参与三法司会审 |
| **权阉专权** | 厂卫架空三法司，司法完全沦为政治工具 |

### 5.3 典型的权力膨胀时期

| 时期 | 权阉 | 厂卫权力状态 |
|------|------|-------------|
| 正德年间 | 刘瑾 | 内行厂监视一切，"文武大臣皆畏之" |
| 天启年间 | 魏忠贤 | 东厂达到权力巅峰，"钩党之狱" |
| 万历至崇祯 | 多位太监 | 厂卫权力起伏不定，与朝臣反复博弈 |

## 六、嘉靖至崇祯的厂卫实况

### 6.1 嘉靖朝

- 嘉靖帝在位45年，对厂卫控制较为得当
- 厂卫主要用于监视朝臣而非大规模迫害
- 但"大礼议"中仍动用廷杖打击异己

### 6.2 万历朝

- 万历帝长期不朝，厂卫权力时大时小
- 东厂掌印太监多为皇帝亲信
- 厂卫与朝臣间的博弈持续

### 6.3 天启朝（权力巅峰）

魏忠贤专权期间（1624-1627），东厂达到权力巅峰：
- "钩党之狱"大肆迫害东林党人
- 杨涟、左光斗、魏大中等"六君子"被逮入诏狱
- 使用"五毒"刑讯逼供
- 杨涟在狱中被土袋压身、铁钉贯耳而死
- 左光斗在狱中被炮烙，面目全非

### 6.4 崇祯朝

- 崇祯帝即位后清除魏忠贤，大幅收缩厂卫权力
- 但明末政局动荡，厂卫仍在运作
- 崇祯十七年（1644）明亡，厂卫制度终结

## 七、厂卫司法的历史评价

### 7.1 消极面

- 破坏正常司法程序，制造大量冤案
- 刑讯酷虐，远超法律规定
- 成为皇帝打击政治异见的工具
- 败坏官场风气，官员人人自危

### 7.2 制约性因素

- 厂卫权力完全依赖皇帝个人意志
- 有为的皇帝可以限制厂卫权力（如弘治帝大幅收缩）
- 言官系统持续对厂卫提出批评和弹劾
- 文官集团与厂卫的博弈贯穿明代中后期
