# 司礼监与宦官机构

## 概述

明朝内廷宦官机构统称"二十四衙门"，由十二监、四司、八局组成，是中国历史上宦官制度最完备、规模最庞大的体系。其中司礼监居于核心地位，掌"批红"之权，与外朝内阁形成"内外相维"的决策体制。嘉靖至崇祯（1521–1644）期间，宦官势力随帝王个人意志起伏剧烈——嘉靖朝宦官受压制、万历朝宦官以矿监税使横行、天启朝魏忠贤专权达到顶峰、崇祯朝则在信任与猜忌间反复。

## 一、二十四衙门

二十四衙门是明朝内廷宦官系统的总称，由朱元璋洪武年间初设，经永乐、宣德等朝完善定型。包括**十二监、四司、八局**，共计二十四个机构。

### 1.1 十二监

| 序号 | 机构名称 | 主要职能 |
|------|----------|----------|
| 1 | **司礼监** | 掌内外章奏、照阁票批红、传宣谕旨；提督东厂；掌管内书堂。为二十四衙门之首 |
| 2 | **内官监** | 掌木、石、瓦、土、搭材、东行、西行、油漆、婚礼、火药十作，及米盐库、营造库、皇坛库；凡营造宫室、陵墓及铜锡妆奁器用、冰窨等事。洪武、永乐时为二十四衙门第一衙门 |
| 3 | **御用监** | 掌御前所用围屏、床榻等贵重器具的日常管理，管理武英殿中书承旨典籍等 |
| 4 | **司设监** | 掌卤簿、仪仗、围幙、帐幔、雨具等 |
| 5 | **御马监** | 掌御厩马匹及诸进贡马骡，兼管草场；后衍生军事（统领直属皇帝禁军——腾骧四卫）和财政（掌皇庄子粒银）两大职能，为仅次于司礼监的权重机构 |
| 6 | **神宫监** | 掌太庙及各庙洒扫、香灯等事 |
| 7 | **尚膳监** | 掌御膳及宫内食用、筵宴等事 |
| 8 | **尚宝监** | 掌宝玺、敕符、将军印信 |
| 9 | **印绶监** | 掌古今通集库，保管诰封、铁券、贴黄、印信、图书、勘合、符验、信符等 |
| 10 | **直殿监** | 掌各殿及廊庑的洒扫清洁 |
| 11 | **尚衣监** | 掌御用冠冕、袍服、鞋袜等事 |

## 二、司礼监

### 2.1 特殊地位

司礼监在十二监中居首位。洪武、永乐时，内官监才是二十四衙门首席；但宣德以后，随着票拟—批红制度的确立，司礼监掌握了代皇帝批红、盖印的权力，迅速跃居内廷各衙门之上。司礼监掌印太监被称为**"内相"**，权势至少与内阁首辅相当。

### 2.2 官职设置

据《酌中志》及《明史·职官志》，司礼监设：

| 官职 | 员数 | 主要职责 |
|------|------|----------|
| **提督太监** | 一人 | 督理皇城内一应仪礼刑名，钤束长随、当差各役；关防门禁；催督光禄供应；掌管内书堂 |
| **掌印太监** | 一人 | 掌理内外章奏及御前勘合；审核批红后盖章用印；为司礼监实际最高权力者 |
| **秉笔太监** | 四至九人不等 | 照阁票批红（用朱笔在奏章上代写皇帝批示）；首席秉笔通常提督东厂 |
| **随堂太监** | 无定员（约四至九人） | 协助秉笔太监处理章奏文书 |
| 书籍名画等库掌司 | 无定员 | 掌管内府书籍、名画收藏 |
| 内书堂掌司 | 无定员 | 管理内书堂的学籍、学规 |
| 六科廊掌司 | 无定员 | 接收六科送入的文书 |
| 典簿 | 无定员 | 掌文书档案 |

### 2.3 掌印太监与秉笔太监的职责区别

**掌印太监**：
- 负责审核由秉笔太监批红后的奏章
- 审核通过后盖上御印（"用印"），使批示正式生效
- 如认为批红不妥，有权打回重批
- 是"批红"最终把关者，相当于内廷的"最终签发人"

**秉笔太监**：
- 按照内阁票拟，用朱笔代皇帝在奏章上写批示（"照阁票批红"）
- 首席秉笔太监通常兼提督东厂
- 各秉笔太监分管内廷各监、司、局
- 相当于内廷的"执行秘书"

**权力大小**：掌印太监 > 秉笔太监。掌印者掌握最终审核盖印权，秉笔者虽执笔但不能最终定夺。但也有例外，如魏忠贤以秉笔太监身份凌驾于掌印太监王体乾之上，取决于个人与皇帝的亲疏。

### 2.4 批红权的具体操作流程

1. **奏章呈入**：各部院衙门的奏章经通政使司汇集，送入内廷
2. **内阁票拟**：内阁大学士阅读奏章后，用小票写上处理意见贴于奏章上，称"票拟"（又称"票旨"）
3. **送司礼监**：票拟后的奏章送入司礼监
4. **秉笔批红**：秉笔太监依照内阁票拟内容，用朱笔在奏章上代写皇帝批示（批红）；若皇帝亲自过问，则依皇帝口谕书写
5. **掌印审核**：掌印太监审核批红内容，认为妥当则盖上御印
6. **发还执行**：盖印后的奏章发还内阁，由内阁据批红撰写正式诏书，分发六部等机构执行

> **关键**：若皇帝勤政（如嘉靖帝），批红多由皇帝亲自裁定，司礼监只是代为书写；若皇帝怠政（如万历、天启），批红权实际由秉笔太监和掌印太监掌握。

### 2.5 司礼监与内阁的权力对接机制

- **制度设计**：明初无内阁制度，也无司礼监批红权。宣德以后，内阁获得票拟权，司礼监获得批红权，二者形成**"对柄机要"**的权力格局
- **相互制衡**：内阁掌"票拟"（建议权），司礼监掌"批红"（批准权），缺一不可
- **文书流转**：内阁 → 票拟 → 司礼监 → 批红 → 用印 → 返回内阁 → 撰诏 → 六部执行
- **实际关系**：
  - 嘉靖朝：皇帝亲自主导，司礼监与内阁均受压制
  - 隆庆—万历初：冯保（司礼监）与张居正（内阁）合作，形成"内外一体"局面
  - 天启朝：魏忠贤控制司礼监，压制内阁，形成宦官专权
  - 崇祯朝：皇帝频繁更换内阁与司礼监人选，制衡失效

## 三、东厂

### 3.1 设立

- **设立时间**：永乐十八年（1420年）十二月
- **全称**：东缉事厂
- **设立者**：明成祖朱棣
- **设立原因**：朱棣以靖难之变夺位，需镇压政治反对力量。因对外朝锦衣卫不完全信任（锦衣卫隶属外朝），故另设由宦官统领的特务机构，直接向皇帝负责
- **驻地**：北京东安门外（北镇抚司北面）

### 3.2 组织结构

**领导层：**

| 官职 | 全称 | 说明 |
|------|------|------|
| **提督太监**（厂公/督主） | 钦差总督东厂官校办事太监 | 由司礼监秉笔太监中位居第二或第三者担任，是宦官中仅次于司礼监掌印太监的第二号人物 |

**属官（贴刑官）：**

| 官职 | 员数 | 来源 |
|------|------|------|
| 掌刑千户 | 一员 | 由锦衣卫千户充任 |
| 理刑百户 | 一员 | 由锦衣卫百户充任 |

**管理层：**
- 掌班、领班、司房共四十余人，由锦衣卫拨给
- 分为子、丑、寅、卯等**十二颗**（即十二个班组）
- 颗管事戴圆帽，着皂靴，穿褐衫

**侦缉执行层：**

| 角色 | 别称 | 人数 | 装束 |
|------|------|------|------|
| **役长** | 档头 | 一百余人 | 戴尖帽，着白皮靴，穿褐色衣服，系小绦 |
| **番役** | 番子、干事 | 每役长统辖数名 | 从锦衣卫中挑选精干分子 |

役长与番役也分十二颗编组。

### 3.3 侦缉制度

东厂的侦缉工作有专门名称与制度：

- **"听记"**：监视朝中各部官员会审大狱、锦衣卫拷讯罪犯的过程
- **"坐记"**：派驻各处地方官府进行访缉（明为坐探，暗为暗探）
- **"打事件"**：各处胥吏将某官员的举措或城门捕获要犯等消息上报东厂

**月度制度**：每月初一，东厂集中布置当月侦缉工作，厂役在东厂内抽签决定所负责的地盘。

**侦缉范围**：
- 监视朝中文武百官
- 监视各地藩王
- 侦缉民间妖言惑众、结社集会
- 监视锦衣卫的办案过程
- 侦察市面物价、民情舆论

### 3.4 与锦衣卫的关系

- **东厂人员来自锦衣卫**：东厂的掌刑千户、理刑百户、掌班、领班、司房、役长、番子均从锦衣卫中选调
- **东厂监督锦衣卫**：东厂有权监视锦衣卫的办案活动（"听记"），锦衣卫无权反向监督东厂
- **东厂地位高于锦衣卫**：东厂直接对皇帝负责，由亲信宦官统领；锦衣卫隶属外朝，受更多制度约束
- **合作办案**：遇大案时，东厂与锦衣卫常联合侦办，但东厂为主导

### 3.5 嘉靖至崇祯的运作情况

| 时期 | 特点 |
|------|------|
| 嘉靖朝（1521–1567） | 嘉靖帝精于驭下，将宦官视为家奴，东厂权力受限。提督太监权力有限，主要执行皇帝旨意 |
| 隆庆朝（1567–1572） | 过渡时期，东厂活动相对平稳 |
| 万历初（1572–1582） | 冯保以掌印太监兼提督东厂，与张居正合作，东厂主要服务于改革需要 |
| 万历中后（1582–1620） | 万历帝怠政三十年，东厂仍持续运作但无突出大案 |
| 天启朝（1620–1627） | **东厂权力顶峰**：魏忠贤提督东厂，大兴冤狱，迫害东林党人。东厂、锦衣卫合力构成恐怖政治 |
| 崇祯朝（1627–1644） | 崇祯初年以曹化淳提督东厂清除魏忠贤余党；后频繁更换厂督，东厂效能下降 |

## 四、西厂

### 4.1 设立与废止

西厂（全称"西缉事厂"）的历史如下：

| 时间 | 事件 |
|------|------|
| 成化十三年正月（1477） | 明宪宗因"妖狐夜出"案及李子龙弑君未遂事件惊恐，设西厂，命太监汪直统领 |
| 成化十三年五月（1477） | 商辂等大臣上疏列汪直十大罪状，宪宗迫于压力废西厂 |
| 成化十三年六月（1477） | 废西厂仅一个月，宪宗恢复西厂 |
| 成化十九年（1483） | 汪直失势，西厂再废，汪直被贬南京御马监 |
| 正德元年（1506） | 明武宗复设西厂，由太监谷大用统领 |
| 正德五年（1510） | 刘瑾倒台，西厂与内行厂同时被永久撤销 |

### 4.2 嘉靖至崇祯期间的情况

**嘉靖朝至崇祯朝（1521–1644）期间，西厂不存在。** 正德五年（1510年）西厂被永久撤销后，此后再未恢复。

### 4.3 内行厂（附）

- **设立**：正德三年八月（1508年），由太监刘瑾设于荣府旧仓地
- **职能**：侦缉范围比东、西厂和锦衣卫更广，并负有监视这三个机构的职责
- **废止**：正德五年（1510年），刘瑾被凌迟处死后，内行厂随即撤销
- **嘉靖后**：同样不再存在

## 五、锦衣卫

### 5.1 概况

锦衣卫全称"锦衣卫指挥使司"，前身为朱元璋所设"仪鸾司"，洪武十五年（1382年）改置锦衣卫。名义上属于"亲军上十二卫"之一，实为皇帝直属的侍卫兼特务机构。

### 5.2 组织结构

**主要官职与品级：**

| 官职 | 品级 | 员数 | 职责 |
|------|------|------|------|
| 指挥使 | 正三品 | 一人 | 锦衣卫最高长官，由皇帝亲信武将担任 |
| 指挥同知 | 从三品 | 二人 | 辅佐指挥使 |
| 指挥佥事 | 正四品 | 二人 | |
| 镇抚 | 正五品 | 二人 | 分管南、北镇抚司 |
| 千户 | 正五品 | 十四人 | 各领千户所 |
| 副千户 | 从五品 | | |
| 百户 | 正六品 | | 各领百户所 |
| 试百户 | 从六品 | | |
| 总旗 | 正七品 | | 每百户所辖两总旗 |
| 小旗 | 从七品 | | 每总旗辖五小旗 |

一般军士称"校尉"、"力士"，雅称**"缇骑"**。

### 5.3 南北镇抚司

| 项目 | 南镇抚司 | 北镇抚司 |
|------|----------|----------|
| **主要职能** | 掌锦衣卫本卫法纪、军纪；管理军匠（制造兵器） | 专理皇帝钦定案件（"诏狱"）；暗中侦伺文武百官；监视藩王；反间谍 |
| **是否有诏狱** | 无 | 有（独立诏狱） |
| **司法独立性** | 受常规司法管辖 | 可自行逮捕、刑讯、处决，不经三法司 |
| **额外编制** | 常规编制 | 增设百户 5 人、总旗 5 人、校尉 100 人、皂隶 30 人、门卫 11 人 |

### 5.4 诏狱制度

- **定义**：诏狱即由皇帝钦定、北镇抚司直接审理的案件及关押场所
- **特点**：
  - 不经刑部、都察院、大理寺（三法司）审理
  - 北镇抚司可自行逮捕、用刑、处决
  - 成化年间，明宪宗规定北镇抚司所理诏狱不必经锦衣卫指挥使过问，直接向皇帝报告
- **刑具**：诏狱使用特殊刑具，包括廷杖、夹棍等
- **著名案件**：明中后期许多政治大案均经诏狱，如嘉靖朝左顺门事件、天启朝东林党人入狱等

### 5.5 与东厂的协作关系

- **东厂监督锦衣卫**：东厂有权派人"听记"锦衣卫的审讯过程
- **锦衣卫为东厂供人**：东厂的掌刑千户、理刑百户、役长、番子全部从锦衣卫调拨
- **合称"厂卫"**：东厂与锦衣卫合称"厂卫"，是明代特务政治的核心
- **权力排序**：通常东厂 > 锦衣卫，但具体取决于皇帝信任度
- **万历以后**：锦衣卫指挥使常由东厂提督太监推荐或控制

### 5.6 人员规模

| 时期 | 缇骑人数 | 备注 |
|------|----------|------|
| 洪武朝 | 约 500 人 | 初设，规模精干 |
| 正德朝 | 约 60,000 人 | 刘瑾用事，大幅膨胀 |
| 嘉靖朝 | 缇骑 + 外围线人约 150,000–160,000 人 | 含外围线人十五六万 |
| 编制理论值 | 5,600 人/卫 | 按卫所制，一卫 5,600 人，一千户所 1,120 人，一百户所 112 人 |

## 六、宦官的来源与管理

### 6.1 宦官的来源

明代宦官主要有以下四种来源：

1. **战争掳掠**：在对外战争中俘获的少数民族幼童（如安南、蒙古等地），阉割后充入内廷。如郑和即为云南回族，靖难后入宫
2. **外国进贡**：朝鲜、安南等藩属国进贡的阉割幼童
3. **籍没入宫**：犯罪官员家属被没入宫廷后，幼童被阉割充作宦官
4. **民间私自净身**：北直隶地区（尤其是京畿附近的青县、静海、河间、沧州、任丘、南皮、枣强等县）家境贫寒者或羡慕宫廷富贵者，将幼子私自阉割后送入宫中

### 6.2 净身制度

- **官方程序**：明代规定，一家有四五个孩子以上者，可自愿将其中一子阉割，由相关部门登记造册，待收补时选用
- **私自阉割禁令**：朝廷明令禁止民间私自阉割，违者重治，邻居知情不报者连坐。但此禁令从未被认真执行
- **净身手术**：由专门的"刀子匠"操作，术前需本人及家属签署"自愿书"，术后在"蚕室"中休养
- **"宝贝房"**：割下的器官放于红绒布袋内，悬挂于紫禁城内阴干室。资深宦官的悬挂位置离地面最高（远离湿气），初入宫者最低
- **录用主管**：明代宦官录用初由**礼部**主管，后逐渐改为**司礼监会同礼部**办理

### 6.3 内书堂（宦官教育机构）

- **设立时间**：宣德元年（1426年）七月，由明宣宗朱瞻基设立
- **别称**：内书院、内馆、书馆
- **隶属关系**：初属内官监，后转隶司礼监；由司礼监提督太监、掌司等官掌管学籍、学规
- **教师**：由翰林院学士充任教习。第一任教师为大学士陈山，另有四名翰林学士为助教。翰林教习内书堂被视为重要差事
- **入学年龄**：约十岁上下的小内使
- **学生人数**：一般二三百人，后一度增至四五百人，最多时达八九百人
- **课程内容**：研读《百家姓》《千字文》《孝经》《大学》《中庸》《论语》《孟子》及《贞观政要》《大学衍义》等
- **意义**：内书堂培养了大批识字、通文墨的宦官，使其能够胜任章奏批红等文书工作。这也是明代宦官能够掌握批红权的基础

### 6.4 宦官的晋升路径

明代宦官的典型晋升路径：

```
内使/火者（品外，杂役）
    ↓
长随（正六品）/ 奉御（从六品）
    ↓
典簿（正六品）
    ↓
副使/副司（从五品）—— 八局副使 / 四司司副
    ↓
大使/司正（正五品）—— 八局大使 / 四司司正
    ↓
监丞（正五品）—— 十二监
    ↓
少监（从四品）
    ↓
太监（正四品）—— 各监掌印太监
    ↓
[特殊]司礼监掌印太监 / 提督东厂太监（名义仍正四品，实权远超）
```

晋升的关键因素：
- **皇帝信任**：这是最重要的因素，宦官的权力来源于皇帝个人恩宠
- **内书堂出身**：内书堂毕业的宦官更容易获得文书类重要职位
- **资历年限**：一般需在宫中服务多年
- **派系关系**：依附于权势太监（如王安门下出曹化淳、王承恩）

### 6.5 宦官与外朝官员的关系规则

- **朱元璋的祖训**：洪武十七年（1384年），朱元璋铸铁牌置于宫门，上书"内臣不得干预政事，犯者斩"，并规定宦官不许识字、品级不过四品
- **祖训的崩溃**：宣德以后，宦官不仅识字（内书堂），还掌握批红权，祖训形同虚设
- **禁止交接**：制度上禁止宦官与外朝官员私下交往，但实际上明中后期宦官与朝臣的交接极为普遍
  - 内阁首辅需与司礼监掌印太监保持良好关系才能施政
  - 张居正—冯保的合作是典型案例
  - 天启朝魏忠贤拉拢外朝官员形成"阉党"
- **出镇制度**：明代向各地派遣**镇守太监**（镇守中官），在军事重镇和财赋重地监督地方文武官员，嘉靖朝曾一度撤回，万历朝以矿监税使形式复出
- **外朝弹劾**：御史、给事中等言官可弹劾宦官，但效果取决于皇帝态度

## 关联主题

- **内阁制度**：票拟权与批红权构成明代决策的核心二元结构
- **厂卫制度**：东厂 + 锦衣卫（+ 西厂/内行厂）构成特务政治体系
- **矿监税使**：万历朝派遣宦官到各地征收矿税，激起大规模民变
- **阉党与东林党**：天启朝最严重的党争，魏忠贤阉党与东林党的对抗
- **镇守太监**（镇守中官）：派驻地方军事重镇的宦官监军制度
- **御马监与腾骧四卫**：宦官掌握的禁军系统

## 存疑 & 待查

1. 司礼监提督太监与掌印太监的权力高低，《明史·职官志》将提督列于掌印之前，但实际政治运作中掌印太监权力更大，学界对此有不同解读
2. 各时期宦官确切总人数缺乏可靠的统计数据，现有数字多为间接推算
3. 嘉靖朝"张佐"任掌印太监的具体年份待考证
4. 都知监的具体职能在史料中记载不甚明确，有"为同僚中最卑闲者"之说
5. 崇祯朝司礼监人事变动频繁，完整的掌印/秉笔太监接任顺序仍待系统梳理
6. 内行厂的具体组织结构与编制，史料记载极为有限
