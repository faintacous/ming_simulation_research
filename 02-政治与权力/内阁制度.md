# 内阁制度

## 概述

明朝内阁是明代中枢决策的核心机构，始设于永乐年间（1402年），终于崇祯十七年（1644年）明亡。其前身是洪武十五年（1382年）设置的殿阁大学士。内阁始终未获得法定的行政机构地位——无专属衙署编制、无官印、无固定官员定额、无法定下属官员——但凭借票拟权，实际成为帝国运转的政务中枢，尤其在嘉靖至万历初年达到权力鼎盛。

内阁制度的演变大致分三阶段：
- **建立形成期**：永乐至正德（1403–1521）
- **发展鼎盛期**：嘉靖至万历十年（1522–1582）
- **变化衰落期**：万历十一年至明亡（1583–1644）

## 一、设立与沿革

### 1.1 洪武废丞相

- **洪武十三年（1380年）**：胡惟庸案后，朱元璋废中书省及丞相，六部直接对皇帝负责，皇帝亲揽行政权。
- **同年九月**：设置**四辅官**（春、夏、秋、冬官），以儒士王本等充任，辅佐皇帝处理政务。
- **洪武十五年（1382年）**：撤四辅官，仿宋制设**华盖殿、武英殿、文渊阁、东阁**诸大学士，以礼部尚书邵质、检讨吴伯宗、翰林学士宋讷、典籍吴沉分任。后又设**文华殿大学士**辅导太子。此时大学士秩仅正五品，职责限于"备顾问"，无决策权，是内阁的雏形。
- 仁宗时增设**谨身殿大学士**，殿阁大学士遂为六个。

### 1.2 永乐设内阁

- **建文四年 / 永乐元年（1402年）**：明成祖朱棣即位后，特简翰林官解缙、胡广、杨荣、杨士奇、黄淮、胡俨、金幼孜等七人入午门值文渊阁，参预机务。因文渊阁地处内廷，故称"**内阁**"，内阁制度正式确立。
- 此时内阁仅为皇帝的私人顾问秘书班子，不是法定权力机构。成祖将决策权握于己手，议政权分给内阁，行政权分给六部。

### 1.3 从秘书机构到决策中枢

| 时期 | 关键变化 |
|------|----------|
| **永乐中后期** | 内阁参预机务日多，权职渐重 |
| **仁宗洪熙年间（1425）** | 升内阁成员**兼任六部尚书衔**，品级从五品跃至二三品；出现"条旨"（票拟前身） |
| **宣宗宣德年间（1426–1435）** | 形成完善的政务流程：通政使司汇总奏章→司礼监呈皇帝→交内阁票拟→司礼监呈皇帝批红→六科校对下发。票拟制度初步确立 |
| **英宗正统年间（1436–1449）** | 英宗年幼，太皇太后张氏专令内阁条旨，票拟正式成为**内阁专责** |
| **宪宗成化至武宗正德** | 内阁首辅地位凸显，票拟权渐归首辅 |
| **世宗嘉靖年间（1522–1566）** | 华盖殿改名**中极殿**，谨身殿改名**建极殿**；阁臣朝位班次列六部尚书之上；夏言、严嵩执政，首辅"赫然真相"，可压制六部 |
| **穆宗隆庆至神宗万历初年** | 张居正改革（1572–1582），内阁权力达到极盛，首辅实际行使宰相职权 |
| **万历中后期至明亡** | 党争加剧，内阁权力衰落；崇祯朝首辅更替频繁（十七年换十余位），内阁票拟徒具虚名 |

### 1.4 内阁与六部的权力关系变化

- **早期（永乐至正统）**：六部直接对皇帝负责，不必听命于内阁。内阁与六部平行，阁臣品级甚至低于六部尚书（正二品）。
- **中期（成化至嘉靖）**：阁臣通过兼衔（兼尚书、加太子三师等）品级提升，逐步凌驾六部。嘉靖后"朝位班次，俱列六部之上"。六部用人权、兵权、监察权等受到内阁支配与制约。
- **晚期（万历至崇祯）**：出现"凡百政事，非下部必不可行"的格局——兵部调兵、工部营造、户部度支等权力均被内阁侵蚀。但内阁权力大小终以皇帝向背为转移，且始终缺乏法律制度保障。

## 二、组织结构

### 2.1 首辅、次辅、群辅

| 称谓 | 定义 | 职权特点 |
|------|------|----------|
| **首辅** | 内阁中入阁最早、资历最深、加官最高者，通常最受皇帝信任 | 主持票拟（尤其嘉靖至万历初年为"专票"），主持内阁大政，地位远高于其他阁臣 |
| **次辅** | 排名第二的阁臣 | 协助首辅，嘉靖后与首辅权力悬殊，天启以前无明确职责分工 |
| **群辅** | 首辅、次辅之外的其余阁臣 | 参与议政、分担票拟，地位较低 |

- 首辅人选的确定依据：入阁先后、资历、才能、声望、皇帝旨意。
- 首辅之称起于天顺年间李贤，此后渐成惯例。嘉靖以来，"首辅莫不由倾轧排挤而得之"。

### 2.2 殿阁大学士官衔体系

嘉靖改制后，明朝共有**六个殿阁大学士**名号，按约定俗成的排序从高到低：

| 排序 | 殿阁名称 | 备注 |
|------|----------|------|
| 1 | **中极殿大学士** | 嘉靖前称"华盖殿大学士"，地位最尊 |
| 2 | **建极殿大学士** | 嘉靖前称"谨身殿大学士" |
| 3 | **文华殿大学士** | 原设以辅导太子 |
| 4 | **武英殿大学士** | — |
| 5 | **文渊阁大学士** | 内阁即以文渊阁为办公地 |
| 6 | **东阁大学士** | 地位最低，新入阁者常授此衔 |

### 2.3 品级

- **大学士本职品级**：正五品（自洪武十五年设置起未变）。
- **实际品级通过加衔提升**：
  - 加公、孤或太子三师、三少→正一品或从一品
  - 加尚书衔→正二品
  - 加侍郎衔→正三品
  - 未加官者仅正五品
- 嘉靖以后，入阁者例兼六部尚书衔，实际品级多为正二品以上。首辅通常加至少师、少傅或太子太师等，为从一品乃至正一品。

### 2.4 内阁人数

- **无固定定额**，一般为**3至5人**。
- 最多时达**10人**（如嘉靖二十一年），最少时仅**1人**（如万历后期方从哲独相七年、叶向高独相数年）。

## 三、票拟制度

### 3.1 定义

票拟（又称"条旨"）是指内阁大学士代皇帝草拟对臣下奏章的批答意见，用小票（纸条）墨书，贴于奏章封面，呈送皇帝裁决。实质上是代拟"御批"草稿。

### 3.2 形成过程

| 时期 | 发展阶段 |
|------|----------|
| **仁宗洪熙元年（1425）** | 出现"条旨"，即阁臣为奏章拟写处理建议，但尚非专责 |
| **宣宗宣德年间（1426–1435）** | 宣宗令内阁用小票墨书贴于奏章封面，再呈皇帝。票拟形式初定 |
| **英宗正统年间（1436–）** | 英宗年幼（9岁即位），太皇太后张氏不便与群臣面议，**专令内阁条旨**，票拟从此成为内阁专责并制度化 |

### 3.3 具体操作流程

```
① 通政使司汇总全国奏章
    ↓
② 司礼监将奏章呈报皇帝过目
    ↓
③ 皇帝发至文渊阁
    ↓
④ 内阁阁臣阅读奏章，写出内容提要（"引黄"或"贴黄"）
    ↓
⑤ 阁臣用小票墨书拟写批答意见（"票拟"），贴于奏章封面
    ↓
⑥ 将奏章连同贴黄、票拟一并呈送皇帝
    ↓
⑦ 皇帝裁决：
   - 同意 → 朱笔批示（"批红"/"批朱"），或口述令司礼监秉笔太监代书
   - 不同意 → 发回内阁令重新拟定（"改票"）
   - 暂不处理 → 搁置（"留中"）
    ↓
⑧ 批红后的奏章由六科给事中校对，下发各部执行
```

### 3.4 票拟与批红的关系

- **票拟**：内阁代拟，用墨笔（黑色）书写。
- **批红**：皇帝（或司礼监代行）最终裁决，用朱笔（红色）书写。
- 批红是最终决策，票拟仅为建议。皇帝可以完全不采纳票拟。
- 实际上多数情况下，皇帝不亲笔批红，而是口述大意，由**司礼监秉笔太监**代为"照阁票批红"。

### 3.5 票拟权的归属

- **天顺以前**：阁臣共同议定票拟，无首辅专票之制。
- **天顺以后**：李贤得英宗信任，主持票拟，此后**首辅主掌票拟（"专票"）**成为惯例。
- **嘉靖至万历初年**：首辅独票权达到顶峰。严嵩专断，"与次辅不相关白"；张居正为首辅时，"三日不出阁，吏以函捧章奏就第票拟，次相在阁坐候，票进乃出"。
- **天启年间**：内阁由首辅"协恭"转为阁臣**"分票"**，即将奏章分配给各阁臣分别票拟。
- **崇祯年间**：因阁臣倾轧，"分票"成为常态，各人署名以便追查责任，票拟权被分散，内阁票拟徒具虚名。

### 3.6 票拟与中旨的冲突

- **中旨**：皇帝绕过内阁票拟程序，直接通过手诏或宦官传口谕下达的旨意。
- 正常程序要求所有政令须经内阁票拟，皇帝主动提出的意见也应"通过手诏下内阁票拟"。
- 对中旨，内阁**可以奉行，也可以拒绝封还**，均属合法。拒绝中旨被视为阁臣维护制度的正当行为。
- 嘉靖、万历等朝，中旨与票拟的冲突时有发生，是皇权与阁权博弈的重要表现。

## 四、入阁途径

### 4.1 基本资格

- 明代阁臣绝大多数出身于**进士→翰林**的路径，即"非进士不入翰林，非翰林不入内阁"。
- 翰林院是入阁的必经之路，其他途径极为罕见。

### 4.2 特简

- **定义**：由皇帝不经吏部，直接以亲自下旨的方式任命阁臣。
- **三种形式**：
  1. 由现任阁臣推荐人选，皇帝从中点用
  2. 宦官援引或皇帝采纳大臣荐举
  3. 皇帝本人亲自选拔
- **适用情况**：内阁初建时期（永乐至正统），阁臣**均由皇帝亲自选任**，全属特简。
- **历史数据**：据统计，明代全部161位阁臣中，特简入阁者46人，占28.6%。
- **案例**：永乐元年解缙等七人入阁、严嵩入阁、张居正入阁等均属特简。

### 4.3 廷推（会推）

- **定义**：由吏部会同九卿（六部尚书、都察院左都御史、大理寺卿、通政使）及科道（六科给事中、十三道监察御史）推举若干人选，呈报皇帝择用。
- **特点**：相对公道，参与面广，具有一定的民主性。
- **适用时期**：正统时始有推举入阁之例（曹鼐、马愉由推举入阁）。**万历以后**，廷推逐渐成为入阁的主要方式。
- **案例**：万历朝王锡爵、叶向高等经廷推入阁。

### 4.4 枚卜

- **定义**：由吏部推荐候选名单，皇帝以**抓阄**方式决定阁臣人选。
- **适用时期**：初行于**万历晚期**，盛于**崇祯年间**。
- **出现原因**：晚明党争加剧，廷推往往陷入僵局，无法达成共识，故以抓阄解决争议。
- **案例**：崇祯朝多次以枚卜方式选任阁臣。

## 五、内阁办公

### 5.1 办公地点

- **文渊阁**：位于紫禁城内左顺门（后称协和门）外东南角，面对皇城砖墙黄瓦。
- 永乐初年内阁设于**东角门内**，阁臣与皇帝"昼夜见无时"。
- **宣德七年（1432年）**：翰林院新署落成，翰林院迁出文渊阁，文渊阁正式成为**内阁官署**。
- 文渊阁共**十间房**：
  - 西五间中间的一间悬"文渊阁"牌匾
  - 西边为阁臣**拟票场所**
  - 东边为**藏书室**
- "文渊阁为禁中深严之地，门榜圣旨，闲杂莫敢窥者。"
- 办公条件并不优越。据明末文秉记载，文渊阁采光不佳，崇祯朝阁臣薛国观不得不额外开窗、打开一道门来解决照明问题。

### 5.2 内阁下属机构

宣德八年（1433年）起，内阁设两个直属文书机构：

| 机构 | 职能 | 人员 |
|------|------|------|
| **制敕房** | 起草制书、诏书 | 中书舍人若干 |
| **诰敕房** | 起草诰命（封赠官员及其先人的文书） | 中书舍人若干 |

- 中书舍人负责书写诰敕、制诏、银册、铁券等，起草初稿通常请翰林院官员执笔，再经内府加盖宝印。
- 大朝会时，中书舍人随班侍立。

### 5.3 日常办公流程

1. **早朝**：阁臣参加早朝（御门听政），与百官面见皇帝。
2. **入阁办事**：早朝后回到文渊阁处理政务。
3. **阅看奏章**：当日由通政使司/司礼监转来的奏章，阁臣逐一阅读。
4. **票拟**：对各件奏章草拟批答意见，用小票墨书贴于奏章封面。
5. **呈送**：票拟完成后将奏章呈送司礼监，转呈皇帝。
6. **起草诏书**：皇帝批红后，内阁据批红起草正式诏令，交制敕房/诰敕房誊写。
7. **面议**：遇重大疑难之事，皇帝可召阁臣入对面议。

### 5.4 与司礼监的文书往来流程

```
外廷奏章 → 通政使司汇总 → 司礼监文书房收发
    ↓
司礼监呈皇帝过目 → 发至文渊阁（内阁）
    ↓
内阁票拟 → 呈送司礼监
    ↓
司礼监掌印太监审核 → 秉笔太监代皇帝批红（或皇帝亲批）
    ↓
批红后下发六科校对 → 各部执行
```

- **司礼监掌印太监**：掌理内外章奏及御前勘合。
- **司礼监秉笔太监**：照阁票批朱（代皇帝批红）。
- **司礼监文书房**：收发内外一切章疏、圣谕、旨意、票拟，功能类似今日政府秘书处。
- 内阁与司礼监构成"内外相维"的双轨辅政体制。皇帝高踞顶端，内阁和司礼监分居底边两端，形成不等边三角形。
- 内阁有票拟权，司礼监有批朱权，互相制约。但内阁权力大小终以皇帝向背为转移。

### 5.5 阁臣的待遇

- **俸禄**：按实际加衔品级领取。大学士本职正五品，月俸16石。加尚书衔后为正二品，月俸61石。加至正一品则月俸87石。（注：明朝1石约合76.5公斤米）
- **赐食**：阁臣例享**光禄寺赐膳**，即由光禄寺每日供给膳食。
- **赐物**：逢年节皇帝赐银、赐衣等。
- **特殊待遇**：首辅可乘轿入紫禁城；阁臣可入内廷办事，与皇帝关系密切。
- 总体而言，明代官俸极薄，"明代官俸之薄，堪称历代之最"，但阁臣因加衔品级高且有各种赐予，实际待遇在文官中最为优厚。

## 关联主题

- **司礼监**：与内阁构成"双轨辅政"体制，掌批红权，是制衡内阁的关键力量
- **六部**：行政执行机关，与内阁存在长期的"阁部之争"
- **通政使司**：负责汇总全国奏章，是票拟流程的起点
- **六科给事中**：负责校对批红后的诏令，有封驳权
- **翰林院**：入阁的必经之路，"非翰林不入内阁"
- **大礼议**：嘉靖初年的政治事件，直接影响了多位首辅（杨廷和、蒋冕、毛纪、张璁）的升降
- **东林党争**：万历后期至天启崇祯，党争深刻影响内阁人事
- **宦官专权**：天启朝魏忠贤专权，"内外大权一归宦寺"，内阁沦为附庸

## 存疑 & 待查

1. 嘉靖朝夏言与严嵩多次交替执政，各次的精确起止月份在不同史料中略有出入，需进一步核对《明世宗实录》。
2. 万历三十六年至四十年间内阁是否完全无首辅，还是有短暂代理者，待查证。
3. 崇祯朝首辅更替极为频繁（一般称"十七年换十九首辅"或"换十八首辅"），不同统计口径略有差异，取决于是否计入代理、短期任职者。本表列出的是主要共识名单。
4. 阁臣光禄寺赐膳的具体规格（菜品、份量等）未能查到详细记载。
5. 顾鼎臣是否正式担任首辅存疑——部分学者认为他仅是"留阁代理"，并非正式首辅。
