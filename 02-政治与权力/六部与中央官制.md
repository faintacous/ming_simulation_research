# 六部与中央官制

## 概述

明太祖洪武十三年（1380年），朱元璋废除中书省和丞相制度，仿《周官》六卿之制，升六部品秩，使六部直接对皇帝负责。六部成为明朝中央行政的核心机构，统管全国政务。本文聚焦嘉靖至崇祯年间（1521–1644）的制度实况，兼及制度沿革。

---

## 一、六部总论

### 1.1 六部排序与地位

六部按传统排序为：**吏、户、礼、兵、刑、工**。其中吏部居首，吏部尚书被称为"天官"、"六部之长"、"文官之首"。

### 1.2 各部通用编制

每部设官如下：

| 职位 | 员额 | 品级 |
|------|------|------|
| 尚书 | 1人 | 正二品 |
| 左侍郎 | 1人 | 正三品 |
| 右侍郎 | 1人 | 正三品 |
| 司务 | 2人 | 从九品 |

各部下设清吏司，司内设：

| 职位 | 品级 | 说明 |
|------|------|------|
| 郎中 | 正五品 | 各司主官 |
| 员外郎 | 从五品 | 副主官 |
| 主事 | 正六品 | 司内办事官 |

- 吏部、礼部、兵部、工部各设 **4个** 清吏司
- 户部、刑部各设 **13个** 清吏司（按省份设置）

### 1.3 六部与内阁的关系

明初六部直隶皇帝，无宰相统辖。永乐以后内阁制度逐步形成，至嘉靖—万历前期达到鼎盛：

- **票拟**：内阁阁臣在奏疏上附写处理意见（用小纸条贴上），称为"票拟"
- **批红**：司礼监秉笔太监用朱笔誊写票拟意见于奏疏上，掌印太监盖章，称为"批红"
- **部议/部覆**：票拟经批红后，经六科发给六部各司，各司再交部里协商处理

**嘉靖—崇祯时期的实际关系：**

| 时期 | 内阁与六部关系 |
|------|--------------|
| 嘉靖朝 | 嘉靖帝乾坤独断，亲自批阅奏疏，内阁权力受限，更像传话筒 |
| 隆庆—万历前十年 | 内阁权力鼎盛，首辅高拱、张居正权压六部，"朝位班次，俱列六部之上" |
| 万历中后期 | 万历怠政，不给内阁票拟权也不给宦官批红权，行政几近停摆 |
| 崇祯朝 | 崇祯勤政但频繁更换内阁，17年间更换50余位阁臣，内阁无法稳定运作 |

**关键制度特征：** 内阁始终不是法定的行政机构，阁臣品级多为五品（翰林学士），依靠加衔（如太子太保、尚书衔等）提升地位。内阁无直接指挥六部的法定权力，其权力依赖皇帝授权和个人政治影响力。

### 1.4 南京六部

明成祖迁都北京后，南京作为陪都保留了完整的六部建制，称"南京六部"。各部均设尚书一人、侍郎等官。

| 南京部门 | 实际职能 |
|----------|---------|
| 南京吏部 | 考核南京六部及南直隶地区官吏，主持南京地区六年一度的京察，北京吏部不得干涉 |
| 南京户部 | 征收南直隶及浙江、江西、湖广三省的布税食粮，管理南直隶户籍；**实权最大** |
| 南京礼部 | 主管南直隶祭祀事务，尤其是明太祖陵墓孝陵的祭祀 |
| 南京兵部 | 负责南京及附近地区的军事守备，由南京守备太监监军；**职权仅次于南京户部** |
| 南京刑部 | 管理南直隶地区的司法审判 |
| 南京工部 | 管理南京地区的工程营建 |

南京六部多被视为闲职或谪贬之所，有"翰林院养望、南京养老"之说。但南京户部因掌管江南财赋重地、南京兵部因掌管南京军事守备，仍具有相当实权。

## 二、吏部

吏部掌全国文官铨选、考课、封爵、勋级之政，号称"六部之首"。

### 2.1 四个清吏司

洪武二十九年（1396年）定制，吏部下设四个清吏司：

| 清吏司 | 主官 | 主要职能 |
|--------|------|---------|
| **文选清吏司** | 郎中1人（正五品）、员外郎1人（从五品）、主事2人（正六品） | 掌文职官员品级、开列、考授、拣选、升调，办理月选 |
| **验封清吏司** | 郎中1人、员外郎1人、主事1人 | 掌封爵、世职、恩荫、难荫、请封、捐封，管理诰敕、承袭、印信等事 |
| **稽勋清吏司** | 郎中1人、员外郎1人、主事1人 | 掌文职勋级、月俸、升转资格、品级、官制、贴黄，办理守制、终养、出继、入籍、复名复姓等事 |
| **考功清吏司** | 郎中1人、员外郎1人、主事2人 | 掌文职官员的处分及议叙，办理京察、大计 |

其中文选司权力最大，掌握文官选拔升迁的实权。

### 2.2 铨选制度（官员选任流程）

明朝选官主要途径：

| 选任方式 | 适用范围 | 说明 |
|----------|---------|------|
| **特简** | 高级官员 | 皇帝直接任命 |
| **廷推** | 三品以上大臣 | 由廷臣集体推举，皇帝裁定 |
| **部推** | 三品以上大臣 | 由吏部主持推举 |
| **听选/铨注** | 中下级官员 | 吏部按资格铨叙注拟 |
| **行取** | 科道官 | 由推官、知县补授科道 |
| **考选** | 科道官 | 举人、贡生、推官、知县考选科道 |
| **月选** | 低级官员 | 文选司每月进行的例行选授 |

**科举正途**：明朝以科举为选官"正途"。詹事府、翰林院、吏部、礼部各司郎官，必须科甲正途出身始能充任。

**铨选流程概要**：
1. 官缺出现 → 文选司按官缺类别汇总
2. 查阅候选官员资格（出身、资历、考核成绩）
3. 按"类推"方式拟定人选 → 呈堂官审定
4. 高级官员经廷推/部推 → 皇帝裁定
5. 中下级官员由吏部铨注 → 请旨发授

### 2.3 考核制度

明朝文官考核分为 **考满** 和 **考察** 两大类。

#### 考满制度
- 官员任职满三年为"初考"，六年为"再考"，九年为"通考"
- 考满等级：称职、平常、不称职
- 称职者升，平常者复职（留任），不称职者降

#### 京察
- **对象**：在京各衙门官员
- **周期**：弘治十七年（1504年）定制，每 **6年** 举行一次
- **四品以上**：经科道纠劾及年老不堪任事者，各自陈请致仕，取自上裁（即自行请辞，皇帝裁决）
- **五品以下**：由吏部主持一体考察
- **流程**：在京各衙门属官由堂上官出具考语 → 送都察院覆考 → 吏部汇总 → 分别处置
- **考察标准（"八法"）**：

| 弊病 | 说明 |
|------|------|
| 贪 | 贪污受贿 |
| 酷 | 残酷暴虐 |
| 浮躁 | 办事浮夸急躁 |
| 才力不及 | 才能不足以胜任 |
| 年老 | 年纪过大 |
| 有疾 | 身患疾病 |
| 疲软 | 懒惰疲怠 |
| 不谨 | 行为不检 |

- **处置结果**：贪、酷者为民（革职为民）；浮躁、才力不及者降调；年老、有疾者致仕（退休）；疲软、不谨者冠带闲住
- 京察中被评为"才力不及"者，一律降一级调外任

#### 大计（外察）
- **对象**：地方官员（外官）
- **周期**：每 **3年** 举行一次（配合朝觐之年）
- **标准**：与京察"八法"基本一致
- **流程**：各省布政使、按察使汇总属官考语 → 报吏部 → 吏部会同都察院考察 → 分别处置

#### 考成法（张居正改革）
万历初年，首辅张居正推行考成法，以六科监督六部，以内阁监督六科，层层考核，限期完成政务，大幅提高行政效率。张居正去世后考成法废弛。

## 三、户部

户部掌全国疆土、田地、户籍、赋税、俸禄及一切财政事务，事务最为繁重。

### 3.1 十三个清吏司

洪武二十三年（1390年）按行省设置属司，宣德年间定为十三司。每司以省名命名，兼管本省钱粮及分管其他专项事务。

| 清吏司 | 郎中 | 员外郎 | 主事 |
|--------|------|--------|------|
| 浙江清吏司 | 1人 | 1人 | 4人 |
| 江西清吏司 | 1人 | 1人 | 4人 |
| 湖广清吏司 | 1人 | 1人 | 4人 |
| 陕西清吏司 | 3人 | 1人 | 4人 |
| 广东清吏司 | 1人 | 2人 | 2人 |
| 山东清吏司 | 2人 | 1人 | 3人 |
| 福建清吏司 | 1人 | 1人 | 4人 |
| 河南清吏司 | 1人 | 1人 | 4人 |
| 山西清吏司 | 4人 | 1人 | 4人 |
| 四川清吏司 | 1人 | 1人 | 3人 |
| 广西清吏司 | 1人 | 1人 | 2人 |
| 云南清吏司 | 3人 | 1人 | 9人 |
| 贵州清吏司 | 3人 | 1人 | 3人 |

> 各司编制不等，取决于所辖省份事务繁简。山西司郎中多达4人，云南司主事多达9人，反映了边疆事务的繁杂。

各司除管本省钱粮外，还分管户部的专项事务（如某司管太仓银出纳，某司管漕粮，某司管盐引等）。

### 3.2 田赋征收

- 洪武二十六年（1393年），全国官私田地共计约 850.7万顷
- 田赋是明朝最重要的财政收入来源
- 征收流程：地方里甲 → 知县汇总 → 知府上报 → 布政使司汇总 → 户部核收
- 赋税种类：夏税（小麦为主）、秋粮（米谷为主）
- 万历九年（1581年）张居正推行 **一条鞭法**：将各种赋役合并，折算银两征收，简化征收流程

### 3.3 太仓银管理

太仓（又称太仓库）是户部直管的中央银库，位于北京。

- 储存来自全国的税银，是国家财政的核心储备
- 岁入项目经历了从实物（粮食、布帛）到白银的转变
- 嘉靖以后，太仓银收入逐年增加但支出亦剧增（军饷、赈灾）
- 万历末年至崇祯年间，太仓银长期入不敷出，是明朝财政崩溃的直接体现

### 3.4 漕运

漕运即通过运河将南方粮食运往北京，是关乎京师命脉的大事。

- 设 **漕运总督** 一员（驻淮安），总理漕务，监督八省漕粮
- 漕运总督非户部属官，由朝廷特派，通常加兵部尚书或侍郎衔
- 每年漕粮运输量约 400万石
- 漕船由各省卫所军士充任运丁
- 户部负责核算漕粮数额、签发仓单，漕运总督负责实际运输管理

### 3.5 盐政

盐税是仅次于田赋的第二大财政收入。

- 明朝实行 **开中法**（以盐引换取商人运粮至边境）→ 后改为 **纳银开中**
- 设盐运使司管理产盐区（两淮、两浙、长芦、山东、河东、福建等）
- 盐运使，从三品
- 嘉靖以后，各地另设 **盐政**（巡盐御史），由都察院差遣监管盐务
- 户部负责盐引的发放和盐课银的核收

### 3.6 户部其他附属机构

| 机构 | 职能 |
|------|------|
| 宝钞提举司 | 管理钞币印造（后期名存实亡） |
| 钞关 | 内河商税征收（嘉靖以后设置渐多） |
| 各仓 | 京师设太仓等十三仓，储存漕粮 |

## 四、礼部

礼部掌典礼、科举、外交、祭祀、宴飨之政。

### 4.1 四个清吏司

洪武二十九年（1396年）定制：

| 清吏司 | 主要职能 |
|--------|---------|
| **仪制清吏司** | 掌嘉礼、军礼及管理学校、科举考试事务。负责朝会、册封、冠婚等礼仪的制定与执行 |
| **祠祭清吏司** | 掌吉礼、凶礼。管理坛庙祭祀、天文历法、僧道寺观、丧葬谥号等事务 |
| **主客清吏司** | 掌宾礼及接待外国使臣。管理朝贡国事务、外交礼仪、边疆民族贡赐往来 |
| **精膳清吏司** | 掌筵飨、牲醴之事。负责宫廷宴会、祭品的准备和供应 |

每司设郎中1人（正五品）、员外郎1人（从五品）、主事1—2人（正六品）。

### 4.2 科举考试管理

礼部（主要通过仪制司）是科举考试的主管部门：

| 考试级别 | 管理方 | 频率 | 说明 |
|----------|--------|------|------|
| 院试 | 各省学政（提学御史/提学副使） | 每年/隔年 | 取秀才（生员），入府州县学 |
| 乡试 | 各省布政使司主持，礼部派考官 | 每三年（秋八月） | 取举人，第一名为解元 |
| 会试 | 礼部主持 | 每三年（春二月） | 取贡士，第一名为会元 |
| 殿试 | 皇帝亲试 | 会试后（三月） | 赐进士及第/出身/同出身，第一名为状元 |

- 会试主考官称"总裁"，由礼部尚书或侍郎担任，另派同考官若干
- 会试录取名额明初约100人，嘉靖以后增至300—400人
- 乡试、会试均用 **八股文** 格式，以《四书》《五经》命题

### 4.3 朝贡体系（外交礼仪）

由主客司主管：

- 朝贡国来使先到礼部报到 → 主客司验看贡物和国书 → 安排馆驿住宿
- 安排朝见皇帝的礼仪、赐宴
- 管理会同馆（外国使臣驻所）
- 颁赐回赠物品
- 主要朝贡国：朝鲜、琉球、安南、暹罗、日本（嘉靖后断贡）、满剌加等
- 嘉靖二年（1523年）"争贡之役"后，明朝与日本断绝贡使往来

### 4.4 皇室礼仪

- 礼部负责制定和执行皇帝即位、大婚、册封后妃、皇子命名、皇室丧葬等一切礼仪
- 嘉靖朝的"大礼议"（1521—1524年）是礼部与皇权冲突的典型案例——嘉靖帝坚持追尊生父兴献王为帝，与礼部及朝臣产生激烈冲突
- 祠祭司负责天坛、地坛、太庙、社稷坛等国家重要祭祀的礼仪安排

## 五、兵部

兵部掌全国武官选授、军政军令、军队调遣、驿传、军器等事务。

### 5.1 四个清吏司

洪武二十九年（1396年）定制：

| 清吏司 | 主要职能 |
|--------|---------|
| **武选清吏司** | 掌卫所土官的选授、升调、袭替、功赏。相当于武官的"人事部门" |
| **职方清吏司** | 掌舆图（地图）、军制、城隍（城防）、镇戍、简练（检阅）、征讨。相当于"作战参谋部门" |
| **车驾清吏司** | 掌卤簿（皇帝仪仗）、仪仗、禁卫、驿传（驿站系统）、厩牧（马政） |
| **武库清吏司** | 掌戎器（兵器）、符勘（兵符）、尺籍（军籍）、武学（武官教育）、薪隶（柴薪供应） |

每司设郎中1人（正五品）、员外郎1人（从五品）、主事1—2人（正六品）。

### 5.2 武官选任

- 武官世袭：卫所军官多为世袭（父死子继），由兵部武选司管理袭替手续
- 武举：始于天顺八年（1464年），分策论和武艺两部分，由兵部主持
- 武学：设于京师，培养武官子弟
- 中下级武官由兵部铨选，高级武官由皇帝特简

### 5.3 军令与军政的区别

明朝军事管理实行 **兵部与五军都督府分权制衡** 的体制：

| | 兵部 | 五军都督府 |
|---|------|----------|
| 性质 | 文官系统 | 武官系统 |
| 核心权力 | **有出兵之令，而无统兵之权** | **有统兵之权，而无出兵之令** |
| 军政 | 武官选授、考核、军籍管理 | 卫所将领的选拔更替（配合兵部） |
| 军令 | 调兵遣将、发布征讨命令 | 负责日常训练、屯田管理 |
| 军事情报 | — | 掌控军事情报（舆图、边报） |

> **制度演变**：明中期以后，五军都督府权力逐渐被兵部侵蚀。嘉靖以后，兵部实际上掌握了大部分军政和军令权力，五军都督府日益虚化。

### 5.4 兵部与五军都督府

五军都督府分为中、左、右、前、后五军，各设：

| 职位 | 品级 |
|------|------|
| 左都督 | 正一品 |
| 右都督 | 正一品 |
| 都督同知 | 从一品 |
| 都督佥事 | 正二品 |

五军都督府的基本职责：
1. 卫所将领的选拔与更替
2. 卫所军士的训练
3. 军户屯田的管理
4. 军事情报的掌控

### 5.5 兵部尚书与总督/巡抚的关系

- **总督**：多加兵部尚书或侍郎衔、都察院右都御史衔，负责数省军务，又称"部堂"
- **巡抚**：多加兵部右侍郎、都察院右副都御史衔，负责一省军民政务，又称"中丞"
- 总督、巡抚名义上受兵部节制，地方军事须向兵部汇报
- 嘉靖以后，重要边镇均设总督（如蓟辽总督、宣大总督、三边总督等），形成文官统兵的格局
- 兵部尚书通过加衔于总督的方式，将中央军事权力延伸到地方

### 6.2 审判流程

根据万历时期刑部主事姚履素的记录，刑部审理案件的一般流程如下：

1. **告状投递**：告状人具通状（诉状）过通政司
2. **分案受理**：通政司准后，解到刑部巡风厅查验过堂 → 过司务厅 → 分拨至十三司中对应省份之司
3. **提取人犯**：本司出牌（传票），行兵马司提人（拘传）
4. **审理**：提到后从巡风厅、司务厅解到本司审理
5. **关押**：重罪者入刑部监狱，轻罪者由兵马司监保
6. **复核**：限日过堂（在本部复审） → 限日过大理寺（先司审，再堂审）
7. **执行**：大理寺评允后送回刑部执行
   - 笞杖罪：送打断官处决杖刑
   - 徒罪：送有司衙门发配
   - 军罪：送兵部发遣
   - 纳罪银：送湖广司
   - 纸赎：送山西司

### 6.3 三法司

明朝中央司法由三法司共同运作：

| 机构 | 长官 | 品级 | 职能 |
|------|------|------|------|
| **刑部** | 尚书 | 正二品 | 主审——受理天下刑名，审判各类案件 |
| **都察院** | 左都御史 | 正二品 | 纠察——监督审判过程，纠举不法 |
| **大理寺** | 大理寺卿 | 正三品 | 驳正——复核刑部、都察院的判决，有权驳回改判 |

> "刑部受天下刑名，都察院纠察，大理寺驳正。"——三者分工明确、相互制约。

**三法司会审**：遇重大疑案，三法司长官会同审理，称"三司会审"。

### 6.4 朝审与热审

#### 朝审
- **设立**：明英宗天顺三年（1459年）正式确立
- **时间**：每年霜降之后
- **参与者**：三法司长官会同公、侯、伯等高官
- **对象**：在京刑部狱中的大案重囚（监候死罪犯）
- **处置分类**：

| 类别 | 说明 |
|------|------|
| 情实 | 案情属实，维持死刑，予以处决 |
| 缓决 | 案情尚有疑虑，缓期再审 |
| 可矜 | 情有可原，免于一死 |
| 留养承祀 | 家中无人奉养父母或承续祭祀，免死 |

#### 热审
- **设立**：始于明成祖永乐年间
- **时间**：每年小满后十余日（夏季）
- **目的**：疏通监狱，防止暑热导致囚犯瘐毙（病死狱中）
- **参与者**：司礼监太监、锦衣卫会同三法司堂官
- **对象**：初期仅审轻罪（徒流以下），成化朝扩大至重罪
- **处置**：笞杖罪案件减等发落，徒刑、流刑酌情处理

## 七、工部

工部掌全国工程营建、器物制造、屯田水利等事务。在六部中地位最低。

### 7.1 四个清吏司

| 清吏司 | 主要职能 |
|--------|---------|
| **营缮清吏司** | 掌经营兴造之事。包括宫殿、陵寝、城濠、坛庙、衙署、仓库、营房的营建，负责调集人力物力、督促工程进度 |
| **虞衡清吏司** | 掌山泽采捕、制陶冶炼。管理铸器、铸钱事宜，各衙门所需铜铁器皿由虞衡司所辖宝源局铸造打制 |
| **都水清吏司** | 掌川渎、陂池、桥道、舟车、织造、衡量之事。职责主要在河渠治理与上供织造两方面 |
| **屯田清吏司** | 掌屯垦、抽分（赋税）、薪炭、夫役、墓碑制定等事务 |

每司设郎中1人（正五品）、员外郎1人（从五品）、主事1—2人（正六品）。

### 7.2 工程营建管理

- 大型工程（宫殿、陵寝、城墙）由工部统筹，营缮司具体执行
- 征调工匠：明初实行 **匠户制**，工匠按籍编入匠户，轮班赴京服役
- 嘉靖以后，匠户可以 **纳银代役**（交钱免除劳役），工部改为雇工兴建
- 重大工程设专职提督官（如嘉靖朝重建三大殿）

### 7.3 军器制造

- 由虞衡司管理，工部所辖的 **军器局** 负责制造火器、兵器等军用物资
- 嘉靖以后，火器制造日益重要（鸟铳、佛郎机等）
- 另设 **兵仗局**（隶属内府），由宦官管理，与工部军器局形成内外两套军工体系

### 7.4 宝源局（铸币）

- 宝源局是工部虞衡司下属机构
- 负责铸造铜钱（制钱）
- 明代主要流通货币：洪武通宝、永乐通宝、嘉靖通宝、万历通宝、崇祯通宝等
- 宝源局在北京设局铸造，各省另设宝泉局（隶属户部）
- 嘉靖六年（1527年）重新开铸制钱，此后铸币规模逐渐扩大

## 九、地方官制简述

### 9.1 总督与巡抚

#### 总督
- **性质**：原为临时差遣，非正式官职。正德、嘉靖年间各地先后设置，但至明亡仍未完全制度化为常设官
- **品级**：本身无固定品级，依加衔确定——通常加兵部尚书衔（正二品）或都察院右都御史衔（正二品），又称"部堂"
- **管辖**：跨省军务，如蓟辽总督（辖蓟州、辽东）、宣大总督（辖宣府、大同）、三边总督（辖延绥、宁夏、甘肃）、两广总督等
- **职能**：总管数省军务，节制巡抚及地方三司

#### 巡抚
- **性质**：初为京官巡视地方的差遣，宣德时逐渐常设化。景泰四年（1453年），巡抚挂衔都御史成为定制
- **品级**：通常加兵部右侍郎（正三品）、都察院右副都御史衔（正三品），又称"中丞"
- **管辖**：一省或数府
- **职能**：总揽一省军民政务，为实际上的地方最高长官

### 9.2 三司（省级机构）

明初在各省设三司，分管行政、司法、军事，互不统属，各自对中央负责：

| 机构 | 长官 | 品级 | 职能 |
|------|------|------|------|
| **承宣布政使司**（藩司/藩台） | 左右布政使各1人 | 从二品 | 掌一省民政、财赋、户籍。下设左右参政（从三品）、左右参议（从四品） |
| **提刑按察使司**（臬司/臬台） | 按察使1人 | 正三品 | 掌一省刑名按劾。下设副使（正四品）、佥事（正五品），分道管理 |
| **都指挥使司**（都司） | 都指挥使1人 | 正二品 | 掌一省军务。下设都指挥同知（从二品）、都指挥佥事（正三品） |

> 嘉靖以后，三司权力逐渐被巡抚接掌。布政使、按察使从一省之长变为巡抚的下属。

### 9.3 府、州、县

| 层级 | 长官 | 品级 | 佐官 | 职责 |
|------|------|------|------|------|
| **府** | 知府1人 | 正四品 | 同知（正五品，无定员）、通判（正六品，无定员）、推官1人（正七品） | 掌一府政务：清军、巡捕、管粮、治农、水利、屯田、牧马等 |
| **直隶州** | 知州1人 | 从五品 | 同知（从六品）、判官（从七品） | 地位与府相当，直属布政使司 |
| **属州** | 知州1人 | 从五品 | 同知、判官 | 隶属于府 |
| **县** | 知县1人 | 正七品 | 县丞1人（正八品）、主簿1人（正九品） | 掌一县政务 |

> 应天府（南京）地位特殊，设府尹（正三品）、府丞（正四品）。顺天府（北京）亦同。

**知县职责（《明史·职官志》）**：

凡赋役，岁会实征，十年造黄册，以丁产为差。养老、祀神、贡士、读法、表善良、恤穷乏、稽保甲、严缉捕、听狱讼，皆躬亲厥职而勤慎焉。

明代知县施政"六事"：兴学校、辟田野、增户口、平赋役、简词讼、息盗贼。

**府的等级划分**：
- 上府：粮20万石以上
- 中府：粮20万石以下
- 下府：粮10万石以下

**县的等级划分**：
- 上县：粮10万石以下
- 中县：粮6万石以下
- 下县：粮3万石以下

全国共有府153个、直隶州与属州合计234个、县1171个。

## 关联主题

- **内阁制度**：票拟、批红流程，内阁首辅的权力消长
- **宦官机构**：司礼监（批红权）、东厂、锦衣卫与六部的互动
- **都察院与六科**：十三道监察御史、六科给事中对六部的监督
- **卫所制度**：兵部与五军都督府共同管理的军事编制
- **财政制度**：一条鞭法、三饷加派与户部的关系
- **科举制度**：礼部会试与翰林院的关系

## 存疑 & 待查

1. 户部十三司各自具体分管的专项事务（如哪个司管太仓出纳、哪个司管盐引），各资料记载不完全一致，需对照《大明会典》原文确认
2. 刑部十三司各司除管辖本省案件外，是否也像户部各司那样兼管专项事务（如湖广司管纳罪银、山西司管纸赎），需进一步考证其完整分工
3. 南京六部各部的具体编制人数（尚书、侍郎以下各司员额），与北京六部有何差异
4. 嘉靖朝至崇祯朝期间六部编制是否有过调整（增减郎中、主事员额等）
5. 文勋"十级"与"十一种"的计数差异——左柱国与右柱国是否算一级还是两级，不同资料表述不一
6. 武散官正一品、从一品的阶名与文散官完全相同（特进荣禄大夫等），是否在实际授予中有所区分
