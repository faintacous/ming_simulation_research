# 党争编年（1521–1644）

## 概述

明朝中后期（嘉靖至崇祯，1521–1644）的政治斗争经历了从个人恩怨到派系对抗、再到党同伐异的演变过程。大致可分为五个阶段：

1. **嘉靖朝**（1521–1567）：大礼议引发的议礼派与守礼派之争，随后演变为严嵩专权与反严力量的对峙。
2. **隆庆至万历初**（1567–1582）：徐阶、高拱、张居正三位首辅之间的权力更迭。
3. **万历中后期**（1582–1620）：张居正去世后朝政松弛，东林党与齐楚浙诸党形成系统性党争。
4. **天启朝**（1620–1627）：魏忠贤阉党与东林党的生死对决。
5. **崇祯朝**（1627–1644）：清除阉党后党争余波不断，内阁频繁更换，复社继承东林遗风。

## 一、嘉靖朝的派系（1521–1567）

### 1.1 议礼之争中的派系

#### 背景

正德十六年（1521），明武宗驾崩无嗣，堂弟兴王朱厚熜以"兄终弟及"入继大统，即嘉靖帝（明世宗）。围绕嘉靖帝生父兴献王朱祐杬的尊号问题，朝臣分裂为两大阵营。

#### 守礼派（继嗣派）

| 人物 | 官职 | 角色 |
|------|------|------|
| 杨廷和 | 内阁首辅 | 守礼派领袖，主张嘉靖应认孝宗为皇考 |
| 杨慎 | 翰林院修撰（杨廷和之子） | 左顺门哭谏的号召者，被廷杖流放云南 |

| 乔宇 | 吏部尚书 | 率群臣近二百五十人进言反对 |
| 蒋冕 | 内阁大学士 | 随杨廷和反对议礼 |
| 毛纪 | 内阁大学士 | 同上 |

**核心主张**：嘉靖帝以小宗入继大宗，应尊孝宗为"皇考"，称生父兴献王为"皇叔父"，即"继嗣"。

**核心主张**：嘉靖帝入继的是皇统而非皇嗣，可尊生父为"皇考"，即"继统不继嗣"。

#### 关键特征

- 此阶段的派系斗争以**个人恩怨和权力争夺**为主，尚未形成以地域或学术为纽带的系统性党派。
- 高拱与张居正的改革路线实质相近（张居正的许多政策以高拱为蓝本），但权力斗争使二人成为政敌。
- 张居正与宦官冯保的结盟，开创了文臣与内监合作的先例，为后来东林党与阉党的对峙埋下伏笔。

## 三、万历中后期（1582–1620）

### 3.1 东林党的形成

#### 起因：癸巳京察（1593）

万历二十一年（1593），吏部尚书孙鑨、左都御史李世达、考功司郎中赵南星主持六年一次的京察（官员考核）。赵南星等人力求公正，罢免了一些内阁大臣子弟和结党营私之人，引起执政内阁辅臣不满。

**后果**：内阁首辅王锡爵以"专权结党"的名义弹劾赵南星。赵南星被降三级调外，吏部尚书孙鑨被迫致仕。此次京察引发了"天下公党"与"辅臣私党"的论战，大批被贬斥的官员回乡后，客观上促成了"东林在野党"的成型。

#### 东林书院的创建

万历三十二年（1604），被罢免的吏部郎中顾宪成回到家乡无锡，会同弟弟顾允成及高攀龙等人，在宋代杨时（杨中立）故址重修东林书院，开始讲学。制定《东林会约》，"讲习之馀，往往讽议朝政，裁量人物"，逐渐形成了以学术为纽带的政治集团——**东林党**。

### 3.3 关键事件编年

#### 国本之争（1586–1615）

围绕册立太子的"国本之争"贯穿万历朝中后期，是党争的核心议题之一。

| 时间 | 事件 |
|------|------|
| 万历十四年（1586） | 万历帝宠爱郑贵妃，迟迟不立皇长子朱常洛为太子。朝臣上疏请立，被贬者多人 |
| 万历二十年（1592） | 王锡爵与万历帝密谋"三王并封"方案（皇长子朱常洛、皇三子朱常洵、皇五子朱常浩一并封王），遭群臣激烈反对，王锡爵被迫自劾请辞 |
| 万历二十六年（1598） | **第一次妖书案**：有人散发匿名传单《忧危竑议》，指控郑贵妃图谋换储。万历帝大怒，大量贬谪官吏 |
| 万历二十九年（1601） | 在慈圣皇太后干预和首辅沈一贯劝进下，万历帝终于册立皇长子朱常洛为太子（时已虚龄二十），皇三子朱常洵封福王 |
| 万历三十一年（1603） | **第二次妖书案**：又有匿名传单《续忧危竑议》出现。浙党首辅沈一贯借机打击东林党人 |
| 万历四十二年（1614） | 福王朱常洵离京赴洛阳就国，太子地位才真正巩固 |

**各派立场**：
- **东林党**：坚决主张"立嫡以长"，力挺皇长子朱常洛。
- **齐楚浙党**（尤其沈一贯）：表面上也支持立长，但利用妖书案等事件打击东林党。
- **郑贵妃集团**：暗中运作为皇三子朱常洵争储。

#### 京察之争

| 时间 | 事件 |
|------|------|
| 万历二十一年（1593） | **癸巳京察**：赵南星主持京察，罢免内阁大臣子弟及结党者。被王锡爵反击，赵南星被贬，东林在野党逐渐成型 |
| 万历三十九年（1611） | **辛亥京察**：东林党人与齐楚浙党围绕京察互相攻讦 |
| 天启三年（1623） | **癸亥京察**：东林党人赵南星以左都御史主持京察，罢黜齐党亓诗教、楚党官应震等人。引发阉党大规模报复 |

## 关联主题

- **矿税之争**：万历帝派遣矿监税使搜刮民间，东林党人强烈反对，齐楚浙党多持默许态度。
- **辽东军事**：党争影响了辽东军事决策，如熊廷弼案在东林与阉党之间成为攻讦工具。
- **江南经济**：东林党人多来自江南富庶地区，反对矿税与其经济利益有关。
- **书院与文社**：从东林书院→复社→几社，形成了江南士人的政治传统。
- **南明党争延续**：弘光朝的东林-阉党之争是明末党争的尾声，阮大铖等阉党余孽在南明短暂复起。

## 存疑 & 待查

1. **"十狗"完整名单**：《明史·阉党传》中"十狗"的确切十人名单各处记载不一，上表所列为逆案中"次等充军"等条目的综合，尚需核对原始史料。
2. **"十孩儿"与"四十孙"名单**：这两个名号的具体人物名单在传世文献中记载不完整，有待查阅《酌中志》《东林始末》等明代笔记。
3. **东林点将录完整108+1人对照表**：目前仅列出部分天罡星对应人物，完整名单需查阅原文全文。
4. **崇祯朝首辅确切数目**：不同史料记载为17–19人不等，原因在于是否计入代理首辅。上表列19人依据较为常见的说法。
5. **各党派的"投机分子"**：如杨维垣，先为东林辩护后投阉党；张四维，先依附高拱后转投张居正。这类人物的派系归属需要具体分析。
