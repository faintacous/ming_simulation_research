# 嘉靖朝重大事件（1521–1567）

## 概述

嘉靖朝历时45年，是明朝在位第二长的皇帝。前期锐意革新（"嘉靖中兴"），中期因壬寅宫变转向怠政修玄，后期严嵩专权、倭寇肆虐、北虏压境，内忧外患并发。这一朝是明中期向晚期转折的关键阶段。

---

## 一、大礼议（1521–1524）

### 事件经过

- **1521**：武宗驾崩无嗣，兴献王世子朱厚熜以"兄终弟及"入继大统，年号嘉靖
- **1521–1524**：大礼议爆发。以杨廷和为首的旧臣要求嘉靖过继为孝宗之子（"继嗣"），嘉靖坚持尊生父兴献王为皇考（"继统不继嗣"）
- **1524年7月12日**：嘉靖诏令去除父母尊号中的"本生"二字，群臣哗然
- **1524年7月14日**：**左顺门事件**。200余名大臣跪伏左顺门哭谏。嘉靖令锦衣卫逮捕为首者8人下诏狱，余者更为激愤，擂门大哭。嘉靖再令五品以下134人下狱拷讯，四品以上86人停职待罪
- **1524年7月20日**：廷杖五品以下官员，**16人杖毙**。大礼议以皇帝全胜告终
- **1524年9月**：嘉靖更定大礼，改称孝宗为"皇伯考"，生父为"皇考"，编纂《大礼集议》和《明伦大典》

### 关键人物

| 人物 | 身份 | 立场与角色 |
|------|------|-----------|
| **朱厚熜**（嘉靖帝） | 皇帝 | 事件主导者，坚持尊生父为皇考，借此确立皇权绝对权威 |
| **杨廷和** | 内阁首辅 | 反对派核心。主持武宗遗诏、安排嘉靖继位，但坚持"继嗣"立场以维护臣权。1524年致仕 |
| **蒋冕、毛纪** | 先后继任首辅 | 杨廷和同盟，延续反对立场，先后被迫去职 |
| **毛澄** | 礼部尚书 | 反对派礼制理论支柱，坚持以孝宗为皇考的礼法依据 |
| **杨慎** | 翰林修撰（杨廷和之子） | 左顺门事件中带头号召"国家养士百五十年，坚守节操大义而死，就在今日"，事后充军云南 |
| **张璁** | 新科进士 → 后升首辅 | 支持派核心。率先上疏提出"继统不继嗣"论，为嘉靖提供礼法理论武器 |
| **桂萼** | 观政进士 | 支持派骨干，与张璁联合上疏，后屡被提拔至礼部尚书 |
| **张太后** | 武宗生母 | 初期倾向杨廷和立场，后被迫接受嘉靖方案 |

### 矛盾核心分析

**表层矛盾——礼法之争**：嘉靖应认谁为父？"继嗣派"依据宗法礼制，认为皇位从孝宗一系传来，嘉靖须过继为孝宗之子；"继统派"认为嘉靖继承的是皇统而非皇嗣，可以尊奉生父。

**深层矛盾——皇权与阁权的较量**：杨廷和在正德末年代理朝政数月，权势极大。要求嘉靖"继嗣"的实质是：由阁臣来定义皇帝的身份与合法性来源，从而维持臣下对皇权的制约力。嘉靖拒绝的实质是：皇帝的身份不由臣下定义，皇权不受制于阁权。

**结构性矛盾——新君与旧臣的权力洗牌**：嘉靖以旁支入继，在朝中毫无根基，杨廷和集团掌控朝局。大礼议本质上也是嘉靖借礼法争端清洗旧臣、扶植自己人（张璁、桂萼等"大礼新贵"）的过程。左顺门廷杖是暴力手段确立新秩序的标志性事件。

[来源：维基百科-大礼议；百度百科-大礼议；知乎-大礼议]

## 二、嘉靖中兴 / 嘉靖新政（约1522–1540）

### 事件经过

- **1522起**：嘉靖重用张璁推行全面改革
- **主要改革内容**：
  - **吏治**：打压宦官集团，严禁宦官干政，削弱司礼监实权；裁汰冗官，严革贪风
  - **庄田清理**：至嘉靖九年（1530），查勘京畿勋戚庄田528处、计57400余顷，其中26000余顷归还业主，严禁再侵占民田
  - **外戚世封革除**：嘉靖与张璁、方献夫达成共识，永远废除外戚世封制度
  - **科举改革**：要求考试文体平实尔雅，改善各省乡试主考官选派制度
  - **军事整顿**：整顿团营编制
  - **罢撤镇守太监**：全面撤回各地镇守太监，收回宦官军权
- **约1530s–1540**：改革成效显著，但张璁本人也卷入政治斗争（与杨一清等的矛盾），三度起落
- **1539**：张璁去世，改革派失去核心推动力，改革逐渐停滞

### 关键人物

| 人物 | 身份 | 立场与角色 |
|------|------|-----------|
| **朱厚熜** | 皇帝 | 改革的最高支持者和推动者，前期英明苛察、励精图治 |
| **张璁**（后改名张孚敬） | 内阁首辅（三度入阁） | 改革总设计师。大礼议中因支持嘉靖崛起，主导全部核心改革。为人刚直清廉但树敌众多 |
| **桂萼** | 礼部尚书 → 内阁大学士 | 改革骨干。率先提出将税粮与徭役统一折银征收的思路，为后来"一条鞭法"开创先河 |
| **方献夫** | 内阁大学士 | 支持革除外戚世封，张璁改革同盟 |
| **杨一清** | 内阁首辅（1529–1531） | 早期帮助张璁入京得到重用，后与张璁产生权力冲突，被排挤致仕 |
| **夏言** | 后期首辅（1536–1541） | 张璁去职后接任首辅，性格刚直自负，延续部分改革但风格不同 |

### 矛盾核心分析

**改革派与既得利益者的冲突**：清理庄田直接触动勋贵外戚的经济利益，罢撤镇守太监削弱了宦官集团的权力根基。这些群体虽在大礼议后被压制，但暗中抵制改革。

**改革派内部的权力倾轧**：张璁、桂萼、杨一清等人虽同属"新贵"阵营，但在首辅之位的争夺上互不相让。张璁三度被排挤出京又三度复出，反映了改革派内部缺乏稳定的权力结构。

**改革的制度依赖性**：嘉靖中兴的动力高度依赖皇帝本人的关注和张璁个人的推动力。一旦皇帝兴趣转移（转向修道）、核心人物去世，改革即失去持续推进的动力——这是人治体制下改革的结构性脆弱。

[来源：维基百科-嘉靖中兴；光明日报-嘉靖革新的历史定位；百度百科-嘉靖新政/张璁]

## 三、壬寅宫变（1542）

### 事件经过

- **1542年10月21日（嘉靖二十一年）夜**：嘉靖帝宿于曹端妃宫中。宫女杨金英率领十余名宫女趁嘉靖熟睡，以黄绫布搓成的绳套套住其脖子，合力勒杀
- 因宫女紧张将绳结打成了死结，未能收紧勒毙
- 宫女张金莲中途逃出，跑往方皇后处报信
- 方皇后闻讯带人赶到救驾，嘉靖获救但已昏迷，经太医许绅急救方才苏醒
- **审讯与处置**：司礼监太监张佐、高忠审讯。结论称杨金英等人与王宁嫔首谋，曹端妃"虽不与，然始亦有谋"。杨金英等**16名宫女**连同曹端妃、王宁嫔一同被凌迟处死，抄没家产

### 关键人物

| 人物 | 身份 | 角色 |
|------|------|------|
| **朱厚熜** | 皇帝 | 被刺对象。长期虐待宫女，是事件的根本诱因 |
| **杨金英** | 宫女（常在/答应级低级嫔御） | 行刺主谋和直接执行者 |
| 苏川药、杨玉香、邢翠莲、姚淑翠、杨翠英、关梅秀、刘妙莲、陈菊花、王秀兰、徐秋花、邓金香、张春景、黄玉莲、张金莲 等 | 宫女 | 参与行刺。张金莲中途逃出报信 |
| **方皇后** | 皇后 | 救驾关键人物。闻讯赶到，组织抢救；主导审讯方向 |
| **曹端妃** | 端妃 | 事发地点的妃嫔。被审讯结论牵连，凌迟处死。历史学家普遍认为可能是冤案——方皇后借机铲除受宠妃嫔 |
| **王宁嫔** | 宁嫔 | 被指为首谋之一，凌迟处死 |
| **许绅** | 太医院御医 | 急救嘉靖帝成功，但因惊恐过度不久后病亡 |
| **张佐、高忠** | 司礼监太监 | 负责审讯，审讯结论的可信度存疑 |

### 矛盾核心分析

**直接导火索——皇帝对宫女的极端虐待**：嘉靖为炼制丹药，先后从全国征选逾千名8–14岁幼女入宫，强迫服食催经药物以采集炼丹原料（"红铅"），造成大量宫女失血、血崩乃至死亡。加之嘉靖性格多疑暴戾、喜怒无常，宫人常被责罚，杖毙者多达200余人。宫女在恐惧与绝望中铤而走险。

**深层问题——皇权不受约束下的极端人身控制**：宫女处于皇权体制最底层，没有任何申诉或逃离渠道。当被压迫者连生存都无法保障时，极端反抗成为唯一出路。

**后宫权力斗争的叠加**：方皇后主导审讯，将曹端妃牵连入案并处死，被普遍认为是借机清除嘉靖宠妃。事件中宫廷权力斗争与宫女反抗交织在一起。

**转折性影响**：壬寅宫变是嘉靖朝的分水岭。此后嘉靖移居西苑、不再上朝、沉迷修道，朝政委托内阁处理——直接为严嵩独揽大权创造了条件。

[来源：百度百科-壬寅宫变；维基百科-壬寅宫变；中国新闻网；故宫博物院]

## 四、严嵩专权（1548–1562）

### 事件经过

- **1542后**：壬寅宫变后嘉靖深居西苑修道，朝政日渐委于内阁
- **1548**：严嵩以"河套"议题构陷前首辅夏言，致夏言弃市。严嵩升任内阁首辅
- **1548–1562**：严嵩父子把持朝政。严嵩以"帝以刚，嵩以柔；帝以骄，嵩以谨；帝以英察，嵩以朴诚"的策略侍奉嘉靖。后期严嵩年老体衰，其子严世蕃实际操控政务，朝中称"大丞相""小丞相"
- **1550**：庚戌之变中严嵩主张"坚壁勿战"，事后将执行其命令的兵部尚书丁汝夔推出斩首替罪
- **1552**：徐阶入阁，韬光养晦，甚至将孙女嫁给严嵩之孙、在江西购置田产与严嵩攀乡谊
- **约1561**：徐阶安排道士蓝道行在扶乩时以"神谕"暗示严嵩为奸臣，动摇嘉靖信任
- **1562**：徐阶授意御史邹应龙弹劾严世蕃贪腐不法。严嵩被迫致仕，严世蕃革职充军
- **1564**：巡江御史林润揭发严世蕃私逃回乡、"乘轩衣蟒"
- **1565**：严世蕃以通倭罪被斩首，严家被抄（抄出黄金3万余两、白银300万两）。严嵩削官还乡，寄食墓舍
- **1567**：严嵩冻饿而死

### 关键人物

| 人物 | 身份 | 角色 |
|------|------|------|
| **朱厚熜** | 皇帝 | 深居西苑修道，但始终保持对朝局的最终控制。利用严嵩代理朝政，在需要时又将其抛弃 |
| **严嵩** | 内阁首辅（1548–1562） | 以柔顺侍君、善写青词（道教祭文）获宠。专权后卖官鬻爵、排斥异己、贪腐巨万 |
| **严世蕃** | 工部侍郎（严嵩之子） | 实际操控者。精明干练但骄横跋扈，收买宦官监视嘉靖起居，为严嵩决策幕后 |
| **夏言** | 前首辅 | 性格刚直自负，被严嵩以"河套"议题构陷，弃市处死。严嵩上位的垫脚石 |
| **仇鸾** | 宣大总兵 | 在狱中与严嵩约为"父子"关系获救，后在庚戌之变中行贿俺答。与严嵩后来反目 |
| **赵文华** | 工部侍郎（严嵩义子） | 严嵩在东南抗倭事务中的代理人，推荐胡宗宪主持军务 |
| **徐阶** | 内阁次辅 → 首辅 | 严嵩的终结者。入阁后隐忍十年，通过联姻、攀乡谊伪装恭顺，最终利用蓝道行、邹应龙等人合力扳倒严嵩 |
| **蓝道行** | 道士 | 徐阶安排的棋子。以扶乩"神谕"向嘉靖暗示严嵩为奸臣 |
| **邹应龙** | 御史 | 接受徐阶提供的材料，正式弹劾严世蕃 |
| **林润** | 巡江御史 | 揭发严世蕃私逃回乡，推动最终清算 |
| **胡宗宪** | 浙直总督 | 严嵩门下但实际专注抗倭。严嵩倒台后受牵连下狱，死于狱中 |

### 矛盾核心分析

**皇帝怠政与代理人治国的结构性矛盾**：嘉靖不上朝但不放权，通过内阁首辅代理日常朝政，自己保留最终裁决权。这种"垂拱而治"的模式必然产生权臣——谁能揣摩圣意、充当皇帝与百官之间的唯一通道，谁就能垄断权力。严嵩恰好具备这种能力（善写青词、性格柔顺、善于揣摩）。

**权力寄生关系**：严嵩的权力完全寄生于皇帝的信任。嘉靖始终是最终决策者——他容忍严嵩贪腐是因为需要一个听话的代理人；当严嵩年老失用时，便毫不留情地抛弃。严嵩倒台不是"正义战胜邪恶"，而是皇帝更换了代理人（从严嵩换成徐阶）。

**权臣更替的零和博弈**：徐阶扳倒严嵩的手段（蓝道行扶乩、邹应龙弹劾、联姻伪装）本质上是一场精心策划的政治阴谋，而非制度化的纠错机制。明代内阁政治中，首辅更替往往是零和博弈——不是你死就是我亡。

[来源：维基百科-严嵩/徐阶；澎湃新闻；百度百科-严嵩/严世蕃；知乎]

## 五、庚戌之变（1550）

### 事件经过

- **背景**：俺答汗多次遣使请求开放朝贡贸易（互市），明廷拒绝并杀来使
- **1550年6月**：俺答率军犯大同，总兵张达、副总兵林椿战死
- 宣大总兵仇鸾以重金贿赂俺答，请其勿攻大同、移攻他处
- **1550年8月14日**：俺答由古北口入关，杀掠怀柔、顺义无数，一路推进至通州东北孤山一带，京师戒严
- 京师兵籍半为虚数，禁军仅四五万、多为老弱，战斗力极差。嘉靖急召各镇勤王
- 严嵩对兵部尚书丁汝夔说："塞上打仗，败了可以掩饰；京郊打仗，败了不可掩饰。俺答不过是掠食贼，饱了自然便去。"丁汝夔遂戒诸将勿轻举
- 俺答放回俘虏宦官杨增，手持书信：**"予我币，通我贡，即解围；不者岁一虔尔郭！"**
- **8月18日**：俺答引兵夺白羊口西走，但因大雨担心伏击折返，最终分两路由高崖口和古北口出边。仇鸾尾随企图邀功，反遭俺答折返伏击，明军溃败死伤千余
- **事后**：严嵩将丁汝夔推出斩首。"诸州县报所残掠人畜二百万"

### 关键人物

| 人物 | 身份 | 角色 |
|------|------|------|
| **俺答汗** | 蒙古土默特部首领、右翼三万户盟主 | 战争发动者。目的是以武力迫使明朝开放互市贸易，而非攻灭明朝 |
| **朱厚熜** | 皇帝 | 深居西苑，急召勤王，事后追究责任 |
| **严嵩** | 内阁首辅 | 主张"坚壁勿战、等敌自退"，事后将执行其命令的丁汝夔推出顶罪 |
| **仇鸾** | 宣大总兵 | 因贿赂严世蕃获任。贿赂俺答移攻他处导致京师直接受威胁；撤退时尾随邀功反被击溃 |
| **丁汝夔** | 兵部尚书 | 听从严嵩建议戒诸将勿战，事后被严嵩当作替罪羊斩首 |
| **张达、林椿** | 大同总兵、副总兵 | 在大同前线战死 |

### 矛盾核心分析

**互市贸易需求与明朝封闭政策的冲突**：俺答汗需要中原的铁器、布帛、粮食等物资来维持对蒙古各部的统治，多次遣使请求互市均被拒绝，甚至使者被杀。明廷拒绝的原因是对"土木之变"的历史恐惧——担心开放贸易会壮大蒙古势力、重演也先入侵。但这种一刀切的封锁政策逼迫俺答以战争手段达成目标。

**军事体制的系统性腐败**：京师禁军兵籍严重虚报（实际战斗力远低于纸面）、将领由行贿而非军功获任（仇鸾贿赂严世蕃）、边防体系形同虚设。庚戌之变暴露的不是某一场战役的失败，而是整个军事体制的溃烂。

**文官避责心态**：严嵩"等敌自退"的策略核心不是军事判断，而是政治避责——京郊战败无法掩饰，不战则不败。事后又牺牲丁汝夔顶罪。这种"不求有功但求无过"的文官心态，在明中后期成为常态。

[来源：维基百科-庚戌之变；百度百科-庚戌之变；知乎]

## 六、嘉靖大地震（1556）

### 事件经过

- **1556年1月23日（嘉靖三十四年腊月十二）夜半**：陕西华州发生约8.0–8.5级大地震
- 有名有姓的死亡记录达**83万人**，未登记者不可计数。为人类有史记载死亡人数最多的地震
- 受灾面积28万平方公里，101个县遭受破坏，波及陕西、山西、河南、甘肃、宁夏等省
- 地形剧变：华阴驻马桥断裂、大荔紫微观干涸、黄河南岸大庆关和蒲州河堤尽数崩塌
- 震后引发大规模瘟疫和次生灾害
- **赈灾**：朝廷直到次年（1557年）二月才派出赈灾官员户部左侍郎邹守愚抵达灾区。先后拨银数万两赈灾、蠲免税粮、停免夏税，并向延绥、宁夏、甘肃、固原调拨军饷银
- 嘉靖下发**罪己诏**

### 关键人物

| 人物 | 身份 | 角色 |
|------|------|------|
| **朱厚熜** | 皇帝 | 下发罪己诏（天人感应传统），批准赈灾拨款 |
| **邹守愚** | 户部左侍郎 | 赈灾主官。抵达后组织地方官共同救灾，从无灾区选调政绩优秀的官员协助 |
| 地方各级官员 | 州县官 | 第一时间上报灾情，但自身也大量伤亡（地震发生在夜半，官民同遭灾难） |

### 矛盾核心分析

**天灾与人祸的叠加放大效应**：83万人死亡的惨重代价不仅是地震本身造成的，更与当时的社会条件有关——关中地区窑洞民居普遍、人口密集、夜半发生时无法逃离、震后医疗和救灾体系几乎不存在。

**中央救灾能力的滞后**：从地震发生（1556年1月）到赈灾官员抵达（1557年2月）长达一年有余，反映了明代中央对地方灾害的响应能力极为有限——交通不便、信息传递缓慢、官僚程序繁琐。

**财政雪上加霜**：地震发生在严嵩专权、北虏南倭并发的时期，国库本已紧张。赈灾拨银导致国库连续两年亏空，税基因人口大量死亡而急剧萎缩，形成恶性循环，进一步削弱明朝应对其他危机的能力。

[来源：维基百科-嘉靖大地震；百度百科-华州大地震；知乎；陕西省社会科学院]

## 七、倭寇之患与东南抗倭（1523–1565）

### 事件经过

- **1523**：**争贡之役**。日本大内氏与细川氏使团在宁波市舶司争贡，酿成武装冲突。明廷随后罢市舶司、严申海禁
- **1540s**：海禁切断正常贸易渠道，走私集团壮大。日本战国浪人南下，与中国沿海走私商人勾结，形成"倭寇"势力（实际以中国人为主体）
- **1548**：浙江巡抚朱纨捣毁走私据点双屿港，但因触动沿海士绅利益集团被弹劾，含冤自杀
- **1553–1555**：倭患达到高峰，威胁杭州、苏州、南京等运河重镇。汪直成为东海最大走私武装集团首领
- **1555**：戚继光调任浙江都指挥使司佥书
- **1556**：胡宗宪升任浙直总督，主持抗倭大局。采取"剿抚并用"策略，招安汪直、招降徐海
- **1557**：汪直被诱降后于杭州处死（胡宗宪承诺赦免但未能兑现）
- **1559**：戚继光在义乌募兵4000余人，编练"戚家军"
- **1561**：**台州大捷**——戚继光转战台州40天，九战九捷，斩俘倭寇4100余人
- **1562–1563**：戚继光援闽，与俞大猷、刘显合力攻克平海卫倭巢，斩倭2200余人
- **1564–1565**：东南沿海倭患基本肃清

### 关键人物

| 人物 | 身份 | 角色 |
|------|------|------|
| **朱纨** | 浙江巡抚 | 早期强硬抗倭派。捣毁双屿港，但因海禁政策触动沿海利益集团被弹劾，含冤自杀。其失败暴露了海禁政策的内在矛盾 |
| **胡宗宪** | 浙直总督 | 抗倭总指挥。虽为严嵩门下（赵文华推荐），但实际专注军务。采取"剿抚并用"策略，招安汪直、招降徐海。严嵩倒台后受牵连下狱，死于狱中 |
| **戚继光** | 参将 → 总兵 | 抗倭核心战将。在义乌编练"戚家军"，发明"鸳鸯阵"战术，创造台州九战九捷的战绩。后调往蓟镇守边 |
| **俞大猷** | 总兵 | 与戚继光齐名的抗倭名将。擅长水战和骑战，在福建、广东多次击败倭寇 |
| **刘显** | 广东总兵 | 与戚继光、俞大猷合力在福建平海卫歼灭倭寇主力 |
| **汪直**（王直） | 海商 / 走私集团首领 | 嘉靖倭寇的核心人物之一。自称"净海王""徽王"，统一东海走私势力。被胡宗宪诱降后处死，其部众失去约束，倭患一度加剧 |
| **徐海** | 倭寇头领（汪直旧部） | 独立自成一股势力，后被胡宗宪招降后杀之 |
| **赵文华** | 工部侍郎（严嵩义子） | 严嵩在东南的代理人，推荐胡宗宪。改变夏言时期的强硬海禁手段，采用剿抚并用 |
| **张经** | 总督 | 早期抗倭主将。取得王江泾大捷，但因赵文华构陷被冤杀 |

### 矛盾核心分析

**海禁政策与民间贸易需求的根本对立**：这是嘉靖倭患的根源。明初海禁的目的是防止沿海居民与海外势力勾结，但随着全球贸易体系的形成（大航海时代），东南沿海的贸易需求急剧增长。海禁将正常贸易逼入走私，走私又催生了武装自卫，最终演变为"倭寇"。所谓"倭寇"实际上中国人占多数（"真倭"比例很低），本质是海禁政策制造的走私武装。

**沿海利益集团与海禁执行者的冲突**：朱纨的悲剧揭示了这一矛盾——沿海士绅、官僚本身就是走私贸易的受益者和参与者，他们在朝中有强大的代言人。任何认真执行海禁的官员都会触动这个利益网络而遭反噬。

**"剿"与"抚"的策略争论**：抗倭过程中始终存在"以剿为主"还是"以抚为主"的路线之争。汪直被诱降后杀之，引发信用危机，部众四散为寇，反而加剧倭患——说明单纯的军事手段无法根除一个由经济需求驱动的问题。最终倭患的解决，军事清剿只是一半，另一半要等到隆庆开关（开放海禁）才真正釜底抽薪。

[来源：百度百科-倭寇/台州之战/胡宗宪；维基百科；澎湃新闻-樊树志；浙江工商大学东亚研究院]

## 八、嘉靖帝驾崩（1567）

### 事件经过

- **1567年1月23日**：嘉靖帝在西苑万寿宫驾崩，享年59岁
- 长期服食丹药（含汞、铅等重金属），晚年频繁失眠、情绪暴躁，1566年11月后已无法批阅奏章
- 在位45年，庙号世宗
- 皇三子裕王朱载坖继位，改元隆庆

### 关键人物

| 人物 | 身份 | 角色 |
|------|------|------|
| **朱厚熜** | 皇帝 | 前期英主、后期昏君的典型。死因可能与丹药汞中毒有关 |
| **朱载坖**（隆庆帝） | 裕王 → 皇帝 | 继位后改弦更张，起用高拱、张居正 |
| **徐阶** | 内阁首辅 | 主持权力交接，拟定嘉靖遗诏，为隆庆新政奠定基调 |

### 矛盾核心分析

**嘉靖之死的历史评价悖论**：嘉靖帝是一个极具矛盾的历史人物——前期展现了卓越的政治才能（大礼议中以15岁之龄击败整个文官集团），中期改革成效显著，但后半生却沉迷修道、虐待宫女、放任权臣。这种转变的根源在于：明代皇帝制度赋予了个人过大的权力，而没有任何制度化的制约机制——当这个个人的兴趣和精力发生转移时，整个帝国都随之转向。

**继承危机的余波**：嘉靖长期不立太子，裕王朱载坖与景王朱载圳"二王并存"的局面延续多年，严嵩父子甚至更看好景王（景王封地在湖广，距京更近），形成暗流涌动的继承政治。直到景王去世（1565），继承问题才算尘埃落定。

[来源：Wikipedia-Jiajing Emperor；Britannica-Jiajing；维基百科-嘉靖]
