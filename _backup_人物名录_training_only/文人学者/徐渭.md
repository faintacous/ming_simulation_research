# 徐渭

## 基本信息

| 维度 | 内容 |
|------|------|
| 姓名 | 徐渭，初字文清，改字文长，号天池山人、青藤道士、田水月、白鹇山人、山阴布衣 |
| 生卒 | 明正德十六年（1521）— 万历二十一年（1593），享年七十三岁 |
| 籍贯 | 浙江绍兴府山阴县（今浙江绍兴） |
| 身份 | 生员（秀才），屡试不中，终未入仕；曾为胡宗宪幕僚，任军事参谋 |
| 擅长 | 绘画（大写意花鸟）、书法（行草）、戏曲（杂剧《四声猿》）、诗文、军事谋略 |
| 谥号 | 无（布衣终身） |

## 生平概要

徐渭生于正德十六年（1521），出身山阴一个走向衰落的地方绅宦家庭。其父徐鏓曾任夔州府同知，但在徐渭出生百日后即去世。徐渭为庶出，生母苗氏为侧室，自幼在嫡母苗宜人的抚养下成长，家庭关系复杂，少年时代即感受到嫡庶之分带来的压力。

徐渭少年聪颖，六岁读书，九岁能作文，有"神童"之誉。然科举之路极不顺利：嘉靖十六年（1537）十七岁考取秀才后，此后八次参加乡试均告落第，终其一生仅为生员。嘉靖二十年（1541）入赘潘家，妻潘似早逝。后又两次入赘或续娶，家庭生活多波折。

嘉靖三十三年（1554）前后，东南沿海倭患日烈。嘉靖三十四年（1555），徐渭以生员身份参与绍兴城防，展现出军事才能。嘉靖三十六年（1557）左右，经人推荐入浙直总督胡宗宪幕府，正式担任军事参谋。在幕期间，徐渭参与谋划招降海盗汪直、剿灭徐海等重大军事行动，深受胡宗宪倚重。胡宗宪对徐渭礼遇有加，允许其不拘官场礼节，徐渭亦自视甚高，常穿布衣往来幕府。

嘉靖四十一年（1562），胡宗宪因严党牵连被捕下狱，幕府散去。徐渭失去庇护，精神状况急剧恶化。嘉靖四十四年（1565），胡宗宪瘐死狱中，徐渭悲惧交加，精神彻底崩溃，先后九次自杀未遂——以锥刺耳、以锤击肾、以柱击首，手段极端。嘉靖四十五年（1566），徐渭在精神错乱中误杀继室张氏，被捕入狱，论死。

入狱后，徐渭得友人张元忭、诸大绶等人多方营救，在狱中度过七年。万历元年（1573），逢万历帝登极大赦，徐渭获释，时年五十三岁。此后他游历北方，曾至宣化、辽东等地，短暂客居好友吴兑幕中，但终因性格狂狷不容于世而南归。晚年在山阴以鬻书画为生，贫病潦倒。万历二十一年（1593），徐渭在极度的孤独与贫困中去世，身边仅有一犬相伴，死时"至籍藁以葬"。

徐渭一生横跨嘉靖、隆庆、万历三朝，在绘画、书法、戏曲、诗文、军事五个领域均有杰出成就。其大写意花鸟画风开一代之先，深刻影响了后来的石涛、八大山人、扬州八怪乃至齐白石。其杂剧《四声猿》被誉为明代戏曲杰作。郑板桥曾刻印"青藤门下走狗"，齐白石称"恨不生三百年前，为青藤磨墨理纸"，足见其艺术影响之深远。

## 关键事迹

1. **幼年失怙，嫡庶困境** — 生百日而父亡，庶出身份使徐渭自幼敏感孤傲。嫡母苗宜人抚养其长大，对其影响极深 [来源：《徐文长三集》卷二十六《嫡母苗宜人墓志铭》]

2. **八试不中，终身布衣** — 自嘉靖十六年（1537）考取秀才后，八次参加浙江乡试均落榜。其文才为时所重，却屡不得志于科场，成为其一生的核心创伤 [来源：《明史》卷二百八十八《文苑传》]

3. **入胡宗宪幕，参与抗倭谋略** — 嘉靖三十六年（1557）前后入胡宗宪幕府，被委以军机要务。胡宗宪"宾之者甚至"，徐渭"亦自负，尝衣白布衫出入辕门，无忌惮" [来源：《明史》卷二百八十八；袁宏道《徐文长传》]

4. **参与招降汪直之谋** — 嘉靖三十七年（1558），胡宗宪以招抚策略诱降大海盗汪直（王直），徐渭参与谋划。汪直最终被擒杀，此役为嘉靖年间东南抗倭之关键转折 [来源：《明史》卷二百五《胡宗宪传》；汪直事件多见于《筹海图编》]

5. **参与剿灭徐海之役** — 嘉靖三十五年（1556）至三十六年间，胡宗宪施反间计瓦解海盗徐海集团，徐海兵败自杀。徐渭在幕后献策，其具体谋略多见于后世记述 [来源：《殊域周咨录》；《胡宗宪传》]

6. **代拟《白鹿表》** — 胡宗宪获白鹿，命徐渭代拟进表。文章辞藻华美，为嘉靖帝所喜，胡宗宪因此得宠。此为徐渭在幕府中文事之代表作 [来源：袁宏道《徐文长传》]

7. **胡宗宪败，精神崩溃** — 嘉靖四十一年（1562）胡宗宪被逮，四十四年（1565）瘐死狱中。徐渭"虑祸及己，遂发狂"，先后以锥刺耳、以铁钉贯耳、以锤击肾、以头撞壁，九次自杀未遂 [来源：《明史》卷二百八十八《文苑传》]

8. **杀妻入狱，论死系囚** — 嘉靖四十五年（1566），徐渭在狂疾发作中击杀继室张氏，被论死罪入狱。关于杀妻原因，一说疑妻不贞，一说纯粹因精神错乱，至今众说纷纭 [来源：《明史》卷二百八十八；《山阴县志》]

9. **友人力救，万历赦出** — 入狱期间，翰林编修张元忭（徐渭早年受业于其父张天复）、翰林院修撰诸大绶等友人全力营救，最终逢万历改元大赦，于万历元年（1573）获释 [来源：《徐文长三集》附录；张元忭《不二斋文选》相关书信]

10. **创作《四声猿》杂剧** — 在入狱前后完成杂剧集《四声猿》，包含《狂鼓史渔阳三弄》《玉禅师翠乡一梦》《雌木兰替父从军》《女状元辞凰得凤》四部短剧，以古讽今、借题发挥，被誉为明代杂剧压卷之作 [来源：徐渭《四声猿》自序；《曲海总目提要》]

11. **北游宣化辽东** — 出狱后约万历四年至八年（1576–1580）间，徐渭游历北方，先后客居宣大总督吴兑幕、至辽东李成梁处短暂逗留。此行留下大量诗文，反映其对边塞军事的持续关注 [来源：《徐文长逸稿》卷二北游诗]

12. **晚年鬻画，潦倒而终** — 归乡后以卖书画为生，穷困潦倒。据陶望龄记载，徐渭晚年"有书数千卷，后斥卖殆尽。帱莞破弊，不能再易，至藉藁以卧"。万历二十一年（1593）卒，年七十三 [来源：陶望龄《徐文长传》；《绍兴府志》]

## 重要人际关系

### 君主关系
- **嘉靖帝（朱厚熜）** — 间接关系。胡宗宪进白鹿表为嘉靖帝所喜，表文出自徐渭手笔。徐渭本人从未获得与皇帝直接接触的机会。

### 盟友与庇护者
- **胡宗宪** — 东南抗倭统帅，徐渭最重要的庇护者。对徐渭以国士相待，徐渭亦以死报之。胡宗宪败后徐渭的精神崩溃，可见两人关系之深。
- **吴兑** — 字君泽，宣大总督，徐渭北游时的重要依靠，为旧友。
- **张元忭** — 翰林编修，其父张天复曾为徐渭老师。张元忭是营救徐渭出狱的关键人物。但二人后因性格冲突疏远。
- **诸大绶** — 翰林院修撰，山阴同乡，参与营救徐渭。
- **沈錬** — 同为山阴人，刚直敢言之臣，与徐渭有交谊，后被严嵩迫害致死。徐渭作《短褐篇送沈王子问余学》等诗文悼念。

### 文人交游
- **袁宏道** — 晚明文坛领袖，万历二十五年（1597）游绍兴时发现徐渭遗作，作《徐文长传》，使徐渭文名大振于死后。此为"公安派"推崇徐渭之始。
- **陶望龄** — 绍兴文人，与徐渭有交游，撰《徐文长传》，是最早的徐渭传记之一。
- **王畿** — 王阳明弟子，山阴人，徐渭曾受其心学影响。

### 家族
- **父：徐鏓** — 字克平，夔州府同知，在徐渭百日时去世。
- **嫡母：苗宜人** — 抚养徐渭长大，对徐渭性格塑造影响极大。
- **生母：苗氏** — 侧室，后被遣出，徐渭对此深感痛苦。
- **兄：徐潞** — 嫡出，与徐渭关系疏远。
- **妻：潘似** — 第一任妻子，入赘潘家，潘似约在婚后数年病逝，徐渭甚为哀痛。
- **继室：张氏** — 为徐渭所杀。关于其是否有不贞行为，史料无定论。
- **子：徐枚、徐枳** — 徐渭有两子，晚景中父子关系不佳。

### 政敌（间接）
- **严嵩、严党** — 胡宗宪被视为严党，严嵩倒台后胡宗宪受牵连，徐渭亦受波及。徐渭与严嵩并无直接冲突，但其命运被严党政治深刻改变。

## 历史评价

### 时人评价

- **胡宗宪** — 对徐渭极为器重，"公（胡宗宪）督数边兵，威振东南，介胄之士膝语蛇行，不敢举足。而文长以部下一诸生傲之，信心而行，恣意谭谑，了无忌惮。公深怜其才，时出重赏以购其文。"（袁宏道《徐文长传》转述）
- **袁宏道** — "古今文人，牢骚困苦，未有若先生者也……先生诗文崛起，一扫近代芜秽之习，百世而下，自有定论。"（《徐文长传》，约作于1597年）
- **陶望龄** — "渭行奇，遇奇，文奇，画奇，书奇，而诗最奇。"（《徐文长传》）

### 后世评价

- **郑板桥** — 刻有"青藤门下走狗"印章，以表对徐渭画艺的崇拜。
- **齐白石** — "青藤、雪个、大涤子之画，能纵横涂抹，余心极服之。恨不生三百年前，为青藤磨墨理纸。"（齐白石题画语）
- **现代美术史** — 徐渭被视为中国大写意花鸟画的真正开创者，将水墨写意推向纵恣狂放的新境界，与陈淳并称"青藤白阳"。
- **戏曲史地位** — 《四声猿》被公认为明代杂剧的压卷之作，在体制上突破了元杂剧的框范，对汤显祖等后辈有直接影响。
- **《明史》定评** — "渭天才绝伦，诗文奇肆，善行草，工画花草竹石。尝自言：'吾书第一，诗二，文三，画四。'识者以为然。"（《明史》卷二百八十八）

## 存疑 & 待查

1. **杀妻真相** — 徐渭杀继室张氏的具体原因，究竟是因疑其不贞、精神错乱中的无意识行为，还是另有隐情，史料记载相互矛盾。张氏是否有不贞行为，无确证。
2. **自杀次数** — 九次自杀之说见于多种传记，但具体次数和顺序各家记载不一，可能有夸大成分。
3. **幕府中具体献策内容** — 徐渭在胡宗宪幕中的军事谋略，多为他自己和后人追述，缺乏独立史料佐证。其对抗倭战事的实际贡献程度难以精确评估。
4. **精神疾病性质** — 徐渭的精神崩溃究竟属于现代医学意义上的何种疾病（抑郁症、精神分裂、躁郁症等），各家学者诊断不一，无法定论。
5. **与汤显祖的关系** — 徐渭《四声猿》与汤显祖"临川四梦"在戏曲史上的先后影响关系，二人是否有过直接或间接的交流，尚待考证。
6. **画作真迹鉴定** — 徐渭传世画作中赝品极多，确定真迹的标准至今存在争议。主要收藏机构包括故宫博物院、南京博物院、上海博物馆等。
7. **入狱确切时间** — 有嘉靖四十四年（1565）和四十五年（1566）两说，需进一步考证。
8. **北游具体行程** — 徐渭出狱后北游的完整路线、时间节点，现有史料无法完全复原。

## 来源汇总

1. 《明史》卷二百八十八《文苑传·徐渭》 — 正史 — 高可信度
2. 袁宏道《徐文长传》 — 明人传记，约作于万历二十五年（1597） — 高可信度，系发现徐渭之关键文献
3. 陶望龄《徐文长传》 — 明人传记，与袁宏道传并称两大徐渭传记 — 高可信度
4. 《徐文长三集》 — 徐渭本人诗文集，明刻本 — 一手文献，最高可信度
5. 《徐文长逸稿》 — 徐渭遗稿汇编 — 一手文献
6. 《四声猿》 — 徐渭杂剧作品 — 一手文献
7. 《明史》卷二百五《胡宗宪传》 — 正史，涉及徐渭参与的抗倭事迹 — 高可信度
8. 《筹海图编》 — 郑若曾撰，嘉靖年间抗倭军事文献 — 一手史料，高可信度
9. 《殊域周咨录》 — 严从简撰，涉及徐海、汪直事件 — 较高可信度
10. 张元忭《不二斋文选》 — 含营救徐渭相关书信 — 一手文献
11. 《山阴县志》《绍兴府志》 — 地方志 — 较高可信度
12. 《曲海总目提要》 — 戏曲目录学文献 — 较高可信度
