# 明代佛教（1521-1644）

## 一、僧录司体系

### 1.1 中央机构：僧录司

僧录司隶属于礼部，是明代管理全国佛教事务的最高行政机构。洪武十五年（1382年）正式设立。

**中央八个官职及品级：**

| 官职 | 员额 | 品级 |
|------|------|------|
| 左善世 | 1人 | 正六品 |
| 右善世 | 1人 | 正六品 |
| 左阐教 | 1人 | 从六品 |
| 右阐教 | 1人 | 从六品 |
| 左讲经 | 1人 | 正八品 |
| 右讲经 | 1人 | 正八品 |
| 左觉义 | 1人 | 从八品 |
| 右觉义 | 1人 | 从八品 |

合称僧录司"八座"。善世为最高僧官，主管全国佛教事务；阐教协助善世处理教务；讲经主管经典讲解；觉义主管教义阐释。

> 来源：[明朝僧、道录司架构及历史 - 知乎](https://zhuanlan.zhihu.com/p/721727297)；[明朝官职表 - 维基百科](https://zh.wikipedia.org/zh-hans/%E6%98%8E%E6%9C%9D%E5%AE%98%E8%81%B7%E8%A1%A8)

### 1.2 地方三级机构

| 层级 | 机构名称 | 主官 | 品级 |
|------|----------|------|------|
| 府 | 僧纲司 | 都纲 | 从九品上 |
| 州 | 僧正司 | 僧正 | 未入流 |
| 县 | 僧会司 | 僧会 | 未入流 |

地方僧官品级极低，"职低权轻"，主要负责本辖区僧尼的登记、考核与管理。僧纲司都纲下设副都纲一员，同为从九品。

> 来源：[僧官制度 - 百度百科](https://baike.baidu.com/item/%E5%83%A7%E5%AE%98%E5%88%B6%E5%BA%A6/2266726)

### 1.3 僧官人数变化

- 南京僧录司名额至成化年间只有十名
- 经嘉靖裁革，至万历年间僧录司仅存四名僧官
- 反映了嘉靖朝崇道抑佛政策对佛教行政体系的冲击

> 来源：[明代的僧官制度 - 参考网](https://m.fx361.com/news/2017/0407/1478308.html)

## 二、度牒制度

### 2.1 基本规定

度牒是封建国家承认并确定合格出家者为僧的资格和身份证明，持有度牒的僧尼可免除赋税和劳役。

**核心规定：**

| 项目 | 内容 |
|------|------|
| 发放周期 | 三年一度（定期给牒） |
| 考试要求 | 须赴京考试，通过后方发给 |
| 年龄限制 | 男子须满14岁（洪武初）/40岁（后期），女子须满40岁（洪武初）/50岁（后期） |
| 名额限制 | 府不过40人，州不过30人，县不过20人 |

> 来源：[度牒 - 维基百科](https://zh.wikipedia.org/zh-hans/%E5%BA%A6%E7%89%92)；[明代的僧官制度 - 参考网](https://m.fx361.com/news/2017/0407/1478308.html)

### 2.2 度牒价格变迁

| 时间 | 方式 | 价格/条件 |
|------|------|-----------|
| 洪武朝 | 免费发放 | 通过考试即可 |
| 景泰二年（1451） | 开始收费 | 纳米五石 |
| 嘉靖十八年（1539） | 纳银给牒 | 白银十两 |
| 嘉靖三十七年（1558） | 降价 | 白银四两 |
| 隆庆年间（1567-1572） | 再调整 | 白银五两 |

景泰以后，鬻牒（卖度牒）逐渐成为朝廷筹集军费和赈灾款项的手段，度牒制度的宗教管理功能日益弱化。

> 来源：[度牒 - 维基百科](https://zh.wikipedia.org/zh-hans/%E5%BA%A6%E7%89%92)；["度牒"经济因何兴盛千年 - 光明网](https://epaper.gmw.cn/wzb/html/2021-01/16/nw.D110000wzb_20210116_3-05.htm)

### 3.1 各朝数据

| 时期 | 僧尼人数 | 备注 |
|------|----------|------|

| 永乐十六年（1418）额定 | 37,090余人 | 全国僧人定额 |

### 3.2 数据分析

- 成化至弘治年间，度牒泛滥，累计度僧逾50万
- 嘉靖朝虽然崇道抑佛，但卖牒制度使实际僧尼人数难以控制
- 明末四大高僧时期（万历至崇祯），佛教再度复兴

## 四、寺院三类分类

### 4.1 洪武三年（1370）分类制度

明太祖将全国佛教寺院分为三类，要求"僧众分别专业"：

| 类别 | 名称 | 主要功能 | 对应宗派 |
|------|------|----------|----------|
| 禅寺 | 禅林/禅院 | 专修禅定，参禅悟道 | 禅宗（临济、曹洞） |
| 讲寺 | 教院/讲院 | 宣讲佛教经典教义 | 天台、华严、法相等 |
| 教寺 | 瑜伽院/应赴院 | 祈福弥灾、追荐亡灵等法事 | 瑜伽僧/赴应僧 |

### 4.2 各类寺院特点

- **禅寺**：僧人以坐禅修行为主，有严格的清规戒律，是最受士大夫推崇的类型
- **讲寺**：僧人专职讲经说法，类似佛教学府，研习经论
- **教寺**：面向世俗社会服务，承接丧葬超度、祈雨消灾等法事活动

教僧占到整个僧侣总数的将近半数，反映了明代佛教日益世俗化的趋势。"教寺"的单列标志着明清以降佛教的社会功能从"治心"变为"荐亡"。

> 来源：[明代佛教 - 百度百科](https://baike.baidu.com/item/%E6%98%8E%E4%BB%A3%E4%BD%9B%E6%95%99/7925594)

## 五、各宗派情况

### 5.1 禅宗

- **临济宗**：明代禅宗中最为盛行的一支，法脉传承不断
- **曹洞宗**：势力次于临济宗，但在南方有较大影响
- 禅宗整体在明代中期一度衰微，至明末四大高僧时期才有复兴

### 5.2 净土宗

- 逐渐成为各宗共同信仰，"禅净双修"成为明代佛教主流
- 莲池袾宏专志净业，被尊为净土宗第八代祖师
- 蕅益智旭被尊为净土宗第九代祖师

### 5.3 其他宗派

- **华严宗**：逐步恢复发展，但未达到唐代盛况
- **天台宗**：在浙江一带仍有传承，传灯等僧人有所弘扬
- **律宗**：明末有复兴迹象，蕅益智旭曾致力于戒律研究
- **法相宗（唯识）**：较为沉寂
- **密宗**：主要在藏传佛教系统中保存，汉地几乎断绝

### 5.4 明代佛教的总体特征

- "三教合一"思潮盛行，儒释道相互融合
- 禅净双修成为普遍修行方式
- 居士佛教兴起，大量士大夫参与佛学研究
- 佛教世俗化，法事活动成为主要收入来源

## 六、明末四大高僧

### 6.1 莲池袾宏（1535-1615）

| 项目 | 内容 |
|------|------|
| 俗姓 | 沈 |
| 字号 | 字佛慧，别号莲池 |
| 籍贯 | 浙江杭州府仁和县 |
| 出家年龄 | 32岁 |
| 常居道场 | 杭州云栖寺（又称云栖蓮池） |
| 宗派归属 | 净土宗第八代祖师 |
| 主要主张 | 禅净兼修，倡导念佛法门 |
| 代表著作 | 《阿弥陀经疏钞》《竹窗随笔》《戒杀放生文》 |
| 圆寂 | 万历四十三年（1615年），世寿81岁 |

莲池大师受具足戒后云游参学，后居杭州云栖寺，大兴念佛之风。他主张禅净双修，认为参禅与念佛殊途同归，对明末佛教复兴起到了关键作用。

### 6.2 紫柏真可（1543-1603）

| 项目 | 内容 |
|------|------|
| 俗姓 | 沈 |
| 法号 | 真可，号紫柏（原名达观） |
| 籍贯 | 南直隶苏州府吴县 |
| 出家经历 | 17岁时仗剑远游，遇虎丘僧明觉而出家 |
| 主要主张 | 立志复兴禅宗，主张儒释道三教调和 |
| 重大事业 | 主持编刻《嘉兴藏》（又称《径山藏》），是中国佛教史上重要的大藏经刻本 |
| 蒙难经过 | 万历三十一年（1603年），因妖书案受牵连，遭东厂逮捕 |
| 圆寂 | 万历三十一年十二月十七日，于狱中沐浴端坐而逝，世寿61岁 |

紫柏大师性格刚烈，一生行脚四方，足迹遍及大江南北。其主持编刻的《嘉兴藏》是中国第一部用方册本而非经折本印行的大藏经，对佛教经典的传播具有划时代意义。

### 6.3 憨山德清（1546-1623）

| 项目 | 内容 |
|------|------|
| 俗姓 | 蔡 |
| 法号 | 德清，字澄印，号憨山 |
| 谥号 | 弘觉 |
| 籍贯 | 南直隶滁州府全椒县（今安徽省） |
| 宗派 | 临济宗 |
| 主要主张 | 精通释、道、儒三家，主张三家思想融合；倡导禅净双修 |
| 蒙难经过 | 万历二十三年（1595年）因私创寺院之罪被充军广东雷州 |
| 重要贡献 | 赴广东曹溪南华寺复兴祖庭，重修六祖惠能真身 |
| 代表著作 | 《憨山老人梦游集》《观楞伽经记》 |
| 圆寂 | 天启三年（1623年）十月十一日于南华寺圆寂，寿78岁 |

憨山德清是明末最具传奇色彩的高僧之一。他曾在五台山修行多年，后遭贬谪至广东，反而促成了曹溪南华寺的复兴，为禅宗祖庭的传承做出了重要贡献。

### 6.4 蕅益智旭（1599-1655）

| 项目 | 内容 |
|------|------|
| 俗姓 | 钟 |
| 法号 | 智旭，号蕅益 |
| 籍贯 | 南直隶苏州府吴县 |
| 出家经历 | 天启二年（1622年），因仰慕憨山大师，从其弟子雪岭出家，时24岁 |
| 宗派归属 | 被尊为净土宗第九代祖师 |
| 晚年驻锡 | 浙江余杭灵峰（故又称灵峰大师） |
| 主要主张 | 融合天台、禅、净、律各宗；注重戒律研究 |
| 代表著作 | 《阅藏知津》《灵峰宗论》《弥陀要解》 |
| 圆寂 | 顺治十二年（1655年），世寿57岁 |

蕅益智旭少年时期好儒学，曾著书批评佛教。后读莲池大师著作而回心转意，皈依佛教。他学问渊博，著述宏富，对天台、唯识、律学、净土均有深入研究，被后人誉为"通宗通教"的大家。

## 七、嘉靖帝抑佛政策

### 7.1 政策背景

明世宗朱厚熜（嘉靖帝，1521-1566在位）是明代最崇信道教的皇帝，其对佛教采取了明显的抑制政策，与其崇道立场形成鲜明对照。

### 7.2 主要抑佛措施

1. **裁减僧官**：大幅削减僧录司编制，从成化年间的十名削减到万历年间仅剩四名
2. **限制度牒**：严格控制新发度牒数量，同时将度牒变为财政工具（纳银给牒）
3. **拆毁寺院**：部分地方执行拆毁佛寺、驱逐僧尼的政策
4. **抬高道教地位**：在朝廷仪式中全面提升道教规格，压缩佛教活动空间
5. **经济压制**：减少对佛教寺院的皇家赏赐和经济支持

### 7.3 政策影响

- 嘉靖至万历间，佛教一度处于低潮期
- 禅宗法脉虽未断绝，但高僧人才匮乏
- 净土宗在民间反而因简便易行而持续传播
- 直到万历后期，四大高僧相继弘法，佛教才迎来明末复兴

### 7.4 明末佛教复兴

嘉靖帝的抑佛政策反而促使佛教界进行自我反思和改革：
- 莲池袾宏整顿净土宗门规
- 紫柏真可推动大藏经刻印
- 憨山德清复兴禅宗祖庭
- 蕅益智旭融通各宗，建立新的教学体系
