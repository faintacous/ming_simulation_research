# 明代祭祀制度（1521-1644）

## 一、大祀/中祀/群祀（小祀）三等分类

### 1.1 制度概述

明代国家祭祀沿袭周礼传统，将祭祀活动分为大祀、中祀、群祀（小祀）三个等级。祭祀等级决定了祭品规格、参与人员、仪式繁简以及是否由皇帝亲祭。

### 1.2 大祀（13项）

大祀是国家最高等级的祭祀，由皇帝亲自主持（或遣亲王代祭），仪式最为隆重。

| 2 | 大雩（祈雨） | 孟夏（四月） | 昊天上帝 | 圜丘 |
| 3 | 大享（秋报） | 季秋（九月） | 昊天上帝 | 圜丘 |

### 1.5 三等祭祀的规格差异

| 比较维度 | 大祀 | 中祀 | 小祀 |
|----------|------|------|------|
| 主祭者 | 皇帝亲祭 | 遣官代祭为主 | 遣官或有司代祭 |
| 斋戒 | 散斋四日，致斋三日 | 散斋三日，致斋二日 | 斋戒一至二日 |
| 祭品 | 太牢（牛羊猪全备）＋玉帛 | 少牢（羊猪）＋帛 | 帛酒果品 |
| 乐舞 | 全套雅乐佾舞 | 简化乐舞 | 无乐舞或简乐 |
| 陪祭 | 王公百官 | 部分官员 | 少数官员 |

## 二、嘉靖天地分祀改革

### 2.1 改革背景

| 项目 | 内容 |
|------|------|
| 改革前制度 | 永乐十八年（1420年）建大祀殿（天地坛），天地合祀 |
| 合祀依据 | 仿南京规制，拆元代祭天圜丘，新建合祀殿 |
| 嘉靖主张 | "天地分祀"更符合古礼 |
| 改革动机 | 嘉靖帝以"大礼议"确立自身权威后，推动全面礼制改革 |
| 朝臣争议 | 许多大臣反对分祀，认为合祀已是祖制 |
| 改革时间 | 嘉靖九年（1530年） |

### 2.2 五坛格局

嘉靖九年改革后，形成了以紫禁城为中心的"五坛"格局：

| 坛名 | 方位 | 祭祀对象 | 祭祀时间 | 建造时间 |
|------|------|----------|----------|----------|
| **天坛（圜丘）** | 南郊偏东 | 昊天上帝 | 冬至 | 嘉靖九年（1530）改建 |
| **地坛（方泽）** | 北郊 | 皇地祇 | 夏至 | 嘉靖九年（1530）新建 |
| **日坛（朝日坛）** | 东郊 | 大明之神 | 春分 | 嘉靖九年（1530）新建 |
| **月坛（夕月坛）** | 西郊 | 夜明之神 | 秋分 | 嘉靖九年（1530）新建 |
| **先农坛** | 南郊偏西 | 先农之神 | 仲春 | 永乐十八年始建 |

### 2.3 天坛改建细节

| 项目 | 改建前 | 改建后 |
|------|--------|--------|
| 名称 | 大祀殿（天地坛） | 圜丘（天坛） |
| 祭祀方式 | 天地合祀 | 仅祭天 |
| 建筑形制 | 殿宇式 | 露天圆形坛台＋祈年殿 |
| 方位定位 | 原址 | 中轴线南延，以"真北"（北斗定位）确定方位 |
| 祭天时间 | 正月合祀 | 冬至专祀天（圜丘）；正月祈谷（祈年殿） |

### 2.4 地坛新建细节

| 项目 | 内容 |
|------|------|
| 位置 | 北京安定门外 |
| 坛台形制 | 方形二层坛台（象征"天圆地方"中的"地方"） |
| 坛面 | 黄色琉璃砖 |
| 方位 | 北偏西约2度 |
| 祭祀时间 | 夏至日 |

### 2.5 改革的深层意义

- 嘉靖帝借分祀改革确立自己在礼制上的权威
- 与"大礼议"一脉相承，都是以古礼否定近世之制
- 形成的五坛格局一直沿用至清末，成为北京城市布局的重要组成部分
- "天地分祀出世宗意"——《明史》明确指出这是嘉靖帝个人的意志

> 来源：[天坛 - 维基百科](https://zh.wikipedia.org/zh-hans/%E5%A4%A9%E5%9D%9B)；[老北京的"五坛" - 北京旅游网](https://www.visitbeijing.com.cn/article/47QrtuflD6U)；[嘉靖朝郊祀礼改革 - 明史研究](https://ming.cnhis.ncnu.edu.tw/wp-content/uploads/2015/10/report2006HeSY.pdf)；[五坛八庙 - 首都之窗](https://www.beijing.gov.cn/renwen/zt/sjc/jz/wtbm1/)

## 三、太庙祭祖制度

### 3.1 太庙概况

| 项目 | 内容 |
|------|------|
| 位置 | 紫禁城东南侧（今北京劳动人民文化宫） |
| 始建 | 永乐十八年（1420年） |
| 初始形制 | 一帝一庙（独立殿宇制） |
| 嘉靖改制 | 嘉靖二十年（1541年）太庙火灾后，嘉靖二十四年（1545年）恢复"同堂异室"制 |
| 建筑规模 | 享殿面阔十一间，比太和殿还高大，体现"位尊权高但不可压祖" |

### 3.2 三种祭祖仪式

#### 时享

| 项目 | 内容 |
|------|------|
| 频次 | 每年四次 |
| 时间 | 春（正月）、夏（四月）、秋（七月）、冬（十月）各孟月 |
| 意义 | 按季节向祖先供奉新鲜祭品，又称"四时之享" |
| 主祭 | 皇帝亲祭 |
| 祭品 | 应季新物（如春季新菜、夏季新果等）＋太牢（牛羊猪） |
| 仪式要点 | 将祖先牌位从寝殿移至享殿神座安放，行三献礼 |

#### 祫祭

| 项目 | 内容 |
|------|------|
| 频次 | 每年一次 |
| 时间 | 除夕前一日 |
| 意义 | 年末大祭，规模最大、最隆重的祭祖仪式 |
| 特点 | 合祭所有祖先，将太庙后殿（寝殿）和两庑中的所有神主牌位移至前殿（享殿）集中祭祀 |
| 乐舞 | 钟鼓齐鸣、韶乐悠扬、佾舞翩迁 |

#### 告祭

| 项目 | 内容 |
|------|------|
| 频次 | 不固定，视需要而定 |
| 时间 | 国家大事发生时 |
| 适用情形 | 登极、亲政、册立皇后/太子、征战出师/凯旋、献俘、诛罪人、重大灾变等 |
| 意义 | 向祖先"报告"国家大事，请示或感恩 |
| 特点 | 仪式简于时享和祫祭，但政治意义重大 |

### 3.3 太庙神主排列

嘉靖二十四年（1545年）重建后的排列：
- 中位：太祖朱元璋（居正中，永不祧迁）
- 左一：成祖朱棣
- 左二：宣宗
- 左三：宪宗
- 左四：睿宗（嘉靖帝之父，通过"大礼议"入庙）
- 右一：仁宗
- 右二：英宗
- 右三：孝宗
- 右四：武宗

### 3.4 祭祀流程概要

1. **斋戒**：皇帝提前斋戒三日（散斋二日＋致斋一日）
2. **省牲**：检查祭牲是否合格
3. **陈设**：在享殿内设木制金漆神座，摆放笾豆案等祭器
4. **迎神**：将寝殿中的祖先牌位移至享殿
5. **初献**：皇帝亲献爵（酒杯），行跪拜礼
6. **亚献**：第二次进酒
7. **终献**：第三次进酒
8. **饮福受胙**：皇帝饮福酒、受祭肉
9. **撤馔**：撤去祭品
10. **送神**：将牌位送回寝殿
11. **望燎**：焚烧祝帛等

## 四、社稷坛

### 4.1 基本信息

| 项目 | 内容 |
|------|------|
| 位置 | 紫禁城西南侧（今北京中山公园内） |
| 始建 | 永乐十八年（1420年） |
| 祭祀对象 | 社神（土地之神）和稷神（五谷之神） |
| 祭祀等级 | 大祀 |
| 祭祀时间 | 仲春（二月）、仲秋（八月）上戊日 |

### 4.2 五色土

社稷坛最著名的特征是坛面铺设的"五色土"：

| 方位 | 颜色 | 象征 | 对应五行 |
|------|------|------|----------|
| 东 | 青色 | 东方大地 | 木 |
| 南 | 红色 | 南方大地 | 火 |
| 西 | 白色 | 西方大地 | 金 |
| 北 | 黑色 | 北方大地 | 水 |
| 中 | 黄色 | 中央大地 | 土 |

五色土的配置象征"普天之下，莫非王土"，代表中国辽阔的国土和五方之地。

### 4.3 坛台规制

| 项目 | 内容 |
|------|------|
| 形制 | 正方形，三层 |
| 材质 | 上层为五色土，下层为砖石 |
| 周围 | 矮墙环绕，四面开门 |
| 社主 | 石柱一座，立于坛台正中（原为木主，后改石主） |

### 4.4 春秋祭祀

| 项目 | 春祭 | 秋祭 |
|------|------|------|
| 时间 | 仲春（二月）上戊日 | 仲秋（八月）上戊日 |
| 主祭 | 皇帝亲祭 | 皇帝亲祭 |
| 祈求 | 五谷丰登，国泰民安 | 感恩丰收，报答土谷之神 |
| 祭品 | 太牢（牛羊猪）＋玉帛 | 太牢＋玉帛 |

### 4.5 洪武朝的社稷改制

洪武十年（1377年），朱元璋认为"社"（土神）和"稷"（谷神）本不可分，令礼官奏议后合社稷共为一坛。嘉靖年间，嘉靖帝又另建"帝社稷坛"（属皇帝自立的"王社"），体现了嘉靖帝对礼制的个人化改造。

## 五、先农坛与先蚕坛

### 5.1 先农坛

| 项目 | 内容 |
|------|------|
| 位置 | 北京南中轴线西侧（今北京先农坛） |
| 始建 | 永乐十八年（1420年） |
| 祭祀对象 | 先农之神（神农氏） |
| 祭祀等级 | 中祀（洪武朝一度为大祀，后降为中祀） |
| 祭祀时间 | 仲春亥日 |

#### 耕籍礼（藉田礼）

每年仲春亥日，皇帝率百官到先农坛行"耕籍礼"：

| 步骤 | 内容 |
|------|------|
| 1. 祭祀先农 | 皇帝在先农坛向神农氏行礼祭祀 |
| 2. 亲耕 | 皇帝在"一亩三分地"（皇帝亲耕田）执耒亲耕 |
| 3. 三推三返 | 皇帝推犁三个来回，象征重农 |
| 4. 百官耕作 | 三公五推五返，九卿九推九返 |
| 5. 收获 | 亲耕所得谷物用于太庙祭祀 |

#### 先农坛内建筑群

| 建筑 | 功能 |
|------|------|
| 先农坛 | 祭祀先农之神 |
| 太岁殿 | 祭祀太岁之神 |
| 观耕台 | 皇帝观看百官耕作 |
| 庆成宫 | 耕籍礼后皇帝更衣休息 |
| 具服殿 | 皇帝祭祀前更换祭服 |
| 神仓 | 储藏亲耕所得谷物 |

### 5.2 先蚕坛

| 项目 | 内容 |
|------|------|
| 位置 | 北京安定门外（与先农坛大致对应的北方位置） |
| 建造时间 | 嘉靖年间 |
| 祭祀对象 | 先蚕之神（嫘祖，黄帝元妃） |
| 祭祀主体 | 皇后 |
| 祭祀时间 | 仲春 |

#### 亲蚕礼

与皇帝的耕籍礼对应，皇后每年仲春行"亲蚕礼"：

| 步骤 | 内容 |
|------|------|
| 1. 祭先蚕 | 皇后在先蚕坛向嫘祖行礼祭祀 |
| 2. 亲桑 | 皇后在坛院桑林中采桑叶 |
| 3. 养蚕 | 蚕妇在坛院内采桑养蚕 |
| 4. 缫丝 | 所得蚕丝用于制作祭服 |

#### 先蚕坛特点

- 院内广植桑树
- 设有蚕室，常年有蚕妇养蚕
- 体现"男耕女织"的农耕文明理想
- 皇帝亲耕先农、皇后亲蚕先蚕，共同构成国家重农政策的仪式化表达

## 六、嘉靖朝祭祀改革的整体评价

### 6.1 改革清单

| 改革项目 | 具体内容 | 时间 |
|----------|----------|------|
| 天地分祀 | 天地坛分为天坛、地坛，新建日坛、月坛 | 嘉靖九年（1530） |
| 太庙改制 | 火灾后恢复"同堂异室"制 | 嘉靖二十四年（1545） |
| 大礼议入庙 | 将生父睿宗入太庙配享 | 嘉靖十七年（1538） |
| 帝社稷坛 | 另建皇帝私人的"王社" | 嘉靖年间 |
| 先蚕坛 | 新建先蚕坛供皇后亲蚕 | 嘉靖年间 |
| 更定祀典 | 调整大祀、中祀分类 | 嘉靖年间 |
| 明堂大享 | 在大高玄殿举行明堂大享礼 | 嘉靖年间 |

### 6.2 嘉靖祭祀改革的特点

1. **以古礼否定近世之制**：嘉靖帝常以周礼为依据推翻前朝惯例
2. **与大礼议相呼应**：祭祀改革是嘉靖帝确立个人权威的系列行动之一
3. **道教色彩渗入**：大高玄殿等道教场所承担了部分祭祀功能
4. **持续时间长**：祭祀改革贯穿嘉靖在位的45年
5. **影响深远**：五坛格局延续至清末，成为北京城市文化遗产的核心组成
