# 明代道教（1521-1644）

## 一、正一派与全真派对比

### 1.1 对比表

| 比较维度 | 正一派 | 全真派 |
|----------|--------|--------|
| 创立时间 | 东汉末年（张道陵） | 金代（王重阳，1167年） |
| 核心经典 | 《正一经》及各种科仪经典 | 《道德经》《清静经》《孝经》 |
| 修炼方式 | 画符念咒、祈禳斋醮、外法驱邪 | 性命双修、先性后命、内丹修炼 |
| 对待符箓 | 以符箓法术为核心 | 不尚符箓，重视心性修炼 |
| 饮食戒律 | 可食荤，不严格戒酒肉 | 素食，严格持戒 |
| 婚姻制度 | 可娶妻生子，不必出家 | 须出家，独身修行 |
| 组织结构 | 以龙虎山张天师为首，联盟性质较松散 | 宫观制度严密，十方丛林 |
| 宫观管理 | 子孙庙制（庙产私有，师徒世代相传） | 十方丛林制（庙产公有） |
| 传承方式 | 父传子为主，兼有师徒传授 | 师徒传承，传戒制度 |
| 传度仪式 | 授箓（授予法箓） | 传戒（受持戒律） |
| 在明代地位 | 受皇室重用，主持国家斋醮 | 地位不及正一，主要在北方活动 |
| 主要活动区域 | 江西龙虎山为中心，南方为主 | 北方为主，全真七子各立门户 |

### 1.2 明代两派关系

明代官方道教以正一派为主体。洪武元年（1368年），太祖封正一天师为"正一嗣教真人"，确立了正一派在官方体系中的主导地位。全真派虽未被取缔，但在朝廷中的影响远不及正一派。嘉靖朝崇道以正一派为主，所用道士多为正一派系统。

> 来源：[道教全真派和正一派的区别 - 福建省人民政府](https://www.fujian.gov.cn/hdjl/hdjlzsk/mzzjt/zj/202012/t20201224_5500308.htm)；[正一道 - 维基百科](https://zh.wikipedia.org/zh-hans/%E6%AD%A3%E4%B8%80%E9%81%93)；[全真道 - 百度百科](https://baike.baidu.com/item/%E5%85%A8%E7%9C%9F%E9%81%93/677438)

## 二、道录司体系

### 2.1 中央机构：道录司

道录司同样隶属于礼部，是明代管理全国道教事务的最高行政机构，洪武十五年（1382年）设立。

**中央八个官职及品级：**

| 官职 | 员额 | 品级 |
|------|------|------|
| 左正一 | 1人 | 正六品 |
| 右正一 | 1人 | 正六品 |
| 左演法 | 1人 | 从六品 |
| 右演法 | 1人 | 从六品 |
| 左至灵 | 1人 | 正八品 |
| 右至灵 | 1人 | 正八品 |
| 左玄义 | 1人 | 从八品 |
| 右玄义 | 1人 | 从八品 |

### 2.2 地方机构

| 层级 | 机构名称 | 主官 |
|------|----------|------|
| 府 | 道纪司 | 都纪（从九品） |
| 州 | 道正司 | 道正（未入流） |
| 县 | 道会司 | 道会（未入流） |

### 2.3 神乐观

神乐观是道录司下属的特殊机构，隶属太常寺管辖，负责国家祭祀中的乐舞演奏。

**关键信息：**
- 位于天坛西侧，又称"神乐署"
- 由道士担任乐舞生，负责祭天、祭地、祭祖等国家大典的音乐仪式
- 嘉靖时期，神乐观道士大量进入太常寺任职，以至有"太常寺由神乐观道士运营"之说
- 神乐观的存在体现了明代道教与王朝礼仪的深度融合
- 明太祖颁布《大明御制玄教乐章》，规范了祭祀乐舞

> 来源：[明朝官职表 - 维基百科](https://zh.wikipedia.org/zh-hans/%E6%98%8E%E6%9C%9D%E5%AE%98%E8%81%B7%E8%A1%A8)；[明清时期的神乐观与王朝礼仪 - 哲学中国网](http://philosophy.org.cn/fzxk/zjx/201509/t20150925_2728449.shtml)；[神乐署 - 维基百科](https://zh.wikipedia.org/zh-my/%E7%A5%9E%E4%B9%90%E7%BD%B2)

## 三、正一天师封号与地位

### 3.1 明代天师封号

| 皇帝 | 天师 | 封号 |
|------|------|------|
| 太祖朱元璋 | 第42代张正常 | "正一嗣教真人"，赐银印，秩视二品 |
| 太祖（洪武三年） | 张正常 | 加赐"永掌天下道教事" |
| 成祖朱棣 | 第43代张宇初 | 继承"正一嗣教真人"，受命编修道藏 |
| 后续各朝 | 历代天师 | 均承袭"正一嗣教真人"封号 |

### 3.2 天师的特殊地位

- 秩视二品，地位极高（高于道录司正六品的最高僧官）
- 有权统领天下道教事务
- 立"万法宗坛"于龙虎山，奠定正一道祖庭地位
- 每代天师须赴京朝觐，接受皇帝册封
- 嘉靖朝期间，天师府在朝廷中的影响力达到顶峰

### 3.3 龙虎山

嗣汉天师府位于江西贵溪县龙虎山，始建于北宋崇宁四年（1105年），是正一道的祖庭和天师的驻锡之地。明代历代天师均以此为根据地，统领全国正一道事务。

> 来源：[正一真人统领下的龙虎山道教与明代官道 - 中国社会科学网](https://www.cssn.cn/zx/zjx/202211/t20221118_5565277.shtml)；[正一道 - 维基百科](https://zh.wikipedia.org/zh-cn/%E6%AD%A3%E4%B8%80%E9%81%93)

## 四、嘉靖帝崇道全面梳理

明世宗朱厚熜（1521-1566年在位）是中国历史上最为崇信道教的皇帝之一，其崇道活动贯穿整个在位时期，对明代政治、宗教格局产生了深远影响。

### 4.1 三个递进道号

嘉靖帝先后为自己加上三个道号，道号长度和等级逐步升级：

| 阶段 | 道号 | 含义 |
|------|------|------|
| 第一阶段 | 灵霄上清统雷元阳妙一飞玄真君 | 以"真君"自称，统领雷法 |
| 第二阶段 | 九天弘教普济生灵掌阴阳功过大道思仁紫极仙翁一阳真人元虚玄应开化伏魔忠孝帝君 | 升格为"帝君"，自诩掌管阴阳功过 |
| 第三阶段 | 太上大罗天仙紫极长生圣智昭灵统元证应玉虚总掌五雷大真人玄都境万寿帝君 | 最终道号，自比"太上"级别的仙人帝君 |

> 来源：[尊号 - 维基百科](https://zh.wikipedia.org/zh-hans/%E5%B0%8A%E5%8F%B7)；[嘉靖皇帝：既胡闹又抓权 - AllHistory](https://m2.allhistory.com/ah/article/5f3124f4d47fa2000162b288)

### 4.2 邵元节和陶仲文的封赠

#### 邵元节（1459-1539）

| 项目 | 内容 |
|------|------|
| 籍贯 | 江西贵溪龙虎山 |
| 派系 | 正一教 |
| 入宫时间 | 嘉靖三年（1524年），因祈雨应验入宫 |
| 主要职务 | 总领道教事，主持斋醮 |
| 最高封赠 | 礼部尚书衔，赐一品服 |
| 影响 | 其侄邵启南及徒陈善道、吴尚礼等均获重用，升任道录司首脑 |
| 去世 | 嘉靖十八年（1539年） |

#### 陶仲文（1475-1560）

| 项目 | 内容 |
|------|------|
| 原名 | 典真 |
| 籍贯 | 湖北黄冈 |
| 入宫经过 | 经邵元节推荐入宫 |
| 获宠原因 | 以符水驱妖、治愈庄敬太子天花 |
| 最高职衔 | 身兼少师、少傅、少保（"三孤"齐集一身，明代道士唯此一人） |
| 同时加封 | 礼部尚书衔，恩荣世袭 |
| 去世 | 嘉靖三十九年（1560年） |

陶仲文是明代历史上唯一一位同时身兼少师、少傅、少保三个头衔的道士，位极人臣。

> 来源：[邵元节 - 维基百科](https://zh.wikipedia.org/zh-hans/%E9%82%B5%E5%85%83%E8%8A%82)；[陶仲文 - 维基百科](https://zh.wikipedia.org/zh-hans/%E9%99%B6%E4%BB%B2%E6%96%87)；[陶仲文 - 百度百科](https://baike.baidu.com/item/%E9%99%B6%E4%BB%B2%E6%96%87/8945098)

### 4.3 斋醮活动

斋醮是道教最核心的宗教仪式，嘉靖帝在位期间大规模举办：

- 日常设醮：嘉靖帝"日夜设醮"，几乎每日都有各种规模的斋醮活动
- 特别斋醮：每逢天灾人祸、生病求嗣等大事，均举行大规模斋醮
- 费用巨大：斋醮所用香烛、纸帛、供品等开支极为庞大，成为国家财政的沉重负担
- 百官参与：重大斋醮要求百官斋戒、行香、禁刑屠

### 4.4 大高玄殿

| 项目 | 内容 |
|------|------|
| 建成时间 | 嘉靖二十一年（1542年）四月 |
| 位置 | 北京紫禁城西北侧（今景山前街） |
| 面积 | 东西宽57米，南北长264米，约1.5万平方米 |
| 供奉 | 玉皇大帝及三清（玉清、上清、太清） |
| 建议者 | 陶仲文 |
| 主持修建 | 工匠郭文英 |
| 木料来源 | 四川、贵州、湖广等地 |
| 用途 | 皇家御用道观，"每岁大旱则建醮祈雨"，平民不得出入 |
| 建筑特色 | 最后面的"乾元阁"上圆下方，象征"天圆地方" |

嘉靖帝建成大高玄殿后"有祷必至"，百官斋戒、行香、禁刑屠十日。

> 来源：[大高玄殿 - 维基百科](https://zh.wikipedia.org/zh-hans/%E5%A4%A7%E9%AB%98%E7%8E%84%E6%AE%BF)；[北京大高玄殿观览记 - 中国旅游报](http://www.ctnews.com.cn/paper/content/202011/10/content_51963.html)

### 4.5 青词宰相（六人名单）

青词是道教斋醮仪式中献给天神的祝祷文书，以朱砂写在青藤纸上，故名"青词"。嘉靖帝极为看重青词的文采，擅写青词者往往获得快速提拔。

**六位"青词宰相"：**

| 姓名 | 入阁时间 | 最高职务 | 特点 |
|------|----------|----------|------|
| 严嵩 | 嘉靖二十一年（1542） | 内阁首辅 | 最著名的青词宰相，专宠20年 |
| 徐阶 | 嘉靖二十八年（1549） | 内阁首辅 | 以青词入直无逸殿，后扳倒严嵩 |
| 李春芳 | 嘉靖四十四年（1565） | 内阁首辅 | "青词宰相"典型代表 |
| 严讷 | 嘉靖四十三年（1564） | 内阁大学士 | 同以青词获宠 |
| 郭朴 | 嘉靖四十四年（1565） | 内阁大学士 | 河南安阳人，嘉靖十四年进士 |
| 袁炜 | 嘉靖四十一年（1562） | 内阁大学士 | 撰写青词能力最强 |

《明史·袁炜传》载："自嘉靖中年，帝专事焚修，词臣率供奉青词。工者立超擢，卒至入阁。时谓李春芳、严讷、郭朴及炜为青词宰相。"

### 4.6 炼丹方术

#### 红铅

- 红铅是以处女初潮经血为原料炼制的丹药
- 陶仲文引导嘉靖帝修炼此术，声称可增加道法和修为
- 为保持"洁净"，宫女只能喝露水、吃桑叶，不许吃五谷杂粮
- 宫中多有宫女因此饿死
- 嘉靖二十一年（1542年）"壬寅宫变"即与此有关：被虐待的宫女杨金英等人企图勒死嘉靖帝

#### 秋石

- 秋石是以童男童女尿液提炼的结晶物
- 号称有延年益寿、补肾固精之效
- 嘉靖帝长期服用

#### 其他方术

- 命宫女凌晨在御花园采集露水，用以炼丹或直接饮用（"吸风饮露"）
- 大量炼制各种丹药，内含铅、汞等有毒金属
- 嘉靖帝晚年长期居住西苑，不上朝，专心修道炼丹

> 来源：[嘉靖皇帝 - AllHistory](https://m2.allhistory.com/ah/article/5f3124f4d47fa2000162b288)；[陶仲文 - 百度百科](https://baike.baidu.com/item/%E9%99%B6%E4%BB%B2%E6%96%87/8945098)

## 五、武当山与真武大帝

### 5.1 武当山的地位

武当山位于湖北省西北部，是道教名山，在明代被尊为"皇室家庙"级别的道教圣地。

### 5.2 永乐大修武当山

| 项目 | 内容 |
|------|------|
| 修建动机 | 明成祖感谢"靖难之役"中真武大帝的护佑，"报答神惠""为天下生灵祈福" |
| 开工时间 | 永乐十年（1412年） |
| 工程规模 | 调集三十万军民工匠 |
| 工期 | 历时约十年 |
| 设计师 | 蒯祥（同时负责紫禁城设计） |
| 建成规模 | 八宫二观、三十六庵堂、七十二岩庙、三十九桥、十二亭 |

### 5.3 真武大帝信仰

- 真武大帝（又称玄天上帝、北极真武、玄武大帝）是道教中的北方之神
- 明成祖靖难夺位后，宣称得到真武大帝庇佑，大力推崇真武信仰
- 北京紫禁城中的钦安殿即供奉真武大帝
- 嘉靖帝继续推崇真武信仰，武当山在整个明代保持崇高地位
- 武当山道教建筑群于1994年被列入世界文化遗产

### 5.4 主要宫观

| 宫观名称 | 位置 | 特点 |
|----------|------|------|
| 紫霄宫 | 展旗峰下 | 明代武当山最大的道教宫观 |
| 南岩宫 | 南岩 | 依悬崖而建，极为险峻 |
| 太和宫/金殿 | 天柱峰顶 | 铜铸鎏金，永乐十四年建 |
| 遇真宫 | 山脚 | 供奉武当派创始人张三丰 |
| 玉虚宫 | 山前 | 规模最大，号称"小紫禁城" |
