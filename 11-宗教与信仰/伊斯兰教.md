# 明代伊斯兰教（1521-1644）

## 一、回族社区分布格局

### 1.1 "大分散，小集中"的分布特点

明代回族延续了唐宋以来"大分散，小集中"的居住格局，主要分布在以下区域：

| 区域 | 主要聚居地 | 特点 |
|------|-----------|------|
| 西北地区 | 陕西西安、甘肃兰州、宁夏、青海 | 人口最密集，经堂教育的发源地 |
| 华北地区 | 北京、天津、河北、山东 | 元代大都回回遗民及明代经商聚居 |
| 华东地区 | 南京、扬州、泉州、杭州 | 海上丝绸之路遗留社区 |
| 西南地区 | 云南昆明、大理 | 元代赛典赤·赡思丁治滇遗留 |
| 中原地区 | 河南开封、郑州 | 分布较散 |

### 1.2 分布特征

- 大都分布于水陆交通线上，经济相对发达
- 形成围绕清真寺的"围寺而居"格局（坊）
- 以清真寺为中心的回族社区称为"教坊"
- 城市中多从事商业贸易（牛羊贩卖、珠宝玉石、香料药材）
- 农村回族兼营农牧业

### 1.3 人口估算

明代回族人口缺乏精确统计。根据学者推算：
- 明初（洪武年间）：回族人口约200万左右
- 明中后期（嘉靖至万历）：约300-500万（含部分汉化和新改宗者）
- 主要依据：元代西北屯戍的"探马赤军"后裔、元代色目人后裔、海上贸易定居者、通婚汉化人口

## 二、清真寺建筑特点与本土化

### 2.1 建筑风格的中国化

明代是中国清真寺建筑风格完成本土化转型的关键时期。清真寺从早期的阿拉伯-波斯风格逐渐转变为中国宫殿式建筑。

| 比较维度 | 西亚伊斯兰风格 | 中国化风格（明代） |
|----------|----------------|-------------------|
| 结构体系 | 砖石拱券结构 | 木结构或砖木结构 |
| 屋顶形式 | 穹顶（dome） | 歇山顶/庑殿顶 |
| 宣礼塔 | 圆柱形高塔 | 中式楼阁或省略 |
| 装饰元素 | 几何纹、阿拉伯书法 | 雕梁画栋＋阿拉伯经文匾额 |
| 院落布局 | 围合式庭院 | 轴线对称的多进院落 |

### 2.2 典型布局："三堂一院"

| 组成部分 | 功能 |
|----------|------|
| 礼拜堂（大殿） | 正殿，面向麦加方向（中国为坐西朝东） |
| 经堂 | 南厢房，教授经典之所 |
| 澡堂（水房） | 北厢房，礼拜前沐浴净身 |
| 庭院 | 中央院落，日常活动空间 |

### 2.3 代表性清真寺

| 清真寺名称 | 位置 | 始建时间 | 特点 |
|-----------|------|----------|------|
| 西安化觉巷清真寺 | 陕西西安 | 唐天宝元年（742），明代重修扩建 | 中国规模最大的古清真寺，五进院落 |
| 北京牛街清真寺 | 北京宣武 | 辽代，明正统年间重修 | 北京最古老的清真寺 |
| 南京净觉寺 | 南京 | 明洪武年间 | 朱元璋御赐建造 |
| 松江清真寺 | 上海松江 | 元代，明代重修 | 江南地区代表 |
| 泉州清净寺 | 福建泉州 | 北宋，明代修缮 | 保留阿拉伯建筑风格 |

### 2.4 数量估计

- 嘉靖十年（1531年）《嘉靖宁夏新志》已有多座清真寺的记载
- 明代全国清真寺数量缺乏精确统计，但据学者估算，明中后期全国清真寺应在数千座以上
- 几乎每个有回族聚居的城镇、村庄都建有清真寺

> 来源：[有容乃大：从回族清真寺建筑看伊斯兰教的中国化 - 陕西省伊斯兰教协会](https://www.sxyslj.com/zcfg_content.php?id=1011)；[回族特色建筑——清真寺 - 中国民族网](http://www.minzu56.net/hz/wh/14502.html)；[回族建筑艺术](http://www.muluanzuo.mzzyk.com/eportal/ui?pageId=663068&articleKey=674010&columnId=682346)

## 三、胡登洲经堂教育制度

### 3.1 创始人与背景

| 项目 | 内容 |
|------|------|
| 创始人 | 胡登洲（约1522-1597年） |
| 籍贯 | 陕西咸阳渭城 |
| 创办地点 | 咸阳渭城里清真寺 |
| 创办时间 | 约嘉靖至万历年间 |
| 教育模式 | 将伊斯兰教传统教育与中国私塾教育方法相结合 |

### 3.2 经堂教育的核心内容

| 维度 | 具体内容 |
|------|----------|
| 教学语言 | 创立"经堂语"——用汉语音译阿拉伯语/波斯语，吸收儒道佛用语 |
| 教学内容 | 阿拉伯语、波斯语、古兰经、圣训、教法、教义学 |
| 教学方法 | 以儒论经，贯通儒学与伊斯兰教义 |
| 教学场所 | 清真寺内的经堂（学堂） |
| 学制 | 初级（念经学阿语）→中级（学教法）→高级（研究经义），一般需8-12年 |
| 师资 | 阿訇（教长）担任教师 |
| 毕业仪式 | "穿衣挂帐"——合格者由师傅授予"伊扎泽"（教学资格证书） |

### 3.3 四大学派

胡登洲创立的经堂教育向全国传播，逐渐形成四大学派：

| 学派 | 代表人物 | 核心实践 | 辐射区域 |
|------|----------|----------|----------|
| **陕西学派**（陕学） | 胡登洲（约1522-1597） | 经堂教育原创体系 | 陕西、甘肃、宁夏、青海（西北地区） |
| **山东学派**（鲁学） | 常志美（约1610-1670） | 经堂教育东传发展 | 华中、华北、东北地区 |
| **金陵学派**（汉学派） | 王岱舆（?-1657）、刘智（约1669-1764） | "以儒诠经"汉文著述 | 南京为中心，影响全国 |
| **云南学派**（滇学） | 马注（约1640-?）、马德新（约1794-1874） | 经堂教育＋原典翻译＋以儒诠经 | 西南、东南及东南亚 |

**传承谱系：**
- 常志美是胡登洲的四传弟子
- 发展轨迹：西北（陕西）→ 华中（山东）→ 江南（金陵）→ 西南（云南）

> 来源：[传承胡登洲经堂教育思想 - 陕西省伊斯兰教协会](https://sxyslj.com/wez_content.php?id=702)；[坚持伊斯兰教中国化方向的典范 - 中国伊斯兰教](http://w.chinaislam.net.cn/cms/zjjy/yjdt/rdyj/201912/03-13653.html)；[中国经堂教育溯源](http://www.pacilution.com/ShowArticle.asp?ArticleID=9362)

## 四、"以儒诠经"运动

### 4.1 运动概述

"以儒诠经"是明末清初中国伊斯兰教学者用儒家思想和汉语来阐释伊斯兰教教义的学术运动，是伊斯兰教中国化的标志性成就。

### 4.2 王岱舆

| 项目 | 内容 |
|------|------|
| 生卒 | ?-1657年（一说1570-1660年） |
| 籍贯 | 南京 |
| 民族 | 回族 |
| 家学 | 世代伊斯兰学者，通阿拉伯语、波斯语 |
| 地位 | "以儒诠经"运动的开创者 |
| 代表著作 | 《正教真诠》《清真大学》《希真正答》 |

**主要贡献：**
- 《正教真诠》是迄今发现的最早的汉语伊斯兰教著作
- 确立了中国伊斯兰哲学中"真一""数一""体一"三大哲学概念
  - **真一**：独一真主（安拉），宇宙的根本存在
  - **数一**：真主的属性和名号
  - **体一**：宇宙万物由真一创造而成的整体
- 开启了用儒学概念解释伊斯兰教义的先河
- 以儒家的"天""理""性"等概念对应伊斯兰教的"安拉""定命""灵魂"

### 4.3 刘智

| 项目 | 内容 |
|------|------|
| 生卒 | 约1669-1764年（一说1660-1730年） |
| 籍贯 | 南京 |
| 民族 | 回族 |
| 别名 | 介廉 |
| 地位 | 金陵学派最杰出代表，"以儒诠经"运动的集大成者 |
| 代表著作 | 《天方性理》《天方典礼》《天方至圣实录》《真境昭微》《五功释义》《天方三字经》 |

**主要贡献：**
- 自幼习读伊斯兰教经籍，青年时博览儒、佛、道典籍
- 隐居南京清凉山麓十余年，潜心著译
- 《天方典礼》系统介绍伊斯兰教的礼仪制度，以儒学五伦对应伊斯兰伦理
- 《天方性理》深入阐述伊斯兰哲学，以儒学"性理"概念解释伊斯兰宇宙论
- 代表了17世纪中国伊斯兰教哲学思想的最高成就
- 将中国的"以儒诠经"运动推向高潮

### 4.4 "以儒诠经"的方法论

| 方法 | 具体表现 |
|------|----------|
| 概念对应 | 用儒学的"天""理""性""命"对应伊斯兰教的核心概念 |
| 经典互释 | 引用《论语》《孟子》《中庸》来阐释古兰经义理 |
| 伦理融合 | 将伊斯兰"五功"与儒家"五常"对应 |
| 哲学贯通 | 用宋明理学的"太极""理气"框架解释伊斯兰宇宙观 |

### 4.5 运动的历史意义

- 实现了伊斯兰教义的汉语化表达
- 促进了回族知识精英与汉族儒生的思想对话
- 为伊斯兰教在中国的长期存续提供了文化合法性
- 形成了独特的"回儒"学术传统

## 五、回族人口估算补充

### 5.1 影响人口统计的因素

- 明代户籍制度中并未单独统计回族人口
- 许多回族在明初已被编入"民户""军户"
- 通婚汉化使部分回族淡出统计视野
- 西北边疆地区的回族人口难以精确计数

### 5.2 间接数据

| 指标 | 数据 | 来源/说明 |
|------|------|-----------|
| 清真寺数量 | 明代全国数千座 | 间接反映回族社区数量 |
| 经堂教育学生 | 各大清真寺常年有数十至百余名学生 | 胡登洲以后的制度 |
| 地方志记载 | 各府县志中有"回民"或"回回"社区记载 | 分散但普遍 |
| 军户 | 明代回族军户遍布各卫所 | 延续元代军事编制 |
