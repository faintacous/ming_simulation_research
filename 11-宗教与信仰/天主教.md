# 明代天主教（1521-1644）

## 一、耶稣会入华完整时间线（1579-1644）

### 1.1 准备阶段（1579-1582）

| 时间 | 事件 |
|------|------|
| 1579年7月 | 耶稣会士罗明坚（Michele Ruggieri）抵达澳门，遵范礼安指示学习中文 |
| 1580年 | 罗明坚初次尝试进入广东，未果 |
| 1582年8月 | 利玛窦（Matteo Ricci）抵达澳门 |
| 1582年12月 | 罗明坚应两广总督陈瑞和肇庆知府王泮邀请，带领利玛窦、巴范济（Francesco Pasio）抵达广东肇庆 |

**关键人物——范礼安（Alessandro Valignano）：**
- 耶稣会远东巡视员
- 制定了"文化适应"传教策略的总方针
- 指示传教士必须学习当地语言和文化
- 未亲自进入中国内地，但在澳门指挥全局

### 1.2 扎根阶段（1583-1600）

| 时间 | 事件 |
|------|------|
| 1583年9月 | 罗明坚和利玛窦获准在肇庆定居，建立"仙花寺"（第一座内地教堂） |
| 1583-1588年 | 利玛窦在肇庆期间，制作世界地图、展示西方科学仪器，吸引士大夫关注 |
| 1588年 | 罗明坚被召回欧洲，利玛窦独自承担中国传教重任 |
| 1589年 | 利玛窦被迫离开肇庆，迁至韶州（今韶关） |
| 1594年 | 利玛窦脱去僧服，改穿儒服——标志性的"合儒"转向 |
| 1595年 | 利玛窦从韶州北上，途经南昌，短暂逗留并结交官员 |
| 1595-1598年 | 利玛窦定居南昌，与当地士大夫广泛交往 |
| 1598年 | 利玛窦首次抵达北京，因时局不稳未能久留 |
| 1599年 | 利玛窦定居南京，在此发展教务 |

### 1.3 发展阶段（1601-1610）

| 时间 | 事件 |
|------|------|
| 1601年1月 | 利玛窦携带自鸣钟、世界地图、三棱镜、圣像等贡品抵达北京，觐见万历帝 |
| 1601年 | 万历帝批准利玛窦在北京永久居住，赐俸禄 |
| 1603年 | 徐光启在南京受洗，圣名保禄 |
| 1604年 | 利玛窦与徐光启合作翻译《几何原本》（前六卷） |
| 1605年 | 北京教徒已达200人，其中有公卿大臣 |
| 1607年 | 《几何原本》前六卷出版 |
| 1610年 | 李之藻在北京受洗，圣名良（Leo） |
| 1610年5月11日 | 利玛窦在北京病逝，万历帝赐地安葬 |

### 1.4 扩展阶段（1611-1616）

| 时间 | 事件 |
|------|------|
| 1611年 | 杨廷筠在杭州受洗，圣名弥格尔（Michael） |
| 1611年 | 龙华民（Niccolò Longobardo）继任耶稣会中国区会长 |
| 1612-1615年 | 传教范围扩大到江西、福建、山西、陕西等省 |
| 1613年 | 艾儒略（Giulio Aleni）来华，在福建传教 |
| 1615年 | 金尼阁（Nicolas Trigault）赴欧洲为中国传教募款 |

### 1.5 挫折阶段（1616-1622）

| 时间 | 事件 |
|------|------|
| 1616年 | **南京教案爆发**（详见第六节） |
| 1617年2月 | 南京传教士被驱逐至澳门 |
| 1617-1621年 | 传教活动受到严重限制 |
| 1621年 | 沈榷被撤职，天主教活动逐渐恢复 |
| 1622年 | 传教士重返各省传教 |

### 1.6 恢复与扩展阶段（1622-1644）

| 时间 | 事件 |
|------|------|
| 1622年 | 邓玉函（Johann Terrenz）、汤若望（Johann Adam Schall von Bell）等来华 |
| 1623年 | 艾儒略在福建发展教务，福建成为传教重镇 |
| 1628年 | 汤若望进入北京，参与历法修订 |
| 1629年 | 崇祯帝命徐光启主持修历，汤若望、罗雅谷等参与 |
| 1630年 | 徐光启主持的"历局"正式开始工作 |
| 1633年 | 徐光启去世 |
| 1634年 | 方济各会、多明我会传教士也开始进入中国 |
| 1644年 | 明朝灭亡，清兵入关。汤若望留在北京，协助清朝编制历法 |

## 二、利玛窦"合儒"策略五要点

### 2.1 策略概述

利玛窦在范礼安"文化适应"方针指导下，结合中国实际，创造了独特的"合儒"传教策略（又称"利玛窦规矩"）。

### 2.2 五大要点

| 要点 | 具体内容 | 策略意义 |
|------|----------|----------|
| **1. 合儒斥佛** | 接受先秦儒家思想，认为古代儒学中的"天"和"上帝"与天主教的唯一真神本质相同；同时批判佛教的核心教义（轮回、偶像崇拜等） | 拉拢儒生士大夫阶层，消除改宗心理障碍 |
| **2. 走上层路线** | 主动结交官员、学者、士大夫，通过精英阶层影响社会 | 获得社会认可和政治保护 |
| **3. 以西学辅教** | 展示西方科学技术（地理学、数学、天文学、机械钟表等）作为传教辅助手段 | 以学术交流打开传教之门 |
| **4. 着儒服说儒语** | 放弃初期穿着的僧服，改穿儒生服饰；精通中文，采用儒学话语体系 | 文化上融入中国社会 |
| **5. 容许祭祖祀孔** | 允许中国天主教徒保留祭祖和祭孔礼仪，认为这些活动仅是表达敬意的社会行为，不具有宗教含义 | 消除信徒在传统与信仰之间的矛盾 |

### 2.3 策略成效

- 成功进入中国内地，直达京师
- 获得万历帝认可，赐地建堂
- 吸引了一批高级知识分子和官员入教
- 翻译出版了大量西方科学著作（150多种）
- 建立了可持续的传教体系

### 2.4 策略争议

利玛窦的"合儒"策略在后来引发了著名的"礼仪之争"：
- 耶稣会坚持利玛窦规矩，允许祭祖祀孔
- 多明我会、方济各会指控耶稣会向异教妥协
- 最终罗马教廷在1704年禁止中国礼仪，导致康熙禁教

## 三、"圣教三柱石"

"圣教三柱石"是指利玛窦在华传教期间培养的三位最杰出的中国天主教徒，他们为天主教在中国的发展做出了最大贡献。

### 3.1 徐光启

| 项目 | 内容 |
|------|------|
| 生卒 | 1562-1633年 |
| 字号 | 字子先，号玄扈 |
| 籍贯 | 上海 |
| 科举 | 万历三十二年（1604年）进士 |
| 最高官职 | 礼部尚书兼文渊阁大学士（入阁拜相） |
| 受洗时间 | 万历三十一年（1603年），在南京由罗如望（João da Rocha）施洗 |
| 圣名 | 保禄（Paulus） |

**主要贡献：**
- 与利玛窦合译《几何原本》前六卷（1607年出版），是近代数学传入中国的里程碑
- 编译《泰西水法》，引介西方水利灌溉技术
- 主持崇祯朝历法改革，聘用汤若望等传教士参与
- 著《农政全书》，融合中西农业知识
- 在上海、嘉定等地劝导家族和乡人入教，数百人受洗
- 是天主教徒中官位最高者，为传教活动提供了有力的政治庇护

### 3.2 李之藻

| 项目 | 内容 |
|------|------|
| 生卒 | 1565-1630年 |
| 字号 | 字振之，号凉庵居士 |
| 籍贯 | 浙江仁和（今杭州） |
| 科举 | 万历二十六年（1598年）进士 |
| 最高官职 | 南京太仆寺少卿 |
| 受洗时间 | 万历三十八年（1610年），在北京因病受洗 |
| 圣名 | 良（Leo） |
| 受洗经过 | 突患重病，皈依天主教后将家中佛像毁弃，换上耶稣像 |

**主要贡献：**
- 与利玛窦合作编译《同文算指》（8卷，1613年），是中国编译西方数学的最早著作
- 编译《浑盖通宪图说》，介绍西方天文仪器
- 刊刻出版了大量西学著作
- 学识渊博，娴于历算
- 在北京朝廷中为传教士争取支持

### 3.3 杨廷筠

| 项目 | 内容 |
|------|------|
| 生卒 | 1562-1627年 |
| 字号 | 字仲坚 |
| 籍贯 | 浙江仁和（今杭州） |
| 科举 | 万历二十年（1592年）进士 |
| 最高官职 | 监察御史 |
| 受洗时间 | 万历三十九年（1611年），在杭州受洗 |
| 圣名 | 弥格尔（Michael） |
| 受洗经过 | 参观李之藻父亲的西式（天主教）葬礼后决定改信 |

**主要贡献：**
- 著《代疑编》《代疑续编》，以儒学立场为天主教辩护
- 著《圣水纪言》《天释明辨》，比较天主教与佛教
- 在杭州建造教堂，资助传教活动
- 南京教案期间，积极为传教士辩护
- 被誉为"圣教三柱石"中的护教者

> 来源：[圣教三柱石 - 维基百科](https://zh.wikipedia.org/zh-hans/%E8%81%96%E6%95%99%E4%B8%89%E6%9F%B1%E7%9F%B3)；[徐光启 - 维基百科](https://zh.wikipedia.org/zh-hans/%E5%BE%90%E5%85%89%E5%90%AF)；[李之藻 - 维基百科](https://zh.wikipedia.org/zh-hans/%E6%9D%8E%E4%B9%8B%E8%97%BB)；[圣教三柱石 - 百度百科](https://baike.baidu.com/item/%E5%9C%A3%E6%95%99%E4%B8%89%E6%9F%B1%E7%9F%B3/2918916)

## 四、教徒人数

### 4.1 三组数据

| 时间 | 教徒人数 | 数据来源/说明 |
|------|----------|---------------|
| 万历三十三年（1605年） | 约200人（仅北京） | 其中有数名公卿大臣 |
| 明末（约1640年前后） | 约150,000人（全国） | 各省公开传教后的累计 |
| 1664年（康熙三年） | 约245,000人（全国） | 清初统计，含明末基础 |

### 4.2 增长轨迹

| 阶段 | 时间 | 特点 |
|------|------|------|
| 起步期 | 1583-1600 | 仅在广东、江西有少量教徒 |
| 发展期 | 1601-1616 | 利玛窦入京后，教徒增长加速 |
| 受挫期 | 1616-1621 | 南京教案，增长停滞 |
| 恢复期 | 1622-1644 | 传教范围扩大到全国十余省 |

### 4.3 传教士人数

| 时间 | 在华耶稣会士人数 | 说明 |
|------|------------------|------|
| 1585年 | 约3-5人 | 仅在广东 |
| 1610年 | 约15人 | 利玛窦去世时 |
| 1644年 | 约30-40人 | 明末 |
| 1664年 | 累计82人 | 含已去世者 |

## 五、南京教案（1616年）

### 5.1 事件背景

| 项目 | 内容 |
|------|------|
| 发起人 | 沈榷（署南京礼部侍郎，后署南京礼部尚书） |
| 发起时间 | 万历四十四年（1616年） |
| 地点 | 南京 |
| 动机 | 沈榷崇信佛教，厌恶天主教；个人与天主教的多次冲突；寻求晋升 |

### 5.2 完整经过

| 时间 | 事件 |
|------|------|
| 1615年 | 沈榷到南京任职，发现天主教在南京发展迅速，教堂已建，教务兴盛 |
| 1616年初 | 沈榷撰写《参远夷疏》上奏朝廷，指控传教士"立教惑众，蓄谋叵测" |
| 前两次上疏 | 未获朝廷重视，无果而终 |
| 第三次上疏 | 沈榷联合皇帝亲信和其他高官，共同发起攻击 |
| 1616年8月31日（万历四十四年七月二十日）| 傍晚，南京大队兵马包围教堂。传教士王丰肃（Alfonso Vagnone）、谢务禄（Alvaro Semedo）被逮捕 |
| 审判与处罚 | 王丰肃被重打十大板，与因病免杖的谢务禄一同被装入囚笼，戴上镣铐 |
| 押解 | 历经一个月颠沛流离，押解至广东 |
| 1617年2月3日（万历四十四年十二月二十八日）| 万历帝下旨："王丰肃等人'立教惑众，蓄谋叵测'，交由广督抚驱逐出境" |
| 北京传教士 | 庞迪我（Diego de Pantoja）等参与修历的传教士也被勒令遣返 |
| 最终处置 | 传教士被赶到澳门 |

### 5.3 各方反应

| 方面 | 反应 |
|------|------|
| 反教方 | 沈榷编纂《南宫署牍》记录此案，士大夫中有人响应反教 |
| 护教方 | 徐光启上疏辩护；杨廷筠在杭州为传教士说情；李之藻也参与抗辩 |
| 朝廷 | 万历帝态度模糊，最终偏向反教方 |
| 民间 | 南京部分教徒遭受迫害 |

### 5.4 教案结束

- 教案持续约三年（1616-1621年）
- 1621年万历帝去世，泰昌帝、天启帝继位
- 沈榷被撤职
- 天主教活动逐渐恢复，传教士重返各地

> 来源：[南京教案 - 百度百科](https://baike.baidu.com/item/%E5%8D%97%E4%BA%AC%E6%95%99%E6%A1%88/9748657)；[南京教案：明朝传教活动的落潮 - AllHistory](https://m2.allhistory.com/ah/article/5f47e235550c6f0001237619)；[在地狱的边缘——万历四十四年南京教案 - 知乎](https://zhuanlan.zhihu.com/p/51400741)
