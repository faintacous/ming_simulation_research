# 明末经世致用思潮（1580s-1644+）

> 聚焦嘉靖至崇祯年间（1521-1644），延伸至明清之际过渡期（至1680s），因经世致用思潮的代表人物主要活跃于明末清初。

---

## 一、经世致用思潮的背景

### 思想背景

| 因素 | 具体表现 |
|------|----------|
| **心学流弊** | 阳明后学流入空谈心性、狂禅，不关心实际政治和社会问题。顾炎武指斥"以明心见性之空言，代修己治人之实学" |
| **理学僵化** | 程朱理学在科举制度下沦为应试工具，士人"束书不观，游谈无根" |
| **明亡刺激** | 1644年甲申之变成为巨大思想冲击，知识分子反思亡国原因 |
| **社会危机** | 嘉靖以来的政治腐败、财政崩溃、流民叛乱、边患加剧 |

### 经世致用的基本主张

1. **反对空谈心性**：学问应该"经世致用"，有益于国计民生
2. **重视实证考据**：以实地调查和文献考证代替空洞议论
3. **关注制度改革**：从历史中寻找治国经验，提出具体改革方案
4. **批判君主专制**：对皇权独裁进行前所未有的理论批判

### 学术渊源

- 先秦儒学中"经世"的实用传统（孔子"学而优则仕"）
- 宋代"经制事功"学派（如叶适、陈亮的永嘉学派、永康学派）
- 明代中期丘濬《大学衍义补》的经世取向
- 东林学者"讲学与论政并举"的传统

## 二、顾炎武（1613-1682）

### 生平

- **姓名**：顾炎武，初名绛，字宁人，曾自署蒋山佣，学者称**亭林先生**
- **生卒**：万历四十一年（1613年）— 康熙二十一年（1682年）
- **籍贯**：南直隶昆山（今江苏昆山）
- **出身**：明代诸生（秀才），嗣母王氏为前绛州知州之女

#### 关键时间线

| 时间 | 年龄 | 事件 |
|------|------|------|
| 1613年 | 出生 | 生于江苏昆山 |
| 1626年 | 14岁 | 加入复社 |
| 1639年 | 27岁 | 科场失意后转向经世之学，开始编纂《天下郡国利病书》 |
| 1644年 | 32岁 | 甲申之变，明朝覆亡 |
| 1645年 | 33岁 | 参加昆山、嘉定一带抗清义军，嗣母王氏绝食殉国 |
| 1645年 | 33岁 | 南明鲁王授为兵部职方司主事 |
| 1647年 | 35岁 | 抗清失败后辗转各地 |
| 1657年 | 45岁 | 离开江南，开始北游，历经山东、河北、山西、陕西、河南等地 |
| 1667年 | 55岁 | 定居陕西华阴 |
| 1670年后 | 58岁+ | 集中精力撰写《日知录》 |
| 1682年 | 70岁 | 卒于山西曲沃 |

### 主要著作

| 著作 | 卷数 | 撰写时间 | 内容要旨 |
|------|------|----------|----------|
| **《日知录》** | 32卷 | 积累大半生，约80万字 | 读书札记，涵盖经术、治道、博闻三大类。顾炎武自评"平生之志与业皆在其中" |
| **《天下郡国利病书》** | 120卷 | 1639-1662年 | 从史书、方志、名公文集中辑录农田、水利、交通、赋税、兵防等材料，寻找国贫民弱的根源 |
| **《肇域志》** | — | 长期编纂 | 地理志，与《天下郡国利病书》互补 |
| **《音学五书》** | — | 晚年完成 | 古音学研究，分古韵为10部，为清代古韵学奠基 |
| **《亭林文集》** | 6卷 | — | 文章汇编 |
| **《亭林诗集》** | 5卷 | — | 诗歌汇编 |

### 主要主张

#### 1."天下兴亡，匹夫有责"

**原文**："保天下者，匹夫之贱与有责焉耳矣"（《日知录·正始》）

**核心区分**：
- **"亡国"**：一姓之兴亡，改朝换代
- **"亡天下"**：仁义充塞，率兽食人，人将相食——文明道德的全面崩溃
- 保国，乃君臣肉食者之事；保天下，则匹夫与有责

> 此语后由梁启超凝练为"天下兴亡，匹夫有责"八字。

#### 2. 经世致用

- "君子为学，以明道也，以救世也。徒以诗文而已，所谓雕虫篆刻，亦何益哉？"
- 反对空谈心性、玄学清谈
- 对王阳明心学评价极低，认为是"误人、误世的学问"
- 主张"文不苟作"，写文章"须有益於天下"

#### 3. 众治与分权

- 反对君主"独治"
- 主张"寓封建之意于郡县之中"——在郡县制框架内增加地方自治
- 提出县令长期任职、给予实权的方案

#### 4. 考据学方法

- 强调"博学于文"，但范围更广，既指文章、文字，更指立身处世之道
- 治学必须先立人格："礼义廉耻，是谓四维"
- 注重实地考察，北游期间遍历各省，验证书本知识

### 学术影响

- 被誉为"清学之祖""清代六大宗师"之一
- 开创了清代考据学（朴学/汉学）的先声
- 古韵学的奠基人

## 三、黄宗羲（1610-1695）

### 生平

- **姓名**：黄宗羲，字太冲，一字德冰，号南雷先生，别号梨洲老人
- **生卒**：万历三十八年（1610年9月24日）— 康熙三十四年（1695年8月12日）
- **籍贯**：浙江余姚
- **师承**：刘宗周（蕺山先生）

#### 关键时间线

| 时间 | 年龄 | 事件 |
|------|------|------|
| 1610年 | 出生 | 生于浙江余姚 |
| 1626年 | 17岁 | 父黄尊素（东林七君子之一）被魏忠贤阉党迫害致死 |
| 1628年 | 19岁 | 入京伸冤，当堂以铁锥刺杀凶手许显纯等阉党余孽，崇祯帝许之 |
| 1630年代 | 20s | 拜刘宗周为师，日闻绪言 |
| 1636年 | 27岁 | 参与复社活动 |
| 1644年 | 35岁 | 明亡 |
| 1645-1649年 | 36-40岁 | 招募子弟数百人组成"世忠营"，举兵抗清。南明授监察御史兼兵部职方司主事 |
| 1653年 | 44岁 | 抗清失败，返回故里，课徒授业，著述以终 |
| 1662-1663年 | 53岁 | 撰成**《明夷待访录》** |
| 1663-1676年 | 54-67岁 | 于慈溪、绍兴、宁波等地设馆讲学 |
| 1676年 | 67岁 | 著成**《明儒学案》** |
| 1695年 | 86岁 | 卒于余姚 |

### 主要著作

| 著作 | 卷数 | 完成时间 | 内容要旨 |
|------|------|----------|----------|
| **《明夷待访录》** | 1卷（13篇） | 约1663年 | 批判君主专制、设计理想政治制度 |
| **《明儒学案》** | 62卷 | 1676年 | 中国第一部完整的学术思想史，记载明代210位学者 |
| **《南雷文定》** | 32卷 | — | 文章汇编 |
| **《明文海》** | 482卷 | — | 明代文章总集 |
| **《行朝录》** | 9种 | — | 南明史料 |
| **《南雷诗历》** | — | — | 诗歌汇编 |

### 《明夷待访录》核心观点

**书名含义**：
- "明夷"出自《周易》，明夷卦象征光明受伤、贤者蒙难
- "待访"意为等待明主来访问——为未来的理想政治设计蓝图

**全书结构**：共13篇——原君、原臣、原法、置相、学校、取士（上下）、建都、方镇、田制、兵制、财计、奄宦

#### 三篇核心

| 篇目 | 核心问题 | 核心观点 | 关键论述 |
|------|----------|----------|----------|
| **《原君》** | 天子是为天下人服务，还是让天下人服务？ | "天下为主，君为客" | 批判君主"以我之大私为天下之大公"，"以天下之利尽归于己，以天下之害尽归于人"，实为"天下之大害" |
| **《原臣》** | 臣子是为天子打工，还是与天子一起为天下人打工？ | "为天下，非为君也；为万民，非为一姓也" | "盖天下之治乱，不在一姓之兴亡，而在万民之忧乐" |
| **《原法》** | 制法出发点是为天下还是为天子？ | "有治法然后有治人" | 批判封建法律为"一家之法，而非天下之法"；主张以天下之法代替一家之法 |

#### 其他重要篇章

- **《置相》**：主张恢复宰相制度，制衡君权
- **《学校》**：主张学校不仅是养士之所，更应成为议政机关，"天子之所是未必是，天子之所非未必非"
- **《取士》**：批评八股取士之弊

### 学术影响

- 被后世称为中国近代启蒙思想的先驱
- 《明儒学案》开创了学案体史书体裁的先河
- 与顾炎武、王夫之并称"明清之际三大思想家"

> **来源**：[黄宗羲 - 百度百科](https://baike.baidu.com/item/%E9%BB%84%E5%AE%97%E7%BE%B2/895771)；[明夷待访录 - 百度百科](https://baike.baidu.com/item/%E6%98%8E%E5%A4%B7%E5%BE%85%E8%AE%BF%E5%BD%95/787890)；[《明夷待访录》读书分享 - 知乎](https://zhuanlan.zhihu.com/p/559960710)

## 四、王夫之（1619-1692）

### 生平

- **姓名**：王夫之，字而农，号姜斋、又号夕堂
- **生卒**：万历四十七年（1619年10月7日）— 康熙三十一年（1692年2月18日）
- **籍贯**：湖广衡州府衡阳县（今湖南衡阳）
- **学号**：晚年隐居石船山麓，世称**船山先生**

#### 关键时间线

| 时间 | 年龄 | 事件 |
|------|------|------|
| 1619年 | 出生 | 生于湖南衡阳书香门第 |
| 1642年 | 24岁 | 中乡试举人 |
| 1644年 | 26岁 | 明亡。张献忠攻入衡阳，王夫之拒绝投降 |
| 1648年 | 30岁 | 于衡阳起兵抗清，兵败 |
| 1650年 | 32岁 | 投奔南明桂王政权，任行人司行人 |
| 1651年 | 33岁 | 因弹劾权臣遭排挤，被迫离开南明朝廷 |
| 1652年后 | 34岁+ | 开始隐居生涯，"披发入山"，在多地辗转 |
| 1675年 | 57岁 | 定居石船山麓（湘西草堂） |
| 1685-1691年 | 67-73岁 | 集中撰写《读通鉴论》 |
| 1692年 | 74岁 | 卒于湘西草堂。自题墓志铭"抱刘越石之孤忠" |

### 主要著作

| 著作 | 卷数/字数 | 内容要旨 |
|------|-----------|----------|
| **《读通鉴论》** | 30卷，约60万字 | 评论《资治通鉴》所记各朝历史，系统阐述政治哲学和历史观 |
| **《宋论》** | 15卷 | 评论两宋历史 |
| **《张子正蒙注》** | 9卷 | 注释张载《正蒙》，系统阐述气本论 |
| **《周易外传》** | 7卷 | 借《周易》阐发哲学思想 |
| **《尚书引义》** | 6卷 | 以《尚书》为据阐发政治思想 |
| **《思问录》** | 2卷 | 哲学札记 |
| **《黄书》** | 1卷 | 民族思想 |
| **《噩梦》** | 1卷 | 政治改革设想 |

**总量**：著作涉及政治、经济、哲学、历史、文学、天文、地理、训诂等诸多方面，凡500余万言。后人辑为**《船山遗书》**（288卷，56种）。

### 主要思想

#### 1. 气本论

- "天下惟器而已矣"——否认独立于物质世界之外的"理"
- "理者气之理也"——理是气的规律，不能脱离气而存在
- 直接批判程朱理学"理先气后"和"存天理，灭人欲"

#### 2. 理欲统一

- "天理即在人欲之中"——不能离开人欲空谈天理
- 反对禁欲主义
- 承认人的物质欲望具有正当性

#### 3. 历史观——理势合一

- 提出"理势合一"的历史进步观
- 批判"复古论"和"循环论"
- 认为历史是不断发展进步的——"势之顺者即理之当然者"

#### 4. 人性论——性日生日成

- "性日生而日成"——人性不是一成不变的，而是不断发展变化的
- 人可以主动地权衡和取舍，参与自身人性的塑造
- 反对"性一成而不变"的先验人性论

#### 5. 均天下

- "平天下者，均天下而已"
- 反对土地兼并和贫富悬殊
- 主张限制君权

## 五、方以智（1611-1671）

### 生平

- **姓名**：方以智，字密之，号曼公、鹿起、龙眠愚者等
- **生卒**：万历三十九年（1611年）— 康熙十年（1671年）
- **籍贯**：南直隶桐城县（今安徽桐城）
- **家世**：桐城方氏世家，祖父方大镇、父方孔炤均为学者
- **并称**：与冒襄、侯方域、陈贞慧并称"明末四公子"

#### 关键时间线

| 时间 | 年龄 | 事件 |
|------|------|------|
| 1611年 | 出生 | 生于安徽桐城学术世家 |
| 1640年 | 30岁 | 崇祯十三年中进士，授翰林院检讨 |
| 1644年 | 34岁 | 明亡，流亡岭南 |
| 1646年 | 36岁 | 曾短暂服务于南明桂王政权 |
| 1650年 | 40岁 | 为避清廷追捕，削发为僧，法名弘智，字无可 |
| 1653年 | 43岁 | 住持青原山净居寺（今江西吉安） |
| 1671年 | 61岁 | 在被押解赴粤途中病逝（一说投水自尽） |

### 主要著作

| 著作 | 卷数 | 内容要旨 |
|------|------|----------|
| **《通雅》** | 52卷 | 综合性百科全书式著作，涵盖天文、地理、音韵、文字、器物等。被收入《四库全书》 |
| **《物理小识》** | 12卷 | 涉及天文、历法、风雷、雨、人身、医药、饮食、金石、器用、草木、鸟兽、鬼神、方术等。融汇西方传教士带来的科学知识。被收入《四库全书》 |
| **《药地炮庄》** | 9卷 | 以儒学注释《庄子》，融通儒释道。被收入《四库全书》 |
| **《东西均》** | 1卷 | 哲学著作，调和东西方知识 |
| **《浮山文集》** | — | 诗文集 |

### 学术特色

#### 1. "质测"与"通几"

方以智将学问分为三大类：
- **质测之学**：相当于自然科学和实证研究——考察天地万物、数学、医学等
- **通几之学**：相当于哲学——探究形上本体和终极道理
- **宰理之学**：相当于政治伦理学——治理社会的学问

主张"质测"与"通几"互补，既要实证考察又要哲学反思。

#### 2. 融汇中西

- 接触传教士带来的西方科学知识（天文、物理、数学）
- 在《物理小识》中批判性地吸收西方知识，不盲从
- 曾用实验验证传教士的某些说法

#### 3. 气一元论

- 哲学上持气一元论立场
- "盈天地之间皆物也"
- 与王夫之类似，反对理学的理气二元论

### 学术影响

- 被认为是明末最具科学精神的学者之一
- 《通雅》和《物理小识》是中国古代百科全书式著作的典范
- 在中西文化交流史上具有重要地位
