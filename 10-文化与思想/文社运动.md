# 明代文社运动（1604-1644）

> 聚焦万历至崇祯年间（1573-1644），以东林书院和复社为核心。

---

## 一、东林书院

### 创建背景

| 要素 | 内容 |
|------|------|
| **创建时间** | 万历三十二年（1604年） |
| **创建地点** | 南直隶常州府无锡县（今江苏无锡） |
| **历史渊源** | 宋代杨时（龟山先生）曾在此讲学，书院始建于北宋政和元年（1111年）。明代重修并恢复讲学 |
| **创建人** | 顾宪成、高攀龙等 |
| **创建原因** | 顾宪成被罢官回乡后，与同志重修东林书院，以讲学为名议论朝政 |
| **著名楹联** | "风声雨声读书声声声入耳，家事国事天下事事事关心"——顾宪成撰 |

### 创始人——顾宪成

- **生卒**：嘉靖二十九年（1550年）— 万历四十年（1612年）
- **字号**：字叔时，号泾阳
- **籍贯**：南直隶常州府无锡县
- **科举**：万历八年（1580年）进士
- **仕途**：官至吏部文选司郎中（从五品），因推选阁臣时忤逆神宗意旨被革职
- **罢官**：万历二十二年（1594年）被削籍回乡

### 东林书院的运作方式——会讲制度

#### 《东林会约》

顾宪成制定《东林会约》，规范讲学活动。会约主要内容：

1. **宗旨**：首列孔子、颜渊、曾参、子思、孟子为学要旨
2. **准则**：次列朱熹《白鹿洞学规》
3. **细则**："饬四要、破二惑、崇九益、屏九损"
4. **精神**：胸怀大志，诚意从学，讲与行一致，为国家社会育人

#### 讲会频次与形式

| 类型 | 频次 | 说明 |
|------|------|------|
| **大会** | 每年春秋各一次 | 大型学术研讨会，参加者数百人 |
| **小会** | 除天寒酷暑外每月一次（九、十、十一日） | 小型学术座谈会 |
| **阅读交流** | 每三日一次 | 日常读书讨论 |

#### 讲会规则

- 每次讲会设**会主一名**，主持讲学
- 通常主讲**四书**，以其中一篇为开头
- 涉及先秦诸子百家、历代儒释道及哲学、文学、史学等各家学派
- 讲完后听讲人提问、辩论、研究
- **不分尊卑、不限地区、不论长少、不收学费**，只要愿意均可参加

#### 学术取向

- 以朱熹白鹿洞学规为准则，倾向程朱理学
- 但并非排斥心学，而是试图调和朱王
- 核心是**学以致用**——将学术讨论与现实政治关怀结合
- 批评阳明后学流于空谈禅学

## 四、复社

### 创建与组织

| 要素 | 内容 |
|------|------|
| **创建时间** | 崇祯二年（1629年） |
| **创建地点** | 苏州府吴江县尹山 |
| **领袖** | 张溥、张采，时称**"娄东二张"** |
| **性质** | 以江南士大夫为核心的政治、文学团体，有"小东林""嗣东林"之称 |
| **组成方式** | 由云间几社、香山同社、浙西闻社、江北南社、江西则社等十几个地方社团联合而成 |
| **成员人数** | 先后共计**2000-2255人**，声势遍及海内 |

### 张溥（1602-1641）

- **字号**：字天如，号西铭
- **籍贯**：南直隶太仓州（今江苏太仓）
- **科举**：崇祯四年（1631年）进士
- **著作**：《七录斋集》
- **事迹**：
  - 少年苦读，手抄书籍，每读必手抄七遍，故名书斋"七录斋"
  - 具有卓越的组织才能
  - 崇祯十四年（1641年）病逝，年仅40岁

### 张采（1596-1648）

- **字号**：字受先，号南郭
- **籍贯**：南直隶太仓州
- **科举**：崇祯元年（1628年）进士
- **与张溥并称"娄东二张"**

### 复社三次大会

| 会议 | 时间 | 地点 | 参与人数 | 主要事件 |
|------|------|------|----------|----------|
| **尹山大会** | 崇祯二年（1629年） | 苏州吴江尹山 | 数百人 | 复社正式成立，制定规章和戒律，各地分设专人负责 |
| **金陵大会** | 崇祯三年（1630年） | 南京 | 数百人 | 扩大影响，联络南方士子 |
| **虎丘大会** | 崇祯六年（1633年） | 苏州虎丘 | **数千人** | 最盛大的一次集会，"观者叹为前所未见"。将复社、应社等合并，刊《国表社集》 |

### 复社的宗旨与活动

**学术宗旨**：
- "宗经复古，切实尚用"
- 以揣摩八股、切磋学问、砥砺品行为日常活动
- 反对空谈，主张改良政治，关注社会现实

**政治活动**：
- 以东林后继自任，继续东林党人的政治路线
- 崇祯年间多次干预政治，如推动周延儒复相
- 作品多揭露权奸宦官，同情民生疾苦
- 崇祯十年（1637年）温体仁试图取缔复社，未果

**出版活动**：
- 刊刻八股文范本、学术文集
- 复社文选在科举士子中影响巨大

### 复社的终结

- 张溥于崇祯十四年（1641年）病逝后，复社群龙无首
- 崇祯十五年（1642年）朝廷下令禁止社团活动
- 1644年明亡后，部分复社成员投身抗清，如陈子龙等殉国
- 清顺治九年（1652年）正式取缔

## 五、几社

### 创建

| 要素 | 内容 |
|------|------|
| **创建时间** | 崇祯二年（1629年） |
| **创建地点** | 松江府（今上海松江） |
| **创建人** | 夏允彝、杜麟征 |

| 2 | **陈子龙** | 字卧子，号大樽 | 1608-1647 | 明末最杰出的诗人之一。崇祯十年进士。清兵南下后组织抗清，兵败被俘，投水殉国 |
| 3 | **徐孚远** | 字闇公 | 1599-1665 | 明亡后追随南明，辗转台湾 |

### 几社的活动

- 汇刻八股文范本**《几社壬申文选》**，集六子之文，人各六十首
- 后汇入复社，成为复社在松江地区的核心力量
- 几社成员普遍注重经世致用，陈子龙主持编纂**《皇明经世文编》**（504卷），是明代经世文献的集大成之作
- 明亡后，几社核心成员多参与抗清，殉国者众

## 七、东林书院与复社的比较

| 维度 | 东林书院 | 复社 |
|------|----------|------|
| **成立时间** | 1604年 | 1629年 |
| **核心人物** | 顾宪成、高攀龙（退休官员） | 张溥、张采（青年士子） |
| **成员构成** | 在职或退休官员、士大夫 | 以举人、秀才等青年士子为主 |
| **成员规模** | 数十至数百人 | 2000人以上 |
| **活动形式** | 定期会讲（书院制度） | 大型集会 + 出版活动 |
| **学术取向** | 调和朱王，讲学论政 | 宗经复古，以八股文会友 |
| **政治参与** | 间接影响朝政（舆论） | 直接干预政治（推动罢相复相） |
| **对手** | 浙党、齐党、楚党 → 阉党 | 温体仁、阮大铖等 |
| **结局** | 天启年间被禁毁 | 崇祯末年衰落，清初被取缔 |
| **历史评价** | "东林"成为正直士人的代名词 | "小东林"，继承东林精神 |

## 八、文社运动的历史意义

### 制度层面

1. **组织化的公共空间**：文社创造了独立于官方的公共讨论空间
2. **规范化的学术活动**：《东林会约》等建立了学术会议的制度化规范
3. **网络化的政治联盟**：各地文社形成横向联络网络，突破地域限制

### 思想层面

1. **学以致用**：将学术讨论与现实政治关怀紧密结合
2. **公共舆论**：形成了对朝政的独立评论传统
3. **士人自觉**：强化了知识分子的社会责任感

### 局限性

1. 党争激烈时常流于意气之争，损害了国家决策效率
2. 东林党人的道德理想主义在面对现实政治时往往力不从心
3. 复社后期在政治操作中也出现了派系私利的问题
