# 西学东渐（1521-1644）

## 一、耶稣会传教士入华时间线

### 1.1 总体背景

明万历年间，天主教耶稣会（Society of Jesus，1540年成立）传教士率先进入中国内地，开启了明末中西方科学与文化交流的序幕。此前，方济各会士方济各·沙勿略（Francis Xavier）曾于1552年在广东上川岛病逝，未能进入中国大陆。

### 1.2 主要传教士时间线

| 传教士（中文名/原名） | 国籍 | 来华时间 | 离华/去世时间 | 活动概况 |
|----------------------|------|---------|-------------|---------|
| **罗明坚**（Michele Ruggieri） | 意大利 | 万历七年（1579）抵澳门 | 万历十六年（1588）离华返欧 | 第一位进入中国大陆的耶稣会士。万历十年（1582）与利玛窦同入肇庆，建"仙花寺"，施洗中国第一位天主教徒。首位编纂中文词典的欧洲人。后赴罗马争取教皇派遣使团赴华 |
| **利玛窦**（Matteo Ricci） | 意大利 | 万历十年（1582）抵澳门 | 万历三十八年（1610）卒于北京 | 西学东渐核心人物（详见下文） |
| **熊三拔**（Sabatino de Ursis） | 意大利 | 万历三十四年（1606）至北京 | 天启元年（1621）被驱逐至澳门后去世 | 与徐光启合译《泰西水法》（1612），参与修历 |
| **庞迪我**（Diego de Pantoja） | 西班牙 | 万历二十七年（1599）入华 | 天启元年（1621）被驱逐 | 随利玛窦进北京，进献自鸣钟等物 |
| **艾儒略**（Giulio Aleni） | 意大利 | 万历四十一年（1613）入华 | 顺治六年（1649）卒于福建延平 | 在华36年。著《职方外纪》（世界地理著作）、《西学凡》（介绍欧洲教育体系），被福建教徒称为"西来孔子" |
| **邓玉函**（Johann Terrenz） | 瑞士/德意志 | 天启元年（1621）入华 | 崇祯三年（1630）卒于北京 | 参与历法改革，与王征合译《远西奇器图说》（机械工程著作） |
| **汤若望**（Johann Adam Schall von Bell） | 德意志 | 天启二年（1622）抵澳门，天启三年（1623）至北京 | 康熙五年（1666）卒于北京 | 参与修订《崇祯历书》（137卷），负责铸造火炮，精通天文学。明亡后继续在清廷任职，任钦天监监正 |
| **罗雅谷**（Giacomo Rho） | 意大利 | 天启四年（1624）入华 | 崇祯十一年（1638）卒于北京 | 参与历法改革，与汤若望合作完成《崇祯历书》 |
| **毕方济**（Francesco Sambiasi） | 意大利 | 万历三十八年（1610）入华 | 崇祯二年（1649）卒 | 在南京、北京传教，著有多种汉文著作 |

## 二、利玛窦——西学东渐的核心人物

### 2.1 生平要略

| 项目 | 内容 |
|------|------|
| 原名 | Matteo Ricci |
| 生卒 | 1552-1610 |
| 籍贯 | 意大利马切拉塔（Macerata） |
| 入会 | 1571年加入耶稣会，在罗马学院受教育 |
| 来华路线 | 1578年离欧→果阿→1582年抵澳门 |

### 2.2 在华活动编年

| 时间 | 地点 | 活动 |
|------|------|------|
| 万历十年（1582） | 澳门 | 抵达澳门，学习中文 |
| 万历十一年（1583） | 肇庆 | 与罗明坚同入肇庆，建仙花寺，穿僧服传教 |
| 万历十七年（1589） | 韶州 | 移居韶州，开始改穿儒服 |
| 万历二十三年（1595） | 南昌 | 移居南昌，交结士大夫 |
| 万历二十六年（1598） | 南京 | 移居南京，首次尝试入京（未成功） |
| 万历二十九年（1601） | 北京 | 正式入京，进呈自鸣钟、《坤舆万国全图》等物，获万历帝允准居留 |
| 万历三十八年（1610） | 北京 | 去世，享年58岁。万历帝赐葬地于阜成门外 |

### 2.3 重要著述

- 《天主实义》（1603）——用儒学概念阐释天主教义
- 《畸人十篇》（1608）——中西文化对话录
- 《交友论》（1595）——引介西方友谊观念
- 《坤舆万国全图》（1602）——与李之藻合作制作的世界地图
- 《西国记法》——介绍记忆术
- 《乾坤体义》——介绍西方天文学基础

## 三、传教策略——"合儒"与"利玛窦规矩"

### 3.1 策略演变

| 阶段 | 时间 | 策略 | 说明 |
|------|------|------|------|
| **僧服期** | 1583-1589 | 以佛僧面貌出现 | 剃发穿僧服，称"西僧"，但发现佛僧在中国社会地位不高 |
| **儒服期** | 1589年后 | 改穿儒服，自称"西儒" | 蓄发留须，穿戴儒者衣冠，以学者身份交游 |
| **上层路线** | 贯穿始终 | 交结官员和文人士大夫 | 通过皈依上层人士来推动传教事业 |

### 3.2 "合儒"策略要点

| 策略 | 具体做法 |
|------|---------|
| **儒学附会** | 从中国古代经典（《诗经》《尚书》等）中寻找与天主教义相通之处，主张"上帝"即古儒所言之"天" |
| **补儒排佛** | 赞扬先秦儒学（尤其是其一神论倾向），批判佛教、道教为偶像崇拜 |
| **容忍礼仪** | 允许中国教徒保留祭祖祀孔等传统礼仪，认为这些仅是社会习俗而非宗教行为 |
| **科学传教** | 以西方科学技术（天文、数学、地图、钟表等）作为敲门砖，吸引中国知识精英 |

### 3.3 "利玛窦规矩"

- 利玛窦去世后，其对待中国礼仪的宽容态度被称为"利玛窦规矩"
- 这一规矩后来成为"中国礼仪之争"的核心议题
- 清康熙帝明确维护此规矩："自今以后，若不遵利玛窦的规矩，断不准在中国住，必须逐回"
- 1704年教皇克莱芒十一世禁止中国礼仪后，引发中国与罗马的长期冲突

> 来源：[利玛窦规矩 - 维基百科](https://zh.wikipedia.org/zh-cn/%E5%88%A9%E7%8E%9B%E7%AA%A6%E8%A7%84%E7%9F%A9)；[福建省人民政府](https://www.fujian.gov.cn/hdjl/hdjlzsk/mzzjt/zj/202012/t20201224_5500307.htm)

## 四、中国合作学者——"圣教三柱石"

### 4.1 概述

徐光启、李之藻、杨廷筠三人因皈依天主教并积极推动中西文化交流，被称为"圣教三柱石"或"天主教中国三大柱石"。

> 来源：[圣教三柱石 - 百度百科](https://baike.baidu.com/item/%E5%9C%A3%E6%95%99%E4%B8%89%E6%9F%B1%E7%9F%B3/2918916)

### 4.2 徐光启（1562-1633）

| 项目 | 内容 |
|------|------|
| 字号 | 字子先，号玄扈 |
| 籍贯 | 上海县（今上海） |
| 教名 | 保禄（Paul），万历三十一年（1603）受洗 |
| 仕途 | 万历三十二年（1604）进士，官至礼部尚书兼文渊阁大学士（崇祯朝入阁） |

#### 主要贡献

| 领域 | 贡献 |
|------|------|

| 天文 | 主持修订《崇祯历书》（崇祯二年起） |
| 农学 | 著《农政全书》（60卷），推广甘薯种植 |
| 军事 | 引介西洋火器技术，主张练兵自强 |
| 水利 | 与熊三拔合译《泰西水法》（1612） |
| 数学 | 与利玛窦合译《测量法义》 |
| 宗教 | 翻译教理书籍，劝化家族入教 |

## 五、传入知识分领域详述

### 5.1 数学

| 项目 | 内容 |
|------|------|
| **核心著作** | 《几何原本》前6卷（1607年，利玛窦口译、徐光启笔受） |
| **底本** | 克拉维乌斯（Clavius）注释的拉丁文版欧几里得《原本》 |
| **翻译过程** | 万历三十四年秋至三十五年春（1606-1607），每天下午徐光启赴利玛窦住所，利玛窦逐句讲述，徐光启逐字记录 |
| **术语贡献** | 首创了大量数学专用名词并沿用至今：几何、点、线、面、直角、锐角、钝角、垂线、平行线、对角线、曲线、曲面、立方体、体积、比例等 |
| **后续影响** | 后6-15卷直到1857年才由李善兰与伟烈亚力翻译完成（相隔250年） |
| **其他数学著作** | 《同文算指》（1613，利玛窦/李之藻，介绍西方笔算法）、《测量法义》（利玛窦/徐光启）、《圜容较义》（利玛窦/李之藻）、《勾股义》 |

> 来源：[几何原本 - 书格](https://www.shuge.org/view/ji_he_yuan_ben/)；[澎湃新闻](https://www.thepaper.cn/newsDetail_forward_1827078)

### 5.2 天文学与历法

| 项目 | 内容 |
|------|------|
| **早期传入** | 利玛窦《乾坤体义》——初步介绍托勒密"九重天说"和亚里士多德"四元行论" |
| **天文仪器** | 利玛窦制作天球仪、地球仪、日晷、象限仪、纪限仪等 |
| **历法改革** | 崇祯二年（1629），因钦天监预报日食又误，徐光启受命督修历法，设"历局"于北京 |
| **《崇祯历书》** | 崇祯三年至七年（1630-1634），由徐光启主持，汤若望、罗雅谷等参与，历时十余年完成，共46种137卷。引入第谷·布拉赫（Tycho Brahe）天文体系 |
| **望远镜** | 崇祯七年（1634），汤若望与罗雅谷向崇祯帝进献望远镜 |
| **关键著作** | 《天问略》（阳玛诺）、《远镜说》（汤若望）、《经天该》 |

### 5.3 地理与地图学

| 项目 | 内容 |
|------|------|
| **《坤舆万国全图》** | 1602年由利玛窦与李之藻合作完成并刊印于北京。六幅拼接，高约1.52米、宽约3.66米。采用伪圆柱投影法，将中国置于地图中央。先后被12次翻刻。万历帝于1608年命绘制12份副本 |
| **此前版本** | 1584年利玛窦在肇庆即编制过《山海舆地全图》，后经多次修订 |
| **《职方外纪》** | 艾儒略著（约1623年成书），详细介绍五大洲地理 |
| **《坤舆图说》** | 南怀仁（Ferdinand Verbiest）著（清初） |
| **地球仪** | 利玛窦制作并展示地球仪，让中国人直观理解地球为球体 |

> 来源：[坤舆万国全图 - 维基百科](https://zh.wikipedia.org/zh-hans/%E5%9D%A4%E8%BC%BF%E8%90%AC%E5%9C%8B%E5%85%A8%E5%9C%96)；[坤舆万国全图 - 书格](https://www.shuge.org/view/kun_yu_wan_guo_quan_tu/)

### 5.4 火器与军事技术

| 项目 | 内容 |
|------|------|
| **徐光启引介** | 徐光启从利玛窦处学习西洋军事科学知识。《明史·徐光启传》称"从西洋人利玛窦学……火器，尽其术" |
| **主要内容** | 火器制造技术、筑造铳台（炮台）方法 |
| **实战应用** | 崇祯年间，为对抗后金（清），朝廷通过传教士从澳门购入西洋大炮（"红夷大炮"），汤若望亦参与铸炮 |
| **相关著作** | 《火攻挈要》（汤若望协助编纂）——系统介绍西方火炮技术 |

### 5.5 钟表与机械

| 项目 | 内容 |
|------|------|
| **进呈自鸣钟** | 万历二十九年（1601），利玛窦入京时进呈自鸣钟（机械钟表）等物品 |
| **万历帝的反应** | 万历帝对自鸣钟极感兴趣，命宫中太监学习钟表维护 |
| **文化影响** | 自鸣钟不仅是计时工具，更引发了中国人对西方机械技术的好奇 |
| **《远西奇器图说》** | 邓玉函口述、王征笔受（1627），3卷，介绍西方机械原理和器具（杠杆、滑轮、螺旋等），是中国第一部系统介绍西方力学和机械学的著作 |

### 5.6 其他领域

| 领域 | 传入内容 |
|------|---------|
| **哲学与逻辑** | 《名理探》（傅泛际口译、李之藻笔受）——亚里士多德逻辑学 |
| **美术** | 西方油画技法（传教士带来的宗教画作引起好奇） |
| **音乐** | 西方音乐乐器（如古钢琴/克拉维科德） |
| **医学** | 西方解剖学基础知识（虽传入有限） |
| **建筑** | 西方建筑技法（教堂建设中有所体现） |

## 七、制度与数据层面

### 7.1 制度层面

| 方面 | 内容 |
|------|------|
| **居留许可** | 传教士在中国居住需获得地方官员默许或朝廷许可，无统一制度。利玛窦最终获万历帝"恩准"居留 |
| **教案** | 万历四十四年（1616）"南京教案"——南京礼部侍郎沈灌上疏攻击传教士，多名传教士被驱逐至澳门 |
| **历局制度** | 崇祯二年（1629）设历局于北京，聘用传教士参与历法改革，是中国政府首次正式雇用西方技术人员 |
| **传教士官职** | 个别传教士获授官职（如汤若望在崇祯年间协助铸炮，明亡后在清廷任钦天监监正） |
