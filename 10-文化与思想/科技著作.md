# 明代科技著作（1521-1644）

## 一、李时珍《本草纲目》

### 1.1 作者生平

| 项目 | 内容 |
|------|------|
| 姓名 | 李时珍，字东璧，号濒湖 |
| 生卒 | 正德十三年至万历二十一年（1518-1593） |
| 籍贯 | 湖广蕲州（今湖北蕲春） |
| 家世 | 出身医学世家，父李言闻亦为当地名医 |
| 仕历 | 曾任楚王府奉祠正（良医所） |

### 1.2 编写过程

| 阶段 | 时间 | 内容 |
|------|------|------|
| **立志编修** | 嘉靖三十一年（1552） | 感于历代本草著作中错误和遗漏甚多，决心重新编纂 |
| **实地考察** | 1552-1578 | 历时27年，足迹遍及湖广、江西、安徽、江苏、河南等地，深入山区采集药物标本 |
| **参考文献** | — | 参阅古今书籍800余种，取访四方 |
| **三次改写** | 1552-1578 | 全书前后修改三次，经历三次大的改写 |
| **定稿** | 万历六年（1578） | 历27年完成定稿 |

### 1.3 出版历程

| 时间 | 事件 |
|------|------|
| 1578年定稿后 | 在蕲州、黄州、武昌寻找书商承印，无人愿意 |
| 1579年 | 赴南京寻求出版，未获成功 |
| 万历十八年（1590） | 获南京藏书家兼出版商胡承龙赞助，开始刻版 |
| 万历二十一年（1593） | 李时珍去世，未能看到刊行 |
| **万历二十三年（1596）** | 在南京正式刊行（**金陵胡承龙本**，即"金陵本"） |

> 从完稿到出版历经18年，从开始编写到出版共计44年。

### 1.4 全书规模

| 数据项 | 数量 |
|--------|------|
| 总卷数 | 52卷 |
| 总字数 | 约190万字 |
| 收录药物 | 1892种 |
| 其中新增药物 | 374种（李时珍新增） |
| 药物图 | 1109幅 |
| 方剂 | 11096首（其中8000余首由李时珍自己收集和拟定） |
| 分类 | 16部60类 |

### 1.5 内容分类

全书将1892种药物分为16部：

| 部 | 内容 |
|----|------|
| 水部 | 天水类、地水类 |
| 火部 | 各类火 |
| 土部 | 各类土 |
| 金石部 | 金属矿物 |
| 草部 | 各类草药（最大部分） |
| 谷部 | 谷物作物 |
| 菜部 | 蔬菜类 |
| 果部 | 水果类 |
| 木部 | 树木类 |
| 服器部 | 衣物器具 |
| 虫部 | 昆虫类 |
| 鳞部 | 有鳞动物 |
| 介部 | 有甲壳动物 |
| 禽部 | 鸟类 |
| 兽部 | 兽类 |
| 人部 | 人体相关 |

### 1.6 学术价值

- 是中国古代最完备的本草学（药物学）著作
- 创立了当时世界上最先进的药物分类法（按自然属性分类，从无机物到有机物、从低等到高等）
- 纠正了前代本草著作中大量错误
- 1596年出版后，先后被翻译为日、朝、拉丁、法、德、英、俄等多种文字
- 达尔文称其为"中国古代的百科全书"

> 来源：[本草纲目 - 维基百科](https://zh.wikipedia.org/zh-hans/%E6%9C%AC%E8%8D%89%E7%BA%B2%E7%9B%AE)；[湖北省文化和旅游厅](http://wlt.hubei.gov.cn/bmdt/mtjj/202208/t20220824_4277457.shtml)

## 二、宋应星《天工开物》

### 2.1 作者生平

| 项目 | 内容 |
|------|------|
| 姓名 | 宋应星，字长庚 |
| 生卒 | 万历十五年至？（1587-约1666） |
| 籍贯 | 江西奉新县 |
| 仕历 | 举人出身，曾在江西、福建等省任地方官（分宜教谕、亳州知州等），后辞官回乡 |

### 2.2 基本信息

| 项目 | 内容 |
|------|------|
| 成书时间 | 崇祯十年（1637） |
| 刊行时间 | 崇祯十年（1637），友人涂绍煃资助刊印 |
| 总卷数 | 3卷18篇 |
| 性质 | 综合性科学技术著作，记述中国古代各项生产技术和工艺 |

### 2.3 内容分类

全书分上、中、下三卷，共十八篇：

| 卷 | 篇目 | 涉及领域 |
|----|------|---------|
| **上卷** | 乃粒（谷物）、乃服（纺织）、彰施（染色）、粹精（粮食加工）、作咸（制盐）、甘嗜（制糖） | 农业与食品加工 |
| **中卷** | 陶埏（陶瓷）、冶铸（金属铸造）、舟车（造船与车辆）、锤锻（锻造）、燔石（烧制石灰砖瓦）、膏液（榨油）、杀青（造纸） | 手工业与制造 |
| **下卷** | 五金（金属开采冶炼）、佳兵（兵器制造）、丹青（颜料制造）、曲蘖（制曲酿酒）、珠玉（宝石加工） | 矿业、军工与奢侈品 |

### 2.4 重要科技记录

- **锌的冶炼**：书中记录的锌冶炼技术是世界上最早的文字记载
- **提花机**：详细记录了能织出精美丝绸的提花机结构
- **养蚕技术**：系统记述了从养蚕到缫丝的全过程
- **炼铁炼钢**：记录了生铁炒钢法等冶金工艺
- **配图**：全书附有大量精美的技术插图

### 2.5 历史命运

- 清代因书中多处涉及满族早期历史的敏感内容，未被收入《四库全书》
- 在国内一度散佚
- 日本最早翻刻（1771年），法文译本（1869年）、英文译本（1966年）先后出版
- 被英国学者李约瑟称为"中国的狄德罗"所著的"技术百科全书"

> 来源：[天工开物 - 维基百科](https://zh.wikipedia.org/zh-hans/%E5%A4%A9%E5%B7%A5%E5%BC%80%E7%89%A9)；[天工开物 - 百度百科](https://baike.baidu.com/item/%E5%A4%A9%E5%B7%A5%E5%BC%80%E7%89%A9/29312)

## 三、徐光启《农政全书》

### 3.1 作者生平

| 项目 | 内容 |
|------|------|
| 姓名 | 徐光启，字子先，号玄扈 |
| 生卒 | 嘉靖四十一年至崇祯六年（1562-1633） |
| 籍贯 | 南直隶松江府上海县（今上海） |
| 仕历 | 万历三十二年（1604）进士，官至礼部尚书兼文渊阁大学士（入阁拜相） |
| 信仰 | 万历三十一年（1603）受洗入天主教，教名保禄 |

### 3.2 基本信息

| 项目 | 内容 |
|------|------|
| 成书过程 | 徐光启在世时已初具规模，但因忙于主持修历，未能最终定稿 |
| 编定者 | 徐光启去世后，由门人陈子龙等负责修订整理 |
| 刊行时间 | 崇祯十二年（1639），徐光启去世后6年 |
| 总卷数 | 60卷 |
| 总字数 | 50余万字 |

### 3.3 内容分类

全书分为12目：

| 目 | 卷数 | 主要内容 |
|----|------|---------|
| 农本 | 3卷 | 农业政策总论、历代农政沿革 |
| 田制 | 2卷 | 土地制度与田赋 |
| 农事 | 6卷 | 耕作技术、农时 |
| 水利 | 9卷 | 灌溉、治水工程（含《泰西水法》内容） |
| 农器 | 4卷 | 农具介绍 |
| 树艺 | 6卷 | 谷物、蔬菜种植 |
| 蚕桑 | 4卷 | 养蚕与桑树栽培 |
| 蚕桑广类 | 2卷 | 棉麻等纤维作物 |
| 种植 | 4卷 | 果树、药材种植 |
| 牧养 | 1卷 | 家畜饲养 |
| 制造 | 1卷 | 农产品加工 |
| **荒政** | **18卷** | **救荒备荒（全书最大部分）** |

### 3.4 特色与价值

- "杂采众家，兼出独见"——以系统摘编前人文献为主，同时融入徐光启自己的研究成果
- 引入西方农业和水利知识（《泰西水法》等）
- **荒政**部分占全书近三分之一，反映了晚明灾荒频仍的现实
- 特别推广甘薯种植以备荒——徐光启是中国推广甘薯（番薯）的先驱之一
- 是中国古代最完备的综合性农学著作

> 来源：[农政全书 - 维基百科](https://zh.wikipedia.org/zh-hans/%E5%86%9C%E6%94%BF%E5%85%A8%E4%B9%A6)；[农政全书 - 百度百科](https://baike.baidu.com/item/%E5%86%9C%E6%94%BF%E5%85%A8%E4%B9%A6/843609)

## 四、徐霞客《徐霞客游记》

### 4.1 作者生平

| 项目 | 内容 |
|------|------|
| 姓名 | 徐弘祖，字振之，号霞客 |
| 生卒 | 万历十五年至崇祯十四年（1587-1641） |
| 籍贯 | 南直隶江阴县（今江苏江阴） |
| 身份 | 终身未仕，以旅行考察为毕生事业 |

### 4.2 旅行历程

| 时间跨度 | 内容 |
|---------|------|
| 万历四十一年（1613）起 | 开始有系统的旅行记录 |
| 至崇祯十二年（1639） | 最后一次远行结束（在云南旅行时双足患疾） |
| 总计 | 旅行考察34年 |

#### 足迹范围

遍及今江苏、浙江、山东、河北、山西、陕西、河南、安徽、江西、福建、广东、湖南、湖北、广西、贵州、云南等21个省区。其中以西南地区（广西、贵州、云南）考察最为深入。

### 4.3 著作信息

| 项目 | 内容 |
|------|------|
| 写作时间 | 1613-1639年旅行期间 |
| 编定过程 | 原稿经王忠纫手校、季梦良续成，成书于崇祯十五年（1642） |
| 总字数 | 遗有60余万字 |
| 版本 | 世传本有10卷、12卷、20卷等数种 |
| 内容形式 | 以日记体为主，按日记述旅行观察所得 |

### 4.4 主要内容

| 部分 | 内容 |
|------|------|
| 名山游记 | 天台山、雁荡山、黄山、庐山、嵩山、华山、太和山（武当）等17篇 |
| 地方游日记 | 《浙游日记》《江右游日记》《楚游日记》《粤西游日记》《黔游日记》《滇游日记》等 |

### 4.5 科学价值

- **地貌学**：对喀斯特（石灰岩）地貌进行了系统观察和记述，是世界上最早的喀斯特地貌考察文献之一
- **水文学**：考察了多条河流的源头和水系
- **植物学**：记录了大量植物的分布和习性
- **地质学**：观察了温泉、火山遗迹等地质现象
- **旅行精神**：被誉为"千古奇人"，是中国系统考察自然地理的先驱

> 来源：[徐霞客游记 - 百度百科](https://baike.baidu.com/item/%E5%BE%90%E9%9C%9E%E5%AE%A2%E6%B8%B8%E8%AE%B0/739945)；[浙江大学](http://gs.zju.edu.cn/2020/0611/c35101a2153801/page.htm)

## 五、其他重要科技著作

### 5.1 数学

| 著作 | 作者 | 成书/刊行 | 卷数 | 内容 |
